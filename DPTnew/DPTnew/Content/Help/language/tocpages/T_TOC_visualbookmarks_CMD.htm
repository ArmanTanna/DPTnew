Visual Bookmarks

Activate Enables you to activate a Visual Bookmark
Delete Enables you to delete a Visual Bookmark or a Category collecting a set of Visual Bookmarks
Export Enables you to export a Visual Bookmark or a Category collecting a set of Visual Bookmarks
Import Visual Bookmark Enables you to import a Visual Bookmark or a Category collecting a set of Visual Bookmarks
Modify Enables you to modify an existing Visual Bookmark
New Visual Bookmark Enables you to create a new Visual Bookmark
Recapture Enables you to recapture a Visual Bookmark
Reverse Enables you to reverse the applied Visual Bookmark
Reload Tree Refreshes the Visual Bookmark Tree Panel
Rename Enables you to rename an existing Visual Bookmark/Category
View Only Enables you to view the Visual Bookmark (Without editing it).
View Exploded Assembly Enables you to view the Visual Bookmark of the Exploded Assembly (Without editing it).
