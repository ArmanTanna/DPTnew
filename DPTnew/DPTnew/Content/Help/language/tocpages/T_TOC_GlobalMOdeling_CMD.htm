Global Shape Modeling

GSM Bend Enables you to bend objects.
GSM Radial Bend Enables you to radially bend objects.
GSM Twist Enables you to twist objects.
Advanced GSM Enables you to model the shape of objects giving you complete control on it.
GSM Replicate Enables you to apply controlled modifications selected on reference entities to other entities to be modified.
Planar Bounding-Box Enables you to define a controlled modification using a set of control points lying on a plane of your choice.
3D Bounding-Box Enables you to define a controlled modification using a set of vertices in the 3D space.
GSM Spined Twist Enables you in providing compensation to those parts in which the spring-back effect generates a twist. Using this command you can provide twist to a part along a predefined spine.
GSM Fillet Reduction Enables you to modifying a set of selected fillet surfaces by changing their curvature, while retaining the desired continuity at borders.
