Format

Colors and Materials Enables you to define environment colors.
Color Enables you to set the current colors.
Color (button) Enables you to set the current color and to change the color of selected entities.
Current Layer Enables you to set the selected layer as current layer.
Lighting Enables you to edit the light studio.
Environment Enables you to edit the environment.
Line Type Enables you to set the current line type.
Line Type (button) Enables you to set the current line type and to change the line type of selected entities.
Line Width Enables you to set the current line width.
Line Width (button) Enables you to set the current line width and to change the line width of selected entities.
Organize Styles Enables you to define or to change the current Styles.
Organize Formats Enables you to define or to change the current Formats.
Organize Weldings Enables you to define welding standard or select the welding type.
Organize Surface textures Enables you to change the current values of surface roughness.
Organize Taper designations Enables you to define or to change the current Formats.
Organize Hole Table Enables you to define or to change the current hole table properties.
Organize Part List Enables you to define or to change the current Part List properties.
