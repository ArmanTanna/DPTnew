History

Roll Back Enables you to cancel the last operation for solids and features.
Roll Forward Enables you to re-apply the last operation cancelled.
Insert Feature Mode Enables you to insert additional features in a model, at a specific feature event that you select within the Model Structure, where a line of demarcation is displayed.
Show Driving Dimensions Enables you to display one or more drive dimensions.
Hide Driving Dimensions Enables you to hide one or more drive dimensions.
Delete Feature Enables you to remove a feature.
Redefine Feature Enables you to change the parameters of the operation used to create a selected feature.
Reorder Feature Enables you to change the order in which a feature is applied to a solid.
Rebuild Model Enables you to update the model following modification of its parametric dimensions.
Reload Structure Enables you to rebuild the ModelDrawing Structure.
Info Structure (root) Displays information about the structure of the ModelDrawing Structure.
Deactivate Temporarily disables a feature without canceling it from the structure.
Reactivate Activates again a feature that was previously deactivated.
Dismantle Enables you to isolate a selected solid within the Model Structure so that you can cancel selected events within its branch of the tree.
Search Enables you to make a search for finding a particular entity in the Model Structure.
