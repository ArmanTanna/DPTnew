Selection

Select All Enables you to select all visible entities
Select Chain Enables you to select a chain of connected curves
Selection Filter Enables you to select entities according to one or more properties common to all of them
Select inside Groups Enables you to select an entity belonging to a group (such as a curve or a line), rather than the group itself
Select Face Enables you to select an individual component of a solid (such as a face or edge), rather than the solid itself
Select Lasso Enables you to select all the entities within a polyline which you define by indicating its points
Select Query Enables you to select entities that are hidden or overlapped
Select Solid Enables you to select the entire solid, rather than its individual component entities
Select Window Enables you to select all the entities inside, crossing or outside a rectangular area
Select Propagated Faces Enables you to make selection of faces in a group that propagate from one another
Select Tangent Faces Enables you to select a group of faces that tangentially propagate from each other
Select Loop Enables you to select curves inside solids
Select From-To Loop Enables you to select only a portion (defined by a start edge and an end edge) of the loop formed by a set of edges
Select Tangent Chain Enables you to select edges tangentially linked to each other
Select Contour Enables you to select curves inside solids
Select Siblings Enables you to select sibling entities
Select Leaf Component Enables you to select and redefine components lower down in the Model Structure without having to set them current
Select Top Component Enables you to select only top level components in the history
