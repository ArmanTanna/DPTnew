Creating Curves

Insert Curve through Control points Enables you to create curves by indicating their control points
Insert Curve through Interpolation Points Enables you to create curves by indicating their interpolation points
Fit Curve Enables you to create NURBS curves fitting points or curves
Hermite Spline Curve Enables you to create cubic spline curves specificing tangency conditions in each of the definition points
Helix Curve Enables you to create helix curves
Join Curve Enables you to create a new curve by joining consecutive curves into just one
Offset Curve on Surface Enables you to create offset curves lying on surfaces
Connect Curve Enables you to create curves connecting other curves
Ellipse Enables you to create ellipses
Hyperbola Enables you to create hyperbolas
Parabola Enables you to create parabolas
Conic Curve Enables you to create conic curves
3D from 2D Curves Enables you to create 3D curves from couples of 2D curves
Spined Curve Enables you to create curves to be used when creating sweep objects along a path
Copy Curve Enables you to create copies of existing curves
Isoparametric Curve Enables you to create isoparametric curves on surfaces
Boundary Curve Enables you to create curves on surface boundaries
Light Contours Enables you to create silhouette curves of objects
Cutting Line Enables you to perform the cutting step of the metal stamping process
Curve Intersection Enables you to create surface intersection curves
Curve Section Enables you to create section curves
Project Enables you to project/map/lay down curves onto surfaces
