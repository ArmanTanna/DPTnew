Modify

Pan/Zoom/Rotate Enables you to pan, zoom, or rotate the view.
Set Target Position Enables you to specify a new target point.
Fit Increases or decreases the zoom magnification so that the entire model fits exactly within the current display.
Zoom Window Adjusts the zoom magnification by enlarging the area within a rectangular window.
Zoom Entities Increases or decreases the zoom magnification so that selected entities fit exactly within the current display.
Zoom on Point of Interest Zooms on the point of interest of a selected annotation.
Pan Translates or shifts the view of the model or drawing.
View on Work Plane Reorients the view of the model so that it appears as a Top view perpendicular to the Work Plane.
Set Direction Enables to align the view direction with respect to a direction identified by a line, a tangent to a curve, or a perpendicular to a surface/solid face.
Properties Enables you to change the properties of the active view.
Step Actions List of the commands controlling the view by step actions.
Set Current View Enables you to define the current view.
Shrink Enables you to shrink a view of the modeldrawing along one of the axes.
Unshrink Enables you to reset a shrunk view.
Remember Enables you to capture and store a particular view orientation of the model.
Define Section View Sections the view of the model along a plane.
Recall Enables you to bring back the last remembered view on screen.
