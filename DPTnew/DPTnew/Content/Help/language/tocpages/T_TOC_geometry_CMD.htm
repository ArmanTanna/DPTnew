Drafting

Insert

Polyline Enables you to create geometric shapes consisting of consecutive lines and arcs.
Fillets Enables you to create arc fillets of a specified radius.
Chamfer Enables you to create a bevel or chamfer line connecting two lines.
Offset on Plane Creates plane offset curves.
X-Hatch Enables you to hatch a closed area.
Text Inserts an item of text whose properties are the currently set properties.
Lines Enables you to create lines.
Circles and Arcs Enables you to create circles and circular arcs.
Rectangles and Polygons Enables you to create rectangles and regular polygons.

Modify

Text Enables you to edit the content of a selected text entity.
Line/Arc Enables you to modify length and direction of lines/arcs.
