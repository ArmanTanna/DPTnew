Capture & Playback
Capture Enables you to record to a log file all the actions performed during a working session
Playback Enables you to play back the actions previously recorded to the current log file or Play list
Debug mode Enables you to play back the current log file in step-by-step mode
Capture & Playback Options Enables you to set the options controlling the way the Capture & Playback utility is performed
