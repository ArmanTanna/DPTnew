Profile Environment

2D Profile Activates the 2D profile environment
3D Profile Activates the 3D profile environment
Add Entities Enables you to add an entity to a profile
AutoConstraint Automatically adds constraints
Check Checks the constraints on a selected profile
Coincidence Constraint Adds a coincidence constraint
Collinearity Constraint Adds a collinearity constraint
Coplanarity Constraint Adds a coplanarity constraint
Curve on Edge Constrains a profile to one or more selected edges of a solid or curves of a profile
Equal Length/Radius Constraint Adds a the Equal Length/Radius constraint
Ground Constraint Constrains a point so that individual or all of its coordinates are fixed with respect to the Reference System
Make/Unmake Reference Adds/removes a reference curve to/from a profile
Move/Copy to Current Work Plane Moves a profile from its placement plane to the current Work Plane
Offset Constraint Adds an offset constraint
Orientation Constraint Adds a constraint to fix the orientation of an entity
Parallelism Constraint Adds a parallalism constraint
Perpendicularity Constraint Adds a perpendicularity constraint
Point-on-Curve Constraint Enables you to constrain entities in a profile such a selected point lies on a target curve
Point-on-Plane Constraint Enables you to constrain entities in a 3D profile such a selected point lies on a target plane
Remove/Substitute Entity Removes/Substitutes one entity in a profile with another
Symmetry Constraint Adds a symmetry constraint
Equal Curvature Constraint Adds an equal curvature constraint
Switch to Model Switches to the Model environment
Tangency Constraint Adds a tangency constraint
Underconstrained Profiles Displays a list of the underconstrained profiles of the model along with their degrees of freedom
Update Updates a profile when resolving is not automatic
Convert Profile Converts the profiles that are created using the old versions of the program (prior to the release 2006.2)
