Windows

Cascade Enables you to arrange windows in a cascade.
Close Enables you to close a window.
Close All Enables you to close all the open windows.
Next Enables the next window and brings it to the foreground.
Previous Enables the previous window and brings it to the foreground.
Tile Horizontally Arranges the windows horizontally so that they are all completely visible.
Tile Vertically Arranges the windows vertically so that they are all completely visible.
Windows Lists all the windows and allows you to enable each of them individually.
