Dimensions

Ordinate Creates dimensions using a reference point.
Smart Enables you to insert dimensions.
Chamfer Enables you to insert chamfer dimensions.
Taper Enables you to insert taper dimensions.
Length Creates a linear dimension for the length of the entity.
Point & line Creates a linear dimension from the linear entity to a special point, measured perpendicular to the linear entity.
Half View Creates a linear dimension from the linear entity to a special point, measured perpendicular to the entity, where the dimension is actually twice the distance.
Angular Creates an angular dimension indicating the angle between the linear entity and the X-axis, the Y-axis, or an axis projected at a user-specified angle.
Parallel Creates a linear dimension indicating the distance between the linear entity and another entity parallel to it.
Snap point Creates a linear dimension measuring the distance between two special points.
Chain Creates a string of linear dimensions drawn end-to-end.
Parallel (Snap point) Creates a set of linear dimensions measured from a common starting point.
Radius Creates a radius dimension.
Diameter Creates a diameter dimension.
Arc length Creates an arc length dimension.
Angular Dimensions Creates angular dimensions between two non-parallel entities.
Angular chain Creates a string of Angular dimensions drawn end-to-end.
Angular parallel Creates Angular dimensions, drawn with a single reference point.
Align Aligns one or more existing dimensions.
Label Enables you to insert either a label or a balloon.
Move Enables you to move or move portions of a dimension.
Modify Enables you to move or edit portions of a dimension.
Surface Textures Enables you to insert surface texture symbols.
Feature Control Frame Enables you to insert geometric tolerance symbols, including feature control frames, datum features, and datum targets.
Datum Reference Enables you to insert geometric tolerance symbols, including feature control frames, datum features, and datum targets.
Datum Target Enables you to insert geometric tolerance symbols, including feature control frames, datum features, and datum targets.
Change View Enables you to move a dimension from one view to another.
Hole Table Enables you to create a hole table in the drawing for all selected circles.
Reference Grid Enables you to create reference grid for the selected drawing and section views.
Modify Reference Grid Enables you to modify a hole table in the drawing for all selected circles.
Modify Reference Grid Enables you to modify reference grid for the selected drawing and section views.
Edit Format/Properties Enables you to edit the format and entity properties of the dimensions created.
Edit Family Enables you to modify all the dimensions that belong to the Ordinate dimensions family.
Welding Section Enables you to draw the section of a weld.
Welding Bead Enables you to draw the bead of a weld along a chain of curves that define the weld path.
Welding Symbol Enables you to insert the symbol of a weld.
