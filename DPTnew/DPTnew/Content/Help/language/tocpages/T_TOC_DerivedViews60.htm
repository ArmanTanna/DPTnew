Derived Views

Projected Inserts a derived two-dimensional drawing view that is projected in a standard orthographic direction from an existing main view.
Auxiliary View Inserts a derived two-dimensional auxiliary drawing view based on a reference line in an existing main view.
Detail View Inserts an enlarged detail view of an area of a main view bounded by a spline, rectangular, or circular border.
Cross-Section view Inserts a cross-section view.
Sectioned view Inserts a section view.
Developed view Inserts a developed view of a section plane.
