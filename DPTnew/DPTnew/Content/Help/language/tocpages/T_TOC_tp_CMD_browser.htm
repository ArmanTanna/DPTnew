thinkparts

Catalog Browser commands
Copy (References) Copies the selected part family reference and can be pasted in to another index
Cut (References) Deletes the selected part family reference when it is pasted to another index
Delete (References) Deletes the selected part family reference from the selected index
Edit Values Enables you to modify the dimensions of the components in the catalog, or add new ones
Export Dumps part or the whole catalog to a portable file
Favorite Enables you to control the display of an index or catalog
Import Adds an external catalog to the current installation
Load Template Enables you to modify the template of the selected catalog
Non-favorite Enables you to control the display of an index or catalog
Open Opens the selected catalog enabling you to use its parts
Paste (References) Moves the cut reference or creates a copy of the copied one to the selected part family index
Properties Enables you to access the properties of the selected index or family of parts
Reload Tree Updates the information displayed in the catalog tree frame
Report in CSV Format Loads the data structure of the catalog to a Microsoft Excel sheet
Send Enables you to send a catalog through your e-mail client
