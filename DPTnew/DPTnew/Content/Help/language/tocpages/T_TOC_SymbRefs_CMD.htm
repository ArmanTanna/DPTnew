Symbolic References

New Symbolic Reference Enables you to declare any geometric entity created by a parametric operation to be a Symbolic Reference
View Symbolic References Displays the Symbolic References that exist at the document current level or inside a component and enables you to change their name, redefine and delete them
Solve Ambiguities Enables you to solve ambiguities involving Symbolic References by selecting the component where ambiguities occur and specifying the Symbolic Reference to be adopted
Redefine Enables you to redefine an existing Symbolic Reference
Delete Enables you to delete the selected Symbolic Reference(s)
Rename Enables you to rename an existing Symbolic Reference
