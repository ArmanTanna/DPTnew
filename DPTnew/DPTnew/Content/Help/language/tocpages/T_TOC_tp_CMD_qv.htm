thinkparts

Catalog View commands
Assign Part Number Assigns the part data to a part in an open catalog
Copy (References) Copies the selected reference awaiting to paste it to another index
Cut (References) Deletes the selected reference when it is pasted to another index
Delete (References) Deletes the selected reference from the selected index
Filters ( thinkparts View) Defines a filter on the data displayed
Import Imports the selected part as component in the model or drawing
Paste (References) Moves the cut reference or creates a copy of the copied one to the selected index
Properties Displays the part data of selected catalog part
Properties ( thinkparts View) Defines the query view layout of the specified catalog
Remove All Filters ( thinkparts View) Removes the filters on the data
Remove Part Number Removes the part number from the selected entry in the thinkparts catalog
