Accessing the Attributes of a Button

This operation lets you display, add and modify attributes of a customized button. The attributes include the button's content, application parameter, full description, tooltip etc.

To access the attributes of a customized button, do the following:

  1. Select ToolsCustomize from the menu bar to open the Customize dialog box.

  2. Move the mouse pointer onto the button whose attributes you want to access and right-click.

  3. Choose Properties from the toolbar popup.

  4. The User Interface Programming dialog box is displayed. It contains the various button attributes, as shown below.
