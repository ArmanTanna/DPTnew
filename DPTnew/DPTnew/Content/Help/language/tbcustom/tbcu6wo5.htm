Copying an icon from one button to another

To copy the icon of a button on any toolbar and paste it to another button on the same toolbar or on another toolbar, do the following:

  1. Select ToolsCustomize from the menu bar to open the Customize dialog box.

  2. Move the mouse pointer over the button whose icon you want to copy and right-click.

  3. Choose Copy Button Icon from the toolbar popup.

  4. Move the mouse pointer over the new button to which you want to paste the copied icon and right-click.

  5. Choose Paste Button Icon from the toolbar popup.



    The appearance of the button gets changed.



Note
  • You cannot change the appearance of a button that is displayed in Buttons area of the Customize dialog box.
  • You cannot copy the icon ( ) of a button that has just been created.
  • When there is no icon to be pasted, the Paste Button Icon option in the toolbar popup is disabled.
