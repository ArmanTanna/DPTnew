Creating a new Dimension Properties Toolbar

In order to create a new Dimension Properties toolbar that collects properties applicable to dimension and text entities, proceed as follows.

  1. Select Tools Options/Properties from the menu bar.

  2. Open the Entity Properties tab and expand the options tree frame to reach the Dimension or Text subcategory that contains the property you wish to add to the new Dimension Properties toolbar.

  3. Right-click the property and select the New Toolbar Object command.

  4. In the New Toolbar Object dialog box that opens, click to uncover the Toolbar area and then click the New Toolbar button.

  5. In the New Toolbar dialog box that opens, enter the name of the toolbar in the Toolbar name: field (for example, TB01) and then click OK.

  6. A new empty toolbar appears, approximately in the center of the screen. The name is not yet visible, due to the toolbar dimension.

  7. In the New Toolbar Object dialog box which is still open, select in the Object type: area the type of object to use to control the property and then click OK to add it to the toolbar.

  8. The object is added to the new toolbar, and the toolbar name is now visible in the title bar. It is now easier for you to move it around, dock it in place and eventually add new buttons to it.
