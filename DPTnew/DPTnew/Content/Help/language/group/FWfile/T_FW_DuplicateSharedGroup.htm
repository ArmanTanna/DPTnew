The InsertGroup Copy command enables you to insert further copies of a shared/non-shared group (either local or X-Reference (groups) already existing in the drawing.

To start the command right-click on an instance of the shared/non-shared group in the drawing or on its icon in the Drawing Structure and select InsertGroup Copy in the context menu. A copy of the selected shared group is displayed: move it to the position where you want to place it.

You will have to specify the position of the group Origin in the drawing. You can also define the scale and the rotation to be used when adding the instance to the drawing. Both the rotation and the scale can be defined By value (typing the corresponding numeric values into the Scale and Angle mini-dialog boxes respectively) or Through 2 points (entering two points: Start point and End point). In the later case, once you have entered the first point, the angle/scale changes interactively as the cursor moves: once you get the desired angle/scale, click to enter the second point.

To insert a group into your drawing, you can also open the Group Library and use the Insert command.
