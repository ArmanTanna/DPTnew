ModifyGroupUnload


 
The Unload Shared Group command enables you to discard the description of X-Reference groups. Selected X-Reference groups are removed from the memory.

Unloading X-Reference groups from the memory can be useful when working on very complex drawings. When the command is applied, the X-Reference group is not displayed any longer in the drawing, but its icon still appears in the Drawing Structure, updated by a red symbol representing the delete operation.

When you start the command, you are prompted to select a shared group with external description (to change the selected groups, right-click on Group, select Reset in the context menu and select other group instances).

The Unload Shared Group command can also be started by right-clicking on a shared group in the drawing or on its icon in the Drawing Structure and selecting Unload in the context menu.

Unloaded X-Reference groups can be reloaded using the Reload Shared Group command.
