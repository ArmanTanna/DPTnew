Creating an X-Reference (groups)

After starting the Make X-Reference command, proceed as follows.

The Make X-Reference command can also be started by right-clicking on a shared group in the drawing or on its icon in the Drawing Structure and selecting Make X-Reference in the context menu.

1 Select the shared group you want to transform into an external reference. To change the selected group, right-click on Group, select Reset in the context menu and select another shared group.

2 Click or . the program displays a standard Save File dialog box. The default file name proposed by the program is the same as the group name. To avoid confusion, it is highly recommended that the file name and the group name be the same.
3 Select Save to save the file.

Note that once you save the group as an X-Reference (groups), the group is no longer a physical part of the drawing. Instead, the drawing contains a reference to the file just created.
