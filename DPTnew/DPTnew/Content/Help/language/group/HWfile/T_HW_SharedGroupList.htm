Shared Group List Tab

When working on a drawing, the document Shared Group list tab is an additional tab of the frame on the left of the graphics area in which the program displays the list of shared groups, both locally defined ( ) and X-References (groups)( ).

The list displays only one icon with the corresponding group name — the archetype — for each of the currently existing shared groups and X-Reference groups. Unloaded X-References (groups) are also displayed ( ).

To display the Shared Group list tab, instead of the Drawing Structure, click the Shared Group icon, as in the following illustration. The document Shared Group list tab will be displayed.
 

  • The Name column contains the name of each local shared group and X-Reference group with at least one occurrence in the drawing, or unloaded.
  • In the case of X-Reference groups, the file path is also displayed in the Path column.
  • The Copies column displays the number of copies of the group actually existing in the drawing (you can delete the archetype — of one local shared group or of all the local shared groups with 0 copies — as described in Commands to remove archetypes of local shared groups with no copies in the drawing)
  • You can click on the column header button to order the list by Name or by Path respectively. When you order the list by Path, local shared groups — which have no path — are displayed on top of the list.
  • The list can be restricted only to those groups whose names match the string you type into the Filter box. You can use the following wildcards:
    • "?" — in the place of one character, whatever it is
    • "*" — in the place of any number of characters, whatever they are
  • Depending on where you right-click within the document Shared Group list tab, different context menus are available.
    • Context menu commands available when right-clicking in an empty area
    • Context menu commands available when right-clicking on a group icon.

 

Context menu commands available when right-clicking in an empty area

When you right-click in an empty area of the document Shared Group list frame, the program displays the following context menu.
 

  • Refresh
    Enables you to update the list after you have added/removed one or more shared groups from the drawing
  • Only local groups
    When selected, only locally defined shared groups are displayed. When not selected, X-Reference groups are displayed as well
  • Delete Unused Groups
    Enables you to delete the archetypes of all local groups with no copies in the drawing (available only if at least one group whose number of copies is 0 exists).

 

Context menu commands available when right-clicking on a group icon

When you right-click on a group icon, the program displays the following context menus.
 
Local Shared Group X-Reference Group
  • Insert Group
    Enables you to insert another copy of a shared group (either local or X-Reference (groups)) already existing in the drawing. See the description of InsertGroup Copy command in the Drawing Structure. See also Inserting a copy of a shared group for further details.
  • Start Local Edit
    Enables you to edit the group not within the drawing, but within its own definition (thus with its original orientation and scale). When the Start Local Edit command is started on a shared group, the background color changes. To quit the command, just double-click on the graphics area, or right-click and select End Local Edit.

    Starting an editing session on another shared group
    After starting an editing session on a shared group, you can automatically end the editing session on the currently selected group and start a new editing session on another shared group group in three alternative ways:
    • By right-clicking in an empty zone of the graphics area and selecting Modify Next or Modify Previous, which will respectively move the editing session onto the next or the previous shared group according to the order of the archetype icons displayed in the Shared Group List Tab.
    • By using the arrow keys of the keyboard ( , ) in the Shared Group List Tab to scroll the list and move the editing session to the next/previous shared group according to the order of the archetype icons displayed in the displayed list.
    • By clicking on the archtype icon of a shared group in the Shared Group List Tab, which will move the editing session onto the group whose icon you clicked on.

    In addition, if the shared group you are editing contains other shared groups, while editing the parent group you can right-click on one of the shared groups it contains and select Start Local Edit. This action will automatically end the local editing session on the parent group and start a new one on the child group.


    See also Locally editing a shared group for further details. If the selected group is an X-Reference (groups), when you close the drawing you will be asked whether you want to save your changes to the shared group file. If your answer is affirmative, the shared group definition will be modified in its own file: next time you load the drawing you will see the group modified according to your changes. On the contrary, if the answer is not affirmative, next time you load the drawing the group will look as it was before you modified it.
  • Delete
    Enables you to delete the archetype of a local group with no copies in the drawing (available only if the number of copies of the selected group is 0).
  • Unload
    Enables you to remove the selected X-Reference shared group from the memory.
  • Reload
    Enables you to update all copies of the selected X-Reference shared group after it has been modified in its external file.
  • Rename
    Enables you to rename the selected group. As soon as the command is selected, an editing session starts on the group name.



    Once you have modified the group name as you like, hit the RETURN key to confirm. A warning message is displayed if you typed in the name of a group already existing.
