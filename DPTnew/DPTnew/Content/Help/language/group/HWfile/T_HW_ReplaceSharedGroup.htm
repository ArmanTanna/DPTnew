Replacing a Shared Group

After starting the Replace Shared Group command, proceed as follows.

The Replace Shared Group command can also be started by right-clicking on a shared group in the drawing or on its icon in the Drawing Structure and selecting Replace in the context menu.

1 Select the shared group you want to replace.

2 To replace all the instances of the selected shared group, select the All Copies check box in the selection list.

3 Click or . the program displays a standard Open File dialog box. Select the X-Reference group file, and then click Open.
