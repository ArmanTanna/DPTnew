Drawing Structure

When working on a drawing, the Drawing Structure tab in the frame on the left of the graphics area displays the list of icons corresponding to groups and views of the drawing. Group icons in the tree can be the following:

Non-shared Group
Local Shared Group
X-Reference Group
Local copy of an X-Reference Group
Unloaded X-Reference Group

In the Drawing Structure, you can use the context menu to access the primary commands used to edit and control the entities represented by the icons.

You can customize the way the Drawing Structure is displayed using the controls of the Drawing Structure category of the System Options.

 

Commands that can be accessed from the Drawing Structure

In the Drawing Structure, you can use context menus and Drag & Drop functions to quickly access frequently used commands.

When you right-click in an empty area of the Drawing Structure frame, the program displays the following context menu.
 


  • Reload Structure
    enables you to rebuild the Drawing Structure
  • Drawing Structure Options
    enables you to control the way the Drawing Structure is displayed.
  • Info
    Provides access to the information commands.
  • Options/Properties
    Provides access to all the categories that enable you to customize the operating parameters of the think3 applications and/or enable you to control the properties of different types of entities.
  • File Properties
    Enables you to view and enter the general file properties.

When you right-click on a shared group, the context menu provides access to the following commands:
 

Local Shared Group X-Reference Group

  • Move/Copy
     Enables you to perform translations or rotations of selected group.
  • Set Current
    Enables you to set the selected group as the Current Group, to edit it in the drawing context.
  • Unlink
    Enables you to modify the selected instance of an existing shared group (either locally defined or X-Reference (groups)) without affecting the other instances.
  • Unload
    Enables you to remove the selected X-Reference (groups) shared group from the memory.
  • Reload
    Enables you to update all instances of the selected X-Reference shared group after it has been modified in its external file, or to load it when unloaded.
  • Start Local Edit
    Enables you to locally edit the selected shared group.
  • InsertNew Group
    Enables you to create a new group.
  • InsertGroup Copy
    Enables you to insert another copy of a shared group (either local or X-Reference (groups)) already existing in the drawing.
  • Delete
    Enables you to delete the selected group.
  • Rename
    Enables you to rename the selected group. As soon as the command is selected, an editing session starts on the group name in the Drawing Structure. Once you have modified the group name as you like, hit the RETURN key to confirm. A warning message is displayed if you typed in the name of a group already existing.
  • Zoom Entities
    Increases or decreases the zoom magnification so that selected entities fit exactly within the current display.
  • Hide Entities
    Enables you to hide the selected group.
  • InfoSingle Entity
    Displays information about the selected group (type: shared or not; name; origin coordinates).
  • Edit Properties
    Provides access to the General and to the Group categories.
  • Select Properties from Entity
    You are prompted to select an entity. Click on the entity whose properties you want to acquire.
  • Set Properties as Current
    The current values of properties will be changed into the ones you just set.
