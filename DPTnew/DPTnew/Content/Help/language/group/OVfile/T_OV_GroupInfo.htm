Specific output data of the Info commands when applied to entities belonging to shared groups

Single Entity Info

When applying the Single Entity Info command to an entity belonging to a shared group:
 

  • a line (see Display information about a selected Line)
  • an arc (see Display information about a selected Arc)
  • an ellipse (see Display information about a selected Ellipse)
  • a NURBS curve (see Display information about a selected NURBS curve)

if the group has been rescaled (for example, if it has been created using the Group Copy command and a scale factor — different from 1 — has been specified) the Scale factor will be displayed along with the other output data.

Info Two Entities

When applying the Info Two Entities command to two entities (lines, arcs, ellipses, NURBS curves) belonging to two different shared groups:
 

  • Specific info will be displayed in the case the two selected entities belong to shared groups with different scale factors
  • The Scale factor will be displayed in the case the two selected entities have the same scale factor, but such factor is different from 1.
