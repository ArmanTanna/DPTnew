Creating the Mesh Entity Associated with a Subdivision solid

Once you have created the controlling mesh from a SUBD solid, then you can select the mesh and output it using the ".obj" format (that works also for generic meshes).
To create the Mesh entity associated with a Subdivision solid proceed as follows.

  1. Start the Extract Subdivision Mesh command.
  2. Select the Subdivision Solid.



    A preview of the new Mesh entity is displayed:


     
  3. Apply the command and the Mesh entity will be created.
