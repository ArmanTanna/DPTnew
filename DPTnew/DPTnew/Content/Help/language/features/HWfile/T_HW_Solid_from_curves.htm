Creating a Solid From Curves

Open the sft_t3logo.e3 file available in the Samples folder of your think3 installation and follow the steps given below.

  1. Start the Solid from Text command.

  2. In the Mode: drop-down list, choose the Select curves option.

  3. For selecting the set of curves, right-click on the graphics area and select Select Chain in the context menu.

  4. Then select the curves to be protruded as shown in the figure.



    If the set of curves selected is not a closed loop, then a skin is created instead of solid.



  6. Click or and create the solid from curves.
