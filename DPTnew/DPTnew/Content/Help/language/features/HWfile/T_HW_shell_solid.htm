Creating a shell from a closed solid

The Shell command enables you to create a shell from a closed solid.

1 Under Mode in the selection list, choose Remove faces.
2 You are prompted to select a face to be removed. Select the faces to be removed from the resulting shelled solid by clicking on the edge of the face. To reselect the faces, right-click on Faces to Be Removed in the selection list, select Reset, and then select the faces again.
3 In the Global Thickness mini-dialog, specify the global shell thickness.
4 By default, the same shell thickness is applied to all the remaining faces. To shell selected faces using a different shell thickness, select Faces with different thickness in the selection list, and then select the face to which a different shell thickness is to be applied.
5 In the Different Thickness mini-dialog, specify the shell thickness to be used for the selected face.
6 To apply a different shell thickness to another face, repeat steps 3 and 4.
7 Click or to confirm your selections and shell the solid. Click to discard your changes.
