Creating a Constant Radius Fillet

The Edge Fillet command enables you to create a constant radius fillet on selected edges of a solid.

Open the Edge_Fillet_Hole.e3 file available in the Samples folder of your think3 installation and follow the steps given below.

  1. Start the Edge Fillet command and select the Constant radius option from the Group 1 drop-down list of the Edge Fillet selection list, if it is not already selected.

  2. You are prompted to select an edge to fillet. Select the edge of the model from the Graphics Area, as shown in the image below.



  3. Enter 20 in the Radius mini-dialog box.

  4. Click on the Group Options to expand it and select the Keep Edges check box to retain the edges of features intersected by the fillet, as shown in the image below.

    Keep Edges selected Keep Edges cleared

  5. To fillet another set of edges using a different fillet radius, select next group from the selection list, and then repeat steps 2 to 4.

  6. Click or to confirm your selections and fillet the selected edges.
