Adding a Thread to a Shaft

To add a thread to a Shaft, proceed as follows.

1 Select the Thread check box under More Options of the Shaft command's selection list.
2 Select the type of thread to be created (Metric Coarse Pitch, Metric Fine Pitch, ANSI UNC, ...) from the Standards: drop-down list.

Based on the chosen type, the Designation drop-down list get displayed in the Graphics Area enlisted with different options.
3 Select the option meeting your requirements from the Designation drop-down list.
4 In the Mode drop down list, select the mode of the thread:
  • Full — creates a thread, starting from the initial edge to the selected face.
  • Depth — creates a thread, starting from the initial edge to a certain Thread depth that you need to type in the corresponding box.
  • Double — creates two threads, starting from the initial edge to the specified First Thread and Second Thread.
5 In the Hand drop-down list, select the orientation of the thread:
  • Right — when viewed axially, it winds in a clockwise and receding direction. This is the default value: a thread is considered to be right-hand unless specifically indicated otherwise.
  • Left — when viewed axially, it winds in a counterclockwise and receding direction.
6 Click or to confirm your selections and create the thread. Click to discard your changes.
