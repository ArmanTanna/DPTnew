Creating a Solid From Existing Text

Open the solid_text.e3 file available in the Samples folder of your think3 installation and follow the steps given below.

  1. Start the Solid from Text command.

  2. In the Mode: drop-down list, choose the Select text option.

  3. For the Text selector, select the text as shown.



  4. The font and font style of the text can be changed in the Character dialog box that is displayed by clicking on the Font option.



  5. Now you need to specify the origin of the text which will define the location of the protrusion of the text to be created. To reselect the origin, click on Origin in the selection list and select the origin again.



  6. Enter the value of depth required for protrusion in the corresponding Depth mini-dialog box.



  8. Click or and create the solid from text.
