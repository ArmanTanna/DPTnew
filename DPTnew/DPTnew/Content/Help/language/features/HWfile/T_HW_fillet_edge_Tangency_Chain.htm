Creating an Edge Fillet using the Tangency Chain option

In this tutorial, you will fillet edges that are tangent to each other using the Tangency Chain option of the Edge Fillet selection list. To do so, open the Edge_Fillet_Simple.e3 file available in the Samples folder of your think3 installation and follow the steps given below:

  1. Start the Edge Fillet command and select Constant radius option from the Group 1 drop-down list.

  2. You are prompted to select an edge to fillet. Select the edge of the model from the Graphics Area, as shown in the image below.



  3. The default preview of the fillet is displayed, as shown in the image below.



  4. Next, click on the Group Options node to expand it. Now select the Tangency Chain check box. The preview of the fillet is displayed, as shown in the image below.



  5. Click or to confirm your selections and fillet the selected edges. The final edge fillet with the Tangency Chain check box selected is shown in image below.
