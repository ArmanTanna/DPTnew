If necessary, click More Options to specify additional options.

  • Select the Axis check box to create the axis of the hole as a reference line.

  • To set the direction of the axis of the hole, select one of the options from Direction in the selection list, and then make the additional necessary selections to define the axis direction. The selections you must make vary, as follows, depending on the direction option you choose:

    • Normal - sets the hole axis normal to the face used to position the hole. No additional selections are needed.
    • 2 points - sets the hole axis parallel to a line between two points. Select the two points that define the axis. To select a new axis, click on 2 points in the selection list, and then select two new points.
    • Line - sets the hole axis parallel to a selected line. Select the line or edge that defines the axis. To select a new axis, click on Line in the selection list, and then select a new line.

  • Select the Skin check box to create the hole on an Open Solid with no side walls throughout the hole's length.

  • To add fillets or chamfers, select Fillet/Chamfer.
