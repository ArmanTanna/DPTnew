If necessary, click More Options to specify additional options.

  • To specify the protrusion angle, select the Angle check box and enter the angle value or drag the handle to the position corresponding to the required angle.

  • To set the direction of the protrusion, select one of the options from Direction drop-down list, and then make the additional necessary selections to define the protrusion direction. The selections you must make vary, as follows, depending on the direction option you choose:

    • Normal - Sets the protrusion normal to the text. No additional selections are needed.
    • 2 points - Sets the protrusion axis parallel to a line between two points. Select the two points that define the protrusion of the solid. To select a new protrusion axis, click on 2 points in the selection list, and then select two new points.
    • Line - Sets the protrusion parallel to a selected line. Select the line or edge that defines the direction. To select a new line, click on Line in the selection list, and then select a line.

  • To set the depth of protrusion, select one of the options from Extension drop-down list, and then enter the necessary value in the mini-dialog box. The inputs vary on the extension parameter you choose.

    • Single depth - Sets the length of protrusion in a single direction.
    • Double depth- Sets the length of protrusion in two opposite direction. Enter the variable in the Depth 1 and Depth 2 mini-dialog boxes.
    • Depth + offset - The length of protrusion in single direction is defined by the value in the Depth mini-dialog box. The protrusion is offset by the distance entered in the Offset mini-dialog box.
