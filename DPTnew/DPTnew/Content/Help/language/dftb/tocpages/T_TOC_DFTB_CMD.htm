Drawing Frames, Title Blocks and Part Lists

Drawing Frame and Title Block Enables you to insert a Drawing Frame and Title Block into your drawing.
Save as Drawing Frame Enables you to save the current drawing as a customized Drawing Frame.
Save as Title Block Enables you to save the current drawing as a customized Title Block.
Part List Enables you to insert the part list into the drawing.
Bend Data Enables you to insert a data chart in the flat pattern view, with all the details of bends present in the sheet metal model.
Edit (Part Properties) Enables you to edit the Part Properties of the current component.
Copy (Part Properties) Enables you to copy the Part Properties of the current component in order to paste them onto another component.
Paste (Part Properties) Enables you to paste the Part Properties previously copied from another component onto the current one.
Item Position Property Enables you to drive the part sequence numbering of the components.
Quantity Property Enables you to define the quantity of the selected component.
Highlight All Parts Highlights the components to which part data have been assigned.
Highlight All Parts without Balloon Highlights the parts to which no balloon has been associated.
