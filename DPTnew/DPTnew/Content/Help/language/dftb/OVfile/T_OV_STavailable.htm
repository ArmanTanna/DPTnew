Available Symbolic Expression Identifiers

Here is the list of the available symbolic texts (for conciseness, only the name argument is displayed in the list):

  • In Models
  • In Drawings
Symbolic texts available in the Model environment
$M{name} if inserted in a text, displays the value of the property called name associated to the overall model.
$E{name} if inserted in a text or in a dimension, displays the value of the property called name which will be subsequently associated to the text or the dimension.
$Q{name} if inserted in a dimension or in a label, displays the value of the property called name associated to the entity - not solid - to which the label (or dimension) refers.
$QS{name} if inserted in a dimension or in a label, displays the value of the property called name associated to the solid containing the entity to which the label (or dimension) refers.
$V{name} if inserted in a text, displays the value of the spreadsheet variable called name. In a drawing, shows the name (or the expression, if the name is not defined) or the value depending on the current value of the Show variables as ... (in dimensions) parameter in the Symbolic Texts area of the Text category in the Document Properties.
$VE{name} if inserted in a text, displays the name (or the expression, if the name is not defined) instead of the value of the spreadsheet variable called name.
$VV{name} if inserted in a text, shows the value of the spreadsheet variable called name.
$VQ{name} if inserted in a dimension or a label, shows the value of the spreadsheet variable called name .
In a drawing, it displays the name (or the expression, if the name is not defined) or the value depending on the current value of the Show variables as ... (in dimensions) parameter in the Symbolic Texts area of the Text category in the Document Properties.

Symbolic texts available in the Drawing environment
$M{name} if inserted in a text, displays the value of the property called name associated to the drawing.
$MM{name} if inserted in a text, the drawing displays the value of the property called name associated to the overall model.
$VM{name} if inserted in a text, the drawing displays the value of the spreadsheet variable called name associated to a model dimension or the name (or the expression, if the name is not defined) depending on the current value of the Show variables as ... (in dimensions) parameter in the Symbolic Texts area of the Text category in the Document Properties.

$VQM{name}

 if inserted in a dimension, displays the value of the spreadsheet variable called name associated to a model dimension or the name (or the expression, if the name is not defined) depending on the current value of the Show variables as ... (in dimensions) parameter in the Symbolic Texts area of the Text category in the Document Properties.
$VEM{name} if inserted in a text, always displays the name (or the expression, if the name is not defined) of the spreadsheet variable called name associated to a model dimension, regardless of the current value of the Show variables as ... (in dimensions) parameter in the Symbolic Texts area of the Text category in the Document Properties.
$VVM{name} if inserted in a text, always displays the value of the spreadsheet variable called name associated to a model dimension, regardless of the current value of the Show variables as ... (in dimensions) parameter in the Symbolic Texts area of the Text category in the Document Properties.
$G{name} if inserted in a text, displays the name of the property called name which is assigned to the parent entity of the dimension or text (for example, in a group, the name property of the group).
QSM{name} if inserted in the text of a dimension of an entity representing a solid or a component in a drawing view, displays the name property of the solid/component.
