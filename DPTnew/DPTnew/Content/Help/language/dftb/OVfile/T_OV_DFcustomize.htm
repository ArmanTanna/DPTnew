Drawing Frames: standard configuration files and customization

A Drawing Frame is the graphical representation of the size of the sheet and, optionally, of the area that will actually be occupied by the drawing.



A frame can be added to the drawing — together with a Title Block — using the Drawing Frame and Title Block command.

A complete set of standard Drawing Frames comes with the think3 application standard installation. They are stored as ".e2" drawing files located in the files\drwFrames subfolder of the application installation folder.
