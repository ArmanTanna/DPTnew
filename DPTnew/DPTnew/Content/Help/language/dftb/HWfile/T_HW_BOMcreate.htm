How to create customized part lists

You are enabled to create multiple custom part lists and recall them whenever you need them. Follow the below steps to create a customized part list.

  1. Start the FormatOrganizePart List command in order to access the part list configuration options.

  2. Set the Part list options to the values that suit your needs.

  3. When ready, save the part list by assigning it a name. To do this, use the Save New Settings to Library command.

  4. You are now able to recall the part list any time by selecting it from the list under the Settings command.
