ModifyItem Position Property

The Item Position Property command enables you to modify the item position numbering of the components.

The Item Position Property command can be used in the following ways:

  • Pre-selection Mode: From the context menu displayed by right-clicking on the group or shared group from the Drawing Structure or a component from the Model Structure. From the context menu displayed by right-clicking on the Graphics Area.
  • Post-selection Mode: From the menu bar (ModifyItem Position Property).

When you start the command, the program displays the Component selector that enables you to select the entity for which you want to modify the position property. On selecting the entity, the Item Position Property dialog box is displayed in which you need to enter the value. Click OK to set the new item position for the selected entity.
