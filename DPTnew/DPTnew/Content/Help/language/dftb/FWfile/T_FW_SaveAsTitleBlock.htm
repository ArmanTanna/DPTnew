File Save as Title Block

The Save as Title Block command enables you to save the current drawing as a Title Block.

When you start the command, a dialog box appears that prompts you for the following information:

  • Name portion:
    Enter the name of the Title Block. The program uses this information to build the actual Title Block filename.
  • Description:
    Enter the Title Block description. This is the text that the program displays in the list of available Title Blocks.

When you click OK, the drawing is saved as User folder\files\drwTBlocks\titleblock_< Name portion:>.e2.
Note that you are enabled to set a different path with the Local/Remote of the Configuration Manager tool. This enables you share a standard configuration among your network clients.
