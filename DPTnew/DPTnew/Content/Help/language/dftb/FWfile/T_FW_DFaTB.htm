InsertDrawing Frame and Title Block

The Drawing Frame and Title Block command enables you to define the drawing size, orientation and scale and insert a Drawing Frame and a Title Block.

In order to insert a Drawing Frame, start with setting Size and Orientation. The combination of these options determines the content of the Drawing Frame drop-down list.

Set the Scale value so that the Drawing Frame contains all the drawing views.

You can see a preview of the above settings if you check the Enable sheet box representation option in the Drawing category of the Document Properties.

Finally, select the Drawing Frame and the Title Block from the relevant drop-down lists.

The Drawing Frame and the Title Block are inserted as a Group entity.
