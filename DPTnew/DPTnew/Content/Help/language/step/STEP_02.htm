Log file of the STEP to think3 conversion

When you open a STEP file, a log file of the conversion is created:

  • in the same directory of the file you opened
  • with the same name of the file you opened and the ".log" extension.

The log file structure

The STEP to think3 conversion log file has a structure made of four main sections:

  • INFORMATION — a section containing information extracted from the STEP file, such as the STEP format (AP203/AP214) of the input file and the system used to create it.
  • Report STEP — a section reporting the results of the STEP code analysis, such as type and number of entities, warnings and errors (if any).
  • Report GDX — a section reporting the results of the first step of the conversion to the intermediate auxiliary "GDX" format, which is an internal format, not directly accessible by the user. Also this section shows warnings and errors (if any).
  • Report EKGK — a section reporting the results of the last step of the conversion process: the one to the think3 format. Types and numbers of converted entities are listed, while two proper sub-sections show warnings and errors (if any).

Sub-sections about warnings and errors help you identify any kind of problem you might have had while converting a STEP file.
