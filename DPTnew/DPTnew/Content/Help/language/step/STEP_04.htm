Log file of the think3 to STEP conversion

When you save a STEP file, a log file of the conversion is created:

  • in the same directory of the file you saved
  • with the same name of the file you saved and the ".log" extension

The log file structure

The think3 to STEP conversion log file has a structure made of two main sections:

  • Report EKGK — a section reporting the results of the first step of the conversion process: the one from think3 format. Types and numbers of converted entities are listed, while two proper sub-sections show warnings and errors (if any).
  • Report GDX — a section reporting the results of the conversion to the intermediate auxiliary "GDX" format, which is an internal format, not directly accessible by the user. Also this section shows warnings and errors (if any).

Sub-sections about warnings and errors help you identify any kind of problem you might have had while converting a think3 file.
