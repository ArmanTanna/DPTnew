The Two Viewports command changes the display so that the program shows the modeldrawing in two viewports, with the display oriented so that the model is viewed from the top (looking down along the Z-axis perpendicular to the XY-plane of the World Reference System) in the left-hand viewport and from a left front isometric position in the right-hand viewport.
