When used during preselection, the Select Entity command enables you to clear all entities from the selection set.

If used after selecting a command, it interrupts the current command, which means:

  • Any currently active actions will be stopped
  • Any currently selected entities will be deselected
  • Any currently open dialog boxes will be closed.
ESC key
Please note that the ESC key can be used instead of the Select Entity command. Its behavior is exactly the same.
