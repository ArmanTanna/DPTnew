Opening a Pro/ENGINEER file

To convert models from the Pro/ENGINEER format (up to version Wildfire 5) to think3, do the following:

  1. Start the Open or Insert from File command.
  2. In the file selection box, under Files of type, select ProEngineer file (*.prt*;*.asm*;*.neu*).
  3. Select the file you want to open.
  4. Select the Options button to open the Pro/ENGINEER Options dialog box and set the parameters controlling the conversion process as you like.
  5. Select the Open button to open the file (no preview is available for Pro/ENGINEER files: the file icon will not be displayed in the preview frame).

When you open a Pro/ENGINEER file, a log file of the conversion is created in the same directory of the file you opened. You are also enabled to store the log file in a specific path by setting the relevant parameters in the Log File category of the Pro/ENGINEER Conversion Options.

At the time of importing a ProE model, you can perform a quality check on the model to know of any prevailing issues by selecting the Check Model Quality after loading check box in the System Options - General category.
