Saving a Pro/ENGINEER file

To convert from the think3 format to Pro/ENGINEER, do the following:

  1. Select Save or Save As
  2. In the Save file dialog box, under Save as type, select Pro/ENGINEER file (*.neu*)
  3. In the File name box, type the name of the file you want to save
  4. Click Options to open the Pro/ENGINEER Conversion Options dialog box and set the parameters controlling the conversion process as you like
  5. Click the Save button to save the file.

When you save a Pro/ENGINEER file, a log file of the conversion is created in the same directory of the file you saved. You are also enabled to store the log file in a specific path by setting the relevant parameters in the Log File category of the Pro/ENGINEER Conversion Options.
