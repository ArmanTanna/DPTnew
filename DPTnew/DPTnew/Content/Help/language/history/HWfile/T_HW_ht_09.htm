Rebuilding Step by Step

The following illustration explains the procedure of using the Rebuild Step by Step command to update the model one feature at a time. This method enables you to visualize and control each step of the model rebuilding process.

1 Open the pillar_any_mating.e3 file from the Samples folder of your installation. Expand the Model Structure to see all the features in it.
2 To enable the Rebuild Step by Step button ( ) in the selection list, ensure that the Show Rebuild Step by Step in selection list box is checked in the Construction category of the System Options.
3 Right-click on the last feature (Linear Sweep) in the model structure and select the Redefine Feature option from the context menu.
4 In the graphics area, change the Depth 1 value from 70 to 20.
5 Select the Rebuild Step by Step button from the selection list; the Rebuild Step by Step dialog box will be displayed. Select the Rebuild Next Event button ( ) from the dialog box to start the Rebuild Step by Step process.
6 The feature that is at the next level gets rebuild. Select the Rebuild Next Event button again to rebuild the next feature.

You can repeat this process till all the features have been rebuild. The feature that has been rebuild is displayed in the graphics area. A red arrow ( ) is displayed in the model structure besides the name of the feature that will be rebuild next.
7 To rebuild the model till a specific feature, for example the Linear Protrusion feature, select it in the Rebuild Step by Step dialog box and then choose the Rebuild till Selection button ( ). The model gets rebuild till the selected feature in a single step.
8 You can choose the Rebuild All button ( ) at any stage of the Rebuild Step by Step process to rebuild all the features of the model in a single step. The fully rebuild model will be displayed as shown below.



Finally choose the Close button to close the Rebuild Step by Step dialog box.
9 If at any stage of the Rebuild Step by Step process, an error occurs in rebuilding a specific feature, it can be redefined or removed from the model by choosing the Redefine or Remove buttons respectively from the Rebuild Step by Step dialog box.
