Showing Driving Dimensions

On starting the Show Driving Dimensions command, a dialog box is displayed.


If you want the dimensions to be visible for the new entities that you will create, click Yes.

Note
Note that you can also hide or show drive dimensions for a particular feature by selecting the feature in the Model Structure and then selecting Show Driving Dimensions or Hide Driving Dimensions from the context menu.
