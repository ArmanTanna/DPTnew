Interaction with other commands from Insert Feature Mode

When you start the Insert Feature Mode command, you activate an insert feature session that may require object updating and intermediate model rebuilding  —  and certainly requires a final rebuild  —  on the solid involved in the action, any time you drag or remove the line of demarcation, displayed by the command.

From this session, you can carry out other actions on the solid you are working at, such as adding a profile on a face, or on different objects, such as adding a new feature to another solid, but some actions/think3 commands are completely or partially inactivated.

Here you find a list of the ones that cannot be carried out, with some notes.

Actions - think3 commands Notes
Save, Save As, Insert from File, Partial Save not allowed
Partial Save, Partial Save, Partial Save not allowed
Geometric Transformations (Move/Copy Entities, Mirror, Scale Entities, Multistretch) not allowed
Roll Back, Roll Back Here not allowed
Deactivate Feature, Delete Feature, Redefine Feature not allowed on the features which are deactivated by the line of demarcation
Edit 2D Profile not allowed on the profiles which are deactivated by the line of demarcation
New Component, Set Current, Break Component, Top Level not allowed; you need to end the command when you apply it to a component
Advanced GSM ( Global Shape Modeling) commands in Associative Mode not allowed
Break Solid with the Global option not allowed
