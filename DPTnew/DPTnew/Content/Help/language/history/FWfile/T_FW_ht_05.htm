ModifySolidDelete


 

The Delete Feature command enables you to remove a feature, provided that there are no other entities positioned relative to the feature or referenced to it.

When the feature you wish to remove has some related entities, the command enables you to manage the deletion of "children", that is, among the related events, the ones that exist only if the selected feature exists.

Any time you encounter a history event where some children are involved, the command displays a dialog box, with the information of the related entity involved, from where you can choose whether to unlink or delete the children of the history event that you are going to delete.

As additional visual feedback, the "child" feature is highlighted on the screen and marked by a red arrow in the Model Structure.

Whenever the feature cannot be deleted directly by the command, but requires some prior actions, the program displays a message with the instructions to follow in order to carry out the deletion of the selected feature.
