New Die Design Functionalities and Improvements

Version 2011.1 introduces the following enhancements to Die Design commands.

  • Inside solid selection for surfaces to be sectioned and to be modified
  • Enhancement in the Fuzzy Offset command

Inside solid selection for surfaces to be sectioned and to be modified

By using the Selection Filter dialog box, when the Solids check box is selected under Inside, if the Surface check box is selected, it si now possible to apply all modification commands (including GSM) to surfaces belonging to solids.

Enhancement in the Fuzzy Offset command

In the Fuzzy Offset command, it is now possible to select surfaces in addition to solids. The new Entities node enables you to select Surfaces in addition to Solid.
