New Mold Design Functionalities and Improvements

Version 2011.1 introduces the following enhancements to Mold Design commands.

  • New Undercut option in the Draft analysis
  • Enhancement in the Separation Surfaces command
  • Installation on 64 bit PC SP1 New

New Undercut option in the Draft analysis

The new Undercut check box in the Draft tab of the Quality Checks dialog box enables you to Mold Design analyze the visible faces of the mold part from multiple directions. Please, take into account the following remarks, before checking the box.

  • Computation time might be rather long (as in Region Analysis) depending on the model
    • .
  • During real time modifications (Control points, Subdivisions, or GSM) it is recommended to uncheck the box and to check it when the drag/modification is completed.

Enhancement in the Separation Surfaces command

In order to manage overlapping contour parts, a new Generate Overlapping drop-down list has replaced the old Generate Overlapped Surfaces check box.

  • None, that is no surface generation is the default choice as in previous versions.
  • Insert surface works as the old Generate Overlapped Surfaces check box when it was selected.
  • Insert filling surface is a completely new option enabling you to insert a filling surface inside the indicated borders.

Installation on a 64 bit PC SP1 New

Starting from v. 2011.1 SP1, the MoldDesign component is available for the installation on a 64 bit PC.
The 32 bit edition of MS Office 2010 Suite is incompatible with MoldDesign. If you need both the products on the same PC, the installation of 64 bit edition of MS Office 2010 Suite is mandatory.

If the incompatibility is found during the installation of MoldDesign, a warning message describes the problem and you can go on only after the deselection of the MoldDesign component.
