The Modify Frame Part command enables you to edit a single frame part. The command can be started by right-clicking on the frame part in the model structure or in the Graphics Area and select the Modify Frame Part from the context menu. You are enabled to change the cross section parameters, alignment, and individual node modification. Once a frame part is modified then it is highlighted in the model structure as edited.
