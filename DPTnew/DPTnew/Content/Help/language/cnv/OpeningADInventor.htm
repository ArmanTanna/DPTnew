Opening an Autodesk Inventor file

Do the following to convert models from the Autodesk Inventor format:

  1. Start the Open or Insert from File command.
  2. In the file selection box, under Files of type, select Inventor file (*.ipt*;*.iam*).
  3. Select the file you want to open.
  4. Select the Options button to open the Autodesk Inventor Conversion Options dialog box and set the parameters controlling the conversion process as you like.
  5. Select the Open button to open the file (no preview is available for Autodesk Inventor files: the file icon will not be displayed in the preview frame).

When you open a Autodesk Inventor file, a log file of the conversion is created in the same directory of the file you opened. You are also enabled to store the log file in a specific path by setting the relevant parameters in the Log File category of the Autodesk Inventor Conversion Options.

Note
For the model with the ipt extension the conversion result is the tessellation representation only.
For the model with the iam extension the conversion result is the tessellation representation of the components and the hierarchy of components.
