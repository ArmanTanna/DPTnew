Settings library

If you need to perform conversions with different sets of parameter values depending on the situation, it is not necessary for you to set the different value for each parameter every time you have to convert a different document. The possibility of creating and using proper settings is supplied in the application by means of the settings library. This document describes:

  • How to create a customized setting
  • How to use a customized setting
The settings library is available for only some of the file formats. If either the Options or Settings buttons are disabled, the function for the specific file format is not supported.

How to create a customized setting

When saving or opening a document in one of the supported file formats and after defining once for all a certain set of parameter values in the Options dialog box, in order to save it do the following:

  1. Click the Settings button in the Options dialog box.
  2. Select the Save New Settings to Library option from the pull-down list.
  3. In the Name: text box enter the name for the configuration you want to save into the library.
  4. Click OK to store the new settings. If you typed the name of an existing setting, you will be prompted to overwrite it.

How to use a customized setting

You can use your customized configurations stored in the settings library, when performing conversions. To convert a document using a specific configuration file, do the following.

  1. After selecting the Save As, Open or Insert from File command and selecting the file format, click the Options button.
  2. Click the Settings button in the Options dialog box.
  3. From the list of names of settings stored in the settings library, select the one you want to use in order to load it.

The selected configuration file will be set as the current one and will be used to perform the conversion of the document being opened or saved.
