Autodesk Inventor Conversion Options

When loading (see Open) a Autodesk Inventor file by selecting Inventor file (*.ipt*;*.iam*) in the Files of type drop-down list, the Options button in the file selection box enables you to define the parameters to be used in the conversion process.

A dialog box displaying the following categories shows up:

  • Load
    This category contains the most common parameters to be set when converting from Autodesk Inventor.
    • Log File
      This category contains parameters controlling the name and location of the log file created during the conversion process.
