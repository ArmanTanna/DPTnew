SolidWorks Conversion Options

When loading (see Open) a SolidWorks file by selecting Solidworks file (*.sld*;*.SLD*) in the Files of type drop-down list, the Options button in the file selection box enables you to define the parameters to be used in the conversion process.

A dialog box displaying the following categories shows up:

  • Load
    This category contains the most common parameters to be set when converting from SolidWorks.
    • Log File
      This category contains parameters controlling the name and location of the log file created during the conversion process.
    • Advanced
      This category contains parameters controlling advanced conversion settings such as the conversion of entities into NURBS.
    • Healing
      The controls in this area determine how to deal with some conversion issues.
