InsertDatumLine

The Datum Line command adds a datum line (a reference axis relative to the typology of an object) to the model, thus creating a geometrical relationship between the object and the line. This relationship is maintained even if you subsequently change the object.

You can create a datum line by selecting one of the options in the Type drop-down list:

  • Edge – from the edge of an existing solid
  • 2 Points – from the line between two selected points
  • Axis – from the axis of a revolved solid or feature.
