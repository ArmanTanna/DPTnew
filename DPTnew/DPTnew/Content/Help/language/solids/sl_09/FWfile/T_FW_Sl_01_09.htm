InsertDatumPoint

The Datum Point command adds datum points to the model with reference to an existing object, thus creating a geometrical relationship between the object and the point. This relationship is maintained even if you subsequently change the object.

You can position a datum point in one of the following ways, to be selected in the Point on drop-down list:

  • Vertex – on a point of the object
  • Edge – on an edge or on its extension
  • Datum Axis – in reference to an existing datum line in the model
  • Face – in reference to one of the faces of the object
