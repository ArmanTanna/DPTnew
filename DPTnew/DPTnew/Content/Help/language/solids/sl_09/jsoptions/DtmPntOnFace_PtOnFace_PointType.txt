DtmPntOnFace_PtOnFace_PointType="Enables you to select the method by which to locate the datum point on the selected face: Point -- the default point is the one used to select the face: you can change it by moving the handle; From 2 Lines -- to define the point by entering its distances from two different lines; From-along edge -- to define the point by entering its distance from the start point of an edge and the distance along the edge; Polar -- to define the point in a polar reference system, entering the polar Radius and the polar Angle. The origin point and the reference direction line have to be selected; Parameters -- to define the point position by entering the U, V face parameters value; Intersection -- to define the point as the intersection between a given axis and the face"
