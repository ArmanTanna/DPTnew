The Make Solid command creates solids from selected surfaces and/or solids.

  • The command is designed to create a solid by "sewing" together adjacent surfaces (and/or solid faces). Therefore, to effectively use it, it is recommendable to avoid situations such as overlapping surfaces and/or intersecting solids (in the latter case, use Solid Union instead).
  • The space bounded by the surfaces does not need to be closed. An open solid will be created in this case.

The Color/Material drop-down list enables you to define the color and material of the resulting solid:

  • Current : All the faces of the solid will be of the current color and current material defined in the statusbar.
  • Keep
    • : The faces of the solid will retain the individual colors and materials of the surface(s)/face(s).
Associativity
To retain links to the component surfaces, so that if you modify a component surface the solid will be modified accordingly, select the Associative Mode check box in the command selection list.

You also have the possibility to do face simplification by selecting the Simplify Faces check box in the Make Solid selection list.

  • Select this check box to simplify planar faces (that is create one single planar surface from each group of different planar surfaces having coplanar and in positional continuity.
  • If this check box is cleared, no face simplification will take place.
Note
During redefinition of the Make Solid feature, the Simplify Faces check box will be displayed but not editable. Also, it will be in the enabled state if and only if the Make Solid feature was previously applied in the associative mode along with face simplification.

Managing Non-Manifold solids

If the resulting solid is non-manifold, you are required to confirm its creation. Non-manifold edges and faces are automatically highlighted. In order to try to obtain a closed solid, you can also choose to edit the tolerance value, supplying a higher value.

On selecting Apply () or OK () a dialog box is displayed informing you about the status of the solid. On selecting the Edit Tolerance button:

  • A specific Check Degenerate Edges check box enabling the degenerate edge detection is available in the selection list.
  • A specific Automatic Mode check box enabling automatic computing iteration is also available. If you select this check box and then do Apply () or OK (), calculation is automatically repeated several times until the required tolerance value (to be specified in the Tolerance mini-dialog box) is achieved.

To break a solid back to its component surfaces, you can use the Break Solid command.
