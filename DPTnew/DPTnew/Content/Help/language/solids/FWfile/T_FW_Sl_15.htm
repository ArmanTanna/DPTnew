EditDimension

The Dimension command changes the value of a parametric dimension by assigning a new numerical value or a expression or a variable. The model is updated accordingly.

When this command is started, an Edit Driving Dimension dialog box opens. Using this dialog box the following operations are possible.

  • A numerical expression can be entered in the Expression text box.
  • A numerical value can be entered in the Value text box.
  • If the variable is not defined, it will be assigned the current value of the drive dimension.
  • The model can be rebuilt using Rebuild command.
