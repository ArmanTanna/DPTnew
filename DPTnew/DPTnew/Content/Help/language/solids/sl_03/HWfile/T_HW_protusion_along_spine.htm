Creating a Generic Protrusion by Sweeping a Boundary Profile along a Spine

The Generic Protrusion command enables you to create a generic protrusion on an existing solid by sweeping a boundary profile along a spine path.

Profile and drive curve Generic Protrusion

To obtain the above result, you need to open the Generic_Protursion.e3 file available in the samples folder of your installation folder and follow the steps given below.

  1. Start the Generic Protrusion command to display the selection list and you will be prompted to select the spine (drive curve) for path transition.

  2. Select the drive curve for the Spine selector, as shown in the image below. In this case the spine is consisting of three entities.



  3. Now, click on the Boundaries selector in the selection list and select the profile you want to sweep from the Graphic Area, as shown the image below.



  4. Next click on the Solid selector from the selection list and select the solid from the Graphic Area.

  5. Next, select the Along one plane option from the Motion Mode drop-down list, if it is not already selected.

  6. To create a shelled solid with a specified thickness, select the More Options to display the 1st Thickness mini-dialog check box. Select the check box to extend the edit box and enter 5 in the edit box.

  7. To create a solid sweep feature you need clear the 1st Thickness check box.

  8. Click or to confirm your selections and create the solid/shelled feature, as shown in the image below.

    1st Thickness check box cleared 1st Thickness check box selected

Click to discard your changes.

See "Using Spines" for details on how to use spine profiles.
