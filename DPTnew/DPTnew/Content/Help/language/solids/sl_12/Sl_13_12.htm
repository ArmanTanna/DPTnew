Check Normals

The Check Normals command enables you to check the normals to the faces of a solid. Note that using this command you cannot invert the normals. To invert the normals, use the Invert Normals command.

1  On starting the command, you are prompted you to select a solid. Click to select the solid.
2 The following dialog box is displayed.


  • Check Show arrows to show the normal vectors on each face.


     
  • Check Darken back-surfaces to show the back surfaces with a dark color, uncheck it to avoid darkening.
    Darken back-surfaces selected Darken back-surfaces not selected
Click OK to apply or Cancel to quit.
