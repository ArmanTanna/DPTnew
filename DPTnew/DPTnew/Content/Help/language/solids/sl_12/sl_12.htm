Tools Modeling


 

This menu provides access to a set of information and editing utilities for the management of solids. You can choose any of the following utilities.

UTILITY DESCRIPTION
Rebuild Object reprocesses a selected object so that any changes made since it was last updated will be displayed.
Rebuild All reprocesses the entire model.
Delete Unused Events deletes unused events resulting from one or more undo operations on solids.
Check Manifold enables you to determine whether a selected object encloses a single full space, or has appended surfaces.
Check Normals enables you to check the normals to the faces of a solid.
Invert Normals enables you to check, and if necessary, invert the normals to the faces of a solid.
