Creating a linear slot through all

The Linear Slot command creates a linear slot through an existing solid by sweeping a profile or chain of curves along a linear path so that it extends through all portions of the solid it encounters.

  1. On starting the command, you are prompted to select a profile. Select the profile or the chain of curves you want to sweep. To reselect the profile, click on Profile in the selection list and select the profile again.

  2. In the Extension drop-down list, select the Thru all option.

  3. The direction in which the profile is swept is selected automatically. To change the direction, select a different face by clicking on Face in the selection list, and selecting a different face.


  5. If the slot intersects components, you can extend the slot through those components by selecting the X-Components check box and then selecting the components through which you want to create the slot.

  6. Click or to confirm your selections and create the slot. Click to discard your changes.
