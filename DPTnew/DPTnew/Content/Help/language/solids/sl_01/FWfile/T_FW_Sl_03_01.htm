InsertSolidSweepLinear Slot

The Linear Slot command creates a slot in a solid by sweeping a chain of curves or a profile along a linear path. In the case of a chain of curves, the program automatically combines the separate curves — which must be coplanar — into a single profile.

When an assembly document is open, you are able to create a linear slot on a part of an assembly as per Assembly Feature concept. To do so, the Type drop-down list gets displayed in the Linear Slot selection list. The following options are available in the Type drop-down list.

  • Assembly Feature: Enables you to add or modify slot on a single component of an assembly.
  • Simple Feature: Enables you to add or modify slot on a solid of an assembly.
  • X-Components: Enables you to add or modify slot on a X-Component of an assembly.

A warning message gets displayed in case a linear slot feature is intersecting the solid.
