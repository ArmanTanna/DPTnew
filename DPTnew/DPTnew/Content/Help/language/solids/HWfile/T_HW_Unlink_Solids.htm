Unlinking Solids

To accomplish this task load file "unlink.e3".

  1. Start the Unlink command.

  2. For Entity, select as shown in the following illustration.

    a. b. c.

  3. If you select a., you have only one option for Break links With that is Related Parents.

    4. If you select b., you have the following options for Break links With:

    • All
    • Related parents
    • Related children

    If you select c., you have only one option for Break links With that is Related Children.

  4. For a., select Related parents and click or to confirm and unlink the solid. Try moving the unlinked solid. You can see that only the unlinked solid (unlinked from related parents) moves as shown in the following illustration.


    For b., select
    1. All and click or to confirm and unlink the solid. Try moving the unlinked solids. You can see all the solids move freely as shown in the following illustration.
    2. Related parents and click or to confirm and unlink the solid. Try moving the unlinked solid. You can see that only the unlinked solid (from related parents) moves as shown in the following illustration.
    3. Related children and click or to confirm and unlink the solid. Try moving the unlinked solid. You can see that only the unlinked solid (from related children) moves as shown in the following illustration.

    For c., select Related children and click or to confirm and unlink the solid. Try moving the unlinked solid. You can see that only the unlinked solid (unlinked from related parents) moves as shown in the following illustration.
