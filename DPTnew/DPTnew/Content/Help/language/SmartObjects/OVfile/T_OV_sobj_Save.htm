Saving a Smart Object

If you need to re-use a specific Smart Object in another model or in another work session, you'll have to save it into a file (with a ".sf" extension). You can save Smart Objects in one of the two following ways:

  • Using the Save Smart Object command (clicking the corresponding button in the toolbar or selecting it in the Model Structure shortcut menu once you have selected its icon). In this case the name of the ".sf" file to which the Smart Object is to be saved will be explicitly requested.

  • Dropping it into the Library palette. In this case, the ".sf" file to which the Smart Object is to be saved will automatically be given the same name as the one of the Smart Object.

The bitmap displayed in the Palette is related to the geometry that is currently visible in the model. A specific bitmap can be assigned to the Smart Object in any moment using the Assign Bitmaps command in the Palette shortcut menu.
