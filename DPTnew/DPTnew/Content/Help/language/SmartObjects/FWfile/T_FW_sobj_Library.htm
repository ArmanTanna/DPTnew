ViewSmart Object Library


 

The Library command activates the Smart Object palette, through which you can:
 

  • Position into your model a number of predefined Smart Objects, belonging to a standard Smart Object library shipped together with the think3 application
  • Save Smart Objects to the library
  • View the Smart Object Description
  • Associate any kind of files to Smart Objects
  • Create and manage the bitmaps of the Smart Objects to be used as a preview or as the library icon
  • Localize the messages associated with parameters and geometric references
  • Associate DLLs to Smart Objects, in order to customize the insertion method. Insertion DLLs can be removed from the objects or extracted (to be modified) and associated again.

Some other commands enabling you to completely manage the Smart Object library are also available.
