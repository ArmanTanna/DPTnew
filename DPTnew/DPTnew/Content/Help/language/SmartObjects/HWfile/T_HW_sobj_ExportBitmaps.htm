Exporting Bitmaps

You can use the Export command to save both the Preview or the Thumbnail icons to external image files (".gif"). You might need to use these files in a number of different situations.

For example, you might want to modify the default bitmaps associated to objects, leaving the objects unchanged. In this case, once you have saved the bitmaps to a file, you can modify them using an image editor program and them associate them again to the Smart Object using the Assign command.

When the Export command is started:

1 Select Preview or Thumbnail depending on whether you want to save the palette bitmap (the icon displayed in the library, 64x64 pixels) or the preview one (to be displayed using Show preview, 128x128 pixels).

2 A file selection box is displayed. In the Save in selection box, select the folder where you want to save the image file.
3 In the File name box, the default image file name will be displayed:
  • "SmartObjectName_preview.gif", if you selected Preview
  • "SmartObjectName_thumbnail.gif", if you selected Thumbnail
You can change the default names just by typing your own in.
4 Click the Save button to save the image file.
