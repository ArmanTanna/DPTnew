Saving Smart Objects

To save a Smart Object to a file, proceed as follows:

1. Select the Smart Object you want to save either by clicking on it in the model or on its icon in the model structure.
   
2. When the Save command is started, a file selection box is displayed. You can save a Smart Object to the local file system.

If you have pre-selected no Smart Object before starting the Save command, you can perform selection after starting it, by clicking on the Smart Object in the model.
