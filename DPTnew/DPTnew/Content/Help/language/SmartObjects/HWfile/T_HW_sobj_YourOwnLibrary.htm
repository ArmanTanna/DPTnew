Creating your own Smart Object Library

To create your own customized library, so that a new tab will displayed in the Library palette, proceed as follows:

1 Start the Save command.
2 In the Save in selection box, choose the "SO_Libraries" subfolder of your think3 installation folder.
3 Select Create New Folder.
A "New folder" icon will be displayed in the box below. Change its name to the one you want. This is the name that will be displayed as the new tab in the Library palette.
 
4 In the File name box, type the name of the Smart Object file you want to save.
   
5 Click the Save button to save the file.

When the Library command is started, a new tab with the folder name defined in step 3 will be displayed, containing the Smart Object you saved to it. The new tab you created will work exactly as the standard ones: you will be able to use it exactly as described in Using the Library palette.
