Inserting Smart Objects

When the Insert command is started a file selection box is displayed.

You can add a Smart Object to your model taking it from the local file system.

Once you have selected the Smart Object to be positioned, the Insert Smart Object selection list is displayed in the graphics area. Proceed as follows.

1. Define which parameters have to be displayed (among the ones defined as exposed parameters), thus being accessible for dimensional changes. In the Show parameters drop-down list, select one of the following items:
None to display no parameters
All to display all the parameters of the Smart objects
Component parameters, depending on the Smart Object to display only parameters of component elements (profile, linear sweep, rotational sweep, ...)
2. If you have selected a Smart Shape or a Smart Profile, select the insertion mode in the Insertion mode drop-down list:
Add to add the Smart Object to the model
Subtract to subtract the Smart Object from the model
New object to add the Smart Object to the model as a new object
3. If any configurations are defined for the Smart Objects to be positioned in the model, select one from the Configuration drop-down list.
4. Select the anchor point and proceed as described in Anchor Handle
5. Confirm by selecting the OK button.
