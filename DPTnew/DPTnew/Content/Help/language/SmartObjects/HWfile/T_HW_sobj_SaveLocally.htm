Saving Smart Objects to the local file system

1 In the Save in selection box, select the folder where you want to save the document (initially, the model directory chosen when installing the think3 application is proposed). You can navigate through your file system like in any Exploring window of the Microsoft® Windows operating system.
   
2 In the File name box, type the name of the file you want to save.
   
3 Click the Save button to save the document.
