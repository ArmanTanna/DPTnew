Localizing Prompts

You may need to have your Smart Objects available to different language users. In this case, you may need to localize Smart Objects prompts to different languages. The think3 application enables you to associate localized prompts with your Smart Object.

To do this, proceed as follows:
 
1 When the Message File command is started, select Extract. A file selection box is displayed. In the Save in selection box, select the folder to which you want to save the message file containing the prompts.
2 Click the Save button to save the message file.
3 Using a text editor, edit the message file you saved in step 2.
4 English prompts are lines like the following:
L.E "English text of the prompt"
To add localized prompts, so that the prompt will be displayed in the same language as the one the think3 application is being run in, just add one line per each language below the "L.E" one:
L.I "Italian text of the prompt"
L.F "French text of the prompt"
L.G "German text of the prompt"
L.J "Japanese text of the prompt"
L.K "Korean text of the prompt"
You can obviously add only the languages you need, and the order is not meaningful.
5 Save the message file.
6 In the Library palette, select the Message fileInsert command.
7 A file selection box is displayed. Select the previous message file and click Save.

The prompts for your Smart object are now localized. You can use it with think3 in each of the languages you added to the message file: the prompts will be displayed in the corresponding language.
