Assigning Bitmaps

When the Assign Bitmaps command is started, a dialog box is diplayed on which you can select the bitmap to associate with the selected Smart Object.

To define an icon to be displayed in the Library palette:
 

  • Under Palette, using the Browse button, select the icon file to associate with the Smart Object as an icon in the Library palette (the required format is ".gif" or ".bmp"; allowed size is 64x64).
  • The chosen icon will be displayed in the Preview: area.
  • Select Show Current to view the current icon again instead of the one you chose.

To define an icon to be displayed as a preview using the Show preview command:
 

  • Under Preview, using the Browse button, select the icon file to associate with the Smart Object as a preview (the required format is ".gif" or ".bmp"; allowed size is 128x128)
  • The chosen icon will be displayed in the Preview: area.
  • Select Show Current to view the current icon again instead of the one you chose.

Select OK or Apply to confirm, Cancel to discard your changes.
