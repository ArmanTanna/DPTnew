Converters

The application enables you to easily exchange data with other packages, by providing the following converters:

  • To and from DXF™, DWG™, and DM formats
  • To and from IGES, STEP, and VDA formats
  • To and from STL format
  • To VRML, DXF™ 3D, Wavefront (Personal Visualizer™, Advanced Visualizer™), IRIS Inventor™ and XGL formats
  • To CATIA® Version 5 format with TD Direct and from CATIA® Version 5 format with TD XchangeReader
  • To Parasolid® format with TD Direct and from Parasolid®; with TD XchangeReader.
  • To Pro/ENGINEER® format with TD Direct and from Pro/ENGINEER® with TD XchangeReader.
  • To PDF format with ThinkDesign Printer.

The TD XchangeReader module is now capable to read:

  • Any SolidWorks© version up to 2016
  • Any Autodesk® Inventor® version up to 2016
  • Any JT version up to 10.0
  • Any Siemens NXTM (Unigraphics) version from V11.0 up to NX 10.0
  • Any CATIA® V5 version from R4 to V5-6R2015
  • Any CATIA® V6 version from 2011 to 2013
  • Any Parasolid® version up to v27.0
  • Any Pro/ENGINEER version up to WildfireTM 5
  • Any CreoTM Parametric version up to 3.0

For the Autodesk® DXF/DWG format, the DWGDirect 3rd party library has been upgraded to Teigha® 4.1.1.

Parameters controlling the conversion processes can be adjusted by selecting the Options button in the file selection dialog box each time you save/load a file.

Help on parameters controlling the conversion process
You can easily display a short description of each parameter controlling the conversion process by issuing the help request as follows:
  1. Click on the button in the top-right corner of the dialog box displayed by selecting the Options button.
  2. Click on the item of the dialog box corresponding to the parameter you need help on.
