InsertDimensionFeature Control Frame


 

The Feature Control Frame command enables you to insert a feature control frame.

When you start the command, the selection list displays the following options:

Reference Entity Enables you to select a reference entity to which you want to attach the feature control frame.
Position Enables you to specify a point/segments of a leader line to position the feature control frame.
Symbol Enables you to choose the geometric tolerance symbol.
Straightness Flatness
Circularity Cylindricity
Profile of a line Profile of a surface
Parallelism Perpendicularity
Angularity Position
Concentricity and coaxiality Symmetry
Circular run-out Total run-out
Diameter Symbol Enables you to choose between the diameter and spherical diameter symbols to be included.
Value Enables you to type in the tolerance value.
Material Condition Enables you to choose the material condition symbol.
Entity Properties Displays the Entity Properties dialog box that enables you to edit the feature control frame properties.

The following options are available under More Options.

Text only When selected, enables you to insert the feature control frame without selecting a reference entity to attach it to. Hence, the Reference Entity option gets hidden automatically on selecting the Text only check box.
Plane Enables you to choose the plane onto which you want the feature control frame to be inserted ( Work Plane, View, World). This option is available only in the 3D environment.

You can attach dimension entities to a measure or witness line of a dimension, as shown in the image below.



The surface texture can be attached only to a measure line for the following type of dimensions:

  • diameter
  • radial
  • linear

    Note that the linear dimension is supported only when it is created by choosing the following dimension types from the Type drop-down list:
    • Snap point
    • Length
    • Point & line

The feature control frame and datum reference can be attached to a:

  • measure line, in case of a:
    • diameter dimension that is passing through the center (point 1, as shown in the image above)
    • diameter dimension as an external label (point 2, as shown in the image above)
    • radial dimension (point 3, as shown in the image above)

  • witness line, in case of a:
    • diameter dimension that is not passing through the centre (point 4, as shown in the image above)
    • linear dimension

To implement the above behavior, the Reference Entity selector in the Surface Textures, Feature Control Frame, and Datum Reference selection lists now enable you to select a dimension (linear, diameter or radial) as a reference. In case of the Datum Reference command, you can select the feature control frame dimension that results in a datum reference attached to the bottom line of the feature control frame symbol.


The surface texture, feature control frame, and datum reference dimensions can be moved only along the measure or witness lines of the dimensions they are attached to. When the dimension is moved, they will also move along with it.
