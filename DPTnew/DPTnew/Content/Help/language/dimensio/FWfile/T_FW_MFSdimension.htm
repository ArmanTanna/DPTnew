The Edit Format/Properties command helps you to change the appearance of the dimensions created. This command can be found in the contextual help, by right-clicking on any of the dimensions created.

Two options are available:

  • Format —  Using this option, you can specify the dimension Prefix, Suffix, Tolerance. You can also limit the precision for the tolerances that you specify using the Precision option, to the desired Decimal digit.
  • Properties —  By clicking on this option, the Entity Properties window pops up, which helps you to modify parameters of both Dimension and Text styles, created earlier.
