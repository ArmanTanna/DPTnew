The Edit Family command enables you to modify all the dimensions that belong to a particular Ordinate Dimension family.

An Ordinate Dimension family is a set of Ordinate dimensions that are created in one session. When you place the cursor on any one of the dimensions, all the dimensions of the family along with the Origin gets highlighted.

You can find this command on the context menu of an Ordinate dimension. As soon as you select this command, the Ordinate Dimension command will be activated, having all the options as set in the previous session. When you select a dimension and make changes, all the other dimensions that belong to the same family get updated with these changes.
