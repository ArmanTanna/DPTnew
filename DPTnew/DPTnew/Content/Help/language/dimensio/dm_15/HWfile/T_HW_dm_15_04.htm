Create angular dimensions with a fixed direction

1  When you start the Smart Dimension command, the program prompts you to select an entity. Click to select a line or edge.



Select the entity anywhere along its length except at a Snap point.
2 Select the Angular option in the Type drop-down list to create an angular dimension indicating the angle between the linear entity and the X- or Y-axis.
3 As soon as you select the Angular option, the Type drop-down list displays the following options:
  • Sector
  • Explementary sector

Select one of the following options from the Orientation drop-down list:
Projected

 To create the angular dimension that subtends the angle from the selected entity to a line projected at the angle specified in the Angle mini-dialog box
Horizontal

 To create the angular dimension that subtends the angle from the selected entity to the X-axis
Vertical

 To create the angular dimension that subtends the angle from the selected entity to the Y-axis

4 By default, the angular dimension subtends the smallest angle between the selected entity and the axis or projected line selected in Step 3. You can change this, however, by selecting one of following options in the Type drop-down list:
Sector
Measures the smallest subtended angle from the selected entity to the axis or projected line, based on the quadrant in which you position the cursor
Explementary sector
Measures the largest subtended angle from the selected entity to the axis or projected line, based on the quadrant in which you position the cursor

Note

You must select the orientation prior to selecting one of these buttons. Once you click one of these buttons, you can no longer change the orientation.

5 You are prompted to enter the origin. Notice that the program displays the dimension, which moves as you move the cursor. To create the dimension, simply specify the location of the dimension line.



As soon as you select the point, the program immediately creates the dimension.


Format
Using the Format option you can create the dimensions in any desired format.

Properties
By clicking on the Properties Option, the Entity Properties window shows up, that enables you to control appearance of the Dimension and Text entities.
