Select a special point

When you start the Smart Dimension command, the program prompts you to select an entity.
As soon as you select an entity, the options change, depending on the type of entity you select and the point at which you select the entity:

  • Line or edge
  • Special point
  • Arc or circle
If you select a special point, the selection list displays the following options:
Orientation Enables you to specify the orientation of the dimension line. Select one from the drop-down list
Type Enables you to specify the type of the dimension to be created, based on the selection you performed. Select one type from the drop-down list
Properties Displays the Properties dialog box
