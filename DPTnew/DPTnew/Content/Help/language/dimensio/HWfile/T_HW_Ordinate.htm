Creating a Linear Ordinate dimension

1  When you start the Ordinate dimension command, first set the Dimension Type as Linear. Then set the Origin by clicking on any of the snap points.

2 The Origin will be the reference point for the dimensions that will be created. The reference point may be changed at any point using the Edit Reference option. Then choose the entity or snap point to create the dimensions.

3 The dimension is displayed as soon as you click on the snap point or the entity. Click to place the dimension in the desired location.

Note
If you select the entity, you will also get the dimension measure from the Origin. But if you select the snap point, you will see the dimension measure of just that snap point.


4
By clicking on the successive snap points or the entities, further dimensions will be created. If the Aligned checkbox is on in the selection list, then the dimensions created will be aligned to the first one. Otherwise you can place them by choosing the location individually.

5 Before creating the dimensions you can set the direction for them using the Direction drop-down list. The three options available are:

Automatic Enables you to measure and create the dimensions along the X-axis or along the Y-axis, depending on the cursor position.
Along X-axis Enables you to measure and create the dimension only along the X-axis
Along Y-axis Enables you to measure and create the dimension only along the Y-axis
6 When you are finished creating the dimensions, end the command.
