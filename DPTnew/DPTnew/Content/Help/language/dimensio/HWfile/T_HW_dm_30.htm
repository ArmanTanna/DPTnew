Move a dimension from a view to another

1  When you start the command, the program prompts you to select the dimension to be moved. Click to select the drive dimension you want to move.



2 The program then prompts you to select a view.
Click to select the view to which you want the dimension to be moved.



Note that the dimension can only be moved to a view in which it is visible; otherwise, the program warns that the dimension is not visible in the selected view.
As soon as you select the view, the dimension is moved to the selected view.
