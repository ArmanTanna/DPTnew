Align a dimension

1  When you start the command, the program prompts you to select the reference entity.
Click to select the dimension to which you want the other dimensions to align.

2 The program then prompts you to select the dimensions to be aligned.
Click to select the dimension you want to align with the reference dimension.

3 As soon as you select the dimension, it is immediately repositioned. The program then prompts you to select another dimension to be aligned.
Click to select another dimension to be aligned with the reference dimension.

4 When you are finished aligning dimensions, end the command.
