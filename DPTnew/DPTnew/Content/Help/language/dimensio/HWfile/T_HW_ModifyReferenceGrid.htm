Modifying a Reference Grid

  • You can modify the Reference Grid using the Modify Reference Grid command from menu bar or using the Modify Entities from the context menu.
  • Double-clicking the grid also enables you to modify the Reference Grid.

1  When you start the Modify Reference Grid command, you are prompted to select a view.
You can select the view by clicking on it either in the drawing or in the Drawing Structure.



(As soon as you select the view, you have the options to redefine the Reference Grid. Using these options, you can reset all the grid properties.
For example:
The above car model has been set with Automatically grid definition mode. You can redefine it to By 2 points mode; change the format of label and display of grid lines.)
2 The program then prompts you to redefine the grid definition mode from the drop-down list of the Set Boundary option.
Select the By 2 points and click Properties to change the grid line properties.
Change the properties and format of the Reference Grid using the Entity Properties dialog box.


 
3 When you are finished modifying the Reference Grid, end the command.
