Compensator FE Commands

Identify Displacement Field Enables you to identify the morphing — that is: the deformation —  defined by a Displacement Field between two sets of points — the "initial points" and the "target points" — (like the ones that can have been derived from Finite Element Analysis files containing meshes, say using the Extract Points command) based on which it finally creates an entity to be used as a reference for the GSM Replicate command, so as to apply the same deformation to any solids/surfaces in the model. The said Displacement Field can result from a spring back computation, as well as be generated by any kind of process.
Show Displacement Field Displays the displacement between couples of points, so as to enable you to analyze input data for the Identify Displacement Field command.
Clean Displacement Field Enables you to remove couples of points (or move some) that may so have been detected as not consistent, that can have been detected through the Show Displacement Field command, in order to avoid the presence of inconsistencies while creating the desired deformation through the Identify Displacement Field command.
Compute Distances Enables you to compute the distances of a specified set of points from a selected shape based on a chosen projection law and also provides info about the maximum, minimum and mean distances.
Process Compensator Enables you to create the compensated shape of any process (for example, like the one required in shape distortions due to spring back effects, such as, for instance, the ones occurring in sheet metal stamping), by using proper "point files", that is FEM files, such as the ones generated using Extract Points.
Blank-Holder Addendum and Part Enables you to modify the die environment based on the result obtained by adaptation/compensation process.
Offset Mesh Enables you to offset a selected mesh by a specified offset value.
Extract Points Enables you to extract points from Finite Element Analysis files containing meshes, so as to facilitate the integration with external processes. Once the points have been extracted using this command and the point sets are saved to proper "point files (*.pt)", it can be used as input files for the Identify Displacement Field command.
Load Mesh Enables you to load mesh entities from Finite Element Analysis files that can be selected in a specific file selection box.
Extract Zones Enables you to clean the displacement fields so as to retain only those areas that satisfy the specified criteria.
Check Symmetry Enables you to detect symmetry conditions in a selected mesh entity.
