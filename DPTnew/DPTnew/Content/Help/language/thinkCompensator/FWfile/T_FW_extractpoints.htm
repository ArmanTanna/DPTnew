The Extract Points command enables you to create ASCII point files ".pt" from Finite Element Analysis files containing meshes.
In the Points drop-down list you can select:

FEM File To select a FEM file using the Browse for FEM File option which displays a specific file selection box, just like in former version.
Mesh nodes To select a mesh in the model (Mesh). In this case, the coordinates of the mesh nodes will be saved to the ".pt" file according to the usual behavior of the Extract Points command.

Once extracted using this command, point sets can be used as input files for the Identify Displacement Field command.

The supported FEM file formats are the following:

  • NastranTM (".nas")
  • Autoform (".af")
  • Forming Technologies (".fpi").

After you have supplied the command with a file in one of the above said formats by choosing it in a specific file selection box automatically activated by the command itself, click or to create an ASCII ".pt" file. A new file selection box will be displayed so as to enable you to define the path and the name of the file to be saved. The default name is the same as the one of the input file, with extension ".pt".
