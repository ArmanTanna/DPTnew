EditAlignmentReplicate

The Replicate Alignment command enables you to modify the position of entities by performing the same controlled transformation as captured by the mesh element (representing the transformation) created by the Pre-Alignment, Optimize Alignment and Optimize Contour Alignment commands.

On starting the command, you have to select (for the Entities selector) the entities on which the transformation is to be replicated.

For the Transformation Reference selector, select the transformation mesh element.

The Copies check box enables you to make copies of the transformation.

  • When this check box is selected, you can make multiple copies of the transformation. Each copy is created by applying the same transformation to the previous instance. The required number of copies can be entered in the corresponding text box.
  • When this check box is cleared, no copies are made.

The Invert Transformation check box enables you the invert the transformation logic.

  • When this check box is selected, the sense of transformation is inverted. Hence, you get a result as if the reference entity has been moved to match the movable entity.
  • When this check box is cleared, no inversion takes place.

Following options are available under More Options:

Linked copies check box
  • If selected, the copies are linked to their original entities. Any time the original entities are modified, the copies will be modified accordingly.
  • If cleared, the copies are not linked to the original. Whatever may happen to the original entities, the copies will not change.
Keep attributes check box
  • If selected, copies of entities retain their attributes (such as color, line type, and layer) of the original.
  • If cleared, copies assume the corresponding current property values.
