The options in this area enable you to define the text (such as the consequence class, the welding class and welding process) that will be displayed in the welding symbol tail.

You can organize the information in up to three lines, each corresponding to one of the available fields. Type in the available fields the text to be displayed, or select the text from the drop-down list. The text stored in the drop-down list is lost when the program exits.

The text in the welding symbol tail is aligned to the left and vertically centered.
