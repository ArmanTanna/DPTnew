An Overview on Weld management

The think3 application features the management of welds with a number of tools that enable you to draw the welds and the welding symbols according to various standards.

 

Welding standards

Many welding standard are supported, each defining the name and the set of welding symbols available. You are enabled to set your preferred standard by choosing it among the list available through the Weldings command.

 

Drawing welds

Specialized commands enable you to draw the section and bead views of the welds.
When you draw the welds, you only need to define the weld parameters once. In fact, whether you draw sectioned or bead representation of the weld first, when you need to draw the other view you are enabled to copy the attributes of the weld from the existing view with the Data from Welding command.
Since most of the times you will use a few of the possible types, rather than scrolling through the complete list of available welding types looking for the one you need, you are enabled to use the Weldings command to define a subset of the types available for the chosen standard. You can return to this command any time to modify the set of preferred types.

 

Welding symbols

The Welding Symbol command enables you to add a welding symbol to the drawing. If you have already drawn the weld, you are enabled to initialize the welding symbol with the data you already applied to the weld with the Data from Welding command.
The symbol offers a comprehensive set of options including, but not limited to, symmetrical welds, peripheral welding, field welding, up to three note fields and other notations.
