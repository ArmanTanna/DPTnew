What's New in Service Pack 2

Version 2008.1 Service Pack 2 introduces the following improvements.

  • Enhancement in the installation procedure
  • The new Taper Dimension command
  • The new Reshape module
  • The new faster drawing views generation mechanism
  • The new MoldDesign module
  • Menu commands reorganization

Enhancement in the installation procedure

The installation procedure has been significantly enhanced and powered by adding the possibility of installing two new modules, Reshape and Mold Design.

To install the said tools, proceed as follows:

  • In the Setup Type installation wizard select Custom and click Next.
  • In the Select Features wizard that will show up:
    • Check the Reshape box to install Reshape. When the box is cleared, Reshape will not be installed.
    • Check the MoldDesign box to install Mold Design. When the box is cleared, Mold Design will not be installed.
  • Click Next to go on with the installation procedure.

The new Taper Dimension command

This version introduces a new InsertDimensionTaper command. Using this command it is possible to dimension the pitch and taper aligned to the standard GB/T 15754-1995. The taper/slope dimension will be associated to the 2 selected edges representing the oblique face and the reference line/plane. For further details, refer to the online help.
In order to insert a taper dimension, a new linear dimension type with a specific display list and a parameter set to define its representation is needed. As a consequence, the following style specializations have been introduced in this version:

  • The new dimension type, Taper dimension, in the Specialize Style dialog box.
  • The new options, Show symbol and Format, in the Taper dimension area of the Entity Properties - Dimension - Symbol and Entity Properties - Dimension - Unit categories.
  • The FormatOrganize Taper designations command has been introduced to predefine the taper dimension designations. A Organize Taper Designations dialog box is displayed when the command is started. See the online help for a complete description of all the options inside the dialog box.

The new Reshape module

Reshape is a fully integrated module of ThinkDesign that provides reverse modeling solutions. It has been explicitly developed for solving the problems of CAD users who need to import and manage files obtained through 3D digitizing operations of physical models. It facilitates fast prototyping, design review, finite element analysis, and inspection. Reshape can be profitably exploited whenever a physical model (which can be wooden, clay or resin made) or even a real object needs to be described as a computerized 3D shape to be processed or modified.


Using Reshape you can import Meshes and STL objects obtained from digitalization machines. On such files, clean-up operations can be performed in case the data is not perfect for further processing. Depending upon the requirement you can use the Reshape tools to modify the Mesh, for example, detect meaningful edges (or character lines), split or cut the mesh, and make standard GSM modifications. All the operations are obtained through robust automatic processes with good quality results. If required, using the modified mesh, Reshape can do surface reconstruction to cover partially or fully the mesh with surfaces.


You are also enabled to intervene manually when required, exploiting all the possibilities offered by ThinkDesign. For instance, you can use the standard surfacing tools of ThinkDesign to generate surfaces by using the Mesh as a template. Also, you can make use of Mesh in the assemblies or generate a drawing from it.


The new faster drawing views generation mechanism

A new option, Hidden line computation, has been introduced in the Entity Properties - Drawing View category to help the user switch between Fast (hardware mode) and Accurate (software mode).


When the hidden line computation is set to Fast (hardware mode), each drawing view will be associated with a precision limit and it can be seen in the Single Entity Info dialog box as shown below:



The hidden line algorithm excludes the entities:

  • smaller than the precision value
  • partially or almost completely hidden (where a very small portion is visible and that could be smaller than the precision limit).

Note that the precision value will depend merely on the size and the graphic discretization of a model, the Graphics card and the kind of drawing view. Also note that the Fast (hardware mode) hidden lines mechanism will be actually effective only if the Graphics card in use is a Professional or High End one, included in the list you can find in the "Graphics cards for high quality rendering" section of the "Graphics cards" document in the online help. In all the other cases, the Accurate (software mode) mode will be automatically used.

In order to make existing options in Options/Properties dialog box consistent with their location under Edit Properties of a drawing view, a few items have been reorganized. Filter entities smaller than and Light representation have been moved from Document Properties - Drawing - General to Entity Properties - Drawing View.

Note

  • The new option is active only when the Representation is set to Hidden lines and the Show dashed hidden lines option is not selected in the Entity Properties - Drawing View - Visibility - Geometry area.
  • The new option works on all views except the flat pattern, exploded, cross-section and developed views. Also, it is not effective with components in light representations.
  • If the option is set to Fast (hardware mode), the resulting drawing file will not be compatible with any of the previous versions.


The new MoldDesign module

MoldDesign is a fully integrated module of ThinkDesign that provides high degree of automation to the mold designing process. The design process of a mold involves project planning & product analysis to develop the Core and Cavity, along with the complete injection mold assembly. The various MoldDesign functionalities are capable of managing all the stages of the designing process, by making use of easy-to-use but powerful and flexible tools to achieve significant reduction in the designing time.


MoldDesign facilitates the automatic extraction of core and cavity along with inserts, creation of 2D tables, form the mold base (with plates and columns) by making use of standard vendor's catalogs, electrode design, insertion of standard parts (like ejector pins, core pins, sprue bush, screws), and creation of cooling systems along predefined 3D paths.


MoldDesign makes use of the same powerful engine that drives ThinkDesign, which supports the well know features like Hybrid Modeling, GSM, and Interactive Solid Modeling. It simply translates to impeccable control over creation & modification of the mold and at the same time being parametric and associative. Hence, any modification of the original part can effectively propagate to the mold assembly structure. By amalgamating MoldDesign with the thinkPLM product, it becomes possible to effectively manage the documentation, revisions and generation of specifications to reduce time and cost.


MoldDesign along with the functionalities of ThinkDesign Suite has the potential to embrace the whole mold design process and efficiently manage specific design modifications at every stage of the design cycle.


Menu commands reorganization

In order to streamline the menu bar organization, the thinkteam menu is displayed only when a thinkPLM license is available.

For this reason, the following commands have been relocated to different menus:


Old Command Path New Command Path Command Name License requirements Notes
thinkteamthinkparts Toolsthinkparts Catalog Browser Any license  
thinkteamthinkparts Toolsthinkparts Catalog Manager thinkparts  
thinkteamthinkparts Toolsthinkparts Create Part Family Any license Available when a 3D Model is active
thinkteam Toolsthinkparts Options thinkPLM Level 0 With other thinkPLM licenses the Old Command Path is retained
thinkteam Tools Assembly Analyzer Any license Available when a 3D Model is active

Also, the thinkPLM Options tab has been removed from the Options/Properties command dialog box. The relevant options are available through the Options command.


The following commands have been removed from the menu bar and have been placed in the context menu of components in the Model Structure and that of groups in the Drawing Structure. The command names, moreover, have been slightly changed to make their function clearer to the user:

  • Edit Part Data
  • Copy Part Data
  • Paste Part Data
  • Define Sequence
  • Edit Relation
