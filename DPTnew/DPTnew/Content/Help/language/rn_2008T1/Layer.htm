Overview

In this version, the application layer management has been completely renewed. The main features of the brand new layer environment are:

  • User interface visibility and availability
  • Better look and representation
  • Possibility to do quick and effective changes
  • Possibility to filter the layers list
  • Possibility to select entities by layer
  • Possibility to filter entity selection
  • Possibility to lock a layer and to assign properties such as Color, Line Type and Width
  • Possibility to protect a layer with password
  • Possibility to move all entities from a layer to another one or to exchange the layer content
  • Possibility to identify the layer by using an alphanumeric value (string)

In former versions each model/drawing contained 1024 layers, no matter whether empty or not. Starting from this version, only layer 0 is there by default.
When an old model/drawing is loaded into the new version, the actual layers automatically defined in the document will be:

  • Default layer 0, which is always there
  • All layers which at least one entity is laying on
  • Layers with an associated description, no matter whether empty or not
  • Current layer.

The new option Show default prefix in new layer names under Layers in the General-Advanced category of the System Options enables you to add the prefix Layer to the name of the new layer.

When importing the content of a document through the InsertFrom File command, all the defined layers will be imported, regardless of the fact there may be entities laying on them or not.

Compatibility
All the customized layers older than version 2008.1 are no more compatible with the current layer management. The renewed layer customization can be applied only to newly created documents.

For complete detail on the compatibility of layers, see in the command description of the online help.

New user interface

In former versions, only limited functionalities (All On, All Off, Show All Layers, Relevant Layers) were permitted on the layers. In this version several new and improved functionalities have been developed, which allow the user to manage the layers in the layer list:

  • Layer Editing to modify the layer list.
  • Layer Filter to display the layers that match the filter criteria.
  • Layer Selection to select the layers.
  • Layer Entity Selection to select the entities on a layer. See "Layers: Entity Selection" in the online help for further details
  • Layer Entity Change to change the entities from one layer to another. See "Layers: Entity Change" in the online help for details.
  • Layer Protection to protect the layer with a password. For further details, see "Layer Protection" in the online documentation.
  • Entity selection filter to select the entities with the common properties on a layer.

To achieve the goals listed above, a brand new user interface, the Layers tab, has been made available on the left pane of the drawing/model document. It consists of several controls enabling you to operate on layers, without starting the layers commands from the menu bar. For further details, see "Layers" in the online documentation. The new interface can be activated using the Layers tab.

Layer Properties

The layer properties are arranged in form of the columns, which can be customized to make them visible by right-clicking and selecting the check box of the required property from the available list. For further details on column customization, complete description of the property columns of an individual layer, and generic information about the layer list, see "Layers" in the online documentation.

New commands

The label of the existing layer command has been standardized to make it more appropriate. As a result, the command Input Layers has been renamed as Current Layer. Also, from now onwards the old Output Layers command has been removed from the Format menu as it is meant to switch to the Layers tab, which is no more useful. As a consequence, the old Layers dialog box has also been removed.


The new layer commands can be reached using the:

  • Buttons: Layer editing is possible using the five buttons — New Layer, Set Current Layer, Delete Layer, Filter Layers, Select Layers. For further details, see "Layers" in the online help.

  • Context menu: See "Layers: Context menu commands" in the online help for further details on the commands accessible from the context menu.

Layer filtering methods

Filters can be defined in order to limit the display of the current layer list. For more details on the selective display of layers, see "Layers: Filtering and Selection Methods" in the online help.

Entity Selection Filter

A new Attributes area has been added to the Selection Filter dialog box (available on right-clicking on the Graphics Area and selecting Select - Filter in the context menu) to provide a filter for the entity selection based on the layer properties.



For example, when a selection filter is set as follows:
Color: By layer
Layer: 0
you can select only those entities that belong to layer 0 and the color attribute of which is set to By layer (in this example, it is Green).

New Layer toolbar

A new Layers toolbar () has been made available for easy management of layers. See "Working with Layers" in the online help for further details.

By layer and selection

A new value called By layer (denote an associated property of a current layer) has been added to the layer properties such as Color, Line type and Width at the following locations:

  • Statusbar
  • Entity Properties - General category
  • Selection Filter dialog box — only Color and Line properties have this new value

To change the property values associated to a layer, left-click on its corresponding icon and then make the new selection from the available list. The layer property affects only the entities on a layer whose value is set to By layer.

For further details on the layer selection, see "Layers: Filter and Selection Methods" in the online documentation.

Visual Bookmarks

While applying the New Visual Bookmark command, it is now possible to define the visibility status of all layers that are created after the current Visual Bookmark has been generated. This status can be defined using the Layers drop-down list present under the New EntitiesMore OptionsCapture Settings branch of the New Visual Bookmark selection list.


Data Exchange (Converters)

During the data exchange to/from think3 to other formats, the layer information has to be correctly converted. Hence, the enhancements in the Layer Management do impact the data conversion functionality. Refer to the following topics for details:

  • New General - Layers category for IGES converter
  • New General - Layers category for STEP converter

Impact on other commands

In several previous version commands you could specify a layer by typing the corresponding number in a box. All those boxes have been replaced with an editable drop-down list in which you can either choose an existing layer or type in the name of a new one to create it. This enhancement has an impact on the following Options/Properties categories and commands:

  • Entity Properties - General category
  • Entity Properties - Drawing - View Attributes category
  • Document Properties - Drawing - View Attributes category
  • Formats
  • Make Solid
  • Print Setup and Print
  • InsertSurfaceBreak Solid
  • ModifySurfaceSplit with Silhouette
