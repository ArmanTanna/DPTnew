Installation and Licensing

Version 2008.1 introduces the following improvements in the installation procedure and in the licensing functionalities.

  • New Windows® Terminal Server Session Licensing method
  • The new Export License file extension: *.tbu

New Windows® Terminal Server Session Licensing method

Starting from this version, users of Windows® Terminal Server Sessions can run think3 applications only using floating licenses after setting the T3TSS system environment variable to "YES" or "yes" on the server.
T3TSS=YES
or
T3TSS=yes

The new Export License file extension: *.tbu

From this version onwards the licenses to be exported manually will be generated with *.tbu extension.
