Comparative Details of thinkreal and ThinkDesign

thinkreal functionalities have been made availble directly inside the new version of ThinkDesign, where a Rendering toolbar can now be activated. This document proposes a comparison between the old and the new rendering functionalities. For further details see "thinkreal vs Rendering in ThinkDesign".

Material

In thinkreal the material was applied using the Material Manager dialog box, whereas in ThinkDesign the material is applied from the Rendering Library.

thinkreal ThinkDesign

In thinkreal the Color, Reflectance, Displacement and Transparency were set using their respective tabs, whereas in ThinkDesign all these options are present in the Edit Material selection list.

thinkreal ThinkDesign

Decal

In thinkreal the decal was applied using the Decal Manager dialog box, whereas in ThinkDesign the decal is applied using the Insert Decal command from the Rendering toolbar.

thinkreal ThinkDesign

In thinkreal the Edit Decal Mask command was used from the Decal Manager, whereas in ThinkDesign this option is in the Insert Decal selection list.

thinkreal ThinkDesign

In thinkreal the Reflectance, Transparency and Displacement were set using their respective tab, whereas in ThinkDesign all these options are present in the Insert Decal selection list.

thinkreal ThinkDesign

Texture

In thinkreal the Texture Space Manager was present, whereas the same functionality is now available in the Edit Material and <Insert Decal selection lists.

thinkreal ThinkDesign



Lighting

In thinkreal the light studio was applied using Light Manager, whereas in ThinkDesign it is applied using the Rendering Library.

thinkreal ThinkDesign

Environment

In thinkreal the Scenery Manager was used to apply scenery, whereas in ThinkDesign it is applied using the Rendering Library.

thinkreal ThinkDesign

In thinkreal the Scenery, Foreground and Background were set using their respective tab, whereas in ThinkDesign all these options are present in the Environment selection list.
