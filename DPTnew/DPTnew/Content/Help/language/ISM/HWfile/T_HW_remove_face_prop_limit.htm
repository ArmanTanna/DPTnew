Removing Faces Using Propagation Limits

This illustration explains the use of Propagation Limits to remove some selected faces from a solid by using the Remove Faces command. Open the ISM_prop_limits.e3 file available in the Samples folder of your think3 installation and follow the steps given below.

  1. Invoke the Remove Faces command.

  2. Select the bottom face of the slot for the Faces selector.



  3. Select the top face for the Propagation Limits selector. All the intermediate faces automatically get selected for removal.



  4. Apply the command. The slot is completely removed from the model.
