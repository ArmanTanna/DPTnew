Roto-translation of Faces

The following illustration explains a situation where you will simultaneously rotate and translate a set of user selected faces using the Interactive Modeling command. Open the ISM_move_face.e3 file available in the Samples folder of your think3 installation and follow the steps given below.

  1. Invoke the Move Faces command and select the face to be moved, as shown below.



    To reselect the face, right-click on the Faces selector, select Reset from the context menu and then make a fresh selection. You can make use of Propagation Limits, if needed.

  2. Drag the Z Axis handle or in the Delta Z mini dialog box, enter 20.



  3. Drag the Z-X rotation handle or in the corresponding Angle mini dialog box, enter 15.



  4. Apply the command to make the transformation.



    The face is moved upwards and also gets tilted in a single step.
