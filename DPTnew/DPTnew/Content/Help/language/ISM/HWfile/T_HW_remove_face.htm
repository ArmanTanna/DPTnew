Removing Faces

This illustration explains a situation where you will remove some selected faces from a solid by using the Interactive Modeling command. Open the ISM_remove_faces.e3 file available in the Samples folder of your think3 installation and follow the steps given below.

  1. Invoke the Remove Faces command.

  2. Select the face to be removed. (Here we intent to remove the extra bulge)



    You can make use of Propagation Limits, if needed.

  3. Apply the command.



  4. Select the next face to be removed. (Here we intend to convert the blind slot to a thru slot)



  5. Apply the command.
