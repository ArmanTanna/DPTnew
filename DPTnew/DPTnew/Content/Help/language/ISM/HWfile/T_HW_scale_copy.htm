Scaling and Copying Faces

The following illustration explains a situation where you will scale and copy the selected faces while moving them using the Interactive Modeling command. Open the ISM_scale_copy.e3 file available in the Samples folder of your think3 installation and follow the steps given below.

  1. Invoke the Move Faces command and select the side face of the cylindrical hole to be moved.



    To reselect the face, right-click on the Faces selector, select Reset from the context menu and then make a fresh selection. You can make use of Propagation Limits, if needed.

  2. You need to reset the handle origin to the center of the hole. Right-click on Start point under Handle Origin option in the selection list. Select Reset from the context menu and then click on the center of the hole.



  3. Drag the Z handle or in the Delta Z mini dialog box, enter 100 such that the preview of the hole comes to the center of the cylindrical head.



  4. In the Move Faces selection list, select the Copy check box. In the Scale Factor, enter 2.5.

  5. Apply the command to make the transformation.
