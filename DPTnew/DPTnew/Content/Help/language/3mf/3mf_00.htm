Conversion to and from the 3MF format

3MF or 3D manufacturing format is a 3D printing format along with AMF meant to supersede the STL format (see www.3MF.io for more details). This document describes how to convert 3MF files to this software format and vice versa.

Introduction

The application enables you to read and write files in the 3MF format.

Do the following:

  • To convert from the 3MF format to this software format, use the Open or InsertFrom File command. In the file dialog box, under Files of type, select the type of file you wish to convert from by selecting the appropriate file format (".3mf").
  • To convert to 3MF format, use the Save or Save As command. In the Save dialog box, under Save as type, select the type of file you wish to convert to by selecting the appropriate file format (".3mf").

Conversion is controlled by the parameters of the Settings tab of the 3MF Options dialog box opened by selecting Options. You can save as many different sets of parameter values  — to be used in different conversion situations —  as you like to the Settings library.
