Log File of the ThinkDesign to 3MF Conversion

When saving a 3MF file, a log file of the conversion is created in the same directory as the 3MF file with the same name and the ".log" extension.

The log file structure

The log file created during conversion of e3 file to 3MF format has a structure made of the following sections:

  • Document scan and mesh creation report — A section starting with "Scanning document for solids, surfaces and meshes to output" that lists found entities and their mesh creation warnings and errors with exhaustive information to help identify possible issues with the entities, the mesh creation process and/or the created meshes.
  • Export information — The remainder of the log file lists information regarding the actual export into the 3MF file.

Example of log file - conversion to 3MF

Scanning document for solids, surfaces and meshes to output
Scanning found entities and creating output meshes.
Processed 1 solids, 0 surfaces and 0 meshes.
Building the components structure.
Finished building the components structure.
Compiling output objects
Compiling build section
Exporting output objects to 3MF
Exporting successful.
