InsertVisual BookmarkNew

The New Visual Bookmark command enables you to insert a new Visual Bookmark.

You can perform the command selecting it from the context menu displayed when right-clicking on an empty area in the Visual Bookmarks tab on the left pane of your model/drawing document.

To order and classify Visual Bookmarks, using the Category drop-down list you can define categories which the Visual Bookmarks will be assigned to. You can always choose to assign a new Visual Bookmark to an existing category or to create a new category and assign the Visual Bookmark to the latter. You can also drag the Visual Bookmark to a different category.

A warning message is displayed if you typed in the name of a Visual Bookmark already existing within the same category.
