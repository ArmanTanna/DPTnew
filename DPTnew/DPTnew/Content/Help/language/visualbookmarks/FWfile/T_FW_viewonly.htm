The View Only command enables you to recall an existing Visual Bookmark any time, so that you can view the bookmark as it is created. You will not be able to edit it though. If you exit the View Only mode, the original view will be restored. This command helps you to perform a quick cross check if the Visual bookmark has captured all the details that you intend to capture without activating it.

You can perform the command from the context menu displayed when right-clicking the selected Visual Bookmark.

You can view the Visual Bookmarks from the Visual Bookmarks tab on the left pane of your model/drawing document.
