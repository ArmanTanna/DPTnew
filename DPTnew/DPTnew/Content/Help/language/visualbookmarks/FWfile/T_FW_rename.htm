The Rename command enables you to rename the selected Visual Bookmark or Category collecting a set of Visual Bookmarks. When you apply the command, you enter in edit mode and are allowed to modify the Visual Bookmark/Category name.

In actual fact, in compliance with Windows Explorer standards, you can rename a Visual Bookmark or a Category also by positioning the selection cursor over its name and typing it.

You can perform the command from the context menu displayed when right-clicking the selected Visual Bookmark.

You can view the Visual Bookmarks from the Visual Bookmarks tab on the left pane of your model/drawing document.

A warning message is displayed if you typed in the name of a Category already existing or a Visual Bookmark already existing within the same category.
