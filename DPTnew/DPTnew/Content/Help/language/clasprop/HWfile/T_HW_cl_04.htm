Assign user defined properties to entity

When you start the Assign UDP command, the Select class(es) to be assigned dialog box is displayed, which contains a list of the classes already defined so that you can choose one of them for assigning properties. Initially, no classes are selected.

1 Select one or more properties.
To select all the properties in the list, click on All. To deselect a property, click on it again.
2 Click OK.
3 The Assign Values to Properties dialog box is displayed. For each property selected, type the value of the property in the Value box.
4 To overwrite properties that have already been assigned, select the Replace if existing check box.
5 Select the entities you want to assign to the modeldrawing.
Or click on Assign to Model to assign to the entire modeldrawing.
6 Click on Close.
