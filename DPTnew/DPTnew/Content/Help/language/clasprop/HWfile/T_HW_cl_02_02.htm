Modify the definition of a class by adding a description

  1. Start the Modify UDC command; the Select class(es) to be modified dialog box is displayed.
  2. In the list of defined classes, choose the class for which the description is to be added; the Property Class dialog box is displayed.
  3. Type the description in the Description: text box.
  4. To apply the change, click on Insert. A message indicates that the change has been applied to the selected class.
    Click OK to proceed.
  5. To close, click on Close in the Property Class dialog box and then on Close in the Select class(es) to be modified dialog box.
