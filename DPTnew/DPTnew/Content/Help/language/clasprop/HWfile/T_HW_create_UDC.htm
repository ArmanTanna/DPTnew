Create a class and assign the corresponding UDP to the model/drawing

An alternative method to create a UDC (User Defined Class) is to perform this operation from the File Properties dialog box (see also Create UDC). To do this, follow the steps below:
 
1 Open File Properties dialog box.
2 Select the Custom tab.
3 In the Name box, type the UDC name, with the following conventions:
  • Maximum length 128 characters.
  • The rest of the characters may be letters, numbers or the - (minus), _ (underscore) and $ (dollar) characters.
4 In the Type selection box, select the class type:
  • Text — character string class.
  • Integer number — integer class.
  • Real number — real number class.
5 In the Value box, type the value of User Defined Property you want to assign to the model/drawing.
6 Select Add to actually create the UDC and assign the UDP to the model/drawing, with the value you set in step 3. The UDP will be displayed in the list.
