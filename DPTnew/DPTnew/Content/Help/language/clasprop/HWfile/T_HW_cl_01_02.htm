Create a class

  1. Start the Create UDC command.
  2. The Property Class dialog box is displayed.
  3. Type the "CODE" name in the Name: box.
  4. Choose Character in the Type: list box.
  5. Deselect the Entity check box in the Assignable to: row, so that the properties can only be assigned to the model and to solids.
  6. Click on Insert to create the class and then Close to close the dialog box.
Note

To create a class and assign it to the model/drawing itself, you can proceed as described in "Create a class and assign the corresponding UDP to the model/drawing", about the Custom tab of the File Properties dialog box.
