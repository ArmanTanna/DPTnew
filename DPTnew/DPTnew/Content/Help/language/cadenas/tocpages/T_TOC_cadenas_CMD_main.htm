PARTsolutions plug-in commands

About PARTsolutions Displays the software version and copyright information
Import PARTdataManager Group Enables you to add a 2D view of an object in the CADENAS catalog as a group in our current drawing
Import from PARTdataManager Catalog Enables you to add a 3D part to our current model
Import from PARTshaft Enables you to add a 3D shaft to our current model
Create Native Part inside PARTdataManager Enables you to add our model to the CADENAS catalog
Import from PARTdataManager File Enables you to import a CADENAS native part file in the current document
Replace Object Enables you to replace existing PARTsolutions solids with a new solid imported from the CADENAS PARTdataManager™
