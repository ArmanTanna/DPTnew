Cadenas software installation guidelines

The following is intended to be a guideline to the installation of Cadenas software products in order to prepare the system for the installation of the PARTsolutions plugin.
It does not cover in detail all aspects of the software installation; please refer to the Cadenas installation guide for more details.

To install the Cadenas catalogs, follow these steps:

  • Insert the Cadenas PARTSolutions and eCATALOGsolutions CD and wait for the setup wizard to start. The reference version of this guideline is 8.0.24. Click Next to continue.



  • Select the STANDARD/PROFESSIONAL version, then click Next:



  • Select the single workstation option if you plan to use a standalone workstation; select the multi workstation if you plan to use the software in a client/server environment. In this example we will install a standalone workstation. Click Next when ready:



  • If you wish to install the catalogs data in a different path than the software installation, select the option to install them in different directories. In this example we will install in the same path. Click Next when ready:



  • Use Browse to choose the target pathname where you want to install the software. For single user installations the default location is: C:\Program Files\cadenas\PartSolutions; for multiuser installations choose the path on a server, e.g. \\[server]\cadenas\PartSolutions. Click Next when ready:



  • Select the modules that you want to install. You will need to install PARTSolutions for the integration with the CAD editor and eCATALOGsolutions for the integration with thinkPLM. Click Next when ready:



  • In order to use the integration with our products, make sure that no other CAD interface is installed. Click Next when ready:



  • In order to use the integration with thinkPLM, make sure that no other EDM/PDM/PLM interface is installed. Click Next when ready:



  • Make sure that no ERP interface is installed. Click Next when ready:



  • Check the classifications you wish to install. Click Next when ready:



  • Check the standards you wish to install. Click Next when ready:



  • Review the installation summary. Click Back to change any installation parameters you wish, according with the aforementioned limitations. Click Next when ready, and wait for the configuration process to complete. You may be required to reboot your system to complete the installation.
