Create Visual Bookmark

The Create Visual Bookmark command in the mating properties dialog captures a Visual Bookmark highlighting the entities involved in the mating. It is important to note that you can use this command at any level of the assembly hierarchy.

When you apply this command on a mating event that is at the top of the assembly:


  • The bookmark is created under a Mating folder in the VB tree
  • When you Activate the bookmark, all the entities that are not part of the mating are hidden.

    How to Unhide the hidden entities?

    When you Activate the bookmark, the entities are hidden to the level of the Profile. In large assemblies this can be a problem when you need to Unhide all the entities. Before activating the created bookmark, its better to create another new bookmark with all the entities visible and keeping the Hide/Unhide and Recursive Capture options checked.

    This Visual Bookmark enables to get back to the initial view.


  • A zoom to fit is applied on the mating entities to give you a clear view.

When you apply this command on a mating event at a lower level in the assembly and Activate it:


  • All entities at the same level that are not part of the mating are hidden.
  • The entities that form the mating are Set Current and are seen.
  • A Zoom to fit is applied on the mating entities.
