The Isolate command enables you to view to the entities involved in the selected constraint in isolation of the other entities that make up the assembly.

The command is available from the context menu displayed when right-clicking the selected mating constraint in the Mating Structure.

When you perform the command on a mating constraint at the top level of the assembly:

  • All entities are hidden showing only those which involve the constraints.
  • The constraints are shown with highlighted faces and arrows.
  • A zoom is applied on the involved components.

When you perform the command on a mating constraint that is at a lower level of the assembly:

  • The top of the Selection list shows the selected constraint that has been isolated.
  • All the components at the same level of the selected mating are hidden except those that involved in the selected constraint which are Set Current.
  • All components higher up in the assembly hierarchy that are not current are visible but greyed.
  • A Zoom fit is applied on the involved components.

You can note that this command can be used on constraints at any level of the assembly hierarchy. When you exit the command, the previous view is restored.
