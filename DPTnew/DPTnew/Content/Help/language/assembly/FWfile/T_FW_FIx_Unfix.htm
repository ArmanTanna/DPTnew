InsertSolidFix
InsertSolidUnFix

The Fix command enables you to fix the location of the selected entity (solids/components) in the 3D space. The representation of the Fix command is an anchor () and is displayed in the Mating Structure..

The Unfix command enables you to make the fixed component unfixed.

Note:
You can also access this command from the context menu by right-clicking the entity in the Graphics Area or in the Model Structure.
