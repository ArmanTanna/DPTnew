InsertSolidMating

The Mating command enables you to position two entities relative one to each other by applying geometrical constraints between the entities that form the solids. Note that when mating solids, it is not necessary to align the reference system of the solid selected as Second Entity with that of the solid selected as Second Entity or vice-versa.

  • By checking/unchecking the Deferred Mating box, you can determine when mating constraints are to be applied.
  • By selecting the Symbolic References Only box (available if the relevant Use symbolic references option is enabled), picking is restricted only to entities associated to Symbolic References.
  • By checking the Tangency chain (First Entity) or Tangency chain (Second Entity) boxes (available if the Tangency constraint has been selected), you can apply the mating constraint to all surfaces with tangency continuity to the selected surfaces.
  • The Mode drop-down list enables you to select the constraint detection mode.
  • In the Type drop-down list you can select the type of positioning constraints.

The Mating command also enables you to drag and drop an entity partially constrained by a mating constraint to a different position. Both the preview and the final position will be consistent with the existing constraints.

You apply the mating constraint by switching to a new command, by selecting Rebuild All (if you chose the Deferred Mating option), or, if you wish to define a new constraint, by picking a new entity, belonging to the object that will be moved in the mating.
