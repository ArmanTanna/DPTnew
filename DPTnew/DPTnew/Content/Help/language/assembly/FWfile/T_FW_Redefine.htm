The Redefine command enables you to modify an existing mating by redefining a single constraint within the mating.

The command is available from the context menu displayed when right-clicking the selected constraint in the Mating Structure.

You can apply the command to constraints belonging to Solid Mating, even if further features have been applied to the involved solids after the mating was performed.

When you perform the command, the whole set of the Mating parameters are displayed in the selection list, so that you can redefine the constraint.
