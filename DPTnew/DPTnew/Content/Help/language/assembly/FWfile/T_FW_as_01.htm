InsertComponentNew

The  New Component command enables you to create a new component.

The component you create is a local one, as you can see from the Component category of the Entity Properties. You can make it an external reference by the Make X-Reference command.

Wishing to create a new component and transform it into an X-Reference component in a single step, perform the New Component as X-Reference command.
