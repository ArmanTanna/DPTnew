ModifyComponentBreak

The  Break Component command breaks a component into the individual entities which represent it.

After this operation, the component is present in the model as one or more entities.

Note that if the component had been positioned using some mating constraints and the component includes more than one solid/component, the mating event is removed and a warning message is displayed.

Also note that if the component includes some variables and the model already contains variables with the same names, you need to provide further information in order to carry out the breaking action.
