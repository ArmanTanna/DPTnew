A new Open X-Reference command has been developed so as to enable you to open the original file of an X-Reference component.

To start the new command do the following:

  1. Right-click on an occurrence of an X-Reference in the model or in the Model Structure.
  2. Select Open X-Reference in the context menu.

The original document of the X-Reference component is opened for editing.

Note
You can always open the original file of an X-Reference component by using the FileOpen command, but using the contextual Open X-Reference command is much easier and quicker.
