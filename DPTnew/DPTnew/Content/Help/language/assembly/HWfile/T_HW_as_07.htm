Reloading an assembly component

1  On selecting the Reload command from the menu bar, the Reload Component dialog box is displayed. Choose one of the following options by selecting the appropriate radio button:
  • Select — reloads selected components. You are prompted to click on the components you want to reload.
  • All modified — reloads all the modified components.
  • All unloaded — reloads all the components previously unloaded using the Unload command.
2 If you want to reload a single component that was previously unloaded, right-click on the component in the Model Structure, then select the Reload command from the context menu.
