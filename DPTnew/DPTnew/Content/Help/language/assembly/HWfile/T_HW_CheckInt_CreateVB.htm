Creating Visual Bookmarks showing interferences

Please refer to Checking interferences in an assembly for the information on the previous steps.

Once you've calculated the interferences in the assembly you are working at, wishing to automatically create Visual Bookmarks showing the single interferences (and the original view), proceed as follows:

3.1 In the tree view of the Interference Results dialog box, select the item that you wish to be viewed in the Visual Bookmark, for instance, "All interferences", and click the Create Visual Bookmark(s) button.

The program displays a message informing you that one or more bookmarks have been created.

3.2 Confirm by OK.
Selecting the Visual Bookmarks tab in the left pane, you can view the bookmarks that have been automatically created:

By selecting any of them, you will view the corresponding interfering objects, while all other objects will be hidden, or the original view.
