The options in this drop-down list enable you to capture images of the steps for defined conditions.

Single: Captures images of the current step.

Sequence: Captures images for a range of steps of the exploded assembly. The Start step and End step edit boxes get displayed on selecting this option. Enter the step number in the respective edit boxes in order to create images for the defined sequence.

In order to create the images of the whole sequence of the explode process you need to enter 0 in the Start step text box and the value of the Last Step text box in the End step edit box.
