Opening a PLY file

To convert from the PLY format to think3, do the following:

  1. Select Open or InsertFrom file
  2. In the file dialog box, under Files of type, select Stanford Polygon File (*.ply).
  3. Select the file you want to open.
  4. Select the Options button to open the PLY Options dialog box and set the parameters controlling the conversion process as you like.
  5. Click the Open button to open the file (no preview is available for PLY files: the file icon will be displayed in the preview frame).
