Visual Properties

The Visual Properties enables you to define the characteristics properties of the material or decal applied. These properties are available in the Edit Material, Decal and Edit Decal selection list. The following are the visual properties available:

Texture Color Enables you to define the color and texture properties.
Reflectance Enables you to define the reflectance properties.
Transparency Enables you to define the transparency properties.
Bumps Enables you to define the roughness and indentations.

Note
The Texture Color visual property is available only for Edit Material selection list.
