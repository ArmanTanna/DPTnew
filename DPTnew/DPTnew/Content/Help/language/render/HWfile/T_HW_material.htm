Applying Material to a Model

In this tutorial you will apply different materials to the model from the Rendering Library palette. Open the Wheel_Rim.e3 file available in the Samples/render folder of your think3 installation and follow the steps given below.

  1. Start the Lighting command. Select Light1 from the Lights drop-down list and select the Lights check box.

  2. Select the Spot option from the Type drop-down list. Now expand the Type node and select the Shadows check box in the selection list.

  3. Now orient this light using the Four Viewports command, as shown in the image below.



    Click or to confirm your selections for lighting.

  4. Start the ViewRendering Library command to get the Rendering Library palette. Click on the Material tab to get the library materials.

  5. Under the Predefined folder, select the Shadow Catcher folder. Right-click on the Shadow Catcher in the preview pane and select Attach from the context menu.

  6. Then select the surface under the rim, as highlighted in the image below. The shadow catcher is applied on to the surface.



  7. Under the Metals folder, select the Steel folder. Right-click on the Polished Steel in the preview pane and select Attach from the context menu. Now select the rim part as highlighted in the image below.



  8. Select the Gold folder under the Metal folder. Right-click on the Polished Gold and select the Attach from the context menu.

  9. Then select the alphabets think in the model one by one to apply the material.

  10. Expand the Metal Finishes node and select the Enamel folder. Right-click on the Red Enamel in the preview pane and select Attach from the context menu. Select the 3 numerical in the rim.

  11. All the materials attached to the model are listed under Materials in the Rendering tree.

  12. Click on the Rendering Preview command to get the final rendered image as shown in the image below.
