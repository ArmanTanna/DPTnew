Applying Texture Space

The texture space can be applied on to the decal or a wrapped material applied on the surface.

  1. The mapping surface is selected using the Surface selector in the Texture Space selection list.

  2. Under the Texture Mapping drop down list, you can select the method of mapping. The following are the different types of mapping available.
    UV A texture space option that assigns the parametric coordinate system to the texture space.
    Sphere A texture space option that assigns the parametric coordinate system to the texture space.
    Plane A texture space in which all points in space are mapped onto a plane of constant x (y or z).
    Cylinder A texture space in which all points in space are mapped onto a cylinder.
    Automatic A texture space selected by the application which is best suited for it.

  3. Additional options can be selected by expanding the Texture Mapping node.
    Scaling factor can be specified in the Scale edit box.

  4. In the Advanced options, you can specify the direction using the Normal or Axis Direction option.
    The Normal option is available when Plane is selected as mapping method. Axis Direction is available when Cylinder is selected as mapping method.

  5. The Show mapping surface and Show Handles check box can be selected to view the mapping surface and mapping handles respectively.

Note
The Show mapping surface check box is available for all options except for UV method.
