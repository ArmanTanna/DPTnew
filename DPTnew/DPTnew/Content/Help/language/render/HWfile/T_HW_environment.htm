Applying Environment to a Model

In this tutorial you will apply an environment for a boat model. Open the boat.e3 file available in the Samples/render folder of your think3 installation and follow the steps given below.

  1. The model is a representation of a boat. Setting up the lighting is important for a good rendered image. For this tutorial we will set up three lights for the model.

  2. Start the Lighting command. Ensure that the Ambient light is selected from the Lights drop-down list.

    Note:
    • It is mandatory to select the Lights check box after selecting the light form the corresponding drop-down list. If the check box is cleared, then the selected light will not be activated and hence no light will be displayed in the Graphics Area.

    • Clear the Fixed To View check box so that the orientation of lights remains fixed with respect to the model.

  3. Now select the Light0 from the Lights drop-down list.

  4. In the Type drop-down list select the Distant option. Expand the node and select the Shadows check box.

  5. Now orient this light using the Four Viewports command, as shown in the image below.



  6. Select the Light1 from the Lights drop-down list and Distant option from the Type drop-down list.

  7. Now orient the light, as shown in the image below.



    Click or to confirm your selections for lighting.

  8. Select one of the view (highlighted in the image above) and start the ViewDisplayViewportsSingle Viewport command to view the model in single view as shown in the image below.



  9. Start the FormatEnvironment command. Ensure that the Image option from the Background drop-down list is selected.

  10. Expand the node and select Browse option from the Image drop-down list. The Browse for File dialog box is displayed.

  11. Select the environment1.jpg image from the Samples/render folder of your installation.



  12. Click or to confirm your selections for the environment.

  13. Start the ViewRendering Library command, the Rendering Library palette is displayed.

  14. In the Material tab, select Shadow Catcher under the Predefined folder.

  15. In the preview pane, right-click on the Shadow Catcher and select Attach from the context menu. Then select the surface under the model to apply the shadow catcher material as shown in the image below.



  16. Now select the FormatRenderingPreview command to view the rendered image, as shown in the image below.
