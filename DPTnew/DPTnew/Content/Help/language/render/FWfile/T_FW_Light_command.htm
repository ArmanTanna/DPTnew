FormatLighting

The Lighting command enables you to edit the default light settings of the application or the light studio applied using the Light Studio tab of the Rendering Library. Start the command to get the following options in the selection list.

The Lights check box when selected enables you to add or delete a light from the light studio. Two buttons are available corresponding to the Lights check box in the selection list.
New Light Each click on the button adds a new light to the light studio. Each new light added to the studio will be displayed in the corresponding drop-down list.

Note
Also a new light can be added by selecting the type of lighting from the context menu by right clicking on the button and select New LightLighting options. Then select one of the lighting options.
Delete Light Each click on the button deletes a light from the light studio. The light that has to be deleted to be selected from the corresponding drop-down list.

The Type drop down list enables you to select different types of light settings.

Under Studio Options the Fixed To View check box is available.

  • When the check box is cleared it sets light orientation with respect to the current view. Hence, if the model is rotated, then even the light changes the position accordingly.
  • When the check box is selected, the light is fixed in a position. Hence if the model is rotated, then the light remains fixed in the same position.

Under More Options following check boxes are available.

  • All Lights Preview:
    • When this check box is selected, all the lights are displayed in the current light studio.
    • When the check box is cleared, only the light selected in the lights drop-down list is displayed.

  • Show Handles:
    • When the check box is selected, the vector handle is displayed for the lights present in the current light studio.
    • When the check box is cleared, the vector handle is not displayed.

After making the required selection, the following buttons are available in the selection list.
/ Hide Preview/Show Preview Toggles between hide and show preview modes of the current lights.
Rendering Performs a quick rendering.

Note
  • All the options in the selection list which are in Bold or appended with (*) sign are used for static rendering; Hence, the settings made in these parameters are visible only in static rendering.
