• Names that match with a string containing the default wildcard at the end. For example, Lay* displays all the layers starting with the string Lay.
  • Name of single layer. For example, Layer1 results to show only Layer1.
  • More than one names, using comma as names separator. For example, 0,Layer* results to show 0 and all the items that contains the prefix Layer.

    Note that the filter operation is case insensitive.
Also, you are enabled to choose the previously entered search string by selecting it from the drop-down list.
