The Select Layers command enables you to select only the layers that meet the chosen selection criteria. This command is also available in the context menu on right-clicking on any layer column or on the empty area.

The below described predefined selection options can be activated in two ways:

  • Left-clicking on the Select Layers button
  • Right-clicking on the layer properties and selecting the Select Layers command from the context menu.

The layer properties such as Color:, Line type: and Width: contains a value called By layer (denote an associated property of a current layer) at the following locations:

  • Statusbar
  • Entity Properties - General category
  • Selection Filter dialog box — only Color: and Line: properties have this new value.

To change the property values associated to a layer, left-click on its corresponding icon and then make the new selection from the available list. The layer property affects only the entities on a layer whose value is set to By layer.
