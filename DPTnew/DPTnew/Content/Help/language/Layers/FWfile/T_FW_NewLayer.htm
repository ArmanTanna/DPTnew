The New Layer command enables you to create a new layer.
The following are the characteristics of this command:

  • By default, the layer name is Layer with an incremented number (for example, Layer1, Layer2). You can also turn off the prefix Layer using the option Show default prefix in new layer names in the System Options-General-Advanced-Layers category.
  • You can rename a layer by clicking the name cell.
  • The new layer inherits the properties of the currently selected layer in the layer list.
