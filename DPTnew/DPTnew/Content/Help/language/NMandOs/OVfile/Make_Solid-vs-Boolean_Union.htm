Open Solids: Boolean Union vs Make Solid

You may want to create an open solid joining two or more open solids together. You can do that using both Make Solid and Boolean Union commands. However, there is a significant difference between those two ways, dealing with normals, that you should be aware of.

The difference between using the two commands concerns the normals of the faces involved. The Boolean Union command ignores the normals and does not affect them in any way. The Make Solid command will flip the normals of any surfaces that do not match those of the first surface selected.
