The Quick Move/Copy Entities command ( ) — available under the Commands tab of the Customize command — enables you to perform translations and rotations. The selected entities can be moved or duplicated as many times as you like. As soon as the command is started you can choose whether to perform a translation or a rotation:


Rotate
Translate

When you start the command for the first time the active option is Translate. Any time you start Quick Move/Copy Entities after the first one, the active option will be the one you set last time you used the command.

Enabling selection inside groups in copy mode
In the drawing environment, w When the Copies check box is selected so as to duplicate the selected entities rather than to move them (see the sections below for further details), the Inside group box enables you to select entities inside groups rather then the groups themselves.



To have the Inside group box work, first select the Group check box in the Inside area of the Selection Filter dialog box. Once you have done so:
  • When the box is checked, clicking on a group will result in selecting the entity you actually clicked on rather than the whole group it belongs to.
  • When the box is cleared, clicking on an entity belonging to a group will result in selecting the whole group rather than the entity you actually clicked on.
Rotate

As soon as the Rotate option is selected, the following options are displayed in the Quick Move/Copy dialog box.



Copies When selected, this check box enables you to duplicate the selected entities as many times as the value you type in the adjacent box.
Center Enables you to select the center of the rotation. Click the button and select the point to be used as the center of the rotation.
Start Enables you to select a new initial point for the rotation to be performed. By default, the initial point of a new rotation is the final point of the previous one.
Angle Enables you to specify the angle of rotation (measured counter-clockwise)
Axis Enables you to select the axis of rotation (not available in the drawing environment). By default, the program rotates entities about the Z-axis. To set a different axis of rotation, select an option in the Axis drop-down list:
X/Y/Z To define the axis of rotation as one of the coordinate axes of the Work Plane.
Line To define the axis of rotation as an existing line to be selected. Click the adjacent Select button and select the line.
2 Points To define the axis of rotation as the one identified by two points to be selected. Click the adjacent Select button and select the two points.

To perform a rotation, proceed as follows:
 

1 You are prompted to select an entity. Select the entities to be rotated and end selection by right-clicking and selecting End of Selection in the context menu.
  • To move the selected entities, leave the Copies box unchecked. To duplicate the selected entities, select the Copies box and type the number of copies to be performed into the adjacent box.
  • By default, the program will rotate the selected entities about the Z-axis. To set a different axis of rotation, use the Axis drop-down list as described above.
     
2 You are prompted to enter the center of the rotation. Select a point (you can use the Snap commands).
3 The program prompts you to enter the base position. Click to specify a point forming a direction vector from the center point.


 
4 The program then prompts you to enter the new position. Click to specify a point defining a new direction vector from the center point. The angle between it and the direction vector defined in step 3 determines the angle by which the entities are rotated.



Alternatively, in the Angle box, type the value of the angle of rotation. Hit ENTER on the keyboard to rotate the entities.

Starting a new rotation
As soon as the entities are rotated, a new rotation is ready to start from the final point of the previous one. To change the starting point of the new rotation, click Start and specify a new position.

Changing to a translation
If, after selecting the entities and specifying the center of the rotation, you select the Translate option, the point previously selected as the center of the rotation will be considered as the base point of the translation.

 

Translate

As soon as the Translate option is selected, the following options are displayed in the Quick Move/Copy dialog box.



Copies When selected, this check box enables you to duplicate the selected entities as many times as the value you type in the adjacent box.
Start Enables you to select a new initial point for the translation to be performed. By default, the initial point of a new translation is the final point of the previous one.
PX/PY/PZ Enable you to specify the components of the translation along the coordinate axes of the Work Plane (PZ is obviously not available in the drawing environment.
Line Enables you to specify an existing line as the direction of translation. After clicking the Line button, select the line. In the adjacent Delta box, type the length of the translation along the chosen direction.
Please note that the point used to select the line determines the orientation of the translation: the selected entities will be moved towards the visible endpoint of the selected line which is closest to the selection point. In this case, the endpoints to be considered are not the actual endpoints of the line, which might lie outside the current view, thus resulting not visible, but the visible line endpoints relative to the current view.

To perform a translation, proceed as follows:
 

1 You are prompted to select an entity. Select the entities to be translated and end selection by right-clicking and selecting End of Selection in the context menu.




To move the selected entities, leave the Copies box unchecked. To duplicate the selected entities, select the Copies box and type the number of copies to be performed into the adjacent box.
 
2 You are prompted to enter the base position. Click to specify the origin of the translation vector.


 
3 The program then prompts you to enter the new position. Alternatively, you can:
  • Click to specify the target point.


     
  • Type the X,  and Y , and Z components of the translation vector in the corresponding boxes (PX/PY /PZ).
  • Click Line and enter a Delta value.

     
To translate the selected entities, hit ENTER.

Starting a new translation
As soon as the entities are translated, a new translation is ready to start from the final point of the previous one. To change the starting point of the new translation, click Start and specify a new position.

Changing to a rotation
If, after selecting the entities and specifying the base point of the translation, you select the Rotate option, the point previously selected as the base point of the translation will be considered as the center of the rotation.
