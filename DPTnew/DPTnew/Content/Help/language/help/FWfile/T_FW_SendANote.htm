HelpSend a Note


 

The Send a Note command provides you with a simple and useful set of forms enabling you to report a bug using the e-mail services.

When reporting the problem, you can attach any type of document (model, drawing, screenshot or others) which can help you to describe what happened.

If your e-mail system doesn't support the MAPI services, you won't probably be able to send the report immediately. In this case, set the Mode option under Send e-mail (General category of the System Options dialog box) to Deferred and send your report using the mail.exe program.
