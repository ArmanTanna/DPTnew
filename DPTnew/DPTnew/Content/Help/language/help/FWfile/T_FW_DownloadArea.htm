HelpDownload Area


 

The Download Area command enables you to access a web page in which you will find links to download the latest version of free2Design.:

  • The Product page, from which you can download updates to standard think3 products.
  • The Utilities page, from which you can download several auxiliary tools not supplied with the think3 products installation CD-ROM.
  • The Documentation page, from which you can download all of the most recent available documentation updates.

Supply your personal accountuser name and password for free2Design community site to sign in.
