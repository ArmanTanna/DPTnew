Attaching documents to the Send a Note report

To attach a document to the Send a Note report, select Add New File to select the file type and add the file to the list displayed in the box below. The file size is also displayed.

Possible file types are the following:

C&P packageA log file (extension ".cp3") obtained by a Record session or a package file obtained selecting the Create CP package command in the Capture & Playback utility.
When you select it, the Logfile file selection box is displayed: 
  1. Select the log file or the package file you want to attach
  2. Click the Open button to attach the file to the report
The file is displayed in the list.
 
Current Document FileThe current model (".e3") or drawing (".e2") file.
When you select it, the model or drawing file is added to the list
 
External FileAny kind of files. This type is the one to be set if you want to attach a text document to the report.
When you select it, the Attach an External file file selection box is displayed
  1. In the file dialog box, under Files of type, select the type of the file you want to attach
  2. Select the file
  3. Click the Open button to attach the file to the report
The file is displayed in the list.
 
Graphics Area Snapshot ImageA GIF file (extension ".gif") is created as soon as you select it and it is added to the list.
The file will contain the current image of the graphics area. You can modify the image selecting it and then clicking Edit.


 

Detail Snapshot ImageA GIF file (extension ".gif") is created. As soon as you select it, a cross cursor is displayed in the graphics area: drag and drop it describing a rectangular window to define the detail of the current image you want to grab.
The file will contain the image of the window captured by the two picks and it is added to the list.

You can modify the image selecting it and then clicking Edit.
 
Application Window Snapshot Image A GIF file (extension ".gif") is created as soon as you select it and it is added to the list.
The file will contain the current image of the application window. You can modify the image selecting it and then clicking Edit.
 
GUI Object Snapshot Image A GIF file (extension ".gif") is created. As soon as you select it, an arrow cursor is displayed: position it on any item of the Graphic User Interface (GUI), such as dialog boxes, toolbars, model/drawing windows and pick on it.
The file will contain the selected item and is added to the list. You can modify the image selecting it and then clicking Edit.
 

Once you have added one or more files to the list you can perform some actions on them. First, you will have to select the file on which you want to make an action.
 

  • To compress the file, click the ZIP button
  • To remove the file from the list, click the Remove button.
  • If the file you selected is an image file, you will be able to Edit it: as soon as you click the Edit button, the image is automatically opened in Microsoft® Paint, where you can modify it.
Please note that the image you get is stored to a GIF file (extension ".gif"). When you edit it, it is converted to the BMP format, that is the one used by MS-Paint, and then converted again to GIF when you finish modifying it.

To allow think3 to publish the files you attached, select the check box:
Choosing this option grants think3 permission ....
When the box is not selected, publishing attached files will be forbidden.
