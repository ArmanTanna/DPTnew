Setting the e-mail options

The Mode drop-down list in the Send e-mail area of the General category of the System Options determines the way e-mail messages are sent by the command:
 

  • Send a note
Possible values are:

Direct If this option is active, e-mail are sent immediately. Execution is slowed down until the e-mail sending process is over. Any e-mail problems might cause the program to get blocked.
Asynchronous When the Send or Finish buttons are pressed, a background e-mail sending process is started in background. Any e-mail problem might cause the sending process to be blocked.
Deferred When the Send or Finish buttons are pressed, an auxiliary "SendMail" folder is created under the "thinkdesign" subfolder in the main MS-Windows directory of your PC. A specific subfolder is created for each e-mail message. The name of this subfolder includes a postfix displaying the sending time.
To send the mail, you have to run the mail.exe program (from Start => Run or from a MS-DOS window).
