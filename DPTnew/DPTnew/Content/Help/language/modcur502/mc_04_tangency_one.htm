Setting Tangency Continuity between curves - Fit Reference

The Fit reference curve method of the Curve Continuity command applies tangency continuity by modifying one curve only.



3 First select the curve you want to change, then the one you want to leave unchanged.



If a curve is specialized and not a NURBS, you are asked to confirm whether you want to convert it into a NURBS (unless you have deselected the Ask before converting check box).

Tolerance for converting into NURBS

You can control the way the curve is converted by setting a suitable Tolerance value. The lower the tolerance value, the more accurate the resulting NURBS will be and, hence, the more control points it will have.

4 Type the Scale factor into the corresponding box under More Options and the propagation factor into the Propagation mini-dialog box in the model.
Scale

Tangency modifies the section of the second curve nearest the point where it meets the other curve. The scale factor controls the amount by which that section of curve is modified. It must be a value greater than 0. The higher the value, the more marked the change in the section of curve.



In technical terms ...
The scale factor controls the distance between the first control point of the curve you want to modify (the one that is in contact with the other curve) and the second control point. Increasing the scale factor means moving the second control point away from the first and hence modifying the shape of the section of curve nearest the contact point.
Propagation The propagation factor controls the extent to which the change is distributed along the curve. It must be a value between 0 and 1. If you set 0, propagation is inhibited and the change remains local. For values greater than 0 (the maximum is 1), the higher the value the more the change extends to the rest of the curve.
