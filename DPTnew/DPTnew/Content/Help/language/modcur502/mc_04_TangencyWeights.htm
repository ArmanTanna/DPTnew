Setting tangency and modifying the shared endpoint: weights

  1. Open file "mc_02_04.e3e2".

  2. Start the Curve Continuity command.
     
  3. In the Degree drop-down list select Tangency.
     
  4. In the Method drop-down list select Modify common point and adjust both.
     
  5. Click on the two curves at points close to the ones shown in the illustration below:


     
  6. Answer Yes each time to enable conversion into NURBS.
