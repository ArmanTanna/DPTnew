Splitting by indicating points

  1. Open file "mc_01_07.e3e2".


     
  2. Select the Split Curve command.
     
  3. In the Mode drop-down list, choose Points.
     
  4. Click on the curve you want to split


     
  5. End selection by right-clicking and selecting Continue in the context menu.
     
  6. Select the points where you want to split the curve (by clicking on them or using a snap action).


     
  7. Click . The curve is split at the selected points:
