System Options

Category: Dynamic Visualization-Advanced

Advanced

Dynamic visualization

The controls in this area determine the dynamic visualization of views when they are dragged (to rotate) using the mouse.

Simplified representation

Simplified representation:

The display of simplified entities can be either points or wireframe boxes or shaded boxes. The desired display setting can be set in the edit box as:

  • Shading - The simplified entities will be displayed as a set of points covering the area of the object.
  • Wireframe - A bounding box in form of a wireframe cuboid will be displayed in place of the simplified entities.
  • Shading - A bounding box in form of a shaded cuboid will be displayed in place of the simplified entities.

Display mode

Display mode:

Determines the appearance of objects when being rotated using the mouse. Choose the required option from the drop-down list.

Standard

Standard

The option set in the drop-down list or check box present beneath will be applied to the standard assemblies and models. The think3 application considers an assembly as 'Standard' or 'Large' as per the settings made in the Large assembly area of the General category.

(current)

(current)

The think3 application considers an assembly as 'Standard' or 'Large' as per the settings made in the Large assembly area of the General category. Depending on the current status of the file, the phrase (current) is displayed either besides Standard or Large assembly.

Large assembly

Large assembly

The option set in the drop-down list or check box present beneath will be applied to the large assemblies. The think3 application considers an assembly as 'Standard' or 'Large' as per the settings made in the Large assembly area of the General category.

Max. rotation angle

Max. rotation angle:

Enables you to define the maximum angle through which an object can be rotated using the mouse. For example, if you move the mouse horizontally from the left-hand edge of the window to the right, the model will be rotated by the angle specified here.

Max. angular step

Max. angular step:

Enables you to define the maximum angular step applied to an object when rotated using the mouse. The higher the value, the faster the object is rotated. This value must be in the range from 1 to 45.
