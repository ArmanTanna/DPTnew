The controls in this area enable you to achieve complete control over the usage of virtual memory by the application.
In fact, you can activate or deactivate the display of a message informing you that the remaining free virtual memory reaches a threshold value that you can freely define.

When the display of the message is active and the corresponding warning is actually displayed, you can perform the appropriate actions so as to avoid crashes or standstills, in the following order.

  • Quit all the other applications that are running contemporarily
  • Close other documents that may be simultaneously open in the think3 application
  • Save the current document, stop the application and then start it again.
