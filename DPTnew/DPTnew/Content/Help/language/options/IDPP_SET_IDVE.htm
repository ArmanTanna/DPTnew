System Options

Category: Input-3D Input Devices

Type

Type:

This drop-down list determines the type of three-dimensional input device.

None no 3D input device enabled.
3Dconnexion Device devices of 3Dconnexion driven by the 3DxWare Driver. The devices supported are SpaceMouse, SpaceBall 4000, SpaceBall 5000, CADMan, Space Traveler, SpaceNavigator, SpacePilot, SpaceExplorer. See "System enhancements" for further details on Special Input Devices.

Options

Options

The controls in this area determine the performance options for the selected three-dimensional input device.

Threshold

Threshold:

The value, expressed in the device's own hardware units, below which any movement of the device is ignored. The proper value prevents accidental, unwanted movement of the device, from being registered.

End motion after

mS

mS

The time, in milliseconds, after which a lack of movement of the input device is registered as the completion of the action. When no movement of the input device is detected for the specified period, all graphic effects that were disabled during the action (such as the substitution of a wireframe for a shaded view during rotation) are restored and all actions of the device are applied.

Dominant axis

Dominant axis

When selected, restricts the action of the input device to the axis in which the most predominant action has occurred.
The movement of a three-dimensional input device can express six possible actions:

  • Panning horizontally and vertically in the view plane
  • Zooming in and out to enlarge or reduce the view
  • Rotating about the Rotation axes (specified on the Navigation-Advanced category of the System Options)
When this option is selected, rather than simultaneous action (such as panning and rotating) resulting in free movement, only the largest of the six possible actions is applied.

Exchange Y with Z

Exchange Y with Z

When selected, the Y and Z axes of the device will be swapped.

When not selected (default), the Y and Z axes in use will be the ordinary ones of the device.

Reverse zoom axis

Reverse zoom axis

When selected, the direction of the Z axis will be reversed. When using the device, you normally reduce the image by pulling the device and enlarge it by pushing the device. But in this checked mode this behavior will be reversed: you will reduce by pushing, enlarge by pulling.

When not selected (default), the zoom behavior will be the ordinary one of the device.

Sensitivity

Rotation

The controls in this area determine the Pan, Zoom and Rotation actions. You can enable or disable the action.
The action is enabled when its corresponding check box is selected.
You can also adjust the sensitivity (speed) of the action. Values less than 1 reduce sensitivity; values greater than 1 increase sensitivity.

Programming

Button

Description:

The controls in this area enable you to program the buttons of the three-dimensional input device (if the device has buttons), by associating the think3 application commands or procedural modules to each button.

To program a button, do the following:

1  From the Button drop-down list, choose the number corresponding to the button you want to program.

2  In the Content box, type the function content you want to associate with the button.
Or, to associate the button with an existing command:
Click the Select button and then choose the command (either a button or menu item). The command you select immediately appears in the Contenent box.

3  In the Description box, type a brief description of the command to remind you of its purpose.


By default, the program preassigns the following commands to the first three buttons of the selected three-dimensional input device:

Button 1 fit all views to show all entities (see Fit View).
Button 2 set current view back to previous state (see Previous View).
Button 3 set view parallel to Work Plane (see Work Plane View ).
