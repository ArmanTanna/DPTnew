Catalog UI folder

Catalog x-reference folder

The folder where the external reference components are cached.
Use the Browse button to select a different one.

View catalogs

View catalogs

The controls in this area enable you to define the catalogs visualization policy.

All

All

When selected, all catalogs are made available to the user.

Favorites

Favorites

When selected, only the catalogs marked as Favorite are made available to the user.

View parts

View parts

The controls in this area enable you to define the part visualization policy.

All versions

All versions

When selected, all revisions of the parts that were assigned a part number are displayed in the catalogs.

Released

Released

When selected, only the last version of the parts that were assigned a part number and are released are displayed in the catalogs.

Last inserted

Last inserted

When selected, only the latest available version of the parts is displayed in the catalogs.

Show codified parts only

Show codified parts only

This option enables you to control the visualization of parts in the Catalog Browser.

When checked, the dialog box that shows when you open a catalog lists only the items to which a part number has been assigned (codified parts), while the uncoded elements will remain hidden. Note that this dialog box has a checkbox with the same function.

Remove part number

Remove files generated with Import command

When checked, the Remove Part Number command will also remove the cached model file from the Catalog x-reference folder in order to insure that the next time the part is imported it will have the correct part data assigned.
