System Options

Category: Drawing Structure

Show entities

Show entities

The controls in this area enable you to define the settings and visualization of entities in the Drawing Structure.

Groups only

Groups only

When this option is selected, only group icons (both shared and non-shared) will be displayed in the Drawing Structure, while single entities either belonging to groups or not will not be displayed.

Entities inside groups

Entities inside groups

When this option is selected, icons of entities belonging to both shared and non-shared groups will be displayed in the Drawing Structure.
Double-click on group icon to view all the entities inside it.

All

All

When this option is selected, all entities inside each group and other entities outside the group will be displayed in the Drawing Structure.

The following are the icons of various entities (outside the group) that are displayed inside the Drawing Structure:

Icon Description
Cross-Hatch
Curve - Lines and arcs
Dimension
Text
Label

Order Groups

Order Groups

The controls in this area enable you to govern the groups according to the order of date/name in the Drawing Structure.

By date

By date

When this option is selected, the Drawing Structure will display the group icons ordered by creation date (the first groups in the list are the last created).
When not selected, the Drawing Structure will display the group icons alphabetically ordered by name.

By name

By name

When this option is selected, the Drawing Structure will display the group icons alphabetically ordered by name.
When not selected, the Drawing Structure will display the group icons ordered by creation date (the first groups in the list are the last created).
