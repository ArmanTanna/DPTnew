System Options

Category: Reference Systems-Appearance

Planes

Planes

The options in this area enable you to define the way the coordinate planes of the Work Plane Reference System are displayed.

None

None

If selected, no planes (including the Work Plane) will be visible on the Graphics Area.

XY plane

XY plane

If selected, only the XY-plane of the Work Plane Reference System will be visible on the Graphics Area. The XY-plane is the same as the Work Plane.

3 planes

Resize on editing

If selected, all the three planes of the Work Plane Reference System will be visible on the Graphics Area. The XY-plane is the same as the Work Plane.

Resize on editing

Resize on editing

While performing a quick editing of the Work Plane by using the Select command, you are enabled to resize the Work Plane.

  • If the Resize on editing check box is selected, green markers are displayed on the Work Plane corners that can be dragged to resize the Work Plane.
  • If the check box is cleared, no markers are displayed and hence, it is not possible to resize the Work Plane.

This check box is active only when the XY plane radio button is selected.

Axes

Axes

The options in this area control the display of the coordinate axes.

Z axis

Z axis
When selected, the program also displays the Z-axis of the Reference Systems.
When not selected, Z-axis of the Reference Systems is made hidden and only the X and Y axes are displayed.
Z axis selected Z axis NOT selected

For further details, see "Tips on the Work Plane appearance".

Axis thickness

Axis thickness
When selected, the coordinate axes are displayed with thick lines.
When not selected, coordinate axes are displayed with thin lines.
Axis thickness selected Axis thickness NOT selected

For further details, see "Tips on the Work Plane appearance".

Size

Size

The options in this area determine the size of the Reference System icons displayed on place.

Document units

Document units

When this radio button selected, the length of the displayed Reference System axes, in the current unit of measurement, is defined as per the value entered in the Size: edit box below.
When cleared, the length of the displayed Reference System axes is defined as per the settings made for the Millimeters on screen option.

Millimeters on screen

Millimeters on screen

This option controls the way the on place Reference System icons are displayed during dynamic zooming/panning/rotating as follows:

  • When NOT selected, the size of the Reference System icons will change proportionally to that of the scene and will be defined by the setting made for the Document units option.
  • When selected, the size of the Reference System icons will remain unchanged and will keep the value defined in the Size: slider below. Using the slider you can define the length (in millimeters with respect to the screen) of the displayed Reference System axes.

Size

Size:

Determines the length of the Reference System axes in the current unit of measurement. You can directly enter the value in the edit box or use the spinner to make increments/decrements of 1 unit at a time.
This edit box is active only when the Document units radio button is selected.

Size

Large

The size (in mm) corresponding to the slider position is displayed as a tool tip over the cursor. The Reference System axes will be displayed as per the length defined by the slider position and will remain unchanged during dynamic zooming/panning/rotating.
The slider is active only when the Millimeters on screen radio button is selected.
