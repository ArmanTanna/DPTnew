System Options

Category: Rendering

Rendering

Rendering

The options under this area enables you to define the parameters controlling the output of the rendered images.

Gamma factor

Gamma factor:

This determines the intensity of light. The value can be entered in the edit box.
This edit box is available only if the ThinkDesign or Separate window radio button is selected.

Output area

Render To

The options under this area control the parameters related to the mode of the output of the rendered images.

ThinkDesign

ThinkDesign

When this option is selected then the image is rendered in the application window.

Separate window

Separate window

When this option is selected then the image is rendered in a seperate window.

File

File

When this option is selected then the image rendered is saved in a specified location.
Click the browser button to open the Save dialog box. Then specify the File name and in the Save as type drop-down list select the format of the image to be saved.

Resolution

Resolution

The options under this area enables you to define the parameters controlling the width and height of the image.

Width

Width:

Specify the value in the edit box which determines the Width of the image

Height

Height:

Specify the value in the edit box which determines the height of the image
