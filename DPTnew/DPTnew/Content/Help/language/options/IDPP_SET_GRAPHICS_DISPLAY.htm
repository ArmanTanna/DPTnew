GDI Generic

Displays the Z-Buffer precision (16, 24 or 32): the "depth" value associated with every pixel, determining the color of the pixel. The higher the value, the higher the precision.

Self Tune

Automatically defines optimized settings for your machine. You are prompted a warning, if the installed Graphics card or drivers do not support all the high quality rendering functionalities.

More Details

More Details

Opens the graphics card document to read more information on the Graphics Card related configuration and troubleshooting.

Enable hardware acceleration

Enable hardware acceleration

When selected, it enables the acceleration for 3D graphics. This can be very useful in some situations, especially when the program can have problems due to the graphics card driver. When not selected, the acceleration 3D graphics is disabled.

Any time you change the status of this check box, we recommend that you stop the think3 application and start it again.

Force Copy swap

Force Copy swap

When this check box is selected, you force the software buffer exchange mode rather than the hardware one.
When the check box is not selected, the hardware mode is enabled.

Enable KTX extension

Enable KTX extension

When checked, the KTX extension of the Open GLTM libraries is activated. This will speed echo operations up. If the KTX extension exists on your machine, the box is checked by default. In some cases, if your screen turns to blue and the program stops responding, you can solve the problem by simply deselecting this box.

Show frame rate (fps)

Show frame rate (fps)

When checked, a box showing the frame rate at which the image is redrawn is displayed in the lower left corner of the current window.



When the box is not selected, the frame rate is not displayed.

Enable pbuffer extension

Enable pbuffer extension

When checked, a box showing the buffer rate at which the image is redrawn is displayed in the lower left corner of the current window.



When the box is not selected, the buffer rate is not displayed.

Enable arc/circle hardware representation

Enable arc/circle hardware rendering
  • When selcted, the hardware acceleration of arcs and circles is enabled if the system is capable of doing it.
  • When cleared, the hardware acceleration of arcs and circles is disabled.
