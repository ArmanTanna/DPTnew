System Options

Category: Assembly/Shared Group

Duplicated name resolution

Duplicated name resolution

The controls in this area enable you to define rules in the duplicated name conflicts occurring when you load models/drawings that include components/shared groups with the same name and different geometry.

See " Loading models/drawings including a component/shared group with the same name and different geometry" for further details.

Show renaming report

Show renaming report

When selected, before renaming or sharing a component/shared group, the Renaming Report dialog box, showing the information on the action, will be displayed.

When not selected, no dialog box will be displayed while renaming or sharing a component/shared group.

See " Loading models/drawings including a component/shared group with the same name and different geometry" for further details.

AutoRename settings

Automatic renaming

The controls in this area enable you to activate/deactivate the automatic renaming assistant and define the way the name is built in duplicated name conflicts. The automatic renaming assistant completes the name by adding a prefix or a suffix, made up at least of a number. Changing options while renaming the component/shared group will automatically update the name.

See also Renaming Report.

Enable

Enable

When selected, the automatic renaming assistant is activated.

When not selected, it is deactivated.

Mode:

Mode:

By selecting Prefix or Suffix in the drop-down list, a prefix or a suffix, composed of a text (optional) and a numeric part (compulsory), will be added to the name automatically proposed for the component/shared group.

Separator:

Separator:

In the name automatically proposed for the component/shared group, you can optionally insert between text, number and old name one of the following separators, available in the drop-down list:

None No separator.
Blank A blank character.
Underscore An underscore.

Text:

Text:

In the name automatically proposed for the component/shared group, you can optionally insert a text as prefix/suffix, associated to a numeric part, that is compulsory.

First number:

First number:

The first number that will be used to build the numeric part of the prefix/suffix of the name automatically proposed for the component/shared group. The program will use the first available number in ascending order.

Min. digits:

Min. digits:

Minimum number of digits of the number making up the prefix/suffix of the name automatically proposed for the component/shared group.

You can enter an integer number from 1 to 6.

AutoRename component/group

AutoRename component/group

The controls in this area enable you to define the component naming behavior.

Assign part number as name

Assign part number as name

When selected, the component name in the document structure is changed to reflect the part number you set in the component or group Part Properties.

When not selected, the component or group name remains unchanged.
