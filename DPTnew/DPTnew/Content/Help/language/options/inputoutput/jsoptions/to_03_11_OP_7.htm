When selected, the type of the cursor changes to the following:

Cursor when a selection is required
Cursor when a point input is required

In this case, the cursor size is determined by the Selection Trap value. As shown in the picture below, this affects the cursor both when the program is waiting for you to select an object and when the program is waiting for the input of a point.
