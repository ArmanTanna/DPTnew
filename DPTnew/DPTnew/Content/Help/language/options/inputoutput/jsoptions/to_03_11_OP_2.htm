This option enables you to sketch on the current XY plane. When this option is selected the points you specify using the mouse will lay on that plane, but those you snap to 3D points will really be 3D.

Use this method, for instance, when adding points - making up curves - to the face of a solid, after first moving the Work Plane to the face. You can then use the curve as a profile from which to create a feature, such as protrusion or shaped hole.
