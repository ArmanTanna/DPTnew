Show in part tooltip

Show in parts tooltip

The controls in this area enable you to define the information displayed in the tooltip that shows when you rest the mouse cursor on a component.

Name

Name

When checked, the component tooltip displays the component name.

Part number

Part number

When checked, the component tooltip displays the part number, if the Part Properties have been assigned to the component.

Description

Description

When checked, the component tooltip displays the part description, if the Part Properties have been assigned to the component.

Item position No.

Sequence number

When checked, the component tooltip displays the Sequence Number defined with the Item Position Property command.

Quantity

Quantity

When checked, the component tooltip displays the quantity defined with the Quantity Property command.

Show balloons tooltip

Show balloons tooltip

The controls in this area enable you to define the information displayed in the tooltip that shows when you rest the mouse cursor on a balloon.

Part number

Part number

When checked, the balloon tooltip displays the Part number.

Description

Description

When checked, the balloon tooltip displays the part description.

Item position No.

Sequence number

When checked, the balloon tooltip displays the component Sequence Number defined with the Item Position Property command.

Quantity

Quantity

When checked, the balloon tooltip displays the component quantity.

You are enabled to edit the component quantity by means of the Quantity Property command.
