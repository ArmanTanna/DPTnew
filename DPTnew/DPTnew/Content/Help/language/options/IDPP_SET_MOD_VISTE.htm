Settings

The controls in this area determine the behavior of some common actions on views.

Enable smooth transition

Enable smooth transition

When selected, zooming , and panning or rotating the model drawing causes a pleasant animated adjustment of the view. This behavior enables you to better understand the model orientation in space, because the object slowly moves under your eyes instead of jumping from the initial to the final positions.

When the check box is not selected, the view changes from its previous to its new status with no intermediate transition.

Previous views

Stored

Stored:

Determines the number of states that are to be stored for the current view so that they can be restored one step at a time using the Previous View command.

Step actions

Step actions

The controls in this area determine the increments used for various zooming , or panning or rotating functions.

Zoom factor

Zoom factor:

Enables you to define the scale factor increment by which the image is enlarged or reduced when using the following commands:

  • Zoom In
  • Zoom Out
  • Zoom on Point of Interest.

View rotate step

View rotate step:

Enables you to define the rotation increment by which the image is rotated when using the Rotate ... commands:

  • Rotate Counter-Clockwise about X
  • Rotate Counter-Clockwise about Y
  • Rotate Counter-Clockwise about Z
  • Rotate Clockwise about X
  • Rotate Clockwise about Y
  • Rotate Clockwise about Z.

View pan step

View pan step:

Enables you to define the translation increment by which the image is shifted when using the Pan ... commands.

  • Pan Down
  • Pan Left
  • Pan Right
  • Pan Up.

It also determines the translation increment by which the clipping plane is moved when using the Define Section View command.
