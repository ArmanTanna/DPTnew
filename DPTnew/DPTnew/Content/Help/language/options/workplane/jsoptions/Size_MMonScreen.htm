This option controls the way the on place Reference System icons are displayed during dynamic zooming/panning/rotating as follows:

  • When NOT selected, the size of the Reference System icons will change proportionally to that of the scene and will be defined by the setting made for the Document units option.
  • When selected, the size of the Reference System icons will remain unchanged and will keep the value defined in the Size: slider below. Using the slider you can define the length (in millimeters with respect to the screen) of the displayed Reference System axes.
