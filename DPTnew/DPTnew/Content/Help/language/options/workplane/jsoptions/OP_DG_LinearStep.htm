Enables you to define the value of the grid linear step (i.e.: the distance between two consecutive nodes of the grid).


Note
  • If the Flexible step check box is not selected, your mouse input points will always be snapped to the nearest node of the virtual grid.
  • If the Flexible step check box is selected, your mouse input points will be snapped to the nearest node of the grid only if the actual position you indicated is inside the Snap trap.

See Hidden Dynamic Grid for further details.
