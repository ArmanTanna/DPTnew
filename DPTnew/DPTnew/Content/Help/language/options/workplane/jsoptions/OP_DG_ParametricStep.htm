Enables you to define the value of the grid parametric step, used whenever parametric values not related to the current unit of measure are involved, such as curve and surface parameters (for example, when you extend a surface by modifying its parameters using the Parameter mode) and scale factors (for example, when you scale entities using the Scale Entities command).

The Parametric step enables you to limit your mouse input actions and the values which can be assigned to mini-dialog boxes to a set of predefined ones at fixed intervals.
