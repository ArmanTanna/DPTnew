You can precisely control the rotation of a model on screen by setting "targets". When you manually set any "targets" for viewing/rotating the model, using View Modify Set Target (or from the View menu in the context menu obtained by right-clicking on the empty space of graphic area), the target you specify is highlighted using this marker:

You can select the Show check box, to toggle the display of the marker in the graphics area.


Note

When you have not manually set any 'targets' for viewing the model, the program tries to find out a point of interest (at that particular instant of time, for that particular model) that can be the 'virtual target' in the graphics window. The program tries to understand and calculates this point behind the screens so that you get the best possible solution. Note that the automatic target is done by the program itself; you don't have to specify anything.
