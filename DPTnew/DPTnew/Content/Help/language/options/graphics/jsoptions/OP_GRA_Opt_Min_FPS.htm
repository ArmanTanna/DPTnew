Enables you to set the Framerate of model Drawing details during panning , zooming and rotations operations.

To set a specific framerate, select the Minimum frame rate (fps): option and enter the value of the framerate.
If you set a constant framerate, whenever you perform a dynamic operation, the program will try to achieve that framerate. This may be done by reducing the graphics details of the model on the fly, during a dynamic action and will be particularly useful while working on complex (heavy) models and assemblies. By setting a lower framerate, say 10, during a dynamic action the navigation of the model on the screen is made very smooth. After you have completed the dynamic action, (say after rotating) the model is again shown with full quality (full details).
