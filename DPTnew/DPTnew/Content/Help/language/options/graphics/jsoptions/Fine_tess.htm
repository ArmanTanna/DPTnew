When you move the slider to this tick, the maximum tessellation quality level is reached, corresponding to the following values:

  • Surface tolerance: 0.001
  • Boundary tolerance: 0.001
  • Surface Accuracy: 100%

As higher values yield better display and mass property calculations but lower performance, when setting this value you will have the best quality but, if the model is a big and complex one, the performance time might be higher.

To display the tessellation on all entities of the model, start the Tessellation command.
