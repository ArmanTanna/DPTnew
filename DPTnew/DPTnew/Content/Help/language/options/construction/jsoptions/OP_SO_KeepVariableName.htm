When inserting a Smart Object into a model, the Smart Object and the model might contain variables with the same names.

  • When this check box is selected, variables associated with exposed parameters are automatically imported into the Spreadsheet with their own names. Only mini-dialog boxes associated with variables not already existing in the Spreadsheet are editable.
  • When not selected, all variables associated with exposed parameters are automatically imported and renamed (a prefix is added, consisting of the letter N, a sequence number and the _ character) and any mini-dialog box is editable.

For further details, see also "Managing Duplicated Variables when Inserting Smart Objects".
