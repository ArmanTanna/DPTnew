This option affects the behavior of the following commands:

  • Offset on Plane
  • Fillet
  • Double Fillet

Offset on Plane

When this check box is selected and you select the Fillet option of the command, if the selected chain of curves has sharp corners and the offset requested is made in the area on the outside of them, the corners are filleted and the portions of curves that extend beyond the fillet are automatically eliminated.
When not selected, the fillets are not created.

Fillet

When this check box is selected and the Trim/Extend drop-down list in the command selection list is set to First or Both, the first or both the lines/curves extending beyond the fillet arc are trimmed.
When not selected, the original lines/curves are not trimmed.

Double Fillet

When this check box is selected and the Extension drop-down list under More Options in the command selection list is set to Trimmed, the portions of the selected surfaces extending beyond the new blend surface are automatically eliminated.
When not selected, the original surfaces are not trimmed.
