This value controls the result precision in several operations. , such as the following commands:

  • Curve Intersection
  • Intersect Solid
  • Solid Difference
  • Solid Union
  • Project Curve

You should set this value as a small fraction of the current unit of measure. Reducing the tolerance value increases the precision of calculations, but those calculations require more time to complete, and, when the result of the operation is a NURBS curve, such as in the case of surface intersection, the resulting curves may have a very high number of control points.
