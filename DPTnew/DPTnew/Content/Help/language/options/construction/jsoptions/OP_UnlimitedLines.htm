The actual length of a Semilimited or Unlimited line (see the 2 Point Line, Parallel Line and Bisector Line commands) can be set in the following ways:

View sizeThe resulting line will be extended up to the view box, that is the visible part of the document.
Document sizeThe resulting line will be extended up to the document box, that is the box that includes all of the entities in the document.
