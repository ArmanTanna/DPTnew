System Requirements

In the following table you will find the system characteristics required to install and use the products of the ThinkDesign Suite.

  • Minimum System Requirements
  • Suggested System Requirements
Minimum System Requirements
Processor Intel® 64 or equivalent AMD64
Operating System Windows® 10 (64-Bit)
System Memory (RAM) 4 GB
Hard Disk Space 1.5 GB for Typical installation
Graphics Card/VRAM 512 MB OpenGLTM 2.0 Graphics Accelerator
Additional Requirements Microsoft® .NET Framework Version 4.0 or higher
Suggested System Requirements
Processor Intel® 64 Xeon or equivalent AMD64
Operating System Windows® 10 (64-Bit)
System Memory (RAM) 8 GB
Hard Disk Space 1.5 GB for Typical installation
Graphics Card/VRAM 1 GB OpenGLTM 3.0 Graphics Accelerator
Additional Requirements Microsoft® .NET Framework Version 4.0 or higher
