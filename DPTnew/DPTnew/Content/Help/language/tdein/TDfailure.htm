Inability to run ThinkDesign after installing

It may sometimes happen that after completing ThikDesign installation and trying to launch it, nothing seems to happen. In such cases, a check through the Task Manager reveals that the ThinkDesign process actually starts but dies soon afterward.

The issue can be due to some missing updates in the operating system. To fix the problem, install the missing updates by connecting to the Windows® Update site and start ThinkDesign again.
