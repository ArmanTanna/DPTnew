Note

For proper installation on Windows® 7 operating system or higher, in case you do not have administrator privileges (Standard User), please run the setup.exe file and then insert the password of the Administrator to go on with the installation.

Before starting the installation process, please verify the following.

  • Make sure to have downloaded all the available updates from Windows® Update
  • Verify from the manufactor web side of your PC to have downloaded all the available updates about:
    • Bios CPU
    • Graphic Card Firmware
    • Graphic Card Driver
