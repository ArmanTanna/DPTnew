X-Camme

X-Camme is a Microsoft® Excel application for the kinematic and dynamic analysis of cam mechanisms and for cam cutting on numeric control machine tools developed by Studio Tecnico Turci.

Data entry is performed through a graphical interface, directly accessible under think3 applications.

Data and results are saved to a unique Excel document. Results are in the form of tables and graphs relative to displacement, speed, acceleration, jerk, pressure angle, radius of curvature, FFT spectrum, CAD system files and Numerical Control cutting machine tools.

X-Camme can be used in technical offices, in order to design the cam mechanism. The resulting model will be automatically transferred to the think3 application.

For further details, see www.turci.biz.
 

A trial version of X-Camme can be installed by choosing the corresponding X-Camme entry in the Products section of your think3 application installation procedure.

KISSsoft

KISSsoft AG develops calculation programs for both mechanical and design engineers in a wide range of fields where safety, reliability, and optimization are the deciding factors.

Making full use of all the latest standards (DIN, ISO, AGMA) KISSsoft software ensures quick and reliable calculation of design parameters, validation, and strength analysis for machine elements with documentation of safety factors and expected life.

KISSsoft is a calculation program consisting of a standard package and expert add-ons for the certification, design and optimization of machine elements.

Standard Packages have been developed to cope with various problems from many fields of application. The program has a modular construction so that only the appropriate amount of software need be acquired.

  • The Standard Gear Package allows the calculation of the geometry and the strength of cylindrical gearing.
  • The Standard Coupling-Element Package contains calculations for shaft-to-collar connections.
  • The Standard Spring Package encompasses calculation modules for the design of Pressure and Tension-springs.

The results of calculations can be transferred to the think3 application very easily thanks to capability of automatically creating the corresponding appropriate models.

For further details, see www.kisssoft.ch.
 

A trial version of KISSsoft can be installed by choosing the corresponding KISSsoft entry in the Products section of your think3 application installation procedure.
