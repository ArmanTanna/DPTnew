How to define a point position

The Type list enables you to define a point in a number of different ways:

Simple Point To select a point just by clicking on it. The selection list displays the Point item. When a point is already selected, you can change it by right-clicking on Point and selecting Reset.
From 2 Edges To select a point by entering its distances from two different edges. The selection list displays two Edge items. When edges are already selected, you can change one or both of them by right-clicking on the corresponding Edge item and selecting Reset.
Along an Edge To select a point entering its distance from the start point of an edge and the distance along the edge. The selection list displays the Edge item. When the edge is already selected, you can change it by right-clicking on Edge and selecting Reset.
Polar To define a point in a polar reference system, entering the polar Radius and the polar Angle. The origin point and the reference direction line have to be selected. As to the line selection, see How to define an axis or a line. The selection list displays the Origin Point and the Reference Line Direction items. When the origin point and the line are already selected, you can change them by right-clicking on the corresponding items and selecting Reset.
