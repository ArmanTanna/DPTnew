Error file - conversion to CATIA

When you save a CATIA file, an error file of the conversion is created:

  • in the same folder of the file you opened
  • with the same name of the file you opened and the ".err" extension or
  • in the path specified by setting the relevant parameters in the Log File category of the CATIA Conversion Options

The error file contains a report of the errors occurred during the conversion process. If no errors occurred, the file will be empty. Please note that the file is created even if the conversion process aborts, enabling you to read the report about the issue that caused the process to abort.
