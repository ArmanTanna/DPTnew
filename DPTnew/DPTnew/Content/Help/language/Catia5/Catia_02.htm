Log file - conversion from CATIA

When you open a CATIA file, a log file of the conversion is created:

  • in the same folder of the file you opened with the same name of the file you opened and the ".log" extension
  • in the folder and with the name you specified in the Log File category of the Options.

In the same folder of the log file, you find also another file with the ".err" extension. It contains the conversion errors if any.

The log file structure

The log file contains a report of the converted entities.

Example of log file - conversion from CATIA



-------------------- HOOPS Conversion Report --------------------

Converted file: bottom_plate.CATPart


Type  Name    Count
----------------------------------------

6     CPntSrf  7
46    CylSrf   1


Total Number of Entities Converted: 8

-------------------------------------------------------------
