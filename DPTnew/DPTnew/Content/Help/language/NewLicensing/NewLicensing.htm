New Licensing System Guide for ThinkDesign 2015 (and successive versions)

ATTENTION: For details on licenses up to ThinkDesign 2014.1, please read this document.

1. Introduction

With the release of ThinkDesign 2015.2, we introduced Sentinel, a new licensing system developed by the company Gemalto, which also resulted in the standardization of node-locked licenses, dongles and network/floating licenses.

think3 License Administrator (cladmin.exe), Machine ID and FlexNet licenses will not be used anymore.
In addition, ThinkDesign will not start in demo mode if no license is registered. In fact, after the splash screen, the program will quickly shut down, showing this error dialog box:



2. User Licenses

The first time you install ThinkDesign on your PC, the service Sentinel Run Time Environment will be installed as well. After running the program and the "Valid license not found" error message, you will be redirected to the "new License Administrator", which is called Sentinel Admin Control Center (ACC).
If the service is correctly installed, you should be able to see the following page.




  1. To ask for a ThinkDesign license, you need first to generate the .c2v (Customer-to-Vendor) file.

    a. Instructions to generate the .c2v file in case your license is tied to a single PC
    b. Instructions to generate the .c2v file after receiving the new dongle

    a. License tied to a single PC

    • If you downloaded ThinkDesign 2018 or a successive version, during the installation, a .c2v file is automatically created on your Desktop. The file will be named after your machine plus a code number, for example: XXXXXX_110137.c2v.

    • If you installed a prior version (2015-2017), proceed as follows.
      Run the executable file DPTGet.C2V.exe that is in the installation folder of ThinkDesign (ex: C:\program files\think3\thindesign 2017\common\utilities).
      The shell window will appear indicating the folder where the .c2v file has been saved, which usually is the Temp folder.

      If you have difficulty in finding the .c2v file, just follow these steps: click Start, select the Run command. A dialog box with a search bar will appear, type "%temp%" and then click "Ok".
      In this way, the Temp folder will open. Sort files by the last modified. The first file on the list will be the .c2v file you are looking for.

    b. New dongle (hardware key):

    After installing ThinkDesign, insert the dongle into the USB slot and the dongle should light up.
    In the Admin Control Center (ACC), select Sentinel Keys in the left-side menu. A table will appear.
    In the row where the hardware key has a Driverless Configuration, select the C2V button in the Actions column.



    Then click Download C2V File. If you didn't change your default download location, the file you've downloaded will be automatically saved in the Downloads folder.




  2. Send an email to pwdrelease@dptcorporate.com attaching the .c2v file you generated.

  3. You will receive an email (License Certificate) from DPT Licensing that provides you with a file named xxx.v2c.
    xxx is the license Key ID, while the V2C extension stands for Vendor-to-Customer.

  4. Attach the .v2c file to the Sentinel Admin Control Center (ACC).
    You can launch the ACC from ThinkDesign installation folder by running the Admin_Control_Center URL (C:\program files\think3\2019.1\common) or by typing http://localhost:1947 in your Internet browser.



    In the Update/Attach section, you can apply the .v2c file to activate the license.



    After this, you will see the license in the Products section of the ACC.

3. Updating the Product Key with a renewal license

In case of product updates or renewals of the maintenance contract, we will generate new licenses to be added on the same Product Key. Simply send an email to pwdrelease@dptcorporate.com and let us know which user needs a new password. We will send you another .v2c file.


4. Floating Licenses

The floating licenses are linked to a specific license server.


Server Installation

  1. Download the Floating Licensing Installer from the Download area of the DPT3Care website.




  2. To extract all files from the downloaded .iso file:
    a. In Windows 10, simply right-click on the .iso file and Open it with Windows File Explorer.
    b. In Windows 7, you can use the utility program 7zip (free download).

    Run the DPT_Licensing_Setup.exe you will find in the folder where all extracted files are.
    Follow the steps of the Setup Wizard and complete the installation.

  3. Insert the dongle into the USB slot and the dongle should light up.
    In the Admin Control Center (ACC), select Sentinel Keys in the left-side menu. A table will appear.
    In the row where the hardware key has a Driverless Configuration, select the C2V button in the Actions column.



    Then click Download C2V File. If you didn't change your default download location, the file you've downloaded will be automatically saved in the Downloads folder.




  4. Send an email to pwdrelease@dptcorporate.com attaching the .c2v file you generated.

  5. You will receive an email (License Certificate) from DPT Licensing that provides you with a file named xxx.v2c.
    xxx is the license Key ID, while the V2C extension stands for Vendor-to-Customer.

  6. Attach the .v2c file to the Sentinel Admin Control Center (ACC).
    You can launch the ACC from the installation folders by running the Admin_Control_Center URL or by typing http://localhost:1947 in your Internet browser.



    In the Update/Attach section, you can apply the .v2c file to activate the license.



    After this, you will see the license in the Products section of the ACC.


Client Installation

  1. Download ThinkDesign from the Download area of the DPT3Care website.




  2. To extract all files from the downloaded .iso file:
    a. In Windows 10, simply right-click on the .iso file and Open it with Windows File Explorer.
    b. In Windows 7, you can use the utility program 7zip (free download).

    Run the DPT_TD_Setup.exe you will find in the folder where all extracted files are.
    Follow the steps of the Setup Wizard and complete the installation.

  3. Now you need to set the system environment variable DPT_LIC_HOSTNAME.
    Control Panel ⇨ System ⇨ Advanced System Settings
    The System Properties window opens, select now the Advanced tab and click the Environment variables button.

    In the System variables section, click the New button.
    DPT_LIC_HOSTNAME corresponds to the Variable name and it should be set according to the following table.

    4. Variable value Description
    LOCALHOST ThinkDesign doesn't use floating licenses, only local licenses
    Server_Name ThinkDesign uses only floating licenses


    If you use on the same PC both local and floating licenses, you don't need to define the DPT_LIC_HOSTNAME variable.

    If you use a VPN connection, you just need to do as follows.
    In the ACC: go to Configuration ⇨ Access to Remote License Managers and write your server name in the Remote License Search Parameters field.



    If you use instead a local network, you just need to set DPT_LIC_HOSTNAME = server_name.
