Log file of the Parasolid to think3 conversion

When you open a Parasolid file, a log file of the conversion is created:

  • in the same folder of the file you opened with the same name of the file you opened and the ".log" extension
  • in the folder and with the name you specified in the Log File category of the Options.

The log file structure

The Parasolid to think3 conversion log file has a structure made of the following sections:

  • Reader options — a section containing the status of the current reading options.
  • Subpart analysis — a section repeated for each model of the assembly, reporting information on the model version and a summary of the translated entities.
  • TD Direct Conversion Report — a section summarizing the entities and files converted.

Warnings and errors help you identify any kind of problem you might have had while converting a Parasolid file.
You can find details about the errors that occurred, if any, in the error log file.
