Parasolid Conversion Options

When loading (see Open) or saving (see Save and Save As) a Parasolid file by selecting Parasolid file in the Files of type/Save as type drop-down list, the Options button in the file selection box enables you to define the parameters to be used in the conversion process.

A dialog box displaying the following categories shows up:

  • Load
    This category contains the most common parameters to be set when converting from Parasolid.
    • Log File
      This category contains parameters controlling the name and location of the log file created during the conversion process.
    • Advanced
      This category contains parameters controlling advanced conversion settings such as the conversion of entities into NURBS.
  • Save
    This category contains the most common parameters to be set when converting to Parasolid.
    • Log File
      This category contains parameters controlling the name and location of the log file created during the conversion process.
    • Advanced
      This category contains parameters controlling advanced conversion settings such as the conversion of entities into NURBS, tolerances and resolutions.
