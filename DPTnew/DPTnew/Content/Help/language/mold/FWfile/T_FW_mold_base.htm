The DME Mold Base command enables you to create the mold base according to the DME catalog. The DME wizard is customized with the terminologies and values related to the DME catalog. The pictorial representation of the different parts of the assembly enables you to make the selection faster and easier.

On starting the command, the Mold Base-DME wizard is displayed. The wizard contains four tabs named as Basic Size, Plates, Guide Elements, and Fasteners as shown in the image below.



Note
If you have not installed the catalog, the software prompts you to install it on starting the command.


In the Basic Size tab, you can select the Mold base series from the Series drop-down list. When the series is selected, the values corresponding to the series are automatically updated in the other tabs for selection. Also you can select the Mold Base to be WidthWise or Lenghtwise from the Oversize drop-down list. The image of the mold base changes according to the selection.
