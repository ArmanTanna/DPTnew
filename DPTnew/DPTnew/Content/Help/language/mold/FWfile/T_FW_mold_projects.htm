InsertMoldDesignProject

The Mold Project command enables you to perform the initial step of creating the project files required for mold design, by using the original mold part.

On starting the command the Mold Project dialog box is displayed as shown in the image below.



Note
The From document:, Save models into archive and Archive name: options are active only when thinkteam is installed.

The options under the Insert Part area enables you to browse the *.e3 file of the part for which the mold is to be designed. There are two radio buttons available:

  • From file:: The file can be browsed and selected from any local folder from your local machine.
  • From document:: The file can be browsed and selected from the document database of thinkteam.

The location of the created project files depends upon whether thinkteam is installed or not. Also the setting of options in the Mold Project dialog box defines the location for the project file creation. For detailed explanation refer to the tutorials.

The Save models into archive check box is used to optionally load the created files into the document archive of thinkteam.

  • When this check box is selected, you can add all the created files into the document archive of thinkteam.
  • When not selected, the project files are created in the parent folder of the selected mold part. Also, the From document: and the Archive name: options will not be active for selection.

The Archive name: text box is used to name the folder which is to be created in the document archive.

The Shrink part area: enables you to specify the scaling factor. The scaling is given to negate the shrinkage involved in the molding. The following two radio buttons are available under the Shrink part area.

  • Isotropic scale: When selected, you can enter the scale value in the corresponding text boxes. The mold part will be scaled in isotropic mode in X, Y, and Z directions.
  • Anisotropic scale: When selected, the three directional axis text boxes are available. You can enter the specific scale factors in the respective Scale X, Scale Y, and Scale Z text boxes.
Select OK to apply the command. Two new files are created as per the settings done; one is a derived component (which is also a scaled component) and another is an assembly.

Note
  • It is mandatory to select a single solid model and not a component.
  • When a part with multiple solid is selected, then the command prompts you to select the particular solid for which mold is to be created.
