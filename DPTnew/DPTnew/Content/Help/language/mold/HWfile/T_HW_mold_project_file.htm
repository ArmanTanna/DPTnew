Creating the Project Files in the Local Folder

In the following tutorial you will create the project files for mold designing using the Mold Project command. This is a situation where think team is not installed in the application. Follow the steps below.

  1. Start the command to display the Mold Project dialog box.

  2. Click on the browse button besides the From file: text box to select the *.e3 file of the mold part from the local folder of your machine.

  3. Enter the scaling factor in the Scale factor: text box.

    Note
    The other options are not active as thinkteam is not installed. So they are not available for selection.

  4. Click OK to apply the command. Two new files are created in the parent folder of the selected mold part. For example if the name of the selected mold part is abc.e3, then the two new files are named as abcderived.e3 and abc_MoldAssembly.e3.
