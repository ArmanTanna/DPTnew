Creating the Project Files with thinkteam Installed

In the following tutorial you will create the project files for mold design using the Mold Project command. This is a situation where thinkteam is installed in the application. There are two ways to create the files:

  • Create the Project Files in the Document Archive of thinkteam.
  • Create the Project Files using the thinkteam Database
Start the command to display the Mold Project dialog box and then follow the steps below.

Create the Project Files in the Document Archive of thinkteam.

  1. Click on the browse button besides the From file: text box to select the *.e3 file of the mold part from your system.

  2. Select the Save models into archive check box. Then enter the name of the archive folder in the Archive name: text box.

  3. Enter the scaling factor in the Scale factor: text box.

  4. Click OK to apply the command. Three new files are created in the document archive of thinkteam. Consider that the name of the file selected is abc.e3, and the files that are created are named as abcderived.e3 and abc_MoldAssembly.e3. Also the selected file is also loaded into the document archive. To view the document archive selects thinkteamDocument Browser from the menu bar.

    For example if thinkteam is installed in the C: drive then the default location of the document archive is C:\Users\<user name>\AppData\Roaming\\thinkteam\<Version Number>/Projects.

Create the Project Files using the thinkteam Database

  1. Select the From document:: radio button and click on the button beside the From document text box to display the thinkteam Query View dialog box.

  2. Select the model for the mold part from the database and click OK to confirm the selection.

  3. Enter the name of the archive folder in the Archive name: text box.

  4. Enter the scaling factor in the Scale factor: text box.

  5. Click OK to apply the command. Two new files are created in the document archive of thinkteam in the specified archive folder.
