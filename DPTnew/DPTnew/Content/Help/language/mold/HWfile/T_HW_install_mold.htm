Mold Base Wizard

The Mold Base wizard is a higlly interactive and pictorial representation of the differenent Mold Base supplier catalogs. The following is the detailed description of the Mold Base wizard and the options available for selection.

The Mold Base wizard usally contains four to five tabs for the selection. The Left pan list all the selections made in the wizard. This lists the total configuration of the selected values of the mold base assembly.
