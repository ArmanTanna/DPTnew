Modifying the Mold Base

The mold base assembly mainly contains the guide elements and plates. The guide elements are linked with the thinkparts catalog whereas the plates are unlinked.

Note
Linked standard parts cannot be redefined. Only unlinked parts can be redefined.

There are several operations which can be performed on the mold base. You can replace the guide elements and modify the plates as per your requirement. The following operations can be performed on the mold base.

  • Replacing the standard parts
  • Modifying the thickness of plates
  • Deleting the plates
  • Moving the Mold Base

Replacing the standard parts

Follow the steps to replace the standard parts such as guide elements.

  • Right-click on the standard part in the Graphics Area or on the corresponding part name in the Model Structure and select Replace Standard Part option from the context menu.
  • The thinkparts catalog dialog box opens with the list of standard parts with various parametric specifications. Select the required part and click OK.

    Note
    Select the Show the codified parts only check box. This saves time and lists the parts related to the selected standard part.

  • The part will be automatically updated in the assembly.

Modifying the thickness of Plates

You can modify the thickness of any plate in the mold base assembly by the following steps.

  • Right-click on the plate and select Set Current from the context menu.
  • Open the Spreadsheet dialog box; specify the values in the corresponding text box.
  • Click on the Rebuild button and the plate is updated to new specification.

Deleting the Plates

There may be situations when some of the plates need to be deleted from the mold base. Follow the steps to delete a plate:

  • Right-click on the plate or on the corresponding plate name in the Model Structure and select Delete from the context menu.
  • Open the spreadsheet and make the thickness of the deleted plate as zero.
  • Click the Rebuild button to get the updated assembly.

Moving the Mold Base

The mold base can be moved to a new position in the Graphics Area by moving the work plane coordinate system. On rebuilding the assembly moves to the new position.

Note
The Work Plane coordinate system is taken as reference to create the mold base.

Also the whole mold base can be deleted (variables in the spreadsheed should also be deleted) and reconfigured to a new specification.
