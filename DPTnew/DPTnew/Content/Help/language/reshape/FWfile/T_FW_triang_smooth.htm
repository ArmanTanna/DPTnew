ReshapeEditSmooth Mesh

The Smooth Mesh command enables you to reorganize the triangles in a more logical and correct way.

  1. Select the mesh you would like to smooth.

    To reset the mesh selection, right-click on Mesh in the selection list, choose the Reset option in the context menu and then select the mesh again.

  2. If you want to use only a section of the mesh, select the Selection Only check box and subsequently select the mesh triangles to be smoothed. You can also make a Window selection.

    Select the Selection through check box to select all the triangles in the selection Window area. Otherwise, by default only the visible triangles are selected.

  3. Make the additional settings in the Smooth Mesh selection list:



    Weight Non-dimensional parameter defining the degree of the influence of adjacent points on the current point smoothing in a single iteration, in the range of 0 to 1
    No. of Iterations Number of smoothing iterations
    Preserve Boundaries Check this box to fix the boundary edges while smoothing
    Preserve Volume This option allows to smooth the mesh without deforming the edgy areas or erasing details otherwise modified by the standard smoothing method. It is mainly used in prismatic models.
    Smoothing Tolerance Constrain smoothing vertex shift within a given tolerance value. Selecting this option displays a further field where to insert the tolerance value.
    Clean Mesh In case of unclean mesh, it improves the mesh topology after the smoothing. This would lead to closing of some holes created by the Remove Wrong Faces command. It is advised not to select this check box for a clean mesh.

  4. Click the OK button to confirm the operation, or the Cancel button to abort.
