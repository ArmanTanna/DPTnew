Insert Surface Specialized AutoPatching

The AutoPatching command enables you to fit a triangle mesh by a stitched set of untrimmed NURB surfaces. All surfaces have the same knot vectors. Adjacent surfaces also satisfy the G1 or G2 continuity condition.

  1. Select the mesh you would like to fit with surfaces.



  2. In the AutoPatching selection list, make the following settings:
    No. of surfaces Use this drop-down list to define whether the number of surfaces to be created is be defined automatically or manually.
    Model Geometry Complexity This option available when the Automatic mode is selected. You can drive the autodetection algorithm as to create more or less surfaces by using the slider.
    No. of surfaces This edit box is available when the Manual mode is selected. You need to enter the approximate number of surfaces to be generated.
    Fitting grid step This check box should be cleared for free form models; should be selected for models with planar faces. The value specified here is used by the algorithm for finding planar faces. (Higher values will give better surface fitting, but it will be more time consuming process).

  3. Under the More Options you can make additional settings to define the surface accuracy and smoothness.



  4. The triangle mesh over which the surfaces is to be created The surfaces created with the default settings
