Reshape

Import Mesh Enables you to import 3D model data from .stl, .rve, .obj and .ply files into the think3's environment.
Export Mesh Enables you to export the selected mesh in one of the supported formats (.stl, .obj, .ply, .rve).
Cut Mesh Enables you to cut or divide a mesh selecting the triangles directly on the mesh.
Split Mesh with Plane Enables you to split the mesh with a plane, which can be defined in an interactive way.
Split Mesh with Curve Enables you to cut the mesh about a chain of curves forming a closed loop.
Smooth Mesh Enables you to reorganize the triangles in a more logical and correct way.
Decimate Mesh Enables you to reduce the number of triangles in the mesh without any significant changes in the object shape.
Refine Mesh Enables you to increase the number of vertices and triangles in the mesh making the surface smoother. This operation subdivides the mesh triangles producing four triangles for every original triangle.
Sew Mesh Enables you to merge the mesh nodes that are closer than a specified tolerance value.
Remove Wrong Faces Enables you to heal the mesh removing abnormal faces, auto-intersecting faces, irregular faces and singularities. The goal of the procedure is to achieve a topologically correct model.
Heal Mesh Normals Enables you to correct the direction and the orientation of the normals to the mesh vertices on the basis of the mesh topology.
Fill Mesh Holes Enables you to heal the holes that a model may have when derived from point clouds obtained by scanning. These empty areas are usually due to markers used for shell registration, high-error zones and other reasons.
Curve on Mesh Enables you to create curves connected with the triangle mesh.
Mesh Boundaries Enables you to extract the bounds of a mesh in form of polyline (NURBS with degree 1) that are created at the boundaries.
Mesh Character Lines Enables you to determine the basic topological structure of the model.
Plane on Mesh Enables you to create reference planes, unconstrained planes characterized by its normal direction and root point.
AutoPatching Enables you to fit a triangle mesh by a stitched set of untrimmed NURBS surfaces in a completely automatic way. All surfaces have the same knot vectors.
