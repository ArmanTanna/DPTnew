Animation Incompatibility with Capture & Playback

The Animation and Capture & Playback functionalities cannot be run simultaneously. If a capture session is active and at the same time the Animation command is invoked, a warning message appears with the fact that the animation cannot work in this situation.

To prevent the accidental usage of the Animation dialog box, it does not show up in the new document that opens either after starting the capture or the playback sessions, while using the Capture & Playback tool. Otherwise, the Animation dialog box follows standard rules for dialog boxes. Hence, if a document is closed with the Animation dialog box active, when a new document is opened, the dialog box appears automatically to show all the animation information for the new document.
