Animation Functionality

The settings for the various animation functionalities are managed by the use of following buttons:

Evolutions : Enables the user to create/modify evolutions and specify the variation parameters.
Animation settings : Enables the user to specify the duration of the animation and also the number of steps to be generated per second.
AVI settings : Enables the user to make the settings for creating an AVI file of the animation. These include specifying the frame rate and the location to save the AVI file.
Collision Detection Settings : Enables the user to make the settings for collision detection. These include specifying the feedback mode when the collision is detected and also making selection of specific components and/or solids among which the collision detection will work.
