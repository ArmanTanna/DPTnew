Tools Animation

The Animation command is used to create the animations. It can be used to animate assemblies, check for interference between components when they are in motion, validate the kinematics of the model etc.

It is possible to create AVI files of the animation that can evidently show how the machine works and how the assembly should be assembled/disassembled.

Collision detection functionality is an integral part of the Animation command, which can be effectively used to verify the free movement of the components in an assembly. It is also possible to create a detailed report of collisions taking place during the animation playback.

On starting the Animation command, the Animation dialog box is displayed. This dialog box contains all the tools required to create the animation. The Animation dialog box has an intuitive graphics user interface providing full control of time line and parallel modifications for high productivity. Please refer to the Animation User Interface document to understand the structure and functions of the Animation dialog box items.
