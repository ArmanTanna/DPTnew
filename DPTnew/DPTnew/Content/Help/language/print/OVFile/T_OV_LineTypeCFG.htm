The Line Type Width Configuration File

The think3 application enables you to define a configuration file that defines the line width of each of the 16 line types defined. If you do, you will be able to use the Line Type width option available in the Print Setup command to control the printing width of your elements.

You can customize the width of the line types by editing the "linetype.cfg" file in the "files/printers" subfolder of your installation folder. "linetype.cfg" is a text file you can edit using any non formatting text editor program such as Notepad.

An Example of Line Type Width Configuration File


Make a copy of this file
It is highly recommended to make a copy of this file before editing it. You might need to restore the default values in case you should input some incorrect ones.

The file contains two columns of numbers:

  • Column 1: the line type identification number. Note that the think3 application provides you with 16 line types, numbered 1 through 16.
    If you look at the figure on the left, you will find a one to one correspondence with the file content, the top solid line being line type 1, the dashed line just below being line type 2, the dotted line below this being line type 3 and so on.
  • Column 2: the line width. In the example above, note that the solid line type (1) has been assigned a 3.5mm line width; the dashed line has been assigned a 0.2mm line width and so on. The default file that comes with the standard the think3 application installation has all widths set to 0.0mm; depending on the printer driver characteristics, this normally produces a 1 pixel line width.
