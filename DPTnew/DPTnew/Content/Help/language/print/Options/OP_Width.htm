Enables you to manage the width (line weights) of printed lines:

  • None — no width management is enabled; all lines will have the same default weight.
  • Colors —  the program prints the different widths according to the current color settings that you can change/define by clicking and editing the parameters in the Colors and Materials dialog boxes displayed (see also Printing Width for the model environment and Setting entity widths for the drawing environment).
  • Entities — the program uses the Width attribute to determine the weight of the lines. You can set the actual width with the Entity Width dialog box displayed when you click .



    In this dialog box, type the width values and click OK.
  • Line Type — when this printing option is selected, the width of printed lines depends on the values specified in the linetype.cfg  configuration file where you can assign a width to the program line types.
