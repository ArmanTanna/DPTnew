When selected, this checkbox enables you to use the same filename for multiple prints. A suffix is appended to the filename specified in Print to file, that is automatically incremented every time you Print a document.
For example, if you specify as output file AutonumberTest of type gif, and check AutoNumber, in the Name field the file changes from path\AutonumberTest.gif to path\AutonumberTest_001.gif. This is the filename used for the first print; if you subsequently print the same document again or a different document, they will be printed to path\AutonumberTest_002.gif, path\AutonumberTest_003.gif, and so on.
