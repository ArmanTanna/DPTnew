Use this command to specify details of the output format when available. The Print to File Details dialog box will be displayed. Select the desired option.



TIFF choose the type of compression: Uncompressed, Macintosh Packbits Encoding or LZW
PPM choose the type of output: Uncompressed (ASCII) or Uncompressed (BINARY)
GIF choose the type of output: background opaque (Interlaced) or transparent ( Transparent background)
JPG choose the type of output: JPG format. You can control the quality of the image by moving the slider.
