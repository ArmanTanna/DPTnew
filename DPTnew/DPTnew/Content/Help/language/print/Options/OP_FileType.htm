Choose the type of file from the drop-down list:

.bmp BMP format (Windows Bitmap); this is the default format
.gif GIF format (CompuServe Graphics Interchange)
.ppm PBM format (Portable Bitmap)
.tif TIFF format (Tagged Image File)
.jpg JPEG format (Joint Photographic Expert Group - SA/CCITT)
.png PNG format (Portable Network Graphics)

For some formats, the Details command is enabled.
