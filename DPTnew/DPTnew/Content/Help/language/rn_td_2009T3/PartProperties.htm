New Part Properties Functionalities and Improvements

Version 2009.3 introduces the following enhancements and improvements to the Part Properties functionalities.

  • Improvement in the Part List customization
  • User interface enhancement in the Save as Drawing Frame command

Improvement in the Part List customization

The following significant improvements have been made in the Part List customization to offer the capability of adapting to different needs.

  • New sub-column functionality
  • Column title in vertical direction
  • Customization of text style and color
  • Configuration of anchor points on a Drawing Frame

New sub-column functionality

You can now have more than one sub-column under a column header (point 1, as shown in the image above). To implement this enhancement, the new Merge titles text box has been introduced in the New Column and Edit Column dialog boxes. These dialog boxes are available on selecting the New Column or Edit Column buttons respectively from the Table Format - Columns category in the Part List Options dialog box.

The columns with the same content in the Merge titles text box will be collected together as merged column, having a common title and individual column titles.

The following steps explain the functioning of this feature:

  1. Activate the Part List Options dialog box.
  2. Expand the Table Format node and select the Columns category.
  3. Create two new columns, Wt and Vol, using the New Column button. While creating the new columns, enter Measure in the Merge titles text box and select OK.
    Note that while editing or adding a new column, the OK button is enabled only if requisite data is specified in the Title: or Name: text boxes.
  4. From the Table Format - Columns category, arrange the Wt and Vol columns in sequence under the Used list by using the Up or Down buttons.
  5. Insert the part list by using the ToolsPart List command. The Part List, as shown in the image above, will be inserted in the drawing.

Column title in vertical direction

It is now possible to have column title text in vertical direction (point 2, as shown in the image above). To implement this enhancement, the Effect button has been introduced in the new Column title tab under the Text font area in the New Column and Edit Column dialog boxes.

Customization of text style and color

In this version, it is also possible to perform text font customization for the row items and the column titles, along with the individual columns, as shown in the image above.

To implement this enhancement, the new Rows and Column title tabs have been introduced in the Text font area of the New Column and Edit Column dialog boxes. The following new button is present in the Column title tab:

Effect Enables you to insert the column title text in the vertical direction.
Note that this is a toggle button, which is active only when the Use table text style check box is cleared.

To implement the color management of title, header, and row items, the new Table color area with the Header/footer and column title and Rows options have been introduced in the Table Format - Advanced category in the Part List Options dialog box.

Configuration of anchor points on a Drawing Frame

It is now possible to configure an anchor point on a Drawing Frame to position a Part List. To implement this enhancement, the new Part List Anchor Point node with the following four options has been introduced in the new Save as Drawing Frame selection list.

  • Select top left anchor point
  • Select top right anchor point
  • Select bottom left anchor point
  • Select bottom right anchor point

These options enable you to specify an anchor point at respective corner positions.

On redefining a Drawing Frame with a Part List placed on an anchor point (using the Drawing Frame and Title Block command), the Part List will retain its attachment to the anchor point with the new Drawing Frame too.


User interface enhancement in the Save as Drawing Frame command

In this version, the old Save as Drawing Frame dialog box has been replaced with the Save as Drawing Frame selection list. The selection list includes all the options available in the old dialog box, along with the new Part List Anchor Point node.
