Enhancements in the Options/Properties Dialog Box

Version 2009.3 introduces the following enhancements in the Options/Properties dialog box.

  • Enhancements in System Options
    • Enhancement in the Clipboard/DragDrop category

  • Enhancements in Document Properties
    • Enhancement in the Drawing - General category

  • Enhancements in Entity Properties
    • Enhancement in the Sheet Metal - Positioning category
    • Enhancement in the Text - Font & Frame category
    • Enhancement in the Cutting-plane Line category
    • Enhancement in the Drawing View - Miscellaneous category

  • Enhancements in Edit Properties
    • Enhancement in the Drawing View category

Enhancements in System Options

Enhancement in the Clipboard/DragDrop category

The Include background check box has been introduced in the System Options - Clipboard/DragDrop category.

  • When selected, it enables you to select the background along with the entities from the Graphics Area and paste it in other supported applications.
  • When cleared, only the selected entitles will be copied, without the background.

Enhancements in Document Properties

Enhancement in the Drawing - General category

In the Next view/section name area, the Show separator check box has been introduced to support the improvement in the detail view notation.

  • When selected, all newly created views will display the separator symbol (—) in between the view name and its scale factor value.
  • When cleared, the separator symbol will not be displayed.

The new Show separator check box is enabled only when the Show view name and Show view scale check boxes are selected.

Enhancements in Entity Properties

Enhancement in the Sheet Metal - Positioning category

The Inner tangent and Outer tangent are two new options available in the Length: drop-down list under the Flange area of the Positioning category. These options are used to define the flange length.


Enhancement in the Text - Font & Frame category

The Vertical button in the Font area of the Entity Properties - Text - Font & Frame category has been enabled to support the improvements made in the single line text alignment.


Enhancement in the Cutting-plane Line category

The new Only thickened portions option has been added to the Style: drop-down list under the Cutting-plane line area in the Entity Properties - Cutting-plane Line category to support the new representation of section line. It enables you to visualize a section line only with thickened portions without cutting the main view.


Enhancement in the Drawing View - Miscellaneous category

The Add to view name check box has been introduced under the Developed view area in the Entity Properties - Drawing View - Miscellaneous category to support the enhancement in the developed section view notation.

  • When selected, you can attach the value of rotation angle and the corresponding rotation symbol.
  • When cleared, only view name and scale factor are displayed (only if the corresponding check boxes are selected).

Enhancements in Edit Properties

Enhancement in the Drawing View category

In the Drawing View area, the Show separator check box has been introduced to support the improvement in the detail view notation.

  • When selected, the separator symbol (—) will be displayed in between the view name and its scale factor value.
  • When cleared, the separator symbol will not be displayed.

The new Show separator check box is enabled only when the Show view name and Show view scale check boxes are selected.
