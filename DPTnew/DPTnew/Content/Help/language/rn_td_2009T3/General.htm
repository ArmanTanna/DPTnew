General Improvements

Version 2009.3 introduces the following general utility improvements.

  • Enhancement in the Show Dimensions/Features/Components command of Spreadsheet
  • Enhancement to Mold Base catalogs
  • Improvement in creating the reference plane by using Coefficients

Enhancement in the Show Dimensions/Features/Components command of Spreadsheet

In this version, the Show Dimensions/Features/Components command of Spreadsheet enables you to highlight not only the feature that has directly assigned variables in the expression but also the feature which has a dependent variable in the assigned expression.


Enhancement to Mold Base catalogs

A Mold Base catalog called Pedrotti has been introduced in this version. The usage of this catalog is similar to the other existing mold catalogs.


Improvement in creating the reference plane by using Coefficients

In this version, the Coefficients method for creating a reference plane has been renamed to Coefficients and through point method. A plane can be created by specifying an orientation point and A, B, & C coefficient values. Since the coefficient D value was redundant, the Coeff.D mini-dialog box has been removed from the Graphics Area. The plane definition by using Coefficients and through point is parametric in nature.
