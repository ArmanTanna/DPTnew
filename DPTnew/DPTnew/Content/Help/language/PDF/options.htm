PDF Conversion Options

Upon the installation of the ThinkDesign printer, when saving (see Save and Save As) a PDF file by selecting PDF file in the Files of type/Save as type drop-down list, the Options button in the file selection box enables you to define the parameters to be used in the conversion process.

A dialog box displaying the following categories shows up:

  • Save
    This category contains the most common parameters to be set when converting into PDF.
  • Log file
    When you save a PDF file, a log file of the conversion is created in the same directory of the file you saved. This category contains the parameters controlling the log file creation.
  • Advanced
    This category contains parameters controlling advanced conversion settings.
  • Visual Bookmark
    This category contains parameters controlling the creation of Visual Bookmarks.
