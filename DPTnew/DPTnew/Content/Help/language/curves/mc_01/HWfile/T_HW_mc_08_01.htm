Creating a new curve by indicating its control points

After starting the InsertCurveControl Points command or selecting the New curve option in the Mode drop-down list after starting the Modify Curve through Control Points command, you can proceed as follows:

  1. Specify the control points one by one (by clicking in the Graphics Area or by using Snap).



    The curve is dynamically displayed while you specify the control points.



    All the edit operations, such as select and move for existing points are available during the creation process.

  2. Double-click to create the curve.
