Analyzing the curvature of a curve

In the following commands:

  • Curve through Control Points
  • Curve through Interpolation Points
  • Curve Shape Constraint

Using the Curvature command you can analyze the curvature of the selected curves. It displays a dynamic plot of the curvature or of the radius of curvature for the selected curves, in which you can change both the Scale and the Density.
 

Curve through Control Points Curve through Interpolation Points Curve Shape Constraint

You can change:

  • the Density value
     
    Curve through Control Points Curve through Interpolation Points Curve Shape Constraint

  • the Scale. If the Scale check box is not selected, the scale of the plot is computed automatically so as to fit the current view. On the contrary, if the check box is selected, the specified scale factor is applied.


     
    Curve through Control Points Curve through Interpolation Points Curve Shape Constraint

If you select the Radius of curvature check box, the plot of the radius of curvature is displayed rather than the one of curvature.

You can project the curvature plot onto the view plane or onto another plane of your choice. If you select the Project check box, the Onto drop-down list will be displayed. The available choices are the following:

View The curvature plot will be created on the view plane. Please note that whenever you change the view orientation, the curvature plot will be updated accordingly.
Views The curvature plot will be created on the view plane of each of the existing views, if more than one.
Plane The curvature plot will be created on a plane to be selected.
The Reference Plane drop-down list enables you to select the method to be used to define the plane. For details on the plane definition method, see " How to define a plane".

A Reverse check box is also available, enabling you to interactively change the orientation of the radius of curvature, thus reverting the curvature plot, so that you can get a better visualization in case you should obtain auto-intersections.

  • When the check box is not selected, the sign of the radius of curvature is not reverted.
  • When the check box is selected, the sign of the radius of curvature is reverted.
