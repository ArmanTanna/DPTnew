Modify Curve Convert to NURBS

The Convert to NURBS command enables you to convert one or more curves into NURBS entities.

After selecting the curves to be modified (the Curves item in the selection list will turn to green), three different conversion methods can be selected in the Fix drop-down list: Tolerance, Parameters, Advanced.

  • Tolerance
    Performs conversion after enabling you to set the tolerance value you want to reach and after specifying the maximum number of control points for the resulting curve in the Max. control points box.
    Unless you require a Specialized conversion by selecting the Specialized check box under More Options, a cubic NURBS curve will be created.
     
  • Parameters
    Performs conversion after enabling you to set the parameter values of the NURBS you want to obtain:
    No. of points to fit Enables you to define the number of points to take into account in the computation for the conversion process.
    Continuity Enables you to define the order of continuity of the resulting curve.
    Arcs Enables you to define the number of arcs of the resulting curve.
    Degree Enables you to define the degree of the resulting curve.
    Using this method you can create NURBS of higher degree, but you will have to try by setting different values until you obtain the desired Achieved Precision.
     
  • Advanced
    Performs conversion after enabling you to set the Boundary continuity and the Continuity for the internal points of the resulting curve. This method will even enable you to force the creation of high knot continuity NURBS (internal knots with multiplicity 1. As multiplicity for a NURBS is the difference between its degree and the value of its continuity, the continuity value for NURBS with multiplicity 1 is maximum). See "Using the Advanced conversion options" for further details.

The Show Control Points check box enables you to display a preview of the resulting curve control points at any time when the command is active.

  • When the box is checked, the control points are displayed.
  • When the box is cleared, the control points are not displayed.

The Show Arc Joints enables you to display a preview of the arc joints that you will obtain on the resulting NURBS entity.

  • When the box is checked, the arc joints are displayed.


     
  • When the box is cleared, the arc joints are not displayed.

The Associative Mode check box enables you to create curves which retain a link to the surfaces/curves they derive from.

When the check box is selected, the resulting curves will be associative. The new curve will retain a link to the entities it derives from. A specific node displaying both the original entities and the resulting associative curve will be added to the Model Structure.
When the check box is cleared, the resulting curves will not be associative.

For a description of the additional conversion options available under More Options, see "Using additional conversion options (More Options)".
