Project a curve onto a surface along a pierce direction

1 Open the file "cv010215.e3". The following is displayed:

2 Start the Project Curve command.

3 Select the curve to be projected.



End selection by right-clicking and choosing Continue in the context menu.
To change the selected curve, right-click on Curves, select Reset in the context menu and perform selection again.
4 In the Method drop-down list select Pierce direction.
5 In the On drop-down list select Plane.
6 In the Plane drop-down list, select Plane. Click on the plane onto which the selected curve is to be projected.



To change the selected plane, right-click on Plane, choose Reset in the context menu and select another plane.
7 In the Direction drop-down list, select the method to be used to define the projection direction:
Line To define the projection direction as the one of an existing line to be selected.
2 Points To define the projection direction as the one identified by the two points to be selected.
X/Y/Z To define the projection direction as the one of a coordinate axis.

Choose Line and select the edge shown in the following illustration:



A preview of the resulting projected curve is immediately displayed:



To change the selected edge, right-click on Direction, choose Reset in the context menu and select another line.
8 Click or to confirm your selections and create the curve.



Click to discard your selections.
