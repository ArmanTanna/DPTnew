Using additional conversion options (More Options)

Depending on the method you have selected in the Fix drop-down list, different options will be available under the More Options node of the Convert Curve to NURBS command (no additional options are available if the Parameters method is selected; in this case the More Options node is not displayed).

Fix set to Tolerance Fix set to Advanced

Specialized Enables you to control the type of the resulting NURBS curves.
  • When this box is selected, curves are converted to specialized NURBS. Specialized NURBS are much more accurate and have a lower number of control points. See "Specialized conversion" for details.
  • When the box is cleared, curves are converted to non-specialized NURBS.
     
Keep initial degree
 		 This check box is active only when the Continuity is set to Maximum.
  • When selected, the resulting curve will have the same degree as the original one.
  • When not selected, the resulting curve degree may differ.
     
Use initial no. of arcs
  • When the box is checked, you can perform the conversion based on the number of arcs of the entity to convert: this number is added to the specified maximum number of arcs, so as to have a more coherent result for each curve.
  • When the box is cleared, the initial number of arcs of the entity to convert is not taken into account.
Update deviation Enables you to control the precision of the resulting curves.
  • When this check box is not selected, the resulting entity will keep its own local precision value.
  • When selected, the achieved precision will be assigned to the local precision of the resulting entity.
