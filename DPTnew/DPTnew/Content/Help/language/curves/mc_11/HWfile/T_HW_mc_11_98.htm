Multi-axis Stretch

  1. Set the reference system for the the stretch axes by selecting the corresponding item in the Reference drop-down list
    Work Plane The Work Plane reference system
    User defined A user defined reference system. To define it, do the following:
    1. Click Define reference
    2. Enter the system origin position
    3. Enter a point defining the X axis direction (i.e.: from the specified origin to this point)
    4. Enter a through point for the XY plane, defining the positive direction for the Y axis.
  2. Define the axes for the multi-axis stretch operation by selecting the corresponding item the Axes drop-down list:

    XY axes X and Y of the selected reference system
    XZ axes X and Z of the selected reference system
    YZ axes Y and Z of the selected reference system
    XYZ axes X, Y and Z of the selected reference system
  3. Type the scale factor value for the stretch in the Scale box.

  4. Click the Origin button and then indicate the point in the model to be used as origin for the stretch or type the stretch origin coordinates in the X, Y and Z and Y boxes.

  5. Double-click to apply the transformation.
