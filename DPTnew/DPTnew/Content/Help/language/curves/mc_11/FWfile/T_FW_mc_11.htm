ModifyCurveInterpolation Points

The Curve through Interpolation Points command provides an environment within which you can create and modify curves by governing their interpolation points. The Mode drop-down list enables you to choose whether to Edit Interpolation Points, to Add Interpolation Points or to create a New Curve.

As to the Edit Interpolation Points mode, there are a number of actions which can be easily performed on selected curves. See the documents in the following links for details.

  • Interactively modify the position of interpolation points, the tangency conditions at the end points or extend the curve, controlling its continuity with other curves
  • Apply geometric transformations to selected interpolation points
    •
  • Delete selected interpolation points from a curve.

If the selected curve is not a NURBS, if the Ask before converting check box is selected, you will be asked to convert it into a NURBS. The conversion precision is controlled by the Tolerance parameter. Please notice the conversion might modify the curve shape.

By default, interpolation point curves are NURBS of degree 3 (ordinary cubic splines), but in the Degree drop-down list under More Options you can also choose 5 to create a quintic curve or 7 to create a septic curve when you need smoother continuity at the interpolation points.

A set of Quality Checks is also available, enabling you to analyze the curvature of the selected curves.
