Editing a user provided Light Representation

Among the actions you can carry out by the Organize command, you can edit a user provided Light Representation.

Proceed as follows.

1 Once you've selected an X-Reference component in Light Representation, perform the Organize command. The Organize Light Representations dialog box is displayed.
2 You are enabled to edit exclusively user provided Light Representations. In fact, when you select a Light Representation in the list on the right side, you can notice the Edit button is activated only for the custom representations. Select the Light Representation you wish to edit and click the Edit button.
3 The Light Representation Data dialog box is displayed. You can modify the information in the Name and Description boxes.
4 As soon as you confirm by OK, the selected Light Representation will be updated in the list of the Light Representations currently available for the selected component.
5 You can proceed with the organization of the Light Representations or close the dialog box and end the command by clicking the Close button.
