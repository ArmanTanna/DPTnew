The New command enables you to define a user provided Light Representation by selecting a model to be used in place of the X-Reference component as a Light Representation.

The command is available from the context sub-menu displayed when right-clicking on the X-Reference (component) and selecting the Light Representations command.

Note that the command can be applied exclusively to X-Reference components.

Also note that, by performing the command, the Model Structure still displays the list of the events, but referred to the model replacing the component.

The model inserted as a new Light Representation will be displayed with the name you specified in the context menu area collecting the representations that require the user's intervention.

When you define a new Light Representation, in fact, the Light Representations sub-menu includes an additional row with the name of the representation that you inserted. You may add different new Light Representations. When you choose any of these, you apply it to the selected X-Reference (component).
