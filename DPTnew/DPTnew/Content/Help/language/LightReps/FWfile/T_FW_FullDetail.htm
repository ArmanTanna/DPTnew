The Full Detail command enables you to apply a full-detailed representation to a/different X-Reference component(s), currently displayed in Light Representation.

The command is available from the context sub-menu displayed when right-clicking on the X-Reference (component) and selecting the Light Representations command.

Note that the command can be applied exclusively to X-Reference components.

Also note that the full-detailed representation is the ordinary, default one for X-Reference components.
