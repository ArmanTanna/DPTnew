Printing documents

You can print either a single topic or all the topics of a heading:

  • How to print the current topic
  • How to print a single topic or all the topics of a heading

How to print the current topic

To print the current topic, do the following:

  1. Right-click on the right-hand frame of the Help (on the very top of the document and do not click on a picture). In the shortcut menu that will be displayed, select Print. The Print dialog box will be displayed.

  2. In the Name drop-down list, choose the printer.

  3. Under Print range, specify the pages you want to print. Select All to print all the pages of the current topic. Select Pages to print only the pages in the range specified in the adjacent from and to boxes.
    To print all linked topics, select the Print all linked documents check box.

  4. Select OK.

How to print a single topic or all the topics of a heading

To print a single topic or all the topics of a heading, do the following:

  1. Right-click on the topic or on the heading in the navigation pane (the left-hand frame of the Help) and select Print. The Print topics dialog box will be displayed.
    If the selected item is a topic ()
    • Select OK.
    If the selected item is a heading ():
    • To print only the document that is associated to the selected heading, select Print only selected topic
    • To print the document that is associated to the selected heading and all its subtopics, select Print selected topic and all subtopics.
    • Select OK to go on.
    The Print dialog box will be displayed.

  2. In the Name drop-down list, choose the printer.

  3. Under Print range, specify the pages you want to print. Select All to print all the pages of the current topic. Select Pages to print only the pages in the range specified in the adjacent from and to boxes.
    To print all linked topics, select the Print all linked documents check box.

  4. Select OK.
