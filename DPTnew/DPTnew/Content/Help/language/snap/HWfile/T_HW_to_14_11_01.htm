AutoSnap - Customize AutoSnap

When you start the Customize AutoSnap command, the program displays the Customize AutoSnap dialog box.

The program has four separate snap modes. Select the current snap mode by clicking the appropriate Snap: button. You can specify different snap point types and selection filters for each of the four snap modes.

The Customize AutoSnap dialog box enables you to define parameters concerning the following:

Description: Contains a user-supplied description to help you remember the purpose of the snap mode
Point Type Enables you to select the check boxes corresponding to the types of points to be automatically selected by the current snap mode
Selection Filter This area provides controls that enable you to select additional selection filters

When the Mandatory check box is selected, the snap action is mandatory and no point input using the mouse is accepted if it does not satisfy the current snap mode, even outside the snap trap.

In the Point Type area, you can select any of the following point types:

End Point When this check box is selected, end points are selected
Arc Center When this check box is selected, centers of arcs or circles are selected
Intersection Point When this check box is selected, Intersections are selected
Mid Point When this check box is selected, middle points are selected
Point on Curve When this check box is selected, points on curves are selected
Surface Vertex When this check box is selected, surface vertices are selected
Text Origin When this check box is selected, text origins are selected
Joint When this check box is selected, arc joints are selected
Tangent When this check box is selected, tangent points are selected
Work Plane Origin When this check box is selected, Work Plane origin is selected
Perpendicular When this check box is selected, perpendiculars are selected

Click the Clear button to clear all check-boxes.

In the Selection Filter area, you can select additional selection filters:

Point When this check box is selected, points are selected
Curve When this check box is selected, curves are selected
Surface When this check box is selected, surfaces are selected
Solid When this check box is selected, solids are selected
Profile When this check box is selected, profiles are selected
Text When this check box is selected, text is selected
Dimension When this check box is selected, dimensions are selected
X-Hatch When this check box is selected, X-Hatches are selected
Mesh When this check box is selected, meshes are selected
Component When this check box is selected, components are selected

Click the All button to select all the check boxes at once.

The check-boxes in the Inside area allow you to extend selection to entities contained in other entities:

Solids Extends selection to entities inside solids
2D Profiles Extends selection to entities inside 2D profiles
3D Profiles Extends selection to entities inside 3D profiles
2D Views Extends selection to entities inside 2D views
Components Extends selection to entities inside components
Only inside When this check box is selected, this indicates that you cannot select entities that do not form part of a solid or component

The following controls complete the AutoSnap selection:

Apply Applies the AutoSnap settings only to the current work session
Save Applies the AutoSnap settings to the current work session and stores them for subsequent sessions
Reset Cancels any AutoSnap settings you just made and restores the snap modes to their previous value
Default Resets the AutoSnap settings to their default values

When you have finished changing the AutoSnap settings, click OK or Cancel to close the Customize AutoSnap dialog box.
