Log file (exporting to CATIA®)

When you save a CATIA file, a log file of the conversion is created:

  • in the same directory of the file you opened
  • with the same name of the file you opened and the ".log" extension.

The log file structure

The log file for conversion to CATIA® lists the conversion parameters and information about the entities converted in the process.

In the log file you find:

  • The tolerance used to convert surfaces to Nurbs.
  • For each solid, the tolerance at which it is manifold.
  • The number of entities not processed (if any).
  • The number of entities processed (if any).
  • The list of entities in layers higher than 255 moved to a different layer (if any).
  • The list and count of CATIA entities in the output file
  • The code page index and data sizes.
  • The model size
  • The tolerance within which curves result identical.
