TD Catia v4 interface Options: Save Tab

Select entities: Determines the entities that are to be translated:
  • Wireframe — if selected, all entities corresponding to the CATIA® Wireframe category will be translated. If cleared, the Wireframe entities will be ignored.
  • Surfaces — if selected, all entities corresponding to the CATIA® Surface category will be translated to spline surfaces. If cleared, the Surface entities will be ignored.
  • Solids — if selected, all entities corresponding to the CATIA® Solid category will be translated to sets of faces. If cleared, the Solid entities will be ignored.
Save options: Determines the file saving mode:
  • Active layer only — if selected, only the entities belonging to the active layers of the model will be translated to the CATIA® model.
  • Partial — if selected, only the entities currently selected in the model will be translated to the CATIA® model.
  • All — if selected, all entities in the model will be translated to the CATIA® model.
Bezier approximation: Enables you to save entities as Bezier ones (they are approximated by entities defined by an arc):
  • Curves — if selected, curves are translated as Bezier curves.
  • Surfaces — if selected, surfaces are translated as Bezier surfaces.
Model size: Enables you to define the Model size, a parameter that CATIA® uses to determine the tolerances in calculations; if not correctly set, the solids loaded into CATIA® may not be closed:
  • Automatic calculation — if selected, the application analyzes the surface related precision of all surface entities, then calculates the model size from the resulting tolerance value.
  • Value — enter in the relevant field the model size. Clear Automatic calculation, if selected, and use this option if you need to impose a Model size coherent with other CATIA® files.
Solid options: Enables you to save solid entities as solids:
  • Create volume — if selected, solid entities are saved as solids and not as single surfaces.
Code page: Enables you to set the text code page:
  • Select from the drop-down list the graphic character set to be used for coding the output file.
Since CATIA® doesn't manage closed surfaces, a closed surface is split into two surfaces, as in the case of a cylinder within a solid.
Entities not correctly converted are moved to layer 1023.
See Exporting a model to CATIA for further details.
