The From Model into Catia Model command saves the current model as a CATIA® .model file.

You can find this command in the TD Catia v4 interface toolbar. The command is active only in the 3D environment of this software. To activate the toolbar, please refer to the TD Catia v4 interface installation procedure.

When you run the command, the file selection dialog box is displayed, from which you can save the document to the local file system or anywhere in your network.
While you are required to enter the file name, the latter will carry a .model extension. If you attempt to specify any other extension, this will be overwritten by .model.

In order to define the parameters controlling the conversion of entities, set the TD Catia v4 interface Options relative to the command before applying it.

Depending upon your workstation performance and the model complexity, the conversion may require some time to complete. To have an idea of the ongoing of the operation, you may want to open the I/O Window, if concealed, where the steps of the conversion process are displayed.

When you save a CATIA® file, a log file of the conversion is created in the same directory of the file you saved with the same name of the file you saved and the ".log" extension.

Known limitations
  • Only 3d entities are converted.
  • Line width management may lead to incorrect conversion.
  • Color mapping may lead to incorrect conversion.
  • More warnings and limitations are listed in the Conversion Table for exporting to CATIA .
