Importing Model From CATIA

Note that only 3d entities are converted.

To convert a CATIA® model file, proceed as follows:

  1. Start the TD Catia v4 interface Options command from the TD Catia v4 interface toolbar and access the Load tab.

  2. Set the conversion options as you like. Please refer to the TD Catia v4 interface Options command help for more information.

  3. Click OK to save the settings.

  4. Click the From CATIA to command from the TD Catia v4 interface toolbar.

  5. In the Look in box of the file selection box that displays, select the folder where the file to open is located.

  6. To help identify quickly the file to open, make sure that Files of type contains CATIA® Model or export (*.model, *.dlv, *.exp).

  7. Select the file you want to open then click the Open button.

  8. A new model window is opened and the converted CATIA® entities are displayed in it.
Different Units of Measure

If the Unit of Measure of the CATIA® file is different from the one currently in use then on opening a CATIA® file the following behavior will be observed:

  • A warning dialog box is displayed, informing you that the Unit of Measure will be changed to the one of the CATIA® file.
  • The current Unit of Measure is changed (not converted) to the one of the CATIA® file.
  • The CATIA® file is opened.
