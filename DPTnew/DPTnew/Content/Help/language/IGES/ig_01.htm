Opening an IGES file

To convert from the IGES format to think3 — both 3D models and 2D drawings —  do the following:

  1. Start the Open or Insert from File command.
  2. In the file selection box, under Files of type, select IGES (*.igs).
  3. Select the file you want to open.
  4. Select Options to open the IGES Options dialog box and set the parameters controlling the conversion process as you like.
  5. Select the Open button to open the file (no preview is available for IGES files: the file icon will be displayed in the preview frame).

When you open an IGES file, a log file of the conversion is created in the same directory of the file you opened.

At the time of importing a IGES model, you can perform a quality check on the model to know of any prevailing issues by selecting the Check Model Quality after loading check box in the System Options - General category.
