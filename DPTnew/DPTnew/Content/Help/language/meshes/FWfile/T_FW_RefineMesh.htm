Modify MeshRefine

The Refine Mesh command enables you to increase the number of nodes and triangles in the mesh making the surface smoother. This operation subdivides the mesh triangles producing four triangles for every original triangle.

  • Select the mesh you would like to refine.



  • To refine only a portion of the mesh, select the Mesh Zones check box and subsequently select the mesh triangles to be included for refining.

  • Make the additional settings under the Mesh control node:
    • By checking Preserve Boundaries check box, the operation will not affect the mesh (or mesh zone) boundaries.
    • A Crease detection value enables you to preserve or not creases on the mesh.
    • A Corner detection value enables you to preserve or not corners on the mesh.


Mesh zone to refine Crease detection = 150° Crease detection = 10°

Mesh zone to refine Corner detection = 150° Corner detection = 10°
