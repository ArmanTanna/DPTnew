Modify MeshSplit with Plane

The Split Mesh with Plane command enables you to cut triangular meshes with a plane that can be defined in an interactive way. This command also provides you with the option to retain or delete either of the split meshes.

  • Select the meshes you would like to split.



  • Choose the method to define the cutting plane in the Plane drop-down list. Depending upon the option you choose, additional selectors will be displayed to make the required selections.

  • The Keep both sides check box enables you to define which side of the meshes will be retained after splitting.
    • When the Keep both sides check box is selected, both sides of the meshes will be retained.
    • When the Keep both sides check box is cleared, red arrow is displayed on the splitting plane indicating the side of the meshes that will be retained. To invert the direction of the arrow, double-click on it.
