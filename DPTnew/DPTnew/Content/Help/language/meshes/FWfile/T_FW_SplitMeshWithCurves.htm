Modify MeshSplit with Curves

The Split Mesh with Curves command enables you to cut triangular meshes with curves. It is not necessary for the curves to be chained, closed or to lie over the meshes.

  • Select the meshes you would like to split.



  • Select the curves about which the meshes will be split. It is not necessary that the curves form a closed loop or a single chain. They can intersect. The curves are projected onto the meshes along the view direction and this projection will be used for trimming.

  • Select the Regions to Keep.
