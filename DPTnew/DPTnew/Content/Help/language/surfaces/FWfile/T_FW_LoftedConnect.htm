InsertSurfaceLoftedConnect


 
The Connect Lofted Surface command enables you to create connection surfaces (surfaces connecting two or more specified surfaces) by starting the Lofted Surface command and automatically setting the proper parameters for the creation of a connection surface.
Specifically, the surface Type under More Options will automatically be set to Connect.
 
  • For details on the command behavior, refer to the page on the Lofted Surface command.
     
  • For a step-by-step example, see Create a lofted surface starting from two boundaries (Connect).
    Please note that the example refers to the Default mode of the Lofted Surface command: some of the initial steps listed therein are not required when you start the Connect Lofted Surface directly.
