InsertSurfaceLoftedGrid


 

The Insert Surface Lofted Grid command enables you to create the following types of surfaces by starting the Lofted Surface command and automatically setting the proper parameters:

  • Grid surfaces (surfaces created starting from a mesh of curves formed by two almost orthogonal subsets of approximately parallel curves)
  • Bi-Rail surfaces (surfaces whose isoparametric curves in the V direction fit a set of selected 3D curves that do not intersect at any point)

The surface Type under More Options will automatically be set to Grid.  

  • The command enables you to manage the selection of closed curves both in Boundary Set A and in Boundary Set B. See "Managing Closed Curves with the Lofted Surface Command" for further details.
  • You can also manage the extension of the resulting surface in both orthogonal directions with no need to spend time providing any additional curve. See "Extending the Resulting Surface with the Lofted Surface Command" for further details.

For further details on the command behavior, refer to the page on the Lofted Surface command.
For step-by-step examples, see:

  • Create a lofted surface starting from two orthogonal sets of parallel curves (Grid)
  • Create a lofted surface starting from a set of parallel curves.

Please note that the examples refer to the Default mode of the Lofted Surface command: some of the initial steps listed therein are not required when you start the Insert Surface Lofted Grid command directly.
