Comparison between the Length and the Height mode

The Length mode applies a constant length along the sweep direction to the base curves.



The Height mode enables you to define the height of the path along the sweep direction. When the base curves are not plane, the height value is computed starting from an "average plane".



If you compare the two illustrations above, the difference between the two modes is outstanding.

If the base curves are plane ...
... the Height and the Length options work exactly the same way.
