Creating a Point Cloud surface by specifying a reference surface

In the following step-by-step example you will create a surface starting from a point cloud and specifying a surface to be used as a reference for the fitting operations.
Please note that you can accomplish this task by loading sample file "sr0508.e3".



1 Start the Point Cloud command.
 
2 If not already selected, in the Surface Type drop-down list select Fit cloud of points.
 
3 In the Input Mode drop-down list, choose Select.
 
4 Using a window selection, select the points to be fitted and end selection by right-clicking and choosing Continue in the context menu.


 
5 In the Fitting Mode drop-down list, select Surface based
 
6 Select the reference surface as in the following picture.


 
7 Under NURBS parameters, set the values for the resulting surface. For example set the following:
Deg. U 1
Cont. U 0
Deg. V 2
Cont. V 1
Click the Preview button ( ). The preview of the resulting surface is displayed:
