Applying additional conditions when creating a plane

Once you started the Plane command and you have selected the Reference Plane, using the More Conditions button, you can apply the following additional conditions to the plane you're creating (depending on the type you selected in the Reference Plane drop-down list):
 

  • None — no additional conditions
     
  • Parallel & point — Enables you to create the final plane parallel to the reference plane and through a point.
    • You will be required to specify the point.
    • To change the selected point (for example if you entered a wrong one by mistake), right-click on Point in the selection list, select Reset in the context menu and perform selection again.

     
  • Parallel & distance — enables you create the final plane parallel to the reference plane at a specified distance.
    • In the Distance box, enter the distance value.

     
  • Perpendicular & axis — enables you to create the final plane perpendicular to the reference plane and through a specified line.
    • In the Axis drop-down list, select the method to be used to define the line:
      Line To define the line as an existing line to be selected. Select the line.
      2 Points To define the line as the one identified by two points to be selected. Select the two points.

     
  • Angle & axis — enables you to create the plane at a specified angle with the reference plane and through a line.
    • First, define the line. In the Axis drop-down list, select the method to be used to define the line:
      Line To define the line as an existing line to be selected. Select the line.
      2 Points To define the line as the one identified by two points to be selected. Select the two points.
    • In the Angle box, tape the value of the angle.
