Creating a Point Cloud surface by fitting an existing surface

In the following step-by-step example you will create a surface by fitting an existing one.
Please note that you can accomplish this task by loading sample file "sr0308.e3".



1 Start the Point Cloud command.
 
2 If not already selected, in the Surface Type drop-down list select Fit cloud of points.
 
3 In the Input Mode drop-down list, choose Fit surface.
 
4 Select surface to be fitted.


 
5 Under NURBS parameters, set the values for the resulting surface. For example set the following:
Deg. U 4
Cont. U 2
Deg. V 4
Cont. V 2
Arcs U 2
Arcs V 2
Click the Preview button ( ). The preview of the resulting surface is displayed:
