The Lasso selection method

In order to select points to apply changes to, in addition to the usual Window selection (see also Multiple Selection Operators, you can use the Lasso selection method, choosing it in the context menu you can display by right-clicking (SelectLasso).

The Lasso selection method enables you to select all the points within a polyline which you define by indicating its points:

Please note:

  • The polyline used as a "lasso" must be a closed one.
  • To close the polyline, you must specify its last point as close as possible to its first point.
