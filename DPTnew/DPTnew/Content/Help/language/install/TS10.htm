Slow opening of assemblies with Quick Load

The Quick Load option in the Open dialog enables you to load any X-Ref Component in the lightest representation, that is in Flat Tessellation, regardless of what is specified in the selected Visual Bookmark.
However, if the model has been created with a version prior to 8, the light representations do not exist in the model file and have to be calculated.

For this reason, loading for the first time an assembly with the Quick Load option active, may result in a lengthy operation signaled by an appropriate dialog.
If the assembly is then saved, along with all of its components, the light representation will be stored in the model files and thus will be available the next time the file is loaded with a sensible decrease in the loading time.
