Warnings and Limitations about some features of v. 2008.1

  • Models created in 2008.1 using decal or materials can be opened, but the materials and decals will be lost.
  • Models created in 2008.1 with new lights / lights studios set up, can be opened, but only some lights (previously supported) will be available, the rest will be lost.
  • Models created/saved in 2008.1 with environments setup can be opened, but with the complete loss of environment.

For further details on the single commands mentioned in this document, consult the online documentation.
