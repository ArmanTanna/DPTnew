The Report in CSV Format command uploads the current contents of the selected catalog or family to a file in CSV format and starts Microsoft Excel to display the file.

This command is located in the context menu available when right-clicking on an index or catalog of components listed in the browser frame of the Catalog Browser and in the toolbar of the command main dialog.

The thinkparts database fields are organized in columns; the data pertaining to the single items are organized in rows. If an index is selected, all components of the selected branch will be reported.

In order to get a correct format of the Microsoft Excel file, the CSV format Separator: parameter in the Report category of the System Options-thinkparts must be aligned with the List separator parameter in the Number tab of the system Regional Settings that are displayed from Control Panel.
