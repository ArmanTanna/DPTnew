The Send... command enables you to send the selected family or catalog as an attachment to a mail message.

This command is located in the context menu available when right-clicking on an index or catalog of components listed in the browser frame of the Catalog Browser or Catalog Manager.

The command exports the selected family or catalog to a temporary file, much as the Export command does; it then opens your email client and loads the exported file as attachment to the message. To send the catalog, simply complete the message with the recipient's address and any relevant text.
