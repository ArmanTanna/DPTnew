The Copy Catalog command creates a copy of an existing catalog.

You can find this command in the context menu that displays when right-clicking on a catalog in the Catalog Manager browser frame.

Upon execution, the command opens the following dialog:



The From: fields reports the Name and Description, separated by a "—", of the original catalog to be copied.

Enter in the To: field the name of the new catalog. Do not enter the description that will be created for you as Copy of <original catalog description>. See the New Catalog command for more details on the catalog name. Exit by pressing the TAB key in order to enable the OK button.

Click OK to start the copy of the catalog. If the OK button is not enabled, check that you entered the new catalog name in the To: field, and that you completed the name by pressing the TAB key.

Click Cancel to close the dialog.

While the copy is being made, you are advised of the progress of its phases by a series of progress bars and checkmarks in the Copy section of the dialog.

The new catalog is inserted at the highest level of your thinkparts tree structure as you can see in the Catalog Manager tree browser frame; you are enabled to drag and drop it into an existing index.
