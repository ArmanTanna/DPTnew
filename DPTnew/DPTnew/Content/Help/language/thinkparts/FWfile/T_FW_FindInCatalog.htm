The Find in Catalog command enables you to locate the catalog where the selected part belongs.

You can find this command in the context menu that displays when right-clicking on a part inserted in the assembly.

The command actually starts the Catalog Browser and opens the thinkparts catalog where the selected part was originally taken from. The line corresponding to the selected part is highlighted in the catalog query view.
This enables you to quickly retrieve a similar object from the same catalog. Note that once the catalog is open, you are enabled to close it and navigate through all available catalogs and idexes.

For more information on how to use the command, please refer to the Open and Import commands.
