Creating a new catalog of flanges

In this example of the usage of the Create Part Family command, you have a model of a flange from which you want to create a catalog of flanges of different diameters.

Prerequisite to this procedure is the availability of a think3 application licence and the installation of thinkparts. A model (a flange in our example) must be open in the think3 application.

These are the steps necessary to accomplish the task:

  1. Rotate the model so that it looks nicely; this view will be captured to obtain the catalog preview. It is also a good idea to hide all dimensions leaving just the ones that correspond to independent variables, as these will be the part attributes.
  2. Start the Tools Spreadsheet to enter the following variables and assign them to the relevant dimensions:

  3. Start the Tools Create Part Family command and enter the following information in the relevant fields:
    Name FLANGE
    Description Flanges
    Part description Flange of diameter <d>
    Press OK when you are ready to create the catalog.
  4. Open the Catalog Browser, if necessary, and verify that the Part Family index has been created and that it contains the Flanges catalog.
  5. You are now enabled to use Edit Values to add new parts to your catalog.
