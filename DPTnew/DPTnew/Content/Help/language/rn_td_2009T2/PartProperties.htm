New Part Properties Functionalities and Improvements

Version 2009.2 introduces the following enhancements and improvements to the Part Properties functionalities.

  • Enhancements in the Part List command

Enhancements in the Part List command

Significant enhancements have been made in the Part List commands, which are:

  • Part List page configuration
  • User interface enhancements
  • Part List redefinition

Part List page configuration

The FormatOrganizePart List command has been enhanced with the new Pages category in the Part List Options dialog box. It enables you to divide the complete Part List table into many parts (called pages) to insert them within the drawing frame for better visibility.

First row number Enables you to specify the row number with respect to the complete Part List, which will be the first row of the currently configured page.
Vertical displacement Enables you to specify the vertical offset to be used for the current page with respect to the previous page.
Add Enables you to load the selected first row number and vertical displacement values to the configuration list to create multiple pages.
Remove Enables you to remove the first row number and vertical displacement values from the configuration list.
Load Lists all the loaded page configurations.

All the settings made in the Part List Options dialog box are saved to the main configuration file and are used while inserting a Part List in a drawing.

User interface enhancements

The ToolsPart List command has been improved with new options and the integration with the FormatOrganizePart List command. The user interface has been significantly refined with the following enhancements to facilitate the new capabilities.

  • The new Position drop-down list contains five options (By selection, Top left, Top right, Bottom left, and Bottom right) enabling you to align the Part List origin marker with respect to a predefined point in a drawing frame. It is available only when a customized drawing frame contains predefined points (minimum of 1 and maximum of 4) inserted using the Drawing initialization command.
  • To offer easy accessibility, the Extend to option available in the Table Format - Advanced category in the Part List Options dialog box is now also available under More Options in the Part List selection list.


In the ModifyPart ListRedefine command, the selection list has been enhanced with the following new options.

  • The new Pages node with the Row option enables you to redefine a Part List page. Select a row at which you prefer to break the Part List page.
  • The new Offset option enables you to redefine the vertical offset distance to be used for the current page with respect to the previous page.

Part List redefinition

In the redefine mode of a Part List, you have the possibility to edit a Part List page by using the following new options available in the context menu.

When the last row in a page is selected When a middle row in a page is selected When the first row in the first page or the last row in the last page is selected When the first row of neither the first nor the last page is selected
  • Move to Next Page
  • Remove Page Break
  • Remove All Page Breaks
  • Split Page Here
  • Remove All Page Breaks
  • Remove All Page Breaks
  • Move to Previous Page
  • Remove All Page Breaks

Move to Next Page The selected row will be the first row of the next page.
Remove Page Break The next page will be merged with the current page.
Remove All Page Breaks The complete part list will be a drawn into a single Part List table until maximum number of rows (999) is reached. The rows beyond the maximum limit are included in the second table.
Split Page Here The current page will be divided into 2 pages. The selected row will be the first row of the next page.
Move to Previous Page The selected row will become the last row of the previous page.
