When the Top Component tool is active, it enables you to select only top level components. Your selection on the graphic area is highlighted in the Model Structure.
Another point of importance here is that the program remembers the filter type set during a working session and opens with the same in the subsequent session.

Note:
When you perform a window selection with this filter ON, all the top level entities within the window will be selected.
