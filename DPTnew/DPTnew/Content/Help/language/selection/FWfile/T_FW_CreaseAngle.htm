Crease Angle


When the current selector of a command accepts the selection of facets, the Crease Angle command ( ) will be contained in the context menu displayed by right-clicking. When you select the Crease Angle command, the cursor immediately changes to .
Picking a facet of the mesh, the connected facets will increase the selection if the angle between their normal and the normal of the picked facet is below the crease angle. The crease angle value can be changed by dragging the Angle handle. This handle can be displayed or not by using the Show Crease Angle option in the right-click context menu.
To confirm the selection you have performed, double-click in an empty zone of the graphics area or right-click and select Exit Crease Angle. As soon as the selection is completed, a preview of the result will be displayed. You can then start the Crease Angle command again in order to modify your selection.
If you press <CTRL> the cursor changes to . Then, if you pick a facet, a removal mesh zone is defined by the crease angle and the facets inside this zone are removed from your selection.
