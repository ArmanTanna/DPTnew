When the Select Solid tool is active, selecting a solid selects the entire solid, rather than its individual component entities (such as faces and edges).

To select the individual component entities of the solid, click Select Face to make it the active selection tool.

Note

When you perform a Window selection including solids, the entire solids will be selected, no matter which of the two possible solid selection modes — Select Solid or Select Face — is currently enabled.
