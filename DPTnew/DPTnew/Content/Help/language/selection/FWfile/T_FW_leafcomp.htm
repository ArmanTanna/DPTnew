When the Leaf Component tool is active, it enables you to select and redefine components lower down in the Model Structure without having to set them current.

Your selection on the graphic area is highlighted in the Model Structure.

The program remembers the filter type set during a working session and opens with the same in the subsequent session.

Note:
When you perform a Window selection with this filter ON, only the lower level entities within the window will be selected.
