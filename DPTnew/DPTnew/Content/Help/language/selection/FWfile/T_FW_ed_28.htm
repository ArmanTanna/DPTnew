Edit Select Chain


The Select Chain command enables you to select a chain of connected curves that can be a series of geometrical entities connected end to end at their endpoints (geometrically or virtually).

The command can also be started by right-clicking on a curve in a chain and choosing SelectChain from This. The whole chain will be automatically selected.

The Select chain of connected curves check box in the System Options-Input-General category of the Options/Property dialog box enables you select the chain of curves in the two ways.

  • When the Select chain of connected curves check box is selected, this command selects curves which are geometrically connected to each other.
  • When this check box is cleared, the curve chain gets selected if it appears to be connected from a particular orientation of the view. These curves may not be geometrically connected.
