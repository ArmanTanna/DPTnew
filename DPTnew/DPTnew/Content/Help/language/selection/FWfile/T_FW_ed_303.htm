When the Select Face tool is active, selecting an individual component of a solid (such as a face or edge) selects only that component, rather than the solid itself.

To select the entire solid, click Select Solid to make it the active selection tool.

Note

When you perform a window selection including solids, the entire solids will be selected, no matter which of the two possible solid selection modes — Select Solid or Select Face — is currently enabled.
