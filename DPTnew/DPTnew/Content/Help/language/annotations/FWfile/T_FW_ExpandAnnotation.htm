The Expand command enables you to expand an annotation (see Expanded Mode in the glossary; see also Expanding an Annotation in Annotations: User Operations).

To start the command, right-click on the annotation label in the model and select Expand in the context menu.

  • If the annotation is of the Text type (see New Annotation), a dialog box displaying the associated text will be displayed.
  • If the annotation type is From file (i.e.:show reference external file in expanded mode), a dialog box displaying the contents of the external document will be displayed.
  • If the annotation type is From Internet, a web browser will appear in the expanded annotation’s frame and will load the page or document identified by the URL address.
