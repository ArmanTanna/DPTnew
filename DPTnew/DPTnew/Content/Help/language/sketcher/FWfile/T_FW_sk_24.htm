ModifyProfileUpdate

The Update command immediately updates the profile to reflect any changes that have been made since it was last updated.

If the automatic updating feature is enabled (as determined by the Automatic resolving check box under General on the Profile category), most changes you make to the profile (such as adding or modifying dimensional and geometrical constraints) cause the profile to update automatically to reflect those changes. When this feature is not enabled, you can use the Update command to update the profile so that you can see the effects of these changes.
