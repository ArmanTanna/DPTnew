The Local button enables you to load the local configuration for the application.

When you click on Local button, the current program's configuration will no more be automatically updated through the network.

The Remote button enables you to load the remote configuration for the application.

When you click on Remote button, the current program's configuration will be automatically updated through the network.

You need to select the remote configuration and restart the application to load the remote configuration.
