ModifyImageScaling

 

The Scaling command enables you to change the scale of the image you imported.

As soon as the command is selected, a dialog box is displayed enabling you to set the parameters controlling the scale change. There are three different ways image objects can be rescaled, depending on the option you select:
 
World units per pixel Select this option if you want to specify (in the adjacent box) the size pixels must have in the real World (in the current unit of measure, whatever it is: inches, centimeters, etc.)
Dots per inch and Scale Select this option to specify the number of dots per inch (for example: 100, 150, 300, 600, ...) and the scale factor (1:50, 1:2500, ...), to be typed in the adjacent boxes
Select Select this option to apply the Distance value to two points you have to select on the image after clicking Two Points. The world units per pixel, in this case, are automatically calculated based on the distance and the two points.
