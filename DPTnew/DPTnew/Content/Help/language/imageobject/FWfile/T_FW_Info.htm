ToolsImageInfo


 

The Image Info command enables you to view the image data, both for the original file and after loading it.

Image Info command can also be activated when you right-click on an image in the context menu. The following data can be viewed using Image Info command:

  • Source file: — The source file name
  • Dimensions (pixels): — Its size in pixels
  • Colors: — The number of colors used
  • Memory used (KB): — The memory space it requires in Kilobytes.
