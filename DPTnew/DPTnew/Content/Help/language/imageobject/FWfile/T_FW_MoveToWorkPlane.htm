ModifyImageMove to Work Plane


 

In the Model enviroment, the Move To Work Plane command enables you to move the image to the Work Plane after you have changed the position of the latter. In fact, when you move the Work Plane after importing an image, the image, originally positioned on the Work Plane, remains in its position. Use the current command in order to update the image position.

In the drawing environment, the Move To Work Plane command can be used to move the image back to the origin once you have positioned it somewhere else in the drawing.
