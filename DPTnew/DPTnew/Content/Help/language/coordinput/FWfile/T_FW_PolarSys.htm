The Polar System command enables you enter the coordinates of points as distance and angle with the X axis, relative to the coordinate system origin.

The command is available whenever the point coordinates are enabled and is executed by either hitting the F10 key or by right-clicking and selecting Polar System from the context menu.

Enter the distance from the origin and the angle with the X axis by typing their values in the relevant minidialogs boxes. You can displace the coordinates origin, which is initially set coincident with the Work Plane origin of the coordinate system, by pressing and holding the Shift key and selecting a new point; you are also enabled to reorient the coordinate system X axis by pressing and holding the Ctrl key while you select a line. The program will orient the X axis of the coordinates system to be parallel with the selected line.
