The Enable Point Coordinates command enables you define a point by specifying its coordinates in either a cartesian or polar system.

The coordinate input method is available inside the commands requiring point input and is enabled by either hitting the F9 key or by right-clicking and selecting Enable Point Coordinates from the context menu.

You can specify the coordinate values by typing them into the corresponding minidialogs boxes, and switch between cartesian or polar systems by selecting the relevant command from the context menu or by hitting the F10 key.
