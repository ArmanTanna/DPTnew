The Cartesian System command enables you enter the coordinates of points as X, Y and Z coordinates with respect to the coordinates system origin.

The command is available whenever the point coordinates are enabled and is executed by either hitting the F10 key or by right-clicking and selecting Cartesian System from the context menu.

Enter the X, Y and Z (Z is displayed only if you set the Mouse Input to On view in the Input category of System Options) and Y coordinates by typing their values in the relevant minidialogs boxes. You can displace the coordinates origin, which is initially set coincident with the Work Plane origin of the coordinate system, by pressing and holding the Shift key and selecting a new point.
