The Synchronize command enables you to re-align the other constrained views after you have changed one of them (by zooming or panning). It works as follows:

  • It retains the view orientation unchanged
  • It copies the current constrained view zoom factor to each other constrained view
  • It applies the current constrained view pan factor to each other constrained view.
