The Centerlines command creates:

  • the centerline axes not only for the selected circle or arc, but also for all 2D entities that are edges of some axial surface in the model as shown in the image below.



  • the circular centerlines and radial axes for the patterned holes in a drawing environment.

The Centerlines command creates the centerline axes for the selected circle or arc.

The Mode drop-down list of the command's selection has the following check boxes. This drop-down list is available only in drawing enviroment.

Axes on hole It enables you to create centerlines only for holes.
Axes on circular pattern It enables you to create radial axes and circular centerlines for the patterned holes.
When this option is chosen from the drop-down list, it displays the following check boxes:
Show circular centerline When selected, enables you to create circular centerlines for the selected patterned holes.
When cleared, circular centerlines creation is not possible.
Show radial axes When selected, enables you to create radial axes for the selected patterned holes.
When cleared, radial axes insertion is not possible.

The lock cursor is displayed during the pre-selection phase when the cursor is moved over a selectable edge, which is an associative entity.
