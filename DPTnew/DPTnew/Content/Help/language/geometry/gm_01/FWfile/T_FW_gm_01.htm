Lines can be created using the following commands.
 
Two-Point Line enables you to create a line by specifying two through points or its components in a Cartesian or Polar reference.
Parallel Line enables you to create lines parallel to existing lines you can select as a reference in the model/drawing.
Angular Line enables you to create a line at an a given angle relative to the X-axis passing through a specified point and extending to the edges of the current view.
Centerlines enables you to create the centerline axes not only for selected circle or arc, but also for 2D entities that are edges of some axial surface in the model.
Bisector Line enables you to create the bisector line of two selected lines.

The following method is also available:
 
Perpendicular Line a method enabling you to create lines perpendicular to existing lines in the model/drawing.
