The Edit X-Hatch command enables you to manually modify the applied hatching.

This command is available from the context menu displayed when you select a hatch and right-click on it.

When you start the command:

  • You are in edit mode of hatch to which you applied the command, but this does not affect the other objects within your document.
  • You are enabled to redefine the applied hatching, that is can flip the hatch lines, increase or decrease the spacing of the hatch lines according to your needs. To do so:
    1. Right-click on the hatch and select Set Current from the context menu.
    2. Select the required hatching you wish to modify. To invert the angle of the hatching, right-click and from the context menu, select Edit X-Hatch Invert Angle
    3. To increase or decrease the spacing of hatching lines, select the options Increase Spacing or Decrease Spacing respectively.


Also fine-tuning of the automatic section hatching can be done by following the below steps:

  1. Double-click on the graphic area to de-select any pre-selected views or views that were Set Current.
  2. Right-click on the sectional view and select Edit Properties from the context menu.
  3. Activate the X-Hatch category under Sectioned of the Entity Properties dialog box that opens, if it is not already activated.
  4. Adjust the properties under Automatic angle and spacing area to your needs.
  5. Click OK to apply your changes.
