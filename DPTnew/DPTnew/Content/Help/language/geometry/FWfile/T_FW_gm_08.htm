Insert Drafting X-Hatch

The X-Hatch command enables you to hatch one or more closed areas. The hatch is created in accordance with the settings you can define using the dialog box opened by clicking Properties in the selection list.

Two different choices are available in the Regions drop-down list for the definition of the regions to be hatched:

  • By boundary — the regions are defined by selecting the boundary curves and any islands enclosed within that boundary (in which case the hatch will be related to the selected boundaries by an associativity link: if you modify the boundary curves, the hatch will be modified consequently).
  • By internal point — the regions are defined by clicking the boundary, which may also contain curves places outside the current viewport and not visible, in that case the program automatically identifies the boundary and any enclosed islands and no associativity link will be established.

    Note
    The final X-Hatch will be just one entity, even if you select any number of disconnected areas.

Applied hatches can be redefined using Edit X-Hatch command in the context menu (pre-select the hatching and right-click).
