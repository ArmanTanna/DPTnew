Rotating a Set of Entities

After starting the Move Copy Entities command, to apply a rotation, proceed as follows:

1 Select the entities you want to apply the transformation to using a window selection.



The set of handles is displayed (for details on the handles, see the description of the Move Copy Entities command). As you selected a set of entities, the origin of the transformation is placed in the center of the viewport. Deselect the vertical line on the left side of the bar by pressing the CTRL key and clicking on the line.



The vertical line is deselected.
2 To define a different starting point, click on Handle Origin in the selection list to expand it, right-click on Start point and select Reset in the context menu. Click on Start point and then specify the point entity right in the center of the circle on the left side of the bar, as shown in the following illustration.



The origin of the transformation is moved onto the selected point.
3 To rotate the selected entities, drag the arc handle. The value of the angle of rotation about the selected origin is displayed in the Angle box (which you can also use to specify the desired value).



Please note that a dynamic preview mechanism is provided, so that you can interctively check the new current position of the selected entities.

4 Click to confirm.
