Creating points on a surface

After starting the On Curve/Surface command, proceed as follows.

1 Select the surface on which you want to create the point entities.


 
2

After choosing the surface, the Points option enables you to select the point creation method:

  • By parameter
  • By distance
The Points Along U and the Points Along V options enable you to specify the number of points to be placed in the respective directions, on the surface.
 

3 After specifying the number of Points Along U, type the value of the u parameter in the Start U box, to specify the initial position where a point entity has to be created (remember that the u parameter varies within the range [0,1]).
 
4 In the End U box, type the value of the u parameter in the final position where a point entity has to be created.
 
5 After specifying the number of Points Along V, type the value of the v parameter in the Start V box, to specify the initial position where a point entity has to be created.
 
6 n the End V box, type the value of the v parameter in the final position where a point entity has to be created.
 
  This is the result if you position a grid of 10 by 10 points on the entire surface.


 
  This is the result if you position a grid of 10 by 10 points on a portion of the surface.
