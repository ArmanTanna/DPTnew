InsertSheet MetalVertex Rip

The Vertex Rip command enables you to rip the vertex on the selected solid.


After selecting a vertex(either the inner or the outer one) for the Vertex selector, the Information on Edge node gets displayed in the selection list.

The number corresponding to each incident edge of the selected vertex is displayed in the Information on Edge drop-down list. The edge number selected in the drop-down list is also highlighted in the Graphics Area as a firm line. You can toggle the edge selection by using the Previous Group and Next Group buttons.

You need to use the Rip By drop-down list to define the bend radius or the rip gap for each edge.

Radius

Select the Radius option from the Rip By drop-down list for any of the incident edge of the selected vertex. The current radius value (defined in the Inner radius: text box under the Bend area of the Entity Properties-Sheet Metal category) is applied while ripping the vertex. To specify a different radius for each incident edge; click on the mini-dialog box and enter the required value.

Gap

Select the Gap option from the Rip By the drop-down list for any of the selected vertex. The current gap value (defined in the Gap text box of the Rip/Corner page of the Entity Properties>-Sheet Metal category) is applied while ripping the vertex. To apply a different gap value:

  1. Click on Local Properties. The Gap text box will be displayed.
  2. Click on the Gap lock icon to unlock it.
  3. Enter the required gap value.
