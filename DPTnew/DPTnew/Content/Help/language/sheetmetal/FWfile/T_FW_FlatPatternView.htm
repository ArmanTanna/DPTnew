InsertDrawing ViewFlat Pattern

The Flat Pattern View command enables you to create a flat pattern view of a Sheet Metal part. This view is a flattened drawing irrespectively of the model state (model could also be in bend state).

Once the command is started, the Flat Pattern View dialog box is displayed. In the graphics display area of the dialog box, select the solid to be flattened and click OK. The view is created: specify its position in the drawing.

Relief lines can optionally be removed from the drawing.

You can control the view parameters in the Drawing View- Flat Pattern category page of Entity Properties.

Note
It is not possible to generate this kind of views in faster way using the Hidden line computation: option in the Entity Properties - Drawing View category.
