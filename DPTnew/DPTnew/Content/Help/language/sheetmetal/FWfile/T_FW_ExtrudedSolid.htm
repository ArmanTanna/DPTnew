InsertSheet MetalSolid Flange

The Solid Flange command enables you to create a sweep solid starting from an open profile containing only lines and arcs with a user defined Length and the Thickness value automatically inherited from the one defined in the Bend page of the Sheet Metal category inside Entity Properties. Bends are automatically created where necessary.

In the Solid Flange command it is possible to hold in check the thickness direction of each edge. Because of this you can create your profile and make it parametric without considering the thickness on the single edge.

The Local Thickness Direction check box is available in the Solid Flange command. When checked, the thickness direction of each edge is displayed as a red arrow. You can change the thickness direction of an edge by double clicking on its arrow.
