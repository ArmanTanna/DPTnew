Managing interference during Unbend

Open the Unbend.e3 file from the Samples folder in your installation and proceed as per the following steps to manage interference warnings.

  1. Invoke the Unbend command and select the horizontal face for the Fixed Face/Edge selector, as shown below.



  2. Select the Show Preview () button on the selection list; the interference warnings are displayed in the graphics area. Also, the interfering regions are highlighted.



  3. The display of warnings can be toggled by using the Hide Warnings () and Show Warnings () buttons. These buttons will only be displayed if a positive interference is found.

  4. To hide the preview, choose the Hide Preview () button.

  5. Even though interference warnings is displayed, the program still allows to perform the unbend operation. On choosing the Apply () or OK () button, the command will be executed.
