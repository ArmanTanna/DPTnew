Adding a flange to a solid with Relief

To accomplish this, task, load file "Flange_Relief.e3".

1 Start the Flange command. Select the Edge as shown in the figure.
2 Click Reliefs under Local Properties. Select Corner from the list. For Type select one of the following:

a. Select Bend from the list.

b. Select Bend and flange from the list.

c. Select Bend and flange chain from the list.

d. Select Rectangular from the list and specify a value in the Gap: edit box.

e. Select Circular from the list and specify a value in the Diameter edit box.
3 Click or to confirm your selections and create the flange.

Note
These reliefs will not appear to be perfectly rectangular or circular when the model is in the bend state. However, exact results are obtained on unbending the model or generating its flat pattern view.

If the specified Diameter (for Circular relief) or Gap: (for Rectangular relief) is less than the minimum possible value, a suitable warning message will be displayed on applying the Flange command.
