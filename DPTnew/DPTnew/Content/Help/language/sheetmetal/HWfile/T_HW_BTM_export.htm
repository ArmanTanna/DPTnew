Exporting a file from the Bend Table Manager

The following illustration explains the procedure to export a *.CSV file from the Bend Table Manager:

1. Start the Bend Table Manager command from the Tools menu; the Bend Table Manager dialog box is displayed.
2. Select the rows that contain the data to be exported into a *.CSV file.


  • To select a single row, click on the row number.
  • To select multiple rows, hold the CTRL key down while making the selection.
3. Right click on the selected rows to display the context menu. Select the Export CSV File option.
4. The Export CSV File from Bend Table Manager dialog box is displayed.
5. Enter the desired file name and browse for the appropriate location. Choose the Save button to export the selected bend table data.
6. The exported bend data can be viewed in the spreadsheet format. Apart from the columns present in the Bend Table Manager, there will be columns for additional parameters that are Interpolation, AngleType and FixedValue. The value of these additional parameters along with that of Unit will be displayed only once for each material.
