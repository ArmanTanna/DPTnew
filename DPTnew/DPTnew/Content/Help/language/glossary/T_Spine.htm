Using Spines

Some rules about spines to be observed when creating spined solids and features.

The conditions applicable to the profiles (boundary and spine) are the same as those that apply when you create Spined surfaces:

  • The curves composing the spine must be wrapped into a profile before you start the command (see "How to create a profile").
  • The placement plane of the boundary profile must be orthogonal to the spine profile.
  • The placement plane of the boundary profile must intersect the spine profile at the point where the spine profile starts or at the point where it ends.
