Tips on the use of GSM

Here are some useful tips on the use of the Advanced GSM command.
 

  • Use Layers to hide/unhide entities
    You can't use the Hide/Unhide commands when the Advanced GSM command is active. In order to control the visibility of entities — which is very important when dealing with overlapped entities or complex cases.
     
  • Use layers to group entities
    When you modify entities using the Advanced GSM command and the Copy check box is selected, the original surfaces are automatically hidden (unless you have deselected the Hide check box), and the new ones are displayed. This could lead to have a lot of surfaces overlapped (the original ones and the new ones), and it might be rather annoying. To avoid this problem it is recommended to group all entities to be modified in one layer, before using the Advanced GSM command.
