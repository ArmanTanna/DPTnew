Filter and Filter Options

The think3 application provides the filter functionality to make a search for a particular visual bookmark or annotation. The same filter is used to make search for markups too. This is especially helpful when multiple visual bookmarks and/or annotations (and/or markups) are present in the file.

Refer to the following documents for details on using the filter and making settings for the filter options.
