ThinkDesign 2018.1 and 2018.1.SP1 Release Notes


Many improvements have been made in version 2018.1. This document contains a list of short descriptions for each new feature, improvement and enhancement.

  • Data Converters Upgrades
  • Multi Flange Overlap Option
  • 3D Manufacturing Format (3MF)
  • New User Interface
  • General Software Improvement and Better Reliability


Data Converters Upgrades

The TDXchangeReader module is now capable to read:


ThinkDesign 2018.1

  • Any SolidWorks version up to 2018

ThinkDesign 2018.1.SP1

  • Any CATIA® V5 version up to V5-6 R2018 (R28)
  • Any Parasolid® version up to v30.1
  • Any Siemens NX™ (Unigraphics) version up to NX 12.0

For the Autodesk DXF/DWG format, ThinkDesign has been upgraded to be compatible with Autodesk® DWG™ 2018 file format.

For the GBG DraftMaker format, ThinkDesign has been upgraded to import Raster Image reference.


Multi Flange Overlap Option

ThinkDesign 2018.1 introduces a new Overlap option in Multi Flange command, available under More Options. With this new option all flanges are created as overlap, while by default flanges are trimmed at the intersection corners.



3D Manufacturing Format (3MF)

The 3D Manufacturing Format (3MF) is a 3D printing format that allows design applications to send full-fidelity 3D models to a mix of other applications, platforms, services and printers. ThinkDesign 2018.1 enables now to import and export 3MF files (see Conversion to and from the 3MF format).


New User Interface

The medium size option for toolbars is available in ThinkDesign 2018.1.SP1.


General Software Improvement and Better Reliability

The source code has been accurately revised so that the program has highly improved its robustness and its reliability has been made much higher.
