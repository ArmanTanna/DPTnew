EditDelete


 
The Delete command deletes the selected entity.

You can preselect the entity, or start the command and then select the entity to be deleted.

When you start the command, you are prompted to select the entity to be deleted. As soon as you click on the entity, it is immediately deleted. The command then repeats, enabling you to delete another entity.

You can also preselect the entity to be deleted. As soon as you start the Delete command, that entity is immediately deleted. When you preselect an entity, however, the command does not repeat.

Applying the command to entities belonging to groups
The command can be applied to entities belonging to groups by using the Select inside Groups command.
