EditWork PlaneThrough 3 Points

The Work Plane through 3 Points command defines the Work Plane based on three specified points. The first point determines the origin of the new system, the direction from the first point to the second point determines the X-axis, and the third point completes the definition of the XY-plane.

If one or more of the points are special points of solids (vertices, centers of holes, and so on) selected using the special point selection commands or with the automatic mouse snap facility, the Work Plane is associatively linked to the solid.
