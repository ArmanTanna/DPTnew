EditWork PlaneUnlink

The Unlink Work Plane command breaks the links that associate the Work Plane with an entity. The associativity could have been created on using the Associative Mode of the Work Plane on Entity command or if snapping was used to position the Work Plane while using the Select command.

As soon as you start the command, any Work Plane link created when the Work Plane was positioned is immediately removed. If the lock marker was previously displayed, it disappears.
