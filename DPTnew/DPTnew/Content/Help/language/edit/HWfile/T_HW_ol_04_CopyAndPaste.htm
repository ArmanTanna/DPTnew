Drag & Drop: Copying and Pasting an Object

  1. Click on the object you want to copy with the left mouse button. Hold the mouse button down.


     
  2. Hold the mouse button down. A preview of the copy is displayed so that you can position it correctly. Move the copy with the mouse, holding down the left button.


     
  3. Hold down the CTRL key and release the left mouse button to insert the copy.
