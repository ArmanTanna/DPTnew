The Search button displays an additional area supplying you with controls that will enable you to easily find parameters you might need to set (among the ones contained in the tabs of the System Options, Document Properties or Organize Styles dialog boxes).
 

You just have to specify a search string. A list of all the category pages containing parameters matching the search string will be displayed. To open a page, you just have to double-click on its name in the list.
 

Proceed as follows:

  • In the Text: box, type the string to be searched for, or select an existing one (if any) from the drop-down list.
  • In the Folder: drop-down list, select the folder where the string is to be searched for.
  • Options:
    • Select the Match whole word only check box to search for the exact string you typed, excluding similar words
    • Select the Match case check box, to perform a case-sensitive search
  • Select Find to start searching.
  • Once the string has been found, a list of the tabs containing the matching controls is displayed in the list box. Double-click on the tab name to open it.
