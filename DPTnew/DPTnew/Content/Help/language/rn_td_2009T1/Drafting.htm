New Drafting Functionalities and Improvements

Version 2009.1 introduces the following enhancements in the Drafting commands.

  • Enhancements in the Automatic Balloon command

Enhancements in the Automatic Balloon command

In this version, the Automatic Balloon command features two new options under the More Options node of the selection list:

  • The Avoid balloon duplications check box
    • When selected, the program assigns the same label to only one of the entities sharing the name.
    • When cleared, the program assigns a label to all entities.
  • The Same item position for same part number check box
    • When selected, the program assigns the same position value to all entities sharing the name.
    • When cleared, the program assigns a different position value to each entity.
