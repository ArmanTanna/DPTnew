New Sheet Metal Functionalities and Improvements

Version 2009.1 introduces the following enhancements and improvements to the Sheet Metal functionalities.

  • The new Vertex Rip command
  • Enhancements in the Unbend command
  • Enhancements in the Flange command

The new Vertex Rip command

The new InsertSheet MetalVertex Rip command has been introduced in this version that enables you to create rips on the selected vertex of a solid, as shown in the images below.


Enhancement in the Unbend command

Starting from this version, it is possible to perform the unbend operation on the doubly curved surfaces by using the Unbend command.

Enhancement in the Flange command

The Keep inner base and Keep outer base are two new options introduced in the Alignment drop-down list of the Flange selection list. They enable you to maintain the constant length of the base (face on which the flange command is applied), as shown in the images below. The red dashed line in the following two images represent the internal and external length of the base, respectively.
