The new Frame functionality

Version 2009.1 introduces the new Frame functionality which is a fully integrated module of ThinkDesign. The principle approach for Frame is an instinctive sketch based method that facilitates in the creation of frame elements. The process of frames involves covering (linear sweeping) the predefined 2D or 3D profile with frame elements consisting of linear beams and nodes.

Frame elements compose to form a complete frame structure. These frame elements can be solid or hollow parts with pre-defined cross section.


Using the Frame Parts command you can create frame elements with the option to choose the unit of measurement and the structural cross section. These structural cross sections can be selected from the Sections drop-down list in the selection list and can also be sourced from the thinkparts catalog. The intersection of one or more frame elements is managed by the node functionality, which enables you to select and modify the nodes.

Also, you can modify each frame element by selecting different cross section parameters and alignment using the Modify Frame Part command. All the structural cross-sections are managed by thinkparts. Hence the modification or creation of new structural cross sections has to be done by updating the Frame catalogs using the thinkparts - Catalog Manager].

The Frame module simplifies the process of creating frame structures with easy-to-use but powerful and flexible tools to achieve significant reduction in the designing time.
