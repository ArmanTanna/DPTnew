New Curve & Surface Functionalities and Improvements

Version 2009.1 introduces the following enhancements in the Curve & Surface commands.

  • Enhancement in the Rotational Sweep command
  • Enhancement in the Drafter command
  • The new Mirror mode for curve and surface editing
  • Smart Delete applicable on COS
  • The new Check Design Differences command
  • Enhanced Section Matching during Global and Rotational sweep
  • Enhancement in the Project Curve command
  • Enhancement in Quality Checks
  • Enhancement in the Offset Curve on Surface command
  • Enhancement in Curvature Plotting within the Insert Curve through Interpolation Points command
  • Enhancement in HQ Reverse Engineering
    • Enhancement in the Mesh Analyzer command

Enhancement in the Rotational Sweep command

The Rotational Sweep command is now associative and hence the Associative Mode check box has been introduced in the selection list. Also, the powerful GSM engine is now associated with this command and can be activated by selecting GSM in the Scaling drop-down list under the Drive Curves node. For GSM scaling transformation, the Flexibility drop-down list is available enabling you to manage the shape of the controlled modification.

Optionally, you can select None in the Scaling drop-down list to allow the use of the drive curve only to control the rotational motion, without having any deformation effect.


Enhancement in the Drafter command

A new Transition check box has been introduced in the Drafter selection list to manage situations where the drive surface is G1 continuous but both drive surfaces and drive curves are not G1-continuous internally. On selecting this check box, some transition surfaces are generated to ensure complete surface continuity.

Also, the Drafter command is now associative, thus the Associative Mode check box has been introduced in the selection list.


The new Mirror mode for curve and surface editing

The new Mirror mode check box has been introduced in the Curve through Control Points and Surface through Control Points selection lists. It enables you to maintain continuity of the curve/surface being edited with respect to the virtual mirror of the entity. Additional options under the Mirror mode node give you the full control to define the type of continuity (Position, Tangent or Smooth curvature) to be maintained about a user selected plane.


Smart Delete applicable on COS

With this improvement you will be able to use the Smart Delete command to delete Curve on Surface (COS) curves too.


The new Check Design Differences command

Check Design Differences is a new info command that enables you to compare two (nearly identical) skins for finding small differences between them. It can typically be used to detect presence of any relative bulge, compute distances between areas having differences or find the modifications applied over the skins.


Enhanced Section Matching during Global and Rotational sweep

It is now possible to achieve transversal G1 continuity of surfaces being produced from section curves having same number of G1 continuous spans while preserving their correspondence. To facilitate this, the All discontinuities with reparameterization option has been introduced in the Matching type drop-down list under the Section Matching node. This new option is analogous to the All discontinuities options.


Enhancement in the Project Curve command

The new Thru all check box has been introduced under More Options in the Project Curve selection list. It is available when the projection Method is selected as Pierce direction or View direction. On selecting this check box, the curve will be projected onto all the underlying surfaces, including on the parts of the surfaces below the silhouette line in the projection direction.


Enhancement in Quality Checks

The Curvature signs tab has been introduced in the Quality Checks dialog box.


Enhancement in the Offset Curve on Surface command

With the improvements in this version the curves with bad continuity are better managed for the offset operation performed by the Offset Curve on Surface command. On using the Use inferred surface check box you can achieve better continuity of the offset curve. In case of a gap (G0 discontinuity) occurs between two parts of the resulting offset curve (due to the configuration of the original curves), selecting the new Blend angle check box obtains a smooth transition.


Enhancement in Curvature Plotting within the Insert Curve through Interpolation Points command

On starting the Curvature command, when the Insert Curve through Interpolation Points command is still running, will henceforth display additional options in the selection list. In previous version the Analysis dialog box used to be displayed instead. These additional options are the same as that of the Curvature selection list. This new behavior is the same as that observed with the Insert Curve through Control Points command.


Enhancement in HQ Reverse Engineering

The following enhancement has been made in the commands of HQ Reverse Engineering.

Enhancement in the Mesh Analyzer command

The new Show curvature variation option has been introduced for the Mode drop-down list in the Mesh Analyzer selection list. It enables you to visualize the presence of variation in the curvature of the mesh. The variation is computed by analyzing the curvature at one point with respect to the curvature computed on its neighboring points.

The new Copy check box has been introduced under More Options. It is available when the Mode is set to Add normal to mesh.

  • When selected, the modifications are applied on the copy of the original mesh. Hence, the original mesh remains unaffected.
  • When cleared, the modifications are applied on the original mesh itself and no new mesh is created.

Note: The Add normal to mesh option was formerly known as Insert Grid with Normal.
