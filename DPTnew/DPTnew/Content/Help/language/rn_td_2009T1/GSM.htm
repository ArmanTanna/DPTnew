New Global Modeling Functionalities and Improvements

Version 2009.1 introduces the following enhancements in the Global Modeling commands.

  • Improvement in the Advanced GSM command
  • The new GSM Fillet Reduction command
  • Enhancement in Quality Checks
  • Enhancement in GSM Transformation of Mesh
  • Enhancement in the GSM Bend and GSM Twist commands

Improvement in the Advanced GSM command

With the improvements made in the Advanced GSM command, the specified constraints for Matching and Preserving conditions are more closely achieved, resulting in better surface continuity.

The new GSM Fillet Reduction command

This version introduces a new GSM command called GSM Fillet Reduction for Die Design.


Enhancement in Quality Checks

The Curvature signs tab has been introduced in the Quality Checks dialog box.


Enhancement in GSM Transformation of Mesh

Previously, while applying the GSM commands on a mesh, it was not possible to modify the original mesh and hence the Copy option had to be selected to apply the command. With the improvements made in this version, it is now possible to modify the original mesh too.


Enhancement in the GSM Bend and GSM Twist commands

This version has improved the management of the results when applying GSM Bend & Twist commands (including Radial Bend and Replicate commands) to curves in associative mode. The result is no longer a joined NURBS curve, and instead, it is in form of individual deformed segments. This allows to replicate the deformation, as well as, to delete only parts of the result without deleting all. Also, the result is represented as MRE (Multiple Resulting Events) in the Model Structure.
