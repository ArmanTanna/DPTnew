New PARTsolutions plug-in Functionalities and Improvements

Version 2009.1 introduces the following enhancements and improvements to the PARTsolutions plug-in functionalities.

  • Improvements to the import commands

Improvements to the import commands

The Import from PARTdataManager Catalog and Import from PARTdataManager File commands benefit from the new Placement option in the selection list.


The selection list now provides you with the following placement methods:

  • Free
    This only requires you to select the insertion point of the part. After you have inserted the part, the program displays the positioning handles that enable you to rotate it and move it at will.
  • Assisted
    This is available only for those parts (for example screws and nuts) that have the data to assist you in selecting the elements necessary to the correct placement in the model. When this method is active, the selection list displays the Face and Hole/Axis selectors that assist you in placing the component.

If you use the Assisted placement method, when you later use Replace Object, the program uses the same constraints for the placement of the new part.
