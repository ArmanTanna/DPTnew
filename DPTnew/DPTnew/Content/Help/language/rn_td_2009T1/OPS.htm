Enhancements in the System Options

The new Startup category

This version introduces a new Startup category. The controls in this category enable you to customize the application startup by showing the dialog box or start with no document, an empty model or an empty drawing.


Enhancement in the Construction category

The new Solid color area has been introduced to support the enhancement in color propagation. The various radio buttons enable you to define the color of the solid generated after a feature creation operation.


Note
The default selected option is First entity color. Hence, while performing a Boolean operation, the resulting solid will be of the same color as that of the solid which has been selected first.

Enhancement in the General - Advanced category

The new Log file management area controls have been introduced in this version to enable you to make settings for configuring the Automatic Logger to send information from your system.


The new General - Backup/Recover category

A new Backup/Recover category has been introduced in this version. The different options under this category are explicitly developed to create and maintain back-up files of the model/drawing. The new mechanism creates a copy at each save made by the user. To avoid overwriting the backup file with the last version that could be already corrupted, a new backup copy is created for each save performed on a file.

Henceforth, the AutoSave area is available in the Backup/Recover category.


The new Output - Tooltip category

The Tooltip category that collects the tooltip options for components and balloons is now available as a subcategory of the Output category.


Enhancement in the Input category

The new Select through transparency check box has been introduced in the Input category. When you move the cursor onto a transparent entity (material or color) then the objects beyond the transparent entity are selected.


Enhancement in the Assembly/Shared Group category

The Assign part number as name check box has been added to this category:

  • When selected, the component name in the document structure is changed to reflect the part number you set in the component or group Part Properties.
  • When cleared, the component or group name remains unchanged.

Enhancement in the Model Structure category

The Show part number and description check box has been added to this category:

  • When selected, the program displays in the structure tree the Part Name and Description that you enter when you edit the Part Properties of the component or group.
  • When cleared, the program displays the component or group name.

Enhancements in the Document Properties

Enhancement in the General category

In the Component area, the Show closed components with component color check box has been introduced to support the enhancement in color propagation. When selected, all elements inside the component are displayed in the component color. Otherwise, the elements are displayed in their individual colors.

The new Profile area has been introduced with the Show profiles with profile properties check box.

  • When cleared, it is possible to change the attributes (color, line type and line width) of entities (curves) inside a profile. After exiting from the profile mode, the entities of the profile retain the applied curve attributes.
  • When selected, attributes of the profile entities cannot be modified inside the profile mode. After exiting from the profile mode, the entities belonging to the profile inherit the profile attributes.

The new Ground category

The new Ground category enables you to control the options that set the shadow and planar reflection of the model on the ground plane.


Enhancements in the Entity Properties

Enhancement in the Drawing View - View Attributes category

The new Inherit attributes area has been introduced with the following two new radio buttons to support the enhancement in color propagation.


Solid color When selected, the edges (belonging to individual solids) in the view will be displayed in their own solid color.
Closed component color When selected, the whole view will be displayed in the color of the closed component, overriding the solids colors.

Note
  • The controls in the new Inherit attributes area are activated only when Keep (which was previously called Keep model attributes) check box is selected.

  • If the view is closed and the Show closed groups with group color check box is selected, then you have to open the view or set it as current to see the inherited colors of the edges.

Enhancement in the Drawing View - Break Lines category

This version introduces the new Inherit breaks from main to derived view check box that enables you to inherit multiple breaks defined on the main view onto the derived views. For this reason the new option, which has been provided in the Entity Properties - Drawing View - Break Lines category of the Options/Properties dialog box is available only for the derived views.

Enhancements in the Document Properties

Enhancement in the General category

In the previous versions, it was possible only to assign colors to solids and not to its individual faces. Hence, when the Edit Properties dialog box was started by selecting a face of the solid, the Color option in the General category was grayed out.

With the enhancement made in this version, you are enabled to apply an explicit color to a face of a solid. To do so, you need to right-click on the face and select Edit Properties from the context menu to display the Edit Properties dialog box. Next, select the General category and click the down arrow on the right side of the Color option to display the color pallet. Select the required color from the palette and choose the OK button to assign the color.

Also, it is now possible to retain color of a face while the solid is exported to another format.

This new behavior has been introduced to support the enhancement in color propagation.
