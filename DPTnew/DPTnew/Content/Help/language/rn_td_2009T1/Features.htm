New Part Modeling Functionalities and Improvements

Version 2009.1 has following enhancements to the Part Modeling functionalities.

  • Enhancement in the Generic Sweep functionalities
  • The new Hole command
  • Simple Shaft command renamed
  • Enhancement in the Solid Difference command (Boolean Difference)
  • Improvement in the Draft Angle command
  • Enhancement in the Extend Faces/Close Solid command (Interactive Modeling)

Enhancement in the Generic Sweep functionalities

The Motion Mode drop-down list is now available in the selection lists of the Generic Solid, Generic Protrusion, and Generic Slot commands, which proposes a number of different methods to perform the sweep along the drive. Also, a new Skin check box has been introduced in the Generic Slot selection list.

Henceforth, it is possible to create generic sweep (Generic Solid, Generic Protrusion, and Generic Slot) along a non-tangential spine to boundaries, as shown in the images below.


The other improvements in the generic sweep are listed below:

  • It is no longer mandatory to position the first boundary on the spine endpoint.
  • The boundary profiles can now have different number of entities with respect to each other.
  • Now, it is possible to redefine the generic solid features.

The new Hole command

In this version, the new Hole command replaces the previously existing Simple Hole command.

The new For screw check box is now available in the command selection list. It can be used for creating and redefining the standard, simple, shaped, and free dimensional holes. The options which were previously available in the Hole Catalog dialog box of the Shaped Hole command are now available under the For screw node of the new Hole command. As a result the Shaped Hole command and the Thread check box in the Simple Hole selection list are no longer available.

Standard values for creating the holes are taken from the predefined table that can be modified by the user from the installation folder. On clearing the For screw check box you are enabled to explicitly define the dimensions of the hole.

Simple Shaft command renamed

The Simple Shaft command has been renamed as the Shaft command.

Enhancement in the Solid Difference command (Boolean Difference)

In order to increase the capability to retain the original solids during the Solid Difference operation, the Keep Original Solids check box has been replaced by the Keep Original Solids drop-down list with three different options: None, Solids B, and All.

Improvement in the Draft Angle command

The Draft Angle command is now much more reliable and can be applied to a much higher number of situations than in the past.

Enhancement in the Extend Faces/Close Solid command (Interactive Modeling)

A new Insert New Faces check box has been introduced in the Extend Faces/Close Solid command selection list.

  • When the check box is selected, new faces are created to fill the selected holes, as shown in the image below.
  • When the check box is cleared, no new face is created to fill the selected holes.
