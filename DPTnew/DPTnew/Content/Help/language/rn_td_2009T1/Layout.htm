New Layout Functionalities and Improvements

Version 2009.1 introduces the following enhancements to the Layout commands.

  • Inheriting breaks from multi-broken views
  • Bisector command associativity
  • New Radial Axes and Circular lines for angular pattern

Inheriting breaks from multi-broken views

A new Inherit breaks from main to derived view option has been introduced in the Entity Properties - Drawing View - Break Lines category of the Options/Properties dialog box to inherit breaks from multi-broken views.


Bisector command associativity

Starting from this version, the entities inserted in the drawing view using the Bisector command have been made associative to the reference model geometry by integrating them with the associativity mechanisms. As a result, any modification done on the model geometry will be propagated to the bisector lines in the drawing view during regeneration.


New Radial Axes and Circular lines for angular pattern

Enhancements have been made in the Centerlines command to create the circular lines and radial axes for patterned holes. To facilitate this, the new Mode drop-down list has been introduced in the Centerlines selection list. The two options present in it are:

  • Axes on circular pattern - It displays two check boxes:
    • Show circular centerline
    • Show radial axes
    Also, the Pattern Center selector is available.
  • Axes on hole - Centerlines are created only on the holes

The application remembers the last mode of axis creation, and hence, the next time you start the command, the same mode is used, unless changed. For an example of use, see "Creating Radial Axes and Circular Lines for Angular Patterns".
