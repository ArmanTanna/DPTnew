New Part Properties Functionalities and Improvements

Version 2009.1 introduces the following enhancements and improvements to the Part Properties functionalities for ThinkDesign users only.

  • Enhancement in the Properties command (File)

Abstract

The set of commands formerly known as Part Data, is now named Edit Part Properties.


Most functionalities have remained the same, but the new commands no longer require other software to be installed besides the standard installation.


To complete the reorganization, some options have been moved to new locations, new commands have been added and some commands have been improved. See "An Overview on Part Properties and Document Data" for details.


Enhancement in the Properties command (File)

The Edit Part Properties tab has been added to the FileProperties command. Its behavior is similar to the Document Properties Edit command.


The new Part Properties commands section

The main commands used to manage the Part Properties (Edit, Copy and Paste), are now consistently collected in the Edit Part Properties section of the Edit menu and of the entity context menu.


These commands enable you to edit the Part Properties of the current component. If no component is selected, a selector in the selection list enables you to select it. To quickly access the top level component Part Properties in a layout or in a 3D assembly model, you can also use the new Edit Part Properties tab of the File Properties command.


The new Item Position Property command

The new InsertItem Position Property command replaces the former Define sequence command and enables you to define the component sequence. This information is used by the ballooning and part list commands.

The selection list enables you to quickly set the numbering criteria.


This command also provides the same feature formerly available in the Edit Relation command.


The new name for Highlight All Parts commands

The former Show All Parts and Show All Parts without Balloon commands have been renamed respectively Highlight All Parts and Highlight All Parts without Balloon.


The new Quantity Property command

The new ModifyQuantity Property command enables you to set the quantity of a component within an assembly.


This command provides the same feature formerly available in the Edit Relation command.


The new Organize Part List command

The new FormatOrganizePart List command enables you to define the default properties of the Part List.


This new command replaces the former way of configuring the Part List based on the "desc.lis" and other files. This integrated tool enables you to easily define the Part List content (Columns, header and footer), graphical aspect (Colors, text fonts, size), behavior (Sorting, origin, hierarchy).


Note
Old configurations need to be recreated using the new command.

Improvements to the Part List command

The ToolsPart List command has been improved with a new selection list and the integration with the Organize Part List command.


Relocated from the former position under InsertDrawing Details, the command now features a new selection list that provides you with full access to the Part List options and a Mode option that enables you to insert the Part List in the drawing or save it to a ".csv" file for later perusal.
The usual controls of origin, sorting and hierarchy are still available and quickly accessible.


The Part List command is also present in the Model environment, with the only option of saving the Part List to a ".csv" file.


The new Part List management commands

The following new commands have been added to manage the Part List after it has been inserted in the drawing with the Part List command:

  • Update
  • Redefine

The commands are present in the menu bar and in the Part List context menu.


The new XML Editor

XML Editor is a new program that enables you to customize the dictionaries that you use to fill the Part Properties and Document Data fields.
