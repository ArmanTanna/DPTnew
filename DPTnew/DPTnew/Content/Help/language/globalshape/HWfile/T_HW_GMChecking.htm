Checking your results

As soon as a preview is displayed, you can check the results you obtained. You can:
 

  • Display warnings on the model
  • Set the precision values to be achieved
  • Set the parameters controlling the shape of the resulting surface
  • Perform a set of Quality Checks

Setting the precision values to be achieved

To define the precision values to be achieved during the computation, you can click the Precision under More Options to open the Precision dialog box:



inside which you can modify the following parameters:
 
No. loops The number of iterations the programs must perform in order to achieve the desired precision. Increase this number to get a higher precision.
No. points The number of points to be used by the program in order to achieve the desired precision. Increase this number to get a higher precision.
Position The precision value to be achieved for positional continuity, in millimeters or inches, depending on the current unit of measure. It is the gap between the resulting surfaces and the constraints.
Tangency The precision value to be achieved for tangency continuity, in degrees. It is the difference between the real tangency and the one you obtained.
Curvature The precision value to be achieved for curvature continuity, as a percentage of the maximum curvature.
Global When the check box is not selected, a set of local tolerance values is attached to each group, as in the following illustration.



To hide the local precision values, select the check box.

If a warning is displayed about some precision value, proceed as follows:
 
1 Increase the value of the No. loops and/or No. points parameters (or decrease the precision to be achieved, if this makes sense).
2 Once you modified the values, click the Preview button ( ).

The model will be rebuilt using the new values and a progress bar will show the completion percentage of the process. When the desired result is achieved, the button (Show warnings) disappears.

As to how to manage precisions when rebuilding the model, see Precision management in Associative GSM.

 

Controlling the shape of the resulting surface

You can set the values of the parameters controlling the shape of the resulting surface: stiffness, roundness and bulge. Click Shape Control under More Options to display the Shape Control dialog box:



  • X stiffness — enables you to set the stiffness value (strictly positive) along the X axis
  • Y stiffness — enables you to set the stiffness value (strictly positive) along the Y axis
  • Z stiffness — enables you to set the stiffness value (strictly positive) along the Z axis
    Stiffness affects results only when constrained entities are very distant from the ones to be modified. Click here to see an example.
     
  • Roundness
    Roundness (strictly positive) mostly affects changes regarding points: when a surface is modified so that it will pass through a point, around that point the surface itself can be more or less "round". Click here to see an example.
    • By selecting the Global check box, you can apply the same roundness to all groups. When it is not selected, a different roundness value can be defined for each group, using the corresponding mini-dialog box
    • By selecting the Optimization check box you can obtain much more accurate results, though execution can take more time.

     
  • Bulge — Enbales you to choose one of three possible values: Small, Medium, Big and Very big.
    Click here to see an example of how the bulge factor can affect objects.
