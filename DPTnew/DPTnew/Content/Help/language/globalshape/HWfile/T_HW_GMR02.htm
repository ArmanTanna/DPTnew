Replicating a GSM Bend and a GSM Twist

In this task you will apply a controlled modification to a solid by performing on it the same actions already applied to a surface (using the GSM Bend and the GSM Twist commands). To accomplish this task, open file "GlobalReplicate_02.e3"

To obtain this model, a GSM Bend command was applied first, and then a GSM Twist.

Start the GSM Replicate command and proceed as follows.



1 In the Deformed reference entity drop-down list, select Surface. Then click on the surface on the right, as in the following illustration:
2 To define the level of the controlled modifications to apply (for example: 1 - only last one; 2 - last one and the one before ... and so on), first click on More Options to expand it, then select the Copy box and choose 1 GSM Twist both in the From level and in the To level drop-down lists.
3 Select the solid you want to modify.
In the Entities drop-down list (under Entities to Be Modified) select Solid, then click on the solid.
4 To see a preview, select the Preview ( ) button.
5 Now, try to see what happens if you apply the first of the two controlled modifications instead of the last one. In the From level and To level drop-down lists select 2 GSM Bend (which is the controlled modification applied first, so its number is the greatest in the list). Then select the Preview ( ) button.
6 Finally, to apply both controlled modifications, select 1 GSM Twist in the From level drop-down list and 2 GSM Bend in the To level drop-down list.
7 Click or to confirm your selections modify the shape of the object.



Click to discard your changes.
