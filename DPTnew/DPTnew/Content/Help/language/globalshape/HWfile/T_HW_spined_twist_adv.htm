Creating a Spined Twist Surface (Advanced)

In this tutorial you will apply a GSM spined twist to a surface after reorienting the section planes. Open the Spined_Twist_Adv.e3 file available in the Samples folder of your think3 installation and follow the steps given below.

  1. Start the GSM Spined Twist command and select the green curve for the Spine Curve selector.

  2. Expand the More Options node and select Advanced Options; the Advanced Options dialog box is displayed. Set the default values by selecting the Default button. Select the Show preview check box and finally select the Update Now button. Select Yes for the displayed warning to proceed.

  3. Press the F8 key to set the Work Plane parallel to the view. The model and the section planes will appear as shown in the image below.



  4. As the punch would be moving vertically down to stamp this part, set the Molding Direction = Z in the drop-down list. The section planes get reoriented such that their X-direction is aligned to the Z-axis of the Work Plane. Therefore, the section planes appear as straight lines.

  5. Right-click on the handle of the 2nd section plane from the right. Select Remove Section from the context menu. Repeat the same with the 4th section plane from the right.

  6. Right-click on the handle of the 2nd section plane from the left and select Enable Drive Section from the context menu. Repeat the same with the 4th section plane from the right. Now you are enabled to set the Angle for this section too. This way you will remove two section planes from the part.

  7. Set the angle values in the Angle mini-dialog boxes, as shown in the image below.



  8. Select all the surfaces of the model for the Surfaces to Be Sectioned selector. Ensure that the Include surfaces to be sectioned check box under the Entities to Be Modified node is selected. This will ensure that the twist is applied to all the selected surfaces.

  9. Clear the Hide and Copy check boxes under More Options and Apply () the command. The original and the twisted surface are shown below.
