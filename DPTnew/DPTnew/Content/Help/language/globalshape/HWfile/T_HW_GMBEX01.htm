Bending a surface changing the continuity conditions at the end points of the bending line

In this task you will bend a surface and change its continuity conditions at the end points of the bending line. You will appreciate how changes are propagated throughout the surface, beyond the area covered by the bending line. To accomplish this task, open file "bend_continuity.e3".

1 Preselect the surface (see also Selection and Preselection).



Start the GSM Bend command.
To create a solid which will retain an associativity link to its base entities, so that if you modify such entities the solid will be updated accordingly, select the Associative Mode check box. In this task, the Associative Mode check box has not been selected.

2 To define the bend line, select 2 Points in the Bend line drop-down list. Click on the two points as shown in the following illustration:



Three handles are displayed.
  • The one corresponding to 2nd Length enables you to modify the bending line
  • The ones corresponding to Angle and Length enable you to perform the bending in a polar reference system with its origin in the endpoint of the bending line.

3 Set constraints at the first end point of the bending line. First of all, click More Options to expand it. Under First:
  • In the Status drop-down list, select Fixed, in order to prevent the surface from moving from the first end point of the bending line.
  • In the Continuity drop-down list, select Curvature, in order to apply curvature continuity about the first end point of the bending line.

4 Drag the handle of the polar axis to the new position (or set the Length value).
5 Set constraints at the second end point of the bending line. Under Second:
  • In the Status drop-down list, select Free, in order to enable the surface to move from the second end point of the bending line.
  • In the Continuity drop-down list, select Curvature, in order to apply curvature continuity about the second end point of the bending line.

6 Click or to confirm your selections modify the shape of the object.



If you set the Copy option, you will obtain the following:



Click to discard your changes.
