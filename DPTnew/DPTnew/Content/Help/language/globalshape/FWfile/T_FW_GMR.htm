ModifyGlobal Shape Modeling Replicate

The GSM Replicate command enables you to modify the shape of surfaces and solids by performing on them the same controlled modifications ( GSM Bend, GSM Twist, Advanced GSM) already applied to another entity, to be selected as a reference.

First, you will have to select the Deformed reference entity (surface or curve) and then the Entities to Be Modified (solids, surfaces, curves, points or meshes — see also " Display information about a selected Mesh" for further details on mesh entities).

If you applied more than one GSM controlled modification to an object, the GSM Replicate command also enables you to select which "level" of the controlled modification to replicate on the selected entities.

A set of Quality Checks commands is also available.

The Associative Mode check box enables you to create relations between the original objects and the modified ones, exactly as in the case of the Advanced GSM command.

The Ratio check box under More Options enables you to apply a scaling when using the command.

  • When the box is checked, a scale value can be used when applying the controlled modification, if only matching constraints were involved in the reference modification. A box is displayed adjacent to the check box where you can type the scale value which will be applied to the selected entities along with the reference modification.
  • When the box is cleared, no scale factor will be applied to the selected entities.
