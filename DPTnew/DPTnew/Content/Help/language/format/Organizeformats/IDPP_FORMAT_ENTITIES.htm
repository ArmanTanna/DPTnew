General Format - General

Note
All the three options in this page, apart from Layer, are enabled only when the check boxes associated with them are selected.

Layer Enables you to specify the layer on which entities are drawn. You can use up to 1024 layers (numbered from 0 to 1023), by just assigning a particular alphanumeric value in the text box. You have the option to either turnoff or turn on a layer, using the check box provided along with it. Each layer can have its own description, so that it will enable you to choose the appropriate layers, when you have many of them.
Color Enables you to either set the color for the entities to be drawn or modify the color of a selected entity.
Line type Enables you to either set the line type for the entities to be drawn or modify the line type of a selected entity.
Width Enables you to set the line width for the entities to be drawn.
