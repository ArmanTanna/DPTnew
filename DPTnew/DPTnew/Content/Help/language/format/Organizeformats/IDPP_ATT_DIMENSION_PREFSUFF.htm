Dimension Format - Prefix/Suffix

Note
The Text and the Position options are enabled only when the checkboxes associated with them are selected.

Prefix Enables you to specify the Prefix before the numerical value of every dimension. You can enter the Text, that acts as the Prefix for the dimension, in the input box provided below the option. After entering the text, you can specify the Position as well, choosing one of the following options:

Left When selected, the prefix is placed to the left of the dimension
Above When selected, the prefix is placed above the dimension

The Preview area, provided on the right hand top portion of the dialog box, enables you to have an updated view and also control the effect of the modifications.
Suffix Enables you to specify the Suffix after the numerical value of every dimension. You can enter the Text, that acts as the Suffix for the dimension, in the input box provided below the option. After entering the text, you can specify the Position as well, choosing one of the following options:

Right When selected, the Suffix is placed to the right of the dimension
Below When selected, the Suffix is placed below the dimension

The Preview area, provided on the right hand top portion of the dialog box, enables you to have an updated view and also control the effect of the modifications.
