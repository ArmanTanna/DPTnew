Balloon Format - General

Note
All the options in this page are enabled only when the checkboxes associated with them are selected.

Text: Enables you to enter the specialized text for the Balloons that you create. See How to customize the balloon label for more details.
Frame: Select from the drop-down list the type of frame for the bBalloons that you create.
Arrowhead: Select from the drop-down list the specialized arrowhead type for the Balloons that you create.
