Dimension Format - Tolerances

Note
All the options in this page are enabled only when the checkboxes associated with them are selected.

Tolerances Enables you to determine the format for tolerances included as part of the dimension. The preview area, present on the right hand top portion of this dialog box enables you to have an updated view and also control the changes made accordingly.

Representation: Enables you to determine the type of tolerance included as part of the dimension. You can choose from any one of the following representations:

+/- Enables you to represent the tolerance value using a single numerical value
Two Values Enables you to represent the tolerance value using upper and lower values, which are specified in the Value Expression category
Limits Enables you to represent the tolerance value using upper and lower limit values, which are specified using the Value Expression category
Basic Enables you to represent the tolerance value inside a box


Add parentheses: Enables you to specify whether parentheses have to be included along with the tolerances. You can choose from one of the following options:

Never Enables you to represent the tolerance value without the paranthesis
with tolerance label Enables you to represent the tolerance value using the paranthesis along with the tolerance label
Always Enables you to represent the tolerance value with the paranthesis always


Tolerance label: Enables you to specify one of the ISO tolerance symbol labels specified in the Isoq.dat and Isos.dat files to automatically calculate and add the specified tolerance value to the dimension.
Limit arrangement: Enables you to represent the tolerance value, having upper and lower limit values in two different ways. The options that are available in the dropdown list are:

Vertical Enables you to represent the tolerance limit values in the vertical form
Horizontal Enables you to represent the tolerance limit values in the horizontal form
