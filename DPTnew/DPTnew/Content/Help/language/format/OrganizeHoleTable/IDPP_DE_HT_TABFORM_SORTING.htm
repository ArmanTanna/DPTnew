Sorting

This category enables you to manage sorting criteria for the Hole Table parameters.

Type

Type:

Enables you to choose an ordering option for the sorting criteria.

Ascending Sorts the parameters in ascending order.
Descending Sorts the parameters in descending order.

Diameter

When not selected, it takes the Diameter, X, and Y as the default sorting criteria.
