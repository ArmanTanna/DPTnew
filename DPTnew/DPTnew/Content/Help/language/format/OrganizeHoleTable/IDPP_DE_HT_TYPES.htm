Holes

Holes

The parameters in this area enables you to customize the hole types.

Type

Type:

Enables you to choose a hole type from the drop-down list. The list items are available based on the configuration file.

Description

Description:

Enables you to customize the content to be displayed in the Description column. You can also use symbolic text in this field. The only specific valid symbolic texts that can be recognized by the application are <D1> and <D2>, which represent the primary and secondary diameter of the Holes.

Marker label

Marker label:

Enables you to specify a specific text input to be used along with the marker for the Hole Type. This text will be concatenated to the original marker to distinguish the Hole Type.
