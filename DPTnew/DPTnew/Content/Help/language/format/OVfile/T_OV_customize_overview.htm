Customization: An overview

As a matter of personal choice, due to specific work requirements or to endorse company-wide settings, you may want to customize think3 application. For example, you can create and customize toolbars and use them in the same way as think3 application's default toolbars. You can set the GUI profile (look and feel of the software) of the program that best suits an industrial design work or set it more productive for a machine design work and so on.

By doing the customization you fine tune think3 application and make it look and work the way you want it to be. There are many ways to perform customizations for the think3 application. One easy way is to use the settings wizard that is shipped with the program. Or, you can always manually set each and every detail for the customization using the system / document properties.
