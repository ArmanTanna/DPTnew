Document Properties

Category: Grid

Static grid

Enable

Enable
When selected, the Static Grid is activated. If the grid is active, when indicating positions using the mouse, the cursor automatically snaps to the nearest point on the grid.
When not selected, the grid is off.

Type

Type:

From this drop-down list, select the type of Static Grid to be displayed.

  • Default : If selected, the Standard Grid is displayed as the Static Grid.
  • Reference: If selected, the Reference Grid is displayed as the Static Grid.

Step

Step:

Determines the size of each grid block in the current unit of measurement for the Static Grid:

1 - Major line subdivisions:
2 - Step:

Major line subdivisions

Major line subdivisions:

Determines the number of grids, displayed between the major (solid) lines based on the following values:

0 the grid appears as dots spaced according to the Step: value.
1 the grid appears as lines spaced according to the Step: value.
Greater than 1 the grid appears as dots spaced according to the Step: value, with lines appearing at specified multiples of the grid.

Hidden dynamic grid

Enable

Enable

When checked, the dynamic grid limiting the possible values to the ones defined by the Linear step and the Angular step is enabled during the input operations.

When not checked, the dynamic grid is not enabled.

See Hidden Dynamic Grid for further details.

Flexible step

Flexible step

When checked, intermediate positions between two successive nodes are enabled (your mouse input points will be snapped to the nearest node of the grid only if the actual position you indicated is inside the Snap trap).

If not checked, only positions corresponding to step values are enabled (your mouse input points will always be snapped to the nearest node of the virtual grid).

See Hidden Dynamic Grid for further details.

Linear step

Linear step:

Enables you to define the value of the grid linear step (i.e.: the distance between two consecutive nodes of the grid).


Note
  • If the Flexible step check box is not selected, your mouse input points will always be snapped to the nearest node of the virtual grid.
  • If the Flexible step check box is selected, your mouse input points will be snapped to the nearest node of the grid only if the actual position you indicated is inside the Snap trap.

See Hidden Dynamic Grid for further details.

Angular step

Angular step:

Enables you to define the value of the grid angular step.


Note
  • If the Flexible step check box is not selected, your mouse input points will always be snapped to the nearest position of the virtual angular grid.
  • If the Flexible step check box is selected, your mouse input points will be snapped to the nearest position of the grid only if the actual position you indicated is inside the Snap trap.

See Hidden Dynamic Grid for further details.

Parametric step

Parametric step:

Enables you to define the value of the grid parametric step, used whenever parametric values not related to the current unit of measure are involved, such as curve and surface parameters (for example, when you extend a surface by modifying its parameters using the Parameter mode) and scale factors (for example, when you scale entities using the Scale Entities command).

The Parametric step enables you to limit your mouse input actions and the values which can be assigned to mini-dialog boxes to a set of predefined ones at fixed intervals.
