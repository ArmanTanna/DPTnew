Category: Drawing View-Sectioned-X-Hatch

X-Hatch

X-Hatch

The controls in this area enable you to define the settings of the X-Hatches representing the sectioned faces.

On sectioned faces

On sectioned faces

When selected, X-hatches are drawn on sectioned faces.
When not selected, no X-hatch is drawn on sectioned faces.

On sectioned surfaces and open solids

On sectioned surfaces and open solids

When selected, enables the computation of cross-hatching inside closed loops resulting from sectioning a surface or open solid.
When not selected, the said computation is not enabled.

Different offsets on adjacent faces

Different offsets on adjacent faces

When selected, parallel X-hatches on adjacent faces of the same solid are displayed with an offset (1).
When not selected, no offset is displayed (2).

Automatic angle and spacing

Automatic angle and spacing

The controls in this area enable you to define the automatic angle and spacing settings of the X-Hatches.

Enable

Enable

When selected, enables to apply X-Hatch spacing and angle for sectional views. The optimal spacing value for the X-Hatch, based on sectioned face area and shape will be automatically applied for each X-Hatch in the view. Angles will be 450 or 1350.
When not selected, the program will use the current X-Hatch settings for all the sectioned faces of the view.

Different x-hatches on adjacent solids

Different x-hatches on adjacent solids

When selected, enables an advanced proximity check so as to ensure that no adjacent X-Hatches with same angle and spacing exists in the view. This is achieved by inverting the angle value (450 or 1350) and increasing spacing density.
When not selected, and the Enable check box is checked in, then modification of spacing and angle value of X-hatches is done, without the proximity (adjacency) conflict check of adjacent solid sections.

Density

Density:

Determines the spacing density for the hatch lines.

Very low will result in hatch farther apart
Very high will result in closely packed hatch lines
