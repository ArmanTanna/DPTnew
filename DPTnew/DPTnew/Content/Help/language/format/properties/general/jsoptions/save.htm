Enables you to control the way the changes you made to the current style will be saved. A dialog box is displayed so as to ask you whether the changes you made are to be applied only to the current document or to the library as well.

  • Yes
    applies the changes to the current style and to the one with the same name in the library — if any — . If no style with the same name exists in the library yet, it will be created and the changes will be saved to it as well.
  • No
    the changes are saved only to the current document.
  • Cancel
    the save operation is aborted.
