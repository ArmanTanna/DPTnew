Document Properties

Category: Reference Grid

Coordinate system

Coordinate system

The controls in this area enable you to change the reference system.

Set to

Set to:

Enables to select the coordinate system to be used for the reference grid. From the drop-down list, choose one of the following:

World the Reference Grid aligns with the world origin.
Current work plane the Reference Grid aligns itself with the current orientation of the Work Plane.
Inherit from 3D model the Reference Grid aligns to the value chosen in the 3D models option (Set to: drop-down list inside Document Properties - Grid - Reference - Advanced category).

Update

Update

Forces reference system to update the Work Plane settings (which are done after setting it as the grid reference system).
The reference system defined using Set to option is used in all the reference grids of the drawing. If you change the reference system and then regenerate the view/drawing, the reference grids will be updated accordingly.

Origin

Origin:

Displays the World coordinates of the origin of the selected system in the three boxes.

Axes

Third axis:

Enables to redefine the names of the axes to be displayed in the labels of the reference grid. The World components of the axis vector are displayed in the Orientation (X,Y,Z) boxes.

First axis Enables you to rename the abscissa axis.
Second axis Enables you to rename the ordinate axis.
Third axis Enables you to rename the applicate (that is height or altitude) axis.

Name

Name

Enables to redefine the names of the axes to be displayed in the labels of the reference grid.

Orientation (X,Y,Z)

Orientation (X,Y,Z)

Displays the World components of the axis vector in the Orientation (X,Y,Z) boxes.
