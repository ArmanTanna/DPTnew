Document Properties

Category: Dimension

General

General

The controls in this area enable you to set the general parameters of dimensions.

Radial angular step

Radial angular step:

Determines the angular step used to insert radius dimensions. For example, if the angular step is alpha, you can insert radius dimensions only at angles that are multiples of alpha. When you indicate the positioning point for a radial dimension, it will be approximated to the nearest multiple of alpha.

Parallel dimensions distance

Parallel dimensions distance:

Determines the spacing between consecutive parallel dimensions.
 

.

The way the distance is computed depends on the status of the adjacent check box.

Show variable names

Show variable names:

Enables you to view variety of information on dimension text when the dimension depends on expression containing adaptive measure or spreadsheet variables:

  • Value —  the dimension shows the value (result of the expression evaluation)
  • Expression —  the dimension shows the expression or variable name
  • <Value> —  the dimension shows the value (result of the expression evaluation) inside angular brackets, so as to show that it is the result of an expression. This is the default value.

Out-of-scale dimensions

Out-of-scale dimensions

When selected, dimension text in a drawing is underlined when the dimension is entered manually (and accepted without rebuilding), to differentiate it from dimensions that are automatically calculated. After rebuilding the drawing, the dimension text will not be underlined anymore.

Remove unrelated dimensions

Remove unrelated dimensions

When selected, dimensions are automatically deleted when the associated entities are removed.
If the box is not selected, dimensions are not deleted when you remove the associated entities

Highlight dimension colors

Driving

Unrelated:

The controls in this area determine the colors used to highlight the different types of dimensions when using the Highlight Dimensions command. Typically, you can perform this command in order to easily visualize the unrelated dimensions and distinguish them from the reference ones. In fact, you can select different colors for each of the following types of dimensions:

Driving determines the color of parametric dimensions (dimensional constraints) of profiles as well as dimensions created automatically by the program during the generation of solids or during machining
Referencedetermines the color of non-parametric dimensions entered by the user
Unrelateddetermines the color of dimensions which have no relation to an entity
