Entity Properties

Category: Sheet Metal-Rip/Corner

Rip gap

Rip

The control in this area determines the gap value in rips.

By value

Gap:

Enables you to specify the gap value to be added between the two adjacent faces.

By thickness

Rip Gap: by thickness

Enables you to specify the value of the gap to be added between the two adjacent faces of the rip. The gap value is to specified in the edit box as the percentage of the Sheet Metal thickness.

Close corner

Close corner

The controls in this area enable you to set the parameters to be used in corner closures.

Include Bend

Include Bend

When this check box is selected, also the bend is processed when creating the corner closure; if the box is not selected the bend is not processed.

The original sheet metal part
Include bend NOT selected

Type:

Enables you to select the type of processing the bend has to undergo when the Include Bend box is selected:
