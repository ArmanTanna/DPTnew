When the check box is selected, you are enabled to create the detail view with the filter value inherited from the parent view.
When the box is not selected, you are enabled to specify a value with which the detail view can be created. If value is set to zero, you can create a detail view from a view with the filter enabled.

(See large assemblies for further information).

Note
  • The Value box to specify a filter value is active only if Inherit value from parent view box is not selected.
  • This field is available only in idle mode (when no view is selected).
