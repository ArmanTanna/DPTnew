When the check box is selected, by default all newly created views will display their own scale factor in the label. In Edit Properties, the Show view scale box will be checked by default (and you can uncheck it to turn off the visualization of the view scale factor).
When not selected, which is the default value, all newly created views will not display their own scale factor in the label, unless you check Show view scale in Edit Properties.


Note
If you load a drawing file (".e2") created with a previous version, the indications of scale factors in the view labels will not be changed (either in format or content). The new scale values will assume the default values.
