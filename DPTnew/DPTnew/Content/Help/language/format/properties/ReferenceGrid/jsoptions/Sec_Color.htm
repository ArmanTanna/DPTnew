Enables you to change the color of the circular frame (if any).
When you click on the arrow adjacent to the color box, the color palette is displayed, so as to enable you to make your choice.


Note
  • Since the reference grid is a group entity, the visualization of the internal color defined for the circular frame depends upon the value of the Show closed groups with group color check box in the Groups category of the Document Properties.
    To display the specific colors of the grid, that is the one of the circular frame, the one defined for major lines and the one defined for minor lines, the Show closed groups with group color check box must be cleared.
  • This applies only to the sectioned grid lines which are created and visible only if the Show sectioned lines check box is selected in the Reference Grid category.
