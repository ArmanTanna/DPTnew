Enables to select the coordinate system to be used for the reference grid. From the drop-down list, choose one of the following:

World the Reference Grid aligns with the world origin.
Current work plane the Reference Grid aligns itself with the current orientation of the Work Plane.
Inherit from 3D model the Reference Grid aligns to the value chosen in the 3D models option (Set to: drop-down list inside Document Properties - Grid - Reference - Advanced category).
