Enables you to specify the position of the axis name in the label with respect to the coordinate value:

Left The axis name is placed before the coordinate value (single line label).
Right The axis name is placed after the coordinate value (single line label).
Above The axis name is placed above the coordinate value (single line label).
Below The axis name is placed below the coordinate value (single line label).

Note
This applies only to the sectioned grid lines which are created and visible only if the Show sectioned lines check box is selected in the Reference Grid category.
