Wizard

Wizard

Options in this area control the various settings for the implementation of the Sheet Metal Wizard command.

Unrecognized edges

Add:

Depending upon the selection made in the Add: drop-down list, all the edges in the solid that have not been selected either for rip or bend creation will be automatically added to either the Bend Group or the Rip Group to create bends or rips respectively.

Add

Add:

Depending upon the selection made in the Add: drop-down list, all the edges in the solid that have not been selected either for rip or bend creation will be automatically added to either the Bend Group or the Rip Group to create bends or rips respectively.

Resulting components

Position:

Depending upon the selection made in the Position: drop-down list, the sheet metal components that are created are either internal components or external components.

  • Internal: The sheet metal components are saved within the same file as that of the solid from which they are being created.
  • External: The sheet metal components are saved as individual e3 files inside the same folder where the file of the solid model is saved.

Color

Color

The options in this area control the color visualization of various selections that are made for the selectors of the Sheet Metal Wizard selection list and the resulting sheet metal components.

Bend edges

Bend edges:

All the edges that have been selected for the Bend Group are displayed as per the color selection made for this option. To modify the selection, click on the corresponding down arrow to display the color palette; make the new color selection in it.

Rip edges

Rip edges:

All the edges that have been selected for the Rip Group are displayed as per the color selection made for this option. To modify the selection, click on the corresponding down arrow to display the color palette; make the new color selection in it.

Resulting components

Resulting components:

All the sheet metal components that are created by the Sheet Metal Wizard command are displayed in the color as per the color selection made for this option.
In the drop-down list, select the component number to display its color setting. To modify the color selection, click on the corresponding down arrow to display the color palette; make the new color selection in it. Using this option, the colors of 32 components can be set.
