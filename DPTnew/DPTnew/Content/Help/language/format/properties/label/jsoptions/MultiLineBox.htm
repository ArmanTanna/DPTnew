Click this button to open the Multiline Text dialog box, enabling you to create multiline text labels. Then, type in your text, that can be:

  • Plain text, in which case:
    • TABS are ignored
    • Blanks are real blanks
    • Hit ENTER to open a new line
  • Text formatted according to the think3 application Text Markup Language. The maximum length, including TML tagging, is 256 characters.
