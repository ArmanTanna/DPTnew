View dimensioning

View dimensioning

The controls in this area determine the automatic creation of dimensions in individual drawing views.

AutoPlace dimensions

Auto-place dimensions:

Specifies when dimensions created in an automatic or derived view can be repositioned so that they can be seen more easily. Select one option from the adjacent drop-down list:

Neverdimensions are not automatically spaced
Only when createddimensions are automatically spaced only when first created
Every regenerationdimensions are automatically spaced whenever they are regenerated (Note that these dimensions can be manually repositioned within the drawing).

Keep model properties at first creation

Keep model properties at first creation

When selected, automatically placed dimensions are drawn using the same properties as those defined for the individual dimensions within the model.

When cleared, automatically placed dimensions are all drawn using the same properties, as defined in the drawing.
