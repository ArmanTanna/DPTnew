Category: Ground

Ground

Ground

The controls in this page enable you to display the shadows and reflection of the model on a selected plane. The following operations can be performed.

  • Set the ground to the current view.
  • Specify an offset distance to the ground plane.
  • Set the intensity of the reflection and enable the shadow formation of the model.

Set from Current View

Set from Current View

When selected enables the application to move the ground plane with respect to the current orientation of view of the model.
Also the ground plane can be set using the Set Ground command.

Offset:

Offset

The ground plane is offset by the distance entered in the edit box. The distance value always has to be positive as the ground plane has to move downwards of the model.

Reflection:

Reflection

Enables you to set the intensity of model reflection from a range of 0% to 100%.

Shadow

Shadow

When the check box is selected, shadow of the model is visualized on the ground plane.
When the check box is cleared, there is no shadow of the model displayed on the ground plane.
