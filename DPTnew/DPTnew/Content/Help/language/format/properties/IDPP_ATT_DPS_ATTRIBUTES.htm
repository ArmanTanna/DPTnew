Document Properties

Category: Drawing-View Attributes

Projection mode

Projection mode

Enables you to determine the projection method used to obtained certain derived views.

First Angle - ISO Third Angle - ANSI

First angle

First angle

Applies a First Angle projection (ISO standard).

Third angle

Third angle

Applies a Third Angle projection (ANSI standard).

Texts

Texts:

Enables you to control the appearance of text, such as section names.

Axes

Axes:

Enables you to control the appearance of the lines representing axes and centerlines.

Cutting-plane lines

Cutting-plane lines:

Enables you to control the appearance of the section cutting plane lines.

X-Hatches

X-Hatches:

Enables you to control the appearance of the cross-hatching applied to sections.

Dimensions

Dimensions:

Enables you to control the appearance of dimensions.

Color

Color:

Enables you to select the color for each type of entity by choosing it from the adjacent drop-down list.

Note that when the Keep model attributes option is selected and any of the drawing view representations (Hidden lines or Wireframe) is set, this option is grayed out.

Layer

Layer:

Enables you to choose an existing layer or type in the name of a new one to create it.

Note that when the Keep model attributes option is selected and any of the drawing view representations (Hidden lines or Wireframe) is set, this option is grayed out.

Representation

Representation

Enables you to determine the way threads and chamfers are represented in the drawing.

Threads

Threads:

Enables you to determine the way in which threads are represented in the drawing. Select one from the drop-down list.

Chamfers

Chamfers:

Enables you to determine the way in which chamfers are dimensioned in the drawing. Select one from the drop-down list.
