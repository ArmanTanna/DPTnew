Viewport properties

Viewport properties

The controls in this area enable you to specify the values of some of the view parameters, such as depth and projection.

Depth

Depth:

Enables you to set the distance from the view plane (positive outward, negative inward; the view plane is the plane of the monitor screen) in a direction orthogonal to it, when the Mouse Input is set to On view.



a Click point
b Screen plane

When you specify points by clicking with the mouse (without snapping to special points), the point you specify is assumed to be parallel to the current view plane at the specified depth. Since the screen plane normal is considered to be directed outward from the screen:

  • A positive depth value implies that points indicated using the mouse lie on a plane parallel to the screen plane, on the same side of the screen as the user
  • A negative depth value implies that the points entered using the mouse lie on a plane parallel to the screen plane on the opposite side of the screen from the user.
The default depth on all views is 0. In that case, all input from the mouse is on the screen plane in each view.


Note: Remember that unless you are viewing the model from a position perpendicular to the Work Plane, the depth doesn't coincide with the z-coordinate of a point provided using the mouse. The z-coordinate and depth coincide only when the z-axis is orthogonal to the screen plane.

Rendering:

Projection

Projection

The controls in this area determine the projection method used to display the model: Orthogonal or Perspective.
See details about projection methods in "Orthographic and perspective views: general concepts".

Current

Current:

Choose the projection method to be used to display the model: Orthogonal or Perspective.
See details about projection methods in "Orthographic and perspective views: general concepts".

Please note that you can also chnge the projection mode by right-clicking on an empty zone of the graphics area and by selecting Orthogonal Projection or Perspective Projection under View in the context menu.

Focal length

Focal length:

The focal length mechanism is the same as in cameras and affects the view angle:

  • If the focal length value is low, the view angle is very big (like when using a wide-angle lens: objects that are close to the camera appear distorted)
  • If the focal length value is high, the view angle is small (like when using a telephoto lens: distortion is lower)

You can set the focal length either by selecting it in a drop-down list of predefined values (1) or by specifying a different value in the edit box on the right-hand side (2).

Set

The buttons in this area enable you to control the view by properly setting the camera and the target.

Camera

Enables you to place the camera at a selected point and to set the view direction along the invisible line between the camera and target position.

View mode

View mode

The controls in this area enable you to choose the view projection in the splitting process and the type of connection among views.

European

US

When splitting views, the options have the following meaning:

  • European the views are arranged in an ISO standard arrangement with the Front view in the upper-left, Side view in the upper-right, Top view in the lower-left, and Isometric view in the lower-right.
  • US the views are arranged in an ANSI standard arrangement with the Top view in the upper-left, Isometric view in the upper-right, Front view in the lower-left, and Side view in the lower-right.

US

US

When splitting views, the options have the following meaning:

  • European the views are arranged in an ISO standard arrangement with the Front view in the upper-left, Side view in the upper-right, Top view in the lower-left, and Isometric view in the lower-right.
  • US the views are arranged in an ANSI standard arrangement with the Top view in the upper-left, Isometric view in the upper-right, Front view in the lower-left, and Side view in the lower-right.
