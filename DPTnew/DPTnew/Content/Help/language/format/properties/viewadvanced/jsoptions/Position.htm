Enables you to select a symmetry plane about which a mirrored preview of the model will be displayed (thus showing a virtual image of the portion which is not actually there).

Custom When selected for the first time, the Custom option sets the virtual mirror plane to the Work Plane. Starting from the very moment you initialize the virtual mirror plane to the Work Plane, switching back and forth from Custom does not alter the custom virtual mirror plane. If you change the position of the Work Plane, you can update the position of the virtual mirror plane by clicking As Work Plane.
XZ, XY, ZY the image virtually mirrored about the chosen plane of the World reference system will be displayed

The mechanism can be enabled separately on each view, so that you can have different symmetry planes in different views. Mirrored images are automatically updated at any geometry change and are insensitive to highlighting and selection actions (i.e: you can't select a mirrored entity). As to components, mirroring is enabled only for the ones at top level.

For further details on the virtual mirroring mechanism, see "Virtual Mirroring".
