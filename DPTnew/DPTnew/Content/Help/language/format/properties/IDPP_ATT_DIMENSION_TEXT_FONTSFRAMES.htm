Entity Properties

Category: Dimension-Text-Font & Frame

Current

Current:

Enables you to set the current style by selecting it in the list of all the available ones.

Text style

Text style:

In the Organize Styles dialog box, it enables you to determine the Text style for the dimension texts. All the Loaded styles are available in the drop-down list.
In the Entity Properties dialog box, it just enables you to view the current style. The drop-down list is greyed out to indicate that you cannot change the style here, except in Organize Styles dialog box.

Font

Font

The controls in this area enable you to set various parameters for the Font.

Standard

Standard:

Enables you to specify the TrueType Standard font to be used to create text. Select one of the fonts in the drop-down list.

Alternative

Alternative:

Enables you to specify Alternative fonts such as Vector fonts when the TrueType Standard font is not specified. Select one of the fonts in the drop-down list.

Aspect

Aspect:

Enables you to determine the aspect applied to the current font:

Bold Text will be displayed in bold (for example: TEXT). This option applies only to True Type (Standard) fonts.
Italic Text will be displayed in italic (for example: TEXT). This option applies both to True Type (Standard) and vectorial (Alternative) fonts.
Underline Text will be underlined (for example TEXT). This option applies both to True Type (Standard) and vectorial (Alternative) fonts.

Frame

Frame

The controls in this area enable you to select the color and the type of the text frame.

Type

Type:

Enables you to determine the Type of Frame used to display text. You can choose one of the following options:

None
Circle (1c)
Circle (2c)
Circle (3c)
Circle (4c)
Rectangle
Rounded rectangle
Rectangle with broken sides
Rectangle with a broken side
Underlining
Arc over text
Break note
Round brackets
Square brackets
Triangle
Rectangle with dashes
JIS Underlining (*) The underlinings are as long as the longest text and they are joined by a vertical line
JIS Underlining (not longest) (*) The underlinings are exactly as long as the text above and they are joined by a vertical line
JIS Underlining (not vert.) (*) The underlinings are as long as the longest text and there is no vertical line
JIS Underlining (not vert.+ not longest) (*) The underlinings are exactly as long as the text above and there is no vertical line

(*) When you choose one of them and run the Label with the Multiline Mode option checked.

Color

Color:

Enables you to specify the color to be used to draw text frames.

Offset

Offset

The controls in this area enable you to specify the offset to be provided for dimensions and labels.

Dimensions

Dimensions:

Enables you to specify the offset to be provided for dimensions within frames.

Labels

Labels:

Enables you to specify the offset to be provided for the labels within frames.
