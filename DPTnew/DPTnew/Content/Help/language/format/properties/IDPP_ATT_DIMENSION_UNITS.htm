Entity Properties

Category: Dimension-Unit

Current

Current:

Enables you to set the current style by selecting it in the list of all the available ones.

Taper dimension

Taper dimension

The controls in this area enable you to define the taper dimension format.

Format

Format:

Enables you to determine the format for the taper dimensions. You can choose any one of the following from the drop-down list options:

  • Ratio (n:1)
  • Ratio (1:n)
  • Ratio (1/n)
  • Percentage (%)
  • Angle
  • Angle or taper designation

Linear dimension

Linear dimension

The controls in this area enable you to define various parameters for the Linear dimension.

Show label

Show label

Enables you to determine whether to show the unit label for the Linear dimension (such as m or m mm) or not. You have the following options:
When selected, the label is displayed.
When not selected, the label is not displayed.

Show dash

Show dash

Enables you to include the dash separator for the Linear dimensions, when the Unit option is set to either Feet inches or m mm.
When the box is selected, a dash separator is included between two units.
When the box is not selected, a dash separator is not included.

Unit

Unit:

Enables you to determine the Unit of measure for the Linear dimension. You can choose any one of the following options:

  • None
  • mm
  • cm
  • m
  • m mm
  • Feet
  • Inches
  • Feet inches
  • '
  • "
  • ' "

Precision

Precision:

Enables you to define the level of Precision for the Linear dimension, i.e.: to specify the number of decimal digits to be displayed in the dimension value. You can select any one option from the drop-down list.

Round off

Round off:

Enables you to Round off the measured distance to the nearest multiple of the entered value. It is not related to the representation of the dimension, but to the value itself. You can increase or decrease the level using the up and down arrows.

Format

Separator:

Enables you to determine the Format to be used to represent the Linear dimension measures. You can choose any one of the following options:

Decimal Enables you to display Linear dimensions using a Decimal representation. As soon as you choose this option, the Separator drop down list appears which helps you to choose one of the following options:
  • .(period)
  • ,(comma)
Fractional Enables you to display Linear dimensions using a Fractional representation. As soon as you choose this option, the Denominator and the Arrangement options show up. The former enables you to specify the denominator, the second enables you to set the arrangement as Horizontal, Vertical or Diagonal.
Scientific Enables you to display Linear dimensions using a Scientific representation.

Separator

Separator:

Decimal Enables you to display Linear dimensions using a Decimal representation. As soon as you choose this option, the Separator drop down list appears which helps you to choose one of the following options:

Zero display

Zero display

The options in this area enables you to determine the visibility of the Leading and Trailing zeroes in the Linear dimension values.

Leading

Leading

Enables you to determine the visibility of the Leading zeroes in the Linear dimension values. Using the check box you have the following possibilities:
When selected, the Leading zeroes are visible.
When not selected, the Leading zeroes are not visible.

Trailing

Trailing

Enables you to determine the visibility of the Trailing zeroes in the Linear dimension tolerance values. Using the check box you have the following possibilities:
When selected, the Trailing zeroes are visible.
When not selected, the Trailing zeroes are not visible.

Show

Show:

Enables you to choose which Leading or Trailing zeroes are to be displayed, When the Unit is set to either m mm or ft in:

Both Enables you to show Leading or Trailing zeroes for the both the units.
0 Feet/meters Enables you to display show Leading or Trailing zeroes only for the meters.
0 Inches/mm Enables you to display show Leading or Trailing zeroes only for the mm.
None Enables you to show Leading or Trailing zeroes for neither of the measures.

Angular

Angular

The controls in this area enable you to define various parameters for the Angular dimensions.

Show label

Show label

Enables you to determine whether to show the unit label for the Angular dimensions (such as Degrees or Radians) or not. You have the following options:
When selected, the label is displayed.
When not selected, the label is not displayed.

Unit

Unit:

Enables you to determine the Unit of measure for the Angular dimensions. You can choose any one of the following options:

None Enables you to show None of the Units for the Angular dimensions.
Degrees Enables you to the specify the Unit of Angular dimensions as Degrees.
Degrees and minutes Enables you to the specify the Unit of Angular dimensions as Degrees and minutes.
Degrees, minutes and seconds Enables you to the specify the Unit of Angular dimensions as Degrees, minutes and seconds.
Radians Enables you to the specify the Unit of Angular dimensions as Radians.
Gradians Enables you to the specify the Unit of Angular dimensions as Gradians.

Precision

Precision:

Enables you to define the level of Precision for the Angular dimensions i.e., to specify the number of decimal digits to be displayed in dimension values. You can choose any one option from the drop-down list.

Zero display

Zero display

The options in this area enables you to determine the visibility of the Leading and Trailing zeroes in the Angular dimension values.

Leading

Leading

Enables you to determine the visibility of the Leading zeroes in the Angular dimension values. Using the check box you have the following possibilities:
When selected, the Leading zeroes are visible.
When not selected, the Leading zeroes are not visible.

Trailing

Trailing

Enables you to determine the visibility of the Trailing zeroes in the Angular dimension values. Using the check box you have the following possibilities:
When selected, the Trailing zeroes are visible.
When not selected, the Trailing zeroes are not visible.

Show

Show:

Enables you to choose which zeroes are to be displayed, when the Unit of measure is set to Degrees and minutes or to Degrees, minutes and seconds:

None Enables you to show no zeroes for the Angular dimensions.
Less significant Enables you to display less significant zeroes (for example: 0'' will be displayed).
More significant Enables you to display more significant zeroes (for example: 0'' will not be displayed).
All Enables you to display all zeroes.
