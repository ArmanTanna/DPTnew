Category: Drawing View-Miscellaneous

Miscellaneous

Miscellaneous

The controls in this area determine some miscellaneous parameters applying to derived views.

Show projection direction

Show projection direction

When selected, a symbol is added to the view from which another view is derived to indicate the projection direction of the derived view; when cleared, no projection direction symbol is created.

Align to main

Align to main

When selected, a derived view is aligned with the view from which it was derived; when cleared, a derived view may be positioned anywhere in the drawing.

Auxiliary revolving direction

Auxiliary revolving direction:

Determines the direction in relation to the selected line in which to create auxiliary views.

Detail view

Detail view

The controls in this area determine the scale of detail views and the appearance of detail view boundaries.

Zoom

Zoom:

Determines the scale of the next detail view relative to the scale of the view from which it is referenced.

Closed boundary

Value:

When selected, the detail view is created with a closed boundary matching the boundary defined when it is referenced.
When not selected, the detail is created with an open boundary visible only where it separates the detail from the objects which it references.

Inherit value from parent view

Inherit value from parent view

When the check box is selected, you are enabled to create the detail view with the filter value inherited from the parent view.
When the box is not selected, you are enabled to specify a value with which the detail view can be created. If value is set to zero, you can create a detail view from a view with the filter enabled.

(See large assemblies for further information).

Note
  • The Value box to specify a filter value is active only if Inherit value from parent view box is not selected.
  • This field is available only in idle mode (when no view is selected).
