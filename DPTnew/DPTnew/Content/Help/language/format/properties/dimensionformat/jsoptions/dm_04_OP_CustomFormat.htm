Enables you to achieve full and detailed control on the whole dimension text, that is on all of the parts displayed in the following illustration, plus the dual measure, if any.

Inside Customized format two very useful tools can be used:

  • The TML coding (think3 Text Markup Language)
  • Some specifically created symbolic notations — along with all the already existing ones — enabling the inclusion of text portions implicitly expanded according to the current property settings.



    They are:
    <MM> The complete dimension according to all current properties (that is: the TML notation emulating the whole dimension as if it were created with no custom format)
    <DM> The dual measure according to current dual measure properties
    <M> The ordinary dimension measure according to current properties
    <T> The ordinary dimension tolerance representation according to current properties
    <S> The fit tolerance symbol
    <ST> Fit symbol and tolerances
    <U> The symbol of the unit of measure for the ordinary dimension measure.

All the above mentioned notations can be automatically applied on selecting the corresponding options available in the drop-down list. Depending on the actual content of the Customized format box, then, some properties will be grayed out or not in order to very clearly display which properties are giving their contribution to the visualized dimension text and which are not.

Some Examples

  • To display the unit of measure after the measure and the tolerances, separated by some additional space (take into account the tolerance values and the dimension unit have to be entered as ordinary properties), the following code can be used:

    <M><T><SP/><U>



  • Some ready-to-use notes spreading across more than one line lines can also be used:

    2<X><SP/><FDIA><M><BR/> <CNB><SP/><FDIA>1.5<BR/><DEP>.062



  • Customizable ways to write tolerances are also available:

    52<VALIGN type="center"><STACK scale="0.60" separator="none">
    <SR> H7(<STACK scale="0.40" separator="none">
    <SR>+0.03</SR><SR><SP/>0</SR></STACK>)</SR>
    <SR> g6(<STACK scale="0.40" separator="none">
    <SR>-0.01</SR><SR>-0.029</SR></STACK>)</SR>
