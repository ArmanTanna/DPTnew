Entity Properties

Category: X-Hatch

X-Hatch

X-Hatch

Enables you to choose the desired X-Hatch.

Type

Type:

Enables you to determine the type of material represented by hatching. The Type drop-down list has the following types of X-Hatch representations:

  • Basic
  • Standard
  • Predefined
  • Custom

A short description of all these four types is given in the Name option page.

Add

Add

By clicking on this control, the Custom Hatch Pattern Editor dialog box shows up. This dialog box enables you to customize your own X-Hatch pattern.

Pattern

Pattern

Enables you to choose the desired pattern, depending on the Type of X-Hatch chosen. This control helps you to design the pattern using the Angle, the Spacing and the Offset options.

Name

Name:

Depending on the Type of X-Hatch chosen, you get different patterns.

Basic Choosing this type, you will get the following X-Hatch patterns
Iron Enables you to apply to the hatching a material of type Iron
Lead Enables you to apply to the hatching a material of type Lead
Brick Enables you to apply to the hatching a material of type Brick
Steel Enables you to apply to the hatching a material of type Steel
Solid Enables you to apply to the hatching a material of type Solid
Standard Choosing this type, you will get all the ISO and ANSI standard complied X-hatch patterns
Predefined Choosing this type, you will get the following X-hatch patterns
Brass Enables you to apply to the hatching a material of type Brass
Clay Enables you to apply to the hatching a material of type Clay
Cork Enables you to apply to the hatching a material of type Cork
Honey Enables you to apply to the hatching a material of type Honey
Custom Choosing this type, you can create your own X-hatch patterns

Edit

Edit

This control enables you to edit the customized patterns.

Angle

Angle:

This control enables you to align the hatchings at a specified angle.

Scale

Scale:

Enables to control the hatch density. Lower the value, closer are the hatcher lines and vice-versa. This parameter is applicable only for standard, predefined and custom hatches. For basic hatches, spacing parameter should be used to get the desired density.

Offset (0->1)

Offset (0->1):

This indicates the distance by which the hatch lines should be moved with respect to default hatch line position. The default hatch line position is when Offset is 0. If the Offset is 1, the hatch lines are offset by a distance equal to spacing parameter value. The offset should be between 0 and 1.

Spacing 1

Spacing 1:

This control enables you to specify the distance seperating two hatches along the first axis.

Preview

Preview

Enables you to get the Preview of an X-Hatch pattern based on the current values. The preview mechanism is enabled when you are trying to change the X-Hatch properties globally, whereas it is disabled while creating or modifying an X-hatch pattern.
