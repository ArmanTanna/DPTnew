Entity Properties

Category: Dimension-Tolerance-Precision

Current

Current:

Enables you to set the current style by selecting it in the list of all the available ones.

Linear dimension

Linear dimension

The controls in this area enable you to define various parameters for the tolerances of Linear dimension.

Precision

Precision:

Enables you to define the level of Precision for the Linear dimension tolerances, i.e.: to specify the number of decimal digits to be displayed in the dimension tolerance values. You can select any one option from the drop-down list.

Zero display

Zero display

The options in this area enables you to determine the visibility of the Leading and Trailing zeroes in the Linear dimension tolerance values.

Leading

Leading

Enables you to determine the visibility of the Leading zeroes in the Linear dimension tolerance values. Using the check box you have the following possibilities:
When selected, the Leading zeroes are visible.
When not selected, the Leading zeroes are not visible.

Trailing

Trailing

Enables you to determine the visibility of the Trailing zeroes in the Linear dimension values. Using the check box you have the following possibilities:
When selected, the Trailing zeroes are visible.
When not selected, the Trailing zeroes are not visible.

Show

Show:

Enables you to choose which Leading or Trailing zeroes are to be displayed, When the Unit is set to either m mm or ft in:

Both Enables you to show Leading or Trailing zeroes for the both the units.
0 Feet/meters Enables you to display show Leading or Trailing zeroes only for the meters.
0 Inches/mm Enables you to display show Leading or Trailing zeroes only for the mm.
None Enables you to show Leading or Trailing zeroes for neither of the measures.
