Entity Properties

Category: Cutting-plane Line

Section text

Section text

The controls in this area enable you to define the settings determining the appearence of section texts.

Name

Name:

Enables you to redefine the section name.

Position

Position:

Enables you to determine the location of the section text in relation to the cutting plane line. Select one from the drop-down list.

Size

Size:

Enables you to determine the height of the section text.

Text size related

Text size related

When this check box is selected, the value specified is interpreted as a percentage of the dimension text height; when not selected, the value specifies the actual text size (measured in current units).

Cutting-plane line

Cutting-plane line

The controls in this area enable you to set the type and the style of the cutting plane line.

Type

Type:

Enables you to determine the line type used when creating cutting plane lines. Select the desired line type from the drop-down list.

Style

Style:

Enables you to determine the line style used when creating cutting plane lines. Select one from the drop-down list.

Arrow

Arrow

The controls in this area enable you to set the settings of the section symbol arrow.

Type

Type:

Enables you to determine the type of arrow placed at the ends of cutting plane lines. Select one from the drop-down list.

Length

Length:

Enables you to determine the length of arrow heads created as part of cutting plane lines.

Text size related

Text size related

When this check box is selected, the value specified is interpreted as a percentage of the Section text size; when not selected, the value specifies the actual arrow length.

Display mode

Display mode:

Enables you to determine the display mode for cutting plane lines. Select one from the drop-down list.
