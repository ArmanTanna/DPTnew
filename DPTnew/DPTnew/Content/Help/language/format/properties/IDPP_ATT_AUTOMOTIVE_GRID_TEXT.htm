Entity Properties

Category: Reference Grid-Text

Text appearance

Text appearance

In this area you are enabled to control the texts to be displayed in the labels of the reference grids.

Standard

Standard:

Enables you to specify the TrueType Standard font to be used to create text, which gets displayed inside the label of the reference grid. Select one of the fonts in the drop-down list.

Alternative

Alternative:

Enables you to specify Alternative fonts such as Vector fonts when the TrueType Standard font is not specified. Select one of the fonts in the drop-down list.

Text size

Text size:

Enables you to specify the text Size of the labels of the reference grids.

Aspect

Aspect:

Enables you to determine the aspect applied to the current font:

Bold Text will be displayed in bold (for example: TEXT). This option applies only to True Type (Standard) fonts.
Italic Text will be displayed in italic (for example: TEXT). This option applies both to True Type (Standard) and vectorial (Alternative) fonts.
Underline Text will be underlined (for example TEXT). This option applies both to True Type (Standard) and vectorial (Alternative) fonts.

Color

Color:

Enables you to change the color of the texts to be displayed in the labels of the reference grids.
When you click on the arrow adjacent to the color box, the color palette is displayed, so as to enable you to make your choice.


Note
Since the reference grid is a group entity, the visualization of the internal color defined for the circular frame depends upon the value of the Show closed groups with group color check box in the Groups category of the Document Properties.
To display the specific colors of the grid, that is the one of the circular frame, the one defined for major lines and the one defined for minor lines, the Show closed groups with group color check box must be cleared.
