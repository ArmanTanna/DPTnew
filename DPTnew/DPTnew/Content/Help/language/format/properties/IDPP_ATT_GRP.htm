Edit Properties

Category: Group

Group

Group

The controls in this area enable you to set all the group parameters.

Name:

Name:

Displays and enables you to modify the name of the group.

Shared group copies

Shared group copies

The parameters in this area display information about the copies of the selected shared group. If the selected group is not shared, the area is grayed out.

Total:

Total:

Displays the total number of copies of the selected shared group.

Shared group definition

Shared group definition

All the parameters in this area apply only to the definition of shared groups. If the selected group is not shared, the area is grayed out.

File:

File:

Displays the file containing the group description, if the selected shared group is an X-Reference.

Status:

Loaded

Displays the loading status of the selected X-Reference shared group. The status is normally Loaded, unless you have discarded the description of the X-Reference using the Unload Shared Group command.

Not Loaded

Loaded

Displays the loading status of the selected X-Reference shared group. The status is normally Loaded, unless you have discarded the description of the X-Reference using the Unload Shared Group command.

Read only

Read only

When checked, the selected group cannot be modified. When not checked, you can modify the selected group (for example using Start Local Edit).

Local copy

Local copy

This check box is available only when you select an X-Reference. After selecting the X-Reference, if you select the check box, a local copy of the X-Reference is created (and its icon in the Drawing Structure changes accordingly). If the check box is not selected, no local copy is created.

Persistent

Persistent

This check box is used to display the status of persistence of the selected shared group and cannot be selected/deselected by the user — When checked, the selected shared group is persistent: if you remove all of its copies from the drawing, its definition or archetype is still available in the Shared Group tab. When not checked, the selected shared group is not persistent (which may happen in the case of an X-Reference group or of a Detail).
