Document Properties

Category: Drawing-Groups

History

History

The controls in this area enable you to define, for the drawing entities, the parameters related to the history.

Enable unrelated entities group with color

Enable unrelated entities group with color:

If this box is selected, when you add an entity to a view in such a way that it is not parametrically linked to the view, it will be automatically added to the 'unrelated entity group' with the color defined in the adjacent table. The Unrelated entity group will be displayed in the Drawing Structure. If this box is not selected, the Unrelated entities group will not be created and displayed in the Drawing Structure.

Group attributes

Group attributes

The controls in this area enable you to define the parameters related to the group.

Show closed groups with group color

Show closed groups with group color

When selected, any entities belonging to groups are displayed with the group color (which can be defined using the other controls in this area). When not selected, any entities belonging to groups are displayed in their own colors.

Ignore layer for closed non-shared groups

Ignore layer for closed non-shared groups

When the check box is selected, the program ignores closed non-shared groups layer — that is the non-shared groups are considered as transparent container as far as concerned layer attribute.

Create group with color

Create group with color:

If selected, it enables you to select in the adjacent selection box the color to be used to display closed groups when the Show closed groups with group color check box is selected.

Create group with current color

Create group with current color

If selected, the program displays closed groups using the current color when the Show closed groups with group color check box is selected.

Make gray all entities outside current group

Make gray all entities outside the current group

When selected, all the entities outside the current group are displayed in gray.
