Drawing

Drawing

The controls in this area determine the drawing Frame and Title Block.

Scale

Scale:

Determines the scale of the document, which will be applied to the title block when it is added to the drawing
If you enter only one number — say x — the actual scale assigned by the program will be x:1. Each former scale value will be available in the drop-down list.


Note
The actual dimension of entities in the drawing views on screen is obtained, for each view, by combining the scale of the view with that of the drawing.

Enable sheet box representation

Enable sheet box representation

When selected, displays a bounding box to facilitate the positioning of drawing objects.
