Entity Properties

Category: Drawing View-Visibility-Image

Image

Image

The controls in this area enable to define the resolution levels, rendering modes and effects on the captured image of the drawing view (if an image is present).

Resolution level

Resolution level:

Enables you to set the resolution level for the captured image.

Resolution level Low     Medium     High     Very high     Custom    
Resolution value (pixels) 512 1024 2048 4096 Any value between 64 and 4096

Note
  • If the resolution level is not set to Custom, the resolution value field is grayed because it depends on the level setting.
  • If the Custom resolution level is selected, the value is editable and any value between 64 and 4096 could be specified.
  • The maximum achievable resolution value depends on the available video RAM of the video card.

Value

Value:

Enables you to set the resolution value for the captured image. The resolution value specifies the number of pixels of the longest edge of the image.

Resolution level Low     Medium     High     Very high     Custom

Rendering mode

Render mode

The control in this area determine the rendering modes on the captured image of the drawing view (if an image is present).

Edges

Edges:

Enables you to choose the display modality for edges and silhouettes in the captured image:

None Enables the view to display shading without any edges. Enable shading is forcedly checked and grayed.
Visible Enables the view to show only visible edges (only for Image only mode)
Visible + hidden Enables the view to show visible edges (continuous line) and hidden edges (dashed line) — (only for Image only mode).
Wireframe Enables the view to show all edges with continuous line (no shading allowed, only for Image only mode)

Enable shading

Enable shading

When the check box is selected, it enables to display the model in high quality rendering.
When the box is not selected, model cannot be displayed in high quality rendering.


Note

If Enable shading is selected, Enable high quality rendering mode can be activated.

Enable high quality rendering mode

Enable High Quality render mode

When the check box is selected, sets if model is shaded in the captured image.
When the box is not selected, shading is not permitted in the captured image.

Show model background

Show model background

When the check box is selected, the background of the model is used for the image.
When the box is not selected, drawing uses white color as background (which is best for printing).
