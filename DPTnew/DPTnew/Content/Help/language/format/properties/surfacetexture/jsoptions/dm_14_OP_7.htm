Enables you to specify the type of surface texture Symbol by selecting one of the following options from the drop-down list:

Generic Enables you to denote a basic surface, which can be produced by any method. This Symbol is chosen, when a particular production method or Lay direction is not specified.
Material removal Enables you to denote a surface, from which can some material has to be removed using machining technique. The horizontal bar indicates that the material has to be removed by machining to produce the surface.
No Material Removal Enables you to denote a surface, for which material removal is prohibited. The circle indicates that the surface has to be produced by using processes such as casting, forging, etc. without subsequent removal of the material.
