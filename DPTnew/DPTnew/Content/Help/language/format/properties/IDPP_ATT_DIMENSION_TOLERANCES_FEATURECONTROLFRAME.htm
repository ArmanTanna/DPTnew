Between two points

Enable

Enables you to specify two points between which the Feature Control Frame specification will be valid. This option works depending on the state of the Enable check box:

  • When selected, you can specify the identifiers of the two points (for example: A, B) in the boxes on the left and on the right of the arrow.
  • When not selected, this option is not enabled.

Line number

Line number:

Enables you to choose the Line number to which a particular tolerance is assigned. You can enter up to 4 lines of tolerances. For each Line number, specify the Line Type and then enter appropriate information in the Feature Control Frame for that particular tolerance.

Line type

Line type:

Enables you to determine the type of information that has to appear in that line. The options available in the drop-down list are:

Feature control frame Enables you to provide information regarding the control frame features such as Straightness, Flatness, etc.
Datum reference Enables you to provide information regarding the datum references.
Proj. tolerance zone Enables you to provide information regarding the projected tolerance zone value.
Simple text Enables you to attach a personalized string at the end of a feature control frame entity.

Symbol

Symbol:

Enables you to determine the Geometric tolerance symbol, to be included in the Feature Control Frame denoted on the current line. You can choose one of the options from the drop-down list:

All around

All around

Enables you to specify that the tolerances specified is All around the profile. This option is available only for Profile of a line and Profile of a surface options.

  • When the check box is selected, the tolerances will be applied All around the profile.
  • When not selected, the tolerances will be restricted to the line or the surface.

Value

Value

Enables you to:

  • specify the tolerance value of the current line
  • choose the diameter or spherical diameter symbol (Ø or SØ) to include along with the tolerance value
  • choose to include the special notation:
    • (-)  concave material
    • (+) protrudable material
    • (>) diminishing from left to right
    • (<) diminishing from right to left

M.C.

M.C.

Enables you to specify an appropriate material condition symbol. You can choose from one of the following options:

Maximum material condition Enables you to specify that it is the condition of the part feature which contains maximum amount of material.
Least material condition Enables you to specify that it is the condition of the part feature which contains Least amount of material.
Regardless of size Enables you to specify that the geometric tolerances and the datum references must be complied with, at all the sizes of the part feature, within the specified tolerance limits.
Statistical Tolerance Enables you to specify that the tolerance for the part feature is specified after a statistical analysis. The desired result is larger tolerances
Free state Enables you to specify that the parts must not be restricted during inspection.
Containing Tolerance Enables you to specify that it is the condition of the part feature which contains a dimension limit.
Reverse Requirement Enables you to specify that it allows to enlarge reversed geometric tolerance number when some function requests have been sufficed.
Projected Tolerance Enables you to specify that it is the condition of the part feature which contains a projected tolerance zone.
