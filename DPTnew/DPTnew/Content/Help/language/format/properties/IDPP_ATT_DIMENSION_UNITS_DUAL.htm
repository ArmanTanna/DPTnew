Entity Properties

Category: Dimension-Unit-Advanced

Current

Current:

Enables you to set the current style by selecting it in the list of all the available ones.

Dual dimension

Dual dimension

The controls in this area enable you to define various parameters for Dual dimensions.

Show label

Show label

Enables you to determine to Show label along with the Dual dimension. You have the following options:
When selected, the label is displayed.
When not selected, the label is not displayed.

Unit

Unit:

Enables you to determine the Unit of measure for the Dual dimension. You can choose any one of the following options:

  • None
  • mm
  • cm
  • m
  • m mm
  • Feet
  • Inches
  • Feet inches
  • '
  • ''
  • ' "

Precision

Precision:

Enables you to define the level of Precision for the Dual dimension, i.e.: to specify the number of decimal digits to be displayed in dimension value. You can select any one option from the drop down list.

Round off

Round off:

Enables you to Round off the measured distance to the nearest multiple. It is not related to the representation of the dimension, but to the value itself. You can increase or decrease the level using the up and down arrows.

Format

Format:

Enables you to determine the Format to be used to represent the Dual dimension measures. You can choose any one of the following options:

Decimal Enables you to display Dual dimensions using a Decimal representation. As soon as you choose this option, the Separator drop down list appears which helps you to choose one of the following options:
  • .(period)
  • ,(comma)
Fractional Enables you to display Dual dimensions using a Fractional representation. As soon as you choose this option, the Denominator option shows up, enabling you to define the denominator value.
Scientific Enables you to display Dual dimensions using a Scientific representation.

Separator

Separator:

Enables you to choose one of the two following separators for the decimal digits:

  • .(period)
  • ,(comma)

Zero display

Zero display

The options in this area enable you to determine the visibility of the Leading and Trailing zeroes in the Dual dimension values.

Leading

Leading

Enables you to determine the visibility of the Leading zeroes in the Dual dimension values. Using the check box you have the following possibilities:
When selected, the Leading zeroes are visible.
When not selected, the Leading zeroes are not visible.

Trailing

Trailing

Enables you to determine the visibility of the Trailing zeroes in the Dual dimension values. Using the check box you have the following possibilities:
When selected, the Trailing zeroes are visible.
When not selected, the Trailing zeroes are not visible.

Show

Show:

Enables you to choose which Leading or Trailing zeroes are to be displayed, When the Unit is set to either m mm or ft in:

Both Enables you to show Leading or Trailing zeroes for the both the units.
0 Feet/meters Enables you to display show Leading or Trailing zeroes only for the meters.
0 Inches/mm Enables you to display show Leading or Trailing zeroes only for the mm.
None Enables you to show Leading or Trailing zeroes for neither of the measures.

Representation

Representation

The controls in this area enable you to choose a proper Representation for the Dual dimensions.

Demarcation

Demarcation:

Enables you to determine the Demarcation symbol for a Dual dimension. You can choose any one of the following options:

Line Enables you to use Line as the Demarcation between a Dual dimension and a primary dimension.
Brackets Enables you to use Brackets as the Demarcation between a Dual dimension and a primary dimension.

Position

Position:

Enables you to determine the Location for a Dual dimension with respect to primary dimension. You can choose any one of the following options:

Below Enables you to place the Dual dimension Below the primary dimension.
Right Enables you to place the Dual dimension to the Right of the primary dimension.
