Entity Properties

Category: Drawing View-View Attributes

Model attributes

Model attributes

The controls in this area enable you to:

  • maintain the attributes of a model
  • control the color of the edges in a view

Keep

Keep model attributes

When selected, it activates the Inherit attributes controls and visibility is set to:

Wireframe enables you to keep the model's color, layer, width and line type
Hidden lines enables you to keep only the model's color and layer.

When not selected, the current settings of the drawing environment will be used instead.

Inherit attributes

Inherit attributes

The controls in this area enable you to determine the color inheritance of the edges on a view.

Solid color

Solid color

When selected, the edges (belonging to individual solids) in the view will be displayed in their own solid color.

Closed component color

Closed component color

When selected, the whole view will be displayed in the color of the closed component, overriding the solids colors.

Visible lines:

Visible lines:

Enables you to control the appearance of the lines representing visible edges.

Hidden lines:

Hidden lines:

Enables you to control the appearance of the lines representing hidden edges.

Thin lines:

Thin lines:

Enables you to control the appearance of the lines representing the margins between tangent surfaces.

Color:

Color:

Enables you to select the color for each type of entity by choosing it from the adjacent drop-down list.

Note that when the Keep model attributes option is selected and any of the drawing view representations (Hidden lines or Wireframe) is set, this option is grayed out.

Layer:

Layer:

Enables you to choose an existing layer or type in the name of a new one to create it.

Note that when the Keep model attributes option is selected and any of the drawing view representations (Hidden lines or Wireframe) is set, this option is grayed out.

Width:

Width:

Enables you to specify the width (line weight) of the visible, hidden, and thin lines, by selecting the desired width from the adjacent drop-down list. The program offers four possible line widths.

Note that when the Keep model attributes option is selected and drawing view representation is set to:

  • Hidden lines, this option is enabled.
  • Wireframe, this option is grayed out.

Line Type

Line:

Enables you to specify the line type of each type of entity (the visible, hidden, and thin lines), by choosing the it from the adjacent drop-down list. The program offers sixteen possible line types.

Note that when the Keep model attributes option is selected and drawing view representation is set to:

  • Hidden lines, this option is enabled.
  • Wireframe, this option is grayed out.
