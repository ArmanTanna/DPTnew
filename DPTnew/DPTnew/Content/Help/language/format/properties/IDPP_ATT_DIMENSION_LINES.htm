Entity Properties

Category: Dimension-Line and Arrow

Current

Current:

Enables you to set the current style by selecting it in the list of all the available ones.

Dimension line

Dimension line

The controls in this area enable you to determine the appearance of the Dimension line for the dimensions you create.

Left portion

Left portion

Enables you to display the Left portion of the dimension line depending on the check box status.
When selected, the Left portion of dimension line is displayed.
When not selected, the Left portion of the dimension line is not displayed.

Central portion

Central portion

Enables you to draw the Central portion of the dimension line, depending on the check box status.
When selected, the Central portion is enabled and the central portion of the dimension line is displayed.
When not selected, the Central portion of the dimension line is not displayed.

AutoHide

AutoHide

Enables you to determine the visibility of the dimension line drawn within an arc or a circle.
When selected, the diameter dimension line normally created within an arc or circle is not drawn.
When not selected, this portion of the dimension line is visible.

Right portion

Right portion

Enables you to display the Right portion of the dimension line depending on the check box status.
When selected, the Right portion of the dimension line is displayed.
When not selected, the Right portion of the dimension line is not displayed.

Arrowhead

Arrowhead

The controls in this area enable you to define the arrowhead settings.

Position

Position:

Enables you to determine the Position for the dimension Arrowhead. You can choose one of the following options from the drop down list:

Follow text Enables you to position the Arrowhead with respect to the dimension text.
Outside Enables you to position the Arrowhead Outside the Extension line.
Inside Enables you to position the Arrowhead Inside the Extension line.

Show

Show:

Enables you to select one of the desired options from the drop-down list to Show the Arrowhead for the dimensions.

None Enables you to show None of the arrowheads for the dimension created
Left Enables you to show only the Left arrowhead for the dimension created
Right Enables you to show only the Right arrowhead for the dimension created
Both Enables you to show Both the arrowheads for the dimension created

Size

Size:

Enables you to determine the Size of the arrowhead.

  • If the adjacent check box is selected, the value you type inside the Length box is a multiplying factor that will be applied to the dimension text Size: the resulting value is the length of the arrowheads.
  • If the adjacent check box is not selected, the value you type inside the Length box is the actual length of the arrowheads, in the current unit of measure.

Witness line

Witness line

The controls in this area enables you to determine the appearance of the Witness lines for the dimensions created.

Show

Show:

Enables you to select one of the desired options from the drop-down list to Show the Witness line.

None Enables you to show None of the witness lines for the dimension created
Left Enables you to show only the Left witness line for the dimension created
Right Enables you to show only the Right witness line for the dimension created
Both Enables you to show Both the witness lines for the dimension created

Extend

Extend:

Enables you to determine the length of the Witness line from the point, where the Dimension line touches it. But the value you enter here is related to the Size of the dimension text. The value entered is multiplied by the Size of the dimension text, which is specified under the Text appearance section of the Text category.

Offset from origin

Offset from origin:

Enables you to determine how the witness lines must be drawn with respect to the reference point/origin. Two alternative methods are available, enabling you to set, for the witness lines, either their distance from the reference point/origin or their length:

Variable Enables you to determine the distance between the witness line and the reference point/origin. When this option is selected, the Gap box is displayed.
  • If the adjacent check box is selected, the value you type inside the Gap box is a multiplying factor that will be applied to the dimension text Size: the resulting value is the distance between the reference point/origin and the starting point of the witness line.
  • If the adjacent check box is not selected, the value you type inside the Gap box is the actual distance between the reference point/origin and the starting point of the witness line, in the current unit of measure.
Fixed Enables you to determine the length of the witness lines. When this option is selected, the Length box is displayed.
  • If the adjacent check box is selected, the value you type inside the Length box is a multiplying factor that will be applied to the dimension text Size: the resulting value is the length of the witness lines.
  • If the adjacent check box is not selected, the value you type inside the Length box is the actual length of the witness lines, in the current unit of measure.

Gap

Gap:

Enables you to determine the Gap between the witness line and the origin or the reference point.

  • If the adjacent check box is selected, the value you type inside the Gap box is a multiplying factor that will be applied to the dimension text Size: the resulting value is the distance between the reference point/origin and the starting point of the witness line.
  • If the adjacent check box is not selected, the value you type inside the Gap box is the actual distance between the reference point/origin and the starting point of the witness line, in the current unit of measure.
