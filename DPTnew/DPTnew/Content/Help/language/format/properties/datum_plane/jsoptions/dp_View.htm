This check box enables you to control the representation for datum planes in the model.

When the box is not checked, datum planes are displayed as ordinary surfaces.





When the box is checked, datum planes are displayed in wireframe mode (as a square frame, the line type of which can be set in the Frame line drop-down list) and with two lines to represent the local X and Y directions.



Please note that the View as reference representation is invariant with respect to zooming actions: however you may zoom the model (either by enlarging it or by reducing it), the size of the datum plane is not affected by the action.
