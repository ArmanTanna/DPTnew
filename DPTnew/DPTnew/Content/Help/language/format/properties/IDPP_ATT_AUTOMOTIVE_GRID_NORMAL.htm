Entity Properties

Category: Reference Grid-Standard Format

Standard format label

Standard format label

In this area you are enabled to control the format of the labels to be displayed in the reference grid of the drawing view.

Prefix

Prefix:

Enables you to define a prefix to be displayed in the labels before the coordinate name and the axis value.

Gap from line

Gap from line:

Enables you to define the distance between the label and the endpoint of the associated line of the grid.

Text alignment

Text alignment:

Enables you to specify the angle for the text of the label.

Fixed The text is always horizontal (angle = 0°).
Parallel to line The text is rotated so as to be parallel to the associated line of the grid.
Perpendicular to line The text is rotated so as to be perpendicular to the associated line of the grid.

Axis position

Axis position:

Enables you to specify the position of the axis name in the label with respect to the coordinate value:

Left The axis name is placed before the coordinate value (single line label).
Right The axis name is placed after the coordinate value (single line label).
Above The axis name is placed above the coordinate value (single line label).
Below The axis name is placed below the coordinate value (single line label).

Min. No. characters

Min. No. characters:

Enables you to specify the minimum number of characters to be used for the coordinate value.
If the value displayed in a label has fewer digits than this value, as many blanks as required are added so as to reach it.
This option can be very useful if you want all the circles to have the same size, since the radius of the circle is proportional to the number of characters inside the label.

Circle frame

Circle frame

Enables you to control the visualization of the circular frame for the labels of the reference grid.

  • When the check box is selected, the frame is displayed.
  • When the check box is cleared, the frame is not displayed.

Color

Color:

Enables you to change the color of the circular frame (if any).
When you click on the arrow adjacent to the color box, the color palette is displayed, so as to enable you to make your choice.


Note
Since the reference grid is a group entity, the visualization of the internal color defined for the circular frame depends upon the value of the Show closed groups with group color check box in the Groups category of the Document Properties.
To display the specific colors of the grid, that is the one of the circular frame, the one defined for major lines and the one defined for minor lines, the Show closed groups with group color check box must be cleared.
