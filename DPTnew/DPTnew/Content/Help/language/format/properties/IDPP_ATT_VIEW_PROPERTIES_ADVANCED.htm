Virtual mirror

Virtual mirror

The controls in this area enable you to define the settings of virtual mirror.

Enable

Enable

Enables you to control the status of the virtual mirroring mechanism:

  • When the check box is selected, the virtual mirroring mechanism is enabled.
  • When not selected, virtual mirroring is disabled.

Please note that the virtual mirroring mechanism can also be activated/deactivated by right-clicking on an empty zone of the graphics area with nothing selected and by choosing ViewVirtual Mirror or ViewNo Mirror in the context menu.

For details on the virtual mirroring mechanism, see "Virtual Mirroring".

Representation

Rendering:

Enables you to select the display mode to be used in the view from the drop-down list.

Same as original mirrored entities will be displayed using the same rendering mode as the real ones
Dim wireframe mirrored entities will be displayed in wireframe mode with dim edges
Shaded view and dim wireframe mirrored entities will be displayed using the same rendering mode as the real ones, but with dim edges
Shaded pattern and dim boundaries mirrored entities will be displayed using a shaded pattern and dim edges

For details on the virtual mirroring mechanism, see "Virtual Mirroring".

Color

Color:

Enables you to select from a color palette, a color to use for the Dim wireframe representation of the mirrored entities.

For details on the virtual mirroring mechanism, see "Virtual Mirroring".

Mirror plane

Mirror plane

Enables you to control the visualization of the virtual mirror plane with respect to factors like position, co-ordinates size (X and Y).

For details on the virtual mirroring mechanism, see "Virtual Mirroring".

Show

Show

Enables you to control the visualization of the virtual mirror plane.

  • When the check box is selected, it displays a virtual mirror plane about which a mirrored preview of the model will be displayed.
  • When not selected, the virtual mirror plane is hidden.

For details on the virtual mirroring mechanism, see "Virtual Mirroring".

Position

Position:

Enables you to select a symmetry plane about which a mirrored preview of the model will be displayed (thus showing a virtual image of the portion which is not actually there).

Custom When selected for the first time, the Custom option sets the virtual mirror plane to the Work Plane. Starting from the very moment you initialize the virtual mirror plane to the Work Plane, switching back and forth from Custom does not alter the custom virtual mirror plane. If you change the position of the Work Plane, you can update the position of the virtual mirror plane by clicking As Work Plane.
XZ, XY, ZY the image virtually mirrored about the chosen plane of the World reference system will be displayed

The mechanism can be enabled separately on each view, so that you can have different symmetry planes in different views. Mirrored images are automatically updated at any geometry change and are insensitive to highlighting and selection actions (i.e: you can't select a mirrored entity). As to components, mirroring is enabled only for the ones at top level.

For further details on the virtual mirroring mechanism, see "Virtual Mirroring".

As Work Plane

As Work Plane

Clicking this enables to update the position of the virtual mirror plane when the position is changed.

Size

Size:

Enables to control the scale factors along the X and Y axes.

X

X:

Use the up and down arrows to increase or decrease the scale factor to be used for mirror plane views along the X axis.

Y

Y:

Use the up and down arrows to increase or decrease the scale factor to be used for mirror plane views along the Y axis.
