Entity Properties

Category: Drawing View-Sectioned

Sectioned View

Sectioned View

The controls in this area enable you to define the settings of sectioned views.

Show only sectioned faces

Show only sectioned faces

When selected, only the sectioned faces of sectioned views are displayed; when cleared, both the sectioned faces and profile are visible.

Show separating segments

Show separating segments

When selected, the separating segment is displayed (at each discontinuity point of the section plane) (2).
When not selected the separating segment is not displayed (1).

General

General

The controls in this area determine the positioning of cross-section views.

Cross-section

Cross-section:

Determines the positioning of cross-section views.

Removed on sight direction places the cross-section view aligned with the sight direction of the cutting plane (as shown in blue - 1)
On place places the cross-section view in place within the view from which it was derived (as shown in green - 2)
Removed on center line places the cross-section view aligned with the center line of the cutting plane (as shown in red - 3)

Slices

Slices:

Determines the number of parallel cross-sectional slices created.

Delta

Delta:

Determines the spacing between multiple parallel cross-sections.

Symmetric

Symmetric

When selected, generates multiple cross-sections symmetrically on either side of the reference cutting plane; when not selected, generates multiple cross-sections in one direction from the reference cutting plane.

Display lines

Display lines

When selected, displays all the cutting plane lines used to generate multiple parallel cross-sections; when not selected, only the original reference cutting plane is displayed.
