System Options

Category: Image

Display

Display

The controls in this area enable you to control the visualization of the image.

For further details see "Images".

Representation

Representation

Enables you to control the behavior of the image when zoomed:

Fastpixels are enlarged when the image is zoomed in (also known as 'point sampling', this may result in a jagged appearance)
Bilinear interpolationpixel borders are interpolated/smoothed when the image is zoomed.

For further details see "Images".

Show

Show:

Enables you to control the positioning of the image in the 3D space:

Defaultthe image is an object lying on the Work Plane
Behindthe image is always behind the model (and rotates with it)
Transparentthe image is transparent and you can pick entities through it.

For further details see "Images".

Lighting

Lighting

When selected, image colors are modified according to the current lights.
When not selected current lights do not affect the image.

For further details see "Images".

Pan/Zoom/Rotate

Pan/Zoom/Rotate

When selected, the image is visible when you dynamically rotate/pan/zoom the model.
When not selected, the image is not visible during the dynamical action: it will be displayed when the action is over.

For further details see "Images".

Perspective correction

Perspective correction

When selected, the image is automatically corrected whenever the graphics card does not support the correct perspective visualization.
When not selected, if the the graphics card does not support the correct perspective visualization, the image might not be correctly displayed.

For further details see "Images".

Transparency

100%

Enables you to control the global transparency factor for images (from no transparency - 0% - to complete transparency - 100%-).

To change the transparency level, just move the slider to the corresponding tick using the mouse selection button. Moving from left to right, you can increase the level of transparency assigning all the values corresponding to the intermediate ticks.


Note
Transparent images can be rendered in the model environment only.
