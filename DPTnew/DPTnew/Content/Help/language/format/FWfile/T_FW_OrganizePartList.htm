Details

Part List page configuration

The FormatOrganizePart List command provides you with the Pages category in the Part List Options dialog box. It enables you to divide the complete Part List table into many parts (called pages) to insert them within the drawing frame for better visibility.

First row number Enables you to specify the row number with respect to the complete Part List, which will be the first row of the currently configured page.
Vertical displacement Enables you to specify the vertical offset to be used for the current page with respect to the previous page.
Add Enables you to load the selected first row number and vertical displacement values to the configuration list to create multiple pages.
Remove Enables you to remove the first row number and vertical displacement values from the configuration list.
Load Lists all the loaded page configurations.

All the settings made in the Part List Options dialog box are saved to the main configuration file and are used while inserting a Part List in a drawing.
