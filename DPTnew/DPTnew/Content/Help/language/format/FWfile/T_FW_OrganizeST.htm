Organize Surface Texture Categories

The Organize Surface Textures dialog box enables you to select the desired surface texture standard and define a set of preferred surface designations in order to ease the selection of the needed type when drawing the surface texture symbol. Note that the lists that appear in the selection list options (1st requirement and 2nd requirement) are the ones you have predefined using the Surface Textures command.

The command dialog box provides you with the following controls:

  • Standards
    Reference standards Enables you to choose the surface texture standard you wish to use. After you select a different standard, the command selection list may change to display different options.


  • Surface Requirements
    Type: Enables you to either set the line type for the entities to be drawn or modify the line type of a selected entity.
    Parameter: Enables you to choose the parameter designations from the available list.
    Band: Enables you to choose the transmission band.
    The transmission band includes the cut-off values of the filters (in mm), separated by a hyphen ("-"), the band is indicated first, and the length second.
    U/L: Enables you to choose the type of tolerance limit of the parameter from the drop-down list.
    Filter: Enables you to choose the filter type (X, Gaussian, 2RC) from the drop-down list.
    No. lengths: Enables you to specify the sampling length.
    Max. When selected, applies max-rule   to a surface texture requirement and adds "max" to the parameter designation. For example: Ramax 0,7; Rz1max 3,3v
    When not selected, max-rule is not applied.
    Limit: Enables you to specify the tolerance limit value in micrometers.
    Add Enables to add a parameter designation to the surface texture symbol or list.
    Remove Enables to remove a selected parameter designation from the surface texture symbol or list.


    Also, you can directly define the surface designations in the text box without support of the above mentioned options. All the predefined designations can be added to the list box and thereby they get reflected in the 1st requirement, 2nd requirement drop-down lists inside the command's selection list.

    Note
    For ISO 1302:1978 standard, all the options in the Surface Requirements tab are grayed out.


  • Processes
    Manufacturing process: Enables you to specify the manufacturing process of the specified surface as a text, which is to be added to the complete symbol. For example: Fe/Ni15p Cr r
    Add Enables to add a process description to the list.
    Remove Enables to remove a selected process description from the list.


    All the predefined processes can be added to the list box and thereby they get reflected in the 2nd requirement drop-down list inside the command's selection list.
