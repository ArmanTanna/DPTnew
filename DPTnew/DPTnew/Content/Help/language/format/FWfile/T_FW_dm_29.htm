Property Categories

The Document Properties and Entity Properties (see Different "Properties" dialog boxes) tabs in the Options/Properties dialog box — depending on the way you opened them and on the environment you are working in (Model or Drawing) — can display all or only some of the categories listed below.

Legend
In the following table:
  • A red bullet in the 3D column means the category is available in the Model environment
  • A red bullet in the 2D column means the category is available in the Drawing environment
  • A red bullet in both columns means the category is available in both environments.

Different "Properties" dialog boxes
  • The Document Properties and Entity Properties tabs are displayed when no entity is selected, if you do one of the following:
    • Right-click on an empty zone of the graphics area and select Options/Properties... in the context menu
    • Start the Tools Options/Properties command.
  • As shown in the above table, some categories of the Entity Properties tab are displayed if you pre-select one or more entities and then right-click and select Edit Properties in the context menu. In this case the dialog box name will be Edit Properties.
  • The Assign Properties dialog box is displayed if you start the Assign Properties command.
