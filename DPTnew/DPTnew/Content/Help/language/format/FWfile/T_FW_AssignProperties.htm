ModifyAssign Properties

The Assign Properties command enables you to modify properties of existing entities.

To assign new properties to one or more entities — thus modifying the ones they originally had — proceed as follows:

  • Start the Assign Properties command. The Assign Properties dialog box is displayed.
  • In the Assign Properties dialog box, change the values of the properties you want to modify
  • Move the cursor — which in the meantime has changed to the following:  — onto the entities you want the changes to be applied to and click to select them, or use a multiple selection operator.
  • You can go on by selecting new entities the changes must be applied to. You can also change new values in the Assign Properties dialog box and then select new entities the new changes must be applied to.
Which properties will actually be changed?
Please note that the only properties that will be changed for the selected entities are the ones you explicitly changed in the Assign Properties dialog box, while the ones you didn't change will not be modified, though their values may differ from those displayed in the dialog box.

Applying the command to entities belonging to groups
The command can be applied to entities belonging to groups by using the Select inside Groups command.

Remember you can modify the properties of existing entities also using the Edit Properties command.
