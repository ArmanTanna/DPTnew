Associate a material with a color in a color set

You can associate a material to one or more colors in a color set. Materials can be chosen in a list of predefined ones, depending on the color set.

To associate a material with a color, do the following:

1 When you select the command, the program displays the Colors and Materials dialog box.
This dialog box shows the color set associated with the current document. The color set consists of both the entity colors (a maximum of 63) and the environment colors (the background color and colors used for highlighting entities).
2 Select a color by clicking on the corresponding number on the left.
3 Under Document color, select Material. The Select Material window is displayed.
4 Select the material you want to associate with the selected color.
5 Select OK (You can double-click on the chosen material instead). The chosen material will be displayed in the color list.

Now you can save your changes: note that standard color sets cannot be overwritten. If you were editng a standard color set, you will have to save it with a different name.
