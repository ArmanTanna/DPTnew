The Point Coordinates Label command enables you to display the X, Y and Z coordinates of the referenced point.

Note
The Point Coordinates Label command is provided in an external plug-in. To install it, select Coordinates Label in the Custom branch of the installation procedure.
After installing the plug-in, you will have to carry out the toolbar customization steps to achieve its own toolbar — from which you can launch the Point Coordinates Label command.

AutoArrange

  • If you select the AutoArrange check box in the selection list, the labels are arranged without any overlap; text movement values are not strictly respected because label texts are arranged in a virtual grid of rectangular cells. In case some existing labels are selected, their position can be changed.
  • When creating a new Point Coordinates Label with selection of new entities as well as existing label, text position for this new label is considered only when AutoArrange is not selected.
