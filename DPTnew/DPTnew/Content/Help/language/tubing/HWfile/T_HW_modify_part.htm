Modifying Tubing Parts

Tubing elements are the basic building blocks to form a complete tubing network. A tubing network can be created using the Tubing Network or Insert Tubing Part commands. The following illustration explains how to use the Modify Tubing Part command to modify the tubing elements of a tubing network. Open the Modify_tubing_parts.e3 file available in the Samples folder of your think3 installation and follow the steps given below.

  1. On opening the file you will see the tubing network that was created using the Tubing Network command.



  2. Start the Modify Tubing Part command.

  3. For the Part selector, select the tubing element you want to modify. You can either make selection from the Graphics Area or from the Model Structure. For this tutorial select the Elbow.

  4. The selection list now displays the Line option; it is the same as that used at the time of element creation and is NOT editable.

  5. On expanding the Line node (), the tubing element option is displayed, in this case the Elbow - Curve.

  6. From the drop-down list choose 90" long radius female-female.

  7. Expand the Elbow - Curve node () to display the catalog. Choose the size 1"1/4.

  8. The preview of the modified Elbow is displayed over the element itself.



  9. Apply () the command to make the changes take effect.

Note
You would have noticed that the diameter of all the tubing elements is automatically modified to keep them consistent with the increased size of the Elbow. This behavior was executed only if the tuning network is originally created using the Tubing Network command. If the tubing network is created using the Insert Tubing Part command, you need modify each element individually to keep their diameters consistent.
