ModifyTubingPart

The Modify Tubing Part command enables you to modify a tubing element present in the tubing network. The tubing network could have been created using the Tubing Network or Insert Tubing Part commands.

On starting the Modify Tubing Part command, you need to select the tubing element you want to modify. Based on the element selected, more options will be displayed in the selection list. The settings for these options will be the same as that was used to create the element.

You can now activate/deactivate the presence of flange or access the corresponding catalog and make selection for a new configuration.

Modifying tubing network created by the Tubing Network command

Once you apply the Modify Tubing Part command to make changes in a particular element, the effect of change in diameter (if any) propagates to all other elements and they are also modified to keep the network diameter consistent with the change.

Note
The Modify Tubing Part command affects a single element at a time. However, you can make a generic change in the complete tubing network by redefining the feature. To do so, choose the Redefine Feature option in the context menu displayed by right-clicking on Tubing Network in the Model Structure. This way you can also change the Line Specification, which is not possible otherwise.

Modifying tubing network created by the Insert Tubing Part command

The effect of the change made to a tubing element is exclusive to it. The other tubing elements on the network are not affected by the change. You need to modify each element individually to keep the tubing network consistent.
