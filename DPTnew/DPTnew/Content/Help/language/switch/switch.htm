Switch tool: management of different applications/versions

You may require to have different think3 applications and/or versions installed on the same computer, for instance the latest the think3 application release and an older one for compatibility with subcontractors, and use them at different moments, as well as launch the same the think3 application version with different options (such as the language). Note that the help system displays online documentation in a language different from the user interface. See, troubleshooting for more details.

In order to switch among all these different think3 applications and/or versions, the think3 software installation CD-ROM includes a specific switch tool which automatically resets the registration of the dlls required to run each version and enables you to directly launch the selected version from the dialog box.

In other terms, the Switch takes care of updating the Registry Editor when you need to run a different version of the think3 application or run it in a different language.

In order to run the switch tool, you must be the administrator of your machine or be logged into the system with administrator privileges.


Desktop icons
If you have different version installed in your system you should have different icons on your desktop. If you register one of the software versions through Switch and you try to use the desktop icons to run a different one, you will receive an error message, since the Switch ignores your desktop icons.
