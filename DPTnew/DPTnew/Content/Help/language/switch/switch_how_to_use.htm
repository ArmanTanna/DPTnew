How to use the switch tool

In order to use a version of the think3 application which is different from the last version installed, or use the same version with different launching options, proceed as follows:

  1. Open the "common\utilities" folder within your installation directory structure.
  2. Double-click the file "switch.exe".
  3. Select the version of the software you wish to use from the list of installations.
  4. Set the language and the other launching options you require.
  5. Check the Register action option if you wish to register the appropriate DLL's. This is mandatory if you wish to change the software version or language.
  6. Check the Run action option if you wish to run the software.
  7. Confirm by clicking OK.

The dll registration will be automatically performed, if you chose the Register option, and the selected version of the think3 application will be started, if you chose the Run option.

Note

In order to run the switch tool, you must be the administrator of your machine or be logged in as privileged user.
If you are not, a warning dialog box is displayed when you run the application.

See Further details for more information on the switch tool setup.
