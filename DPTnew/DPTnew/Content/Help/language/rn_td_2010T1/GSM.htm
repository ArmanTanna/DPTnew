New Global Modeling Functionalities and Improvements

Version 2010.1 introduces the following enhancements to the Global Modeling commands.

  • Enhancement in the Planar Bounding-Box and 3D Bounding-Box commands
  • Enhancement in the Advanced GSM and Zone Modeling commands
  • Enhancement in the GSM Fillet Reduction command
  • Enhancement in Quality Checks

Enhancement in the Planar Bounding-Box and 3D Bounding-Box commands

The options in the Planar Bounding-Box and 3D Bounding-Box selection lists have been enhanced so as to have similar capabilities in each command. The Increase Control Points, Decrease Control Points, Edit control points, and Add control points options are now available in both the commands.

Enhancement in the Planar Bounding-Box command

The following two options have been introduced in the Planar Bounding-Box selection list.

Increase Control Points Selecting it increases one control point each in both the directions of the grid.
Decrease Control Points Selecting it decreases one control point each in both the directions of the grid.

Enhancement in the 3D Bounding-Box command

The new Mode drop-down list is available in the 3D Bounding-Box selection list with the following two options.

Edit control points Set this option in order to continue with the modification.
Add control points By clicking on a line between two control points a new grid of control points gets added in the plane perpendicular to the selected line at the pick location.
Note
  • The Update Projection () button is available (in both the command selection lists) after rotating the current view, enabling you to refresh the control point grid for the new view.
  • User modifications are preserved when new control points are added.
  • When control points are decreased, the grid is re-initialized. If you already have made any deformation then a warning is displayed informing that all changes will be lost.

Enhancement in the Advanced GSM and Zone Modeling commands

The Advanced GSM and Zone Modeling commands enable you to specify shape constraints on curves and points. To apply these constraints, you can select Curves or Points in the new Target Shape drop-down list under the Target Shape node.

Once the curve/point selection has been done, the Constraint mini-dialog box is displayed on the curve/point in the Graphics Area and you can select one of the following constraints to be applied from the corresponding drop-down list.

Target Plane The selected boundaries are mapped to a user defined plane
Target Line The selected boundaries are mapped to a user defined line
Target plane + symmetry The selected boundaries are mapped to a user defined plane while maintaining G2 continuity along the boundary. Hence, the resulting surface and its mirrored copy with respect to the plane will be G2 continuous.
Match Plane The selected boundaries are mapped to a plane
Match Sphere The selected boundaries are mapped to a sphere
Match Cylinder The selected boundaries are mapped to a cylinder
Note
For Match Plane, Match Sphere, and Match Cylinder the characteristics of the surfaces are internally computed and not defined by the user.

Enhancement in the GSM Fillet Reduction command

The following options have been introduced under the Advanced Options node of the Fillet Reduction selection list. They enable you to display (preview) the computed preserving, matching & virtual vertices to have a feedback on the effect of various related parameters.

1 Show Matching Curves Enables you to see the preview of curves where matching is applied.
2 Show Virtual Vertex Location Enables you to see the locations where virtual vertices are present.
3 Show Preserving Boundaries Enables you to see the preview of boundaries that are being preserved.

Enhancement in Quality Checks

The Density option has been introduced in the Plane Curvature tab of Quality Checks enabling you to define the curvature plot density. You can define a number in the Density text box to display the same number of plots for each surface section arc.
