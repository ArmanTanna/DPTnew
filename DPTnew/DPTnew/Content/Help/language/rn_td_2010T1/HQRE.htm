New HQ Reverse Engineering Functionalities and Improvements

Version 2010.1 introduces the following enhancements to the HQ Reverse Engineering commands.

  • HQ Reverse Engineering commands in the menu bar
  • Enhancement in the Super Capping command
  • Enhancement in the Mesh Analyzer command
  • The new ReCapping command

HQ Reverse Engineering commands in the menu bar

In this version, the commands in the HQ Reverse Engineering toolbar are also available in the menu bar at the following locations:

  • InsertSurfaceSuper Capping
  • InsertSurfaceReCapping
  • InsertPointSampling
  • InsertPointExtract Zones
  • ToolsInfoAnalysisCheck Symmetry
  • ToolsInfoAnalysisMesh Analyzer

Enhancement in the Super Capping command

The new Symmetry check box in the Super Capping selection list enables you to define the symmetry conditions for the capping surface.

  • When this check box is selected, you will get additional options to define the symmetry plane.
  • When this check box is cleared, no symmetry criteria are applied on the capping surface.

Enhancement in the Mesh Analyzer command

The Mesh Analyzer command now enables you to do noise estimation also for point cloud data. The following two options have been introduced in the Points drop-down list.

  • ASCII file - Enables you to browse for an ASCII file in which the noise will be measured. Also, you can define the Unit of Measurement of the ASCII file.
  • Standalone - Enables you to individually select the necessary points to measure noise on them.

Note
These two new options are available only when the Mode is set to Compute noise.

The new ReCapping command

The new ReCapping command enables you to create a capping surface by using a mesh and some limiting curves as the only input. On applying the command, you get a quality surface representing this mesh inside the defined limits.

This command is a simplified version of the existing Super Capping command and can be applied to relatively simpler cases. However, as the user inputs are very few, its processing is faster.
