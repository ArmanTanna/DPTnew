New Data Exchange Functionalities and Improvements

Version 2010.1 introduces the following enhancements to the Data Exchange functionalities.

  • Save *.e2 file as PDF
  • Save as STL file
  • Enhancement in the Save category of Catia V5 Options, ProEngineer Options, and Parasolid Options dialog boxes.
  • Enhancement in the IGES Options - Load category

Save *.e2 file as PDF

You can save *.e2 files in the PDF (Portable Document Format). The Adobe PDF file (*.pdf) option has been introduced in the Save as Type drop-down list of the file selection box.

To support this conversion process, the ThinkDesign PDF Printer gets automatically installed during the installation process.

Save as STL format

It is possible to save solids, surfaces, and open solids/skin into STL format to convert them into meshes. The current enhancement generates more accurate mesh; as a result, you can get better 3D printing of STL models.

Enhancement in the Save category of Catia V5 Options, ProEngineer Options, and Parasolid Options dialog boxes.

The new Non-manifold solids to surfaces option has been introduced in this category. When selected, it enables you to convert non-manifold solids into surfaces.

Enhancement in the IGES Options - Load category

In this version, the Match UV curves with edge curves option has been introduced in the Healing area of the Load category in the IGES Options dialog box.

A trimmed surface boundary is determined by UV curves and these UV curves need to match with 3D curves. In order to ensure the regeneration of UV-curves from the 3D curves, the new Match UV curves with edge curves option has to be selected.
