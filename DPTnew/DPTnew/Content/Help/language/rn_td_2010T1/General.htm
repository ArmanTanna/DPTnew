General Improvements

Version 2010.1 introduces the following general utility improvements.

  • Enhancement in the ANSYS plug-in installation
  • Enhancement in the Two-Entity Distance Map command
  • Enhancement in the Define Section View command
  • Enhancement in the Scale Entities command
  • Zoom using the entity selector in the selection list
  • Improvement in the Offset on Plane command
  • Enhancement in making Search in Model StructureSearch in Drawing Structure
  • Enhancement in the Selection Filter dialog box

Enhancement in the ANSYS plug-in installation

In this version, you are provided with the choice of installing the ANSYS 11.1 and/or ANSYS 12.1 plug-ins from the custom window of the installation wizard during the installation process. With this enhancement, you are enabled to choose the plug-in that supports the ANSYS application available on your workstation.

Note
Prerequisite of using the ANSYS plug-in is to have the ANSYS application installed in the workstation.

Enhancement in the Two-Entity Distance Map command

For Meshes, a new method of computing distance has been introduced, that detects and takes into account the correspondence between the closest points. The new Closest point option is available in the Method drop-down list. When this option is selected, the nodes will always have a correspondence (except if filtered by the Max. presumed distance option), and the mesh will not have non-computed areas due to missing projection.

In this version, under More Options the Absolute Distance check box has been introduced.

  • When selected, the distance map does not show any relative positioning as the computation is based only on the absolute distance.
  • When cleared, the distance map allows visualizing entities that are crossing each other (or if there are up and down oscillations between them).

Enhancement in the Define Section View command

The Flip Section View Plane check box has been introduced in the Define Section View selection list.

  • When selected, the section view is inverted along the section plane.
  • When cleared, the primary section view is displayed.

Enhancement in the Scale Entities command

The threetwo new Scale text boxes (Scale X, and Scale Y, and Scale Z) are displayed in the Graphics Area on selecting the modeldrawing by using the Scale Entities command. You can scale the model independently in the X, Y, and ZX and Y directions by entering the corresponding values in the Scale X, and Scale Y and Scale Z text boxes, respectively.

Zoom using the entity selector in the selection list

Once an entity has been selected for a selector in the selection list, the context menu on that selector will have the Zoom entities option. On selecting it, the entities selected for the selector will be zoomed in.

Improvement in the Offset on Plane command

It is now possible to offset the curves without converting them into NURBS by using the Offset on Plane command.

Enhancement in making Search in Model StructureSearch in Drawing Structure

The Warnings option has been introduced in the Search in: drop-down list of the Search in Model Structure dialog box. It enables you to make a search only within those features/entities that have warnings on them.

Enhancement in the Selection Filter dialog box

The following are the enhancements in the Selection Filter dialog box.

  • The Component check box has been introduced in the Inside area.
    • When selected, the components inside a component are selected.
    • When cleared, only the top level component is selectable.
  • The Material drop-down list has been introduced in the Attributes area. This drop-down displays the materials applied on the model. It enables you to set the selection filter to a specific material, similar to that of color filter.
