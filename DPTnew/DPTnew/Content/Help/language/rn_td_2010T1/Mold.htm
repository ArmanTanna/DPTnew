New MoldDesign Functionalities and Improvements

Version 2010.1 introduces the following enhancements to the MoldDesign functionalities.

  • The new Region Analysis command
  • Enhancements in the Mold Project command
  • The new Separation Surfaces command
  • Enhancements in the Mold Electrode command
  • Enhancements in the Mold Standard Parts command
  • The new Mold Base sub-commands
  • Enhancement in the Mold Cooling command
  • Enhancement in the Split Mold command

The new Region Analysis command

The new Region Analysis command replaces the former Undercut Analysis command with many more functionalities. This new command enables you to analyze the visible faces of the mold part from multiple directions. The faces are grouped as visible, under draft, undercut, and overlapping faces. On selecting the view direction, you can define with specific colors the faces as the core, cavity, and sliders. Also, you can create skins of the selected faces that can be used by the Region based option in the Split Mold command.

Note
The Region Analysis command is available only with the MoldDesign license.

Enhancements in the Mold Project command

The Shrink part area has been introduced in the Mold Project dialog box. The following two radio buttons are available under the Shrink part area.

  • Isotropic scale: When selected, you can enter the scale value in the corresponding text boxes. The mold part will be scaled in isotropic mode in X, Y, and Z directions.
  • Anisotropic scale: When selected, the three directional axis text boxes are available. You can enter the specific scale factors in the respective Scale X, Scale Y, and Scale Z text boxes.

Enhancements in the Mold Electrode command

The following enhancements have been made in the Mold Electrode command.

  • The new None option in the Head Shape drop-down list enables you to create an electrode without head part.

  • The Show all reference points check box has been introduced in the selection list.
    • When selected, the reference points of the electrode head are highlighted in the Graphics Area.
    • When cleared, the reference points are not highlighted.
    On double-clicking any of the reference points, a datum point is created at the same location. The datum point can then be used for dimensioning the electrode by using the Enable for electrode origin option.

  • The new Precision check box enables you to select the precision value for the distance between the electrode head and the Work Plane origin.
    • When selected, its drop-down list is activated from which you can choose the precision value.
    • When cleared, you cannot choose the precision value.
  • The new End Axis Alignment node enables you to position the cubed electrode head to the required direction. The alignment can be done by using the X Axis and Y Axis options under the End Axis Alignment node.

    Note
    The End Axis Alignment node is available only when you select Cube from the Head Shape drop-down list.

Enhancements in the Mold Standard Parts command

The following enhancements have been made to the Mold Standard Parts command.

  • You can delete individual standard parts from the mold by right-clicking on the standard part component in the Model Structure and selecting Delete from the context menu.
  • You can insert a new mold catalog and custom part catalog by editing the index of suppliers in the \files\MoldDesignData\StdPartCatalogIndexMap.xml file in your installation.
  • Using the new Customize option in the Part drop-down list, you can integrate the customized catalog into the command.

The new Mold Base sub-commands

The Mold Base command has been completely overhauled into an advanced and easy to use feature. The following are the new enhancements.

  • The DME, HASCO, Futaba, Pedrotti, and Meusburger commands have been introduced, which represent each mold base supplier.
  • The Mold Base wizard has a highly interactive user interface and is customized to their respective catalog manuals.
  • In the Mold Base wizard, when you select a specific mold series or assembly, then the wizard automatically lists the related values in the corresponding tabs for selection. The pictorial representation of the different parts of the assembly enables you to make the selection easier.
  • The selected parts are listed in the left pane of the wizard, which enables you to view the complete mold base configuration.

Enhancement in the Mold Cooling command

The following are the enhancements in the Mold Cooling command.

  • The preview of the standard parts is displayed on the selected cooling path.
  • You can change the standard parts in the cooling path by using the Information on Hole dialog box.

Enhancement in the Split Mold command

The new Mode drop-down list enables you to choose the method of creating the core and cavity by using the Split Mold command. The following are the two options available:

  • Automatic: When selected, the core and cavity are created as it used to happen in previous versions.
  • Region based: When selected, the core and cavity are created using the region skins formed by the Region Analysis command. The new Group node enables you to select multiple regions skins by using the Region selector.
