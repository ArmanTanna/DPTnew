New Curve & Surface Functionalities and Improvements

Version 2010.1 introduces the following enhancements to the Curve & Surface commands.

  • The new Approximation by Lines-Arcs command
  • The new Geodesic Curve command
  • Enhancement in the Invert Surface command
  • Enhancement in the Global Sweep command
  • Enhancement in the Rotational Sweep command
  • Enhancement in the Connect Curve command
  • Enhancement in the Modify Curve through Control Points command
  • Enhancement in the Modify Surface through Control Points command
  • Enhancement in the Advanced Surface Continuity command
  • Enhancement in the Split Surface with Silhouette command

The new Approximation by Lines-Arcs command

The new Approximation by Lines-Arcs command enables you to convert a generic planar curve into a sequence of line segments or a combination of circle arcs and line segments. This is useful as a tool path input for NC machines.

You can specify the Linear Tolerance value to limit the deviation from the original curve. For arcs, you can also define the Min. radius and Max. radius values.

The new Geodesic Curve command

The new Geodesic Curve command enables you to create geodesic curves on a surface. It represents the shortest path between points on the surface.

To create a geodesic curve, you need to define the point on the surface through which the geodesic curve must pass and the direction of the geodesic curve at its initial point.

This command can create multiple geodesic curves simultaneously, and also insert them on the surface to generate a COS (Curve On Surface). The following image shows single and multiple geodesic curves created over the surface in X-direction.

Enhancement in the Invert Surface command

The Invert Surface command now enables you to apply a global inversion to all normals to make them point towards the same direction. As a result, you do not have to select the normals individually to invert their direction. The inversion takes place based on one of the following options selected for the new Invert Normals drop-down list in the Invert Surface selection list:

Manually You need to manually invert the normals; automatic inversion is not applied.
By direction Normals are oriented towards the specified direction
By axis Normals are oriented towards the defined axis
By point Normals are oriented towards the defined point
By skin Individual normals of surfaces forming the skin are oriented in the same direction.

Refer some examples below:

Set of surfaces Orientation of surface normals
Normals oriented towards the Y-direction Normals oriented in reference to the axis

The direction of all the normals can be simultaneously inverted by using the new Exchange All Normals option.

Enhancement in the Global Sweep command

In this version, the AutoPlace option is available in the context menu of the Curves selector instead of the Section Group(s) node. The same is true for the AutoInvert and AutoPlace option which is displayed when the Motion Mode is Bi-driven.

 

Also, the context menu of the Curves selector has the following new options:

  • Show Invert and Place
  • Show All Invert and Place
  • Hide Invert and Place
  • Hide All Invert and Place

With the introduction of above mentioned options, the old behavior of displaying Position of Section and Invert on all sections by expanding the More Options node has been removed.

 

In this version, the new Transition drop-down list replaces the With Transitions check box. The following options are available in it.

None No transition is applied.
Global mode The effect of transition is computed by the global section interpolation process.
Local mode A local computation of transition is performed that is effective in a restricted area around the transition, as shown in the following images.
Local mode Global mode

 

Using the following options in the new Restrict drop-down list under More Options, you can restrict the development of the sweep surface within limits.

  • None - No restriction is applied and the sweep will be created over the complete drive curve.
  • Between parameters - The sweep is limited between two user defined points on the drive curve.
    Regular Complementary
  • Between sections - The sweep is limited between the first and the last section curves.
    Regular Complementary

The complementary sweep is obtained by selecting the Complementary Shape check box under the Restrict node.

Enhancement in the Rotational Sweep command

The Rotational Sweep command now enables you to perform GSM based transformations while maintaining additional tangency constraints. For example, a section with a tangent perpendicular to the axis will be swept in such a way that it remains perpendicular throughout the sweep. Two Constraint drop-down lists (mini-dialog box) have been introduced, which are displayed over the section curve and drive curve, respectively.

The Constraint drop-down list for the section curve has the following options:

Keep position Sweep surface boundary is unchanged.
Keep position + tangent Sweep surface boundary is unchanged and also its tangency field is preserved.

The Constraint drop-down list for the drive curve has the following options:

Target position Sweep surface is transformed to match the drive.
Target position + keep tangent Sweep surface is transformed to match the drive and also its tangency field is preserved.
Target position + target tangent This option is available only if the drive curve is a surface boundary. Sweep surface is transformed to match the drive and ensure tangent continuity with the boundary surface along the drive.

Enhancement in the Connect Curve command

The Connect Curve command now enables you to create the connecting curve such that it fits with an existing curve. The Fit Curves option has been introduced in the Edit Mode drop-down list of the Connect Curve selection list. On selecting the Fit Curves option, the Reference Curves selector is available; through it you can select the curve to fit. The weights of the connecting curve are automatically calculated to generate the best possible fit.

Enhancement in the Modify Curve through Control Points command

The Mirror mode option in the Modify Curve through Control Points command now supports NUBS (B-Spline without rational parts) curves.

Enhancement in the Modify Surface through Control Points command

The following two options have been introduced in the Continuity drop-down list under the Mirror mode node of the Surface through Control Points selection list.

Smooth curvature Enables you to maintain smooth curvature continuity between the original surface and its virtual mirror.
Perpendicular tangent Enables you to maintain tangent continuity between the original surface and its virtual mirror, such that the tangent is perpendicular to the virtual mirror.

The Minimize drop-down list is now available under the Advanced Options node. The various options in it enable you to define additional criteria according to which the control points are moved to achieve the specified continuity constraint.

The Mirror mode option now supports NUBS (B-Spline without rational parts) surfaces.

Enhancement in the Advanced Surface Continuity command

Previously, the Advanced Surface Continuity command applied the Continuity constraints to match a curve. Now, it is also possible to match a target shape. The Mode drop-down list has been introduced in the Advanced Surface Continuity selection list with the following two options.

Curves - The continuity constraint is maintained between selected curves. (It corresponds to the previous behavior of the software.)
Shape - The continuity constraint is maintained between the selected curves and a shape. The shape can be a user defined or an automatic one depending on the constraint selected in the Continuity drop-down list. It is displayed in the Graphics Area as a mini-dialog box over the boundary curves to be modified. Some additional selectors can be displayed in the selection list depending on the type of selected constraint.

The various available constraints in the Continuity drop-down list are:
Target Plane The selected boundaries are mapped to a user defined plane
Target Line The selected boundaries are mapped to a user defined line
Target plane + symmetry The selected boundaries are mapped to a user defined plane while maintaining G2 symmetry along the boundary. Hence, the resulting surface and its mirrored copy with respect to the plane will be G2 continuous.
Keep position + make symmetric Existing planar (boundary) curves keep their position while obtaining the G2 symmetry along the boundary
Keep plane + make symmetric Existing planar (boundary) curves are "allowed" to move in its plane while obtaining the G2 symmetry along the boundary
Match Plane The selected boundaries are mapped to a plane
Match Sphere The selected boundaries are mapped to a sphere
Match Cylinder The selected boundaries are mapped to a cylinder
Note
For Match Plane, Match Sphere, and Match Cylinder the characteristics of the surfaces are internally computed and not defined by the user.

Enhancement in the Split Surface with Silhouette command

The Hide check box has been introduced under More Options in the Split with Silhouette selection list.

  • When this check box is selected, the original surfaces are made hidden and only the preview is visible.
  • When this check box is cleared, the original surfaces are visible along with the preview.

You can hide and show the preview by using the Hide Preview () and Retrieve Preview () buttons, respectively.
