How to Go On


In order to complete the execution of the command, you have to:

  • Supply all the input arguments that are marked in red in the selection list:
  • When the input of the last argument is required, the marker turns to
  • Apply the command by specifying the last argument. If it is a point, click the mouse or hit ENTER (the cursor position will be achieved).
  • Once you have confirmed, the input argument markers turn to again and the command is ready for a new execution.
  • Click to quit the command.

Before applying the command, you always can:

  • Change the selected entities (right-click on the corresponding selector, choose Reset in the context menu and perform selection again)
  • Change the values of arguments you have already entered
  • Set the values of optional arguments.
