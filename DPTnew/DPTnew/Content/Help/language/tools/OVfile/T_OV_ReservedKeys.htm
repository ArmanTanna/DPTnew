Reserved Keyboard Keys

The following keyboard keys are reserved by the program for its own internal use and cannot be customized with shortcuts of any kind.

Keyboard Key Common behavior Details and related topics
Backspace Deletes the current selection. When editing a field, it deletes the selected text or the character to the left of the cursor.
B Blocks the value in a mini-dialog box.
Ctrl+C Executes the Copy command.  
Ctrl+F Executes the Front View command.  
Ctrl+H Executes the Highlight Dimensions command.  
Ctrl+N Enables you to open a new document.  
Ctrl+O Executes the Open command.  
Ctrl+P Prints the active document. The current Print Setup options are used without interaction with the user.
Q Starts the Select command. Enables you to perform a quick editing of the Work Plane.
Ctrl+R Executes the Right View command.  
Ctrl+S Executes the Save command.  
Ctrl+T Executes the Top View command.  
Ctrl+V Executes the Paste command.  
W Executes the Work Plane command.  
Shift+W Executes the Absolute Reference System (World) command.  
Ctrl+X Executes the Cut command.  
Y Toggles the display of the Profile Reference System. Same behaviour is executed by the use of the 3D Reference system check box in the System Options - Profile category.
Ctrl+Y Executes the Redo command.  
Ctrl+Z Executes the Undo command.  
Del Deletes the current selection. When editing a field, it deletes the selected text or the character to the right of the cursor; when editing a model or drawing, starts the Delete command.
Enter Terminates the data input and applies the command. Overview table on how to use the keyboard
Esc Aborts the command. Overview table on how to use the keyboard
F1 through F12 Various functions depending on the context. Shortcut Function Keys
Shift+N Executes the Snap to None command.  
Shift+R Executes the Redraw All command.  
Space bar Starts an editing session. Starts the input on the mini-dialog with the focus enabled when a command with a selection list is active. When no command is active, it moves the focus to the command line of the I/O Window if the latter is displayed.
TAB, Shift+TAB, Ctrl+TAB Moves onto the next field when entering data. Overview table on how to use the keyboard
