Using the Spreadsheet

The program enables you to define variables, which can then be used to interrelate the model's various parametric dimensions. Parametric dimensions can be accessed from the model's related drive dimensions, which are created and assigned automatically when you create a parametric solid or feature or add dimensional constraints to a profile.

By assigning different values to variables, you can create different models from a single basic model.

Variables can be controlled (assigned, deassigned, modified, deleted, and so on) from the Spreadsheet dialog box.

The Spreadsheet dialog box contains a working area, where you can manage variables.

The Spreadsheet work area contains the following fields:

Type Displays the type of the variable. There are five kinds of variables: Standard, Dependent, Set, Range and Measure. To know how to define these variables, see Defining a variable.
Name Enables you to enter a name for the variable.
Unit Enables you to select an unit for the variable.
Expression Enables you to enter an expression for the variable. Note that the total length of the expression cannot exceed 256 characters.
Value Displays the result of the expression.
Used Displays if the variable is used in another variable.
Assigned Displays the number of time the variable in assigned in the model.
Min Enables you to enter a minimum value that you can use.
Max Enables you to enter a maximum value that you can use.
Set Enables you to specify the value set for the variable, by entering one value in a row.
Comment Enables you to enter a comment for the variable.
