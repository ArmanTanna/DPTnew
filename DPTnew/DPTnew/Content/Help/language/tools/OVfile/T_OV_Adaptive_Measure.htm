Integrating the Adaptive Measure mechanism in think3 application commands

You can use the Adaptive Measure (AM) mechanism for any command that uses mini-dialogs with a numerical input. You can enable the AM mechanism through the context menu on the Graphic Area and in the mini dialog box.
