The Toolbars tab of the Customize dialog box gives you full control on the program default and custom toolbars. You can activate and deactivate, create, delete and rename toolbars, and choose the button Look: and Size:.

  • A complete scroll-down list of the available toolbars — both built-in and custom — is available in the Toolbars: frame. By checking or clearing the box adjacent to each name in the list you can turn the corresponding toolbar respectively on or off.
  • Three buttons are available just above the frame:
    New Enables you to create a new custom toolbar. When you click the New button you will be required to type in the name of the new custom toolbar. Once you've done so, you will be enabled to add buttons to the new toolbar.
    Reset Enables you to reset a default toolbar to its original status, if you added or removed buttons. To reset a default toolbar:
    1. Select the toolbar in the Toolbars: scroll-down list. The name will be displayed in the Toolbar name: box just below the frame.
    2. Click the Reset button: the toolbar will be reset to is default status by removing any added button and by restoring any removed one.
    Delete Enables you to delete a custom toolbar.
    1. Select the custom toolbar in the Toolbars: scroll-down list. The name will be displayed in the Toolbar name: box.
    2. Click the Delete button: the custom toolbar will be deleted.
      If you select a built-in toolbar the Delete button is grayed out, as only custom toolbars can be deleted.
  • In the Button settings area, you can set:
    • The button Look:, which can be New or Old. Please note that your choice can be interactively assessed — before applying it —  in the Example: area.
    • The button Size: —  In this case the choice depends on the chosen Look:: Small, Medium or Large, when the Look: is New; Small or Large, when the Look: is Old.
