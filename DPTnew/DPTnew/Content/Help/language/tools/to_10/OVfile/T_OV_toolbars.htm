Toolbar specific actions

The last button on the right side of each toolbar is a drop-down button. It can be displayed in the two alternative ways shown below:
 

A More Buttons downward arrow is displayed in the button. In this case, you can always:
  • Add or remove buttons: you can add new buttons to the toolbar in use or remove existing ones from it.
  • Reset the toolbar: you can restore the original buttons of the toolbar in use, in case the latter had been modified by adding or removing buttons.
  • Customize the toolbar: you can access the Toolbars tab of the Customize dialog box.
In the More Buttons downward arrow, an additional "double" arrow () is displayed. This happens if you have reduced the size of the think3 application's window. In this case toolbars keep their original position — that is, they are not moved around — and are truncated on their right side to match the new window size (that is: some buttons are hidden).
The three functionalities described above are still available. The meaning of the additional arrow is there are hidden buttons, which you will be able to access by right-clicking.
Floating Toolbars
When a toolbar is floating, a special button with a small arrow on the right side of the banner enables you to access all the functionalities mentioned above.

Drop-Down buttons

Drop-Down buttons are buttons with an arrow marker in the bottom-right corner indicating they give access to a set of different commands. In order to display the drop-down list of commands they give access to, right-click on the button. The list will be displayed.

Left-click on an item in the drop-down list to start the corresponding command. The icon of the latter will also be displayed in the drop-down button and will be the current one until you select a different command in the list.
