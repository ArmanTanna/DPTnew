Customizing context menus

You can customize context menus by adding or removing commands to/from them.
Once you have added a command to a context menu, whenever you open that particular context menu (that is, whenever you are in the same situation — context — and right-click), the command you added will be available in the context menu, until you decide to remove it.
Very similarly, once you have removed a command from a certain context menu, whenever you open that menu the command will not be available anymore, as long as you don't add it again or restore the menu to its default.

Context menus depend on GUI profiles.

If you change the current GUI profile, any context menu customization you had made when the current GUI profile was a different one will not be available.

Customizations are stored among the user data and can be easily shared with other users through the Configuration Manager (see the Customized Context Menus component under GUI Customization in the Configuration Options).

How to add commands to a context menu

To add commands to a context menu, proceed as follows:

  1. Right-click to display the context menu and select Customize:



    The Customize dialog box shows up adjacent to the context menu. Select the Commands tab.


     
  2. Select the icon of the command you want to add to the context menu and drag it onto it.


     
  3. Drop the command icon onto the context menu.


     
  4. Select Close. The new command has now been added to the context menu.

    Adding drop-down buttons to a context menu
    Please note that also drop-down buttons can be added to context menus.



    An appropriate sub-menu will be added to the customized context menu.

How to control the applicability of context menus

If you have added a command to a context menu in a certain situation, you may want to extend its applicability to different situations.
For example, if you added the command to a context menu when no entity was selected, you might want to get the same command also when one or more entities are selected. In these cases, once you have added an item to your customized context menu, you can right-click on the newly added item and select Set Conditions, which will enable you to:

  • Control the number of selected entities. If you have selected just one entity, you might want the command to show up also in case you selected any or a specified number of entities.
  • Control the type of entities it applies to (for example, if you selected a line, you might want the command to show up also when you select a curve; if you selected no entities, you might want the command to be available also when an entity of some or all types is selected).
    You will also be enabled to distinguish between associative and non-associative entities (suppose you selected an associative surface: you might want the newly added command to be displayed also when you click on a non-associative surface, or vice versa).
  • In case of surface selection, to extend the applicability from trimmed to non trimmed surfaces and vice versa.

Suppose you have added a command to a context menu. To change the applicability context for the newly added command, before closing Customize dialog box, proceed as follows:
 

  1. Right-click on the newly added command and select Set Conditions.



    The Set Conditions dialog box is displayed.



  2. In the Number of entities drop-down list, select to how many entities the selected command must apply:



    Any The command will apply to any number of selected entities (≥0).
    Greater/equal than The command will apply to any number of selected entities equal or greater than the one you can specify in the adjacent box on the right.
    Equal The command will apply only to a number of selected entities that is the same as the one you can specify in the adjacent box on the right.
    Less/equal than The command will apply to any number of selected entities equal or smaller than the one you can specify in the adjacent box on the right.
    Note that the initial value displayed in the box is the number of entities you actually selected when you started the customization of the context menu.
     
  3. Unless the Number of entities reads Greater/equal than 0 (in which case the area below is grayed out), you will be enabled to choose the entity types the command must apply to by properly setting the controls of the Entities area.
    For example, if you check the All box, the command will be displayed for any type of entity you may select.



    If the box is cleared, three different drop-down lists are available for you to choose three different types of entities (the same types available with the Selection Filter command).



    If the type you select is curve or surface, you will also be enabled to select the sub-type in the Subtypes drop down list.
    Specific warnings will be displayed if you select the same sub-type more than once.
     
  4. If the type you selected is either curves or surfaces, the Associative and Non-associative are available to further filter the applicability of the newly added command.
    • When the Associative check box is selected, the command will be available when the selected entities are associative.
    • When the Associative check box is cleared, the command will not be available when the selected entities are associative.
    • When the Non-associative check box is selected, the command will be available when the selected entities are non-associative.
    • When the Non-associative check box is cleared, the command will not be available when the selected entities are non-associative.
       
  5. Finally, if the selected entity type is surfaces, then you can further filter the applicability of the newly added command to trimmed or non-trimmed surfaces.
    • When the Trimmed check box is selected, the command will be available when the selected surfaces are trimmed.
    • When the Trimmed check box is cleared, the command will not be available when the selected surfaces are trimmed.
    • When the Not trimmed check box is selected, the command will be available when the selected surfaces are non-trimmed.
    • When the Not trimmed check box is cleared, the command will not be available when the selected surfaces are non-trimmed.
       
  6. Click OK to apply your changes and close the dialog box.

How to remove commands from a context menu

To remove commands from a context menu, proceed as follows:

  1. Right-click to display the context menu and select Customize. The Customize dialog box shows up adjacent to the context menu.
     
  2. Select the icon of the command to be removed from the context menu and drag it outside the menu, then drop it.



    The command will be removed from the menu.


     
  3. Select Close. The new command has now been removed from the context menu.

Other customization commands

If you right-click on the context menu to be customized after selecting the Customize command, a context menu is displayed.



New Item Enables you to add a new command to the selected context menu just on top of the selected item. Once the new item has been added, you can edit its properties using the Properties command (see below), thus creating macro commands with their own contents, labels and parameters.
Add Separator Enables you to add a separator line to the context menu just on top of the selected item.
Delete Enables you to remove the selected item from the context menu.
Restore Displays a dialog box on which you can choose to:
  • Restore the selected context menu to its original version
  • Restore all customized context menus to their original version.
Properties Displays a dialog box enabling you to change the properties of a selected item in the context menu in terms of functional contents, label and parameters.

Select Close to apply your changes and close the Customize dialog box.
