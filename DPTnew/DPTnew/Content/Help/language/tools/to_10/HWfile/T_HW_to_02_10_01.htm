Customize Commands - Add a command button to a toolbar

1 Under Categories, select a category to display the buttons for commands in that category.
2 Pause the cursor over a button to display a ToolTip identifying its command name. Click on the button to display its description in the Description box.
3 Click and drag the desired command button and drop it onto the toolbar on which you want it to appear.
Note that if you drop the command button onto an area of the screen where there is no toolbar, the program creates a new toolbar containing that button. The name Toolbar # is assigned to the new toolbar, where # is a number starting from 0 and automatically incremented for successive toolbars so that each toolbar created in this way is assigned a unique name.
