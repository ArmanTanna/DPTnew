The Info command provides access to the following set of commands which display various types of information about the objects in the model/drawing:

Single Entity Displays specific information about a selected entity
Two Entities Displays geometric information about pairs of entities
Two-Entity Distance Map Computes the distance between two entities that can be meshes, surfaces and curves, providing result in form of a graph and of a color map.
Zebra Displays zebra lines so as to assess the shape of surfaces
Curvature Displays the curvature plot of a selected curve
Analysis Provides access to a set of functions used to analyze curves, surfaces and solids
Check Continuity Enables you to check the continuity between two curves or two surfaces.
Check Interferences Provides information about interferences among solids, components and sets of solids and components
Entity Relations Enables you to retrieve information about the history of the selected entity belonging to the Model Structure
Tessellation Displays the tessellation on all the entities
Mass Properties Calculates and displays the mass properties of one or more selected solids and/or surfaces
Area Properties Calculates and displays the area properties of a selected X-hatch
Underconstrained Profiles Enables you to check the degrees of freedom of all the underconstrained profiles on current output layers of the current model
Image Displays the information about an image, if one is inserted into the application
