Display information about entities

You can display information about entities using either of the following methods:

  • Choose the Info command from the menu and then select the entity.
  • Select the entity and then right-click and choose the Info command from the context menu.

If you access the Info command from the menu, the program displays a drop-down menu from which you can choose the specific information command. After choosing the command, you must select the entities whose information you want to display.

If you select the entities first and then right-click to display the context menu, the options displayed by the Info command depend upon the entity or entities you selected. The following situations are possible: If you have selected the entity first and then called up the context menu:

  • If you selected a two-dimensional entity such as an arc or line, only the Info Single Entity command appears.



  • If you selected a three-dimensional entity such as a surface or solid, both the Single Entity and Mass Properties commands are available.



  • If you selected two two-dimensional entities, only the Info Two Entities command appears.
