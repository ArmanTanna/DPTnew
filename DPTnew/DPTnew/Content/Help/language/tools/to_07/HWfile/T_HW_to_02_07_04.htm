Display geometric relation between a pair of curves

To start the Two Entities command, do one of the following:

  • From the menu, choose the Two Entities command, and then select the two entities whose information you want to display.
  • Select the two entities you want information about, and then right-click and select the Info Two Entities command from the context menu.

If you select two curves, the program displays the Two Entities Info dialog box and also displays a circle around the two closest points on the curves, as well as a dashed line indicating the distance between them.


If you select two curves that are not coplanar (curves lying in two planes) then the projection distance of the first curve to the second one will also be shown.

When applying the command to two entities belonging to two different shared groups:

  • Specific info will be displayed in the case the two selected entities belong to shared groups with different scale factors
  • The Scale: factor will be displayed in the case the two selected entities have the same scale factor, but such factor is different from 1.
