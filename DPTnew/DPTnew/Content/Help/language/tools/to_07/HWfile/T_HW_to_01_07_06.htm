Display information about a selected Profile

The Single Entity command can be used to display various information regarding profiles. The command can be invoked in the following ways inside the program:

  • Single Entity command ( Info Single Entity):

    • To view the feedback on profiles, right-click a profile in the Model Structure and select Info Single Entity. The Single Entity command on a profile gives the feedback in a Single Entity Info dialog box and contains information on items such as Internal entities:, External entities:, Dangling ext. reference(s): and so on.




  • By right-clicking on the required items

    • You can also right-click on a entity inside the Profile environment and select Info Single Entity. The Single Entity Info dialog box then will display the information regarding the entity accordingly.



The information is available as:

  • Entity type:

    • Displays the type of the entity that you selected; that is, whether it is a profile, a constraint (for example, tangency), and so on.
  • Internal entities:

    • Displays the number of internal entities of the selected profile.
  • External entities:

    • Displays the number of External entities of the selected profile. Select the Show check box to display them.
  • Under Constraints
    Degrees of freedom: Displays the number of degrees of freedom of the selected profile
    Constraints: Displays the number of constraints currently applied to the selected profile
    Dangling ext. reference(s): Displays the number of dangling references
  • Under Loops:
    Loops: Displays the number of islands of the selected profile
    Open profile This check box is selected only when the selected profile is open
  • Length:
    Displays the length of the selected profile.
