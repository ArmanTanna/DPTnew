Calculating and displaying the Area Properties of a selected X-Hatch

1

After starting the Area Properties command, the program prompts you to select the X-Hatch on the document.

2 Once you select the X-Hatch, the Area Analysis dialog box pops up, which displays the following information of that particular X-Hatch:
  • Total area
  • Total perimeter
  • Center of the selected area
  • Moments of inertia
  • Principal axes and products of inertia

The UI of the Area Analysis dialog box is shown below:
3 By clicking on the Units.. button on the Area Analysis dialog box, another dialog box pops up, which helps you to change the unit of measurements.
4 Now you are enabled to choose from any units of measurement of your requirements, that is available on the Area Properties Measurement Units dialog box. Once the desired unit is chosen, you can click on any one of the following buttons:
  • OK - to apply the selected units only to the current calculation.
  • Save - to save the new settings so that it can applied to all subsequent calculations.
  • Reset - to reset the settings to their previous units.
  • Default - to restore the program's original settings.
5 Once the desired unit is set, you have the option to save the Area Properties into a file by clicking on the Save As.. button on the Area Properties dialog box. In the Save as type drop-down list, you can specify whether the data is to be saved as a word document(*.doc) or as a simple text file(*.txt).
