Display information about a selected NURBS curve

If you select a NURBS curve, the program displays a dialog box showing:

  • Coordinates at Endpoint 1
  • Coordinates at Endpoint 2
  • Number of Arcs
  • Degree of the curve
  • Continuity of the curve
  • Length of the curve
  • Flags indicating if the curve is Closed, Rational, Planar

Numbers within the model or drawing are also displayed to identify the first and second endpoints of the curve.

If the selected entity belongs to a shared group and the group has been rescaled (for example, if it has been created using the Group Copy command and a scale factor — different from 1 — has been specified) the Scale factor will be displayed along with the other output data.

The check boxes under Show enable you to display additional components of the NURBS curve:

  • Control points — displays control points.



  • Arcs — displays arc joints.



  • Cusps — displays sharp corners.



  • Inflexion points — displays inflection points.



If you click on the Create points button, a point entity is created at each of the points shown.
