ToolsInfoSingle Entity


 

The Single Entity Info command displays specific information about a selected entity. Data displayed depend on the type of the selected entity:

  • Point
  • Line
  • Arc
  • Ellipse
  • NURBS curve
  • Curve on Surface
  • Profile
  • Plane
  • Cylindrical surface
  • Ruled surface
  • Rotational surface
  • Proportional surface
  • Coons surface
  • Fillet surface
  • NURBS surface
  • Solid
  • Mesh
  • Group
Applying the command to an entity belonging to a shared group
If the selected entity belongs to a shared group and the group has been rescaled (for example, if it has been created using the Group Copy command and a scale factor — different from 1 — has been specified) the Scale factor will be displayed along with the other output data.
Applying the command to an entity belonging to a drawing view
For drawing views the dialog shows an additional information about the applied value of small entities filter.
In case of faster hidden line computation, each generated view will be associated with a precision value and the entities smaller than this value cannot be excluded by hidden line algorithm. The precision value will merely depend on the size of the model, the Graphics card and the kind of drawing view.
