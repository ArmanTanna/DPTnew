Tools Info Two-Entity Distance Map


 

The Two-Entity Distance Map command enables you to compute the distance between two entities that can be meshes, surfaces and curves, providing result in form of a graph and of a color map.
After starting the command, proceed as follows.

  • Select the first set of entities (First Entities). In the following example it is a curve.



    By right-clicking on the selector and choosing Curves Only you can restrict selection to curves (surface boundaries, solid edges). You can reset the selection back to curves, surfaces and meshes by selecting Curves-Surfaces-Mesh.



    End selection by right-clicking and selecting Continue in the context menu.
  • Select the second set of entities (Second Entities). In the following example it is a surface.



  • In the Method drop-down list select the computation you want the application to perform and then click the preview button (  ):
     
    Normal When this option is selected the Normal To drop down list is displayed:
    First Entities To compute the values of distance along the direction normal to the first set of entities.
    Second Entities To compute the values of distance along the direction normal to the second set of entities.
    Pierce direction To compute the values of distance along a pierce direction to be defined (for details see "How to define an axis or a line" in the online documentation.
    Node correspondence To display the correspondence between the nodes of the two selected meshes (this option is available only if both selected entities are meshes with identical topology (same number of nodes and faces).
    Mesh-Mesh Curve-Surface

    The values displayed include:
    • the minimum and maximum distance between the two entities
    • the absolute and algebraic mean values
    • a graph of the distribution.
       

    By choosing Repartition in the Type drop-down list under Graph you can also get the graph of the distance repartition.

    Distribution Repartition
     
  • The following check boxes are available under More Options:
    Hide Enables you to control the visibility of the second entities:
    • When the box is checked, the second entities are not visible.
    • When the box is cleared, the second entities are visible.
    Max. presumed distance Enables you to reject distance values higher than the value you specify in the box (that is a tolerance value).
    Curve sampling density Enables you to define the number of points to be used for the discretization of curve. The check box is available only when at least a curve is selected.
    Surface sampling density enables you to define the number of points to be used for the discretization of the surface. The check box is displayed only when at least a surface is selected.
