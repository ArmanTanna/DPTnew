The Draft analysis command enables you to analyze draft angles.

Note

Draft analysis is always performed in the Work Plane reference system.

The program applies shading to the selected surface to highlight those zones where the draft angle does not lie within the specified values. This is particularly important in mold engineering, since draft enables a part to be removed from a mold. The program produces a diagram of the measured values and assigns the color blue to the smallest draft angle and red to the largest draft angle.
