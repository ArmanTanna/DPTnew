ToolsInfoAnalysisArea Properties


 
The Area Properties command enables you to calculate and display the area properties of a selected X-hatch.

Note

Tolerance used when calculating areas and volumes is defined by the Tolerance option in the General - Advanced category of the Document Properties.

By increasing this tolerance value you reduce execution time but also execution precision. By reducing tolerance you improve execution precision but increase execution time.


The command also enables you to do the following:
  • set the units of measurement of length for output and check the ones in use in the document
  • save the calculated data to a file.
