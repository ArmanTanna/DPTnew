ToolsInfoAnalysisCurve Curvature Diagram


 

The Curve Curvature Diagram command under Analysis enables you to create the graph of the curvature of a plane curve (or of the projection of a skew curve onto a plane) as a function of the u parameter.

The command also enables you to obtain the values of the parameter, curvature, and curvature radius at an indicated point on the graph (Point on diagram).

Select the Create diagram check box to retain the generated diagram even after exiting from the command.

Note that, you can obtain the curvature plot of a curve using the Curvature command.
