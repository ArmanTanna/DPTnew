You can assign a variable to an entity using the Assign command, which can be accessed from the Spreadsheet dialog box.

The Spreadsheet dialog box can be viewed using the Spreadsheet command.

You can assign the variable to the following:

Dimension/Feature/Component Enables you to assign the variable to a drive dimension or to a feature/datum entity. In the case of threaded holes/shafts, choosing this option will cause the variable to be assigned to the hole/shaft diameter rather than to the thread diameter or pitch
Thread Diameter Enables you to assign the variable to a thread diameter, when the model involves threaded hole/shaft, rather than to the hole/shaft diameter or to the thread pitch.
Thread Pitch Enables you to assign the variable to a thread pitch, when the model involves threaded hole/shaft, rather than to the hole/shaft diameter or to the thread diameter.
Try to ...
  • Assign/deassign a variable to/from a drive dimension
  • Assign/deassign a variable to/from a feature or a datum entity
  • Assign/deassign a variable to/from an X-reference component
  • Assign variables to features/datum entities and check the items that have variables assigned to them
