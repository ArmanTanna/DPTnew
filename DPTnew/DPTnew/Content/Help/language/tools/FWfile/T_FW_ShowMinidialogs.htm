The Show Minidialogs command enables you to restore the visualization of the hidden minidialogs.

You can find this command in the context menu that becomes available when you right-click in the graphic area whenever the minidialogs are hidden, or you can run the command by hitting the F7 key.

The program automatically restores the minidialogs visualization when the current command is completed.
