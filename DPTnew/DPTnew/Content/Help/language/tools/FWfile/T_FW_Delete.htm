You can delete a variable using the Delete Variables command, which can be accessed from the Spreadsheet dialog box.

The Spreadsheet dialog box can be viewed using the Spreadsheet command.

Note
You cannot delete a variable that is assigned to a dimension in the model. You must first deassign also those dependent on it before you can delete the variable.

You can select the variable, right-click on the spreadsheet area and select Delete Variables, to remove the variable from the Spreadsheet dialog box.
