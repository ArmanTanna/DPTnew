The Custom Visual Bookmark command enables you to create a drawing view (default or sectioned) based on the visual bookmark defined on the referenced model. The sectioned view is created while creating a drawing view from the visual bookmark without having an intermediate view.

When you start the command, the Visual Bookmark View dialog box is displayed. The Visual Bookmark drop-down list contains all the visual bookmarks defined in the referenced model. You can select the visual bookmark based on which the view is to be created.

If the chosen visual bookmark does not contain information about the view orientation (i.e.: the View Data check box under Capture Settings was not selected when the visual bookmark was defined), you can select one of the default view orientations in the Default drop-down list under View orientation. If the visual bookmark does contain information about the orientation, the View orientation area is grayed out.

The view properties related to the type of entities that are to be visible or not, to the layers to be displayed and so on are overwritten by the corresponding settings of the selected visual bookmark.
