You can use the Cross-Section command to insert multiple parallel cross-sections in a single step.

Note that before you can use the Cross-Section command to create multiple parallel cross-sections, you must first create a section plane by using one of the following commands:
  • Define Section
  • Define on Model
  • Place on Drawing
You must also change the number of parallel cross-sections you want to create in the Slices box of the Options/Properties.
