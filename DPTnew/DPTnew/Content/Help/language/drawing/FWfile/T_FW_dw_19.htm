The Layout Configuration command inserts an arrangement of two-dimensional views in an empty drawing, based on the current Layout Configuration, as specified in the Drawing category of the System Options.

You will be required to associate a model with the current drawing, so that it becomes its referenced model. The Edit Reference Model dialog box will be displayed, on which you can select the model to be referenced.
