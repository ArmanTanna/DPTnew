The Modify Detail View command enables you to redefine the existing detail view.

To perform the command, right-click on a detail view or on its icon in the Drawing Structure and select Modify Detail View from the context menu.

This Modify Detail View command causes the think3 application to reset its Undo/Redo buffer. Once you start the command:

  • — you would not be able to undo it or any previous operation.
  • — the existing detail view disappears from the drawing and you are enabled to redefine it by providing exactly the same parameters as the ones required for the initial definition.
