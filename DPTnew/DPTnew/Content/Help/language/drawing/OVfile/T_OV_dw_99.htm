Layout Configurations

Layout configurations describe the way in which two-dimensional views are arranged in a drawing and can be used to create the same arrangement of views in other drawings.

The software ships with five predefined layout configurations for arranging views based on the ISO, ANSI, GB, JIS, and KS international standards. You can also create and save your own custom layout configurations.

When you save a custom layout configuration, you save the arrangement of the views in that drawing, not the actual views. You can then use a layout configuration, either a predefined configuration or a custom configuration, to produce the same arrangement of views in another drawing.

Layout configuration files have the file extension .LYA. The predefined layout configurations are saved in the thinkdesign\files\Templates directory of your installation folder. Custom configurations are stored in the user folder which depends on your operating system.

You can use a predefined configuration to arrange drawing views when you do the following:

  • create a new drawing from the current model
  • load a configuration in the current drawing

You can also insert the current layout configuration into the current drawing. You can choose the type of configuration to be used (ANSI, ISO, GB, JIS, KS) in the File: drop-down list under Layout Configuration in the Drawing category of the System Options.

Custom Configurations

After you arrange several two-dimensional drawing views in a drawing, you can save that arrangement as a layout configuration. You can then reuse that configuration to create the same arrangement of views in other drawings.

Create the views

Arrange the views on the drawing sheet.

Save the Configuration

Use the Save Layout Configuration command to save the view arrangement as a configuration.

The program stores custom configurations in the directory specified by your user profile as determined by your operating system. Layout configuration files are assigned the file extension .LYA.

Reuse the Configuration

You can use a custom configuration to arrange drawing views when you do the following:

  • create a new drawing from the current model (see Drawing from Current Model)
  • load a configuration in the current drawing (see Load Layout Configuration)


When you apply the custom configuration to a different model, the program creates a drawing of that model containing the same view arrangement as the configuration.
