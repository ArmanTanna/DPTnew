Editing entities inside drawing views

Drawing views are 2D representations of a 3D model used to create production drawings. Therefore, they have the following characteristics.
  • The following commands:
    Hide Entities, Delete, Smart Delete and Properties, Trim, Insert Fillet, Insert Chamfer can be used, but all the user's changes will be lost as soon as the view or the drawing are rebuilt ( see Rebuild View and Rebuild Drawing ). A specific message is displayed to inform the user about this behavior.
     
  • The following commands:
    • Move/Copy Entities,
    • Mirror Entities,
    • Scale Entities
    can be used with the Copies option. Transformed copies of entities will be automatically added to the Unrelated entity group, thus being not linked to the view. They will be retained when the view is rebuilt.
     
  • None of the Modify Curve command can be applied to curves belonging to drawing views.
     
  • Any other editing actions on lines, arcs and curves are not enabled.
     
  • Dragging & dropping of entities is not enabled. Only whole views can be dragged and dropped.
     
  • Copying & Pasting is enabled, while Cutting & Pasting is not.
