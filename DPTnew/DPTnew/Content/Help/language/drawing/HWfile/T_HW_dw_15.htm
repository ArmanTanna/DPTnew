Hiding Dimensions, Axes or Edges

To hide the dimension, axes or edges in a drawing view, start the Hide Dimensions Axes Edges command and proceed as follows:

1 Click to select the view in which you want to hide dimensions, axes or edges.
Please note if you preselect a view, the program skips this step.
 
2 The program prompts you to select the entity that you want to hide.

The command enables you to hide texts, if the Text option has been selected under Model entities visibility filter, in the Entity Properties - Drawing View category of the Options/Properties dialog box.

 
3 Specify the entity type you want to hide by selecting from the Entity Type drop-down list:
  • Dimensions — to hide dimensions
  • Axes — to hide axes
  • Edges — to hide edges
4 Click to select each entity you want to hide. As you select the entities to be hidden, the status bar updates to indicate the number of hidden entities. The dimensions, axes or edges you select are immediately hidden. Note that you can also use a window selection to select several dimensions, axes or edges in single action.

Note
Only for dimensions, you can introduce the additional pick modality by feature, by selecting the Feature Dimensions check box.
  • When selected, you can pick all the dimensions linked to the selected feature.
  • When not selected, you can pick only individual dimensions.
Use the Unhide Dimensions Axes Edges command to restore the visibility of the hidden entities.
