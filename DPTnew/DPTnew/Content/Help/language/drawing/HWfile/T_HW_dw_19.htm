Inserting a Layout Configuration

1  When you start the Layout Configuration command, the program displays the Edit Reference Model dialog box.
2 Specify the model you want referenced by the current drawing by doing one of the following:
  • From the drop-down menu, choose one of the models that is currently open.
  • Click the Browse button and select the model file you want to reference.
3 You can prevent the referenced model from being modified from the drawing. For example, if the drawing views you will insert into the drawing contain drive dimensions, you might want to prevent them to be changed, so that the model will not be modified. To so, select the Open with read-only access check box.
4 Click OK to confirm.

The program immediately generates two-dimensional drawing views based on the current Layout Configuration (as specified in the Drawing category of the System Options).


Note

If you start the Layout Configuration command in a drawing to which a model has already been referenced, the program displays a dialog box warning you that the current drawing will be cleared. If you click OK, the program immediately clears the contents of the current drawing and then displays the Edit Reference Model dialog box. If you don not want to clear the current layout, click No.
