Automatic naming mechanism

By default the automatic naming mechanism, of the think3 application, for sections names assignation starts from A to Z, then AA to ZZ, AAA to ZZZ and so on.
This can be set by selecting the Automatic names check box in the Drawing - General category of the Document Properties.
You can also set a different initial value (default is "A"), so that the section names will start for example from E, F or whatever value you specify in the Start from section: box.

Excluding some letters from the automatic naming mechanism
For readability reasons and compliance to the standards, you need to exclude some letters from the progressive auto-naming mechanism. To obtain this do the following:

  • Open the I/O window (View I/O Window).
  • In the Command line: of the I/O window, type:
    ekpar EP_SECT_WRONG_NAMES"list"
    where list is a list of characters to be skipped by the automatic numbering mechanism, like, for instance "IJOQSXZY" (default list of skipped letters).
  • To restore the automatic naming mechanism so that it will skip no letters, type:
    ekpar EP_SECT_WRONG_NAMES""
