Managing Views

The view management dialog box (which is different depending on what you are actually doing, for example: creating a Custom View or an Exploded View) enables you to manage view creation in the 2D environment. The tools within this dialog box enable you to reorient the model to create the desired custom view, using a graphics preview area in which the model or the views are displayed.

The following tools are available:

  • A set of controls enabling you to control the orientation of the view. See Changing the View Orientation.

  • A Section in use: drop-down list under Sectioned view enables you to select a section from which you want to create a sectioned view. This option is only available if the drawing already contains a main view with at least one existing section already created by using the InsertCutting Plane LinesDefine on Drawing command. See Creating a Main Sectioned View to learn how to create a section view.
  • A set of four buttons located above the graphic preview area enabling you to control the visibility of the objects depicted in the resulting custom view, as follows:
     
    View Properties Displays the Drawing View category and all of its sub-categories, which enables you to control the name and scale of the custom view and also determine the types of entities that will be visible in the view.
    Select Visible Entities Displays an additional Selected entities: drop-down list. Choose either Visible or Non-visible from this list, and then select the entities in the preview area to determine whether they will be visible or non-visible in the resulting custom view. Note that when you select Non-visible, the button is grayed out.
    Select Non-Sectionable Solids Enables you to select the solids that you do not want to allow to be sectioned.
    Add to Non-Sectionable ( ), present in the context menu available on right-clicking the solid icon in the Model Structure, enables to add an entity to the set of non-sectionable ones.
    Remove from Non-Sectionable, present in the context menu available on right-clicking the solid icon in the Model Structure, enables to define an entity as sectionable one.
    Note that these two context menu commands work like a toggle switch.
    Reset Restores the visibility settings to those set in the model being referenced.


  • Note
    These four buttons are available only in Custom view Redefinition dialog box.


  • The two tabs, Shared Groups and Layers, of the frame on the left of the Graphics Area in the dialog box enabling you to manage the shared groups and layers respectively.

    The Layers tab structure looks similar to the one available in the left pane of the model/drawing document, except for the following considerations:

    • It contains only Filter Layers and Select Layers buttons.
    • The layer list displays all the layers belonging to all the components of an assembly (not only the ones of the root). If there are two layers with the same name, the layer list shows the only one occurrence of that layer. However, when the layer is turned on/off, all layers with this name will be turned off. Moreover, the Entities column will show the number of visible and hidden entities of the layer in hierarchy.
    • The New Layers: drop-down list allows the user to handle the visibility status of all layers that will be created in the model after the custom view definition.

      Unhide Make the next created new layers ON (visibile) in custom view definition. Henceforth, the entities belonging to new layers will be displayed in the corresponding view at first drawing rebuild or custom view redefinition.
      Hide Make the next created new layers OFF (hidden) in custom view definition. Henceforth, the entities belonging to new layers won’t be displayed in the corresponding view.
