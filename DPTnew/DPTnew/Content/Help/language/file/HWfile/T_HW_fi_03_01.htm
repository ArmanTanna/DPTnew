Inserting a file from the local file system into a model

On selecting the Insert from File command, the file selection dialog box is displayed. To insert a file from the local file system into a model, you will have to:

  1. Select the document to open.
  2. Choose whether to import the entire model or just some entities.
  3. Choose whether to import the model or some entities statically or with their parametric structure.
  4. Keep or change the position of the Work Plane in the document you are importing.
  5. Define how to deal with variables, in case they have the same name in the document you are importing and in the current one.

Proceed as follows:

1 Select the document.
a Under Files of type, select the type of document you want to open in the list of formats supported by think3.
b In the Look in selection box, select the folder containing the document to open (initially, the model directory chosen when installing the think3 application is proposed). You can navigate through your file system like in any Exploring window of the Microsoft® Windows operating system.
A list of all documents of the specified type located in the selected folder will be displayed in the box below.

Please note steps a and b can be performed in reversed order.
 
c Click on the document you want to open. The document name will be displayed in the File name box.If the document type is ".e3" or ".e2" (and when the selected document was saved the proper options were set in the Preview tab of the FileProperties dialog box), a preview image of the file contents will be displayed in the frame on the right.
d Click the Open button to open the document.

Note

If you select a drawing document (".e2") including views of a model, a warning will be displayed, informing you that all the references to the model from which the views were derived will be lost, that is any changes made to the model won't be applied to the views.

The Import dialog box is displayed.
2 Choose whether to import the entire model or just some entities. In Transfer, select All or Selected entities.
3 Choose whether to import the model or the selected entities statically or with their parametric structure. In Load as, select Parametric or Static.
4 Define the position of the Work Plane of the source document. A transformation is applied to the imported items to make their Work Plane coincide with that of the target document. You can change the position of the Work Plane in the document to be imported before importing it, by selecting the Work Plane Position check-box under Modify. Once you've finished editing the Work Plane position in the source document, double-click the mouse to close the temporary window (or right-click and select End of Input in the shortcut menu), and the selected object will be imported into the target document.
5 Define how to deal with the variables of the document you're importing that have the same names of the variables of the current document, if any.
6 Select OK to import the document.
