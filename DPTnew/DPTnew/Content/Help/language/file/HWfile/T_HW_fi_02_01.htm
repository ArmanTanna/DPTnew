Opening files including entities with corrupted data

When you open a file including entities with corrupted data, a dialog box is displayed warning you that there are some errors loading the file.

You can then do the following:

  • Click for More Details
  • Continue the loading action
  • Close file

By clicking More Details, you get the list of the single entities which cause the errors in loading. Switching to any row in the list by the arrow keys, a dialog box is displayed including some information about the selected entity such as identification number, type, visibility, layer.

By selecting Continue, the program continues the loading action without inserting the entities with corrupted data.

By selecting Close file, the program closes the loading action and ends the Open command.
