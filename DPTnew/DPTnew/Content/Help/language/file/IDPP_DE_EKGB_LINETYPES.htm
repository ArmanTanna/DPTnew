DXF/DWG

Category: General-Line Type

Line Type

The controls in this group determine how the line types will be converted.

Add

Enables you to map a new element.

Modify

Enables you to modify the mapping of an element.

Delete

Enables you to delete the mapping of an element.

Show Map

Displays the default map table.

Save

In this area you can specify the scale factor applied to all line types when saving to DXF/DWG formats.

Scale factor

Specify the scale factor to be applied to all line types when saving to DXF/DWG formats.
