STEP

Category: Load-Healing

Healing

The controls in this area determine how to deal with some conversion issues

Enable

When checked, enables the options in the group

Maximum error threshold for closing gaps

Specifies the maximum error allowed before considering a gap closed in a curve loop

Remove small boundary curves

When checked, the boundary curves smaller than the specified length will not be converted

Remove sliver/small surfaces

When checked, surfaces smaller than the specified area will not be converted
