General

The controls on this page determine general aspects of the conversion process for both importing and exporting files.

Logfile

Determines the name of the log file. You can specify the file name by typing the name into the box, or by clicking the Browse button, then specifying the file using a standard file dialog box.

Verbosity

Determines the level of detailed information recorded to the log file, by choosing one of the following: Standard Output - normal level of detail; Debugging - high level of detail.

Disable font remapping

If DraftMaker's fonts are the standard ones, when this box is not selected, think3's vector fonts are automatically remapped to their corresponding DraftMaker's fonts and vice versa (based on the font appearence). On the contrary, if this box is selected, fonts are converted based on their font number (so that font number 1 is converted into font number 1, number 2 into number 2 and so on). This might be useful if DraftMaker's fonts have been customized.
