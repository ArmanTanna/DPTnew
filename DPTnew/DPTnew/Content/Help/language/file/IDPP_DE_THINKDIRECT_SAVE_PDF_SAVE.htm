PDF Options

Category: Save

Save

The controls in this area determine general aspects of the conversion from think3 into the selected format

Convert profiles to curves

When selected, these entities are converted. When not selected they are ignored.

Convert datum entities

Controls the conversion of datum entities. When this check-box is selected, datum entities are converted.

Convert not visible entities

When selected, these entities are converted. When not selected they are ignored.

Convert solid entities

Controls the conversion of solid entities. When this check-box is not selected, solids are converted into their component surfaces.

Save hidden layers

When selected, the hidden layers and the entities belonging to them are saved in the PDF output file. When not selected, they are ignored.

3D format to PDF

The controls in this group allow to define the accuracy of the description of the model.

B-Rep

Enables you to save the B-Rep (boundary representation) of the model. The size of the output file is bigger than the one obtained by the Tessellation option, because it includes both geometry and topology of the model.

Tessellation

Enables you to save the model by the tessellation, that is the 3D static representation of the model. The size of the output file is the smallest of the three options.

B-Rep + Tessellation

Enables you to save the B-Rep (boundary representation) of the model along with its tessellation (3D static representation of the model). The size of the output file is the biggest of the three options.

Show warnings/error messages

When selected, warnings and error messages are displayed in a popup window.
