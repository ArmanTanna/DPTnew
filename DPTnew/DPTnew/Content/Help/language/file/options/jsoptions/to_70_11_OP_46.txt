Associated dimensions converts all the IGES Dimension  entities to dimensions associated to the entities referred to. Associated dim., angle dim. excluded converts all the IGES Dimension  entities to dimensions associated to the entities referred to, except the Angular Dimension , which do not maintain any relations with the entities referred to. Associated dim., angle dim. with set text excluded converts all IGES Dimension  entities to dimensions associated to the entities referred to, except Angular Dimension , whose measurement is the wrong format (a set text rather than the calculated measurement), which do not maintain any relations with the entities referred to."
