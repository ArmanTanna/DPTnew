STEP

Category: Load

Load

The parameters in this area control file opening operations.

Display modes

Enables you to select the viewing representation of the objects that are converted.

Set tolerance

When not selected, the tolerance value used for the conversion is the one that is currently specified for the model. The model tolerance value is defined in the Tolerance edit box under the Precision area of the General - Advanced category of Document Properties. When selected, the tolerance value can be specifically defined in the corresponding edit box.

Create boundary curves

Creates the boundary curves to surfaces.

Assembly conversion modes

The options in this area enable you to control the conversion of assemblies, sub-assemblies and components.

Keep hierarchy

When selected, it enables the think3 application to read and convert also the hierarchy information. Each 'leaf' component will be converted to an X-Reference Component. When not selected, information about hierarchy is ignored (default).

Print report

This parameter is enabled only if the Keep hierarchy box is also selected. When selected, it enables you to create a report file (with the same name of the input file and extension '.rpt') — in the same folder of the file you are loading — containing the hierarchy information: parent-child relations, unit of measure, transformation matrix associated with each child and component name. When not selected, no report file is created (default).

Keep original names

This parameter is enabled only if the Keep hierarchy box is also selected. When selected, it enables you to assign components the same names they had in the system where the model was created. When not selected, components are assigned the name of the corresponding '.e3' file (default).
