File Open


 

The Open command opens a new window in think3's client area containing the document you selected. As the think3 application is a Multiple Document Interface system, in that area you can open as many documents as you like.

The Open command can open several types of documents.

In addition, according to the type of document that you open, you may define some specific Options.

From the file selection box displayed when you perform the command, you may also:

  • Select a Visual Bookmark that will be applied at loading. Choose it in the drop-down list that shows the different bookmarks and the categories to which each bookmark belongs.
  • Select the Quick Load option, enabling you to load any X-Reference component in the Lightest Representation, that is in Flat Tessellation, regardless of what is specified in the selected Visual Bookmark.
