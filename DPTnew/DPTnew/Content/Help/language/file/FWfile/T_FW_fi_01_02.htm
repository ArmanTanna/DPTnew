FileNew Drawing from Template

By selecting Drawing from Template in the New command, the program opens a new document from the chosen template in the drawing environment, from where you can create a new drawing.

All the commands for the drawing environment are available.

The new document will be given a provisional name like the following: Drawing<number>, where <number> is a sequence number (Drawing1, Drawing2, ...). The provisional name will be replaced with save it.

This command meets your requirements when you need to create a stand-alone drawing file without reference to a model.

In order to create views of a model in the drawing you can:

  • Associate a model with the drawing using the Reference Model command, and then insert the views using the InsertDrawing View commands
  • Use the Layout Configuration command. You will be required to select a model to associate with the drawing, and a predefined arrangement of views will be automatically created.

Furthermore, the New command, performed from the model environment, includes Drawing from Current Model, enabling you to create a new drawing containing drawing views of the current model.
