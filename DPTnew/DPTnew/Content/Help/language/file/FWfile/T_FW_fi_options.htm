When you load (see Open) and save (see Save and Save As) a file in a different format, that needs to be converted to the think3 format, the Options button enables you to define the parameters, specific for the actual data exchange you require.

You can define Options in the following cases:

  • Model File — extension ".e3"
    • Load
    • Save
  • Drawing File — extension ".e2"
    • Load
    • Save
  • DXF File — extension ".dxf"
    • General
    • General - Colors
    • General - Layers
    • General - Line Type
    • General - Font
    • Load
    • Load - Advanced
    • Save
    • Save - Advanced

  • DWG File — extension ".dwg"
    • General
    • General - Colors
    • General - Line Type
    • General - Font
    • Load
    • Load - Advanced
    • Save
    • Save - Advanced

  • IGES — extension ".igs", ".iges"
    • The following options are available when 2D is selected in the Conversion type: drop-down list of the file selection box:
      • 2D
      • 2D - Layers
      • 2D - Text
      • 2D - Dimension
      • 2D - Geometry

    • The following options are available when 3D is selected in the Conversion type: drop-down list of the file selection box:
      • General
      • General - Layers
      • Load
      • Load - Log File
      • Load - Advanced
      • Load - Healing
      • Save
      • Save - Log File
      • Save - Header Details
      • Save - Advanced
      • Save - Healing

  • STEP — extensions ".stp", ".step"
    • General
    • General - Layers
    • Load
    • Load - Log File
    • Load - Advanced
    • Load - Healing
    • Save
    • Save - Log File
    • Save - Header Details
    • Save - Advanced
    • Save - Healing

  • VDA — extension ".vda"
    • General
    • Load
    • Load - Log File
    • Load - Advanced
    • Load - Healing
    • Save
    • Save - Log File
    • Save - Header Details
    • Save - Header Details - VDA-FS
    • Save - Advanced
    • Save - Healing

  • GBG DraftMaker drawings (for plotting using GBPLO - drawings only) — extension ".d"
    • Settings
    • Load
    • Save

  • STL — extension ".stl"
    • Settings

  • ASCII Points — extension ".pt"
    • Load

  • Parasolid — extension "*.x_*"
    • Load
    • Load - Log File
    • Load - Advanced
    • Save
    • Save - Log File
    • Save - Advanced

  • CATIA — extension "*.CATP*"
    • General
    • General - Layers
    • Load
    • Load - Log File
    • Load - Advanced
    • Save
    • Save - Log File
    • Save - Advanced

  • Pro/ENGINEER — extension "*.prt*;*.asm*;*.neu*"
    • Load
    • Load - Log File
    • Load - Advanced
    • Save
    • Save - Log File
    • Save - Advanced

  • VRML — extension ".vrl"
    • Settings

  • PDF — extension ".pdf"
    • Save
    • Log file
    • Advanced
    • Visual Bookmark
