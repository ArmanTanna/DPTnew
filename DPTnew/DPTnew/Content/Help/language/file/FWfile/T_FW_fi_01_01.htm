FileNewModel from Template

By selecting Model from Template in the New command, the program opens a new document from the chosen template in the model environment, from where you can create a new model.

All the commands for the model environment are available.

The new document will be given a provisional name like the following: Model<number>, where <number> is a sequence number (Model1, Model2, ...). The provisional name will be replaced with the one you give it the first time you save it.
