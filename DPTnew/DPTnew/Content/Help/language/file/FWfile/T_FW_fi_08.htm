FilePartial Save

 

The Partial Save command enables you to save a part of your document, that is entities you select in the current model/drawing, to a new document.

Once you have selected the entities, the command behavior doesn't differ too much from that of the Save and Save As commands.

You can save the new document in several different formats, therefore, according to the type of document that you save, you may define some specific Options.
