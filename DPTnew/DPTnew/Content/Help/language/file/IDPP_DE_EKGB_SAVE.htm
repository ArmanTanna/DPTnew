DXF/DWG

Category: Save

Save

The controls in this area determine the format used to save DXF, DWG and DraftMaker files.

2D data only

When selected, enables conversion to DXF 2D format.

3D faces only

When selected, enables conversion to DXF 3D format.

Disassemble on saving

The controls in this area determine whether certain think3 entities are converted into their corresponding DXF, DWG, or DM entities when saved, or broken down into their component entities.

Dimensions

Using this selection list it is possible to select which dimensions will be saved as disassembled geometry; None : No dimension will be saved as disassembled geometry, When needed : Only those dimension will be saved as disassembled geometry that cannot be converted preserving 100% of their parameters, All : All the dimensions will be saved as disassembled geometry.

X-Hatches

When selected, cross-hatches are broken down into their component entities.

Entities inside groups

Using the options is the selection list, select the way the entity colors are preserved while saving the document; Current : Entities will be saved as per the selection made for the Show closed groups with group color option in the Group attributes area of the Drawing - Groups category in the Document Properties dialog box for drawings ('e2' file), As group : Entities will be saved in the same color as that of the group they belong to, As entity : Entities will be saved in their own color.

Color

Using the options is the selection list, select the way the entity colors are preserved while saving the document; Current : Entities will be saved as per the selection made for the Show closed groups with group color option in the Group attributes area of the Drawing - Groups category in the Document Properties dialog box for drawings ('e2' file), As group : Entities will be saved in the same color as that of the group they belong to, As entity : Entities will be saved in their own color.
