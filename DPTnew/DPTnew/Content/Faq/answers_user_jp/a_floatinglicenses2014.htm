FlexNet (formerly FLEXlm) is a network license manager used by many software developers to control the use of their software products with a client-server approach. You just install a license server on a PC with a dongle, and all clients get the licenses from that server, simply defining a few system variables.


Server Installation

  1. Check if your server has one of the following, supported operating systems:
    • Microsoft® Windows® 2008 R2 Server 64 bit
    • Microsoft® Windows® 2003 R2 Server 64 bit
    • Microsoft® Windows® 7 32 and 64 bit

  2. Install ThinkDesign on a computer (not necessarily the server), in order to download the software required to perform the server installation.

  3. Copy the installation directory C:\Program Files\think3\version number\common\drivers\hardlock on the server, double-click on the hldrv32.exe file and confirm all steps to install the dongle driver; then plug the dongle in the USB port.

  4. Send an email to pwdrelease@think3.eu, specifying the dongle ID and asking for password computing.

    • For products with version up to 2011.1, you will receive password strings like:
      FEATURE <product code> thinkflx <product version> <date> <quantity> <license code> VENDOR_STRING=<customer name> NOTICE=<notice code> INCREMENT <product code> thinkflx <product version> <date> <quantity> <license code> VENDOR_STRING=<customer name> NOTICE=<notice code>
    • For products with version starting from 2012.1, you will receive password strings like:
      FEATURE <product code> thinkdmn <product version> <date> <quantity> VENDOR_STRING=<customer name> NOTICE=<notice code> SIGN=<license code> INCREMENT <product code> thinkdmn <product version> <date> <quantity> VENDOR_STRING=<customer name> NOTICE=<notice code> SIGN=<license code>

    You have a string beginning with FEATURE when you add new product licenses; while you have a string beginning with INCREMENT when you add new licenses of a product already inserted in the license file (i.e. you already have a string beginning with FEATURE for the product).



  5. Copy the installation directory C:\Program Files\think3\version number\common\drivers\flexlm on the server directory C:\flexlm, and configure the license.lic file.
    • Uncomment the first row #SERVER <hostname> CLCODE=<hostid>7171, and replace <hostname> with the name of the server and <hostid> with the dongle ID, so that you should obtain a row of this type:
      SERVER ServerName CLCODE=HUS00123 7171
    • Wishing to run products starting from version 2012.1, uncomment the second row, getting:
      DAEMON thinkdmn C:\flexlm\thinkdmn.exe
    • Wishing to run products up to version 2011.1, uncomment the third row, in order to have:
      DAEMON thinkflx C:\flexlm\thinkflx.exe
    • Add the password strings that you received from pwdrelease@think3.eu


  6. Double-click on the lmtools.exe file included in the C:\flexlm directory, and choose the Config Services tab.

    • In the Service Name box, insert the name of the Service you want to define.
    • In the Path to the lmgrd.exe file box, click on the Browse button and select the C:\flexlm\lmgrd.exe file.
    • In the Path to the license file box, click on the Browse button and select the C:\flexlm\license.lic file.
    • In the Path to the debug log file box, define the file in which you want FlexNet to write the log (for example the C:\flexlm\thinkdesign.log file).
    • Check the Use Services checkbox if you want FlexNet to run as a service.
    • Check the Start Server checkbox at Power Up if you want FlexNet to automatically start at the server boot.
    • Click on the Save Service button to save the service just configured (the Remove Service button can be used to delete it).


  7. In the Start/Stop/Reread tab, you can:

    • Start the license server clicking on the Start Server button.
    • Stop the license server clicking on the Stop Server button.
    • Force the re-read of the license file, when you edit and update it with the license server running, clicking on the EeRead License File button.


  8. In the Server Status tab, you can see the license server status clicking on the Perform Status Enquiry button.



Client Installation


1. Check if your client has one of the following, supported operating systems:

  • Microsoft® Windows® 7 32 and 64 bit
  • Microsoft® Windows® XP 32 and 64 bit

2. Install on the computer the products you want to run.


3. Define the following system variables in Control Panel → System → Advanced System Settings.
The System Properties window opens, select now the Advanced tab and click the Environment variables button.
In the System variables section, click the New button.


  • LICETYPE
    LICETYPE=FLEX
    The system uses local licenses first, then floating ones
    LICETYPE=NOFLEX
    The system does not use floating licenses, but only local ones
    LICETYPE=FIRSTFLEX
    The system uses floating licenses first, then local ones
    LICETYPE=ONLYFLEX
    The system uses only floating licenses

  • LM_LICENSE_FILE
    It has to be defined with the value portnumber@<hostname>.
    If in the license file of the server, the first row is defined as follows:
    SERVER ServerName CLCODE=HUS00123 7171
    in the client, you have to define the system variable as follows:
    LM_LICENSE_FILE=7171@ServerName

4. Only if you want to use our products by Windows® Terminal Server Sessions, you have to define another variable T3TSS=YES to have the products running, and each instance of the products will get a license from the license server.
