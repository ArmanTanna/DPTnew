It may sometimes happen that your online documentation looks bad, with different character sets and widths inside the same document.
In these cases, it is clear that the CSS Style Documents are ignored and images are not displayed. To solve this problem and well display the online help, proceed as follows:


  • Internet Options → Advanced → Show Pictures (must be CHECKED)
  • Internet Options → General → Accessibility → Format documents using my style sheets (must be UNCHECKED)
