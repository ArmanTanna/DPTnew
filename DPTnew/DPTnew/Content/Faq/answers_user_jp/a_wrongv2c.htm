If you get this error message ...



... you may have not inserted the right V2C file. It is mandatory that your V2C files are applied in the order that they were issued. Therefore before applying your last V2C file, make sure you have applied - and in the righe order! - all previous V2C files issued for the same license.

The error message HASP_UPDATE_TOO_NEW means that you have not applied yet a previous V2C file issued for the same license.

If you have not collected your V2C files, please contact pwdrelease@think3.eu to obtain them.
