In case of error messages appearing at ThinkDesign startup, it is possible to use a manual Registration Tool to try to fix the incomplete registration.



  1. Open the Installation folder of ThinkDesign think3\XXXX.X\common\utilities
  2. Run the switch.exe application
  3. In the list of the products installed, select the product version you want to register
  4. If the Register only button is not enabled, select Edit → Force Registration from the menu bar
  5. Click the Run button to save and execute ThinkDesign.

In the case of error messages appearing during the registration, it is possible to try to repair the installation as described in the What should I do before reinstalling? box here below.
