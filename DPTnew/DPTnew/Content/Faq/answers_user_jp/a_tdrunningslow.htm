If ThinkDesign is slow in starting up or running, this may depend on some Performance Counters that are installed and enabled with the Microsoft updates or software. For more information refer to this link.


To disable the Performance Counters, follow these steps:

  1. Download the Extensible Performance Counter List from:
    ftp://ftp.think3.it/outgoing/Performance_counters/exctrlst_setup.zip
  2. Run exctrlst_setup.exe and install the software.

  3. Run exctrlst.exe located in the Resource Kit folder, inside Program File (or Program Files (x86) on 64-bit systems).
  4. For each item appearing in the Extensible Performance Counter List, deselect the item Performance Counters Enabled as shown in the image below.
