If you install AMEC under Windows 7, folders and file are downloaded in a different way with respect to the old operating systems.

All file and folders which were usually located under ap40 are now placed directly under the installation root folder, at the same level as dm91.

From the tests run under Windows 7, we gathered:


  • Install dm91 in a path with no blanks (for instance: C:\think3\dm91) and not under C:\Program Files (x86)\think3
  • After installing GBGDM and AMEC, re-organize files so as to match the file structure expected by AMEC procedures, that is:
    1. Create an ap40 folder at the same level as dm91 (C:\think3\ap40).
    2. Copy all folders and files located under C:\think3 – except dm91 – to folder ap40.
    3. Copy folder .\ap40\infotmm_i to C:\think3 and rename it infotmm
    4. Copy files gap_pre.cnf and tmm_pre.prc from .\ap40 to .\dm91.

After these interventions, if you start GBGDM → Tools → Applications, it should be possible to activate AMEC, having previously entered AMEC password through dm91 License Administrator.
