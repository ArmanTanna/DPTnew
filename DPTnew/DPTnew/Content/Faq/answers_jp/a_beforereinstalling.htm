In case of a malfunction that suggests uninstalling and reinstalling the product, you can first try to repair the installation, using Repair/Change installation functions.


A similar procedure is available to update the installation when it is necessary to add optional components that were not selected during the first installation.



  1. Open the Control Panel and select Programmes and Features to access the list of the installed applications.
  2. Scroll down the list until you find the ThinkDesign installation that you want to repair or update and select it.
  3. R-click the mouse button to open the drop-down menu and select Change.
  4. In the Wizard that opens, select one option among the following:
    1. Edit to update the installation by choosing the add-ons.
    2. Repair to re-run the latest installation.
    3. Uninstall to completely remove the installation.

To run the Repair or Change functions, the file folder resulting from the first installation must be available; otherwise you will be prompted to indicate the path of the folder where to find the files.
