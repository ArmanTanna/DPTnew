Here are described the export modalities.


First of all, please note that an export activity can be:

  • REGULAR
    This is the case when TD2014 is regularly exported and a valid .tbu file can be produced, sent to us and validated by us.
    This is also the case when TD2018 is regularly exported using DPT3Care website ‘export/validation’ features.
  • NOT REGULAR
    This is the case when with TD2014 a .tbu file cannot be produced (PC has been lost, PC has been formatted, PC has been stolen) or .tbu file is not a valid one (our check finds out whether it is empty or not valid). This is also the case when with a TD2018 license cannot be exported for analogous reasons.
    Please note that in some situations our central Customer Care team is able to ‘fix’ some NOT REGULAR cases. Such ‘recovering’ activity requires a remote connection with the customer’s PC (when possible). If this activity is a successful one, export will be promoted to REGULAR.
