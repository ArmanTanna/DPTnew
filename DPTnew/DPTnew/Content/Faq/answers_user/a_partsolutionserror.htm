If, when trying to insert a PARTsolution component, you get the following error message:



It means that Cadenas did not create the file (in a particular format) that the plug-in loads and converts to .e2/.e3.
The Windows user on the PC might not have the right permission to save the file in the mentioned folder (see CADENAS_DATA Windows environment variable).
The solution is to set the UAC to zero or to change the user permissions on this folder.
