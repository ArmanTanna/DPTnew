Do not insert the dongle into the USB slot yet; if you have already inserted it, remove it and restart your PC.

First of all, you should install the dongle driver on the new PC. You will find it into TD installation directory:

C:\Program Files (x86)\think3\software version\common\drivers\hardlock or C:\ Program Files\think3\software version\common\drivers\hardlock.

With Windows7/Windows8 operating systems, run the hldrv32.exe file.
With Windows 8.1, run the HASPUserSetup.exe file. In both cases confirm the default values.

Now you can insert the license code on the new PC using the License Administrator, using the same password as before (the password has not changed, it depends just on the dongle).
