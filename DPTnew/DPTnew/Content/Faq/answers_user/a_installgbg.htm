In order to install the software you need Administrator rights.

Do not insert the dongle into the USB slot yet; if you have already inserted it, remove it and restart your PC.

  1. Download dongle driver from this link: ftp://ftpitapub.think3.eu/outgoing/hardlock.zip
  2. Install USB dongle driver:
    • run hldrv32.exe (follow the instructions) if you have installed Win7 x64 or
    • run HASPUserSetup.exe if you have installed Win8.1 x64
  3. Insert the dongle into the USB slot.
  4. Download GBGDM 9.1 from this link: ftp://ftpitapub.think3.eu/outgoing/GBG/dm91_A7.zip
  5. Open the .zip file you have just downloaded and proceed with installation by double clicking Setup.exe.
    1. Follow the instructions and install all the suggested products.
    2. When choosing the License Type, select Local licenses.
    3. Answer Yes for opening the License Administrator.
    4. Verify that your machine ID corresponds to the USB dongle ID code.
    5. Select Cancel and proceed with installation.
    6. Insert the password in the License Administrator.
