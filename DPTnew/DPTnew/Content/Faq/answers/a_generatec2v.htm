OPTION A

If you installed version 2015, 2016 or 2017, please proceed as follows:

You should run the executable file DPTGetC2V.exe present in the installation folders (example: C:\program files\think3\thindesign 2015.2\common\utilities).
A shell window will appear, showing the folder where the .c2v file has been saved (usually the Temp folder).


If you do not find the generated .c2v file, just follow these steps: click 'Start', select the 'Run' command. A dialog box with a search bar will appear, type '%temp%' and then click 'ok'. The 'Temp' folder will open. Sort files by the last modified. The first file on the list will be the .c2v file you are looking for.



OPTION B

If you downloaded ThinkDesign 2018 or successive versions, during the installation, a .c2v file is automatically created on your Desktop.
The file is named after your machine plus a code number, for example: PC-John_110137.c2v.

In case you don't find the file or need to re-generate it, please follow the procedure described in "OPTION A". However, instead of the "Temp" folder, the .c2v file will be located on your Desktop.

Now please send an email at pwdrelease@dptcorporate.com attaching your .c2v file or activate the license on your own through DPT3Care website.
