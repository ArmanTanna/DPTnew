OPTION A

If you have donwloaded ThinkDesign 2018, during the installation, a .c2v file is automatically created on your Desktop. The file will be named with your machine name and a code number, for example: XXXXXX_110137.c2v.



OPTION B

If you have installed a prior version, proceed as follows:

You should run the executable file DPTGetC2V.exe, from your installation folder (example: C:\program files\think3\thindesign 2015.2\common\utilities). A shell window will appear, showing the folder where the C2V file has been saved (usually the Temp folder).


If you do not find the generated C2V file, just follow these steps: click 'Start', select the 'Run' command. A dialog box with a search bar will appear, type '%temp%' and then click 'ok'. The 'Temp' folder will open. Sort files by the last modified. The first file on the list will be the C2V file you are looking for.


Now you can send an email to pwdrelease@think3.eu, attaching your C2V file, or you can activate the license on your own through DPT3Care website.
