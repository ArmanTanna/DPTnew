1. Select the row of the license you want to export (do NOT click the green cross button, but the row itself) and click the Export button in the Command bar.





2. Your will receive a V2C file from DPT Licensing at your company email address (the one you selected for receiving license passwords and all other types of sales/marketing messages).


3. Sentinel Admin Control Center: apply the received V2C file to the Update/Attach section.




4. Sentinel Admin Control Center: in the Sentinel keys section, click the C2V button in the Actions column and download the C2V file. This file gives us a picture of the situation on your machine. You’ll have to use it for the export validation.





5. Refresh the DPT3Care page. Select the license’s row (do NOT click the green cross button, but the row itself) and click the Validate Export button in the Command Bar. A new tab will open (pop-ups must be enabled!), please upload the .c2v file you have just downloaded (it contains the evidence for DPT that you really exported the license).





6. Refresh the DPT3Care page. Select the row of the license you want to import (do NOT click the green cross button, but the row itself) and click the Install button in the Command bar. A new tab will open (pop-ups must be enabled!), you now need to upload the new C2V file generated for the new machine (see FAQ: How to generate a C2V file?).
Then click Submit.





7. Sentinel Admin Control Center: apply the V2C file you have just received at your company email address to the Update/Attach section.
