The floating licenses are linked to a specific license server. In order to manage them you should go through a Server/Client (simple!) installation.


Server


1. Download the 'Safenet Sentinel Installer' from the Download Area of think3 Customer Care website and run the executable file.



2. Copy the two following files from a PC where you have previously installed TD 2015.2:
- vcredist_x86.exe, from 'Temp' folder in dvd_td_64bit\thinkdesign\ISSetupPrerequisites\{D240878C-0495-42D7-844F-E0617616DF90},
- DPTGetC2V.exe, from the Installation folder of TD 2015.2
(example: C:\program files\think3\thinkdesign 2015.2\common\utilities).


3. Run vcredist_x86.exe. (Please, note that this operation is necessary only for floating licenses)


4. Run DPTGetC2V.exe. Now follow the instructions provided in the FAQ How to generate a C2V file?, How can I get my V2C file? and finally How can I use my V2C file?.



Client


First of all, you need to set the system environment variable DPT_LIC_HOSTNAME.

Control Panel → System → Advanced System Settings
The System Properties window opens, select now the Advanced tab and click the Environment variables button.
In the System variables section, click the New button.



DPT_LIC_HOSTNAME corresponds to the Variable name and it should be set according to the following table.


Variable value Description
LOCALHOST TD uses only local licenses, not floating ones
Server_Name TD uses only floating licenses, not local ones
not defined TD uses both local and floating licenses

In the Sentinel ACC:







If you use a VPN connection, you just need to do as follows:


In the ACC: go to Configuration → Access to Remote License Managers and write your server name in the Remote License Search Parameters field.




If you use instead a local network, you just need to set DPT_LIC_HOSTNAME = server_name.
