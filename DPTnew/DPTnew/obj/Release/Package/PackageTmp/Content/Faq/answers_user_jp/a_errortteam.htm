When ThinkTeam starts (either standalone or in ThinkDesign) the following error message might show up:



The error message is about the connection of ThinkTeam to the database. It is a generic error message.


Possible causes:

  1. if you use Access database (the default database) the installation might turn out as incorrect. So, please repeat it as Local Administrator after setting the UAC to 0 and Windows updates completely installed.
  2. if you use a different database you may have to install the appropriate database client and test the connection to the database
  3. in system.xlm (check the ThinkTeam registry variable CONFIGFILE in HKEY_LOCAL_MACHINE\Software\Wow6432Node\think3\thinkteam\2015.2\Environment) the name or the database type is not aligned with the installed one.
