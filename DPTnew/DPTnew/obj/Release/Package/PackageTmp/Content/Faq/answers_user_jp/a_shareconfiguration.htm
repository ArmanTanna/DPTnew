As an example, let's centralize the configuration of Title Blocks.

  • Start ThinkDesign with no document open on a PC where the Title Blocks you want to centralize have been created.
  • Tools → Configuration Manager → Save. In the dialog box click Options, then check the Drawing Frames box under Frames and Title Blocks → Title Blocks
  • Copy the configuration (for instance conf1.t3c and folder conf1) onto one PC of your network, not necessarily where ThinkDesign is installed.
  • On each PC, in order to load the centralized Title Blocks at the start, open ThinkDesign with no document open.
  • Tools → Configuration Manager → Local/Remote → Remote → select file conf1.t3c from the network.

Afterwards, if you need to modify or add the centralized Title Blocks: modify the Title Blocks on one of the PCs, save the configuration and replace the one on the server PC with it.
