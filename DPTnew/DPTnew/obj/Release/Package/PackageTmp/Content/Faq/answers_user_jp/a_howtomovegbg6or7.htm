If you are simply moving the USB dongle to another PC:

  1. Install USB dongle driver.
  2. Copy the cadlab21.dat file from C:\ onto the new PC under C:\.

If you want to install a new USB dongle:

  1. Install USB dongle driver.
  2. Start GBGDM and you will be asked to enter your Customer Name.


  3. Enter the password.


Possible error messages:

  1. GBGDM doesn't recognise the USB dongle.


  2. GBG reads a wrong cadlab21.dat file or you don't have the rights to access C:\.
