A frequent issue: a customer installs version 2015.2, tries to start it but it does not (neither in demo mode, which is no longer included with the new licensing system). At such an extent, the customer tries to start version 2014.1 back again, but it fails (since the new registered version is 2015.2).


Solution:

  • Apply for a 2015.2 license by sending the C2V file. Follow the instructions provided in the FAQ How to generate a C2V file?.
  • Use the Switch tool to start the old version. Instructions on how to use the Switch tool can be found in the online Help: Installing think3 products → Coexistence of different versions.
