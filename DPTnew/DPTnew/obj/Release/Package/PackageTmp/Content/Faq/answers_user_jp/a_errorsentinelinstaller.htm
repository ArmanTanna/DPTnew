If, when launching DPT_Safenet_Sentinel_Installer.exe (which can be found in the Download area of the Customer Care portal), the following message shows up...



...it means that on that very PC the Sentinel service has already been installed previously through the Installer.
Actually, no comparison is made between the versions of the new Installer and of the already installed one, but the error message is issued even if the new one is more recent.


Solution:

Uninstall DPT Safenet Sentinel from the Windows Control Panel and install it back again.
If, on the PC, the Sentinel Service has already been installed previously during ThinkDesign installation or using haspdinst.exe from ThinkDesign installation folder, the Installer will work correctly by overwriting the existing driver.
