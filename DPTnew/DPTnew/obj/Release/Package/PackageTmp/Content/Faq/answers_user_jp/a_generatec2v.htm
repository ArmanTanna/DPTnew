You should run the executable file DPTGetC2V.exe, from your installation folder (example: C:\program files\think3\thindesign 2015.2\common\utilities). A shell window will appear, showing the folder where the C2V file has been saved (usually the Temp folder). The file will be named with your machine name (example: DellTower5810.c2v).


If you do not find the generated C2V file, just follow these steps: click 'Start', select the 'Run' command. A dialog box with a search bar will appear, type '%temp%' and then click 'ok'. The 'Temp' folder will open. Sort files by the last modified. The first file on the list will be the C2V file you are looking for.


The C2V file you have obtained contains information about the actual status of the licenses on your machine. If this is your first license, the C2V file contains only your machine ID.


Now you can send an email to pwdrelease@think3.eu, attaching the C2V file and specifying your company name, your name and surname and your email address.
