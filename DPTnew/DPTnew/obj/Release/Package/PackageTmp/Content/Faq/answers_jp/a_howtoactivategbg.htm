How to activate a GBGDM 6.0 or 7.0 license:


In order to create the license file cadlab21.dat for a new USB dongle, you need to launch GBGDM 6.0/7.0 on a PC with the Windows XP OS. On Windows 7 an error message is issued instead.


  1. On the PC with the Windows XP OS, rename or move the old cadlab21.dat from C:\.
  2. Start GBGDM 6.0: in the I/O window, the USB dongle code is displayed (if the USB dongle cannot be regularly read, a “NO-SYSID” code is displayed).
    You will be asked to insert the Customer Name, which will be sent you along with the password.
  3. Enter the Customer Name.
  4. Enter the password.

This way, the new cadlab21.dat file will be created under C:\. Finally, the said file must be copied onto the new PC under C:\.



How to activate a GBGDM 9.1 license:


The Password Release Service will send you a t3license.lcm file.
Copy it in C:\windows and start the program.
