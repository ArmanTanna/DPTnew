1. Start ThinkDesign with no document open.
  2. Tools → Configuration Manager → Save → Options. Then select what you need to save.
    Sometimes in the Saved Preferences some internal parameters are saved together with the others (for example the setting for the correct management of the graphics video card in use on the PC). Such parameters may not be correct for the target PC (it might use a different video card, for instance) thus causing issues or system instability. Our suggestion therefore is to avoid using Application settings → Preferences and Settings → Saved Preferences.
  3. Copy the configuration (for instance conf1.t3c and folder conf1) onto the new PC inside a temporary folder.
  4. Start ThinkDesign with no document open.
  5. Tools → Configuration Manager → Load.
  6. Select the t3c file previously saved and, finally, restart ThinkDesign.
