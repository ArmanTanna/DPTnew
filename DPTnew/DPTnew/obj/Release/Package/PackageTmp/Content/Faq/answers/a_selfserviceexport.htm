1. Select the row of the license you want to export. If the license’s version is 2015 and it is under a maintenance contract the “Export” button will be enabled; click it.





2. Your will receive a V2C file from DPT Licensing at your company email address, that is the one you have selected for receiving licenses passwords and all other types of messages (sales/marketing).

3. In the Sentinel ACC: apply the received V2C file in the “Update/Attach” section.




4. In the Sentinel ACC: in the “Sentinel keys” section download the C2V file.





5. Refresh the DPT3Care page. Click on the license’s row. The “Validate Export” button is enabled. Select it and upload the C2V file you have just downloaded (it contains the evidence for DPT that you really exported the license).





6. Refresh the DPT3Care page. Click on the license’s row. The “Install 2015” button is enabled. Select it and you will be able to generate a new license. A new webpage opens, at the bottom of it you need to upload a new C2V file.





7. Create the C2V file by running the executable file DPTGetC2V.exe, which is in your installation folder (C:\program files\think3\thindesign 2015.2\common\utilities). Then upload it in the DPT3Care page.


In the Sentinel ACC: apply the V2C file you have just received at your company email address in the “Update/Attach” section.
