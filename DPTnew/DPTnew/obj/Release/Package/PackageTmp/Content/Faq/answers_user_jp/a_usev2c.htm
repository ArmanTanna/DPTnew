You are now ready to use your xxxx.V2C file.


In order to do it, you should open the ACC (Sentinel Admin Control Center), either by running the Admin_Control_Center URL from the TD installation folder ...





... or by launching TD and being redirected to the ACC.

Now you can activate the license using your V2C file, applying it in the Update/Attach section of ACC.




When you are done, you will see the license in the Products section. In the example below, it’s a TDProfessional_2015.2 valid for 30 days.





Select the Products section in the left-side menu.





After selecting the Features button, you'll be able also to see the eventual expiration date of your license.
