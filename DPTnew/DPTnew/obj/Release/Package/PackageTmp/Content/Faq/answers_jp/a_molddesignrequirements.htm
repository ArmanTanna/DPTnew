On 64-bit operating systems, the 32-bit version of Microsoft Office 2010 is not compatible with the 64-bit Access database engine 2010, requested by Mold Design. If both products need to be installed on the same machine, the installation of the 64-bit version of MS Office is mandatory.


If any incompatibility is detected during the setup procedure of Mold Design, a warning message appears notifying the problem and the procedure stops until the installation of the Mold Design component is disabled.
