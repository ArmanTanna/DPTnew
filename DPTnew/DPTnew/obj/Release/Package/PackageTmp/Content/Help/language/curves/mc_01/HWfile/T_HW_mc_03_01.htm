Deleting control points from a curve

The Delete Points option of the Curve through Control Points command enables you to remove one or more control points from the selected curves.
Removing control points from a spline curve will modify the curve shape.
 
1 Select the control points.



  • To select all the control points that lie between two of them, select the first and hold down the SHIFT key while selecting the last point.
  • To deselect a selected point, click on it while holding down the CTRL key.
2 Right-click and select Delete points from the shortcut menu. A preview of the modified curve is displayed.
3 Double-click to confirm. The selected control points will be deleted.
