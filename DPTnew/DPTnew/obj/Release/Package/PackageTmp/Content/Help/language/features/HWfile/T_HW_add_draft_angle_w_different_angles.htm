Adding Draft Angle to a Solid with Different Angles

To accomplish this task load file "DraftAngle.e3".

  • Adding draft angle with different angles using Neutral Plane
  • Adding draft angle with different angles using Neutral Edges

Adding draft angle with different angles using Neutral plane

1 After starting the Draft Angle command, select the Reference Plane as shown.


 
2 Select the Faces as shown:

First Face Second Face
To change the selected face, right-click on Faces, select Reset in the context menu and select another face.
3 Check Different Angles option. Enter the angles as shown:




4 Click or to confirm your selections and create the draft.



Click to discard your changes.

Adding draft angle using Neutral edges

1 After starting the Draft Angle command, select the Edges as shown.
First Edge Second Edge
2 For Reference Direction, select Line.

Note
    Make sure that the direction for the angles are same as shown in the figure.



3 Check Different Angles option. Enter the angles as shown:



To change the selected edge, right-click on Edges, select Reset in the context menu and select another edge.
4 Click or to confirm your selections and create the draft angle.
