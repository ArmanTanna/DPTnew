Inserting a Fillet welding section

In this task you will use the Welding Section command to add the section of a Fillet weld to a drawing. To follow this step-by-step example, load the Weld_Fillet.e2 drawing.

Before you start, check that the welding standard set in the Organize Weldings dialog box suits your needs, then familiarize with the scenario: you have two parts to weld together with a Fillet weld:



1 Start the Welding Section command.
 
2 From the Basic Symbol drop-down list, select Fillet.
 
3 From the Dimensioning mode drop-down list in Section Dimensioning, select Side.
 
4 The program prompt is 2 Curves. Select the two sides to be welded together:
 
5 Enter 5 as side dimension of the weld, then click the to create the weld.
