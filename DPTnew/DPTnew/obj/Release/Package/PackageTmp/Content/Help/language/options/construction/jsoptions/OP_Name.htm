Enables you to define a specific and separate name for the component/shared group.
If this box is not filled in, the component/shared group name will be the same as the one of the file it is contained in.

Please note that the component/shared group name thus defined is taken into proper account in any operation the component/shared group is involved in (save and load, insert, etc.).

The component/shared group name can be displayed in the Model Structure/ Drawing Structure Drawing Structure — in brackets on the right of the model/drawing name — by selecting the new shared group name in structure Show component/shared group name in structure box in the Assembly/Shared GroupShared Group category of the System Options.
