ModifyComponentUnlink

The  Unlink command breaks the link between an X-Reference component and the file which represents it. After this action, the X-Reference component is transformed into a local component.

Similarly, applying this command to a single occurrence of a local component in a model containing multiple occurrences of that component breaks the link between the occurrence and the reference.
