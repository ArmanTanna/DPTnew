InsertDesign AidsTangent Plane

The Tangent Plane command enables you to easily display and create the common tangent plane to two curve(s) and/or surface(s).

Several common tangent planes can exist for the selected entities; in this case, the program will create the one that is closest to the point you specified on the second entity.
