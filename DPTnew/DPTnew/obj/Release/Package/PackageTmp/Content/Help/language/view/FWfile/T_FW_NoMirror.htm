The No Mirror command (which you can select under View in the context menu displayed by right-clicking on an empty zone of the graphics area) enables you to deactivate the virtual mirroring mechanism in the current view. All the virtual symmetric representations of the model will disappear from the current view.

To activate the virtual mirroring mechanism you can:

  • Right-click on an empty zone of the graphics area with nothing selected and choose ViewVirtual Mirror in the context menu.
  • Select the Enable check box in the View-Advanced category of the Document Properties.

A set of appropriate options to control the virtual mirroring mechanism is available in the Virtual mirror area of the View-Advanced category of the Document Properties.

For details, see "Virtual Mirroring".
