Spreadsheet - How to define a variable

A variable is completely defined (see "Defining a variable") when you have set:

  • its name
  • its expression or current value

The name of a variable may be up to 256 characters in length. The first character must be a letter of the alphabet; the remaining characters may include numbers, letters, and other characters. Variable names are case sensitive:  Var  is different from  VAR  and from  vAr  etc.

A variable may be independent or dependent. An independent variable is a variable having a value assigned as a fixed value, as a continuous range of values between minimum and maximum limits, or as a set of separate values. A dependent variable is a variable defined based on an expression involving one or more independent variables (up to a total expression length of 256 characters).

You can define a variable using the Spreadsheet or in thefollowing manner:

  • Click on a drive dimension.
  • Right-click to display the context-sensitive pop-up menu.
  • Choose the Edit Dimension command to display the Edit Driving Dimension dialog box.
  • Under Expression, type the name of the variable, and then click OK or press Enter.

The Value box contains the current value of the selected parametric dimension, which can also be changed within this Edit Driving Dimension dialog box.
