Edit Properties

Category: General

Entity type

Entity type:

Displays the type of the selected entity. No type is displayed in case of multiple selection of entities of different types.

ID

ID:

Displays the entity identifier (for internal use only).

Layer

Layer:

Enables you to specify the layer on which entities are drawn. You can use up to 1024 layers (numbered from 0 to 1023).
See "An Introduction to Layers" for further details.

Color

Color:

Enables you to determine the color to be used to draw entities. Besides the option you will see:
By solid : if the color of the entity is the same as that set as the sold color.
 By layer : if the color of the entity is the same as that set as the layer color.

For details, see also Color and Setting and Modifying the Color.

For details about how to control entity attributes, see Setting and Modifying Current Attributes and Entity Attributes

Line type

Line type:

Enables you to determine the line type to be used to draw entities. For details, see also Line Type and " Setting and Modifying the Line Type".

For details about how to control entity attributes, see "Setting and Modifying Current Attributes and Entity Attributes".

Width

Width:

Enables you to determine the line width (sometimes also called "line weight") used to draw entities. For details, see also Line Width and " Setting and Modifying the Line Width".

For details about how to control entity attributes, see "Setting and Modifying Current Attributes and Entity Attributes".
