InsertFrameParts

The Frame Parts command enables you to create frame structures on a selected predefined 2D or 3D profiles. On starting the command the Frame Parts selection list is displayed.

For the Beams selector select the predefined 2D or 3D profile that define the frame structure.

Thereafter, you have to select the line specification using the Line drop-down list.
Then expand the Line node () to view the various cross sections associated with the line specification in the Sections drop-down list. When the cursor is moved on to option in the drop-down list, then a graphical image of the cross section is visualized at the tooltip. Expand the Sections node () to view the dialog box. The dialog box displays all the data related to an individual option and double-click to make a selection.
The Angle mini-dialog box in the Graphics Area enables you to specify an angle for the inserted beam.

There are additional options available under the More Options.

The Insert component check box enables you to create the frame part as a component.

  • When the check box is selected the frame parts are created as components and are also highlighted in the Model Structure.
  • When the check box is cleared the frame parts are created as solids/skins.

The Joint Type: drop-down list enables you to select the type of joint for intersection of frame parts.

The Trim Mode: drop-down list enables you to select the method of trimming on the frame parts.

Once the profile is covered with all the required frame parts, you can Apply the command to generate the Frame Structure. Each frame part inserted along the profile to form the structure is displayed in the Model Structuremodel .

Once the Frame Structure has been created then the individual Frame parts can be modified using the Modify Frame Part command.
