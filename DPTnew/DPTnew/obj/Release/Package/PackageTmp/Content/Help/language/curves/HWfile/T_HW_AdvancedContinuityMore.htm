Additional Advanced Curve Continuity Options

Several options are available under More Options in the selection list of the Advanced Curve Continuity command, in order to enable you to better control the shape of the resulting curve(s). Please refer to the model in Modifying the continuity between two chains of curves for the illustrations below.

Shape factor This parameter affects the resulting curve shape based on two different criteria:
  • Shape similarity — the shape of the resulting curve will have to be as similar as possible to the initial part of the contour
  • Distance — the shape of the resulting curve will have to be as close as possible to the initial part of the contour.
When the value is 0.0 the result will be computed only based on theshape similarity criterion with the initial part of the chain. On the contrary, when the value is set to 1.0, the result will be computed only based on the distance from the initial part of the chain.

Shape factor 0.0 Shape factor 1.0
When the value is set to an intermediate value between 0.0 and 1.0, both the said criteria will be taken into account proportionally to the value itself.
 
Flexibility This parameter enables you to set the flexibility of the resulting curve. Four values are available(Minimum, Small, Medium and Big). The higher the chosen value, the more flexible and complex the resulting curve will be. So, if you use a high value, you will be able to reach a result also in complex cases, but the resulting curve may have oscillations.
