InsertProfileSwitch to Model

The Switch to Model command deactivates the currently active Profile environment and switches back to the Model tab.

Click here to know the other methods to end the profile creation/editing session and switch back to the model.

In case of editing a 2D profile, when you switch back to the model, the original view you had when you started the profile editing session will be restored (unless you have dynamically rotated the model).
