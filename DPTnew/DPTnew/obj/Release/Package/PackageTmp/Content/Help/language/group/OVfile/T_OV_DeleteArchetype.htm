Commands to remove archetypes of local shared groups with no copies in the drawing

Though you may have deleted all the copies of a local shared group, its definition or archetype is still available in the Shared Group tab (so that, for example, you can still right-click on the corresponding icon and place a copy in the drawing using the Insert Group command).

There are two ways to delete the archetype of a certain group with no copies in the drawing or all the archetypes of groups with no copies:

  • To delete the archetype of a group with no copies in the drawing:
    1. Right-click on the icon of the shared group in the Shared Group tab.
    2. Select Delete from the context menu. Please note that the Delete command is available only if the selected group icon is that of a group with no copies in the drawing.
       
  • To delete the archetypes of all groups with no copies in the drawing:
    1. Right-click in an empty area of the Shared Group tab.
    2. Select Delete Unused Groups from the context menu. Please note that the Delete Unused Groups command is available only if at least the archetype of one group with no copies in the drawing exists.

Details are non-persistent local shared groups
Please note that local shared groups created using the Detail command are non-persistent: their archetype is automatically removed when you remove all of their copies from the drawing.
You can check whether a selected shared group is persistent or not by right-clicking on it in the graphics area, selecting Edit Properties from the context menu and verifying the status of the Persistent check box in the Group category.
