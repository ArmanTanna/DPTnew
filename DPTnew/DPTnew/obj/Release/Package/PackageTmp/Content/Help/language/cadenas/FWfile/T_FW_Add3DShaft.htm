PARTsolutionsImport from PARTshaft


 

The Import from PARTshaft command enables you to add a 3D shaft to our current model.

When you run the command, the program starts CADENAS PARTshaft™ that enables you to design the shaft to import.
When you have completed the design of the shaft, click ExportExport. This action closes CADENAS PARTshaft™ and imports the shaft in our model for you to move it in place.

Refer to the Move/Copy command for information on how to use the positioning handles in order to place the shaft.

After positioning the shaft in the model, click:
to exit the command without any further action.
to apply the new shaft and quit the command.
to apply the new shaft and position another copy of it. The new copy is generated in the center of the current view.
