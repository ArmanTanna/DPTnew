InsertSheet Metal Flange

The Flange command enables you to add a flange to a sheetmetal on one of its linear edges selected for the Edge selector. Multiple flanges can be added contemperarily by using the specific Multi Flange command.

  • The Use Profile check box is used to create flange of various shapes by selecting the flange profile. On selecting this check box, the Edit Profile option is displayed. On selecting it, the profile mode is invoked and you can edit the profile of the flange as per your requirement.

  • The Length drop-down list is available on clearing the Use Profile check box. It enables you to specify the length of the flange. The Length drop-down list has following two options.
    By value Enables you to enter the required flange length value in the Length mini-dialog box in the Graphics Area.
    Up to point Enables you to create a flange by defining its upto a selected point.

  • By selecting the Offset check-box; an offset can be added to the flange from the selected edge.
    By value Enables you to specify the required offset distance in the Offset mini-dialog box.
    Up to point Enables you to extend the offset upto a selected point.

  • On selecting the Local Properties node, the Radius mini dialog box gets displayed in the Graphics Area. You can set bend radius value in it. The other following parameters of flange will get display on selecting the Local Properties node.
    Reliefs Enables you to provide relief while creating flanges. On selecting this node, the Reliefs dialog box gets displayed. The Bend and Corner are two types of relieves available in the drop-down list availble on the top of the Reliefs dialog box.
    Allowance Enables you to provide allowance to reduce the compression and tension effect on the bend of the flange.
    Rolling Threshold Enables you to define the value of the radius beyond which the k-factor values is set to 0.5, as the cylindrical surface is to be considered as rolled rather than as bent.
    Alignment Enables you to specify the alignment of the flange with respect to the selected edge.
    Length Enables you to select the method to be used for computing the unbend length.

    Note:
    To make any global change of the Sheet Metal parameters -see Sheet Metal category of the Entity Properties. And to change the parameters locally, select Local Properties and proceed as described in Local Properties and the Lock Marker.

  • Following options are available on selecting the More Options node:
    Side Angles Enables you to create side faces of the flange at an angle. To enter the angle values, the 1st Angle and 2nd Angle mini-dialog boxes are provided in the Graphics Area.
    Orientation Enables you to create a flange at an angle. On selecting the check box, the Reference Plane drop-down list is available to select a reference plane for the bend angle. The Angle drop-down list in the Graphics Area enables you to set value of the angle.
    • Edge face: The reference plane is the base face of the selected edge.
    • Line: The reference plane is the one defined by the selected edge and a line to be selected.
    • 2 points: The reference plane is the one defined by the selected edge and a line identified by two points to be selected.
    • X, Y, and Z: The reference plane is one of the X, Y and Z axes of the selected edge.
    Extension Enables you to specify the width of the flange. To do so, following are the different ways:
    • 1st distance and 2nd distance: To specify the distances from the endpoints of the selected edges.
    • 1st distance and width: To specify the distance from the first end point of the selected edge and the width value.
    • 2nd distance and width: To specify the distance from the second end point of the selected edge and the width value.
    • Keep width: The width of the flange will be same as the width of the selected edge.
    Adjacent Normal Side Face
    Note
    The Adjacent Normal Side Face check box gets displayed only in case the sheet metal model contains inclined face.
    • When this check box is cleared, the angle of the adjacent face is retained and flange to be created propagates with the adjacent face angle.
    • When this check box is selected, the adjacent inclined side face of the selected edge attains 90 degrees to create the flange.
