The New File command enables you to open a new document in a new empty window where you can create a drawing. the following ways:

  • clicking the icon, displays the ThinkDesign dialog box, from where you can select the document in the New File tab, if new, or in the Recent Files tab, if recent:
    New File tab Shows the different document icon types that you can select. The Change button enables you to define the current template to be used to open a new document.
    Note that you can customize the application startup by showing the dialog box or start with no document, an empty model or an empty drawing using the Startup options in the Options/Properties dialog box.
    Recent Files tab Shows the list of the recent files, from where you select one, and More files..., which opens the Open file selection box. If available, you get the document preview.
  • clicking the arrow button or performing the command from the menu bar and selecting the new document from the pull-down menu displayed.

The document that you open can be:

  • Model from Template — a new empty window in which you can create a model
  • Model Derived from Current — a model derived from the current one, linked to it so that when the former changes, the latter is updated accordingly.
  • Drawing from Template — a new empty window in which you can create a drawing
  • Drawing from Current Model — a new window containing drawing views of the current model.

Note

The new document is given a provisional name, which can be:
  • Model<number>, where <number> is a sequence number (Model1, Model2, ...), in the case of a model
  • Drawing<number>, where <number> is a sequence number (Drawing1, Drawing2, ...), in the case of a drawing
The provisional name will be replaced with the one you give it the first time you save it.
