Trimming a curve at its virtual intersection with another one

The Trim/Extend Curves command can trim two curves also when they are not lying on the same plane (unless they do not intersect in any plane or view direction). When a virtual intersection is detected, a proper message is displayed. Upon confirmation, the trimming operation is performed along the current view direction.

To accomplish the following task on your own, load file "TEC.e3", then start the Trim/Extend Curves command.

1 In the Mode drop-down list select First, so as to cut only the first of the two curves.
 
2 Select the first curve close to the virtual intersection point.


 
3 As soon as you move the cursor onto the second curve, a preview of the result is displayed.



When you click to select the second curve, a message is displayed to warn you a virtual intersection along the current view direction has been detected. Confirm to go on and cut the curve.
