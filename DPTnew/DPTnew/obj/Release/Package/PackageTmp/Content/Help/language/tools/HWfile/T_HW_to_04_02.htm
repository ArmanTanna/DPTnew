Spreadsheet - Assign/deassign a variable to/from a feature or a datum entity

It is possible to associate a variable with a feature or a datum entity. This association controls the enable/unable operation for these items:

  • When the variable value is 0, the item is not enabled
  • When the value is not 0 the item is enabled.

To assign a variable to any of these items there are two possible ways. Here's the first:

1 Click the line number corresponding to the variable you wish to assign.
2 In the Assign drop-down list, select the assignment mode:
 
Dimension/Feature/Component Enables you to assign the variable to a drive dimension or to a feature/datum entity. In the case of threaded holes/shafts, choosing this option will cause the variable to be assigned to the hole/shaft diameter rather than to the thread diameter or pitch.
Thread diameter When a threaded hole/shaft is involved, this option enables you to assign the variable to the thread diameter, rather than to the hole/shaft diameter or to the thread pitch.
Thread pitch When a threaded hole/shaft is involved, this option enables you to assign the variable to the thread pitch, rather than to the hole/shaft diameter or to the thread diameter.

3 The program prompts you to select an entity. Click on the item to which you wish to assign the variable.

Note

Do not try to assign variables to ...
  • Linear or Rotational sweep solids
  • Spined sweep solids
  • Sweep solids connecting two boundaries
If you try to do it, you'll get an "Invalid selection" message.
4 Click Rebuild.

The other way is the following:

1 In the Model Structure, right-click on the icon of the item you want to assign the variable to and select Assign Variable/Feature in the shortcut menu.
2 The spreadsheet is automatically displayed. Click the line number corresponding to the variable you wish to assign.
3 In the Assign drop-down list, select the assignation mode as described above. The Model Structure displays a specific icon (looking like a small "spreadsheet"), adjacent to the corresponding item.

If you change the value of the variable to 0, the item is deactivated and the Model Structure changes accordingly.

To deassign a variable from any of these items:

1 Click Deassign.
2 The program prompts you to select an entity. Click to select the item from which you wish to deassign the variable.
