Searching documents

This help is composed of two frames. The frame on the right-hand side gives you instructions on how to use the Help and perform the most frequent actions. The frame on the left-hand side, the Navigation Frame, helps you locate information. This frame includes fourthree tabs — Contents, Index, Search, and Favorites — enabling you to locate the information you are seeking by using the most convenient method:

  • Search by subject
  • Search by alphabetical list
  • Search by keyword/phrase
  • Frequently used topics

Search by subject

To search by subject, do the following:

  1. Select the Contents tab, which lets you locate information by subject category.
  2. Clicking on the plus sign (+) adjacent to a particular subject, you can expand that category to locate more detailed information.
  3. When you have located the information you are seeking, click on the category to view its description in the adequate pop-up window.

Search by alphabetical list

To search by alphabetical list, do the following:

  1. Select the Index tab, which lets you locate information by using an alphabetical list of subject categories.
    Note that you can scroll through the list, or begin typing in the box directly above the list.
  2. When you have located the desired subject, select it and then click the Display button (or double-click the subject) to view its description in the adequate pop-up window.
    As you begin entering, the list immediately scrolls to match the letters you have entered.

Search by keyword/phrase

To search by keyword/phrase, do the following:

  1. Select the Search tab, which lets you locate information by searching the entire Help for occurrences of a particular keyword or phrase.
  2. Enter the word or phrase that you want to find, then click the List Topics button to display a list of topics containing those keywords.
  3. To display the description, select it in the Topic list and then click the Display button (or double-click the subject).
    You can refine your search in many ways and use advanced search methods, as described in Advanced Search.

Frequently used topics

You can create your own list of frequently used topics and add topics to it. To add a topic to the Favorites list, do the following:

  1. Display the topic.
  2. Select the Favorites tab and select Add.
