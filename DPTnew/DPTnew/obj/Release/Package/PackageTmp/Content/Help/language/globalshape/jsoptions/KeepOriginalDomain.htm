This option applies to surfaces only and essentially affects the program behavior about surfaces.

  • When this check box is selected, the domain of the resulting entity will have the same size as the one of the original entity.
  • When not selected, the domain of the resulting entity may differ from the original (and be smaller: in fact it will be reduced to the trim boundary domain).
