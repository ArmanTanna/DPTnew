InsertSheet MetalBend

The Bend command enables you to create bends at sharp edges and/or fillets or on all the edges of a selected face of a solid.

After starting the command you will have to choose the command mode in the Bend drop-down list:
Sharp Edge/Fillet to create a bend on a selected sharp edge or a fillet.
Face to create bends on all the edges or adjacent fillets of a selected face.
After choosing the mode, you just need to select the face or the sharp edge/fillet (depending on mode you chose). In the case of edges/fillets, the command will help you perform your selection by not enabling you to select edges which wouldn't make sense.

Reliefs are automatically created when necessary, according to the parameters defined in the Corner Reliefs page inside Sheet Metal category of the Entity Properties.

You can assign specific values to the related Sheet Metal parameters (different from the general ones defined in the Sheet Metal category of the Entity Properties) by selecting Local Properties and proceeding as described in Local Properties and the Lock Marker.
