Computation in Mirror Mode

In the Process Compensator and Identify Displacement Field commands, you can work based on the displacement field symmetry.

  • If a symmetry is there, by selecting the displacement field symmetry option you are ensured the final Global Model deformation will be a symmetric one.
  • You can also set the symmetry option to run the computation on either half or on the entire displacement field and be sure to obtain the result relative to the entire displacement field.
  • A check tool is available to detect symmetries in meshes (see "Check Symmetry"). Once a symmetry is detected, you can use the symmetry option to ensure the result is symmetric.

To activate the symmetry method, select the Symmetry check box under More Options.
 

Defining the symmetry plane
The Reference Plane drop-down list enables you to choose the selection method to define the symmetry plane:
On Face A face of a solid is to be selected (the Face selector will then turn to green).
3 points Three points identifying the plane are to be specified (the 3 points selector will then turn to green).
Datum plane An existing datum plane is to be selected (Plane).
Through axis & point Select an axis (Axis, see "How to define an axis or a line") and a point (Point) which the plane must be through.
Perpendicular to axis & through point Select an axis (Axis, see "How to define an axis or a line") the plane must be perpendicular to and a point (Point) the plane must be through.
Coefficients Specify the coefficient values of the cartesian equation of the plane (Ax+By+Cz+D=0) in the respective mini-dialog boxes (Coeff. A, Coeff. B, Coeff. C, Coeff. D), and a through point (Point).
