Entity Properties

Category: Drawing View-Visibility

View mode
View mode

The control in this area enables you to specify the mode to display the drawing view.

Show
Show:

Enables you to choose the type of mode to display the drawing view with. The possible selections are as follows:

Geometry only (default) In this mode, the view has no shading and is drawn as a set of curves (only this mode was present in previous releases).
Geometry and Image In this mode, shading is added to the view using a bitmap image as background. Since geometry is still present, the view is fully associative — it is possible to add dimensions and geometry. You are enabled to customize the image resolution, rendering modes, and effects parameters in this mode using the Image category under the Drawing View - Visibility category.
Image only In this mode, the drawing view is represented as a bitmap image captured from the 3D model. The views created in Image only mode are much faster to regenerate, but the 3D geometry represented in the drawing as image is not pickable and then it is not possible to add dimensions and geometry. You are enabled to do the similar customization of the image as done in Geometry and Image mode. Additionally you can also choose the "edge" types to be displayed in the view.
Geometry
Geometry

The controls in this area determine the geometric types of entities that are visible in the next view you create.

Representation
Representation:

Determines the way in which the next view is displayed, by selecting one of the following from the drop-down list:

Wireframe displays the view in wireframe mode
Hidden lines displays the view in wireframe mode with hidden lines
Hidden Lines (ver. < 2008.1-SP1) when editing old drawing views created using obsolete options that have been removed (Hidden line+axis and Add axes to holes and rotational slots), this option displays the view in wireframe mode with hidden lines and makes the mentioned removed options available for your use.
Show boundaries between tangent faces
Show boundaries between tangent faces

When selected, boundary lines between consecutive tangent faces are displayed.
When not selected the said boundary lines are not displayed.
Show dashed hidden lines
Show dashed hidden lines

When selected, hidden lines are represented by dashed lines.
When not selected, hidden lines are not displayed using dashed lines.
Add axes to rotational surfaces
Add axes to rotational surfaces Add axes to rotational surfaces

When this check box is selected, axes are automatically added to all rotational surfaces while creating custom views. Also, the options in the Filter axes area are grayed out.
When not selected, axes are not drawn in the said cases.
Filter axes
Filter axes

The controls in this area enable you to determine the type of axes to be created in a drawing view.

Datum axes
Datum axes

When selected, datum axes are displayed.
When not selected, datum axes are not displayed.

This option is grayed out when the Add axes to rotational surfaces option is selected.

Centerline curves
Centerline curves

When selected, centerline curves are displayed.
When not selected, centerline curves are not displayed.

This option is grayed out when the Add axes to rotational surfaces option is selected.

Profile references
Profile references

When selected, profile references are displayed.
When not selected, profile references are not displayed.

This option is grayed out when the Add axes to rotational surfaces option is selected.
