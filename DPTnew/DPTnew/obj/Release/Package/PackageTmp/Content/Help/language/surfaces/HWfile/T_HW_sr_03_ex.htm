Creating a Revolved Surface

  1. Open the file "sr_01_03.e3".



    You will create a revolved surface starting from the two blue curves on the left and using the red vertical line as the axis of revolution.
     
  2. Start the Rotational Surface command.
     
  3. Select the two blue curves.



    To end selection, right-click and select Continue in the context menu.
     
  4. Select the axis of revolution by clicking on the red line:



    A preview is immediately displayed.


     
  5. Select to create the surface:
