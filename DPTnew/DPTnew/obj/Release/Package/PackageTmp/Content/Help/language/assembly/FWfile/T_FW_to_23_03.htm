In the lists displayed in the Document View tab and in the Part View tab, each item is associated with two icons:

  • a color-coded "floppy disk", the color of which shows the information related to the item as follows:
    • green: the item includes both the document and the part data
    • yellow: the item includes the document data, but lacks the part data
    • red: the item lacks valid document data

    In addition, the items which haven't been previously saved into any project can be distinguished by a "spark" on the top-left-hand corner of the "floppy disk" icon. Components not bound to be saved in the archive (local components or X-Refs in local copy or read-only ones) don't have an associated "floppy disk" icon.

  • an Item-Type icon — identical to the ones in the ModelDrawing Structure — which shows the type of item (local or external component, elementary part or sub-assembly, assembly).
