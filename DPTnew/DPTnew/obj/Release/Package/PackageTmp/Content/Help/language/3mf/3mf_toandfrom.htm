Converting models to and from the 3MF format

From 3MF to think3

  • The result of the conversion is a Mesh entity (for details, see "Display information about a selected mesh")

From think3 to 3MF

  • As a result of the conversion you will get 3MF archive file. Standalone curve and points will be ignored since the 3MF file contains exclusively meshes and components. Surfaces and solids need to be transformed into meshes prior to creating the file, this is performed according to the current values of the Settings tab in the 3MF Options dialog box opened by selecting Options.
  • If there are problems in the objects inside the 3dmodel.model file of the archive, check the log file of the conversion. This log is created in the same directory as the saved file.
