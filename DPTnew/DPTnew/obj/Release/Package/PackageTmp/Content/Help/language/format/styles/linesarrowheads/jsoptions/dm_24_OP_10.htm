Enables you to determine how the witness lines must be drawn with respect to the reference point/origin. Two alternative methods are available, enabling you to set, for the witness lines, either their distance from the reference point/origin or their length:

Variable Enables you to determine the distance between the witness line and the reference point/origin. When this option is selected, the Gap box is displayed.
  • If the adjacent check box is selected, the value you type inside the Gap box is a multiplying factor that will be applied to the dimension text Size: the resulting value is the distance between the reference point/origin and the starting point of the witness line.
  • If the adjacent check box is not selected, the value you type inside the Gap box is the actual distance between the reference point/origin and the starting point of the witness line, in the current unit of measure.
Fixed Enables you to determine the length of the witness lines. When this option is selected, the Length box is displayed.
  • If the adjacent check box is selected, the value you type inside the Length box is a multiplying factor that will be applied to the dimension text Size: the resulting value is the length of the witness lines.
  • If the adjacent check box is not selected, the value you type inside the Length box is the actual length of the witness lines, in the current unit of measure.
