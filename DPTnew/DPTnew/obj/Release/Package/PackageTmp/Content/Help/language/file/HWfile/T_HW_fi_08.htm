Saving your document partially

1 On selecting the Partial Save command, the Partial Save file selection dialog box is displayed.
2 In the File name box, specify the file that will include the entities of your partial saving.
3 Select the entities you want to save and end selection.
4 The program displays a dialog box where you are asked to confirm the selection. Click Yes to save the selected entities to the new document, or No to interrupt the command without saving the new document.
Tips and Tricks
  • In order to save the file in different formats, define specific options, save to the local file system or to the Intranet, see the Partial Save or Save As commands.
