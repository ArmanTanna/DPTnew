Applying constraints along the cutting lines

When modeling an object, the line you select as a reference also defines the portion of the object which the controlled modification will be applied to. One or two "cutting lines" are thus identified passing through the reference line end points.
 

(1) — the line used as a reference
(2) — the First cutting line
(3) — the Second cutting line.

Constraining conditions to be applied along each of the two cutting lines can be selected under More Options respectively in the First and in the Second drop-down lists.

  • A description of each constraint that can be applied along the first cutting line is provided in "Constraints along the first cutting line".
  • See "Constraints along the second cutting line" for a complete description of constraints that can be applied along the second cutting line.
