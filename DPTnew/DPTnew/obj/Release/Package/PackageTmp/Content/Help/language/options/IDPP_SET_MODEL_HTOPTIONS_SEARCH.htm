System Options

Category: Model Structure-Search

Search

Search

Options in this area define the advance settings for the Model Structure search.

Include hidden layers

Include hidden layers

When selected, all the entities on the hidden layers will be included while performing the search in the Model Structure.

Only in current component

Only in current component

When selected, the Model Structure search will be performed only inside the component that has been set current.

Inside thinkteam description

Inside thinkteam description

When this option is checked, the search is extended to the thinkteam part description, provided that the latter is displayed by checking the thinkteam description option of the ThinkDesign category of the thinkPLM Options.
