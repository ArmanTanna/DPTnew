3MF Conversion Options

When saving (see Save and Save As) a 3MF file by selecting 3MF file in the Files of type/Save as type drop-down list, the Options button in the file selection box enables you to define the parameters to be used in the conversion process.

A dialog box displaying the following categories shows up:

  • Settings
    This category contains the most common parameters to be set when converting into 3MF.
