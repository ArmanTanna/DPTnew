ModifySolidReorder


 

The Reorder Feature command enables you to change the order in which a feature is applied to a solid, thus applying the selected feature before or after another feature applied to the same solid. This capability can be quite useful, particularly when a dimensional change interferes with two features and their logical order is not correct.

When you perform the command, you are enabled to reorder features only in a consistent sequence, that is, you can never reorder features before their parent one(s).

Note that you can also use the Drag & Drop function to reorder the structure by selecting a feature icon and dragging it to a new position within the Model Structure.

Regardless of the method you use, you may need to rebuild the solid in order to see the changes.
