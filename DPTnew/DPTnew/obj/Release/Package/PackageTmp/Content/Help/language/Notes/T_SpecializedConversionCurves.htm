Specialized conversion

When the Fix mode of the Convert to NURBS command is set to Tolerance, all curves are generally converted to cubic curves.
Selecting the Specialized check box under More Options you can obtain "specialized" NURBS curves instead.

Specialized conversion should be performed only if really needed

Take into account what follows:

  • Many packages do not support rational curves; if you need to exchange data with one of those packages, you will have to deselect the Specialized check box, as the curves you obtain from a special conversion process are generally rational.
  • On the other hand, rational curves obtained running a special conversion process are much more accurate and have a lower number of control points.
