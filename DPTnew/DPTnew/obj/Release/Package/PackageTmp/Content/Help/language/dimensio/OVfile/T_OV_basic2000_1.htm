Drive dimensions

Drive dimensions can be divided into the following two types:

  • Dimensions of constraints on variational profiles, which you can add to profiles using the Smart Dimensions command
  • Dimensions generated upon creation or editing and mating of solids, which are automatically added by the program.  

Drive dimensions constitute the framework of the dimension-driven structure of think3 application's solids. It is by modifying them that you can make dimensional changes to the model. They are like other dimensions only in their graphic aspect but their function is completely different.

In addition to Smart Dimensions, here is a list of commands which automatically add driving dimensions to the corresponding solids/features. See the corresponding pages to know more.



Linear Slot/Protrusion to create Linear Sweep Solids, Slots and Protrusions
Rotational Slot/Protrusion to create Rotational Sweep Solids, Slots and Protrusions
Pipe Slot/Protrusion to create Pipe Sweep solids
Hole & Shaft to create Holes and Shafts
Fillet to create Fillets
Chamfer to create Chamfers
Shell to create Shell
Draft Angle to add Draft Angles
Pattern to create Patterns
Mating Mating tools: Coincident, Parallel, Concentric and Side

You can blank drive dimensions from view or display them using the Show Driving Dimensions and Hide Driving Dimensions commands. While you can add or delete dimensional constraints to profiles as you like, you cannot delete drive dimensions automatically inserted by the program, as they form part of the structure of the model.

The parametric values of driving dimensions can be changed using the Edit Dimension command.
