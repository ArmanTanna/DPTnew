An Overview of Adaptation

During manufacturing processes, prototypes are created to validate the part before the production starts. Many a times these prototypes are modified (either by hand or by machining) to meet specific design requirements. Since the high quality digital data already exits (in form of the CAD model), it is sensible to match it with the measured data of the modified prototype (obtained by scanning process).

Adaptation tool solves this problem of creating a digital model which is identical to the physical tool. The original surfaces of the 3D model are adapted on the basis of the physical model data (obtained by scanning).

1 is the original CAD data

The Adapt on Measured Data command performs the adaptation computation. The result is a small GSM surface, which contains the computed transformation. This transformation surface can then be used by the GSM Replicate command to apply the modification on the base entity to match it to the measured entity.
