STL

Category: Settings

Save

The controls in this area control the conversion to STL format

Tolerance

Determines the tolerance for tessellation as the distance between the real surface and the plane of approximation

Use current tessellation

When selected, tessellation is controlled by the shading parameters specified in Tessellation on the Graphics of the Options dialog box and all the other parameters on this page are disabled.

File format

The controls in this area determine the output format of the resulting file

ASCII

Determines the format of the resulting STL file: ASCII - ASCII STL format; Binary - binary STL format

Binary

Determines the format of the resulting STL file: ASCII - ASCII STL format; Binary - binary STL format

Save assembly as

The controls in this area determine the output format of the assembly

Single file

Determines how think3 saves STL files of models containing more than one solid: Single file - all the solids are saved in one file; Multiple files - each solid is saved to a separate file. The name of the first file is specified by the user. The program assigns the name to subsequent files by appending _ (underline) character and a unique number to the user-specified file name (for example, file .stl, file _1.stl, file _2.stl, and so on).

Multiple files

Determines how think3 saves STL files of models containing more than one solid: Single file - all the solids are saved in one file; Multiple files - each solid is saved to a separate file. The name of the first file is specified by the user. The program assigns the name to subsequent files by appending _ (underline) character and a unique number to the user-specified file name (for example, file .stl, file _1.stl, file _2.stl, and so on).
