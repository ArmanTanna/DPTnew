Solids

Solid Difference Creates a solid by subtracting one or more solids from a base solid.
Intersect Solid Creates a solid from the intersection of two or more selected solids.
Solid Union Joins one or more selected solids to form a single solid.
Edge Chamfer Enables you to chamfer selected edges of a solid.
Face Chamfer Enables you to chamfer all the edges of the selected face of a solid.
Cut/Split Cuts a solid using any surface already existing in the model.
Rib Enables you to create a rib to support the component.
Datum Line Adds a reference axis relative to the typology of an object to the model.
Datum Plane Adds a datum plane to the model, thus creating a geometrical relationship between the object and the plane.
Datum Point Adds datum points to the model with reference to an existing object.
Draft Angle Adds a draft angle to one or more faces of a solid.
Edge Fillet Enables you to fillet selected edges of a solid.
Face Fillet Enables you to fillet all the edges of the selected face of a solid.
Solid Fillet Enables you to fillet all the edges of a selected solid.
Face to Face Fillet Enables you to create a constant radius fillet in between two selected faces.
Generic Protrusion Creates sweep solids and protrusions by moving a boundary along a profile or connecting two or more profiles (along a spine or not).
Generic Slot Creates slots by moving a boundary along a profile or connecting two or more profiles (along a spine or not).
Generic Solid Creates solids by moving a boundary along a profile or connecting two or more profiles (along a spine or not).
Linear Protrusion Adds a linear protrusion to a face of a solid by sweeping a profile or chain of curves along a linear path.
Linear Slot Creates a slot in a solid by sweeping a chain of curves or a profile along a linear path.
Linear Solid Creates a solid by sweeping a chain of curves or a profile along a linear path.
Solid from Text Creates a set of solids from curves defined by a selected font.
Make Solid Creates solids from selected surfaces, which become the faces of the resulting solid. An associative link can be created between the solid and the surfaces.
Convert to Subdivision Solid Converts any type of solid or mesh into a Subdivision solid.
Subdivision Solid from Sections Creates Subdivision solid from Sections.
Subdivision Solid Enables to modify a Subdivision solid in a number of possible manners.
Extract Subdivision Mesh Creates the Mesh entity associated with a selected Subdivision solid. The mesh can be output in the ".obj" format.
Mirror Enables you to copy a solid or a feature or a set of features by mirroring them about a plane of symmetry.
Edit Dimension Changes the value of a parametric dimension.
Move Base Profile Enables you to move a base profile, which forms part of a solid.
Pattern Enables you to create multiple copies of a set of features or a solid in a linear or angular pattern.
Pipe Protrusion Adds a pipe protrusion to an existing solid.
Pipe Slot Creates a slot in a solid with a circular cross section along a path.
Pipe Solid Creates a solid with a circular cross section along a path.
Rotational Protrusion Adds a protrusion to a face of an existing solid by revolving a profile or chain of curves about an axis of revolution.
Rotational Slot Creates a cutout in a solid by revolving a profile or chain of curves about an axis of revolution.
Rotational Solid Creates a solid by revolving a profile or chain of curves about an axis of revolution.
Shell Enables you to create a shell from a closed solid, or a solid with parametric thickness from an open solid.
Joint Lip Enables you to create a lip or a groove feature by driving a profile along a drive curve.
Hole Enables you to create a simple and shaped holes in a solid.
Shaft Enables you to add a simple shaft to a solid.
Solid from Faces Creates a static solid obtained as the copy of one or more faces of existing solids.
Split Faces Enables you to split the faces of a selected solid at their intersections with a set of other solids.
Unlink Solid Enables you to remove external links that may have been previously created.
Thread Enables you to add threads to the inner or outer parts of solids or features bounded by cylindrical surfaces.
Modeling Provides access to a set of information and editing utilities for the management of solids.
Zone Draft Enables you to add a draft angle to one or more faces of a solid a posteriori.
Zone Modeling Enables you to efficiently apply controlled modifications to zones of a solid.
