ViewModifySet Current View


 

The Set Current View command enables you to define the current view, when the display is split into multiple views (see Set Views).

After starting the command, click in the view to be set as the current one.

Using the context menu
An alternative method to set the current view is to right-click on the the view you want to set as the current one and to select Set Current View in the context menu.
