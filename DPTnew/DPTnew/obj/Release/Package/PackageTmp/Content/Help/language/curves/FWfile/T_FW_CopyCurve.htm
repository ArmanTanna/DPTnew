Insert Curve Copy

The Copy Curve command enables you to perform the copy of one or more existing curves, or to create curves as copies of selected solid edges.

Select each curve/solid egde you need to create a curve from (the Curves item in the selection list will turn to green) and then select ( ) Apply or ( ) OK to create the curves.

Each newly copied entity will have its own entry in the Model Structure, thus retaining the same geometric characteristics as the original one, while being absolutely separate from it. Nevertheless, the command also enables you to assign associativity relationships and to decide whether to keep the resulting curves linked to the original ones or not, as explained below.

The Associative Mode check box controls associativity relationships:

  • When the check box is selected, the resulting curves will be associative. The new curve will retain a link to the entities it derives from. A specific node displaying both the original entities and the resulting associative curve will be added to the Model Structure.
  • When the check box is cleared, the resulting curves will not be associative.

In addition:

  • If the Associative Mode check box is selected, the Linked copies box shows up in the selection list.
    • If Linked copies is cleared, the copies are not linked to the original. Whatever may happen to the original curves, the copies will not change.
    • If Linked copies is checked, the copies are linked to their original curves. Any time the original curves are modified, the copies will be modified accordingly.
  • If the Associative Mode check box is cleared, the Linked copies box is not available.

If you interrupt the command session, your selections and settings are not lost: they are "frozen" so that you can reuse them as described in "Restore Selection modality for selections and settings"

Finally, please note that in order to create curves from surface boundaries you can use the Boundary Curve command.
