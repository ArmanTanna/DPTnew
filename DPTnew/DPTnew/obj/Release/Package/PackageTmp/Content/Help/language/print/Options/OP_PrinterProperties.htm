Opens the current printer driver properties dialog.
If the selected driver is a Windows printer, please refer to the manufacturer documentation.
If it is a think3 internal driver, the following information is displayed:

  • The printer Name as it is defined in the Name=  field of the relevant initialization file.
  • The printer Type as it is defined in the Type=  field of the relevant initialization file.
  • Where the output is currently being redirected (if the destination is a file, you are enabled to change its path).
  • A comment with further notes about the printer as defined in the Comment=  field of the relevant initialization file.
  • The current output Paper Size, selected from the list of formats defined in the relevant initialization file. You are enabled to change the paper size by selecting a different one from the list.
  • The current paper Orientation, which you are enabled to change between Portrait and Landscape.
  • The Special Options, as defined in the Area SPECIAL  of teh relevant initialization file.
