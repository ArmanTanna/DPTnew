InsertCurveSpined

The Spined Curve command enables you to position plane curves along another curve which will be used as a spine.

A spine can be used as a curve controlling the shape of a surface and/or as a drive curve used to obtain spined surfaces. Spined surfaces are created by "moving" one or more boundary curves along a given path. To obtain this type of surface, the following conditions must be satisfied:

  • The placement plane of the boundary curve must be orthogonal to the spine.
  • The spine must intersect the placement plane of the boundary curve.

Both of these conditions can be obtained automatically starting with any curve by using the Spined Curve command.

After selecting the boundary curve to be moved and the spine curve, in order to carry out the operation, use a Reference plane onto which the boundary curve is translated so that it is orthogonal to the spine curve.

You should therefore define the active reference system in such a way that:

  • The origin coincides with the point on the boundary curve that must be positioned on the spine.
  • The xy plane coincides with the "reference plane" which you intend to define for the spine: the directions will be defined by the intersection of the axes of the spine curve reference system (Z axis in the direction of the tangent; X axis in the direction of the normal; Y axis in the direction of the binormal) with the selected plane.
A new curve, identical to the original, will be created and positioned in a plane perpendicular to the spine, with the origin of the active reference system at the preselected point on the spine curve. You can move the curve along the spine by dragging a handle or by specifying the Parameter value.
