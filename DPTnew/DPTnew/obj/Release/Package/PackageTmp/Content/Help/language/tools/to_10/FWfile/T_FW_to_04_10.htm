The Shortcuts tab of the Customize dialog box enables you to create and modify keyboard shortcuts.

The program enables you to assign shortcut keys to commands. The Shortcuts tab contains the following tools that enable you to assign, remove or modify the shortcuts:

Category: Select the command category that contains the command of which you want to assign or change the shortcut keys.
Commands: Choose from the list of commands belonging to the selected category the one you need.
Description: A short description of the command. It appears only when a command is selected from the Commands: list.
Current keys: The list of shortcut keys associated to the command selected from the Commands: list. Although the program does not allow you to assign the same shortcut to different commands, it will accept multiple shortcuts to the same command.
Press a new shortcut key: After you click in this box, the system captures any combination of keys you press on the keyboard; this will become the new shortcut to the selected command.
Insert the new voice text: Available only if the Speech Interface has been installed and is enabled: otherwise the box is grayed out. It enables you to customize the voice associated to a command (that is: the sentence you have to pronounce to start a command through the Speech Interface — please note that each default command voice is displayed in the English and Italian documentation on top of the command description page). In order to customize the voice associated to a command, proceed as follows:
  • In the Category: list, select the category which the command you want to customize the voice of belongs to.
  • Once the category has been set, in the Commands: list select the command you want to customize the voice of.
    The current voice shortcut associated with the command will be displayed in the Current keys: box.
  • In the Insert the new voice text: box, type the new voice shortcut for the selected command.
  • As an interactive check, you can click the Play Voice Text arrow to have the program read out the new shortcut that you have associated with the selected command.
  • Select Assign to confirm. The new voice shortcut is thus actually associated with the selected command.

    Customization and Online Help
    Please take into account that if you customize the voice shortcut for a command, the one specified in the command description of the online documentation will not correspond to the active one any longer. As a consequence, in case you should forget a voice shortcut you have customized, you cannot refer to the online help so as to remember it. The only way is through the Customize- Shortcuts tab.
Currently assigned to: This textbox appears only when entering a new shortcut of the combination you are trying to use. Note that if you assign the shortcut to the current command, it will be removed automatically from the old command.
Assign Click this button to assign the newly defined shortcut. The command is enabled only if you enter a valid shortcut in the Press a new shortcut key: field; some shortcuts, such as CTRL+O – File Open, cannot be changed. If you try to assign these, the command will remain disabled.
Remove Click this button to remove the shortcut highlighted in the Current keys: list. Some shortcuts, such as CTRL+O – File Open, cannot be removed. If you select one of these, the command will remain disabled.
Reset All Removes all custom assigned shortcuts if you click OK in the confirmation dialog.
