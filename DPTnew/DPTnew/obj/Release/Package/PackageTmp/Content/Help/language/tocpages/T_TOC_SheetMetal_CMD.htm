Sheet Metal

Bend Enables you to bend solids along edges or fillets.
Close Corner Enables you to close the corner where two walls meet.
Face Rip Enables you to create a rip by removing one or more faces of a solid.
Extruded Flange Enables you to create a flange starting from a profile.
Flange Enables you to add a flanges on a solid edge.
Multi Flange Enables you to add flanges on solid edges.
Flat Pattern View Enables you to create a flat pattern view of a Sheet Metal part.
Hem Enables you to add hems on solid edges.
Rebend Enables you to rebend selected (or all) faces of an unbent solid.
Edge Rip Enables you to create rips on edges of solids.
Sketched Bend Enables you to bend a solid along a profile lying on a face.
Sketched Rip Enables you to create rips along profiles lying on solid faces.
Vertex Rip Enables you to create rip on the vertex of a solid.
Solid Flange Enables you to create a sweep solid using the current thickness of the Sheet Metal Properties.
Step Bend Enables you to replace an existing fillet and/or bend with a step bend.
Unbend Enables you to unbend selected (or all) faces of a bent solid.
Bend Table Manager Enables you to manage bend table data.
Sheet Metal Wizard Enables you to create multiple sheet metal components starting from a single solid (with or without sheet metal attributes).
Convert Solid to Sheet Metal Enables you to convert select solids to seheet metal entities, complete of their sheet metal properties, and capable of being used for a Flat Pattern View.
