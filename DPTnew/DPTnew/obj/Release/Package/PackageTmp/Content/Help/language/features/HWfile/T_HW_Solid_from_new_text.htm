Creating a Solid From a New Text

  1. Start the Solid from Text command.

  2. In the Mode: drop-down list, choose the Enter text option.

  3. An edit box is displayed by clicking on the graphics area next to the Text selector in the selection list. Enter think3 in the edit box.



  4. The font and font style of the text can be changed in the Character dialog box that is displayed by clicking on the Font option.



  5. Now you need to specify the origin of the text which will define the location of the protrusion of the text to be created. To reselect the origin, click on Origin in the selection list and select the origin again.





  7. Click or and create the solid from new text.
