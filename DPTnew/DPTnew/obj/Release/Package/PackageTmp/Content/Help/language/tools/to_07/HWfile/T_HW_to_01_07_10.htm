Display information about a selected Rotational Surface

If you select a rotational surface, the program displays a dialog box showing:

  • Coordinates of the surface Origin
  • Components of the surface Axis
  • Revolution angle
  • If the surface is trimmed, the Trimmed check box will be selected and the Limiting curves and Limiting islands boxes will show the corresponding values.

Furthermore, in the graphics area it also displays:

  • The axis of revolution
  • Arrows indicating the two parametric directions.

Finally, if you select the Area check box, the program also displays the surface area.
