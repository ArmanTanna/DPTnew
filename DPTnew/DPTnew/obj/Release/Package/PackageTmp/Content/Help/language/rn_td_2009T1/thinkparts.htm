New thinkparts Functionalities and Improvements

Version 2009.1 introduces the following enhancements and improvements to the thinkparts functionalities.

  • The new Import from report CSV command

The new Import from report CSV command

The new Import from report CSV command enables you to import the values of a catalog from a ".csv" file.


You can find this command in the dialog box of the Edit Values command.


Combined with the existing CSV Report, it provides you with a powerful tool to edit the values of a catalog. In fact, you are enabled to use CSV Report to open an Excel file with the current values of the selected catalog, and then use Import from report CSV to load the edited values.
