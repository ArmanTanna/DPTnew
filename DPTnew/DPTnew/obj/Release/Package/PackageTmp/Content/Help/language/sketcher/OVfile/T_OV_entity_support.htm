Entity Support in the Profile Environment

Both the 2D and 3D profile environment of the think3 application supports creation of various entities. These are:

2D Profile environment
  • Points
  • Straight Lines
  • Arcs
  • Circles
  • Conics
  • Splines
3D Profile environment
  • Points
  • Straight Lines
