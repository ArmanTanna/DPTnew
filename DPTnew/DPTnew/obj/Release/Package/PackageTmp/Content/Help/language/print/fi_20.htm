FilePrint Preview

The Print Preview command enables you to preview how the document will look when you print it. On a sheet with the proportions of the selected format, inclusive of the hardware margins of the printing device, Print Preview draws the picture of the objects to be printed and enables you to set parameters such as drawing scale and paper rotation.

When executed, the command opens a dialog that carries on the top part the following commands:

Command Description

Print		 Prints the document as it appears in the preview window. To exit the preview window without printing, click the Close button.

Print Setup		 Starts the Print Setup command that enables you to define the printer to be used and its setup.

Zoom		 Lets you zoom in and zoom out alternately by preset zoom factor.
Zoom Factor Enables you to set the zoom factor to be used by Zoom. Choose a value from the pull-down list to change the zoom factor.
Scale 1: Specifies the print scale. If you set a value greater than 1, the program prints a reduced image. If you set a value less than 1, the program prints an enlarged image. Press TAB after entering the new scale to see the effect. If you press RETURN the dialog will close without any further action.
Fit to page Calculates the scale so that the drawing fits exactly into the page, whose size you can set using Paper size:.
Paper size: Specifies the print sheet size. (This function is enabled only if the printer you are using supports it. Otherwise, it can be accessed from Print Setup.)

Portrait		 Positions the sheet with the long side vertical.

Landscape		 Positions the sheet with the long side horizontal.
View Mode Enables you to control the visualization characteristics of the 3D model being printed by selecting one of the following modes:
Wireframe View
Shaded View
Shaded View + Boundaries
Shaded View Boundaries + Hidden Lines
Boundary View
Boundary View + Hidden Lines
These commands are enabled when the With shading option is checked and a 3D model is being printed.

Help		 Displays the help file.
Close Closes the dialog box.

Using the mouse

When the preview window is displayed, the mouse buttons have the following functions:

Button Function
Left Lets you zoom in and zoom out alternately by preset scale factors.
Middle (scroll) Lets you zoom in and zoom out alternately by preset scale factors.
Right (Keep pressed) MovesMoves the drawing on the sheet.
Ctrl + Right Moves the image (including the format and the title block) on the sheet. The document is scrolled in any direction ("Pan" function).
Shift + Right Enables the dynamic zoom function.
