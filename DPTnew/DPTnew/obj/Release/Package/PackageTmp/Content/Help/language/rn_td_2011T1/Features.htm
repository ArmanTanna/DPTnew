New Part Modeling Functionalities and Improvements

Version 2011.1 introduces the following general enhancements to the Part Modeling functionalities.

  • The new External Relations Info command
  • Enhancement in the Hole, Linear Slot, Rotational Slot, Generic Slot, and Pipe Slot command
  • New Subdivision Commands and Functionalities

The new External Relations Info command

The new Parent/child relations command has been introduced. It enables you to check all external relations that a solid/component has with the external entities. To do so, right-click the solid/component from the Model Structure or Graphics Area, select the External Relations Info option from the context menu to display the Entity External Relations dialog box. The following two new options have been introduced in the Show relations drop-down list.

  1. Events with external parents: Enables you to list the events inside the solid that use reference geometry that is outside the solid itself (external parents). This list must be empty in order to be able to move the solid.

  2. External children: Enables you to the list the events outside the solid that use reference geometry inside the solid.

Enhancement in the Hole, Linear Slot, Rotational Slot, Generic Slot, and Pipe Slot commands

A warning message gets displayed in case of following features - Hole, Linear Slot, Rotational Slot, Generic Slot, and Pipe Slot has been added to a solid and the feature is intersecting with the solid.

New Subdivision Commands and Functionalities

Subdivision surfaces (from now on: SUBD) are a new powerful modeling tool available for the control of solids. Similarly to how NURBS are a way to control a surface using points, SUBD are a way to control a solid (closed or not) using points. While in a NURBS the control points forms a rectangular mesh, SUBD have a generic mesh with arbitrary topology. What is really outstanding is that when you move the points the solid will keep connected and the faces will retain their reciprocal tangency and curvature continuity conditions.

The new commands available are the following:

  • InsertSolidConvert to Subdivision Solid
    It converts any type of solid or mesh into a Subdivision solid, which can be modified using the Modify Subdivision Solid. See Creating a New Subdivision Solid.
  • ModifySolidSubdivision Solid
    It enables you to modify a Subdivision solid in a number of possible manners. See Modifying a Subdivision Solid.
  • InsertSolidExtract Subdivision Mesh
    It creates the Mesh entity associated with a selected Subdivision solid, so that you can communicate it by outputing it in the ".obj" format. See Creating the Mesh Entity Associated with a Subdivision solid.

 

Creating a New Subdivision Solid

In order to create a Subdivision solid proceed as follows.

  1. Start the Convert to Subdivision Solid command.
  2. Select a mesh or a solid (Subdivision Solid/Mesh).

    Note
    Please take into account the conversion will be much more effective and accurate if the faces of the selected mesh/solid are as square as possible and contain as few holes as possible.

Modifying a Subdivision solid

You can interact with a Subdivision solid in all usual ways, that is by creating features on it, moving it, creating drawing layouts of it, and so on. However, there is a new and special way that is modifying it by editing its controlling mesh. To do it proceed as follows.

  1. Start the Modify Subdivision Solid command.
  2. Select the Subdivision Solid.



    The associated mesh is displayed:



  3. Select the Edit Mode:
    • Move face
      Enables you to perform the modification by selecting one face and moving it with the usual transformation handle, which shows up as soon as the selection is done. Please pay attention when selecting the face: you need to move the cursor onto the inside part of the face in the mesh selection is performed using a ray-cast.
    • Move edge
      Enables you to perform the modification by selecting an edge and moving it with the usual transformation handle, which shows up as soon as the selection is done. Please pay attention when selecting the edge: you need to click very close to in the internal side of the face in the mesh.
    • Move vertex
      Enables you to perform the modification by selecting an edge of a face and moving it with the usual transformation handle, which shows up as soon as the selection is done. Please pay attention when selecting the vertex: you need to click very close to in the internal side of the face in the mesh. In order to select more than one vertex contemporarily, you can use the Window and Lasso tools, and also the <SHIFT> and/or <CTRL> keys of the keyboard as described in the note below.
    • Move chain
      Enables you to perform the modification by selecting an edge, and all the edges connected to it in a chain will be selected as well.
    • Subdivide face
      Enables you to perform the modification by splitting a selected face. This can be most useful if you need to add details.
    • Extrude face
      Enables you to perform the modification by extruding a selected face. In this case, new faces will be automatically created according to the extrusion.
    • Remove face
      Enables you to perform the modification by removing a selected face. Since there is a one-to-one correspondence between mesh faces and subdivision surfaces, one surface will be deleted and the adjacent surfaces will change their shape accordingly.
    • Split face
      Enables you to perform the modification by splitting a selected face. Please pay attention when the selecting the face: need to pick it near an edge along which the face will be split. The split will be propagated along the neighboring faces.

A number of useful options are available so as to make the command more effective.

  • Symmetric Movements
    When you move some vertices, edges or faces and there is some symmetry, then also the symmetric vertices, edges and faces will be moved. The possible symmetry modes are the following:
    • Work Plane based
      The symmetry is computed with respect to the current Work Plane.
    • Automatic
      The software tries to understand by itself where the symmetry plane is and then behaves accordingly.
    • None
      No symmetry is to be taken into account, and the movement will be extended to no symmetric vertices, edges and faces (even if any).
  • Edit Reference System
    Controls how the transformation handle is set when performing move face/edge/vertex operations. The possible choices are:
    • Local
      The handle is set using local reference system of the selected element.
    • Work Plane
      The handle is set according to the current Work Plane.
    • Fixed Z
      The handle is set like in Local mode but trying to keep the z-axis fixed.
  • Move Mode
    Controls the type of movement to be applied to the selected elements.
    • Roto-translation
      The usual transformation handle will be made available.
    • Scale
      The scaling handle will be made available instead.
  • Align Current Selection
    If you have a set of vertices currently selected, it will move all of them along the line built between the first and the last selected. This mode can also be activated by pressing the <F2> key.
  • Align to Surface
    If you have a set of vertices currently selected, they can be projected in such a way as to be tangent to a selected surface as much as possible. You can use also the <F3> key to activate this command. This alignment will work only on open edges.
  • Split in half
    When selected, when you split a face, the split is the middle of the face. When not selected, the split is performed at the pick point.
  • Split propagation
    When selected, the split is propagated to the neighboring faces. When not selected, the split is not propagated to neighboring faces. Not propagating usually creates worse shapes.
Notes on Selection and Related Movement
Please take into account that pressing the <SHIFT> or <CTRL> keys will enable you — as it ordinarily is in multiple selection commands of this type — to select/deselect multiple entities respectively in a row/adjacent or freely add/subtract one or more entities to the current selection.

 

Creating the Mesh Entity Associated with a Subdivision solid

Once you have created the controlling mesh from a SUBD solid, then you can select the mesh and output it using the ".obj" format (that now works also for generic meshes).
To create the Mesh entity associated with a Subdivision solid proceed as follows.

  1. Start the Extract Subdivision Mesh.
  2. Select the Subdivision Solid.
