System Options

Category: Output

Reference system

Set

Set:

Determines the reference system when saving points and other entities in ASCII format, by selecting from the drop-down list:

Worldsets the reference system to the global reference system (see "Work Plane, Sketching Plane and Datum Plane")
Work Planesets the reference system to the current Work Plane.

Printout

Printout:

Determines the reference system for the output of data, by selecting from the drop-down list.

World sets the reference system to the global reference system (see "Work Plane, Sketching Plane and Datum Plane")
Work Plane sets the reference system to the current Work Plane.

Decimal digits

Display

Display:

Determines the number of decimal places contained in output data files. Specifying a negative value automatically eliminates insignificant digits.

Trailing zeros

Trailing zeros

If checked, trailing zeros will be displayed at the end of the decimal digits.
If not selected, trailing zeros will not be displayed.

Entities with history

Entity with History

The two options in the Entities with history area displays the geometric data and event number of the feature in the Model Structure.

Show history index

Show history index Show history index: It enables you to hide/unhide the event number in the Model Structure.

Show geometric data

Show geometric data Show geometric data: It enables you to hide/unhide the geometric data of the feature in the Model Structure. For example, in case of the counter drilled hole it displays head diameter, head depth, and diameter of the hole (e.g D=10, Head=40*20).
