Relief is metal cutout from the edges where two bends meet to prevent interference. By selecting the Reliefs node, the Reliefs dialog box is displayed.

The Bend and Corner are two types of relieves available in the drop-down list of the Reliefs dialog box.

Bend This type of relief is generally created if the flange does not cover the whole edge length.

By selecting the Enable check box, the options in the Type drop-down list get activated.

The Rectangular and Oblong are two types of the Bend relief.

By selecting the Remove Remnant check box, you can enter the minimum size of the remnant into the adjacent box. When the size of the remnant is lower than this value, the remnant is removed.

In case of the Rectangular type the width and height of the remnant can be set in the Width and Height edit boxes, respectively.

In case of the Oblong type the diameter and height of the remnant can be set in the Diameter and Height edit boxes, respectively.

The Remove Remnant check box is cleared The Remove Remnant check box is selected
Corner The Corner Relief is provided to prevent interference between the corners of two adjacent bends.
It enables you to select the type of corner reliefs to be created when creating flanges. The following are the types of the Corner Relief.

Bend only Creates a corner relief only in the bend region.
Bend and flange Creates a relief in the first adjacent flange.
Bend and flange chain Creates a relief in all the adjacent flanges.
Rectangular Creates a relief that is rectangular in shape as per the specified Gap: value.
Circular Creates a relief that is circular in shape as per the specified Diameter: value.

Note
When a colored edit box is displayed, you can type in the name of the variable of the entity associated to it.

Reliefs are automatically created if the flange does not cover the whole edge length, according to the parameters defined in the Bend Reliefs page of the Sheet Metal category inside Entity Properties.
