Circles and circular arcs can be created using the following commands.
 
Center Circle based on a center point and other specified parameters:
  • radius or diameter
  • through point
  • two endpoints
  • radius, starting angle and subtended angle.
Three-Point Circle based on three through points.
    Radius Circle based on a radius or diameter and other specified parameters:
    • two points
    • center point.
    When working in the model (three-dimensional) environment, the resulting circle and arc entities are created on the Work Plane.

    Note
    The diameter mode:
    • Is available only for circles and not for arcs
    • Is not available if you use the command inside the profile environment.
