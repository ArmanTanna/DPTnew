Using Assembly Components in Smart Objects

If an assembly component is selected as a part of a Smart Object to be defined, there can be two different cases:

  • The component is an X-Ref: no dimensional parameters can be selected and the component will be kept unchanged
  • The component is not an X-Ref: dimensional parameters can be selected, but, at the end of the definition, the assembly component will be unlinked and a local copy created.
