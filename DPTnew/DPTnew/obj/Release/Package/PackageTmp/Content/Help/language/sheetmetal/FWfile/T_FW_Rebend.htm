Insert Sheet Metal Rebend

The Rebend command enables you to completely or partially bend solids previously flattened using the Unbend command.

After selecting the fixed face or edge, in the Type drop-down list you can select the type of rebending to perform:

Global to rebend all of the bends existing in the object.
Local to rebend only some of the bends of the flattened object. If you select this mode, you will have to select the bends to be rebend.

While performing the Rebend operation if any interference is observed in the model, a warning will be displayed in the Graphics Area with the interfering regions getting highlighted.
