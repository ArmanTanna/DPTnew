Customize

Customize Enables you to access the customization tools.
Customize GUI Profiles Enables you to save the current user interface configuration.
Customize Commands Enables you to add commands to toolbars by selecting them from a list.
Customize Toolbars Enables you to turn toolbars on and off, create, delete, and rename toolbars, and display either large or small toolbar buttons.
Customize Statusbar Enables you to add commonly used commands to the statusbar so as to have them easily available.
Customize Shortcuts Enables you to create and modify customize shortcuts.
