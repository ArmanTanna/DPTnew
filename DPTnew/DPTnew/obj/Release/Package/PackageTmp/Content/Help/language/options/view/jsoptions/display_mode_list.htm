To determine the appearance of objects when being rotated using the mouse choose one of the following options from the drop-down list:


Same as current

 While being rotated, the object will be visible as per the current Display setting. 
Wireframe While being rotated, the object is displayed as wireframe; its shaded representation being restored when you complete the rotation. This method results in the fastest rotation.


 
No boundaries While being rotated, the object is shaded, but the edges are not shown.


 
Flat shading While being rotated, the object is shaded, but without smoothing (tessellation is visible and all the polygons appear in a single color). This method results in faster rotation than the No boundaries setting; use it when you want to rotate large models but require a more realistic appearance than the Wireframe method.


 
HW Wireframe The object is displayed as wireframe using the OpenGL® libraries; its shaded representation being restored when you complete the rotation. This method results in the fastest rotation with some accelerated graphics cards.
