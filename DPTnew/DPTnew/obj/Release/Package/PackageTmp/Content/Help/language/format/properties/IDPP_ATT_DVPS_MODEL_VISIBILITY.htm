Model dimension filter on features

Model dimension filter on features

The controls in this area determine the dimensions that are automatically displayed in the drawing views or in the selected view, by specifying the dimensions driving the different features, the reference dimensions and the ones placed not parallel to the view.

Mating

Others

When selected, displays, in the drawing views or in the selected view, the dimensions driving the specific feature.
When not selected, the said dimension are not displayed.
