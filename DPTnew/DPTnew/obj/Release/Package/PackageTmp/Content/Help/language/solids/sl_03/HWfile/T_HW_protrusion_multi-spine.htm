Creating a Generic Protrusion by Using Constant Axis Type Motion Mode

The Generic Protrusion command enables you to create a protrusion on a solid by using the Constant axis type of Motion Mode.
Profile and drive curve Generic Protrusion

To obtain the above result, you need to open the Generic_Protursion.e3 file available in the samples folder of your installation folder and follow the steps given below.

  1. Start the Generic Protrusion command to display the selection list.

  2. Select the drive curve for the Spine selector and the profile for the Boundaries selector, as shown in the images below.



  3. Next, select the solid from the Graphics Area for Solid selector.

  4. Select Constant axis from the Motion Mode drop-down list of the selection list and Y from the Direction drop-down list.

  5. Click or to confirm your selections and create the solid, as shown in the image below.



    6. Click to discard your changes.

See "Using Spines" for details on how to use spine profiles.
