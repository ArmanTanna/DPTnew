Work Plane Enhancements and Improvements

Version 2008.1 introduces the following enhancements to the Work Plane editing and management.

  • Enhancement in the Work Plane visualization
  • Enhancement in the Work Plane editing techniques
  • The new On Entity command
  • Enhancement in the Move command
  • Enhancement in the Align to X command
  • Enhancement in the Align to Z command
  • Enhancement in the Set on Face command
  • Quick editing of Work Plane by the new Select command
  • Enhancements in the general method for axis, line and direction definition
  • Enhancements in the general method for plane definition

Enhancement in the Work Plane visualization

From this version onwards it is possible to visualize all the three coordinate planes on the Graphics Area, as shown in the image below.



The 3 planes option under Planes area of the Reference Systems - Appearance category in System Options controls the display of the three planes. The XY-plane is the Work Plane.

Enhancement in the Work Plane editing techniques

The Work Plane editing methodology has been changed in this version. Previously the Edit Work Plane dialog box was used to define the various parameters for Work Plane positioning. Now this dialog box is no longer available. However, with the enhancements made in this version, new commands have been introduced and the functionality of many existing commands has been augmented to perform all kinds of Work Plane editing, that too in a more intuitive manner.


The following Work Plane editing commands present in previous versions of the think3 application will no longer be available henceforth.

  • Edit
  • Point on Face
  • Set on Profile
  • On Curve
  • On Surface
  • On Point
  • Exchange Y with Z

Instead you can make use of the On Entity or the Select commands to get the required result.


Also note that the labels of the following Work Plane editing commands have been changed:


Old name New name
Set to World On Absolute Reference System (World)
Set on View On View
Set on Face On Face or Datum Plane
Set on Profile On Profile
Align to X Align X-Axis
Align to Z Align Z-Axis

The display and appearance of the Work Plane are now controlled by the options of the System Options - Reference Systems and System Options - Reference Systems - Appearance categories. Previously they were controlled by the options in the Document Properties - Work Plane category, which has been removed henceforth.


Note
The Work Plane is now always visible during the redefinition of a "feature" that used a Work Plane element for its creation. (For example, the Work Plane axis could have been used to define the direction during a command's execution.) This way you can get the proper feedback (highlight) on the used Work Plane element.

The new On Entity command

The On Entity command enables you to position the Work Plane over a selected entity. This command can be started in the following three ways:

  • Selecting EditWork PlaneOn Entity
  • Double clicking on the Work Plane
  • Selecting the On Entity option from the context menu displayed by right clicking on the Work Plane

After starting the On Entity command you need to select an entity that can be profile, line, curve, surface, face or datum plane. Depending on the entity selected, additional options in the selection list will be displayed to define the origin and orientation of the Work Plane.



On selecting the Associative Mode check box, the Work Plane becomes associative with the entity that was selected to position it (only if the selected entities are associative). In such a situation, if the selected entity is transformed, the Work Plane also gets repositioned accordingly.


Note
You can select a curve inside a 2D profile to define the Work Plane only if the 2D Profiles check box is selected in the Inside area of the Customize AutoSnap dialog box.

Enhancement in the Move command

On starting the Move, the origin of the Work Plane gets selected. Now you just need to select another point to move the origin to it. You can also drag the origin to a new location. Note that while the origin of the Work Plane moves to the new point, the orientation of the Work Plane does not change. Once the Work Plane is moved, you exit the command automatically.


The Move command can also be accessed from the context menu displayed by right clicking on the Work Plane.


Enhancement in the Align to X command

From this version the Align to X command is now termed as Align X-Axis. After starting the command you need to pick a direction to align the X-axis in the same direction. This can be done in two ways:


  • By selecting a straight entity that can be either a line or an edge.
  • By selecting a point. In this case the direction is specified by the virtual line joining the Work Plane origin and the projection of the selected point on the Work Plane.

Note that while the X-axis of the Work Plane is realigned with the specified direction, the position of the Work Plane does not change.


The Align X-Axis command can also be accessed from the context menu displayed by right clicking on the Work Plane.


Note
In case you select an entity such that the X-axis cannot be aligned with it, you will receive a warning. This will happen if you try to select a straight line that is aligned with the Z-axis.

Enhancement in the Align to Z command

From this version the Align to Z command is now termed as Align Z-Axis. After starting the command you need to pick a direction to align the Z-axis in the same direction. This can be done by selecting a straight entity that can be either a line or an edge.


After the Z-axis has been aligned, you can optionally align also the X-axis by selecting a line or a point, as described in the previous topic. If you don't want to align the X-axis, press the ESC key to quit the command.


The Align Z-Axis command can also be accessed from the context menu displayed by right clicking on the Work Plane.


Enhancement in the Set on Face command

From this version the Set on Face command is now termed as On Face or Datum Plane. As the new name suggests, this command now allows you to not only set the Work Plane on a selected face but also on a Datum Plane.


Quick editing of Work Plane by the new Select command

A new EditWork PlaneSelect command has been introduced in this version. This command enables you to quickly reorient and reposition the Work Plane. This command can also be started just by clicking on the Work Plane or by pressing Q on the keyboard.


On starting the Select command, translation and rotation markers are displayed over each axis, and can be effectively used to transform the Work Plane.


The Select command is very versatile and offers you numerous methods to transform the Work Plane. Refer to the command description in the Online Documentation for details.


Enhancements in the general method for axis, line and direction definition

There are several methods to define a line, an axis or a direction, as stated in the "How to Define an Axis or a Line" document of the online help. Usually this is done by selecting a straight entity. At times, the program supports the selection of the coordinate axis as the straight entity. In previous versions, the coordinate axis selection in such situations was possible only by the explicit use of the X, Y, Z options from the direction/axis definition drop-down list.

With the enhancement made in this version it is now possible to select the axes of the Work Plane as the straight entity by directly clicking on them in the coordinate axes symbol displayed over the Work Plane (see the command Show Work Plane). To do so, choose the Line option (instead of X, Y, Z) from the drop-down list and then click on one of the axes, as displayed in the image below.



In this case, the direction is defined as the line parallel to the selected axis and passing through the origin of the coordinate system. However, you can change the origin by resetting the Origin selector and specifying another point the line must pass through, though retaining its direction.

Nevertheless, the method used in former versions is still available. A new Set direction/axes using X, Y, Z check box has been added to the Input category of the System Options so as to enable you to choose the method you feel more comfortable with.


  • When the Set direction/axes using X, Y, Z check box is selected, the direction/axis definition drop-down list will still contain the X, Y, Z or (Parallel to X and through point, Parallel to Y and through point, Parallel to Z and through point) options, with the same behavior as in former versions. Also, the interactive modality (direct pick of coordinate axes) will be possible.

  • When the Set direction/axes using X, Y, Z check box is NOT selected, the direction/axis definition can be done only by direct selection (clicking the coordinate axis after selecting the Line option). In this case the drop-down list will not include the X, Y, Z options. Yet, once an axis has been selected, its corresponding letter (X), (Y), (Z) will be displayed adjacent to Line.

(See also another example for the definition of a direction: "New Drafter command")


Enhancements in the general method for plane definition

To execute some commands you need to define a plane. From this version onwards, apart from selecting a planar face or datum plane, it is possible to select one of the three coordinate planes for the Plane selector.
