ModifyAnnotationRedefine

The Redefine command enables you to modify an annotation. The command can also be started selecting Modify Entity from the context menu displayed by right-clicking in the Annotation Tree Panel.

You can modify each parameter used to define the annotation (see New Annotation), using exactly the same procedure.

Refer to Creating a New Annotation for a step-by-step explanation.
