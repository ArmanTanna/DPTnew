Curve/Surface editing mode

Base entities:

When modifying a non-associative curve or surface, the options of this drop-down list determine how the program shall apply the change:

Keep the program creates a copy of the original entity and applies the changes to the copy. The original entity remains visible.
Hide the program creates a copy of the original entity and applies the changes to the copy. The original entity is hidden.
Modify the program applies the changes to the original entity.


1 —  Modify
2 —  Keep

Please note that this option does not apply to associative entities, which are always actually modified when they undergo a change, whatever the option value may be.

Component

Solid_color

Options under this area enables you to define component name and color.

Name:

Name:

Enables you to define a specific and separate name for the component/shared group.
If this box is not filled in, the component/shared group name will be the same as the one of the file it is contained in.

Please note that the component/shared group name thus defined is taken into proper account in any operation the component/shared group is involved in (save and load, insert, etc.).

The component/shared group name can be displayed in the Model Structure/ Drawing Structure Drawing Structure — in brackets on the right of the model/drawing name — by selecting the new shared group name in structure Show component/shared group name in structure box in the Assembly/Shared GroupShared Group category of the System Options.

Show closed components with component color

Solid_color
  • When selected, the whole component will be displayed in the same component color, overriding the solids colors.
  • When cleared, the individual solids of the component will be displayed in their own solid color.

Profile

Profile

The option in this area enables you to define the way attributes are applied to profile entities.

Show profiles with profile properties

Show profiles with profile properties
  • When this check box is cleared, it is possible to change the attributes (color, line type and line width) of entities (curves) inside a profile. After exiting from the profile mode, the entities of the profile retain the applied curve attributes.
  • When this check box is selected, attributes of the profile entities cannot be modified inside the profile mode. After exiting from the profile mode, the entities of the profile possess with profile attributes.
