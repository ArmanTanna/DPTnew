File Save As

The Save As command saves the current document under a new name and/or in a different format and/or in a new location.

The document can be saved in several different formats.

In addition, according to the type of document that you save, you may define some specific Options.

The think3 application, saves by default, the Drawing (*.e2) and Model (*.e3)files in the compressed format.
