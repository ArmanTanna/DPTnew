ModifyComponentDerive and Replace


 

The Derive and Replace command enables you to derive an X-Reference component and also replace it with the same.

This command is available in the context menu on right-clicking the component from the Model Structure. It essentially combines the capabilities of both the Model Derived from Current and Replace commands.

The command is particularly useful, when in the design change/update process, you want the changes made to a particular model not to affect the base model. It is also beneficial when you are using multiple instances of a base model with minor modifications in multiple assemblies and need to retain the base model as a template so that the changes do not affect it.
