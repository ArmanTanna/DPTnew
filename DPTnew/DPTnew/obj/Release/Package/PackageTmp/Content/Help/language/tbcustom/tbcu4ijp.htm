Customizing a Button on a Default Toolbar

You can customize a button on a default toolbar so that from being a “default” button, it becomes a “customized” button. Customizing a button means changing the graphic appearance and programming of the buttons on a program's default toolbar. To restore a default toolbar to its original condition after you have customized it, use the Reset Toolbar command.

  1. Select ToolsCustomize from the menu bar to open the Customize dialog box.

  2. Move the mouse pointer onto the button you want to customize and right-click.

  3. Choose Customize from the toolbar popup.



    Now the button is customizable.
Note
You can only customize default buttons. If you have selected a “customized” button, the Customize option is disabled.
