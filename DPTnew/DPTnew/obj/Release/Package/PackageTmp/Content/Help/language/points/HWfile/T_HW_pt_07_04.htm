Project points along a line given the endpoints

1 Open the file pt_07_02.e3.



In this example, you are explained about how to project the points along a line given the endpoints.
2 To do this, start the Project command and you are asked to select the points you want to project.
In this case, you can select the endpoints of the guide line instead of the line itself. This technique may be useful to project the points along an imaginary line, without having to actually create the line (in this example, the line is shown to illustrate the technique).
3 Now select the points you want to project.
4 Then indicate the first endpoint of the guide line.
5 Later indicate the second endpoint.
6 The points are projected.
