ProEngineer

Category: Load-Advanced

Set tolerance

When not selected, the tolerance value used for the conversion is the one that is currently specified for the model. The model tolerance value is defined in the Tolerance edit box under the Precision area of the General - Advanced category of Document Properties. When selected, the tolerance value can be specifically defined in the corresponding edit box.

Convert arcs to NURBS

When selected, these entities are converted into NURBS and their conversion is controlled by the tolerance value.

Convert curves to NURBS

When selected, these entities are converted into NURBS and their conversion is controlled by the tolerance value.

Convert surfaces to NURBS

When selected, these entities are converted into NURBS and their conversion is controlled by the tolerance value.

Convert solids to NURBS

When selected, these entities are converted into NURBS and their conversion is controlled by the tolerance value.

Convert to multi-units

When checked, the model units will be converted to the current think3 units. Change this option only when the attempt to load the document fails.

(Temporary setting. Only the next loaded file will be affected)

If you change this option, the new setting will last only for one load or save operation, then it will be reset to its default value.

Insert all component/assembly paths

Enter, separated with semicolons, the folders where the system is to look for external components other than the folder where the file to be converted is.
