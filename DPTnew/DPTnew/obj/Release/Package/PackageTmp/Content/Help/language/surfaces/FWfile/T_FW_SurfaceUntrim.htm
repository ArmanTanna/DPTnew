The Untrim Surface command enables you to reconstruct the original surfaces from surface with parameter limitations, however those were created (by trimming an existing surface, creating bounded planes, etc.).

Depending on the method selected in the Mode drop-down list, reconstruction can be done in one of two different methods.

  • Global
    After selecting the surfaces (Surfaces), all of the trimmed boundaries will be reconstructed and all of the existing islands will be removed.
     
  • Local
    Performing the selection of the boundaries (Boundaries), you can:
    • Select single islands to be removed. Only the selected islands will be removed.
    • Select a trimmed boundary. All of the trimmed boundaries will be reconstructed. If there are islands, they will not be removed, unless you explicitly select them.

You can specify what to do with the original surfaces. In the Base Entities drop-down list, select one of the following:

  • Modify — to apply the changes to the original surfaces.
  • Hide — to hide the original surfaces, though keeping them in the model.
  • Keep — to keep the original surfaces, leaving them visible.
