InsertDraftingLineBisector


The Bisector command creates the bisector line of two selected lines. The two lines can have a real or a virtual intersection point, or be parallel:

If the two lines have a real intersection point, the bisector line will start from that point.
If the two lines have a virtual intersection point, the bisector line will start from that point.
If the two lines are parallel, the line will be parallel to them and at half the distance from each of the two.

In the Type drop-down list you can choose the line extension as Length, Semilimited or Unlimited.



If you choose the Length mode, type the length value into the Length mini-dialog box and press TAB.

Associativity
The entities inserted in the drawing view using this command are associative to the reference model geometry, which means any modifications done on the model geometry will be propagated to the bisector lines in the drawing view during regeneration.

Due to associativity:

  • Lock symbol () appears on the entities (when the cursor control is placed on them) based on whether associativity can be established with the model entities or not.
  • The inserted entities are placed inside the related group in the Model Structure.
  • The added entity will be treated like an added geometry.
