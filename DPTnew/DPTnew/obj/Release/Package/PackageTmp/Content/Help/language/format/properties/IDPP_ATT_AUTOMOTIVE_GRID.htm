Entity Properties

Category: Reference Grid

Label

Label

In this area you are enabled to define the properties affecting the Label of the reference grid in the drawing views.

Placement

Placement

The controls in this area enable to specify the placement of the labels in the reference grid.

Horizontal

Horizontal:

Enables you to select the vertical placement for the reference grid labels:

None inhibits the visualization of labels along vertical lines
Left places the labels on the left of the grid
Right places the labels on the right of the grid
Left & right places the labels both on the left and on the right of the grid

Vertical

Vertical:

Enables you to select the horizontal placement for the reference grid labels:

None inhibits the visualization of labels along horizontal lines
Above places the labels above the grid
Below places the labels below the grid
Above & below places the labels both above and below the grid

Display units

Display units:

Enables you to select the unit to be used to display the coordinate values of the labels.

0 The value displayed in the label will be the actual one. That is, if the value is 1200, 1200 is the number displayed in the label.
Units>0 The actual value will be scaled by the Unit and the label will display its integer part. That is, if the Unit is 100, when the value is 1200, 12 is the number displayed in the label, but, if the value is 120, the label will display 1.

Options

Options

In this area you are enabled to adjust the construction properties controlling the way reference grid is displayed.

Step

Step:

Enables you to change the distance between two subsequent grid lines. This value is defined at the grid creation time.

Offset

Offset:

Enables you to specify how much the grid is to be extended beyond the specified boundaries. This value is defined at the grid creation time.

Major line subdivisions

Major line subdivisions:

Enables you to specify how many parts the major block can be divided in both the directions using 'n' number of minor lines.
For example, selecting 5 for the Major line subdivisions: will divide a major block into five parts in both the directions by creating four minor lines.

Show sectioned lines

Show sectioned lines

Enables you to control the sectioned lines (for sectioned views only) display.

  • When the check box is selected, sectioned lines are displayed.
  • When not selected, sectioned lines are not displayed.
