The Align To Target command enables you to re-align constrained views after you have moved the target point in the free view using the Set Target command, which can be managed using the Target area of the Advanced category under Navigation of the System Options dialog box.

Start the Align To Target command by right-clicking on one of the constrained views and selecting Align To Target in the context menu.

The command works as follows:

  • It re-evaluates the view orientation from the Work Plane position and the view type
  • It copies the isometric view zoom factor to each constrained view
  • It resets the pan factor so that the target point is displayed in the middle of each viewport.
