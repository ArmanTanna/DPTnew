Animation Playback Controls

Following buttons control the playback of animation:



 Jump to Start Position : Displays the first frame of the animation


One Step Backward : Displays the previous animation frame


Play : Starts the playback of the animation


Create AVI : Creates an AVI of the animation (displayed only when the Create movie check box is selected in AVI settings)


Stop : Stops the playback of the animation (displayed only when the animation is running)


Stop AVI creation : Stops the creation of AVI (displayed only when an AVI is being created)


One Step Forward : Displays the next animation frame


Jump to End Position : Displays the last frame of the animation


Jump to Following Step : Displays the frame whose number is specified in the Jump Step edit box


Minimize
(Toggle button)		 : Displays a compressed version of the Animation dialog box containing only the playback buttons. This increases the visible portion of the Graphics Area.
