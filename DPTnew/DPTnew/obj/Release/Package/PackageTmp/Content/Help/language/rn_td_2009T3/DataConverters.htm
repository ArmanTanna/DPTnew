New Data Exchange Functionalities and Improvements

Version 2009.3 introduces the following Data Exchange enhancements.

  • Save *.e2 file as PDF

Save *.e2 file as PDF

In this version, you will be able to save *.e2 files in the PDF (Portable Document Format). To implement this enhancement, a new Adobe PDF file[pdf] option has been introduced in the Save as Type drop-down list of the file selection box.

To support this conversion process, the ThinkDesign Printer gets automatically installed during the installation process.
