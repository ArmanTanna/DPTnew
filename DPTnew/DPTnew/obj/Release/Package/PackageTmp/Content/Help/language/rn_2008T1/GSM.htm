New Global Modeling Functionalities and Improvements

Version 2008.1 introduces the following enhancements to the Global Modeling commands.

  • New Global Shape Modeling command names
  • New 3D Bounding-Box command
  • Enhancements in the Advanced GSM command

New Global Shape Modeling command names

In order to homogenize the User Interface and to simplify the user access to the global modeling commands, their names and paths have been improved as follows.

New 3D Bounding-Box command

The 3D Bounding-Box command (ModifyGlobal Shape Modeling3D Bounding-Box) enables you to define a controlled modification using a set of vertices in the 3D space.



See the online help for a complete description of the command.

Enhancements in the Advanced GSM command

Significant improvements have been made in the Advanced GSM command.

  • Starting from this version, some appropriate additional warnings will be displayed when the selected constraints do not fulfill the target tolerances.
  • Should any incompatibility be generated when you select Highlights under Passing Through, proper messages will also be displayed.
