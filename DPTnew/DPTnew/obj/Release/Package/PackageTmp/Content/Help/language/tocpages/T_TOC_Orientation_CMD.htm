Orientation

Named Views Provides commands to reorient the display.
Top View Reorients the display so that the model is viewed from the top (looking down along the Z-axis) perpendicular to the XY-plane of the World Reference System.
Bottom View Reorients the display so that the model is viewed from the bottom (looking up along the Z-axis) perpendicular to the XY-plane of the World Reference System
Left View Reorients the display so that the model is viewed from the left (along the X-axis) perpendicular to the YZ-plane of the World Reference System
Right View Reorients the display so that the model is viewed from the right (along the X-axis) perpendicular to the YZ-plane of the World Reference System
Front View Reorients the display so that the model is viewed from the front (along the Y-axis) perpendicular to the XZ-plane of the World Reference System
Back View Reorients the display so that the model is viewed from the back (along the Y-axis) perpendicular to the XZ-plane of the World Reference System
Right Back View Reorients the display so that the model is viewed as an isometric view from the right-back (along the line from coordinate 1,1,1 to the origin) relative to the World Reference System
Left Back View Reorients the display so that the model is viewed as an isometric view from the left-back (along the line from coordinate -1,1,1 to the origin) relative to the World Reference System
Left Front View Reorients the display so that the model is viewed as an isometric view from the left-front (along the line from coordinate -1,-1,1 to the origin) relative to the World Reference System
Right Front View Reorients the display so that the model is viewed as an isometric view from the right-front (along the line from coordinate 1,-1,1 to the origin) relative to the World Reference System
