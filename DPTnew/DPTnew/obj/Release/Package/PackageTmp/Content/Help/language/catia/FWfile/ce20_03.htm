The TD Catia v4 interface Options command enables you to set some aspects of the data conversion with CATIA®.

You can find this command in the TD Catia v4 interface toolbar. The command is active both in the 3D and in the 2D environments of this software. To activate the toolbar, please refer to the TD Catia v4 interface procedure.

When you run the command, the TD Catia v4 interface Options dialog box is displayed that contains the parameters which control the conversion of entities to and from CATIA® organized in two tabs:

  • Load —  Defines the options relevant for conversion to CATIA®.
  • Save —  Defines the options relevantfor conversion from to CATIA®.

Once you have set the options you like, click:

  • OK to confirm the options you entered
  • Cancel to terminate the operation discarding the changes
  • About to get information about TD Catia v4 interface
