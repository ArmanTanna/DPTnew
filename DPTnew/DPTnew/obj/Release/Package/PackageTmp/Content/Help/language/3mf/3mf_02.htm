Saving a 3MF file

To convert from the think3 format to 3MF — , do the following:

  1. Select Save or Save As
  2. In the Save file dialog box, under Save as type, select 3MF Format (*.3mf)
  3. In the File name box, type the name of the file you want to save
  4. Click Options to open the Settings tab of the 3MF Options dialog box and set the parameters controlling the conversion process as you like
  5. Click the Save button to save the file.
