Log file (importing from CATIA®)

When you load a CATIA file, a log file of the conversion is created:

  • in the same directory of the file you opened
  • with the same name of the file you opened and the ".log" extension.

The log file structure

The log file for conversion from CATIA® lists the conversion parameters and information about the entities converted in the process.

In the log file you find:

  • Information about the CATIA model creation such as date, time and CATIA release.
  • The tolerance within which curves are considered identical.
  • The CATIA entity inventory (type and count).
  • The tolerance at which each solid is manifold.
  • The number of 2D entities processed (if any).
  • The number of not processed entities, if any, both visible and hidden.
  • The list and count of entities in the output file.
  • Warnings, if any.
