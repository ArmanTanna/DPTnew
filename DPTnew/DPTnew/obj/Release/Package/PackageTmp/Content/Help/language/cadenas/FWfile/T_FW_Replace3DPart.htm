PARTsolutionsReplace Object


 

The Replace Object command enables you to replace existing PARTsolutions solids with a new solid imported from the CADENAS PARTdataManager™.

When you start the command, the program prompts you to select the solids to replace. Select the solids in our model, then click Replace Part(s) With:.
Wait for CADENAS PARTdataManager™ to start and select the new part from the catalog. When you click ExportExport, all the solids you have selected will be replaced by the new CADENAS part.
Click Replace Part(s) With: again to replace the part just loaded from the CADENAS catalog with a new part.

While choosing the solids to replace, the program prevents you from selecting parts not created with PARTsolutions plug-in.

After selecting the new part from the catalog, click:
to apply the new part and quit the command.
to return to the original part and quit the command.
to apply the new part and continue to select new components in the model to replace with new parts.
