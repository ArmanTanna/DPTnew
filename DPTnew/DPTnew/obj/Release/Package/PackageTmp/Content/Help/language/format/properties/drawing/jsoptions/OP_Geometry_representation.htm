Determines the way in which the next view is displayed, by selecting one of the following from the drop-down list:

Wireframe displays the view in wireframe mode
Hidden lines displays the view in wireframe mode with hidden lines
Hidden Lines (ver. < 2008.1-SP1) when editing old drawing views created using obsolete options that have been removed (Hidden line+axis and Add axes to holes and rotational slots), this option displays the view in wireframe mode with hidden lines and makes the mentioned removed options available for your use.
