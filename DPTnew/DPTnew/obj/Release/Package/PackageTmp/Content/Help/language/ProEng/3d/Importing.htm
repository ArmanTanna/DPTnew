Converting from Pro/ENGINEER to think3

The think3 application can open Pro/ENGINEER models.

General concepts

  • Entities are converted according to the Pro/ENGINEER to think3 conversion table.
  • Conversion is performed according to the current values of the parameters of the Pro/ENGINEER Conversion Options dialog box opened by selecting Options.

Tips on converting models from Pro/ENGINEER to think3

Try to open your Pro/ENGINEER file and see what you get. After examining the results, set to the proper values those parameters which mainly affect conversion accordingly to the results you've obtained and open the file again.

Reading the error file of the conversion process is the most direct and simplest way to check conversion errors. In such file, errors are logged, which will help identify where the problem is.
