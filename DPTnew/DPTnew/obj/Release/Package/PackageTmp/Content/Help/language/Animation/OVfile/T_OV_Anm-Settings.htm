Animation Settings

The settings for the animation are done using the Animation settings button ( ) in the Animation dialog box. On clicking this button, the setting parameters are displayed as shown below.



Duration: : Total duration of the animation (avi) in seconds
Steps per unit: : Number of steps to be generated per second
Initial status of first variation : Select this check box to initialize all the variations to their starting position on selecting the Jump to Start Position button. Otherwise, only the variation(s) from which the animation begins get initialized.

After making all the settings, click the Evolutions ( ) button to return to the evolution window.
