You can deassign a variable from an entity using the Deassign command, which can be accessed from the Spreadsheet dialog box.

The Spreadsheet dialog box can be viewed using the Spreadsheet command.

You can deassign the variable from the following:

Dimension/Feature/Component Enables you to deassign the variable from a drive dimension or from a feature/datum entity.
Thread Diameter Enables you to deassign the variable from a thread diameter.
Thread Pitch Enables you to deassign the variable from a thread pitch.
Try to ...
  • Assign/deassign a variable to/from a drive dimension
  • Assign/deassign a variable to/from a feature or a datum entity
  • Assign/deassign a variable to/from an X-reference component
