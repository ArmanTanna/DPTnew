Compatibility with AutoCAD 2010 enhancements

Now loading and saving of dxf/dwg file format version 2010 is supported in this application. To do so, you need to select the 2010-2011 option from the Version number drop-down list of the Save Advanced category of the DXF/DWG Option dialog box.
