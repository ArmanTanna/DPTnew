The Edit command enables you to modify the index and catalog properties.

You can find this command in the context menu that displays when right-clicking on a catalog or index in the catalog tree frame of the Catalog Manager and in the toolbar of the command main dialog.

If you are editing the properties of an index, a wizard will guide you through the process of modifying the name, description and preview image of the index. As this wizard is also used by the New Index command; please refer to this article for more information.

If, instead, you are editing the properties of a catalog, the same wizard used by the New Catalog command will guide you in modifying the name, descriptions, preview image and fields of the catalog.
