ModifyDrawing ViewEdge Visibility

The Edge Visibility command enables you to change the appearance of any edge within a drawing. The selected edge can be made dashed, visible, thin, or restored to its original condition.

The command allows multi-edge selection at a time using any of the selection commands: Lasso, All, Window, Chain.
