Drawing a Cutting Plane Line as a Group

To draw a cutting plane line as a group, start the Draw Section as Group command and proceed as follows:

1 In the Name box, define the name of the cutting plane line.
2 From the Orientation drop-down list select the orientation mode:
Free To orient the section lines freely in any direction
Horizontal vertical To orient the section lines only in horizontal and vertical directions
3 You can also set the following parameters, if required:
  • Select the Invert check box to reverse the direction of the cutting plane arrows.
  • Select the Properties node, which opens Entity Properties dialog box to set various style parameters for the section lines created.
4 Select the points to insert the section line.
5 The starting point of the cutting line is marked and you are prompted to specify its endpoint. Click to specify the endpoint of the cutting plane line. The program continues to prompt for the endpoint of the line, so that you can create a cutting plane line with more than one segment.
6 Click or to confirm your selections and create the cutting plane line.



Click to discard your selections.


Note

During insertion mode, it possible to edit the points and segments using the contextual options.
