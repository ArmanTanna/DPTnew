Enables you to determine the orientation to be applied to the text characters:

Upside down Enables you to mirror the text characters about the horizontal axis. This option applies both to True Type (Standard) and vectorial (Alternative) fonts.
Backward Enables you to display the text characters in the backward direction. This option applies both to True Type (Standard) and vectorial (Alternative) fonts.
Vertical Enables you to display the text characters vertically. This option applies only to True Type (Standard) fonts.
