ModifyComponentAutomatic Weight CalculationEnable/Disable

This command allows you to Enable or Disable the Automatic Weight Calculation for a part.

The command selection list features the following options:

  • Select whether to enable/disable the automatic weight calculation for Top components or for Leaf components
  • Select the very components you want to enable/disable the automatic weight calculation for.

You will be asked whether to apply the automatic weight calculation also to child components or not:



The Automatic Weight Calculation can also be enabled or disabled from the Edit Part Properties tab, which is available in the File Properties dialog box which is displayed by the Properties command.

.

If you have enabled the automatic calculation, the weight value will be displayed in the mentioned tab. A message will be issued to warn you the calculation time depends on the complexity of the assembly.
