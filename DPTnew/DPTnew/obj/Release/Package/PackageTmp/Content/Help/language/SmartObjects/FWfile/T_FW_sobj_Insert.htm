The Insert command enables you to add a Smart Object to your model.
The Smart Object can be taken from a document residing in your local file system.

When inserting the Smart Object, all mandatory geometric references have to be selected.

The exposed dimensional parameters can be modified only if no configuration tables are defined; otherwise, it is possible to select a different configuration. When positioning the Smart Object, an anchor point is usually to be selected: an "Anchor Handle" is displayed in this case.
