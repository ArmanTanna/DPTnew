Enables you to specify the layers of the drawing/model to be printed.
Also, you can specify the layers to print in the corresponding edit box.

  • All — prints the entities on all layers, including hidden layers.
  • Current — prints only the entities on currently visible layers.
  • Selected — prints the entities on layers that you select in the dialog box that is displayed when you click the button.



    The dialog box displays the names of the layers. Select the layers you want to print and click OK.
