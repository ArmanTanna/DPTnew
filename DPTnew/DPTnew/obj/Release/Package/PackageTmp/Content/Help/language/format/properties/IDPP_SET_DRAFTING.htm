Category: Centerlines

Centerlines

Centerlines

The controls in this area enable you to define the length and the color to be used in the drawing of arc centerlines.

Extension

Extension:

Enables you to select the method you want to use to determine the arc centerline extension:

Length the value in the adjacent box determines the arc centerline extension as the total length of the centerline axis, based on the current unit of measurement.
Half axis % the value in the adjacent box determines the arc centerline extension as a percentage half the length of the centerline axis.

Length

Length:

Choose this method to specify the arc centerline extension as the total length of the centerline axis, based on the current unit of measurement. Type the value in the corresponding box.

Half axis %

Half axis %:

Choose this method to specify the arc centerline extension as a percentage of half the length of the centerline axis. Type the value in the corresponding box.

Axis will be extended of a percentage of the half length of the entire axis itself.

Color

Color:

Determines the color used to draw the centerline axes of arcs and circles.
