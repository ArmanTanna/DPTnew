Setting tangency and modifying both curves: weights

This example shows the effect of changing the weights of the two curves while leaving unchanged their propagation factors and the scale factor.
 
  1. Open file "mc_02_04.e3e2".


     
  2. Start the Curve Continuity command.
     
  3. In the Degree drop-down list select Tangency.
     
  4. In the Method drop-down list select Adjust both.
     
  5. Click on the two curves at points close to the ones shown in the illustration below:


     
  6. Answer Yes each time to enable conversion into NURBS.
     
  7. In the corresponding boxes, type the following values:
    • Scale: 1
    • 1st Weight: 1
    • 2nd Weight 2: 1
    • 1st Propagation: 0
    • 2nd Propagation: 0.5


       
  8. Click to obtain the following result:


     
  9. Repeat the exercise using 0.2 as the weight of the first curve and leaving all the other values unchanged:
    • Scale: 1
    • 1st Weight: 0.2
    • 2nd Weight: 1
    • 1st Propagation: 0
    • 2nd Propagation: 0.5
