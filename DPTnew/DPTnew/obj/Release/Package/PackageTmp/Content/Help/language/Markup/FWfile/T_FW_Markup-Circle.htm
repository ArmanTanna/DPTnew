InsertMarkupCenter Circle

The Center Circle Markup command enables you to add a circle markup to a drawing file opened inside in this software. Note that by using this command you can add both circles and arcs. The method of applying the Center Circle Markup command is the same as applying the Circle given Center command. Various options are provided in the command's selection list to assist in creating the markup as a circle or an arc by defining its radius or diameter.

Any circle markup that you add will appear on the Visual Bookmarks tab, as an entry on a collapsible list. In order to delete a circle markup, delete its entry from this list by right-clicking on it and selecting Delete from the context menu.

See the topic An introduction to Markups for an overview on markups.
