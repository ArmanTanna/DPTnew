Create chain dimensions using two special points

1  When you start the Smart Dimension command, the program prompts you to select an entity. Click to select an entity or edge at a Snap point.

 

2 Select the Chain option in the Type drop-down list to create a string of linear dimensions drawn end-to-end. See also "Serial/Chain Dimensioning" for further details.

 

3 You are prompted to select a special point. Click to select another Snap point.



Notice that the program immediately displays the dimension, which moves as you move the cursor. The alignment of the dimension is determined by the current Orientation.

 

4 Select the desired alignment from the Orientation drop-down list:
 
Distance
measures distances parallel to the line through the two endpoints
Horizontal
measures distances parallel to the X-axis
Vertical
measures distances parallel to the Y-axis
Projected
measures distances parallel to a specified Angle
Automatic
measures distances parallel to the line through the two endpoints or parallel to either the X-axis or the Y-axis

 

5 You are prompted enter the origin. Simply click to specify the location of the dimension line.



Prior to specifying the location of the dimension line, you can select Format and Properties to change the appearance of the dimension.

 

6 After inserting the first dimension in the chain, the program prompts again to select a special point. To create another dimension measured from the end point of the previous dimension, click to select another Snap point.



The new dimension is immediately created.



The prompt repeats so that you can continue to insert additional dimensions along the chain.

Format
Using the Format option you can create the dimensions in any desired format.


Properties
By clicking on the Properties Option, the Entity Properties window shows up, that enables you to control appearance of the Dimension and Text entities.
