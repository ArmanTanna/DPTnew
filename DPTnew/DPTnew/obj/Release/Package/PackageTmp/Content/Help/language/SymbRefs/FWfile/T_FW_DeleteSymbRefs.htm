The Delete command enables you to delete an existing Symbolic Reference.

The command is available from the context menu displayed when right-clicking the selected Symbolic Reference in the Model Structure.

You also get the command from within the View Symbolic References command, when you right-click on a Symbolic Reference mini-dialog box.

Another way to delete Symbolic References is simply selecting one or more in the Model Structure and entering the "Cancel" key: the program displays a confirmation request; once you've confirmed, the Symbolic Reference(s) is/are deleted.
