Cutting Plane Lines

Define on Drawing Enables you to create a cutting plane line and position it in a two-dimensional drawing view.
Define on Model Enables you to define a cutting plane in the referenced model which can then be used for creating sections.
Draw as Group Enables you to draw a cutting plane line as a 2D-symbol of a section, without actually deriving any sectioned views from it.
Extend Section Enables you to change the length of a cutting plane line.
Invert Section Reverses the direction of the cutting plane line from which you derive a section.
Redefine Enables you to modify the defined cutting plane line from which you derive a section.
Multiple Parallel Cross-Sections Inserts multiple parallel cross-sections in a single step.
Place on Drawing Inserts a cutting plane line based on a previously defined cutting plane into a selected view.
