Current

Current:

Enables you to set the current style by selecting it in the list of all the available ones.

Dimension line

Dimension line

The controls in this area enable you to define additional parameters for the dimension line, such as the stem.

Stem

Flip orientation

Enables you to control the usage of the Stem in the Dimension line.

  • When the Enable check box is selected, the Stem is displayed. In this case you can specify the Length of the stem and even invert its orientation by selecting Flip orientation.
  • When the Enable check box is not selected, no stem is displayed.

Extensions

Extensions

The controls in this area determine the creation of Extensions when adding dimensions on virtual intersection points, i.e.: intersection points on extensions of lines.

Draw

Draw

When the Draw check box is selected, extension lines are drawn when adding dimensions on virtual intersection points.
When the checkbox is not selected, extension lines are not drawn.

Line

Line

Enables you to select the type of Line for drawing the extension lines. If the Draw check box is not selected, this control is grayed out.

Slant

Slant:

Enables you to determine the angle at which extension lines are drawn when you insert a Distance type of dimension.

45° Chamfer

The controls in this area enable you to define the format for chamfer dimension leader line.

Aspect:

Enables you to specify the leader line representation of chamfer (45º) dimensions:

Perpendicular leader line Leader line represented perpendicular to the edge
Parallel leader line Leader line represented parallel to the edge
