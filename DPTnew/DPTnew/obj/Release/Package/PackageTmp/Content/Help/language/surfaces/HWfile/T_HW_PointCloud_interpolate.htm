Creating a Point Cloud surface by interpolating a grid of points

In the following step-by-step example you will create a surface by interpolating a grid of points.
Please note that you can accomplish this task by loading sample file "sr0608.e3".



1 Start the Point Cloud command.
 
2 In the Surface Type drop-down list select Grid points interpolation.
 
3 Select the points of the first row by clicking or by using a selection operator, if possible. End selection by right-clicking and choosing Continue in the context menu.


 
4 Select the points of each of the remaining rows and end selection by right-clicking and choosing Continue in the context menu.



To avoid twisting, use the same selection order for all the rows. Make also sure you specify the same number of points for each row.
 
5 Click the Preview button ( ). The preview of the resulting surface is displayed:


 
6 Click or to confirm your selections and create the surface.



Click to discard your changes.
