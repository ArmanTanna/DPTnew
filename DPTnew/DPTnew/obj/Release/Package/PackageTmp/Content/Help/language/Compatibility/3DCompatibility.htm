Conditions for models created with version 2007.1

Sheet Metal commands

Models created and saved in 2007.1 using one of the following features cannot be loaded with previous version:

  • Models created with the Extruded Flange command using the 1st distance and 2nd distance and the At each edge options.
  • Additional settings changed for the Wizard command in the new Sheet Metal category of the Document Properties.
  • Any model created using the Sheet Metal Wizard.

Curve and Surface commands

Models created and saved in 2007.1 using one of the following features cannot be loaded with previous version:

  • Curve Target Curvature command in associative mode (that is Associative Mode check box selected).
  • Global Sweep command in associative mode (that is Associative Mode check box selected).
  • Lofted Surface command in associative mode (that is Associative Mode check box selected).
  • Blending Shapes command with the Side Curves option in associative mode (that is Associative Mode check box selected).
  • All Curve and Surface commands in associative mode (that is Associative Mode check box selected), if an adjacent edge is selected as a boundary of a surface.
  • Helix curves created using the Helix Curve command in associative mode (that is, Associative Mode selected).
  • Helix skins created using the Helix Surface command in associative mode (that is, Associative Mode check box selected)
  • Skins created with the Primitives commands in associative mode (that is, Associative Mode check box selected).
  • Any entity made associative using the Enable Associativity command.

Solid commands

Models created and saved in 2007.1 using one of the following features cannot be loaded with previous version:

  • Holes or shafts created on cylindrical faces using the Hole or Shaft commands respectively.
  • Fillets created using the Setback option of the Edge Fillet command.
  • Fillets created with the Face-to-Face command.
  • Models containing features created with Boolean operations with the Keep Original Solids check box selected.
  • Models saved after using the Extend Faces/Close Solid commands ( Interactive Modeling)
  • Models containing holes (simple or shaped) or shafts with a Direction.

Profile commands

Models created and saved in 2007.1 using one of the following features cannot be loaded with previous version:

  • Models containing profiles for which the Equal Curvature constraint has been used.

Drafting and Dimension commands

Models created and saved in 2007.1 using one of the following features cannot be loaded with previous version:

  • Models containing 2007.1 new Welds and new serial and parallel dimension families.

Assembly commands

  • Assemblies created with the Make Flexible option will be loaded, and the flexibility can be exploited, though not disabled/enabled.

Transformation commands

Models created and saved in 2007.1 containing entities such as the ones in the following list cannot be loaded with previous version:

  • Parametric entities created using the Move/Copy command with both the Copies and the Linked copies check boxes selected.
  • Parametric entities modified with the Scale command or created using the Scale command with both the Copies and the Linked copies check boxes selected.
