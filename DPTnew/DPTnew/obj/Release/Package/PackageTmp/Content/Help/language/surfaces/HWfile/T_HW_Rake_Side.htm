Creating a Sweep Surface by Selecting the Side of the Motion

In this task you will use the Rake Sweep command to create a surface by sweeping one contour along two drive contours.



In this case there two possible sides for the motion. You can accomplish this task by loading file "RakeSweep.e3"

1 Start the Rake Sweep command.
 
2 Select the first drive curves and end selection by right-clicking and selecting Continue.


 
3 In the Second Drives drop-down list, select Curves.
 
4 Click on the curve on the left and end selection by right-clicking and selecting Continue.
 
5 In the Constraint drop-down list select G0-G0.
 
6 Select the section curve:



The handle shows up:



Then you can see the preview of the resulting surface.
