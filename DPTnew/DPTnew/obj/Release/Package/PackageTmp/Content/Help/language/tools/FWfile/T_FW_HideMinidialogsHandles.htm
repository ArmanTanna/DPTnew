The Hide Handle and Minidialogs command enables you to temporarily conceal the visualization of minidialogs and handles (to hide only minidialogs, use Hide Minidialogs), when using the Move Copy Entities, Scale Entities and Multistretch Entities commands.

When using the said transformation commands, you can find this command in the context menu that becomes available when you right-click in the graphic area if minidialogs and handles are visible, or you can run the command by hitting the Shift+F7 keys contemporarily.

From time to time, the presence of minidialogs and handles on the screen may prevent you from correctly evaluating the effect of the command preview; in this case, concealing minidialogs and handles may help you take the right decision faster.

The Hide Handle and Minidialogs command is automatically reset when you apply or cancel the feature.
