What's New in Service Pack 3

Version 2008.1 Service Pack 3 introduces the following improvements.

  • Enhancement in the Entity Properties - Drawing View - Flat Pattern category
  • The new HQRE toolbar
    • The new Super Capping command
    • The new Sampling for Recapping command
    • The new Extract Zones command
    • The new Mesh Analyzer command
  • Enhancement in the Data Exchange functionalities
    • Enhancement in the IGES Converter Options
    • Healing enhancement in the STEP and IGES Converter Options

Enhancement in the Entity Properties - Drawing View - Flat Pattern category

This version introduces a new option, Angle precision in the Flat Pattern category. It enables you to specify the precision value up to five decimal digits of the bend angle (bend data) in the flat pattern view.


The new HQRE toolbar

The new HQRE toolbar (High Quality Reverse Engineering) has been introduced in this version. As the name suggests, it constitutes of the tools which have been developed specifically to facilitate reverse engineering, and especially for class A shapes; the following commands are part of it:

  • Super Capping   (accessible only through HQRE toolbar)
  • Sampling for Recapping   (accessible only through HQRE toolbar)
  • Extract Zones   (also available with Compensator)
  • Check Symmetry   (also available with Compensator)
  • Mesh Analyzer   (accessible only through HQRE toolbar)

Licensing: The HQRE tools will be available with the Reshape module.


The new Super Capping command

The new Super Capping command is comparable to the existing Capping command. This new command is capable of producing class A surfaces. Apart from standard capping functionalities, the Super Capping command enables to:

  • provide Constraint input as Smooth points for smoothening of noisy points. The Mesh nodes (present within defined Limits) can also be the input points.
  • create a bezier capping surface by selecting bezier surface in the Generic Shape drop-down list
  • define the untrimmed shape of the capping surface by using the Smart Domain option

The new Sampling for Recapping command

The new Sampling for Recapping command enables you to create some sampling points over a set of surfaces or faces that needs to be capped later on. The created points can be used as reference while performing the capping operation. The options in the Sampling for Recapping selection list enable you to define:

  • the mode to be adopted for doing the point sampling
  • the density of the points over the reference surface

The new Extract Zones command

It enables you to extract zones on Mesh (or couple of Meshes) based on several criteria, and to create associated set of points. These criteria include:

  • Noise
  • Small Curvature Radius
  • Curvature variation
  • Curvature difference %
  • Free boundaries

Note: In Compensator, the new Extract Zones command replaces the old Extract Small Curvature-Radius Zones command.


The new Mesh Analyzer command

The new Mesh Analyzer command enables you to perform analysis of a Mesh to:

  • compute noise
  • see the minimum curvature radius
  • see gauss curvature
  • insert grid with normal
  • see osculating surface

Enhancement in the Data Exchange functionalities

Enhancement in the IGES Converter Options

The new Load colors according to IGES format specifications check box has been introduced in the Load category of IGES Options dialog box. When selected, entities are loaded with the proper color shade as specified in the IGES format.


The new Save colors according to IGES format specifications check box has been introduced in the Save category of IGES Options dialog box. When selected, entities are saved with the proper color shade into the IGES format.


Healing enhancement in the STEP and IGES Converter Options

The new Match face parametric curves with edge curves check box has been introduced in the Load - Healing category of the STEP Options and IGES Options dialog box. When selected, the application will detect and fix any mismatch between the parametric (UV) curve of the solid face and the actual 3D edge curve. Hence, it prevents gaps appearing at face boundaries due to disparity between edge curves and UV trimming curves.
