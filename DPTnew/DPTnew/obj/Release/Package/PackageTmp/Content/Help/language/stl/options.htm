STL Conversion Options

When saving (see Save and Save As) or loading (see Open and Insert from File) a STL file by selecting STL file in the Files of type/Save as type drop-down list, the Options button in the file selection box enables you to define the parameters to be used in the conversion process.

A dialog box displaying the following categories shows up:

  • Load
    This category contains the most common parameters to be set when converting from STL.
  • Save
    This category contains the most common parameters to be set when converting to STL.
