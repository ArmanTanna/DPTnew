Modify MeshSew

The Sew Mesh command enables you to merge the mesh nodes that are closer than a specified tolerance value. It is possible to select one or more meshes to apply the command giving you the following results.

  • If a single mesh is selected, all the nodes closer than the specified tolerance value are merged together.
  • If multiple meshes are selected, the meshes are first merged to create a single mesh. Thereafter, all the nodes closer than specified tolerance value are merged together.
  1. Select the mesh you would like to sew.



  2. Enter the tolerance value in the Tolerance edit box.
