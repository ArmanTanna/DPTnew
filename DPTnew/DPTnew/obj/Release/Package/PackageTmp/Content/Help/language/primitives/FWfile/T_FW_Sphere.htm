InsertPrimitivesSphere

The Sphere command enables you to create a sphere.


Obtaining a parametric solid or a surface
The final result of the command can be either a spheric surface or a parametric solid sphere. The Associative Mode check box enables you to control the result:
  • When the box is checked, the resulting entity will be a parametric solid. A drive dimension will be associated with the length of the sphere radius and the icon of a solid will be displayed in the Model Structure.
  • When the box is cleared, the final result will be a spheric surface.

To start the creation, you will have to specify a point to be used for the placement of the sphere.
At any time during the creation of the sphere, you can change the position of the placement point just by clicking in a different position.

Finally, you will have to define the length of the Radius. You can either type the value into the corresponding mini-dialog box, or define it interactively by dragging the corresponding handle.

A dynamic preview of the sphere is displayed since the very moment you specify the placement point, so as to enable you to immediately assess the result of your actions.
