The Help System

The help system goal is to enable users to easily and naturally access the online documentation and get the information they need. The online documentation has been structured in such a way that users can access only the information they really need: low level documentation about each option and command, a task accomplishing job, or a high level methodology article.

Here is a short description of the different help requests you can issue:

  • Help on commands
  • Help on controls of dialog boxes
Help on commands

When no command is currently being executed, you can easily display a page containing a short description of a command by issuing the help request as follows:

  1. Select the What's This button ( ) in the Standard toolbar
  2. Click on the command button or on the corresponding menu item.
Help on controls of dialog boxes

You can easily display a yellow-background pop-up window containing a short description of a control by issuing the help request as follows:

  1. Select the button in the top right corner of the dialog box.
  2. Click on the control you need help about.
