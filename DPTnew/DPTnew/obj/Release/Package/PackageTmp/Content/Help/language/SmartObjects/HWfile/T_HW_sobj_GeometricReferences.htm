Viewing geometric references

When the Show references command is started, the program displays a dialog box on which you can read the mandatory geometric references to be requested at insertion time (position point, profile, face, ...).

  • Boxes in the Reference column display the prompt to be displayed as a request for the reference to be selected at insertion time (if a a ";" is present, what follows is the label to be displayed on the corresponding item in the Selection List).
  • Boxes in the Type column display the type of entity to be selected as a reference at insertion time
  • The Parameters and Variable columns display parameters and variables associated with the Smart Object (if any).
