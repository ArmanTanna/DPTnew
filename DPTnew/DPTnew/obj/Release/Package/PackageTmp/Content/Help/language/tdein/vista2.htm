Video Refresh Issue on Windows Vista®

Sometimes, when using ThinkDesign on Windows Vista®, it may happen that upon certain actions, such as switching from a window to another one, the automatic video refresh is not accomplished correctly, so that you may have some flickering or some blanked zones.

You can easily bypass the problem by turning the Enable Desktop Composition option off before starting the application.

To turn the Enable Desktop Composition option off:

  • Open the Control Panel
  • Select System Properties
  • In the Advanced tab select Settings in the Performance area
  • Clear the Enable Desktop Composition check box in the Visual Effects tab of the Performance Options dialog box which will show up.
