How to insert a custom part list in the drawing

When you insert a part list in a drawing, you are enabled to select a custom part list among those you have previously created. Follow these steps to insert a custom part list.

1 Start the Part List command in order to insert the part list in the drawing. The program builds the default part list that you can see attached to the mouse pointer.
 
2 Click Options in the command selection list to open the part list options dialog box.
 
3 Click the Settings command to display the list of available custom part list configurations.
 
4 Select one of the part list configurations listed. Note that the part list attached to the mouse pointer is instantly rebuild according to the configuration that you selected.
 
5 If you wish, you are enabled to change the part list options to make last-minute refinements.
 
6 Select the insertion point to complete the positioning procedure. Note that if a part list was already present in the drawing, it is removed and replaced by the new one.
