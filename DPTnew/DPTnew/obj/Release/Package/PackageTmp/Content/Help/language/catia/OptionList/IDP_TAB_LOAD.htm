TD Catia v4 interface Options: Load Tab

Select entities: Determines the entities that are to be translated from CATIA®:
  • Model — if selected, all CATIA® entities of the Model category will be translated. If cleared, the Model entities will be ignored.
    • Wireframe — if selected, all CATIA® entities of the Wireframe category will be translated. If cleared, the Wireframe entities will be ignored.
    • Surfaces — if selected, all CATIA® entities of the Surface category will be translated. If cleared, the Surface entities will be ignored.
    • Solids — if selected, all CATIA® entities of the Solid category — except mock-up solids, which are converted to wireframe — will be translated (into solids or surfaces, depending on the current value of the Create solid check box). If cleared, the Solid entities will be ignored.
  • Drawing — if selected, all CATIA® entities of the Drawing category will be translated. If cleared, the Drawing entities will be ignored.
Hidden entities: Determines how to manage the translation of blanked (hidden) entities:
  • Hide — if selected, CATIA® blanked entities are translated as hidden entities.
  • Show — if selected, CATIA® blanked entities are translated as visible entities.
  • Delete — if selected, CATIA® blanked entities are not translated.
Solid options: Defines how solids are transformed:
  • Create solid — if selected, CATIA® entities of the Solid category will be translated into solids. If cleared, they will be translated into surfaces. Entities of the Solid category are converted only if the Solids check box in the Select entities area is checked.
  • Convert to layers — if selected, all the faces of each CATIA® solid are converted to surfaces belonging to a different unique layer, from 256 to 1023.
Surface options: Defines how surfaces are transformed:
  • Convert to Bspline — if selected, all CATIA® entities of the Surface category will be translated into Bspline surfaces, otherwise planes and cylinders are loaded as such.
CATIA Project file : Defines the CATIA® project file that will be loaded by TD Catia v4 interface and used when loading a drawing that refers to it. The CATIA® project file usually contains some drawing information, like user defined cross-hatches or text parameters. You can browse for the CATIA® project file by using the Browse button.

Note that the From CATIA to command automatically loads models and/or drawings, depending on the Select entities option that you selected.

Also note that when you import a drawing from CATIA®, usually you need to select only the Drawing option and keep all the other options unchecked. However, in some specific cases (for instance, a CATIA® 3D model projected onto a drawing plane), if the conversion seems to be not complete (that is, dimensions and texts are converted but geometric entities are missing), try to select also Wireframe.
