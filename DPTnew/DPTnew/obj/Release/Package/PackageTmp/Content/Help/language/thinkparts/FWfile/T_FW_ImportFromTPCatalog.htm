The Import from thinkparts Catalog command enables you to quickly import the selected part from the thinkparts catalog into the current think3 document.

You can find this command in the context menu of the codified thinkparts components listed in a Query View on Parts when a think3 document is open.

Upon execution, thinkteam opens the thinkparts catalog to which the selected component belongs.
 By enabling the Show codified parts only options, thinkparts presents you with a view of the components that have been assigned a Part number, from which you are enabled to Import the part into the current think3 document.
