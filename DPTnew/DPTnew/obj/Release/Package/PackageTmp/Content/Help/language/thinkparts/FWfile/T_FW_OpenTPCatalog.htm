The Open thinkparts Catalog command opens the thinkparts catalog that contains the selected part.

This command is located in the context menu of thinkparts derived parts in the query and BOM views of the thinkteam Windows Client and of the think3 applications provided that no document is currently open.

When a document is open in the think3 application, you will find the Import from thinkparts Catalog in place of Open thinkparts Catalog.

When you start the command, the program opens the thinkparts catalog that contains the selected part, selects the row related to the relevant part number, sets the Show codified parts only checkbox and hides the OK button.
