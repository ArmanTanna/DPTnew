Bend Table Parameters

The following bend table parameters are displayed in the user interface of the application:

  • Radius — the bend internal radius, in mm or inches (default is mm). The radius value can also be 0.
  • Thickness — it can be in mm or inches (default is mm). The thickness value cannot be 0.
  • Angle — the angle is measured in degrees. The range is 0° to 360°. There are two different ways to measure the bend angle:
    • Inside (1) — the angle is measured from inside
    • Outside (2) — the angle is measured from outside (default).
