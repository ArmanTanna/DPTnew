This option enables you to control the behavior of X-Reference components when they are modified in their parent file:

Always reload automatically reloaded into any open model in which they are referenced after any change has been made on them. This is the default option.
Never reload never automatically reloaded (if this option is set, after an X-reference component has been modified in its parent file, you need to use the Reload Component command to see it modified also in the open models in which they are referenced).
Ask user confirmation request will be issued before reloading a modified X-Reference component.
