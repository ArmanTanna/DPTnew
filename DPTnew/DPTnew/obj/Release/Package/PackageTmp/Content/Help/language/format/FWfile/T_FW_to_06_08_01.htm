FormatColor Map

The Color Map command enables you to determine the colors used by the program.

Note
When performed from the model environment, the command defines the setting referred to 3D models.

You can set the following colors individually:

  • the color used to draw entities
  • the background color
  • the selection highlight color
  • the dynamic highlight color (when you move the cursor over an entity)
You can also assign a material and density to each color and set or edit the following advanced color settings:
  • the diffusive color
  • the specular reflection color
  • the transparency coefficient
  • the reflection coefficient

The colors, materials, and properties you assign are retained as the current color set and can be saved together with the modeldrawing when it is saved.

For further details on properties of colors and materials, see Properties of colors and materials.

Note
The rendering commands is taken as priority in the application. This command works only if there is no rendering information applied using the rendering toolbar.
