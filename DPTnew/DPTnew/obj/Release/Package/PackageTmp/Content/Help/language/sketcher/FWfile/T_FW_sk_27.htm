ModifyProfileCheck

The Check command checks the constraints on a selected profile.

The command checks if the profile is well defined, under defined, over defined, or inconsistent and displays a summary of the current constraint conditions of the profile. The command also helps you visualize the constraint conditions by using different colors to display the entities of the profile that are underconstrained, overconstrained, or inconsistent.

When you start the Check Profile command, the results are displayed as a selection list:

In the selection list, the program uses different colors to highlight any entities that are underconstrained, overconstrained, or inconsistent. These colors can be changed by clicking on the color spots for the respective entity states, and picking another color:

A warning button will appear inside check profile whenever the evaluation fails. If you click this button, an error message explaining the cause of the failure, will appear at the end of selection list. This message will disappear when you click the warning button again.
