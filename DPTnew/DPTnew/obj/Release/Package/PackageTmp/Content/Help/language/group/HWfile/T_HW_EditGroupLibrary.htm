Editing a Group Library

When you start the Group Library command, the Group Library window is displayed (some groups are already present in the Group Library shown in the illustration below, but none will be present the first time you start the command).



The Group Library window displays the preview image of each group belonging to the current Group Library.

This document describes how to:

  • Move and resize the Group Library window
    (just using the mouse)
  • Control the visualization of the Group Library window
    (commands of the context menu displayed when you right-click in an empty zone of the Group Library window)
  • Perform actions on groups in the Group Library window
    (commands of the context menu displayed when you right-click on a group icon/bitmap in the Group Library window).

To close the Group Library window, select the cross button in the top-right corner.

 

Moving and resizing the Group Library window

The Group Library window is initially docked to the right side of the program's window. You can move it and resize as you like.

  • To move the Group Library window
    • Press and hold down the mouse selection button on a gray area of the Group Library window. The window starts moving together with your mouse.
    • Position it anywhere you like on the screen (no matter whether it will be docked to the main window edges or not), dragging it with the mouse.
    • Release the button in the desired position.



  • To resize the Group Library window
    • Simply select one edge and drag it with the mouse until the window is the desired size.



 

Controlling the visualization of the Group Library window

When you right-click on an empty zone in the library frame, a context menu containing the following commands are displayed:
 
   

RefreshEnables you to reset the alphabetical order in the group list inside the Group Library window after adding some new group (see "Adding a group to a library").
 
ViewBitmaps
ViewIcons
Each group belonging to the library can be displayed using a file icon instead of its preview bitmap.
The Bitmaps option enables you to display the group Bitmaps The Icons option enables you to display the group Icons.
 
Arrange TabsRight
Arrange TabsDown
Tabs can be arranged horizontally or vertically.
The Down option arranges tabs horizontally on the bottom side of the library window The Right option arranges tabs vertically on the right side of the library window.

Note that the display options you choose remain unchanged until you explicitly change them, even if you stop the application and run it again.

 

Performing actions on groups in the Group Library window

When you right-click on a group icon in the library frame a context menu containing the commands explained below is displayed. These commands enable you both to load groups into a drawing and to edit the group library:
 


InsertImports a group into your drawing (See Inserting a group into the drawing)
Cut SelectionCuts the selected group from the current Group Library to the Windows clipboard. You can then paste the group into a different library. (Click the corresponding tab to switch to a different Group Library. For example, you might move your group from the Generic Groups tab to the Generic Shapes tab just by cutting it from Generic Groups, clicking Generic Shapes and pasting it by right-clicking in an empty zone and selecting Paste Selection in the context menu)
Copy SelectionCopies the selected group(s) from the current Group Library to the Windows clipboard. You can then paste the group(s) into a different library. (Click the corresponding tab to switch to a different Group Library, right-click in an empty zone and select Paste Selection in the context menu)
Delete SelectionRemoves the selected group from the current Group Library
RenameRenames the selected group. The name of the group is immediately selected so that you can change it. The group contained in the ".e2" file is also renamed to the new name you specify.
Properties Opens a dialog box displaying the properties of the file containing the group (Size, Creation, Modification and Access dates, Read only or not attribute).
