Inserting Title Block Fields

The Text command enables you to insert symbolic texts referencing document and part properties and use them as fields when you are designing your custom Title Block.

1 You can insert either a predefined Document property or a predefined Part property by selecting the relevant value from the Text from Dictionary drop-down list in the Text command selection list:
  • To insert a predefined Document property, select Document properties and expand the Text from Dictionary node:

  •  
  • To insert a predefined Part property, select Part properties and expand the Text from Dictionary node:
2 Double-click the property you wish to add to the Title Block. The dialog box is closed and the predefined property string is displayed: drag it into position:


 
3 Click in the final position. If the Symbolic text expansion check box in the Symbolic texts area of the Text category in the Document Properties dialog box is selected, the value of the predefined property will be displayed:
