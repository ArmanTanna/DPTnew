Creating a Walk-through Animation

It is strongly recommended to go through the Creating an Animation and Generating Camera Movements During Animation documents before attempting this tutorial.

Using the new Camera evolution it is also possible to create a walk-through animation of the model. This is made possible by making the camera move on a curve (representing the path of the walk-through) and at the same time keeping its focus on the same curve. Follow the steps given below.

1. Open the walk_through.e3 file from the Samples folder of your installation. In the model you will find the red curve, which denotes the path for the walk through.
2. Make an exact copy of the red curve at the same location using the Copy and Paste commands, so that both the curves overlap each other.
3. To facilitate the selection of the curves in the right order (during setting the camera data), extend one of the curves tangentially along its length by using the Extend command. Make the extension at the farther end from the house.
4. Invoke the Animation command; the Animation dialog box is displayed. Create a new animation, the Camera evolution and its variation.
5. Set the variation parameters as:
Start time = 0, End time = 20
6. Right-click on the variation for the camera and select the Set Camera Data option from the context menu; the Animation - Evolution selection list is displayed in the Graphics Area.
7. For the Camera Path selector, select the extended curve. For the Target Path selector, select the shorter curve. Make both the selections towards the farther end from the house.
8. Set the value of No. of Control Points option under More Options to 1.
8. Once all the parameters have been specified, select the OK () button to continue.
9. Now playback the animation by referring to the Animation Playback document. Ensure before playback that the current view has been set to Perspective Projection.
10. During the playback, the view of the animation will move along the specified path. To reverse the movement direction, use the Invert Movement Direction option on both the curve selectors.
