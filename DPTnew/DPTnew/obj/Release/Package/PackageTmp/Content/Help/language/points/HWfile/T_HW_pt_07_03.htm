Project points on a surface along a guide line

1 Open the file called pt_07_02.e3.



In this example, you are explained about how to project a set of points on a surface along a guide line.
2 To do this, start the Project command and you are asked to select the points you want to project.
3 In this case, after selecting the points and the projection surface, select the Pierce option from the Direction drop-down list. Once you select this option, you will see one more Direction drop-down list, under this option. Now, select the Line option from this drop-down list.
4 If you select the guide line,



the points are projected on the surface along lines parallel to this guide line.
