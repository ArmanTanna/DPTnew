Document Properties

Category: Grid-Reference-Advanced

Coordinate system

Set to

Set to:

The options in this drop-down list decide the alignment of the Reference Grid.
If World is selected, the Reference Grid aligns with the world origin.
If Current work plane is selected, the Reference Grid aligns itself with the current orientation of the Work Plane.

Update

Update

This button is active only when the Current work plane option is selected in the Set to: drop-down list. On choosing this button, the Reference Grid updates its alignment in reference to the current orientation of the Work Plane.

Origin

Origin:

This is a read-only option and displays the coordinates of the Reference Grid's origin.

Axes

Name

Name

The entries in the edit boxes of this column specify the characters that will appear in the label of the corresponding axis.

Orientation (X,Y,Z)

Orientation (X,Y,Z)

The entries under these columns are read-only and give the orientation of the three axes of the Reference Grid.

3D plane

3D plane

The Show check boxes under this column toggle the display of the corresponding 3D plane of the Reference Grid in the rotated views.

First

First:

The options in this row control the display of the first axis of the Reference Grid.

Show

Show

When selected, the first plane of the Reference Grid is displayed while viewing the grid in rotated views.
When cleared, the first plane of the Reference Grid is made hidden while viewing the grid in rotated views.

Second

Second:

The options in this row control the display of the second axis of the Reference Grid.

Show

Show

When selected, the second plane of the Reference Grid is displayed while viewing the grid in rotated views.
When cleared, the second plane of the Reference Grid is made hidden while viewing the grid in rotated views.

Third

Third:

The options in this row control the display of the third axis of the Reference Grid.

Show

Show

When selected, the third plane of the Reference Grid is displayed while viewing the grid in rotated views.
When cleared, the third plane of the Reference Grid is made hidden while viewing the grid in rotated views.
