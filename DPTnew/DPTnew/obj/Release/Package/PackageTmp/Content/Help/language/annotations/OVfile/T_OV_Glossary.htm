Browsing

Using a Boolean search string to control the visibility of the annotations.

Category

The category is a string that is used to organize the annotations.

Expanded Mode

The annotation possesses two graphical modes. The first being a compact mode in which an icon and the short text label is displayed. The expanded mode of the annotation is the advanced mode usually represented by a resizable dialog containing a sub-application allowing the user to view the notes additional content.

Keywords

Keywords allow the annotations to be searched for via a Boolean search string.

Point of Interest

The “Point of Interest” is the point, on the model, to which the annotation is anchored. It indicates the subject of the annotation's text or content. The point can be:
  • Any point on a face of a solid
or may reside at the center of, end point, mid point, or any where on a curve (as long as it is linked to history). Good points of interest, for example, are:
  • Vertex on solid
  • Point (center, end, or mid point) on edge
  • Point on the corners of an associative surface
  • Points on profile geometry.

URL — Uniform Resource Locator

A URL is an address for a document on the Internet. The syntax of this type of address is standardized at international level.
For example, the URL of our web site is http://care.think3.eu.
