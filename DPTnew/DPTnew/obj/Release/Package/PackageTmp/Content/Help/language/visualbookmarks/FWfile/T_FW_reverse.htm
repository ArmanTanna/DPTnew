The Reverse command enables you to de-apply the visual bookmarks that you applied.

A button (  ) is displayed in the right-bottom corner of the Visual Bookmark panel. The name (category name+ Visual Bookmark name) of the last Visual Bookmark applied – the one the Reverse command will be applied to - is displayed in the gray area on the left of the button. You can use the Reverse command to simplify the model/drawing to work only on a part of it.

Note
This command is available only on the last Visual Bookmark applied.
