Displaying the properties of a selected Smart Object

The Properties command displays a dialog box containing information about a selected Smart Object. The Smart Object Properties contains two tabs:
 

  • General
  • Preview & Description

 

General

This tab displays the following data:
 
The name of the Smart Object
Type The type of the object: Smart Object, Smart Shape or Smart Profile
Location The location of the folder which the Smart Objects is in
Size The size of the Smart Object (both in KB and bytes)
Created The date the Smart Object was created
Modified The date of the last change made to the Smart Object
Accessed The date of last access to the Smart Object
Attributes/Read Only If this box is checked, the Smart Object is read only.

 

Preview & Description

This tab displays:
 

  • The Smart Object Preview
  • The Smart Object Description, associated with the Smart object when it was defined
  • The File list area, on which you can attach files to a Smart Object, open them or delete them from the list of the attached files (see Attaching files to a Smart Object).
