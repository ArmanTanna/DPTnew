The following commands enable you to insert a predefined two-dimensional view of a 3D model — say, for example, the following

 — into the current drawing. You can insert any of the following predefined views:
Top model viewed from the top (looking down along the Z-axis) perpendicular to the XY-plane of the World Reference System.
Front model viewed from the front (along the Y-axis) perpendicular to the XZ-plane of the World Reference System.
Left model viewed from the left (along the X-axis) perpendicular to the YZ-plane of the World Reference System.
Right model viewed from the right (along the X-axis) perpendicular to the YZ-plane of the World Reference System.
Rear model viewed from the back (along the Y-axis) perpendicular to the XZ-plane of the World Reference System.
Bottom model viewed from the bottom (looking up along the Z-axis) perpendicular to the XY-plane of the World Reference System.
Right Back model viewed as an isometric view from the right-back (along the line from coordinate 1,1,1 to the origin) relative to the World Reference System.
Left Back model viewed as an isometric view from the left-back (along the line from coordinate -1,1,1 to the origin) relative to the World Reference System.
Left Front model viewed as an isometric view from the left-front (along the line from coordinate -1,-1,1 to the origin) relative to the World Reference System.
Right Front model viewed as an isometric view from the right-front (along the line from coordinate 1,-1,1 to the origin) relative to the World Reference System.
