The Organize command enables you to arrange the different Light Representations for the selected X-Reference component.

The command is available from the context sub-menu displayed when right-clicking on the X-Reference (component) and selecting the Light Representations command.

Note that the command can be applied exclusively to X-Reference components.

The command displays a dialog box where you can view the current representation and add a new Light Representation, as well as edit, remove or apply an existing one.

Typically, the command enables you to organize and clean up the Light Representations of a specific X-Reference (component).
