Adding a Free Frame

To add a Free Frame, refer to the general procedure described in How to create customized Drawing Frames, following some additional specific rules.

  • The size of the frame is determined by the dimensions of the frame bounding box.
  • When using the Save as Drawing Frame command, you will not be able to set the orientation of the frame, as it depends on the bounding box orientation.
  • When you select Free from the Size drop-down list of the Save as Drawing Frame command, the program displays all free frames available.
  • When you select a free frame, the program displays the X and Y dimensions of the frame.
