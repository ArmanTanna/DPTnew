FormatColor Map

The Color Map command enables you to determine the colors used by the program.

Note
When performed from the model environment, the command defines the setting referred to 2D layouts.

You can set the following colors individually:

  • the color used to draw entities
  • the background color
  • the selection highlight color
  • the dynamic highlight color (when you move the cursor over an entity)

In addition to the color used to draw entities, you can associate a width value with every color. This parameter determines plotted width of lines (line weight) of the current color when the Colors option is selected in Width, in the Print and Print Setup dialog boxes.

Note
You can assign width values to colors also when defining the print options, that is directly from the Print and Print Setup dialog box.

The colors and widths you assign are retained as the current color set and can be saved together with the drawing when it is saved.
