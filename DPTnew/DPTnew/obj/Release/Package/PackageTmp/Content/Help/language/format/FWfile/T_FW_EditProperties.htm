The Edit Properties command — available in the Edit category of the Commands tab of the Customize dialog box — enables you to modify the properties of existing entities. You can use the command in the following ways:

  • Pre-selecting entities and then starting the command
    You pre-select one or more entities and then start the command (please note that it is available also in the context menu displayed by right-clicking after pre-selecting one or more entities). The Edit Properties dialog box will be displayed. Change the values as you like and select OK to confirm the changes or Cancel to reject them.
     
  • Starting the command, selecting entities and changing the properties
    You first start the command and select the entities whose properties you want to change. The Edit Properties dialog box is displayed: then you can change the property values. Select OK to confirm the changes or Cancel to reject them.
Applying the command to entities belonging to groups
The command can be applied to entities belonging to groups by using the Select inside Groups command.

Remember you can modify the properties of existing entities also using the Assign Properties command.
