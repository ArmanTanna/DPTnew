think3 to Parasolid conversion table

Here is a general conversion table showing how think3 entities are mapped to the Parasolid ones when converting a think3 file to the Parasolid format.

think3 entity Parasolid entity to which it is converted
Point Point
NURBS curve BSpline curve
Line BSpline curve
Arc BSpline curve
Ellipse BSpline curve
Offset curve BSpline curve
NURB surface BSpline surface
Cylinder Cylinder
Plane Plane
Offset surface BSpline surface
Ruled surface BSpline surface
Blend surface BSpline surface
Tabulated cylinder Sweep
Surface of revolution BSpline surface
Coons surface BSpline surface
Trimmed surface Face
Vertex Vertex
Edge Edge
UV curve Coedge
UV loop Loop
Face Face
Shell Shell
Manifold solid Body, if the relevant option is checked
Color Top entity only (No solid multicolor and color for trimmed surface)
Layer Not converted
Component name Not converted
Font style Not converted
Width Not converted
Blank Solids only (No curves nor surfaces)
Datum plane Plane, if the relevant option is checked
Datum point Point, if the relevant option is checked
Datum curve BSpline curve, if the relevant option is checked
Profile BSpline curves, if the relevant option is checked
