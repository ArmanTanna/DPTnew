The Modify Dimension command enables you to modify portions of a dimension. The command can also be started by preselecting a dimension entity and selecting any of the following:

  • Modify Dimension ( ) toolbar button
     
  • ModifyDimensionModify command
  • Modify Dimension command from the context menu

Depending upon the type of entity selected (Dimension, Label, Surface Textures, Feature Control Frame, Datum Target and Datum Reference), the selection list options will be displayed.

You can perform the Modify Dimension command functionalities when the Move command is active. To do so:

  1. Start the Move command.
  2. Select a dimension text or a witness line.
  3. Then, right-click on the Graphics Area. A context menu will be displayed showing the relevant options belonging to the part of the dimension selected.
