Corner Reliefs

Corner Reliefs

The controls in this area enable you to define the relief parameters affecting corners.

Enable

Enable

When this box is selected, a slot is created on the corners. The slot size is sufficient to enable the creation of fillets andthe gap value is also added to it; when the box is not selected, no slot is created.

Type

Type:

Enables you to select the type of relief to be created on corners:

Rectangularrelief has a rectangular shape defined by the bend radius (the gap value is also added)
Triangularrelief has a triangular shape defined by the bend radius (the gap value is also added)
Circularrelief has a circular shape defined by a diameter

Gap

Gap:

Enables you to specify the gap value to be added to the corner relief when it is rectangular or triangular.
