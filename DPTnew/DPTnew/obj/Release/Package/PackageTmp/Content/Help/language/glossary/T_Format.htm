Organize Formats

Formats are a collection of attributes that the user needs to change often. They have the purpose to speed up your work by quickly changing a number of settings at once. You can create your own formats and store them in the library, so that you can use them readily.

A drop-down list on the top right corner of the Organize Formats dialog box enables you to choose a format from the existing list.

Organize Formats provides you two classes of Formats:

  • General (Color, Line type, Width and Layer)
  • Dimension (Prefix/Suffix, Tolerance, Precision and Symmetric)

Color:

The color specified here will be stored in the general entity format that you create.

Line type:

The Line type chosen here will be stored in the general entity format that you create. And this Line type will be default for all the entities that will be created.

Width:

The width chosen here will be stored in the general entity format that you create.

Prefix/Suffix:

The text entered here will be stored in the dimension format that you create. But to enter a prefix or a suffix, the check box, associated with this category, has to be checked. When the check box is not checked, the default system values are applied to the current format.

Tolerance:

The tolerance category has various options like Representation, Add parenthesis, Tolerance label and Limit arrangement. Under Tolerance, we have a sub-category, Value Expression. This expresses the tolerance value with either Deviations or Limits, Linear or Angular, by specifying the Value above and Value below.

Precision:

The precision value could be Linear, Angular or Dual.

Symmetric:

The symmetric symbol could be either enabled or disabled, by choosing the appropriate option.
