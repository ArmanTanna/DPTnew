InsertSolidSweepPipe Protrusion

The Pipe Protrusion command adds a pipe protrusion to an existing solid. The pipe protrusion is created moving a circular cross section along a path defined by an existing open or closed spine curve profile. The resulting protrusion can be either a solid rod (by specifying only the diameter of the cross section) or a hollow tube (by specifying both the diameter of the cross section and the thickness of the tube wall).

The resulting solid is connected directly to the base spine profile. If you change the profile in any way (such as by using any of the profile edit commands), the solid changes accordingly.
