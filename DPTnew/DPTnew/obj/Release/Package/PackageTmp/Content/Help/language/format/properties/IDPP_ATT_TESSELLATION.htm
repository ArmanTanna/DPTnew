Tessellation parameters

Tessellation parameters

The controls in this area govern the tessellation operations: the dividing of smooth surfaces into polygons which approximate the surface. The values are all measured in the current unit of measurement.

Fine

Fine

When you move the slider to this tick, the maximum tessellation quality level is reached, corresponding to the following values:

  • Surface tolerance: 0.001
  • Boundary tolerance: 0.001
  • Surface Accuracy: 100%

As higher values yield better display and mass property calculations but lower performance, when setting this value you will have the best quality but, if the model is a big and complex one, the performance time might be higher.

To display the tessellation on all entities of the model, start the Tessellation command.

Rough

Rough

When you move the slider to this tick, the minimum tessellation quality level is reached, corresponding to the following values:

  • Surface tolerance: 10
  • Boundary tolerance: 10
  • Surface Accuracy: 0%

As higher values yield better display and mass property calculations but lower performance, when setting this value you will have the worst quality but the performance time will be minimum even on complex models.


To display the tessellation on all entities of the model, start the Tessellation command.

Custom

Custom

When selected, it enables you to directly specify the numerical values of the parameters which control tessellation operations:

  • Surface tolerance
  • Boundary tolerance
  • Surface Accuracy

Surface tolerance

Surface tolerance:

Determines the distance between the surface and the tessellation approximation plane. Smaller values yield a closer approximation of the surface by creating a larger number of smaller polygons.

To display the tessellation on all entities of the model, start the Tessellation command.

Boundary tolerance

Boundary tolerance:

Determines the distance between the boundary curve and the polyline which approximates it in the resulting tessellation. Smaller values yield a closer approximation of the surface by creating a larger number of smaller polygons.

To display the tessellation on all entities of the model, start the Tessellation command.

Surface accuracy (%)

Apply Changes

Determines the percentage of tessellation points used to describe connected surfaces. Higher values yield better display, mass property calculations but lower performance.

To display the tessellation on all entities of the model, start the Tessellation command.
