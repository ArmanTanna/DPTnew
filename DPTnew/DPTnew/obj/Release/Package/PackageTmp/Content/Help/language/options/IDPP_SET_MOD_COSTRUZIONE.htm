Construction

Construction

The controls in this area enable you to govern the behavior of a large set of construction operations.

Use symbolic references

Use symbolic references

When selected, it activates the Symbolic References Only option in the Mating command, which restricts selection exclusively to entities associated to Symbolic References.

When the check box is not selected, the Symbolic References Only option is not active.

Keep variable name by inserting smart objects

Keep variable name by inserting smart objects

When inserting a Smart Object into a model, the Smart Object and the model might contain variables with the same names.

  • When this check box is selected, variables associated with exposed parameters are automatically imported into the Spreadsheet with their own names. Only mini-dialog boxes associated with variables not already existing in the Spreadsheet are editable.
  • When not selected, all variables associated with exposed parameters are automatically imported and renamed (a prefix is added, consisting of the letter N, a sequence number and the _ character) and any mini-dialog box is editable.

For further details, see also "Managing Duplicated Variables when Inserting Smart Objects".

Show on failed update

Show on failed update

When selected, in the case a feature update (see for example: Rebuild Model, Update (Profile), Redefine Feature) should lead to an unexpected result, such as an open or a not manifold object, the feature will be completed all the same, thus letting you operate on the modified solid (see also Check Manifold).

When this check box is not selected, if a feature leads to an unexpected result, it will not actually be made (see also "Recovering errors when rebuilding").

Hide dimensions in feature creation

Hide dimensions in feature creation

When selected, all existing dimensions will be hidden as long as a feature is being created.

When not selected (default), dimensions will always be visible.

Show Rebuild Step by Step in selection list

Show Rebuild Step by Step in selection list

When selected, the Rebuild Step by Step button is displayed in the selection list, whenever a update of the model is required following the modification of its features.

When not selected (default), the Rebuild Step by Step button is not displayed in the selection list.

Unlimited lines:

Unlimited lines:

The actual length of a Semilimited or Unlimited line (see the 2 Point Line, Parallel Line and Bisector Line commands) can be set in the following ways:

View sizeThe resulting line will be extended up to the view box, that is the visible part of the document.
Document sizeThe resulting line will be extended up to the document box, that is the box that includes all of the entities in the document.

Solid color

Solid_color

The radio buttons in this area enable you to define the color of the solid being generated during an operation. It is quite relevant for Boolean.

First entity color

First entity

When selected, the resulting solid takes up the color of the first entity that was selected during the solid creation operation.

Current color

Current color

When selected, the current color as set in the status bar is applied to the solid during the solid creation operation.

Keep face color

Solid_color

When selected, the solid retains the colors that are already applied to the faces prior to the solid creation operation.
