The Remove all Filters command resets all the filters set within the current query view.

You can find this command as a button in the thinkparts query view dialog that opens when you run the Open command on a catalog of parts or in the thinkparts view section of the context menu of a part in the catalog.

After resetting the filters a new query is automatically started; as a result, all parts in the database will be listed in the view.

A similar command is available in the context menu that displays when you right-click on a column header that has a filter defined. In this case, however, only the filter of the selected column is deleted.
