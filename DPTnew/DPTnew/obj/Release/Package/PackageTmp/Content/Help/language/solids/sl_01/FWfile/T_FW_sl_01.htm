InsertSolidSweepLinear Protrusion

The Linear Protrusion command adds a solid linear protrusion to the face of an existing solid by sweeping a profile or chain of curves along a linear path. In the case of a chain of curves, the program automatically combines the separate curves — which must be coplanar — into a single profile.

This command can manage multiple complexes. A complex is a closed loop containing multiple disjoint closed loops, also called as islands. Islands cannot contain other islands nor intersect other islands.

Correct
The rectangle is a closed loop that contains various disjoint islands (circles).		 Incorrect
An island is inside another one.		 Incorrect
Two islands intersect.
