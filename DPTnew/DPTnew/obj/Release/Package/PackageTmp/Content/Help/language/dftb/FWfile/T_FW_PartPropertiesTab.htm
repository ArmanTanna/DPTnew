The Edit Part Properties tab is available in the File Properties dialog box which is displayed by the Properties command.

It enables you to edit the following Part Data:

Part number: Enter the name or number of the part (or "componentgroup"). Note that if the Assign part number as name option of the Assembly/Shared Group Assembly/Shared Group category of the System Options is checked, the componentgroup name in the ModelDrawing Structure is updated with the value of Part number:.
Description: Enter the part description. You are enabled to access a dictionary of standard descriptions by means of the button.

Double clicking on a description with <> symbol as postfix from the Dictionary (for example, AGITATOR <par1>), the program displays a Variables dialog box that enables you to customize part description.


Measurement unit: Optionally enter or select from the drop-down list the measurement unit for the part.
Item type: Optionally enter or select from the drop-down list the part item type.
Material: Optionally enter or select from the drop-down list the part material.
Heat treatment: Optionally enter or select from the drop-down list the part heat treatment.
Surface treatment: Optionally enter or select from the drop-down list the part surface treatment.

You are also enabled to execute the following commands:

  • Copy
    To copy the current Part Properties in order to paste them onto another componentgroup.
  • Paste
    To paste the Part Properties previously copied from another componentgroup.
  • Remove
    To clear all fields.
