Editing Curves

Curve through Control Points Enables you to modify a curve by editing its control points.
Curve through Interpolation Points Enables you to modify a curve by editing its interpolation points.
Free Hand Curve enables you to quickly and easily modify a curve you created through the Insert Free Hand Curve command by using the very same technique.
Curve Shape Constraint Enables you to select a curve in an arbitrary location and modify its shape using a temporary handle.
Convert Curve to NURBS Enables you to convert a curve into a NURBS entity.
Convert Curve to 3D Curve Enables you to convert a curve of the Curve on Surface type into a NURBS entity
Curve Continuity Enables you to control continuity between curves.
Advanced Curve Continuity Enables you to control continuity between curves/chains of curves.
Curve Flexibility Enables you to modify degree and continuity of curves.
Split Curve Enables you to split one curve into many ones.
Extend Curve Enables you to modify the extension of a curve
Through Points Enables you to make a curve pass through a set of points.
Invert Orientation Enables you to invert the orientation of curves.
Update Parameters Enables you to modify a curve by editing its own parameter.
Curve Tension Enables you to modify a contour — that is: a chain of consecutive curves — by changing its tension value.
Curve Target Curvature Enables you to modify a contour by changing its curvature plot.

Context menu commands

Disable Associativity Enables you to remove associativity from entities that have been promoted to history graph genitors, that is generic root associative curves/surfaces.
Enable Associativity Enables you to apply associativity to non-associative curves and surfaces, thus promoting them to history graph genitors, that is generic root associative curves/surfaces.
