The Expand All command enables you to open a selected icon in a tree displayed in the different left panes of the think3 application windows.

The command applies to different contexts (Model Structure/Drawing Structure, Visual Bookmarks and Annotations panels).

In drawings views, see the specific command enabling you to open a selected icon in the Model Structure window of the Custom and Exploded view command dialog box.

To start the command, right-click on an icon in the tree and select Expand All in the context menu.

To close an open icon, select the Collapse All command.
