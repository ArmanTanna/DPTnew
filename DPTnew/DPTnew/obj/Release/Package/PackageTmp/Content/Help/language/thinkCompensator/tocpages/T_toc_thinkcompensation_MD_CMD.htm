Compensator MD Commands

Identify Measured Data Enables you to find the correspondence between an initial and measured mesh, but will identify the deformation corresponding to the compensation to be applied on the base Mesh.
Blank-Holder Addendum and Part Enables you to modify the die environment based on the result obtained by adaptation/compensation process.
Offset Mesh Enables you to offset a selected mesh by a specified offset value.
Check Symmetry Enables you to detect symmetry conditions in a selected mesh entity.

Adaptation Command

Adapt on Measured Data Enables you to find the correspondence between an "Initial Mesh" (corresponding to a base geometry) and a "Measured Mesh" (corresponding to a modification of the base geometry). The result can then be used to match the initial mesh with the measured one.
