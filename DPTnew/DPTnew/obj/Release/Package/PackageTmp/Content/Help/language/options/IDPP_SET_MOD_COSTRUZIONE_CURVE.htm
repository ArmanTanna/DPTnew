Ask before converting

Ask before converting

When selected, the program asks whether specialized curves and surfaces should be converted into NURBS curves and surfaces.

Please note that when more than one entities are to be converted within the same command, a specific option (OK to All) will enable you to confirm conversion for all of them with just one click, without having to confirm each time.
Also note that if you select No the command will be interrupted.

Conversion to NURBS curves and surfaces can occur when using the following commands.

When modifying curves:

  • Curve Continuity
  • Curve Flexibility
  • Extend Curve
  • Modify Curve Shape Constraint
  • Modify Curve through Control Points
  • Modify Curve through Interpolation Points
  • Through Point Curve

When modifying Surfaces:
  • Join Surface
  • Modify Surface Shape Constraint
  • Modify Surface through Control Points
  • Normal to Degenerate Surface
  • Surface Continuity
  • Surface Flexibility
  • Invert Surface

When not selected, the conversion occurs automatically.

Tolerance

Tolerance:

This value specifies the tolerance used when converting specialized curves and surfaces to NURBS curves and surfaces.
See also Ask before converting.

Automatic cut

Automatic cut

This option affects the behavior of the following commands:

  • Offset on Plane
  • Fillet
  • Double Fillet

Offset on Plane

When this check box is selected and you select the Fillet option of the command, if the selected chain of curves has sharp corners and the offset requested is made in the area on the outside of them, the corners are filleted and the portions of curves that extend beyond the fillet are automatically eliminated.
When not selected, the fillets are not created.

Fillet

When this check box is selected and the Trim/Extend drop-down list in the command selection list is set to First or Both, the first or both the lines/curves extending beyond the fillet arc are trimmed.
When not selected, the original lines/curves are not trimmed.

Double Fillet

When this check box is selected and the Extension drop-down list under More Options in the command selection list is set to Trimmed, the portions of the selected surfaces extending beyond the new blend surface are automatically eliminated.
When not selected, the original surfaces are not trimmed.

Rebuild

Propagate properties

Propagate properties

When the box is checked, textures and tessellation properties are retained after rebuilding. When the box is cleared, textures and tessellation properties are lost after rebuilding.
