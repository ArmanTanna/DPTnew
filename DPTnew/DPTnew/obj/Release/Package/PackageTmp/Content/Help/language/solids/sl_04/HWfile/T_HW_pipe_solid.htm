Creating a pipe sweep solid

The Pipe Solid command enables you to create a solid by sweeping a circular profile along a spine path.

1 On starting the command, you are prompted to select a profile or curves. Select the spine along which you want to sweep the pipe, and then end selection. To reselect the spine, click on Spine in the selection list and select the spine again.
2 In the Diameter mini-dialog, specify the diameter of the pipe
or
click and drag the handle to interactively set the diameter.
3 If necessary, click More Options to specify additional options.

  • Use the Thickness mini-dialog to create a shelled solid
    or
    drag the handle to set the required thickness.
4 Click or to confirm your selections and create the solid. Click to discard your changes.
