TD Catia v4 interface installation

You can add TD Catia v4 interface to your system during the software installation or add it to an existing installation of the latter. In either case, follow these steps:

  1. Insert the software installation medium in the drive. If it does not start automatically, browse its contents and run the setup program ()
  2. Click Products.
  3. Click .
  4. Click Install and wait for the installation wizard to start.
  5. Click Next in the welcome page.
  6. If the product installation is already present on your system, a dialog enables you to update the installed software. If this is your case, select Update the installation and click Next.
  7. Select the Custom installation option to access the components to be installed and click Next.
  8. Check TD Catia V4 interface to add the component to your installation.
  9. Continue accepting the default installation options and close the final dialog that informs you that the setup is complete.
  10. Start the application and open a new model.
  11. Access the Toolbars tab of the Tools Customize command, and turn on the TD Catia v4 interface toolbar for the modeling environment.
  12. Dock the toolbar in any position you like.
  13. Open a new drawing.
  14. Access the Toolbars tab of the Tools Customize command, and turn on the TD Catia v4 interface toolbar for the drafting environment.
  15. Dock the toolbar in any position you like.
