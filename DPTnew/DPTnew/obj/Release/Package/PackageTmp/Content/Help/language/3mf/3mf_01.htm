Opening a 3MF file

To convert from the 3MF format to think3, do the following:

  1. Select Open or InsertFrom file
  2. In the file dialog box, under Files of type, select 3MF Format (*.3mf)
  3. Select the file you want to open
  4. Click the Open button to open the file (no preview is available for 3MF files: the file icon will be displayed in the preview frame).
