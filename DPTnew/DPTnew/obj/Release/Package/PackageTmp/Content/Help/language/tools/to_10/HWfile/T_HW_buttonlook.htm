Choosing look and size for toolbar buttons

The Button settings area of the Toolbars tab of the Customize dialog box enables you to change both the Look and the Size of toolbar buttons.

Changing the look

The Look drop-down list enables you to choose which look to use for the toolbar buttons:

New This look is completely Microsoft® Windows® XP compliant and provides smart looking toolbar button images (with a number of colors increased up to 16.8 million). It is active also on Microsoft® Windows® 2000. It also affects Model Structure and Drawing Structure icons.
Old This look is the one used in all previous versions. The number of colors used is 16.

Please note that your choice can be interactively assessed — before applying it —  in the Example area, displaying a preview of some icons with the chosen look and feel.

New Old

A comparative table of the toolbar icons in the two looks is available in "Toolbar Icons: Old Look/New Look Map".
A map of the Selection List items is also available in "Selection List Items: Old Look/New Look Map".

As soon as you select a different Look, a message will show up to warn you your look changes will be applied next time you start the program.



When you click Close, the following message will be displayed:


 

  • Select Yes to immediately quit. As soon as you restart it, the chosen look will be applied.
  • Select No to go on with the current session. The look change will be applied in next session.
Using the program Settings Utility
Please note that you can always change the look and feel of buttons also in the specific page of the program setting wizard activated by selecting:
Start  Programs  think3  thinkdesign 2006.1  Utilities  Settings Wizard.
Once you have selected the GUI Profile to be used, you are enabled to choose whether to activate the old or the new look and feel for toolbar buttons.

Changing the size

The Size drop-down list enables you to select the button size, which can be:

  • Small, Medium or Large , when the Look is New
  • Small or Large, when the Look is Old.
    •

As the available sizes are two for the old look and three for the new one, there is no biunique correspondence. For this reason, after changing the Look you will have to restart the program before being enabled to change the size as well.
