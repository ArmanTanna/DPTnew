Drive and Reference Dimensions

The think3 application is a parametric and dimension-driven solid modeler. In fact, you can create solid models whose dimensions can be changed at any time; you can change the size of a part by selecting and editing its corresponding parametric dimension values. But you can also add simple dimensions, which do not control the parametric structure of a solid and which you can use just as references.

Therefore, you can find two different types of dimensions in a model:

  • Drive (or driving) dimensions
  • Reference (or driven) dimensions.

Note that you can distinguish the two types of dimensions selecting the Highlight Dimensions command, after properly setting the Driving and Reference options under Highlight dimension color in the Dimension category of Document Properties.
