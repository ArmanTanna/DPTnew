Spreadsheet Commands

Assign Enables you to assign a variable to a drive dimension or a feature or a datum entity or an X-reference component
Deassign Enables you to assign a variable from a drive dimension or a feature or a datum entity or an X-reference component
Rebuild Enables you to rebuild the model with the selected variable
Hide Variables Enables you to hide the selected variable in the spreadsheet window
Unhide All Enables you to unhide all the variables in the spreadsheet window
Show Dimensions/Features Enables you to highlight the selected variable in the model
Delete Variables Enables you to delete the selected variable from the spreadsheet
