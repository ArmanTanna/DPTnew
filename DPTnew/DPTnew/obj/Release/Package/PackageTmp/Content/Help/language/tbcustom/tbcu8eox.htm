Resetting a Default Toolbar

To restore a default program toolbar to its original condition after it has been modified, do the following:

  1. Select ToolsCustomize from the menu bar to open the Customize dialog box.

  2. Move the mouse pointer over the toolbar you want to reset and right-click.

  3. Choose Reset Toolbar from the toolbar popup.

Note
You cannot reset a custom toolbar that has been created from scratch. In such a case, the Reset Toolbar option is disabled.
