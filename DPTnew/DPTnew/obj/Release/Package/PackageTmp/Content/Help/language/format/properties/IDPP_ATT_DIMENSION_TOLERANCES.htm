Current:

Enables you to set the current style by selecting it in the list of all the available ones.

Tolerances

Tolerances

Enables you to specify the type of Tolerances to be included as part of the dimension.

Representation

Representation:

Enables you to select a proper way of Representation for the Tolerances included as part of the dimension. You can choose from one of the following options:

None Enables you not to represent any tolerance
+/- Enables you to represent the tolerance value using a single numerical value
Two Values Enables you to represent the tolerance value using upper and lower values, which are specified in the Value expression section
Limits Enables you to represent the tolerance value using upper and lower limit values, which are specified using the Value expression section
Basic Enables you to represent the dimension value inside a box, as a "basic dimension symbol".

Add parentheses

Add parentheses:

Enables you to specify whether to Add parentheses along with the tolerances. You can choose from one of the following options:

Never Enables you to represent the tolerance value without the parenthesis
With tolerance label Enables you to represent the tolerance value using the parenthesis along with the tolerance label
Always Enables you to always represent the tolerance value with the parenthesis

Label

Label:

Enables you to enter the text that will appear as the Tolerance label for the dimension.

Limit arrangement

Limit arrangement:

Enables you to set the Limit arrangement for the tolerance limits. The options that are available in the drop-down list are:

Vertical Enables you to represent the tolerance limit values in the vertical form
Horizontal Enables you to represent the tolerance limit values in the horizontal form

Value expression

Value expression

The controls in this area enable you to define the tolerance settings separately for linear and angular dimensions.

  • The Mode drop-down list enables you to choose whether to specify the tolerance values as Deviations or as Limits.
    Value above: Enables you to determine the upper tolerance value.
    Value below: Enables you to determine the lower tolerance value.
  • The Measure drop-down list enables you to select the type of dimension the Mode choice will apply to: Linear or Angular.

Mode

Mode:

Enables you to choose whether to specify the tolerance values as Deviations or as Limits.

Deviations The Value above and Value below values will be displayed by the dimension side.


 
Limits The Value above and Value below values will be algebraically added to the dimension value. The two resulting numbers will be displayed by the dimension side.

Measure

Measure:

Enables you to select the type of dimension the Mode choice will apply to: Linear or Angular.

Value above

Value above:

Enables you to determine the upper tolerance value for the linear or the angular dimension, depending on the current choice in the Measure drop-down list.



The entered value will be used either as a deviation value or to determine the limit value depending on the current choice in the Mode drop-down list.

Value below

Value below:

Enables you to determine the lower tolerance value for the linear or the angular dimension, depending on the current choice in the Measure drop-down list.



The entered value will be used either as a deviation value or to determine the limit value depending on the current choice in the Mode drop-down list.

Size ratios

Size ratios

The options available in the Size ratios area enable you to define sizes for parameters like Size, ISO symbol used while specifying tolerances, and Vertical value gap, etc.

Size

Size:

Enables you to specify the tolerance Size as a ratio with respect to the dimension text size.

ISO symbol

ISO symbol:

Enables you to specify the size of the ISO symbol as a ratio with respect to the dimension text size. The ISO symbols are generally used to represent the dimension Text, Geometric Tolerances, etc.

Vertical value gap

Vertical value gap:

Enables you to specify the Vertical distance between two tolerances, as a ratio with respect to the text size.
