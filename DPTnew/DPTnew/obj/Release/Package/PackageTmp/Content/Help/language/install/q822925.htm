Script error occurs when trying to get help on a command

On some installations the following error may occur when trying to access the contextual help:
"line:12 Char: 1 error: object doesn't support this property or method code".

This could be due to a problem with the Microsoft® Internet Explorer Q810847 security update required to address a security vulnerability in Help. However, Q810847 affects the normal behavior of Help. The security update 811630 restores normal Help functionality after installing Q810847. Download it and install it to fix the script error problem.
