Drawing Environment

Commands of general utility in the drawing environment.

In the File menu:

Drawing from Current Model Enables you to create a new two-dimensional drawing from the current 3D model.
Open Reference Model Enables you to open the reference model of the current drawing.
Load Layout Configuration Enables you to call up a view layout that has been saved.
Save Layout Configuration Saves the way the views are arranged in a drawing.

In the Modify Drawing View menu:

Regenerate Regenerates a selected two-dimensional drawing view and all of its derived views.
Move Enables you to move a two-dimensional drawing view and all of its derived views.
Align Enables you to align one view with another.
Disalign Enables you to release a selected view from its alignment constraint with another view.
AutoPlace Dimensions Restores dimensions to their default positions in the selected drawing view.
Unhide Dimensions Axes Edges Enables you to restore the visibility of axes and dimensions previous hidden by using the Hide Dimension command.
Hide Dimensions Axes Edges Hides selected axes or dimensions in a view so that they are no longer visible.
Partial Break on View/Section Enables you to modify the partial break applied on a view/section.
Detail View Enables you to modify the existing detail view.
Delete Breaks on View/Section Enables you to delete the breaks on any view or section.
Multibreaks on View Enables you to modify the multiple breaks applied on a view using Multibreaks on View command.

In the Edit menu:

Regenerate Drawing Regenerates the drawing by rebuilding all the existing views (available only when the model has been modified).
Edit Reference Model Enables you to change the reference model of the current drawing.

In the Tools menu:

Rebuild Drawing Regenerates the drawing by rebuilding all the existing views (always available).
