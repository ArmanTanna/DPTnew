Entity Properties

Category: Dimension-Symbol

Current

Current:

Enables you to set the current style by selecting it in the list of all the available ones.

Symmetric

Symmetric

Enables you to include a Symmetric symbol as part of the dimension.

  • When selected, the chosen Symbol is displayed.
  • When not selected, no symbol is not displayed.

Show symmetric symbol

Show symmetric symbol

The options in this area control the symmetric symbol that can be shown in symmetric dimensions.

Symbol

Symbol:

Enables you to determine the Symbol used to indicate a symmetric dimension, applied when the Show symmetric symbol check box is selected. The following options are available in the drop-down list:

Slash
Ballpoint
Square
Integral Sign
Filled ballpoint
Filled square
Filled slash

Linear

Linear

The controls in this area enable you to choose a proper Symbol to be shown along with the Linear dimension.

Symbol

Symbol:

Enables you to include a proper Symbol as a part of the Linear dimension. You have the following options in the drop-down list:

None Enables you to show None of the available symbols along with the dimension.
DIA before text Enables you to include the DIA symbol before the Linear dimension.
DIA after text Enables you to include the DIA symbol after the Linear dimension.
Diameter symbol before text Enables you to include the Diameter symbol (fi) before the Linear dimension.
Diameter symbol after text Enables you to include the Diameter symbol (fi) after the Linear dimension.
Spherical diameter symbol Enables you to include the Spherical diameter symbol along with the Linear dimension.
Square symbol before text Enables you to include a Square symbol before the Linear dimension text.

Diameter

Diameter

The controls in this area enable you to determine the format for Diameter dimensions.

Symbol

Symbol:

Enables you to choose a proper Symbol to be attached along with the Diameter dimensions. These are the following symbols available in the drop-down list:

None Enables you to show None of the available symbols along with the dimension.
DIA before text Enables you to include the DIA symbol before the Diameter dimension.
DIA after text Enables you to include the DIA symbol after the Diameter dimension.
Diameter symbol before text Enables you to include the Diameter symbol (fi) before the Diameter dimension.
Diameter symbol after text Enables you to include the Diameter symbol (fi) after the Diameter dimension.
Spherical diameter symbol Enables you to include the Spherical diameter symbol along with the Diameter dimension.
Square symbol before text Enables you to include a Square symbol before the Diameter dimension text.

Angle

Angle:

Enables you to specify the Angle at which the Diameter dimensions are drawn.

Arc length

Arc length

The controls in this area enable you to determine the format for Arc length dimensions.

Symbol

Symbol:

Enables you to choose a proper Symbol to be attached along with the Arc length dimensions. These are the following symbols available in the drop-down list:

Arc symbol before text Enables you to include the arc symbol before the arc length dimension.
Arc symbol after text Enables you to include the arc symbol after the arc length dimension.
None Enables you to show none of the available symbols along with the dimension.

Taper dimension

Taper dimension

The options in this area control the taper symbol that can be shown in taper dimensions.

Show symbol

Show symbol

Enables you to include a Taper symbol as part of the dimension.

  • When selected, an appropriate symbol is displayed.
  • When not selected, no symbol is displayed.

Radius

Radius

The controls in this area enable you to determine the format for Radius dimensions.

Symbol

Symbol:

Enables you to choose a proper Symbol to be attached along with the Radius dimensions. These are the following symbols available in the drop-down list:

None Enables you to show None of the available symbols along with the dimension.
R before dimension Enables you to include the R symbol before the Radius dimension.
R after dimension Enables you to include the R symbol after the Radius dimension.
R before dim., no space Enables you to include the R symbol before the Radius dimension without any space between the symbol and the dimension text.
R after dim., no space Enables you to include the R symbol after the Radius dimension without any space between the symbol and the dimension text.
Spherical radius symbol Enables you to include the Spherical radius symbol along with the Radius dimension.
Controlled radius Enables you to include a Controlled radius symbol for the Radius dimension.

45° Chamfer

45° Chamfer

The controls in this area enable you to determine the format for chamfer dimension symbol.

Symbol

Symbol:

Enables you to specify the symbol representation to be attached along with the Chamfer (45º) dimensions. These are the following symbols available in the drop-down list:

C distance Includes the C symbol before the distance.
45° X distance Includes 45º X symbol before the distance
Distance X 45° Includes 45º X symbol after the distance
Distance X distance Includes X symbol in between two distances (in the format: Distance X distance )
