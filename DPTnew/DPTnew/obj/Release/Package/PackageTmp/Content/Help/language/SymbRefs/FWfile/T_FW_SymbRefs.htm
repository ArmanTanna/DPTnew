InsertSymbolic ReferenceNew


 

The New command enables you to declare any geometric entity created by a parametric operation to be a Symbolic Reference.

You can attribute a Symbolic Reference any time and to any of the following parametric entities:

  • solid/component faces, datum planes
  • solid/component edges, datum axes
  • solid/component vertices, datum points
  • profile curves and vertices

Typically, Symbolic References are used when you move or replace objects, that is in mating events and as parametric references in profiles and components.
