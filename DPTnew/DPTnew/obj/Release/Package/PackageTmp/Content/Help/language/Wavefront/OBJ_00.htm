Conversion to and from the Wavefront format

Wavefront's Advanced Visualizer's object file describe geometries along with their attached properties. These files can be formatted in either ASCII(.obj) or binary(.mod). This documentation and this software are only concerned about the ASCII format and mesh objects. This document describes how to convert Wavefront OBJ files to this software format and vice versa.

Introduction

The application enables you to read and write files in the Wavefront format.

Do the following:

  • To convert from the Wavefront format to this software format, use the Open or Insert From File command. In the file dialog box, under Files of type, select the type of file you wish to convert from by selecting the appropriate file format (".obj").
  • To convert to the Wavefront format, use the Save or Save As command. In the Save dialog box, under Save as type, select the type of file you wish to convert to by selecting the appropriate file format (".obj").

Conversion is controlled by the parameters of the OBJ Conversion Options dialog box opened by selecting Options. You can save as many different sets of parameter values  — to be used in different conversion situations —  as you like to the Settings library.
