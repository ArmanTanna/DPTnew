Unhiding Dimensions, Axes or Edges

To unhide the dimension, axes or edges in a drawing view, start the Unhide Dimensions Axes Edges command and proceed as follows:

1 Click to select a view in which the entities were previously hidden. As soon as you select the view, all the entities become temporarily visible (depending on the type of entity selected from the Entity type: drop-down list from the selection list).
If you were to end the command at this point, all the entities would immediately become hidden again. Note also that if you preselect a view, the program skips this step.
2 The program prompts you to select the entity that you want to unhide.

The command enables you to also Unhide texts, hidden using the Hide Dimensions Axes Edges command, if the Entity Properties - Drawing View - Model entities visibility filter - Text option has been selected from the Options/Properties dialog box.

3 Specify the entity type you want to unhide by selecting from the Entity Type drop-down list:
  • Dimensions — to unhide dimensions
  • Axes — to unhide axes
  • Edges — to unhide edges
4 Click to select each entity you want to unhide. As you select entities to be unhidden, the status bar updates to indicate the number of hidden entities. The dimensions, axes or edges are unhidden and are indicated using the same color as the selected view. Note that you can also use a window selection to select several dimensions, axes or edges in single action.

Note
Only for dimensions, you can introduce the additional pick modality by feature, by selecting the Feature Dimensions check box.
  • When selected, you can pick all the dimensions linked to the selected feature.
  • When not selected, you can pick only individual dimensions.
