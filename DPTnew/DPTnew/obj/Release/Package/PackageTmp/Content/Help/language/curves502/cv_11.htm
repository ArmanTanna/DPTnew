Inserting curves on surface boundaries

After starting the Boundary Curve command, proceed as follows.



1 In the Boundaries drop-down list, select the method to be used to create the boundary curves:
All to create all the boundary curves of the surfaces you will select
Select To create only the curves corresponding to the boundaries you will select.
Free Boundary (Applies to solids only) To create the curves corresponding to the free boundaries of the selected faces belonging to the same solid: free boundaries are the ones that are not common to more than one of the selected faces. The resulting curves will be the free boundaries of the selected set of faces.


2 Further selections depend on the method you chose:
  • If you chose the All method, click on the surfaces whose boundary curves you want to create:



    To change the selected surfaces, right-click on Surfaces, select Reset in the context menu and perform selection again.
  • If you chose the Select method, click on the boundaries you want to create curves on:



    To change the selected boundaries, right-click on the Boundaries selector item, choose Reset in the context menu and perform selection again.
3 Click or to confirm your selections and create the boundary curves.
 
Method: All Method: Select

Click to discard your selections.
 
Curves On Surfaces
The Insert Curve on Surface check box controls the type of the resulting curves, enabling you to create curves of the "Curve on Surface" type (See " Curves on Surfaces" for details).
  • When it is selected, the resulting curves will be of the Curve on Surface type. In this case, if the surface is modified, the curves will follow the change.
  • When it is cleared, the resulting curves will be NURBS and they will not follow any surface change. In this case, an additional option is available under More Options: High quality approximation.
    • When the check box is selected, the resulting curves will be G3 or G4 continuous and will have higher degree.
    • When the check box is cleared, the resulting curves will have degree 3 and continuity 2.
