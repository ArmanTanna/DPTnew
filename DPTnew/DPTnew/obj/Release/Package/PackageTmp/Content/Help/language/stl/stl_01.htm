Opening an STL file

To convert from the STL format to think3, do the following:

  1. Select Open or InsertFrom file
  2. In the file dialog box, under Files of type, select STL Format (*.stl)
  3. Select the file you want to open
  4. Click Options to open the Settings tab of the STL Options dialog box and set the parameters controlling the conversion process as you like
  5. Click the Open button to open the file (no preview is available for STL files: the file icon will be displayed in the preview frame).
