Category: Sheet Metal-Bend Reliefs

Bend Reliefs

Bend Reliefs

The controls in this area enable you to define the relief parameters affecting bends.

Enable

Enable

If the bend is to be created shorter than the selected edge, when this check box is not selected the object is just cut along the bend boundaries and no reliefs are created; if the box is selected reliefs are created instead.

Type

Type:

Enables you to select the type of relief to be created when creating bends:

Rectangularrelief has a rectangular shape defined by a width and a height
Oblongrelief has an oblong shape defined by a diameter and a height

Remove remnant

Remove remnant

When this check box is selected, you can enter the minimum size of the remnant into the adjacent box. When the size of the remnant is lower than this value, the remnant is removed.

Remove remnant NOT selected
Remove remnant selected

Width

Width:

Enables you to specify the width of the relief (when it is rectangular).

Height

Height:

Enables you to specify the height of the relief.
