Inserting Smart Objects from the local file system

1 In the Look in selection box, select the folder containing the document to open (initially, the model directory chosen when installing the think3 application is proposed). You can navigate through your file system like in any Exploring window of the Microsoft® Windows operating system.
A list of all documents containing Smart Objects located in the selected folder will be displayed in the box below.

2 Click on the Smart Object document you want to open. The document name will be displayed in the File name box. If available, a preview image of the file contents will be displayed in the frame on the right.
3 Click the Open button to open the document.

Proceed as described in Inserting Smart Objects
