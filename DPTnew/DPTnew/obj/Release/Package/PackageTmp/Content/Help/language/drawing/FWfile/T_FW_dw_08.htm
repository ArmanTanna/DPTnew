InsertDrawing ViewsExploded

The Exploded View command enables you to insert a two-dimensional exploded view of a 3D assembly model into the current drawing. When you create a 3D explosion of an assembly, a special type of exploded view Visual bookmark is created in the VB tree under the Exploded Assemblies folder. It is distinguished from the normal bookmark with a different icon.
