Creating Centreline Axes

To insert the centerline axis, start the Centerlines command and proceed as follows:

  1. Select the circle/arc.



  2. In the Axis angle mini-dialog box, type the value of the angle relative to the X-axis and press Tab/Enter.



  3. The centerlines will be created at the specified angle.





Note
The Axis angle mini-dialog box is available only when cursor is placed on a circle or arc but not in case of selecting generic 2D entities that are edges of some axial surface in a model.
