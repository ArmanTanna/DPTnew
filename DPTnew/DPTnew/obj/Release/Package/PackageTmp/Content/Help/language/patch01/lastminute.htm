New Chain Selection Process

In this version, the new Select chain of connected curves check box has been introduced in the System Options-Input-General category in the Options/Properties dialog box. This check box enables you to select a chain of curves in two ways using the Select Chain command.

  • When the Select chain of connected curves check box is selected, the Select Chain command selects curves which are geometrically connected to each other
  • When this check box is cleared, the curve chain is selected if it appears to be connected from a particular orientation of the view. These curves may not be geometrically connected.
Enhancement in the System Options

In this version, the new Entities with history area have been introduced in the System Options-Output category in the Options/Properties dialog box. The two check boxes under this area are as follows:

  • Show history index: It enables you to hide/unhide the event number in the Model Structure.
  • Show geometric data: It enables you to hide/unhide the geometric data of the feature in the Model Structure. For example, in case of the counter drilled hole it displays head diameter, head depth, and diameter of the hole (e.g. D=10, Head=40*20).

The Show history index option is applicable for all commands, however, the Show geometric data is applicable only to Hole command.

The new Reset to Default option

A new Reset to Default option has been introduced in the context menu (right-click either on the Light Studio or Environment part in the Rendering tab) in the rendering environment. This option enables you to set the lighting and the environment to the default settings of the application.

This option is also used in situations when an old *.e3 file from a previous version is opened in this version. In this case, the Reflection: will be automatically set to zero in the Document Properties - Ground category. Now by using the Reset to Default option, you can visualize the model with the current default reflection and environment.

The new FUTABA catalog in Mold Design

A new FUTABA catalog is available with Mold Design. This catalog can be installed by the same procedure as followed for other mold catalogs like D.M.E etc.

The new Part Lists check box for configuration files

A new Part Lists check box has been introduced under the Configuration Files node available in the Configuration Options dialog box, which is displayed on starting the ToolsConfiguration ManagerSave command. This option enables you to include the Part List settings (made using the FormatOrganizePart List command) in the configuration file.

Explicit dimension type declaration in the 2D profile environment

In this version, the enhancement of explicitly declaring the dimension type is also applicable to the 2D profile environment. Refer the "New Drafting & Dimension Functionalities and Improvements" document for more details.
