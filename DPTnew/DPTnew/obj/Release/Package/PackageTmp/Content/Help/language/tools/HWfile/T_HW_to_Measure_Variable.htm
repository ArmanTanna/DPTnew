Spreadsheet - Defining a Measure variable

To accomplish this task, load the file "measure_variable.e3".

  1. After starting the Spreadsheet command, enter M_Variable under Name column.


  2. Click on the variable row as shown.



    Click Assign to assign the variable to a dimension.


  3. Select the dimension as shown.



    The Type column changes to Measure.



    The variable takes the value of the dimension.
