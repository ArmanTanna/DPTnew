InsertSurfaceReCapping

The ReCapping command enables you to create a capping surface by using a mesh and some limiting curves as the only input. On applying the command, you get a quality surface representing this mesh inside the defined limits.

On starting the command, you need to select the reference mesh for the Mesh selector. The capping surface will be created to match this mesh.

For the Limits selector, select the boundaries that will define the bounds for the capping surface.

In case the right projection of the boundary on the mesh for creating the capping surface is not found, a Show Warnings () button is displayed besides the Limits selector. Click on it to view the warning.

Select the Preview button to display the preview of the capping surface.

Additional settings can be made under More Options.

The Hide check box enables you to hide the mesh during the preview.

  • When selected, the mesh is made hidden during the display of the capping surface preview.
  • When cleared, the mesh is displayed along with the capping surface preview.

Select the Quality Checks node to perform quality checks on the capping surface.
