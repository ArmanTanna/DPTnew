Excluding some parts from the drawing of a model

In some situations you might need to create a drawing of a model excluding from it some standard parts (for example screws or washers). You can obtain the desired result by properly creating a Custom view. You can proceed as follows.

Suppose you need to create a drawing of your model hiding the screws it contains.

1 Create a custom view of your model. In the Model Structure panel of the Custom view definition dialog box:
  • Right-click on the icon of the solid to be displayed



  • Select Make Entity the Only One Visible in the context menu.
The selected object will be highlighted, while the other objects will be grayed out.
2 Select OK to define the view. The created view will display only the object you selected.
