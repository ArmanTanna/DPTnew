New Licensing System Guide for ThinkDesign 2015 (and successive versions)

ATTENTION: For details on licenses up to ThinkDesign 2014.1, please read this document.

Since USB dongles are no longer managed and supported by DPT, any reference to them in this guide must be considered obsolete.

1. Introduction

With the release of ThinkDesign 2015.2, we have introduced Sentinel, a new licensing system developed by the company Safenet, which will also result in the standardization of node-locked licenses, dongles and network/floating licenses.

think3 License Administrator (cladmin.exe), Machine ID and FlexNet licenses will not be used anymore.
In addition, ThinkDesign will not start in demo mode if no license is registered. In fact, after the splash screen, the program will quickly shut down, showing this error dialog box:



2. User Licenses

The first time you install ThinkDesign on your PC, the service Sentinel Run Time Environment will be installed as well. After running the program and the "Valid license not found" error message, you will be redirected to the "new License Administrator", which is called Admin Control Center (ACC).
If the service is correctly installed, you should be able to see a page with the Safenet logo.




  1. To ask for a ThinkDesign license, you need first to generate the .c2v (Customer-to-Vendor) file.

    a. Instructions to generate the .c2v file in case your license is related to a single PC
    b. Instructions to generate the .c2v file after receiving the new dongle

    a. License related to a single PC

    Run the executable file DPTGet.C2V.exe that is in the installation folder of ThinkDesign (ex: C:\program files\think3\thindesign 2017\common\utilities).
    The shell window will appear indicating the folder where the .c2v file has been saved, which usually is the Temp folder.
    The file will be named with your machine name, let's say "DellTower5810.c2v".

    If you have difficulty in finding the .c2v file, just follow these steps: click Start, select the Run command. A dialog box with a search bar will appear, type "%temp%" and then click "Ok".
    In this way, the Temp folder will open. Sort files by the last modified. The first file on the list will be the .C2V file you are looking for.


    b. New dongle (hardware key):

    After installing ThinkDesign, insert the dongle into the USB slot and the dongle should light up.
    In the Admin Control Center (ACC), select Sentinel Keys in the left-side menu. A table will appear.
    In the row where the hardware key has a Driverless Configuration, select the C2V button in the Actions column (the Key ID should match the one on the label of the key sent by DPT).



    Then click Download .C2V File. If you didn't change your default download location, the file you've downloaded will be automatically saved in the Downloads folder.




  2. Send an email to pwdrelease@think3.eu attaching the .c2v file you generated.

  3. You will receive an email with (License Certificate) from DPT Licensing that provides you with a file named xxx.v2c.
    xxx is the license Key ID, while the extension .v2c stands for Vendor-to-Customer. Save this xxx.v2c in any temporary folder.

  4. Attach the xxx.v2c in the ACC.
    You can launch the ACC from ThinkDesign installation folder by running the Admin_Control_Center URL, or you can start ThinkDesign and then be redirected to the ACC as seen before. You can launch it also by typing in your Internet browser http://localhost:1947.



    In the Update/Attach section, you can apply the .v2c file to activate the license.



    After this, you will see the license in the Products section. In the example below, it's a TDProfessional 2015.2 valid only for 30 days.







3. Updating the Product Key with a renewal license

In case of product updates or renewals of the maintenance contract, we will generate new licenses to be added on the same Product Key. Simply send an email to pwdrelease@think3.eu and let us know which user needs a new password. We will send you another .v2c file.


4. Floating Licenses

The floating licenses are linked to a specific license server.


Server Installation

  1. Download the "Floating Licensing Installer" from the Download area of DPT3Care website and run the executable file.



  2. Insert the dongle into the USB slot and the dongle should light up.
    In the Admin Control Center (ACC), select Sentinel Keys in the left-side menu. A table will appear.
    In the row where the hardware key has a Driverless Configuration, select the .c2v button in the Actions column (the Key ID should match the one on the label of the key sent by DPT).



    Then click Download C2V File. If you didn't change your default download location, the file you've downloaded will be automatically saved in the Downloads folder.




  3. Send an email to pwdrelease@think3.eu attaching the .c2v file you generated.

  4. You will receive an email with (License Certificate) from DPT Licensing that provides you with a file named xxx.v2c.
    xxx is the license Key ID, while the extension .v2c stands for Vendor-to-Customer. Save this xxx.v2c in any temporary folder.

  5. Attach the xxx.v2c in the ACC.
    You can launch the ACC from ThinkDesign installation folder by running the Admin_Control_Center URL, or you can start ThinkDesign and then be redirected to the ACC as seen before. You can launch it also by typing in your Internet browser http://localhost:1947.



    In the Update/Attach section, you can apply the .v2c file to activate the license.



    After this, you will see the license in the Products section.




Client Installation

First of all, you need to set the system environment variable DPT_LIC_HOSTNAME.

Control Panel ⇨ System ⇨ Advanced System Settings

The window opens, select now the Advanced tab and click the Environment variables button.
In the System variables section, click the New button.

DPT_LIC_HOSTNAME corresponds to the Variable name and it should be set according to the following table.

Variable value Description
LOCALHOST ThinkDesign doesn't use floating licenses, only local licenses
Server_Name ThinkDesign uses only floating licenses


If you use on the same PC both local and floating licenses, you don't need to define the DPT_LIC_HOSTNAME variable.
