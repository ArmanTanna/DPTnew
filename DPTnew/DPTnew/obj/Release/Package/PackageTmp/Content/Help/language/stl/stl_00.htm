Conversion to and from the STL format

STL or stereo lithography format is an ASCII or binary format used in manufacturing as the main industry standard for the rapid prototyping system. This document describes how to convert STL files to this software format and vice versa.

Introduction

The application enables you to read and write files in the STL format.

Do the following:

  • To convert from the STL format to this software format, use the Open or Insert From File command. In the file dialog box, under Files of type, select the type of file you wish to convert from by selecting the appropriate file format (".stl").
  • To convert to STL format, use the Save or Save As command. In the Save dialog box, under Save as type, select the type of file you wish to convert to by selecting the appropriate file format (".stl").

Conversion is controlled by the parameters of the Settings tab of the STL Options dialog box opened by selecting Options. You can save as many different sets of parameter values  — to be used in different conversion situations —  as you like to the Settings library.

Help on parameters controlling the conversion process
You can easily display a short description of each parameter controlling the conversion process by issuing the help request as follows:
  1. Click on the button in the top-right corner of the dialog box displayed by selecting the Options button.
  2. Click on the item of the dialog box corresponding to the parameter you need help on.
