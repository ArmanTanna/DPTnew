Entity Properties

Category: Drawing View-Flat Pattern

Show
Show

The options in this area control the visibility of the sheet metal parameters in the flat pattern view(s).

Bend lines
Bend lines

When selected, bend lines are displayed in the flat pattern view. When not selected, bend lines are not displayed.
Bend data
Bend data

When selected, bend data are displayed in the flat pattern view. When not selected, bend data are not displayed.
Supplementary angle
Complementary Angle

When selected, the complementary angle of each bend is displayed in the flat pattern view. When not selected, the complementary angle of each bend is not displayed.
Angle precision
Angle Precision

It enables you to specify the precision value up to five decimal digits of the bend angle (bend data) in the flat pattern view.
Remove reliefs
Remove reliefs

When not selected, reliefs are displayed in the flat pattern view. When selected, reliefs are not displayed.
Relief radius
Relief radius

The options in this area control the reliefs of the flat pattern view. The radius of the circle with the center on the end of each bend line which completely encircles the relief curves is known as relief radius.

Length
Length:

Enables you to define the absolute radius of the circle.
Thickness*factor
Thickness*factor:

Enables you to define the the radius of the circle by multiplying the factor with the thickness.
Angle precision
Forming tools:

Select the type of representation for forming tools in the flat pattern view:

  • With thin lines
  • Inherit from view
  • Remove (no representation)
