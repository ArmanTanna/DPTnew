Creating a new “empty” toolbar

To create a new empty toolbar, do the following:

  1. Select ToolsCustomize from the menu bar to open the Customize dialog box.

  2. Move the mouse pointer into the Graphics Area and right-click.

  3. Choose New Toolbar from the toolbar popup.

  4. The program creates an empty toolbar, as shown below.



  5. You can now add tool buttons to the empty toolbar. Click here for details.
