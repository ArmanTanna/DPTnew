System Options

Category: Startup

Startup dialog box

Startup dialog box

The controls in this area enable you to to customize the application startup.

Show at startup

Show at startup

When the check box is selected, the thinkdesign dialog box is displayed on each startup of the application from which you can choose to open a type of document template.

Start with

Start with

The controls in this area determine the type of document opened automatically when you start the application, and how that document is displayed.

No document

No document

When selected, no document is displayed when you start the application.

New model

Error Report

When selected, a template file in the model environment is automatically opened when you start the application.

New drawing

Error Report

When selected, a template file in the drawing environment is automatically opened when you start the application.
