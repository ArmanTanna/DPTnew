The Remove from Non-Sectionable command enables you to define an entity as sectionable in a Custom view, after you have defined it as non sectionable using either the Add to Non-Sectionable command or the Select Non-Sectionable Solids tool.

To define the entity as non sectionable again, after the entity has been added to the set of the sectionable entities, use the Add to Non-Sectionable command.
