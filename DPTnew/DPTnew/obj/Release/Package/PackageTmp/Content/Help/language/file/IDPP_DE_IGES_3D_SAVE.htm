Category: Save

Save

The controls in this area determine general aspects of the conversion from think3 into the selected format

Convert datum entities

Controls the conversion of datum entities. When this check-box is selected, datum entities are converted.

Save JAMA-IS format

When checked, loads the optimized settings for the JAMA-IS format

Set tolerance

When not selected, the tolerance value used for the conversion is the one that is currently specified for the model. The model tolerance value is defined in the Tolerance edit box under the Precision area of the General - Advanced category of Document Properties. When selected, the tolerance value can be specifically defined in the corresponding edit box.

Save colors in IGES format

On selecting this check box, the true color of the entities in the ThinkDesign file is retained while saving in the IGES application. On clearing this check box, the color mapping is done as per the setting of the 'Map ThinkDesign colors to IGES colors' option.

Map ThinkDesign colors to IGES colors

The IGES and ThinkDesign have their own set of color index, and a number is associated with each color. On selecting this check box, the ThinkDesign color number is mapped to the same color number in IGES. For example, if 28 represent red in ThinkDesign format and in IGES 28 represent blue, the red ThinkDesign entities will change from red to blue on saving the ThinkDesign file in IGES format. On clearing this check box, the colors are mapped according to the similarity of shades. Hence, the color numbers might change. Note that this check box is available only when the 'Save colors in IGES format' check box is cleared.
