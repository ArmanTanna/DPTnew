Configuration Manager e-Learning list

In this web training you will learn how to use the configuration manager. It allows centralized management of ThinkDesign configuration settings in companies with several installations of ThinkDesign. You can also move configurations to newer ThinkDesign versions and among different computers. In this task you will learn how to work with centralized configuration settings, how to copy configurations between computers, how to keep configurations over different ThinkDesign versions, how to create and manage different configurations.
