Modifying the parameter of a curve

After starting the Update Parameters command, proceed as follows.

1 Select the curve to be modified.



As soon as the curve is selected, a handle and a mini-dialog box (Start parameter, End parameter) show up at each of the endpoints.


 
2 To modify the starting endpoint of the curve, move the handle or type the desired parameter value into the Start parameter box.



To modify the second endpoint of the curve, do the same with the corresponding handle or the End parameter box.

.

 
3 Click or to confirm your selections and create the curve.



Click to discard your changes.
