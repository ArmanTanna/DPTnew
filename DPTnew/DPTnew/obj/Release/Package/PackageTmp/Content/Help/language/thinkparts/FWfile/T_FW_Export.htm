The Export command enables you to save a copy of the selected index or catalog to a file with the purpose of distributing it to other thinkparts installations.

This command is located in the context menu available when right-clicking on an index or catalog listed in the browser frame of the Catalog Browser or Catalog Manager.

Upon activation, it silently dumps to a file the selected branch of your tree. Depending upon the workstation speed, the operation may take several minutes to complete. A confirmation message informs you of the file location and name, the latter being built upon the Part Number of the branch being exported. You are then enabled to copy this file onto another workstation and Import it in thinkparts.
The resulting file will contain a single catalog or a complete structure of catalogs and indexes depending on the type of object selected in the catalog tree.

The Export command is not designed to export the thinkteam information associated with the codified parts, namely those to which part numbers have been assigned. If you need to export such information as well, please refer to the Export Partnumbers command.
