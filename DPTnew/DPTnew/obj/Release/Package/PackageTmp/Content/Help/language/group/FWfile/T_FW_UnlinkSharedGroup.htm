ModifyGroupUnlink


 
The Unlink Shared Group command enables you to modify an instance of an existing shared group (either locally defined or X-Reference (groups) without affecting the other instances. A new local shared group is created as a result of the unlinking operation.

When you start the command, you are prompted to select a shared group. You can assign a new name to the unlinked group by typing it into the Name box (default is "sharedGroup#", where # is a sequence number automatically increased by the program).

If the selected group is an X-Reference (groups), it will be changed into a local shared group.


The Unlink Shared Group command can also be started by right-clicking on a shared group in the drawing or on its icon in the Drawing Structure and selecting Unlink in the context menu.
