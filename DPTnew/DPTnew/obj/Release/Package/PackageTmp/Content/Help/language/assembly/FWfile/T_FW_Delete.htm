The Delete command enables you to delete an existing mating as well as the single constraint within the mating.

The command is available from the context menu displayed when right-clicking the selected mating or single constraint in the Mating Structure.

Another way to delete a mating/constraint is simply selecting it in the Mating Structure and entering the "Cancel" key.
