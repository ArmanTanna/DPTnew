When inserting a welding section, select from the drop-down list the type of surface of the weld:

Flat
Convex
Concave

When inserting a welding symbol, select from the drop-down list the type of backing strip, if any:

None
No backing strip required.
Permanent backing strip
The application of a permanent backing strip is required.
Removable backing strip
The use of a removable backing bar is required.
