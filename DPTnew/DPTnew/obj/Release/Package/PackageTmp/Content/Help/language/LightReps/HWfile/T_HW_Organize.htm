Organizing Light Representations for an X-Reference Component

Suppose you've handled the LReps model/assembly including the a1, a2, a3 X-Reference components

a1 a2 a3

and set different Light Representations for some of them as shown in the following Model Structure:

Now you wish to arrange the different Light Representations for one of the components in Light Representation.

After starting the Organize command — available from the context sub-menu displayed when right-clicking on the X-Reference (component) and selecting the Light Representations command — proceed as follows.

1 From the Organize Light Representations dialog box that is displayed you are enabled to make the following actions.

  • View the list of the Light Representations currently available for the selected component:

    In the above illustration, note that, in addition to the full set of the standard Light Representations, a custom one, named yy, was inserted by the New command.

  • View the representation currently assigned to the component:

  • Add a user provided Light Representation.

  • Edit the name and the description of user provided Light Representations.

  • Remove any Light Representation except the Full Detail representation and the one presently set as current.

  • Apply any representation presently not set as the current one.
2 Once you've arranged the Light Representations for your component, click the Close button that will close the dialog box and end the command.
