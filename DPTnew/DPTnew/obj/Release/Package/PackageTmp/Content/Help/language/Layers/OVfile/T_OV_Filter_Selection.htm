Layers: Filter and Selection Methods

The layers display and selection can be managed using the following commands:

  • Filter Layers
  • Select Layers

Filter Layers

If the drawing has a limited number of layers it is easy to scan through them. However, if the drawing has a large number of layers, it is sometimes difficult to search through the layers. In such cases, you can use the Filter: Layers button and Filter field. By defining the filters, you can specify the properties and only the layers that match those properties will be displayed in the layer list. The below described predefined filtering options can be activated in two ways:

  • Left-clicking on the Filter Layers button
  • Right-clicking on the layer properties and selecting the Filter Layers command from the context menu.

Using the Filter Layers button, you can define the following type of filters:

All Layers Displays all layers defined in the model/drawing document. The All Layers filter is selected by default, which ensures that all the layers are displayed.
Layers On Displays only the visible layers.
Locked Layers Displays only the locked (that is: locked and locked with snap) layers.
Layers with Entities Displays layers that contain entities.
Selected Layers Displays layers that are currently selected by the user in the layer list.
Filter: Enables to define a filter on name in the following cases:

The Filter result field displays results of the matched filter criteria (that is: filter on name or that of the defined filtering option — All Layers, Layers On, Locked Layers, Layers with Entities, Selected Layers).
Also, you are enabled to:
  • choose the previously entered search string by selecting it from the drop-down list.
  • invert the defined filter (filter on name or the filtering option) by selecting the Invert filter check box. For example, if you have selected the filter to show all layers, none of the layers will be displayed.

Select Layers

Layer selection has been made much easier by providing the new Select Layers command. The below described predefined selection options can be activated in two ways:

  • Left-clicking on the Select Layers button
  • Right-clicking on the layer properties and selecting the Select Layers command from the context menu
