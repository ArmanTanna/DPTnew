Selection List

For a number of commands for which it is helpful, once the command is selected, a Selection List is displayed in the graphics area.

The Selection List reminds you of what you have to do, giving you very natural and easy to understand graphical feedbacks on what you have already done and what you still have to do. It contains:
 

  • The command name as the first item on top:
     
     
  • An item for each element to be selected. Markers can be:
    redfor each element that has to be selected
    yellowfor each element that can optionally be selected

    The program proposes a selection order: the one items are displayed in the Selection List. Such order is not mandatory and you can change it as you like: right-clicking and selecting Continue in the context menu will enable you to move to the next selection item. You can do this as many times as you like, thus completely redefining the selection order your own way.

    You can redefine selections you have already performed at any time by right-clicking on the corresponding item and selecting Reset in the context menu: the previous selection will be cancelled and you will be enabled to perform selection again. If you don't need to completely redefine a selection you have made, but simply to deselect an entity, hold down the CTRL key and click on the entity.

    Each red and yellow marker correspond to a certain number of elements of the same type.

    • If the number of entities to be selected for each item marker is a definite one, as soon as all the required entities are selected, selection automatically moves to next item
    • If the number of entities to be selected is not a definite one, selection has to be ended explicitly by doing one of the following:
      • Right-clicking and selecting Continue in the context menu
      • Clicking on another item of the Selection List
