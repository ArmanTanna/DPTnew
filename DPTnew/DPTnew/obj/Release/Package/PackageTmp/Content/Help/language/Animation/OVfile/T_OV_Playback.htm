Animation Playback

The following documentation explains the use of Animation Playback Control buttons to play the animation effectively.

  • If multiple animations have been created, select the animation from the List of Animations that you need to play.

  • To start the animation playback from the very first step, choose the Jump to Start Position button. This will bring the model to its initial state. Now click the Play button to start the animation playback. The step number being played is displayed in the Current Step display box.

  • The animation playback automatically stops after it is completed. To play the complete animation again follow the steps explained above.

  • Once the animation playback has started the Play button is replaced by the Stop button. Choose this button to stop the animation playback immediately. Choosing the Play button again will resume the animation from the step it was stopped.

  • To start the animation playback from a specific step, specify the step number in the Jump Step edit box. Click the Jump to Following Step button. The model will immediately transform till the specified step. Now choose the Play button to start the animation playback.

  • Use the One Step Forward or One Step Backward buttons to play the animation one step at a time in the forward or backward directions respectively.

  • To transform the model directly to the last step, click the Jump to End Position button.

  • Before playing the animation choose the Minimize button to display a compressed version of the Animation dialog box containing only the playback buttons. This increases the visible portion of the Graphics Area. Click this button again to restore the dialog box size.

  • Player Bookmarks can be used to playback an intermediate potion of the animation.
Note
There is no specific sequence in which the animation playback control buttons should be used. Hence, the above-mentioned steps are not sequential and can be executed randomly as per the need of the animation playback.
