Categories

Categories collect sets of Visual Bookmarks and Annotations and can be used to organize them.

Both in In Visual Bookmarks and Annotations, in fact, you can define categories.

When creating and modifying a Visual Bookmark/Annotation, in the selection list, you can select the category. Select an existing category in the drop-down list or New Category to introduce a new one in the box that is displayed, where you can insert the category name. You can also drag the Visual Bookmark/Annotation to a different category.

In Visual Bookmarks, you can import and export the single bookmark as well as the whole category.

Since categories are attributes of Visual Bookmarks/Annotations, they exist once you've created a Visual Bookmark/Annotation, therefore you cannot create an empty category.
