The Partial Break on View/Section command enables you to define a partial or broken view/sectioned view consisting of user-defined portion(s) of an existing view bounded by spline, rectangular, or circular border(s). The portion(s) of the sectioned view(s) inside the boundary remains/remain in section while the portion(s) outside the boundary appears/appear as a projected view.

Note: The view is created by altering an existing view. The command resets the Undo/Redo buffer, thus preventing you from undoing any previous operations.
