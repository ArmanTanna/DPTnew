InsertCutting Plane LinesPlace on Drawing


The Place on Drawing command inserts a cutting plane line based on a previously defined cutting plane into a selected view. You can then use the cutting plane line to create sectioned views.

Note: This command is not available if the drawing doesn't contain any views or if the drawing or the referenced model don't contain any previously defined cutting planes.
