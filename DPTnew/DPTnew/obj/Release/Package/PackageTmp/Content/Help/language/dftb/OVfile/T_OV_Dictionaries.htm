An Overview of Dictionaries

The application provides you with a series of customizable dictionaries designed to limit the typing of commonly used data and expedite the filling of the data associated with parts and documents. These dictionaries containing standard names are displayed by the program each time you are asked to enter a name or description for any kind of element. The dictionaries are simple lists of definitions in Extended Markup Language (XML) format and are stored in the "Installation folder/thinkdesign/files/dictionaries" folder. They can be modified by using the XMLEditor tool to customize them to your in-house requirements. See the "Customizing a Dictionary" document for further details.
