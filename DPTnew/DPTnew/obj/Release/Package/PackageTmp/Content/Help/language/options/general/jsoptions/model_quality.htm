The models imported into think3 might not be perfect, as they can have issues that can be of topological nature, specific issues on curves, surfaces and points or also geometric problems. All these issues can be checked by the Model Quality.


  • When the Check Model Quality after loading check box is selected, after loading the model using the converter, the program automatically starts and applies the Model Quality command on it to perform the quality check.

    Note
    The quality check is done only for some supported formats; these are STEP, IGES, VDA, Parasolid, CATIA V5 and ProEngineer.

  • When the check box is cleared, no quality check is performed on the loaded model.
