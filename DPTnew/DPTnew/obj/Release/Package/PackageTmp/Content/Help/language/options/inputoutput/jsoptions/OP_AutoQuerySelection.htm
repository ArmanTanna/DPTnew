If selected, when the cursor stops on a position in which the selection trap matches two or more entities, the query selection is automatically started: the Page Up/Page Down keys ( ) is displayed and you can use the corresponding keys to move your selection from one entity to the next or previous one.



You will have to confirm your selection by clicking the mouse selection button.



To disable the automatic query selection, just move the mouse away (see also Select Query).

When the check box is not selected, the automatic query selection is disabled.
