ToolsKeyboard Map


 

The Keyboard Map command displays a list of keyboard shortcuts to the program commands.

When you start the command, the program displays the Help Keyboard dialog box that shows a table with the program commands grouped in functional categories with the associated shortcuts and a brief command description.

On the top of the form the following tools are available:

Print Use this button to print the list of the currently displayed category on your default system printer.
  Category Click and select from the list the category which the commands to be displayed must belong to. Once you have selected it, only commands belonging to that category will be displayed in the list.

The columns in this table display the following information:

Category The name of the category which the commands to be displayed belong to.
The Bound Commands category enables you to display all commands that already have a shortcut assigned; this is the default category shown when you run the command for the first time.
Command The name of the command.
Keys The associated shortcut, if any. In order to customize it please refer to the documentation of the Shortcuts tab of the Customize command.
Description The description of the command.

Click on a column header to order the list by the topics of that column. For example, to order the list by the contents of the Command column in descending order, click on Command. To get the reverse order, click on Command again.
