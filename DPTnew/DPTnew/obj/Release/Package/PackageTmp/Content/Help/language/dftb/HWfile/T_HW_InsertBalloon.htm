Inserting Balloons Manually

After starting the Manual Balloon command, proceed as follows.

1 Select the format you wish to use from the drop-down list in the command selection list.
 
2 Select the reference point on the part to which you need to associate the balloon call-out. Note that the program has highlighted the parts in your drawing that have no balloon associated.
3 Select the balloon call-out origin and then the point where you want the balloon text. Note that if you checked the Start from Reference Point option, you only have to define the point where you want the balloon text.
4 Repeat steps 2 and 3 for all the items requiring the balloon call-outs. Note that when you associate a balloon to a part, the latter is no longer highlighted.
 
5 When all the balloons are placed, click to terminate the command.
