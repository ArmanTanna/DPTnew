Properties: Sheet Metal

Bend ReliefsEnables you to control the parameters containing bends reliefs.
Corner ReliefsEnables you to control parameters governing relief corners.
Flange CornerEnables you to control parameters governing flange corners.
PositioningEnables you to control parameters governing bend and flange positioning.
Rip/CornerEnables you to control parameters governing rips and corner closures.

Assigning variables to Sheet Metal parameters of selected objects

To assign variables to Sheet Metal parameters of an object, proceed as follows:
  1. Right-click on the object
  2. Select Edit Properties
  3. In the box adjacent to the parameter you want to associate the variable with



    type the variable name.

If the variable does not exist yet, it will be created, assigned to the parameter and added to the Spreadsheet. If the variable already exists, it will be simply assigned to the parameter (while the counter of its occurences will be incremented in the Spreadsheet).
