Attaching files to a Smart Object

To attach files to a Smart Object, start the Properties command and select the Preview&Description tab.

Using the File list area, you can attach files to a Smart Object, open them or delete them from the list of the attached files.

  • To open an attached file:
    1. Select it in the list
    2. Select Show
    The selected file will be opened in the application associated with it (if any).
     
  • To attach a file to the selected Smart Object, thus adding it to the list:
    1. Select Add. A file selection box is displayed.
    2. In the Look in selection box, select the folder containing the document to open. You can navigate through your file system like in any Exploring window of the Microsoft® Windows operating system. A list of all documents located in the selected folder will be displayed in the box below.
    3. Click on the document you want to open. The document name will be displayed in the File name box.
    4. Click the Open button to attach the document to the Smart Object.
    The selected document will be displayed in the list of the attached files.
     
  • To remove a file from the list, so that it will no more be attached to the Smart Object:
    1. Select it in the list
    2. Select Delete
    3. Select Yes to confirm its deletion from the list (note that the file won't be deleted, but simply removed from the list and un-attached from the Smart Object).
    The selected file will be removed from the list.
