Adding your own company's logo

To add your own company's logo to the title block, you can use either an image file or a drawing file.

Before inserting your company's logo in the title block, no matter whether it is an image or a drawing file, remove the think3 logo from the drawing of the standard title block copy you have created as described in "How to create customized title blocks".

  • If your company's logo is an image file, do the following:
    1. Select the Insert Image command and import the image file containing your company's logo
    2. Select the Edit Image command
    3. Drag the origin of the image anchor handle to the new position you want the image to be placed in
    4. Resize the image by dragging the circular marker on the higher-right corner to a new position: the image will be enlarged/reduced accordingly.



    Please note that only one image at a time can be positioned in a drawing. So, if your title block contains an image, do not import another image into the drawing. On the other hand, if you try to load the title block in drawing already containing an image, the one used as the company's logo will not be loaded.
     
  • If your company's logo is a drawing file, do the following:
    1. Select the Insertfrom File command.
    2. position your own company's logo into the title block.

    An alternative way

    Another way to add your own company's logo is the following:
    1. Create it as a group
    2. Using the Group Library command, add it to a group library
    3. Remove the think3 logo from the copy of the standard title block
    4. Using the Group Library command again, position the group containing your logo into the customized title block.
