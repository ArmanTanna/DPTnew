Selective Closure of Open Loops

The following illustration explains a situation where you will selectively close a loop present in a open solid (skin) by using the Interactive Modeling command. Open the ISM_selective_closure.e3 file available in the Samples folder of your think3 installation and follow the steps given below.

  1. Invoke the Extend Faces/Close Solid command.

  2. Select the skin for the Base Skin selector.



  3. Set the Mode to Local.

  4. For the Edges selector, select the edges of the open loops to be closed.



  5. Note that even if a portion of the loop is selected, the complete loop gets selected automatically.



  6. Apply the command to extend the faces and close the open loop.
