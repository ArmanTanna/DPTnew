Bending a solid along a profile lying on one of its faces

To accomplish this, task, load file "BendOnProfile.e3".
 
1 After starting the Sketched Bend command, select the profile on which the bend must be created.



To change the selected profile, right-click on Profile, choose Reset in the context menu and select another profile.

The face to be fixed is automatically selected. To change the face, right-click on Face, choose Reset in the context menu and select another face.
 

2 In the Angle mini-dialog box, type the value of the bend angle, or drag the corresponding red handle



3 Click the vertical arrow to invert the bend angle.
4 Click the horizontal arrow to invert the bend direction.
5 Click or to confirm your selections and add the bend.



Click to discard your changes.
