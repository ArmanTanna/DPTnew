License Administrator

The License Administrator utility helps you in managing the license codes you will be provided with our company products. It means you are able to insert, to renew, to check, to export or to import your liceses.

The following contents are fully detailed in the License Information document on the website in the Customer Care area accessible through Login CUSTOMERCARE; in order to sign in you have to supply your personal account as described below.

Personal account

If you do not have a personal account yet to access the Customer Care's website, you can get it in one of the following ways.

  • Apply to your company primary contact (the persons in your company who are in charge of creating new users for our company's Customer Care, of computing licenses and of giving rights to compute licenses to the other company users). They can obtain up to one personal account for each of the users of our company products.
  • Connect to the website (www.think3.eu) and click Access Issues at the login page. Now fill in the form and press Submit to forward your request to the Customer care team.

When you start the License Administrator, a dialog box will show up, displaying a list of all the active company licenses, their Acronym (please note that the number displayed in the acronym is not relevant), their License type and their Expire date and the Machine ID: (in the right bottom corner):




Note that if you set the Options to Network Licenses there will be another column (In use/total) that shows number of licenses in use and the total number of licenses. Only currently active licenses are displayed in the list: expired licenses or licenses computed for a different Machine ID are not displayed in the list (see Help commands below to know how to display them).

To close the utility, select FileExit.

The License codes of our company products are stored to a license file whose name is displayed in the left bottom corner of the License Administrator dialog box.

Help commands

In the Help menu you will find some Diagnostics commands, which can be used in case of trouble with your licenses:

  • DiagnosticsDate of Last Usage — displays a dialog box showing the date of last usage of any of our company product.
    This command can be useful if for any reason your PC's clock has been modified and the date has been moved back. In this case the last usage date is "in the future" if compared with your current system date and thus you will get the following message:

    The current system date is previous to the last application usage
    In this case the temporary licenses are disabled.
    Please check your system date and time zone and if they
    are correct request a password of type S to your provider.
    Otherwise set the correct data and start the application again.

  • DiagnosticsShow Machine ID Column — displays the license code list adding a column where you can read the Machine ID, so that you can check whether it is the correct one (as the Machine ID changes, for example, any time you re-install the operating system).
  • DiagnosticsShow All Licenses — displays the complete license code list, including expired licenses and licenses computed for a different Machine ID.
  • DiagnosticsProperties — displays data about the selected license: product, import/export information, Machine ID of the original machine from which the license was imported.
  • DiagnosticsFLEXlm Version: — displays the version details of the FlexNet you are using. Note that this option is active only when the Options is set to Network Licenses.

You can check the version of the License Administrator you are using by selecting About CLAdmin.
