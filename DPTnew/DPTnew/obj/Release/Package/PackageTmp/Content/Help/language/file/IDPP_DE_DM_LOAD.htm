Category: Load

Load

The controls in this area determine general aspects pertaining to the action of loading a document into the think3 application.

Symbols as non-shared groups

When selected, symbols are converted to non-shared groups.

Only symbols with symbolic texts

When selected, only those symbols that contain symbolic texts are converted as non-shared groups.

Disassemble on loading

The controls in this area determine whether certain entities are converted into their corresponding think3 entities when the file is converted, or broken down into their component entities (such as segments, arcs, and so on).

Dimension/label

Using this selection list it is possible to select which dimensions will be saved as disassembled geometry; None : No dimension will be saved as disassembled geometry, When needed : Only those dimension will be saved as disassembled geometry that cannot be converted preserving 100% of their parameters, All : All the dimensions will be saved as disassembled geometry.

X-Hatch

When selected, dual dimensions are broken down into their component entities.
