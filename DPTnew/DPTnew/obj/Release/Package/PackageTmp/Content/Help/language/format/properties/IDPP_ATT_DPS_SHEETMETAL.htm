The controls in this area control the way bend data are displayed.

text

The boxes in this column enable you to specify the text to be positioned along with the upwards and the downwards bends.

line type

line type

The boxes in this column enable you to specify the line type to be used for the representation of both upwards and downwards bends.

Text position

Text position

The options in this area enable you to determine whether text is to be positioned as a postfix or as a prefix.

Prefix

When this option is selected, text is positioned as a prefix.

Suffix

When this option is selected, text is positioned as a postfix.
