The Debug button expands the Capture & Playback dialog box to enable you to play back the current log file in step-by-step mode, using either context tags or log file line numbers.

The Capture & Playback dialog box then provides the following additional functions:

Play up to User Action

Selecting this button results in the play back of the log file up to the next user action, which is displayed in the corresponding box. Right-clicking this button expands the action box to display a list of user actions. You can then select the user action and play back the log file up to the selected action.
Step-by-Step

Clicking this button plays back the current log file in step-by-step mode. Each time you click this button, the program plays back the number of lines from the log file specified in the corresponding box. You can specify the number of lines to be played back by typing a value in the corresponding box, or by selecting the arrows button (to increase or decrease the number of lines by 10). Each line being executed is printed in the I/O window, preceded by its own number.
Play up to Line

Clicking this button plays back the current log file in step-by-step mode, up to the log file line number specified in the corresponding box. You can specify the line number by typing a value in the text box, or by selecting the arrows button (to increase or decrease the line number by 10).
Show Marker Clicking this button places a white "X" mark on the last graphical input (either a pick or selection) while playing back a log.
