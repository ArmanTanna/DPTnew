The Replace Standard Part command enables you to replace the currently selected parts with one chosen from a thinkparts catalog.

You can find this command in the context menu that displays when right-clicking on a part inserted in the assembly.

The command starts the Catalog Browser and opens the thinkparts catalog where the selected part was originally taken from. The line corresponding to the selected part is highlighted in the catalog query view.
Choose the new part from the catalog and import it in the model.

If you select more than one part before running Replace Standard Part, all components will be replaced with the one selected from the catalog. Also, note that if the selected parts are of different types or measures, the catalog opens on the last one in the selection sequence. This provides you with a quick mechanism to substitute many different parts with a particular one already inserted in your model: select the latter as last one, then start the Replace Standard Part command and simply close the catalog selection window.

The Replace Standard Part command will be correctly completed if you replace a part with another having the same mating methods. If you choose a completely different part, having different mating methods — which doesn't generally make sense — it will be positioned as well as possible, but no mating constraints will be applied to it.


If the command does not behave as expected, please verify that there are no duplicate symbolic matings with the same name.
