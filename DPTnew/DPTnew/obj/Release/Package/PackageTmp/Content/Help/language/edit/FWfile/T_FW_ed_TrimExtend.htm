EditTrim/Extend Curves


 

The Trim/Extend Curves command trims or extends curves according to the method selected in the Mode drop-down list:


First
Trims or extends the first of two selected curves so that it ends on the second selected curve.

The results depend on the points used to select the curves. If the curves intersect at more than one point, the think3 application considers the one closest to the selection point of the first curve and trims the curve so that it ends at that point. If the curves do not intersect, the program extends them along their existing curvature until they do intersect.


Both
Trims or extends the two selected curves so that they have a shared endpoint.

The results depend on the points used to select the curves. If the curves intersect at more than one point, the think3 application considers the one closest to the selection point of the first curve and trims the curves so that they end at that point. If the curves do not intersect, the program extends them along their existing curvature until they do intersect.


After selecting the trimming method, select the curves to be trimmed.
Once the curves have been selected, by checking the Quadrant box you can choose the portions of curves to be kept: the quadrant changes as you move the cursor and the portion of curves which will be kept is highlighted. Click to select the quadrant and apply the command.

Virtual intersections
The command can trim two curves also when they are not lying on the same plane (unless they do not intersect in any plane or view direction). When a virtual intersection is detected, a proper message is displayed. Upon confirmation, the trimming operation is performed along the current view direction.

Applying the command to entities belonging to groups
The command can be applied to entities belonging to groups by using the Select inside Groups command.
