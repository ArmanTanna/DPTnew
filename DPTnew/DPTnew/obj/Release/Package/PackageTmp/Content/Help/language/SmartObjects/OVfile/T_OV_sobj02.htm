Smart Profile

Smart Profiles are Smart Objects composed of a single profile.

Smart Profiles can be added to a model just by dragging them from the Palette.

If a profile is being edited in the current model, the new Smart Profile will be merged and trimmed with it: the result will be a unique profile.

Please note what follows:

  • The redefinition of this type of profile is restricted to two basic operations: adding or removing external references
  • All the external references of the Smart Profile, if any, will be removed at merging time
  • Trimming when merging a Smart Profile is possible only if the outer loop is closed. If this condition is not true merging is performed without automatic trim.
