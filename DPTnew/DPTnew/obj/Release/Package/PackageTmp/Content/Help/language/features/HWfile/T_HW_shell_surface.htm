Creating a solid with a parametric thickness starting from a surface

The Shell command enables you to create a solid with parametric thickness from an open solid (a solid created from the surfaces that bound it using the Insert Solid Make Solid command).

1 Under Mode in the selection list, choose Add thickness.
2 You are prompted to select a solid. Select the surface that you made into a solid. To reselect the solid, right-click on Solid in the selection list, select Reset, and then select the solid again.
3 In the Global Thickness mini-dialog, specify the global shell thickness.
4 Click or to confirm your selections and shell the solid. Click to discard your changes.
