Creating a Tool Button

You can add new buttons to a toolbar and then program them with the think3 application or Graphics Procedure Language commands.

To create a new button, do the following:

  1. Select ToolsCustomize from the menu bar to open the Customize dialog box.

  2. Move the mouse pointer onto the toolbar you want to add a button to and right-click.

  3. Choose New Button from the toolbar popup.




    The new button gets created.
