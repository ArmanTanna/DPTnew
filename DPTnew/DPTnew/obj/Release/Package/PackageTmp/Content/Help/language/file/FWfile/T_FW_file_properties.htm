FileProperties


 

The FileProperties command displays information about the current document in the following tabs:

  • Version Compatibility
    Enables you to check the compatibility of the currently open files with former versions of the application.
  • General
    Includes the file general information, derived automatically.
  • Custom
    Includes additional alphanumeric, customizable information, that can be associated to the model/drawing of parts of the model/drawing (solids, simple entities).
    Click here to define a property class and assign a user defined property UDP to the model/drawing itself or here to remove a property assigned to the model/drawing itself.
  • Preview
    Displays the file preview, when available and the options to define the preview and its details.
  • Reference Model Info
    Includes the reference model name and status.
    This tab is displayed only for drawings created from models.
  • Document Data
    Enables you to edit and inspect the Document Data.
  • Edit Part Properties
    Enables you to edit and inspect the Part Properties pertaining to the top level component.

Either derived automatically and customized or changed, this information is stored in all documents.
