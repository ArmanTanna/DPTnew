Modifying the degree of a curve

  1. Open file "mc_01_02.e3e2". The following curve is displayed:



    2.  
  2. Start the Curve Flexibility command.
     
  3. Select the curve.


     
  4. Select the Show Control Points check box to display the control points associated with the curve.


     

  5. Type 7 in the Degree box and hit the Tab key on the keyboard. A preview of the modified curve is displayed (as you can see the number of control points is higher):


     
  6. Click to apply the change and quit the command.
