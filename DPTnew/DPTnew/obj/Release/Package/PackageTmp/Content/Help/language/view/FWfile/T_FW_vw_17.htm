ViewModifyFit


 

The Fit command increases or decreases the zoom magnification so that the entire modeldrawing fits exactly within the current display.

If the display contains multiple views, the command changes the zoom magnification of all of the views so that the modeldrawing fits within each view at its maximum magnification.

Note: If you start the command within a split view by right-clicking and selecting it from the shortcut menu, the command affects only that view.
