New Profile Functionalities and Improvements

Version 2010.1 introduces the following enhancements in the Profile environment.

  • Improvement in the Constraints representation
  • Enhancement in the 3D profile environment

In case you are not able to see the new constraint representation on opening an old e3 file, you need to upgrade the Graphics Driver of your machine. Also, your machine should have any one of the Graphics Card mentioned in the OpenGL Graphics Accelerators document.

Enhancement in the 3D profile environment

It is possible to create arcs in the 3D profile environment. Hence, the following commands are now enabled in the 3D profile environment.

  • Three-Point Circle
  • Circle given Center
  • Circle given Radius
  • Fillet

Also, the Mode drop-down list is available in the Polyline selection list when working in the 3D profile environment. The options in this drop-down list enable you to create tangent arcs and 3 point arcs.

Also, the Geometrical and Dimensional constraints relevant to arcs and circles are applicable.
