New Drawing Frame and Title Block management

Version 2008.1 introduces a new management of the Drawing Frame and Title Block.

  • Overview
  • New Save as Drawing Frame command
  • New Save as Title Block command
  • New user interface of the Drawing Frame and Title Block command
  • Enhancements in the Drawing category of the Document Properties

Overview

The management of Drawing Frames and Title Blocks has been renewed in this version. Although the inner format of the drawings used as Frames and Title Blocks does not change, so that you can keep using your customized ones, you are provided with new and revisited commands to manage them.


New commands are provided for the creation of Drawing Frames and Title Blocks:

  • Save as Drawing Frame
  • Save as Title Block

The Drawing Frame and Title Block command has been completely renovated and has been moved at the first level of the Insert menu.


The options that enabled you to manage the Drawing Frame and Title Block are still present in the Drawing category of the Document Properties tab, albeit updated.


If you would like to use your existing customized Drawing Frames and Title Blocks, see "Using customized Drawing Frames and Title Blocks from versions prior to 2008.1" in the online help.

New Save as Drawing Frame command

The Save as Drawing Frame command enables you to save the current drawing as a Drawing Frame.


When you start the command, the program displays a dialog box that prompts you for the following information:


  • Name portion, used by the program to build the actual Drawing Frame filename.
  • Description, that the program displays in the list of available Drawing Frames.
  • Size
  • Orientation
  • Anchor point

When you click OK, the drawing is saved as <User folder>\Application Data\think3\thinkdesign\2008.1\files\drwFrames\<Size>_<Orientation>_<Name portion>.e2.

New Save as Title Block command

The Save as Title Block command enables you to save the current drawing as a Title Block.


When you start the command, the program displays a dialog box that prompts you for the following information:


  • Name portion, used by the program to build the actual Title Block filename.
  • Description, that the program displays in the list of available Title Blocks.

When you click OK, the drawing is saved as <User folder>\Application Data\think3\thinkdesign\2008.1\files\drwTBlocks\titleblock_<Name portion>.e2.


Note that in order to add document or component properties or descriptions while designing your custom title block, you are now enabled to use the Text from Dictionary option of the Text command.

New user interface of the Drawing Frame and Title Block command

The Drawing Frame and Title Block command has undergone a review due to the new Drawing Frame and Title Block setup.


The command has been moved from InsertDrawing Details to the first level of the Insert menu and it features a new user interface based on a selection list that enables you to enter the following information:


  • Size
  • X and Y dimensions when you set Size to Free
  • Orientation
  • Scale
  • Drawing Frame, chosen from the list of those delivered with the software and those that you have created with the Save as Drawing Frame command specifically for the current drawing Size and orientation.
  • Title Block, chosen from the list of those delivered with the software and those that you have created with the Save as Title Block command.

The program displays a bounding box preview in order to give you an immediate feedback of the changes to the selection list options.

Enhancements in the Drawing category of the Document Properties

The Drawing category of the Document Properties has been changed to reflect the new management of Drawing Frames and Title Block.


Specifically, note that the former Pathname and AutoLoad options have been removed.
