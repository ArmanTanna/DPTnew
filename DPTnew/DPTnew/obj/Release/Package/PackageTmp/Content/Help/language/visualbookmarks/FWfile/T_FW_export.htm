The Export command enables you to save the Visual Bookmark or the Category collecting a set of Visual Bookmarks in a specific format, .vbk, in order to export it to another computer/document, where it can be loaded by the Import command.

The command displays a file selection box where you can enter the information required to save the Visual Bookmark/Category.

You can perform the command from the context menu displayed when right-clicking the selected Visual Bookmark/Category.

You can view the Visual Bookmarks from the Visual Bookmarks tab on the left pane of your model/drawing document.
