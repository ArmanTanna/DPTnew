Adding a Group to a Library

To add a group to a library, start the Group Library command and proceed as follows.
 
1 Create the group.

2 To switch to a different Group Library, click the corresponding tab. To know how to add a new tab, see "Creating a new Group Library"

3 To add the group to the current Group Library:
  1. Drag it from the drawing



  2. Drop it into the Group Library window. A preview image of the group is immediately displayed in the Group Library window.



An ".e2" file containing the group and with the same name as the group is created in the current Group Library folder. If a group with that name already exists in the current Group Library, a warning message is displayed. Select Yes to replace the existing group with the new one, or No cancel the operation.

  About shared groups
Please note that when you add a shared group to a library by dragging and dropping it from the drawing, the selected shared group in the drawing is either a local group or an X-Reference depending on whether the currently selected option in the Shared group drag&drop area of the Drawing category of the System Options is Local or X-Reference.
