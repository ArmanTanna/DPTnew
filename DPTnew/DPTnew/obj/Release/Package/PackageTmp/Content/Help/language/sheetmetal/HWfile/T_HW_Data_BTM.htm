Creating Sheet Metal Features using Bend Table data

The following illustration explains the procedure of creating Sheet Metal features using the bend table data:

1. Start one of the Sheet Metal feature creation commands, which can be Solid Flange, Extruded Flange, Flange, Bend, Step Bend or Sketched Bend.
2. Make the entity/feature selection as per the prompt.
3. Select Bend table as the Type for the Allowance option of Local Properties/ Properties in the selection list.

Prior to the previous step, you need to unlock the lock of the Allowance option, if it is locked. See Local Properties and the Lock Marker for details.

4. Select the required material from the Material drop-down list, which displays all the available materials in the bend table.
5. If you are using the Solid Flange command, select the Thickness from table check box; the corresponding drop-down list gets activated and displays the available thickness values for the selected material in the bend table. Make the required selection from the drop-down list.
6. Select the Radius from table check box; the corresponding drop-down list gets activated and displays the available bend radius values for the selected material and specified thickness in the bend table. Make the required selection from the drop-down list.
7. On applying the command, the program retrieves the value from the drop-down lists and uses it in creating the Sheet Metal feature.
Note
The From table option is also available in the Sheet Metal category of the Entity Properties dialog box to set the bend table values in the global settings.
