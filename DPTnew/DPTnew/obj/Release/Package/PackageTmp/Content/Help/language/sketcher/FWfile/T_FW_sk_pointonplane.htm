InsertProfilePoint on Plane

The Point-on-Plane Constraint command enables you to constrain point entities in a 3D profile so that they align with a plane.

To apply the constraint, after starting the command, simply select a planar entity for the Plane selector. The planar entity can be a planar face or a datum plane. Next select a point to be placed over the plane. The program immediately applies the constraint and adds the PoP identifier.

The program determines which entities to move based on the other constraints on the profile. If there are no constraints, the program moves the entities that produces the least change to the profile. Note that the command also supports Single or Multiple selection sequence.
