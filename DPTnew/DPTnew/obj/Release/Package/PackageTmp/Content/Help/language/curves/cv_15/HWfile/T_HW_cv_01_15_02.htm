Project a curve in a direction normal to a plane

After starting the Project Curve command, proceed as follows.
 
1 Select the curve to be projected.



End selection by right-clicking and choosing Continue in the context menu.
To change the selected curve, right-click on Curves, select Reset in the context menu and perform selection again.
2 In the Method drop-down list, select Normal.
3 In the On drop-down list, select Plane.
4 In the Plane drop-down list select the method to define the projection plane:
3 Points The projection plane is defined as a plane through 3 points to be specified.
Plane The projection plane is defined as an existing plane to be selected.
Perpendicular to axis & point The projection plane is defined as perpendicular to an axis and a point to be specified.
Select Plane and click on the plane to be used as the projection plane:



A preview of the resulting projected curve is displayed. An arrow shows up to display the projection direction:



To change the selected plane, right-click on Plane, choose Reset in the context menu and select another plane.
5 Click or to confirm your selections and create the curve.



Click to discard your selections.
