Pro/ENGINEER Conversion Options

When loading (see Open) a Pro/ENGINEER file by selecting ProEngineer file in the Files of type drop-down list, the Options button in the file selection box enables you to define the parameters to be used in the conversion process.

A dialog box displaying the following categories shows up:

  • Load
    This category contains the most common parameters to be set when converting from Pro/ENGINEER.
    • Log File
      This category contains parameters controlling the name and location of the log file created during the conversion process.
    • Advanced
      This category contains parameters controlling advanced conversion settings such as the conversion of entities into NURBS.
  • Save
    This category contains the most common parameters to be set when converting to Pro/ENGINEER.
    • Log File
      This category contains parameters controlling the name and location of the log file created during the conversion process.
    • Advanced
      This category contains parameters controlling advanced conversion settings such as the conversion of entities into NURBS.
