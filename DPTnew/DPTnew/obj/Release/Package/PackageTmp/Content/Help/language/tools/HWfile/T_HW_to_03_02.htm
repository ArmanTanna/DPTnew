Spreadsheet - Assigning/deassigning a variable to/from a drive dimension

To accomplish this task, load the file "to_90_02.e3".

To assign a variable to a drive dimension:

1 After starting the Spreadsheet command, click the line number 1 as shown.

2 Click Assign.
3 The program prompts you to select an entity. Click to select the drive dimension to which you wish to assign the variable. As soon as you select the dimension, the variable name appears in place of the dimension value.

4 Click Rebuild to see the effect of the assignment on the model.

To deassign a variable from a drive dimension:

1 Click Deassign.
2 The program prompts you to select an entity. Click to select the drive dimension from which you wish to deassign the variable. As soon as you select the dimension, its associated variable is deassigned and the current dimension value appears in place of the variable name.
