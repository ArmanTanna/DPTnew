Assembly e-Learning list

Below the complete list of self-tutorial videos available on Assemblies. Follow the links to discover all functionalities of Assemblies in ThinkDesign.

  • Introduction to assemblies (e-Tutorial): local and X-ref components, create, replace, derive, cross component and assembly features, matings, component creation in the assembly, exiting component reuse.

  • Introduction to assemblies (e-Training): the task introduces the concepts of assembly modeling in top-down (from assembly to component) and bottom-up (from component to assembly) modes. In the task you will learn how to create a local component, create an external reference, insert an existing component, deal with matings.
