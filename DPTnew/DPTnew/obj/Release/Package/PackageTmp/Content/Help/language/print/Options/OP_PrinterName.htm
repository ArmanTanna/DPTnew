Displays the current printer in use.
To change the printer, click on the drop-down list and select a different one. The list of printers that appears depends on your configuration and may include the following:

  • your local system printers;
  • the think3 application internal printer drivers for BGL, CCRF, HPGL, HP-RTL, PCI-907, HP-PCL peripherals and various graphics formats such as GIF, JPEG, BMP, TIFF and others;
  • the list of the available thinkprint virtual printers if Batch is selected.
