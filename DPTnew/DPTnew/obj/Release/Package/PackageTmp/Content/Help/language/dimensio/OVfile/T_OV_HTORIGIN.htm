Hole Table: Customizing the Symbol of the Origin

In the "files/HoleTable" subfolder of your installation folder you can find the "HT_origin.e2" drawing file, containing the symbol of the origin which all the coordinates in the Hole Table refer to.

  • You can customize the drawing to conform the symbol to your company standards. We suggest to make a spare copy of the file before editing it.
     
  • The drawing size corresponds to a text character height of 1. When the symbol is positioned in the drawing, it is scaled by a factor equal to the current value of the text character height (see Size in the Font category of Entity Properties).
     
  • If the HT_origin.e2 file is missing from the files/HoleTable subfolder of your installation folder, a standard symbol will be used instead.
