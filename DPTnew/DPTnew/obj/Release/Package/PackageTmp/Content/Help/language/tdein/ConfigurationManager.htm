Retaining reserved settings through the Configuration Manager

Configuration Manager "local" parameters are controlled in a separate branch of the registry (Local Preferences) — rather than in the ordinary Saved Preferences branch — so as to prevent parameters related to local configurations from being overwritten by those in the Saved Preferences.

Correct setting management when moving to a new version (CAD managers only)
As a consequence, in order to provide a correct setting management for the new versions, CAD managers are strongly recommended to proceed as follows:
  • First of all clean up the registry. Specifically, remove the Saved Preferences and Graphic branches.
  • Run the think3 application and perform the new parameter setting as required/desired.
  • Stop the application.
Only at this point it is possible to obtain the new configuration from the Saved Preferences branch (otherwise some parameters may be retained from former installations).
