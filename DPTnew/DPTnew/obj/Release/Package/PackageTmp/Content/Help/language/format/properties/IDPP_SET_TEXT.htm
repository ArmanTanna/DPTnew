Display

The controls in this area determine the way dimensions and texts are displayed in the views.

In all views

When selected, text and dimensions are visible in all views.

Only if view angle <

Only if view angle <

When selected, text and dimensions are visible in a given view only if the angle between the view direction and the placement plane of the dimension or text is greater than the value in the adjacent box.

Reference view

Reference view:

Specifies the identification code number for the view to which the dimensions refer; all the dimensions entered will lie in the projection plane of this view.

AutoHide in model rebuilding

AutoHide in model rebuilding

When selected, the program only displays the dimensions for the most recent operation, and hides all the other dimensions. Select this check box when working with a complex model, to hide dimensions that might otherwise make the model more difficult to understand.
When the check box is not selected, all dimensions are displayed.

Symbolic texts

Symbolic texts

The controls in this area determine the way Symbolic Texts are displayed in text and dimensions.

Symbolic text expansion

Hide unresolved symbolic texts

When selected, text and dimensions show the value of the Symbolic Texts.
When not selected, the string which describes the notation is displayed. That is, the display of the value  of the symbolic annotation, rather than the string that describes it, is set by checking the Symbolic text expansion box.

In the case a symbolic text is unresolved,  that is, should the program still display the symbolic annotation, not being able to replace it with a proper value, you can decide whether to display it or to hide it by properly setting the Hide unresolved symbolic texts check box.

After selecting or deselecting this check box, click the Redraw Texts button to update the display.

Hide unresolved symbolic texts

Hide unresolved symbolic texts

This item — enabled only if the Symbolic text expansion check box is selected — enables you to control the cases when a symbolic text should be unresolved — that is: should still display the symbolic annotation, not being able to replace it with a proper value. In such cases you can decide whether to display it or to hide it.

  • If the check box is selected, the unresolved symbolic text is not displayed.
  • If the check box is cleared, the unresolved symbolic text is displayed (that is: the symbolic annotation string is displayed in the place of the value).

Please note the option applies to any unresolved symbolic texts, no matter whether included in wider text strings or not.
After selecting or deselecting this check box, click the Redraw Texts button to update the display.

Redraw Texts

Redraw Texts

Forces text redrawing after changing the status of the Symbolic text expansion control.

Partial save command expands unrelated symbolic texts

Partial save command expands unrelated symbolic texts

When selected, Symbolic Texts are expanded when a Partial Save command is started, so that they will show the value even if they should become unrelated.
If the box is not selected, symbolic texts are not expanded; in this case, if they become unrelated, the symbolic notation will be displayed.
