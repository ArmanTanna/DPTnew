The Redefine command enables you to redefine an existing Symbolic Reference.

The command is available from the context menu displayed when right-clicking the selected Symbolic Reference in the Model Structure, as well as from within the View Symbolic References command, when you right-click on a Symbolic Reference mini-dialog box.

The command displays a selection list, from which you can select the geometric entities that you declared to be Symbolic References and the single Symbolic Reference, in order to modify it.

To end the command and update the model, click the (Rebuild) icon in the selection list.
