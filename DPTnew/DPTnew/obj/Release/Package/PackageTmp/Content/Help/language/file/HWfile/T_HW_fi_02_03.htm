Opening a file from the local file system


1 In the Open file selection dialog box, under Files of type, select the type of document you want to open in the list below:
  • think3 3D model or 2D drawing — extensions ".e3", ".e2"
  • think3 3D model — extension ".e3"
  • think3 2D drawing — extension ".e2"
  • GBG DraftMaker drawing — extension ".d"
  • ASCII points — extension ".pt"
  • IGES — extensions ".igs", ".iges"
  • VDA — extension ".vda">
  • DXF file — extension ".dxf"
  • DWG file — extension ".dwg"
  • think3 neutral format — extension ".gkd"
  • STEP — extensions ".stp", ".step"
  • STL — extension ".stl"
  • All files

2 In the Look in selection box, select the folder containing the document to open (initially, the model directory chosen when installing the think3 application is proposed). You can navigate through your file system like in any Exploring window of the Microsoft® Windows operating system.
A list of all documents of the specified type located in the selected folder will be displayed in the box below.

Please note steps 1 and 2 can be performed in reversed order.
 
3 Click on the document you want to open. The document name will be displayed in the File name box. If the document type is ".e3" or ".e2" (and when the selected document was saved the proper options were set in the Preview tab of the FileProperties dialog box), a preview image of the file contents will be displayed in the frame on the right.

4 Click the Open button to open the document.

A new window containing the selected document will be opened in think3's client area.
