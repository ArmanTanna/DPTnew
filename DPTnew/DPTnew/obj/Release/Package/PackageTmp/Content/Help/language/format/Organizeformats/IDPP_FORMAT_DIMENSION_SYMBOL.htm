Dimension Format - Symmetric symbol

Note
This option is enabled only when the checkbox associated with it is selected.

Symmetric symbol Enables you to include a symmetric symbol as part of the dimension created. This option is useful to show the symmetry of the dimension created. The preview area, provided on the right hand top portion of the dialog box, enables you to have an updated view and also control the effect of the changes made. You have the following two options:

Enable Enables you to show the symmetric symbol for the dimension
Disable Enables you to not to show the symmetric symbol for the dimension
