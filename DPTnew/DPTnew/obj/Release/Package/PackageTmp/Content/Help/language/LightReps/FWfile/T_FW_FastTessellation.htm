The Fast Tessellation command enables you to apply a Light Representation at an intermediate tessellation level to a/different X-Reference component(s).

The command is available from the context sub-menu displayed when right-clicking on the X-Reference (component) and selecting the Light Representations command.

Note that the command can be applied exclusively to X-Reference components.

The Fast Tessellation Light Representation is generated automatically. When you perform the command, the Model Structure displays the hierarchy of the sub-components but doesn't show any more the list of the events.
