New Welding Functionalities and Improvements

Version 2011.1 SP2 New introduces the following enhancements to the Welding commands:

  • New Symbol Scale option in the Welding Symbol command

New Symbol Scale option in the Welding Symbol command

The new Symbol Scale option in the Welding Symbol command allows to change the size of the welding symbol. By entering different sizes of the welding symbol in the Symbol Scale option, the symbol preview changes to reflect your input.
