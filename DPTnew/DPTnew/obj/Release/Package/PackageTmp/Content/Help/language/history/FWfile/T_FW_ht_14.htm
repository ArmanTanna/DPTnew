ModifySolidDeactivate

The Deactivate command temporarily disables a feature without canceling it from the structure.

You can start the command both from the menu bar and from context menus, displayed when you right-click the selected feature(s). In addition to solid features, the command can be applied to other entities such as datum entities.

The Mode drop-down list of the command's selection has two options as follows:

Selection: It enables you to deactivate the features or the datum entities of a solid.
To select the features/datum entities from the solid the Features/Datum Entities selector is provided.
Feature type: It enables you to deactivate all the features of a solid. To select a solid you are provided with the Solid selector. On selecting a solid different check boxes get displayed with respect to the different feature created in the model. Keep All/Reject All enables/disables you to select all displayed check boxes. Following are few check boxes that get displayed with respect to the feature present in the model.

Hole: It highlights the hole features present in the model to be deactivated.
Linear Slot: It highlights the slot features present in the model to be deactivated.
Edge Fillet: It highlights the edge fillets if any present in the model to be deactivated.
Flange: It highlights the flange features present in the model to be deactivated.
Pattern: It highlights the pattern features present in the model to be deactivated.
Shell: It highlights the pattern features present in the model to be deactivated.

The command can be used in sequence with the Reactivate command, enabling you to temporarily deactivate one or more features, then activate them again when required, or on its own, deactivating one or more features that are kept in the structure, which you don't want to delete.

Typically, you could make some features deactivated to study some design alternatives.

Feature activation status can be managed through Visual Bookmarks.
