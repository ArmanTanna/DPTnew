Assembly

New Component Enables you to create a new component.
Open X-Reference Enables you to open the X-Reference component.
X-Reference ( Component) Enables you to insert a component by referring to an external file.
Make X-Reference Enables you to transform a local component into an X-Reference component.
New Component as X-Reference Enables you to create a new component and transform it into an X-Reference component.
Set Current Enables you to set a selected component as the current component so that you can modify it.
Top Level/ Reset Current Component Resets the conditions of the assembly so that there is no longer a current component.
Unload Unloads selected components from memory.
Reload Reloads one or more components that have been modified or removed from memory using the Unload command.
Unlink Breaks the link between an X-Reference component and the file which represents it.
Break Breaks a component into the individual entities which represent it.
Replace Replaces a component with another component.
Derive and Replace Derives the selected X-Reference component and Replaces it with the same.
Assign Converts two components containing the same entities into two instances of the same component.
Explode Assembly Enables you to create or edit the exploded view of an assembly.
Assembly Analyzer Provides you with an overview of the current assembly and information about existing Document Data and Part Properties for the components.
