FormatOrganizeWeldings


 

The Weldings command enables you to select the desired welding standard and define a set of preferred welding types in order to ease the selection of the needed type when drawing the weld or the welding symbol.

The command dialog box provides you with the following controls:

  • Standards:
    Select from the drop-down list the welding standard you wish to use. Your choice determines the list of available welding symbols and their names.
  • Symbols to be viewed:
    Lists the welding symbols available for the current standard. Next to each name, a checkbox indicates whether the symbol will be present in the list of symbols made available to the commands that you use to actually insert the symbol in the think3 document. Check the checkbox to show it, uncheck the checkbox to hide it. You may want to keep checked in each standard only those you use most frequently.
  • Show All
    Click this button to check all the symbols in the Symbols to be viewed: area.
  • Hide All
    Click this button to uncheck all the symbols in the Symbols to be viewed: area.
