Changing the visibility of an edge

To change the visibility of an edge, start the Edge Visibility command and proceed as follows:

1 Select the drawing view, which include the edges you want to change the appearance of.
2  Select the edge display mode from the Line Type drop-down list. You can set the edge display to any of the following modes:
Dashed–medium displays the edge using a dashed line
Continuous–medium displays the edge using the Visible lines: settings specified in the View Attributes category of the Options/Properties dialog box
Continuous–thin displays the edge using the Thin lines: settings specified in the View Attributes category of Options/Properties dialog box
Default restores the edge to its original condition.

3  You are prompted to select the edge(s). Click the edge whose display mode you want to change. As soon as you click on the edge, its visibility immediately changes to the mode you specified.
4 The program continues to prompt you to select the edges. To select more edges belonging to the selected (set current) view, repeat the step 3.
5 To end the command, start another command, click the Select Entity command, or press Esc key.
Note
  • To change the edge display mode of an edge in another drawing view, proceed as follows:
    • Click View selector from the selection list.
    • Choose the drawing view.
    • Then, select the edge.


  • To change the visibility mode of an edge that you had previously hidden, follow the steps below:
    • Reset the current view.
    • Select the drawing view whose edge you want to edit, right-click, select the Unhide Dimensions Axes Edges command from the context menu.
    • Choose the Edges from the Entity Type drop-down list from the selection list.
    • Then, select the edge(s) to be unhidden.
