Splitting a curve by indicating intersecting surfaces

  1. Open file "SplitCurve_RuledBoundary.e3".


     
  2. Select the Split Curve command.
     
  3. In the Mode drop-down list, choose Boundaries.
     
  4. Click on the curve you want to split


     
  5. End selection by right-clicking and selecting Continue in the context menu.
     
  6. As boundary surface, select the ruled surface in the illustration:


     
  7. Click . The curve is split at each of its intersections with the selected surface:
