DXF/DWG Conversion Options

When saving (by Save or Save As) or loading (by Open or Insert from File) a DXF/DWG file by selecting DXF File (*.dxf) or DWG File (*.dwg) in the Save as type/Files of type drop-down list in the file selection box, the Options button displays the DXF/DWG Options dialog box. It enables you to define the parameters to be used in the conversion process that are present in the following categories:

  • General
    This category contains the most common parameters to be set when converting DXF/DWG files.
    • General- Colors
      The controls in this category determine how the colors will be converted.
    • General- Layers
      The controls in this category determine how the layers will be converted.
    • General- Line Type
      The controls in this category determine how the line types will be converted.
    • General- Font
      The controls in this category determine how the fonts will be converted.
  • Load
    This category contains the most common parameters to be set when loading DXF/DWG files.
    • Load- Advanced
      This category contains parameters controlling advanced conversion settings.
  • Save
    This category contains the most common parameters to be set when saving DXF/DWG files.
    • Save- Advanced
      This category contains parameters controlling advanced conversion settings.
