Select an arc or a circle

When you start the Smart command, the program prompts you to select an entity.
As soon as you select an entity, the options change, depending on the type of entity you select and the point at which you select the entity:

  • Line or edge
  • Special point
  • Arc or circle

If you select a curve, the selection list displays the following options:
Orientation Enables you to specify the orientation of the dimension line. Select one from the drop-down list
Type Enables you to specify the type of the dimension to be created, based on the selection you performed. Select one type from the drop-down list
Entity Properties Displays the Properties dialog box


You can then create a radius, diameter, or arc length dimension using one of the following options (to be selected in the Type drop-down list):
 
Radius
Diameter
Arc length

  • Horizontal
  • Vertical
  • Projected
    • At an Angle: Enables you to set the dimension line angle (which is measured counter-clockwise in relation to the X-axis)
You can also specify that the dimension be created without extension lines and control the position of the dimension line.
