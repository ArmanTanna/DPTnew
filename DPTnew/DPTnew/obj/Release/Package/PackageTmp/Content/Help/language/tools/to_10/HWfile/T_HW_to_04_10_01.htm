Customize shortcuts - Associate a keyboard shortcut with a command

After opening the Shortcuts tab in the Customize dialog box, follow these steps to associate a command to a keyboard key combination, so that by pressing the program executes the command.

1 Under Category, select the command category to display a list of commands in that category.
2 Under Commands, select the command with which you want to associate the keyboard shortcut. A description of the command is displayed in the Description area of the dialog box. Note that any shortcuts already assigned to the command are displayed in the Current keys box.
3 Click inside the Press new shortcut key box, and then press the keyboard shortcut combination you want to associate with the command. For example, press the CTRL and I keys simultaneously to assign that keyboard combination as the shortcut.
4 Click the Assign button to complete the keyboard shortcut assignment. The new keyboard shortcut appears in the Current keys box.

To cancel all keyboard shortcut modifications you have made click Reset All.

To delete a specific keyboard shortcut, select the category and command to display its keyboard shortcut, and then click Remove.
