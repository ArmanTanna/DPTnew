Display parameters

Display parameters

In this area you are enabled to adjust the properties controlling the way curves and surfaces are displayed.

Tolerance

Tolerance:

Enables you to determine the discretization adopted when displaying curves on the screen. The higher the tolerance value, the less accurate the display tolerance, the faster the drawing is executed, and the lower its quality.

U Curves

U Curves

Enables you to define parameters for U curves.

No. of isoparametric curves

No. of isoparametric curves:

Enables you to determine the number of isoparametric curves used to represent the surface in the U direction. Higher values result in more accurate on-screen representations but require longer to display.

Click Apply Changes to update the visualization.

No. of points

No. of points:

Enables you to determine the discretization of each display isoparametric curve, by setting the number of points to be used in the U direction.

Click Apply Changes to update the visualization.

V Curves

V Curves

Enables you to define parameters for V curves.

No. of isoparametric curves

No. of isoparametric curves:

Enables you to determine the number of isoparametric curves used to represent the surface in the V direction. Higher values result in more accurate on-screen representations but require longer to display.

Click Apply Changes to update the visualization.

No. of points

No. of points:

Enables you to determine the discretization of each display isoparametric curve, by setting the number of points to be used in the V direction.

Click Apply Changes to update the visualization.

Show discontinuities

Show discontinuities

Tessellation takes into account also cusps, and cusps are visible also when the model representation is shaded. In addition, the isoparametric curve visualization can be activated also for arc joints.

  • When the check box is selected, isoparametric curves on surfaces are displayed for each discontinuity (cusps or arc joints) (in wireframe mode) and the tessellation is also taking into account the cusps (in shaded mode). In this case No. of isoparametric curves applies between each discontinuity and/or surface border pair rather than to surface borders only.



    No additional isoparametric curve corresponding to arc joints will be displayed if you selected Cusps.

    .
     
  • When not selected, isoparametric curves related to cusps are not displayed.

About tessellation
Please note that when you open a model that was saved using a version prior to 2005.1, the applied tessellation parameters are the current ones, that is the ones that will be applied to all newly created entities.

Behavior on multiple selection
Also note what follows:
  • If the values of the tessellation parameters are the same for all the selected entities, then the values displayed in the category page are the ones common to the selected entities.
  • On the contrary, if the selected entities have different values for the tessellation parameters, the default document values are displayed instead.

Once you have changed the status of this check box, click Apply Changes to udpate the visualization of the selected surface.

Apply Changes

Apply Changes

Automatically updating the visualization of complex curves/surfaces each time a setting changes might be time consuming. So you can first change any of the settings of the following list, and then click this button to udpate the curve/surface visualization once for all.

  • No. of isoparametric curves (u curves)
  • No. of points (u curves)
  • No. of isoparametric curves (v curves)
  • No. of points (v curves)
  • Show discontinuities
