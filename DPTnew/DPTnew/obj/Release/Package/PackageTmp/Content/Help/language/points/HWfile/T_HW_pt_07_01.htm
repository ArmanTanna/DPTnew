Project points onto entities (generic)

1 Select the points you want to project.
2 End the input of the points.
3 Select the entity onto which you want to project the points.
4 The direction of projection can be chosen from the Direction drop-down list.
5 If you choose the Normal option,



the points are projected at right angles to the surface.
6 If you select the Pierce option, then you will see another Direction drop-down list, which has the following options:
  • Line
  • 2 Points
  • X
  • Y
  • Z
7 If you choose one of the X, Y or Z options,



the points are projected along the corresponding axis.
8 If you select the Line option,



you will be asked to select a guide line.
The points are projected along the axis defined by this line.
9 If you select the 2 Points option,



you will be asked to select two points.
The points are projected along the axis defined by these two points.
