System Options

Category: Input-Snap

Snap

Snap trap

Snap trap:

Specifies the size of a square area, centered on the selection point, inside which the program automatically locates any valid location matching the current snap mode.



Any point specified using the mouse inside the snap trap will be snapped to the specific special point corresponding to the one of the current snap operator.

Any point outside the trap will not be snapped to the special point.



See "About the snap" and " Snapping onto special points" for further details.

Selection trap

Selection trap:

Specifies the size of a square area, centered on the selection point, inside which entities are selected:

  • If the entity to be selected is inside the Selection trap of the position specified using the mouse, the entity will be selected.


     
  • If the entity to be selected is outside the Selection trap of the position specified using the mouse, the entity will not be selected.

Middle button snap

Middle button snap:

The think3 application has four separate snap modes. This option enables you to determine the current snap mode associated with the middle mouse button: None, 1, 2, 3, 4.
You can specify different snap point types and selection filters for each of the four snap modes. See "Customize AutoSnap" for further details.

AutoSnap

AutoSnap:

The think3 application has four separate snap modes. This option enables you to determine the current automatic snap mode: None, 1, 2, 3, 4.
You can specify different snap point types and selection filters for each of the four snap modes. See "Customize AutoSnap" for further details.

Customize AutoSnap

Customize AutoSnap...

Displays the AutoSnap dialog box, which enables you to customize the automatic snap as described in "Customize AutoSnap".
