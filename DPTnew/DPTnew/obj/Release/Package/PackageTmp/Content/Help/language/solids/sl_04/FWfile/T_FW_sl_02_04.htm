InsertSolidSweepPipe Slot

The Pipe Slot command creates a slot in a solid with a circular cross section along a path defined by an existing open or closed spine curve profile. The resulting slot can be one of the following:

a slot with a simple circular cross section, created by specifying only the diameter of the cross section
a solid tube centered within the slot, created by specifying both the diameter of the cross section and the thickness of the tube wall.

The resulting solid is connected directly to the base spine profile. If you change the profile in any way (such as by using any of the profile edit commands), the solid changes accordingly.
