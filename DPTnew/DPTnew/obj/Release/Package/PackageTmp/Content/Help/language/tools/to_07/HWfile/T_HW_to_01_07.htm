Display information about a selected entity

To start the command, do one of the following:

  • From the menu, choose the Single Entity command, and then select the entity whose information you want to display.
  • Select the entity you want information about, and then right-click and select the Info Single Entity command from the context menu.

The information displayed depends on the type of entity you select. The program displays information about the following types of entities:

  • Point
  • Line
  • Arc
  • Ellipse
  • NURBS curve
  • Profile
  • Plane
  • Cylindrical surface
  • Ruled surface
  • Surface of revolution
  • Proportional surface
  • Coons surface
  • Fillet surface
  • NURBS surface
  • Solid
  • Mesh
  • Group
