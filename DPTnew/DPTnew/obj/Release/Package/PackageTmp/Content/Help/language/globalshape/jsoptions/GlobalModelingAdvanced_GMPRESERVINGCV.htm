Enables you to define the type of entities that are to be preserved:

  • Curves
    to preserve the shape of the entities to be modified along selected curves.
  • Points to preserve the shape of the entities to be modified at selected locations.
  • Virtual vertices
    to preserve the shape of the entities to be modified at points defined by curves supposed to 'cross' each other without necessarily exactly intersecting.
  • Highlights
    (Advanced GSM only) to preserve the shape of the entities to be modified along highlight lines.
  • Silhouette
    (Advanced GSM only) to preserve the shape of the entities to be modified along silhouettes.

Once the choice has been performed, you will have to select the required groups (Group) of entities along/at which the shape is to be preserved.

In case of Curves, you will be enabled to select the desired Constraint:

  • Free — no constraint at all.
  • Keep position — to preserve the position of the selected curves.
  • Keep position + tangent — to preserve the position of the selected curves and their tangency conditions.
  • Keep position + tangent + curvature — to preserve the position of the selected curves and their tangency and curvature conditions.
  • Keep tangent — to preserve the tangency conditions of the selected curves.
  • Keep tangent + curvature — to preserve the tangency and curvature conditions of the selected curves.
  • Keep plane — to preserve the plane on which the selected curves lie.
  • Keep projection — to preserve the projection of the selected curves along a direction to be specified.

In case of Points, in the Group drop-down list, you can specify the type of points to be selected:

  • Standalone — single points to be selected ( Points).
  • Points on surface — points ( Points) to be indicated on a surface to be selected ( Support Surface).
  • ASCII File — points the coordinates of which are contained in an ASCII file to be specified ( Browse) and for which you can specify the Units of measurement.

You will then be enabled to select the desired Constraint:

  • Free — no constraint at all.
  • Keep position — to preserve the position of the selected points.

In case of Virtual vertices, you will be enabled to define the settings as described in Applying a controlled modification to a shape using Virtual Vertices.

In case of Highlights ( Advanced GSM only), you will have to specify the highlight curve ( Curve) for each Group and the Projection direction (see " How to define an axis or a line" for details).

In case of Silhouette ( Advanced GSM only) you will have to select the type in the Silhouette type drop-down list: a Curve to be selected ( Silhouette), or a Direction, to be selected in the Silhouette direction drop-down list.
