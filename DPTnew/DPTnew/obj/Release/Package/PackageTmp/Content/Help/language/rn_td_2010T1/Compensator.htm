New Compensator Functionalities and Improvements

Version 2010.1 introduces the following enhancements to the Compensator commands.

  • New compensator concept
  • Compensator commands in new toolbars and menu bar
  • The new Adapt on Measured Data command
  • The new Identify Measured Data command
  • The new Blank-Holder Addendum and Part command
  • The new Offset Mesh command
  • The new Alignment commands

New compensator concept

Currently, designers predict the springback deformation by using FEM (Finite Element Analysis) tools. However, it requires considerable time and effort in integrating the FEM results into the modification of the tool design. Hence, the FEM tools are not always used and a measurement process is applied instead.

In this version, a new concept has been introduced in the compensator solution. A new set of tools have been developed to realize Compensation (or Adaptation) based on Measured Data. These new tools are available in the new Compensator MD toolbar.

Adaptation on measured data

It is possible to apply deformation to adapt the original surfaces of the model. It is done by identifying the deformation from the physical model (obtained by scanning) as compared to the initial digital model. Then the deformation is applied to the digital model. Hence, it resolves the problem of creating a digital model that is identical to the physical tool.

Compensation on measured data

It is possible to compensate the part based on the measured data rather than on FEM data. The reverse deformation data from the physical model are applied to the digital model employing an inverse law. Minor design changes can be applied even without an additional FEM calculation.

These new functionalities make it possible to maintain quality throughout the design process, considerably reducing time and cost. It becomes easier to make complex components that maintain the original design intent, with the possibility of handling design changes in a very effective way.

Compensator commands in new toolbars and menu bar

Compensator commands are now available in the following two new toolbars.

Compensator FE Contains the commands previously available under the Compensator toolbar that make use of FEM results.
Compensator MD Contains the new commands that make use of Measured Data.

All Compensator commands are now available under ModifyCompensator in the menu bar.

The Adaptation command is available under ModifyAdaptation in the menu bar.

The new Adapt on Measured Data command

The Adapt on Measured Data command enables you to find the correspondence between an Initial Mesh (corresponding to a base geometry) and a Measured Mesh (corresponding to a modification of the base geometry). At the same time, the command will identify a deformation representing the correspondence.

The new Identify Measured Data command

The Identify Measured Data command enables you to find the correspondence between an Initial and a Measured mesh, but will identify the deformation corresponding to the compensation to be applied on the Initial Mesh. Here, the Measured Mesh corresponds to the shape obtained as a result of the manufacturing process using the Initial Mesh geometry.

The new Blank-Holder Addendum and Part command

The Blank-Holder Addendum and Part command enables you to modify the die environment based on the result obtained by the adaptation/compensation process.

This command is intended to solve a common problem related to the die design and the springback compensation. During the design process, the user must have completed one or more process compensation obtaining some deformed surfaces. These deformations compensate only the Part areas of the die. The Blank-Holder Addendum and Part command enables you to change other portions of the die to accommodate the compensation result.

While applying this command, you need to make the following inputs in the form of solid skins:

Addendum It is the portion of the die that has to change to accommodate the compensated parts.
Blank-Holder It is the area that connects the Addendum to the die sides.
Deformed Surface It is the modified surface obtained after applying the process compensation.
Part It is the part on which compensation has been applied.
Closing Faces It represents the holes (open areas) of the part.

You can use some GSM Parameters and Approximation while applying this command.

The Fix compensated part check box enables you to define whether the compensated part remains fixed or not while applying this command.

The new Offset Mesh command

The new Offset Mesh command enables you to offset a selected mesh by a specified offset value to be entered in the Offset Size text box. Select the Preview () button to see the preview of the offset mesh. In case the offset mesh is self-intersecting, a warning is displayed on selecting the Show Warnings () button. Also, the location of self-intersection is highlighted on the mesh.

The Always recalculate normals check box enables you to manage the normals of the mesh being offset.

  • When the check box is selected, the normals on mesh are always calculated and reoriented, even if they already exist on the mesh.
  • When the check box is cleared, the normals on mesh are calculated and reoriented only if they do not exist on the mesh.

The new Alignment commands

The new Alignment commands enable you to manage alignment of two entities (mainly Meshes) that represent the similar shapes. The source of these entities can be a mesh measured on a physical model and that of a digital model. The physical model could have been modified, so that its measure could be used to make compensation on the digital model. Hence, first you need to align the two meshes using the same reference. The various Alignment commands are:

Pre-Alignment Enables you to align two distant meshes by using 4 user-defined point couples.
Optimize Alignment Enables you to further optimize the alignment of the meshes that are brought close to each other. Also, it supports the alignment of multi-zones and hence, you can define a set of corresponding zones on the two meshes to align.
Optimize Contour Alignment Enables you to optimize the alignment along curves and hence, can be useful in cases where the correspondence is about boundaries.
Replicate Alignment Enables you to replicate the transformation generated by the previous commands to other entities.
