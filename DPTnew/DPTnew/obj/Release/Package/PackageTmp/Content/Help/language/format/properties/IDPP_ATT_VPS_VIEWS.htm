Entity Properties

Category: Drawing View

Drawing View

Drawing View

The controls in this area enable you to set the main properties of the selected view.

Name

Name:

Determines the name of the selected view.

Show view name

Show view name

When selected, the view name is displayed in the drawing
When not selected, the view name is not visible.

Show separator

  • When selected, the separator symbol (—) will be displayed in between the view name and its scale factor value.
  • When cleared, the separator symbol will not be displayed.

The Show separator check box is enabled only when the Show view name and Show view scale check boxes are selected.


Scale

Scale:

Enables to specify the scale for the selected view.
If you enter only one number — say x — the actual scale assigned by the program will be x:1. Each former scale value will be available in the drop-down list.

Show view scale

Show view scale

When the check box is selected, the scale factor is displayed in the drawing view.
When not selected, the scale factor is not displayed.


Note
By default, the check box is selected in the case of the Detail View and not selected in all the other cases.

Type

Type:

Displays the type of view (such as Main, Top derived, or Sectioned).

Align to main

Align to main

When selected, the derived views can be moved only in the direction of their alignment with the main view.
When not selected, the derived views can be moved to anywhere in the drawing.


Note
The detail view can always be moved anywhere unlike other views, irrespective of the option selection. By default this option is not selected. Also, it is activated only when the Inherit breaks from main to derived view option is not selected.

— When selected, the arrow on the main view changes its orientation with respect to the new detail position.
— When not selected, the arrow on the main view remains fixed, that is, it cannot change its orientation with respect to the new detail position.

Frozen view

Frozen view

When selected, the view will not be updated by any Rebuild operation.
When not selected, the view will be updated at any rebuild operation.

Filter entities smaller than

Filter entities smaller than:

Enables you to skip entities smaller than the specified value for the specific view (extensively helpful when dealing with large assemblies).
If the edited value is different from the current one, a view regeneration is automatically launched with the click of OK button.
If the filter is set to a value greater that zero (a warning icon will appear on the right of the value box (), all entities smaller than that value will be automatically skipped by the view regeneration.

Hidden line computation

Hidden lines computation:

Enables you to choose the computation mode to generate a drawing view. The possible computation modes are as follows:

Software In this mode, the drawing view is generated accurately based on the software information.
Hardware In this mode, the drawing view is generated faster based on the hardware information. In this case, each generated view will be associated with a precision value and the entities smaller than this value cannot be excluded by hidden line algorithm.


Note
  • The new option is active only when the Representation: is set to Hidden lines and the Show dashed hidden lines option is not selected in the Entity Properties - Drawing View - Visibility - Geometry area.
  • This option works on all views except the flat pattern, exploded, cross-section and developed views. Also, it is not effective with components in light representations.
  • If the option is set to Hardware, the resulting drawing will be not  compatible with any of the previous versions.
