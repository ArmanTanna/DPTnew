Insert a Datum Reference

To insert a datum reference proceed as follows:
1  In the Name box, type or select the entity to which the tolerance refers.
2 You are prompted to select an entity. Click to select the entity to which you want to attach the datum reference.
3 The program then prompts you to enter the positioning point. Click to specify the endpoint of the leader line. This prompt repeats, so that you can create multiple leader line segments.
Select Text only option under More Options to insert the control frame without selecting a reference entity to attach it to.
