Creating a Reference Grid in the Drawing View

After creating a Drawing from current model, start the Reference Grid command and proceed as follows:


1 When you start the Reference Grid command, you are prompted to select a view.
You can select the view by clicking on it either in the drawing or in the Drawing Structure.
2 Click on the view you want to insert the Reference Grid in.
3 The program immediately displays a Set Boundary option, which permits two possible grid definition modes.
Select one of the grid definition mode from the drop-down list.

Automatically If this mode is selected, the boundaries of the grid are automatically computed based on the bounding box of the selected view, so as to cover the whole view and extend beyond its boundaries as much as specified in the Offset box.
When you Regenerate View, grid bounds are automatically updated as to the new view bounding box.


 
By 2 points If this mode is selected, you will specify two points which will define the grid extension.



If a non-null Offset is defined, the grid will be enlarged as much as specified in both directions with respect to the bounding box determined by the two selected points.
 
4 In the Step box, specify the distance between two subsequent grid lines.
5 In the Offset box, specify how much the grid is to be extended beyond the specified boundaries (both Automatically and By 2 points modes).


  • You can reset the Set Boundary option by choosing other option from the drop-down list before the command is applied.
  • You can also reset the selection list options using Modify Entities from the context menu of the Drawing Structure after the command is applied.

6 Click Properties, to set or change the properties of the reference grid.
7 Click one of the following to complete the command action:
Apply to insert Reference Grid in the selected view and keep the command active.
OK to insert Reference Grid in the selected view and quit the command.
Cancel to discard all the operations and quit the command.
The program immediately creates Reference Grid with the defined properties.


Note
  • Other factors which affect the display of the label, construction properties of the grid, format of the labels, display of grid lines, and display of the text in the label, are controlled from the various categories under Reference Grid of the Options/Properties... dialog box.
  • After you create a reference grid, you can change many of these properties from the various categories of the Reference Grid
  • As the Reference Grid is a group entity, the visualization of the internal colors defined for the major and minor lines depends upon the value of the Show closed groups with group color check box in the Groups category of the Document Properties. To display the specific colors of the grid — that is: the one of the circular frame, the one defined for major lines and the one defined for minor lines — the said box must be cleared.
