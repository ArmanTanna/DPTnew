System Options

Category: Reference Systems

Work Plane

Work Plane

The options in this area enable you to control the display and appearance of the Work Plane.

For further details, see "Tips on the Work Plane appearance".

Show

Show
When the Show check box is selected, the program displays the Work Plane, according to the current display settings.
When the Show check box is cleared, the Work Plane is not displayed.
Show selected Show NOT selected

Please note that you can also display or hide the Work Plane by using the View Work Plane command.
For further details, see "Tips on the Work Plane appearance".

Representation

Representation

The radio buttons in this area define the location where the Work Plane Reference System will be displayed.

On place

On place

Select this radio button to display the Work Plane Reference System at its actual location (over the Work Plane) in reference to the Graphics Area.

As proxy

As proxy

Select this radio button to display a dummy representation of the Work Plane Reference System (as a proxy) at one of the corners of the Graphics Area. The corner at which the proxy is displayed is defined by the Position: drop-down list under the Representation as proxy area of the System Options - Reference Systems - Appearance category.


With this radio button selected, the Work Plane itself is not displayed in the Graphics Area.

On place and as proxy

On place and as proxy

Select this radio button to display the Work Plane Reference System both at its actual location as well as a proxy in the Graphics Area. The Work Plane itself is displayed only at its actual location and not over the proxy.

Automatic

Automatic

When this radio button selected, the Absolute Reference System will be automatically displayed as a proxy if its actual location is not within the visible portion of the Graphics Area.

Absolute Reference System (World)

Absolute Reference System (World)

The options in this area enable you to control the display and appearance of the Absolute Reference System.

Show

Show

When the Show check box is selected, the axes of the Absolute Reference System (World) are displayed on the Graphics Area.
When the Show check box is cleared, the axes are not displayed.
