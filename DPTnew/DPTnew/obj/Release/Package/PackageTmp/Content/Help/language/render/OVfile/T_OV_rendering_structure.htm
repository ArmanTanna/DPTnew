Rendering Structure

The Rendering structure is on the left pane of the application window. The rendering structure is available by clicking on the Rendering Tab. While applying the render properties to the models, the rendering structure collects the data and lists it under appropriate nodes. Using the listed render properties you can edit, delete, rename and save the property to the library. The following are the rendering properties available in the Rendering Structure as shown in the image below:



Materials Displays the materials which have been applied on to the model.
Decals Displays the decals applied on to the different surfaces of the model.
Light Studio Displays the current light studio active on the model.
Environment Displays the current environment active on the model.
Texture Space Displays the texture space applied on to the model.
