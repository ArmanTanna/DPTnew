Delete user defined property to entity



  1. Start the Delete UDP command; the Select class(es) to be deleted dialog box is displayed containing the list of properties.
  2. Select one or more properties. To select all the properties in the list, click on All. To deselect a property, click on it again.
  3. OK.
  4. Select the entity from which you want to delete the property.
Note

To remove a property assigned the model/drawing itself, you can proceed as described in "How to delete a UDC, removing its assignment from the model/drawing", in Custom tab of the File Properties dialog box.
