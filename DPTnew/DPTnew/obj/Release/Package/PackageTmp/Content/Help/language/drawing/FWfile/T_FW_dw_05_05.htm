InsertBreak LinesMultibreaks on View

The Multibreaks on View command enables you to create multiple breaks on views, so as to provide a more compact representation in the layout of very long parts.
It has a selection list that enables you to create an arbitrary number of breaks along the selected direction or along two perpendicular directions. The selected direction can be associatively related to model edges represented in the drawing view. When the selected direction is associative, a feedback by means of a lock cursor is given during the input phase.

Note:

The view is created by altering an existing view. The command resets the Undo/Redo buffer, thus preventing you from undoing any previous operations.
