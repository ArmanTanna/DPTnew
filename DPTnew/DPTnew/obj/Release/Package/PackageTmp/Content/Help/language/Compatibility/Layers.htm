• Layers: layers will keep their visibility value (on/off), their status of current/non current and their description, while they will loose their original names and will automatically be assigned a number (apart from layer 0, which will keep its 0 name=number). So, in general, the properties of name, lock, color, line type and width will be lost.
    Furthermore, as to names, a particular case is when a document created in versions older than 2008.1 containing layers n1,...., nm is first loaded into version 2008.1, in which case the layer names are automatically generated as "Layern1",...., "Layernm" and then saved. If such a document is reloaded with a previous version, the layer names are lost but their numbers will be the same as in the original document (n1,...., nm).
