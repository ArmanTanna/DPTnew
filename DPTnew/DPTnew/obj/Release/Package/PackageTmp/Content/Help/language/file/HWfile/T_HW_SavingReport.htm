Saving/Renaming reports

In some circumstances the program displays the following dialog boxes:

Saving Report When you are trying to save a file including components/shared groups which are read-only within the file and components/shared groups which are read-only from the file system, in order to list the items that have not been saved and require some specific actions to be saved
Renaming Report After renaming/sharing some items in the duplicated name conflicts occurring when you load models/drawings that include components/shared groups with the same name and different geometry. In this case the report shows the information of the actions that you have already carried out.

Saving Report

Any time you save a file including components/shared groups which cannot be saved because they are read-only within the file or read-only in the file system, you get the Saving Report dialog box, showing the complete information on the items involved, that is names, diagnosis and path.

This dialog box helps you understand why the items were not saved. You can save the report to a .txt file.

Thanks to this information, when you close the dialog box, you can act on the different items and carry out the adequate actions in order to save the file.

You may handle files including some read-only items in different situations: for instance, when you don't use any vaulting tool such as the think3 vaulting system in your company. Typically, assemblies may include local/X-Reference components which were made read-only within the file, and X-Reference components which were made read-only from the file system.

 

Renaming Report

After renaming/sharing components/shared groups from the Duplicated Name Resolution dialog box, the Renaming Report dialog box may be displayed, showing a report of the action(s) that you've carried out.

The dialog box is displayed only if the Show renaming report option is selected in the Assembly/Shared Group Shared Group category of the System Options.

You can close the dialog box and save the report to a .txt file.
