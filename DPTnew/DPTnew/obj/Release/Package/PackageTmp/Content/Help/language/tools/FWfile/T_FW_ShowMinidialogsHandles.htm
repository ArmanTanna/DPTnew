The Show Handle and Minidialogs command enables you to restore the visualization of the hidden handles and minidialogs (to restore the visualization only of minidialogs, use Show Minidialogs), when using the Move Copy Entities, Scale Entities and Multistretch Entities commands.

When using the said transformation commands, you can find this command in the context menu that becomes available when you right-click in the graphic area if handles and minidialogs are hidden, or you can run the command by hitting the Shift+F7 keys contemporarily.

The program automatically restores the visualization of handles and minidialogs when the current command is completed.
