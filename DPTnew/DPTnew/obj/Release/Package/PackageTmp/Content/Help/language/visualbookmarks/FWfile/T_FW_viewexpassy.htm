The View Exploded Assembly command enables you to recall a Visual bookmark that is created using the Explode Assembly command. This is also a type of View Only bookmark where you will not be able to edit it. If you exit the command, the original view will be restored. This command helps you to make a quick cross check of the bookmark that is created for the explosion parameters and you will also be able to capture images of the steps of the explode sequence.

You can perform the command from the context menu displayed by right-clicking on the Visual Bookmark of your choice.

You can view the Visual Bookmarks from the Visual Bookmarks tab on the left pane of your Graphic Area.
