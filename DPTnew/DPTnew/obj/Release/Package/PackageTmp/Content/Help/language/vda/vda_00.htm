Conversion to and from the VDA format

VDA-FS is a data format for the exchange of free formed curves and surfaces. This document describes how to convert VDA-FS files to the format of think3 and vice versa.

Introduction

think3 application enables you to read and write files in the VDA-FS format; click here to know about the supported versions.

Within think3 application, you can do the following:

  • To convert from the VDA-FS format to think3, use the Open or Insert From File command. In the file dialog box, under Files of type, select the type of file you wish to convert from by selecting the appropriate file format (".vda").
  • To convert from think3 to VDA-FS, use the Save or Save As command. In the Save dialog box, under Save as type, select the type of file you wish to convert to by selecting the appropriate file format (".vda").

Conversion is controlled by the parameters of the VDA Options dialog box opened by selecting Options. You can save as many different sets of parameter values  — to be used in different conversion situations —  as you like to the Settings library.

Help on parameters controlling the conversion process
You can easily display a short description of each parameter controlling the conversion process by issuing the help request as follows:
  1. Click on the button in the top-right corner of the dialog box displayed by selecting the Options button.
  2. Click on the item of the dialog box corresponding to the parameter you need help on.
