Select from the drop-down list the format of the current welding representation.

When you are inserting or editing a bead or a section, the program provides you the following possibilities:

Single
Double
Select from the Current side drop-down list whether the current options apply to the weld above or below. Each side, therefore, has an independent set of options.
Symmetric
A single set of options is available and it applies to both sides.

When you are inserting or editing a welding symbol, the program provides you with the following possibilities:

Single
Double
Select from the Current symbol drop-down list whether the current options apply to the weld above or below the reference line. Each side, therefore, has an independent set of options.
Symmetric
A single set of options is available and it applies to both sides.
Sequence
Indicates that a sequence of up to three independently defined welding steps apply (the figure shows a sequence of two steps). Select from the Current symbol drop-down list whether the current options apply to the first, second or third step of the sequence.
