System Options Categories

The System Options categories enable you to customize the following operating parameters of the think3 application.

Legenda
In the following table:
  • A red bullet in the 3D column means the category is available in the Model environment
  • A red bullet in the 2D column means the category is available in the Drawing environment
  • A red bullet in both columns means the category is available in both environments.

When you begin using the program, each operating parameter is assigned a default setting. When you change any of these values, the new value replaces the previous value.

An additional area is displayed if you click the Search button, enabling you to easily find parameters you might need to set (among the ones contained in the tabs of the System Options, Document Properties or Entity Properties dialog boxes; options of the Open and Save commands are not included).
