The Paste command enables you to paste to the selected component the Part Properties previously copied from another component.

In order to paste the Part properties that you have copied from a component, select the target component then execute the Paste command.

The command does not override the Part Properties that are already defined in the target component. The Paste command copies only the fields that are empty in the target component.
