Importing Drawing From CATIA

Starting from a CATIA® file containing 2D entities, the From CATIA to command enables you to import them.

Note that when the CATIA® file is created working with transparency in AuxView and AuxView 2, the CATIA®entities can't be imported. However, when the file is created working with Draw Elements, they can be imported.

To convert the CATIA® 2D entities included in a file to a drawing, proceed as follows:

  1. Start the TD Catia v4 interface Options to display the TD Catia v4 interface Options dialog box and access the Load tab.

  2. In the Select entities area, select the Drawing check box and clear the Model check box.

  3. Click OK to save the settings.

  4. Click the From CATIA to command. Selecting the command having open a new model window doesn't affect its running, while if you start the command having an open drawing window in the client area, a warning message is displayed, that the layout environment is still active and you have to close it or switch to model.

  5. In the Look in box of the file selection box that displays, select the folder where the file to open is located.

  6. To help identify quickly the file to open, make sure that Files of type contains CATIA® Model or export (*.model, *.dlv, *.exp).

  7. Select the file you want to open then click the Open button.

  8. The converted CATIA® entities are displayed in a new window.
