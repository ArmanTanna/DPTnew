Inserting an X-Reference Component

1.  Start the Component Cross Reference command to display the standard file selection dialog box.
2. Select the file you want to insert as external reference and click Open.
3. Next, the Insert X-Reference selection list displays and the preview of the component positions along with the cursor. You need to click in the Graphics Area to place the component. To exit from the command, press ESC key.

Note that from the file selection box displayed when you perform the command, you can select a Visual Bookmark that will be applied at loading. An additional check box in the file selection box enables you to load any X-Reference component in the Lightest Representation (i.e. Quick Load), corresponding to the Flat Tessellation Light Representation, regardless of what is specified in the selected Visual Bookmark.
