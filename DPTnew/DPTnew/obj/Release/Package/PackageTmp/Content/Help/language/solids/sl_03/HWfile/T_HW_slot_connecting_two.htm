Creating a Generic Slot by Connecting two Boundary Profiles

The Generic Slot command enables you to create a generic slot in an existing solid by sweeping a boundary profile along a spine path.

Spine and profile Generic Slot

To obtain the above result, you need to open the Slot_Multi_Profile.e3 file available in the samples folder of your installation folder and follow the steps given below.

  1. Start the Generic Slot command to display the selection list.

  2. Select the drive curve for Spine selector, as shown in the image below.



  3. Next, click on the Boundaries selector from the selection list and select the profile from the Graphics Area, as shown in the image below.



  4. Click on Solid selector in the selection list, and then select the solid on which the slot to be added.

  5. Select the Along one plane option from the Motion Mode drop-down list

  6. If required change the direction of the material to be removed by using red arrow in the Graphics Area.

  7. Click or to confirm your selections and create the slot, as shown in the image below.



Click to discard your changes.
