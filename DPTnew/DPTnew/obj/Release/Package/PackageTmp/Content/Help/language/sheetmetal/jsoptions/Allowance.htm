Allowance is provided to reduce the tension and compression effect on the bend of the sheetmetal. Following are the two options available in the Type drop-down list under the Allowance node.

K factor k-factor is a ratio that represents the location of the neutral sheet with respect to the thickness of the sheet metal part. This option enables you to provide allowance by specifying the k-factor value. The K factor edit box is available to specify the value. The k-factor controls the positioning of the neutral radius: 0.5 identifies the middle of the thickness, 0 identifies an outwards development, 1 an inwards development. For example, very hard materials have k-factors reasonably about 0.1, while k-factors for metals are about 0.5.
Bend table Enables you to select the material from the Material drop-down list. On selecting the Radius from table check box, you will be able to select the bend radius of the flange from the available values in the bend table.
