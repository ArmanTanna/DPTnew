Display mathematical information about a curve or surface

1  To start the command, choose the Local command, and then click on the entity about which you want to display the information.

When you select the Local command, the cursor changes to indicate that the command is active.



As soon as you select the entity, the program displays a dialog box containing information about the entity at the point at which is was selected. The dialog box title indicates the type of entity selected.



In addition, the program displays information graphically within the drawing. The information displayed depends on the type of entity selected:



  • Curves—the program displays the tangent (indicated by a continuous line), the normal (indicated by a dashed line) and the binormal (indicated by a dotted line).
  • Lines—the program displays the direction vector (indicated by an arrow).
  • Surfaces—the program displays the tangents in each of the two parametric directions (indicated by a continuous line in the U-direction and a dashed line in the V-direction), the related tangent plane, and the normal (indicated by a dotted line).
2 To display information about another point on the selected entity, type the new parameter values of the desired point in the u box (in the case of lines and curves) or in the u and v boxes (in the case of surfaces), and then click OK or press the Enter key.

The dialog box immediately updates, displaying the data for the new point. The graphic information within the drawing also updates to reflect the new point location.
