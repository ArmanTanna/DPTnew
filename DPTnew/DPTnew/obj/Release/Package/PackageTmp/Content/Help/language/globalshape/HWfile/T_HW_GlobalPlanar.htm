Applying a controlled modification through control points

In this job you will apply a controlled modification to a surface through a set of control points by using the Planar Bounding-Box command.
Please note that you can accomplish this task by loading sample file "globalplanar01.e3"



1 Define the type of entity the controlled modification is to be applied to. In the Entities drop-down list under Entities to Be Modified you can select the type of entities the deformation is to be applied to:
  • Solid
  • Surfaces
  • Curves
  • Points
Select Surfaces.
2 Select the entities the controlled deformation is to be applied to:



A rectangular grid of five control points along each side will be displayed :



Note that you can always add points to the initial grid by proceeding as follows:
 
  1. Set the Mode to Add control points. A Points item will be displayed in the Selection List just below the Mode node.
     
  2. Click on a line in between two control points.



    A whole row of control points will be added:



    Finally, note that though being in the Add control points node, if you click on an existing point you will be enabled to move it about.

 
3 Define the deformation plane — that is: the plane containing the control points to be used to deform the shape — by choosing one of the items in the Mode drop-down list:
On View To automatically define the deformation plane as the view plane.
Select plane To define a customized deformation plane. Once you have selected it, you can access the plane definition items. The picture below shows what you would get if you defined the deformation plane as perpendicular to the red line and through its right endpoint.


 
4 Apply the controlled deformation using the control points. Once the deformation plane has been selected, the deformation can be applied by setting the Mode to Edit control points and then by selecting control points (about the selection of control points, see " Edit Control Points") and dragging them about to the very position corresponding to the deformation you want to apply:



The following picture shows a different deformation applied using the Work Plane as the deformation plane (the view plane coinciding with the Work Plane):





Note
Some actions can be performed at any time during the command session:

  • Adding further points to the initial grid
    See the description above for details.
     
  • Adding new entities to the currently selected ones
    Please take into account that any time you add a new entity to the current selection the grid is re-initialized (for this reason, if you had added further points, they will be lost and only the initial five points will be left; if you had moved them, the movement will be reset).
     
  • Changing the deformation plane (that is, the plane containing the control points)
     
  • Undoing the movement of a control point
    just by clicking the Undo button ( ) which shows up on top of the Selection List as soon as a point is moved.
     

Further options are available under More Options (see the command description for details)

 
5 Click or to confirm your selections modify the shape of the object, or click to discard your changes.
