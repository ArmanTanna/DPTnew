InsertSheet MetalSketched Bend

The Sketched Bend command enables you to create bends in an object along a user-defined profile.

The bending line must be an open profile composed of a single straight line drawn on a face (not on the "thickness" side faces), dividing the face into two or more separate parts. The straight line can pass through one or more linear slots.

The bend can be added both when the object is flattened or when it is bent.

A red arrow is displayed to show the bend direction. Click the arrow to invert the bend direction.

A red arrow is also displayed to define the fixed part. Click the arrow to change it.

Type the bend angle value into the Angle mini-dialog box, or drag the corresponding red handle.

To locally change any of the Sheet Metal parameters related to this command (see Sheet Metal category of the Entity Properties), select Local Properties and proceed as described in Local Properties and the Lock Marker.

Try to ...
  • Bend a solid along a profile lying on one of its faces
  • Bend a solid along a profile (containing slots) lying on one of its faces
