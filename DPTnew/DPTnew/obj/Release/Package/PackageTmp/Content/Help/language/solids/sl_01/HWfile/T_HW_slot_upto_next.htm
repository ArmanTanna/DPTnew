Creating a linear slot through next face

The Linear Slot command creates a linear slot in an existing solid by sweeping a profile or chain of curves through the next face it encounters.

  1. On starting the command, you are prompted to select a profile. Select the profile or the chain of curves you want to sweep. To reselect the profile, click on Profile in the selection list and select the profile again.

  2. In the Extension drop-down list, select the Thru next option.

  3. Click on the Face selector; the direction in which the profile is swept is selected automatically. To change the direction, select a different face by again clicking on the Face in the selection list, and selecting a different face. Also you can right-click and select Invert from the context menu to reverse the direction.



    • To specify a draft angle, select the Angle check box and enter the angle value or drag the handle to the position corresponding to the required angle.

    • Use the Offset mini-dialog box to create a slot with an offset distance from the profile plane. Select the check box and enter the offset value or drag the handle to the required offset value.

  5. If the slot extends to another solid, you can click Face to select the solid to which the slot will belong.

  6. Click or to confirm your selections and create the slot. Click to discard your changes.
