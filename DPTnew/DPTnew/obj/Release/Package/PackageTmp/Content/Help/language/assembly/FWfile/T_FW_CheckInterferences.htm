ToolsInfoCheck Interferences

The Check Interferences command provides information about interferences among solids, components and sets of solids and components.

Typically, the command enables you to statically check interferences in assemblies, providing a simple way to achieve a complete and fast analysis of interferences between all components.

You can identify the parts (solids or components) of the model, which collide with each other, selecting objects by the ordinary post-selection options and also components inside components directly in the Graphics Area. You can define to check interferences among or inside sets of solids and components and view the results of the interferences that have been detected with different modalities.

Interferences are checked with no limit to the number of objects, and calculated by identifying the colliding objects and showing the involved faces.

You can also:

  • save text files with the interference results
  • automatically create interference solids
  • automatically create Visual Bookmarks, showing the interference solids.

In addition, you can introduce a tolerance value and consider as colliding two objects which are too close to each other, by defining a minimum distance between them.
