Copying data in the Bend Table Manager

The following illustration explains the procedure to copy bend table data from one row to another in the Bend Table Manager:

1. Start the Bend Table Manager command from the Tools menu; the Bend Table Manager dialog box is displayed.
2. Select the rows that contain the data to be copied into new rows.


  • To select a single row, click on the row number.
  • To select multiple rows, hold the CTRL key down while making the selection.
3. Right click on the selected rows to display the context menu. Select the Copy option.
4. Again right click and select the Paste option from the context menu to paste the data; the New Material Name dialog box is displayed.
5. Specify the material name in the Material name text box and choose the Ok button. The data gets copied to the blank row at the bottom of the table.
6. Choose the OK button to save the changes and close the Bend Table Manager.
