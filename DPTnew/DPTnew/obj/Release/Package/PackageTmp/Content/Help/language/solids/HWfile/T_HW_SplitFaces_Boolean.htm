Splitting solid faces with Boolean option

In the following example you will use the Split Faces command to split the faces of a selected solid at their intersections with a set of other solids. To accomplish the task, load file "SplitFaces.e3".

1 Start the Split Faces command, for Solid A select the cylinder and for Split with option, select Solids B from the list and select the cuboid.

Note
  • To change the solid, you can right-click on the solids, select Reset and select a different solid.
  • To reset the current selection, you can right-click on the graphics area and select Reset current selection.

    • 2 For Boolean option, you can do the following:

    a. Select Global from the list.
