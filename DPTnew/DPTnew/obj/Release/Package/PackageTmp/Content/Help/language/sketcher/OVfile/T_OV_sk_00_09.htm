Information about profiles

If the Enable graphic area tooltips check box is selected in the Input category, when you pause the cursor over a constraint, the program displays ToolTips to indicate the type of constraint. In addition, when the ToolTip appears, the program also highlights the constrained entities.

If you want to know more, the program provides the following commands for displaying information about a profile:

  • To display information about a constraint, use the Single Entity command.
  • To check for underconstrained, overconstrained, and inconsistent profiles, use the Check command.
  • To check the degrees of freedom for all the underconstrained profiles in the model, use the Underconstrained Profiles command.
  • To display external entities used to constrain the profile, use the Show External Entities command.
