The Configurations tab of the Define Smart Objects command enables you to define predefined values corresponding to all the exposed parameters. You can modify the ones that are automatically proposed by the think3 application and/or add your own.

  • Boxes in the Conf. Name column can be edited to change/define the name of the configuration
  • The Clear Conf. and Clear table buttons can be used to delete respectively one or all the defined configurations.
