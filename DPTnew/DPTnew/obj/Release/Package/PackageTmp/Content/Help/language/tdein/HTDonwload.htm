How to Download

The products are available in the Downloads page of the Customer Care area in the www.think3.eu web side.
Supply your personal account to sign in.

The steps are self explaining

  1. Choose a product
  2. Choose a version
  3. Choose a topic
  4. Choose an item
  5. Push the Go button to start the download.

You have to run the EXE package you obtain at the end of the download and then follow the instruction to obtain the DVD image.

To perform the installation click on setup.exe in the folder you obtain.

For multiple installation read the documentation about the Silent Installation.
