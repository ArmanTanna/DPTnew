ModifyPart ListUpdate

The Update command enables you to redefine the Part List currently inserted in the drawing.

The command is enabled only after a Part List has been inserted in a drawing with the Part List command. It is replicated in the context menu of the Part List group and is the default command that the program executes when you double-click the Part List.

For details on the command options and operation, refer to the Part List command documentation.
