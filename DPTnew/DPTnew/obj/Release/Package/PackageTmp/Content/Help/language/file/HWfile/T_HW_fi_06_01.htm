Saving a file to the local file system

Both when saving a document that hasn't been saved before (by selecting the Save command) and when selecting the Save As command, the program displays the file selection dialog box, from which you can save the document to the local file system.

  1. In the Save file selection dialog box, under Files of type, select the format you want to save the document in, in the list below:


    • think3 3D model — extension ".e3"
    • think3 2D drawing — extension ".e2"
    • GBG DraftMaker drawings (for plotting using GBPLO - drawings only) — extension ".d"
    • DXF File — extension ".dxf"
    • DWG File — extension ".dwg"
    • ASCII points — extension ".pt"
    • IGES — extension ".igs", ".iges"
    • VDA — extension ".vda"
    • think3 neutral format — extension ".gkd"
    • STEP — extension ".stp"
    • STL — extension ".stl"
    • VRML — extension ".wrl"
    • Wavefront — extension ".obj"
    • Inventor — extension ".iv"

  2. In the Save in selection box, select the folder where you want to save the document (initially, the model directory chosen when installing the application is proposed). You can navigate through your file system like in any Exploring window of the Microsoft® Windows operating system.


    3. Please note steps 1 and 2 can be performed in reversed order.

  3. In the File name box, type the name of the file you want to save.

  4. Click the Save button to save the document.
