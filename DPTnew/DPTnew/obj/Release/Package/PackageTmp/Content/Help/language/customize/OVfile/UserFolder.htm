User folder

  • C:\Documents and Setting\user name (Windows 2000)
  • C:\Documents and Setting\user name (Windows XP)
  • C:\Users\user name (Windows® Vista, Windows 7, Windows 8, Windows 10)
The C: disk drive is the O.S. default value. If the O.S. on your computer was installed to a different one, consider the latter as the disk drive.
