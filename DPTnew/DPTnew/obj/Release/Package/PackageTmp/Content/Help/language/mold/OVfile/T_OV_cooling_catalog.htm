Catalogs

The catalogs are set of standard parts provided by different vendors like D.M.E etc. Insertion of standard parts from the catalog is an important operation while creating a cooling system. In a cooling network the standard parts which are inserted into the system are Plug, O-Ring, and Nipple.

A list of installed catalogs will be available in the Catalog Type drop-down list under More Options in the Mold Cooling selection list. You will also find the following check boxes which on selection will insert the corresponding standard part into the cooling system.

Note
You have to check for the values of the standard part that is best suited for your design and select the same from the part catalog. The default values may not always work.

The following are the standard parts:

Plug Plugs are usually positioned in the path where the flow is to be blocked.
O-Ring O-rings are positioned between the plates to avoid leakage.
Nipple Nipples are positioned at the inlet and outlet of the cooling network in the plates.

The Refuge Hole is a hole (slot) is created at the position where nipple is inserted. The refuge hole allows the nipple to be inserted into the plate and provides housing to the nipple. In the default mode the Refuge Hole option is selected.
