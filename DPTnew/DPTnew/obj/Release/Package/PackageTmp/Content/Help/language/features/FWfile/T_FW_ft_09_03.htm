InsertSolidFilletSolid

The Solid Fillet command enables you to fillet all the edges of a selected solid. A single, constant radius fillet is applied to all the edges. Start the Solid Fillet command and select the model; the selection list is displayed. Enter radius in the Radius mini-dialog box to create the fillet. The other options in the selection list are discussed in the Edge Fillet.
