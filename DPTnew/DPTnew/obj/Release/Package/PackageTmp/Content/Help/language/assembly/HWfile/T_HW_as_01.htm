Creating a new component

1  Once you've started the command from InsertComponentNew (or choosing InsertNew Component from the context menu on the selected item, both in the Model Structure and in the Graphics Area), the program displays the New Component dialog box, from where you can select the entities that will be included in the component and introduce the component name.
2 Click Select, to select the entities to be included in the component, then End of Selection, to end the selection.
3 In the Name box, type the name you wish to assign to the component (or accept the default name assigned by the program).
4 Click OK to confirm your selections and create the new component.

To create a new component and transform it into an X-Reference component in a single step, perform the New Component as X-Reference command.
