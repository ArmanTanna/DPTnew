IGES Conversion Options

When saving (by Save or Save As) or loading (by Open or Insert from File) a IGES file by selecting IGES Format in the Save as type/Files of type drop-down list in the file selection box, the Options button displays the IGES Options dialog box. It enables you to define the parameters to be used in the conversion process that are present in the following categories:

  • If you are converting 2D data (select 2D in the Conversion type: drop-down list):
    • 2D
      This category contains the most common parameters to be set when converting to and from IGES.
    • 2D - Layers
      The controls in this category determine how the layers will be converted.
    • 2D - Text
      The controls in this category determine general aspects of the conversion of texts.
    • 2D - Dimension
      The controls in this category determine general aspects of the conversion of dimensions.
    • 2D - Geometry
      The controls in this category determine general aspects of the conversion of geometric entities.

  • If you are converting 3D data (select 3D in the Conversion type: drop-down list):
    • General
      This category contains the most common parameters to be set when converting IGES files.
      • General- Layers
        The controls in this category determine how the layers will be converted.
    • Load
      This category contains the most common parameters to be set when converting from IGES.
      • Log File
        The controls in this area determine general aspects of the file where the conversion events are logged.
      • Advanced
        The controls in this area determine advanced aspects of the conversion.
      • Healing
        The controls in this area determine how to deal with some conversion issues.
    • Save
      This category contains the most common parameters to be set when converting into IGES.
      • Log File
        The controls in this area determine general aspects of the file where the conversion events are logged.
      • Header Details
        The controls in this area enable you to customize the IGES file header.
      • Advanced
        The controls in this area determine advanced aspects of the conversion.
      • Healing
        The controls in this area determine how to deal with some conversion issues.
