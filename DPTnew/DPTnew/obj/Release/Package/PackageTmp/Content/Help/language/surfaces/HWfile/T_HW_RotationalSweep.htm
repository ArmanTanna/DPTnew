Creating a Rotational Sweep Surface

In this tutorial you will use the Rotational Sweep command to create a rotational sweep surface given some sections. Open the RotSweep.e3 file available in the samples folder of your think3 installation and follow the steps given below.



  1. Select each section curve for a different group, as shown below.



  2. In the Axis drop-down list select Line. Select the line to be used as the axis of rotation.



  3. Click on Drive Curves and select the curves to be used as drives.



  4. Select Preview ( ) to display the preview of the result.



  5. Apply the command to create the surface.
