The Set Ground command enables you to set the ground plane of the model. The command can be started by right-clicking in the Graphics Area and select ViewSet Ground from the context menu.

Note
This command is available in the context menu only when the application is in High Quality Shaded display.

This command is used when the ground plane need to be reset (that is the ground plane to be positioned under the model). This can also be set by using the Set from Current View option in the Document Properties - Ground category.
