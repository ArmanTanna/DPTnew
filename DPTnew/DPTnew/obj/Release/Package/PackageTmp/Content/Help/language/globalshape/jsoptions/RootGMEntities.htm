The Entities drop-down list under Entities to Be Modified enables you to select the type of entities the controlled modification is to be applied to:

  • Solid
    to apply the change to solid entities.
  • Surfaces
    to apply the change to surface entities.
  • Curves
    to apply the change to curve entities.
  • Points
    to apply the change to point entities.
  • Mesh
    to apply the change to mesh entities (see also "Display information about a selected Mesh" for further details on mesh entities).
