Alignment Methods of Label Dimensions

The think3 application enables you to align the label dimension:

  • when creating the new one
  • when modifying the existing one
When creating a new label dimension
  1. Click to activate the Reference for Alignment selector.
  2. Then, select the existing label(reference label) to which you want to align the new label.
    As you select the reference label, the program allows to move the new label only horizontally or vertically, while alignment is maintained with the reference label.



  3. Click on the final position
When modifying an existing label
  1. Select the label dimension to be modified.
  2. Then, select the reference label.
    As you select the reference label, the program permits to move the label you are modifying only horizontally or vertically, while alignment is maintained with the reference label.
  3. Click on the final position.


The reference point for alignment depends on the type of the label frame:
  • If the frame is circular, the reference point is the center of the frame.
  • If the frame is not circular or there is no frame, the reference point is the leader text/frame attachment point.

If the vertical distance between the reference points is greater than the horizontal distance, the labels will be aligned vertically and vice-versa.
