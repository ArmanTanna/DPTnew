Close a file

If you have changed the document after you opened it as a new document or you last saved it, as soon as you select the Close command, the program will display a dialog box, from where you can choose one of the following options:

Yes Saves the changes you made to the current document and closes it. If the document has never been saved before, you will proceed as if you were using the Save As command. See the description of the Save As command for further details.
No Closes the current document; any changes you made to it after you last saved it are lost.
Cancel Interrupts the command, so that the current document is not closed.

You can also close the document by clicking on the close box in the topmost right corner of the document window: the Close command will be started.
