Reference Manager

The Reference Manager utility helps you restore the references which belong to an assembly file.

The Reference Manager is automatically activated any time the program is unable to load a reference to a component file.

A file being referenced that has been moved from one folder to another or has been renamed are some typical examples of when this situation may occur.
