Enhancements in the Options/Properties Dialog Box

Version 2010.1 introduces the following enhancements in the Options/Properties dialog box.

  • Enhancements in Document Properties
    • Enhancement in the General - Advanced category
    • Enhancement in the Drawing - Dimensions category
    • Enhancement in the Drawing - Flat Pattern category

  • Enhancements in Entity Properties
    • Enhancement in the Drawing View - Flat Pattern category

Enhancements in Document Properties

Enhancement in the General - Advanced category

The new Tessellation area has been provided with the following three options for controlling the maximum edge length of the tessellation.

  • When the Enable max. edge length: check box is selected you are enabled to enter the maximum edge length value in the corresponding text box.
  • When the Skip planar surfaces check box is selected, the maximum edge length is applied only on curved surfaces, and not on the planar ones.
  • The effect is applied on selecting the Apply Changes button.

Enhancement in the Drawing - Dimensions category

The Automatic dimensioning area with the Enable for electrode origin check box has been introduced to perform the automatic dimensioning for an electrode.

You can specify the length of the reference axis in the new Axis length text box.

Note
These options are available only when you have installed the MoldDesign module from the custom window of the installation wizard during the installation process.

Enhancement in the Drawing - Flat Pattern category

The following enhancements have been made to support the new layers feature for bend data of a flat pattern view:

  • The new Color and Width options in the Bend area enable you to customize the color and width of the upward and downward bends.

  • The Layers area has been introduced to enable you to create bend lines, bend data, and bend table of the flat pattern view on different layers. This is exclusively designed to manage the visibility of layers in case of bulk flat pattern views.

Note
The settings specified in this category are applied only after regenerating the corresponding flat pattern view.

Enhancements in Entity Properties

Enhancement in the Drawing View - Flat Pattern category

The Bend table area with the Bend table check box and the view attributes (Text style:, Color:, Width:, and Line type:) have been introduced to make the bend table customization.
