System Options

Category: Input

General
General

The controls in this area enable you to define the settings of the input operations.

Dynamic highlighting
Dynamic highlighting

When selected, the program automatically highlights entities when the cursor moves onto them.




When the check box is not selected, entities are not highlighted when you move the cursor onto them.

Right-click keeps pre-selection
Right-click keeps pre-selection

When selected, wherever you right-click in the graphics area, the shortcut menu you get refers to the preselected entities (if there are any).When not selected, the shortcut menu refers depends on the position you right-clicked on.
Command confirmation in context menu
Command confirmation in context menu

(This setting applies only to commands without Selection List)
When selected, you can use the shortcut menu to exit a command.
Automatic query selection
Automatic query selection

If selected, when the cursor stops on a position in which the selection trap matches two or more entities, the query selection is automatically started: the Page Up/Page Down keys ( ) is displayed and you can use the corresponding keys to move your selection from one entity to the next or previous one.



You will have to confirm your selection by clicking the mouse selection button.



To disable the automatic query selection, just move the mouse away (see also Select Query).

When the check box is not selected, the automatic query selection is disabled.
Enable pickbox cursors
Enable pickbox cursors

When selected, the type of the cursor changes to the following:

Cursor when a selection is required
Cursor when a point input is required

Enable graphic area tooltips
Enable graphic area tooltips

When selected, the behavior is the following:

The AutoSnap cursor includes a tooltip indicating the type of snap point acquired



When you pause the cursor over an entity, a tooltip displaying information appears



When you pause the cursor over a constraint in the Profile environment, the constrained entities are highlighted and a tooltip containing the name of the costraint appears

When not selected, no tooltip is displayed.

Enable extended insertion
Enable extended insertion

If selected, all the insertion commands are displayed in the right-click context menu.



If not selected, only the insertion commands relative to the type of the selected entity will be displayed.



Set direction/axes using X, Y, Z
Set direction/axes using X, Y, Z

Enables you to choose how to define direction/lines as parallel to one of the axes of the Work Plane.

  • When the check box is selected, the direction/axis definition drop-down list will include:
    Parallel to X and through point To define the direction as parallel to the X axis of the Work Plane and through a point to be specified.
    Parallel to Y and through point To define the direction as parallel to the Y axis of the Work Plane and through a point to be specified.
    Parallel to Z and through point To define the direction as parallel to the Z axis of the Work Plane and through a point to be specified.
  • When the check box is not selected, instead, you can define the direction as parallel to one of the axes of the Work Plane just by choosing the Line option and by clicking on one of them as displayed in the image below.



    In this case by default the actual line will be the one through the origin of the system. However, you can change the origin by resetting the Origin selector and by specifying another point the line must be through, though retaining its direction.
    In this case the drop-down list will not include the X,Y,Z choices. Yet, once a Work Plane direction has been selected, the corresponding letter (X),(Y),(Z) will be displayed adjacent to Line.

See "How to Define an Axis or a Line" for further details.
Select through transparency
Using Transparency

When the check box is selected, it enables you to select the objects beyond the transparent entities. The cursor highlights the objects for selection further than the transparent entities.
When the check box is cleared, you cannot select the objects beyond the transparent entities.
Select chain of connected curves
Select chain of connected curves

  • When the check box is selected the command selects curves which are geometrically connected to each other
  • When the check box is cleared the curves that appear to be connected from a particular orientation of view is selected. These curves need not be geometrically connected.
Mouse Input
Mouse Input

The options in this area enable you to control the input of points when using the mouse. See "Tips on Work Plane settings" for further details.

On view
On view

When this mode is selected, all of the input operations you perform using the mouse will apply to the view plane, according to the following rules:

1 The view plane is parallel to the screen and is through the origin of the Work Plane.


 
2 A point freely specified using the mouse just by clicking somewhere (from now on: free points: points entered with no snap command or further constraint) is created on a plane that is parallel to the view plane at the Depth: distance.
3 If a point is specified using a snap command, than the view plane is through the snapped point and parallel to the screen. All free points specified afterwards — within the same command — will lie on a plane that is parallel to the view plane at the depth distance.
4 After a rotation, the orientation of the view plane changes accordingly.
5 When a command is quitted, the view plane is reset as described at point 1.
 

For details on how this mode affects the behavior of several drafting commands, see "Using drafting commands when the input mode is On View".

On Work Plane
On Work Plane

This option enables you to sketch on the current XY plane. When this option is selected the points you specify using the mouse will lay on that plane, but those you snap to 3D points will really be 3D.

Use this method, for instance, when adding points - making up curves - to the face of a solid, after first moving the Work Plane to the face. You can then use the curve as a profile from which to create a feature, such as protrusion or shaped hole.
Projected onto Work Plane
Projected onto Work Plane

When this option is selected, all points including snap points are automatically placed on the Work Plane. This option is set automatically when you are in profile mode.
This setting can be very useful, but we recoomend that you be careful with it since it can become tedious, especially when creating 3D splines or other 3D curves.
