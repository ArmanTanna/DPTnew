This check box enables you to manage sectioning in derived views obtained from sectioned views:

  • When the box is cleared, the derived view is kept as it is, that is a sectioned one.
  • When the box is checked, sectioning is removed from the derived view, so that it isn't a sectioned view any longer.
    Please note that it is not possible to undo this action and thus the view will remain a non-sectioned one.
