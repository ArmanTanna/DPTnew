Creating Fillets on a Selected 3D Model

The Solid Fillet command enables you to create a constant radius fillet on all the edges of a selected solid. To do so, open the Edge_Fillet_Simple.e3 file available in the Samples folder of your installation and follow the steps given below.

  1. Start the Solid Fillet command; you are prompted to select a solid. Click on the solid you want to fillet for the Solid selector, as shown in the image below.



  2. Enter 5 in the Radius mini-dialog box to specify the fillet radius.

  3. Click on More Options to expand it and select the Continuous tangency check box from the Use Capping drop-down list.

  4. Click or to confirm your selections and fillet the edges of the solid, as shown in the image below.
