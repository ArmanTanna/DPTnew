How to use Adaptive Measure

In order to activate Adaptive Measure, right-click on the mini-dialog box and select Enable Adaptive Measure from the context menu.

In order to complete the execution of Adaptive Measure, you have to:

  • Supply all the input arguments that are marked in red () in the Select drop-down list.
    (For further details on the use of Adaptive Measure, see "Overview of Adaptive Measure.")
  • Select the necessary argument from the list. When you provide the inputs, the red marker turns green.
  • Confirm Adaptive Measure by pressing TAB or selecting another mini-dialog box or selecting
    OK () or Apply () (or sometimes Rebuild () also while rebuilding the model).

Before confirming, you can always, change the selected entities by right-clicking on Select and select Reset.

After you confirm, if you can see the ruler icon () next to the corresponding feature in the Model Structure, the measure of the feature is associated with the selected measure i.e. it will change with the associated feature.

If you do not see this icon, the measure of the feature is not associated with the selected measure, but only the value of the measure is used.

For example, when you use a drafting command (in 2D Profile environment), say creating a Circle given Center, you enable adaptive measure, select a radius (from other model or profile) and press TAB to create the circle. The circle will be created with the value of the selected measure, but it is not associated with the selected measure.
