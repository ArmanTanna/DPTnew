Defining the Reference Model

1  When you start the Edit Reference Model command, the think3 application displays the Edit Reference Model dialog box.

2 Specify the model you want to define as the reference model for the current drawing by doing one of the following:
  • From the drop-down menu, choose one of the models that is currently open.
  • Click the Browse button and select the model file you want to reference.

3 You can prevent the reference model from being modified from the drawing. For example, if the drawing views you will insert into the drawing contain drive dimensions, you might want to prevent them to be changed, so that the model will not be modified. To so, select the Open with read-only access check box.

4 Click OK to confirm.

 Rebuild the drawing to see your changes!

If a drawing was already associated with the model, you must rebuild the drawing or the views before you can see the effects of your change. After rebuilding the views, all of the views, sections, details, and so on that were in the drawing remain in the same position as they were previously, but now reference the model you specified.
