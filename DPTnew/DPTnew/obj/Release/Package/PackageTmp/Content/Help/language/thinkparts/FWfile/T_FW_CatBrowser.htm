The Catalog Browser command is the thinkparts run-time module that enables you to consult the catalogs in a standard part library.

The Catalog Browser command opens the thinkparts Catalog Browser main window which is composed of two frames: the Catalog Tree at the top and the Preview frame at the bottom.

The Catalog Tree frame displays the hierarchical structure of the set of all catalogs that are contained in the database and that are available to the user. The red closed books in the tree mark the indexes. Each index contains a set of indexes or catalogs, while a catalog (marked by a gray open book in the tree) is a set of parts. In the standard catalogs that come with the think3 application, the Phillips head screws and Hexagon head screws are examples of catalogs of parts, while SKF bearings, Screws and Washers are indexes.
A set of commands is available in the context menus that pop up when you right-click on one of the objects listed in the frame.

The Preview frame displays the documents associated to the currently selected object. The documents can be in the form of images, such as the component previews, Microsoft® Word or any other document format with, for example, technical information on the components of the catalog. By double-clicking on the document icon the associated application will open enabling you to access the document contents.
