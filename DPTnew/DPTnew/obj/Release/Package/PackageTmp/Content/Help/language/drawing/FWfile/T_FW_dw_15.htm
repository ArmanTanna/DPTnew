ModifyDrawing ViewHide Dimensions Axes Edges


The Hide Dimensions Axes Edges command enables you to hide the selected dimensions, axes or edges in a view. You can do multiple selection of all these entities using the selection window. And this command works on most of the views like section views, Auxiliary views, detail views, etc. The command can be operated only when you select entities of the same type in preselection mode, if not the program warns you displaying a warning message.

The number of hidden entities will be shown as a part of description in the statusbar.

Note:
This command works only on the Driving dimensions but not on Driven ones.
