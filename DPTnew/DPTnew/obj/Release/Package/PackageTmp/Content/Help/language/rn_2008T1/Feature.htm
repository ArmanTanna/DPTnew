New Feature Functionalities and Improvements

Version 2008.1 introduces the following enhancement to the Feature functionalities.

  • Enhancement in the Linear Protrusion and Linear Slot commands
  • Enhancement in the Linear Solid command
  • Enhancement in Boolean behavior of Simple Hole and Simple Shaft commands
  • Enhancements in the Solid from Faces command
  • The new Joint Lip command
  • Improvement in the behavior of the Edge Fillet command
  • Improvement in the behavior of the Edge Chamfer command
  • Improvement in the behavior of Interactive Modeling commands
  • Enhancement in the Interactive Modeling Move face command
  • Enhancement in the Make Solid command
    • New Simplify Faces check box for face simplification.
    • New Default layer check box for error management

Enhancement in the Linear Protrusion and Linear Slot commands

Significant enhancements have been made in the Linear Protrusion and Linear Slot commands.

  • It is now possible to create a protrusion/slot in both the directions of the selected profile. To facilitate this, new options are available in the Extension drop-down list of the Linear Protrusion and Linear Slot selection list.

    In the Linear Protrusion command, choose the new Double depth option from the Extension drop-down list and select the profile as shown in the image below.



    Then the depth values can be specified in the Depth 1 and Depth 2 mini-dialog boxes to create the protrusion in both the directions of the selected profile. The depth direction can be inverted by double-clicking on the arrow handle.


    In the Linear Slot command, Blind double depth has been introduced in the Extension drop-down list. The slot can be created on both the directions of the selected profile.



    From this release, the Blind option is now called Blind single depth (the functionality remains the same).

  • From this version you can create a protrusion/slot with the required offset distance from the profile plane. The Offset check box is visible on the preview of the selected profile by expanding the More Options category. The Offset check box is available also in the Linear Solid Command



    Note:
    The Offset check box is not available for the From-to and Double depth options of the Extension drop-down list in both the commands.

Enhancement in the Linear Solid command

The Depth+Offset option has been removed from the Extension drop-down list from this version. Instead, an Offset check box is available on the preview of the selected profile, when the Single depth option is selected from the Extension drop-down list.



Although the functionality is unchanged, the changes have been made to keep the user interface consistent with the enhancement made to the Linear Protrusion and Linear Slot commands.

Enhancements in the Solid from Faces command

From this version onwards it is possible to select a solid whose faces will be used to create the static (or the parametric) solid while applying the Solid from Faces command. This is extremely useful in case of complicated solids with many faces and all its faces need to be selected.


To facilitate this enhancement, the Mode drop-down list has been made available in the Solid from Faces selection list. There are two options in it:

  • Local: Individual faces can be selected for creating the resulting solid.
  • Global: All the faces of a selected solid are used to create the resulting solid.



On selecting the Global option, you are prompted to select a solid. On making the selection, all its faces get automatically selected for the command's application.


On selecting the Local option, you are prompted to select the faces. You can individually select the faces or make use of the Semantic Selection functionality.


Note
Another significant enhancement in this command is the feasibility of face (and solid) selection belonging to any level in the assembly hierarchy. In the earlier versions, only the faces at the current level were selectable. Hence, this enhancement is very useful in situations where you need to do a boolean between solids belonging to different assembly hierarchy levels. You can define the boolean operation at the preferred level by selecting the copies of the required solids created in the same level by the Solid from Faces command.

Note
The Linked copies option is also valid for multiple hierarchy selections. As a result you can create associative solids related to faces/solid present at any level of the assembly hierarchy. The Linked copies option can also be used with both the selection conditions (Local and Global).

Backward compatibility

2008T1 models containing Solid from Faces events become no longer compatible with older version ONLY if:

  • the event has been generated by selecting a solid instead of individual faces
  • the event has been generated by selecting faces/solid present at different levels of the hierarchy

The new Joint Lip command

This version introduces a new InsertSolidJoint Lip command. Using this command it is possible to create a lip feature. The same command is also used for creating a groove. The difference being that for creating the lip feature, material is added and for the groove, material is removed. The shape of the lip is defined by a profile. To generate the lip feature, the profile is moved along a chain of solid edges.


The situation is highlighted hereafter. You may need to create a lip/groove on a face(s). To define the shape of the lip/groove, you need to create a profile. Refer to the image below.



After invoking the Joint Lip command, you need to select the profile (for the Profile selector) and the solid edges (for the Drive Curves selector) along which the profile will sweep itself, as shown in the image below. Associative curves on surface can also be selected as drive curves. You can specify the method to be used to perform the sweep along the drive curve by using the Motion Mode drop-down list.



Note
You can make use of Smart Selection Commands such as Tangent Chain to select the drive curves. This way the associativity between the selected edges and the lip feature can be maintained.

Depending upon whether you decide to add or remove material using the Material drop-down list, a lip or a groove will be created respectively and the same will be displayed in the Model Structure, as shown in the images below.

Improvement in the behavior of the Edge Fillet command

There are improvements in the behavior of the Edge Fillet command.

  • From this version onwards, the Edge Fillet command has been improved so as to have the vertices terminate perfectly when filleted, also in the case when the edge of a solid is filleted by not selecting the Tangency Chain check box in the Edge Fillet selection list.



  • In the former versions, the fillet created was not terminating at the vertices as shown below:


    Similar behavior was also occurring while using the Edge Chamfer command. This is now managed correctly from this version onwards.

  • From this version onwards, when the Tangency Chain check box is selected, the tangency chain selection terminates when there is change in the concavity.



    In former versions the selection curve would not terminate even when there was a change in the concavity.



Improvement in the behavior of the Edge Chamfer command

From this release onwards the direction arrow is visible immediately after selecting the edge for chamfering using the Distance+Angle option in the selection list of the Edge Chamfer command.



When the Edge Chamfer command was used to chamfer a high order curve/surface or a edge where the angle of the faces were not constant, the resulting chamfer would be weak or possibly not able to create a chamfer in some situations.
From this release improvement has been made in the Edge Chamfer command and now chamfers correctly in all complex situations.

Enhancement in the Interactive Modeling Move face command

A new Associative Transformation check box has been introduced in the Move Faces command selection list. This check box is available only after the face to be moved is selected.

When the check box is selected, the resulting transformed entity is associated with the original entity. Now when the original is again moved, then even the entity associated with it also moves accordingly.
When the check box is cleared the resulting transformed entity is not associated with the original entity.


Enhancement in the Make Solid command

The following enhancements have been made in the Make Solid command.

New Simplify Faces check box for face simplification.

The new Simplify Faces check box has been made available in the Make Solid selection list.

  • Select this check box to simplify planar faces (that is create one single planar surface from each group of different planar surfaces having coplanar and in positional continuity.
  • If this check box is cleared, no face simplification will take place.

Note
During redefinition of the Make Solid feature, the Simplify Faces check box will be displayed but not editable. Also, it will be in the enabled state if and only if the Make Solid feature was previously applied in the associative mode along with face simplification.

New Default layer check box for error management

While applying the Make Solid command, if any error arises due to the generation of an open solid, a dialog box is displayed in order to give the user the possibility to move the wrong entities onto a selected layer. To make this functionality compatible with the enhancements made to the Layer Management, the dialog box now shows the Default layer check box and a drop-down list, as shown in the image below.


  • When the Default layer check box is selected, the wrong entities will be moved to a new layer with a default name (for example "AutoLayer_ MakeSolid_Small", where AutoLayer_ MakeSolid_ is prefixed to the nature of the error).
  • When this check box is cleared, the edit box and drop-down list are enabled. The user can either select an existing layer from the drop-down list or enter the name of a new layer in the edit box to move the wrong entities onto it.
