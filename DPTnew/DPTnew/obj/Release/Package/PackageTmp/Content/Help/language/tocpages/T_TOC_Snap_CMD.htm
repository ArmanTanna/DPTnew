Snap

End Point Enables you to select the end point of a curve or line
Surface Vertex Enables you to select the vertex of a surface
Mesh Node Enables you to select a node of a mesh
Arc Center Enables you to select the center of a circle or arc
Text Origin Enables you to select the origin of a text entity
Group Origin Enables you to select the origin of a group entity
Intersection Point Enables you to select the intersection between two curves
Mid Point Enables you to select the midpoint of a curve or line
Point on Curve Enables you to select any point projected onto a curve
Joint Enables you to select the arc joint of a NURBS curve
Intersection on Extensions Enables you to select the point at which two lines would intersect if extended
Perpendicular Enables you to select the perpendicular to a selected curve
Tangent Enables you to select the tangent to a selected curve
Work Plane Origin Enables you to select the origin of the Work Planecoordinate system
Enable Point Coordinates Enables you to specify a point using its Work Plane coordinates
Reference Grid Enables the cursor to snap to the nodes of the Reference Grid
AutoSnap Toggles the current automatic Snap mode
Snap to None Disables AutoSnap for the selection of the next point
Customize AutoSnap Toggles the current automatic Snap mode
