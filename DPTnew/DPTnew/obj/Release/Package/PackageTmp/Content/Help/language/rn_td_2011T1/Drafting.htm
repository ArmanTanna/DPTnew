New Drafting & Dimension Functionalities and Improvements

Version 2011.1 introduces the following enhancements to the Drafting & Dimension functionalities.

  • Usability improvements in the drafting commands
  • Usability improvements in dimension commands
  • New Part List formatting options

Usability improvements in the drafting commands

The following usability improvements have been done in the drafting environment:

  • Adding custom format field to dimension format
  • Double-click on label text to edit it
Adding custom format field to dimension format

The new Customized Text sub-category has been added under the Dimension category to the Organize Formats dialog box. It contains the new Customized text check box. When selected, the customized text (entered in the edit text box) will be displayed as part of the dimension created. The preview area, provided on the right-hand top portion of the dialog box, enables you to have an updated view and also control the effect of the changes made.

Double-click on label text to edit it

You can now edit labels with a double-click on the entity. The Label Text dialog box will be displayed when you double-click on a label. You can specify single line text (Up/Down) as well as multiline text in this new dialog box.

Usability improvements in dimension commands

Significant usability improvements have been made in the drawing environment. They are:

  • New representation of an unrelated dimension
  • Redefining the missing references
New representation of an unrelated dimension

All the unrelated dimensions are much easily located, even on the printed paper, with a cross along with the existing non-persistent color highlighting. The new representation is supported with the introduction of Cross unrelated dimensions option in the Options/Properties dialog box.

Note
The new representation of unrelated dimension is visible only after:
  • activating the new option and in event if the option is already activated
  • starting the Highlight Dimensions command, (CTRL+H)
  • loading a document
  • regenerating a drawing layout
Redefining the dimension with missing references

It is now possible to recreate the dimension with missing references with the help of the new Reinsert Dimension command in the context menu presented by an unrelated dimension. The dimension type will be explicitly declared in the command title inside the selection list.

Dimensions will be recreated for the following dimension types:

  • Line length
  • Arc length
  • Distance
  • Half distance
  • Angle
  • Diameter
  • Radius

New Part List formatting options

Two new formatting options have been made available for Part Lists in the Part List Options dialog box.

  • The new Fit/stretch automatically check box in the Table Format-Text tab:
    • When selected, texts will be stretched or extended so as to fit into the cell.
    • When cleared, texts will be left as they are and may exceed the length of the cell.
  • The new Row/text height ratio box in the Table Format-Advanced tab enables you to specifiy the row height as a ratio with respect to the text height.
