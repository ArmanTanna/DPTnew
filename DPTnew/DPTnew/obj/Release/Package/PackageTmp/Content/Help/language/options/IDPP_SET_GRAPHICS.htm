The controls in this area determine the quality settings for the graphics representations.

Silhouette:

Silhouette:

This option controls the visualization of silhouettes within the program.


The silhouette visualization mode is based on tessellation. In Shaded modes, the program has to perform tessellation computation. In Wireframe mode the program generally skips tessellation computation. Once the program has determined the tessellation, the silhouette mode is readily available.See Wireframe and Shading.


Default The silhouettes are visualized only in Shaded modes, in which mode, the program has to perform tessellation computation. So when this option is selected, silhouettes will not be not visualized in the Wireframe mode.
None The silhouettes rendering is disabled altogether, in any mode.
Always Force the program to render silhouettes every time, even in Wireframe mode, in which mode the program generally skips tessellation computation.
Note, however, silhouettes are not rendered during dynamic transformations (i.e.; Pan, Zoom, Rotate operations).

Silhouette

The silhouette of a surface is normally composed of:

  • The apparent contours of the surface — the curves defined as the sets of the points at which the normal to the surface is parallel to the view plane.
  • The real boundaries of the surface — the isoparametric curves with the lowest and highest parametric values, or in the case of surfaces with parameter limitations, the limiting curves which do not coincide with other boundaries of the surface.
  • The edges of the surface (if any) — the isoparametric curves defined as the sets of the points where the surface has only positional continuity.
For further details, see Silhouette.

Backface culling

Backface culling

When selected, the program ignores faces whose normal points away from the current viewpoint. Selecting this check box can improve the redraw speed of shaded images.

When not selected, all faces are taken into account, which can slow the redrawing operation down in the case of complex shaded images.

Shading in surface highlight

Shading in surface highlight

When selected, moving the cursor onto a shaded surface or solid face causes the color of the surface or solid face to change.

When not selected, only the edges of the shaded surface or solid face are highlighted when you move the cursor onto them.

Dashed hidden line mode

Dashed hidden line mode

When selected, the program displays all the hidden lines as dashed. This applies only when using view modes displaying hidden lines (see "Wireframe and Shading").

When the box is not selected, hidden lines are displayed as continuous. This can be useful to more clearly display complex models, where dashed lines might result confusing.

Black borders

Black borders

When checked, all edges and isoparametric curves will be displayed in black, no matter the color of objects.

When not checked, edges and isoparametric curves are displayed using the color defined in the Wireframe area of the Advanced Color Editing dialog box which you can access selecting Advanced in the Color and Materials dialog box.

Curve antialiasing

Curve antialiasing

When selected, anti-aliasing on curves and lines (including iso-parametric curves) is enabled.

When not selected, anti-aliasing is not enabled.


Note
  • Anti-aliasing is a technique which enables lines and curves to be smoothly displayed, avoiding aliasing effects (that is: the appearance of jagged distortions). It is only applied to the line and curve entities of the model (including iso-parametric curves).
  • When the Curve antialiasing check box is selected, avoid activating further anti-aliasing effects using the graphics display driver options.

Automatic working space:

Automatic working space:
  • When this box is selected, the correct values for both near and far clipping planes are automatically re-computed after each pan-zoom-rotation action, taking into account the bounding box of the whole scene.
  • When it is not selected, the old computation method is enabled and you will have to set the proper value in the adjacent box.
