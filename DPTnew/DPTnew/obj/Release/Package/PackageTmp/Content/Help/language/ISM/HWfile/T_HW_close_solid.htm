Closing an Open Solid

The following illustration explains a situation where you will close an open solid by using the Interactive Modeling command. Open the ISM_closure.e3 file available in the Samples folder of your think3 installation and follow the steps given below.

  1. Invoke the Extend Faces/Close Solid command.

  2. Select the open solid for the Base Skin selector.



  3. Ensure that the Mode has been set to Global.

  4. Apply the command to form a perfectly close solid.

In some cases the default extension is not enough to close the solid. In such case the command fails but it is possible to increase the extension by specifying a value for the Extension Factor.
