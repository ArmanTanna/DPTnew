Improvement in the Underconstrained Profiles command

In this version, the Underconstrained Profiles command has been disabled in the 2D and 3D profile environments and is only available in the Model environment.


This has been done because the 2D and 3D profile environments allow profile editing and can distinguish between fully constrained and under constrained entities by their color.
