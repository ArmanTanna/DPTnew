Symbolic Texts

Texts and dimensions may include symbolic annotations which enable you to display the values of user defined properties or spreadsheet variables.

Although you can use them anywhere, symbolic expressions are typically used in Title Blocks and Part Lists.

You can control the display of symbolic annotations — deciding whether to display the value of the annotation or the string that describes it — by selecting/deselecting the Symbolic text expansion check box in the Symbolic texts area of the Text category in the Document Properties. After selecting or deselecting this check box, click the Redraw Texts button in order to update the display.
