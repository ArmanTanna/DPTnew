InsertCutting Plane LinesDefine on Drawing

The Define on Drawing command enables you to create a cutting plane line and position it in a two-dimensional drawing view. You can then use the cutting plane line to create cross-section, sectioned, and developed views.

Note

This command is not available if the drawing doesn't contain any views:

  • Section name can be redefined by changing it in the Name: box in the Cutting-plane Line category of the Edit Properties.
  • When the section is automatically created by the program (by selecting the Automatic names check box in the Drawing - General category of the Document Properties), you can set a different initial value (default is "A"), so that the section names will start for example from E, F or whatever value you specify in the Start from section: box. The automatic naming mechanism assigns sections names from A to Z, then AA to ZZ, AAA to ZZZ and so on.
