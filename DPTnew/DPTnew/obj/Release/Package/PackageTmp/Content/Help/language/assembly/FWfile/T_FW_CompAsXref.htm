The New Component as X-Reference command enables you to create a new component and transform it into an X-Reference component.

The command corresponds to the New Component and Make X-Reference commands, carried out in a single step.

The command is available exclusively from context menus and can be applied to one or more entities selected with the ordinary pre-selection modalities.

Once you've selected the item(s), right-click and select InsertNew Component as X-Reference from the context menu. A file selection box is displayed, from where you can save the component. A component name is proposed. You can accept it or type a different name. When the saving is completed, an X-Reference component is created and you get visual feedback by its icon being updated in the Model Structure.
