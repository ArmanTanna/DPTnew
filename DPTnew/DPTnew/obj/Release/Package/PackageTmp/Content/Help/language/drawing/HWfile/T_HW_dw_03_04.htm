Inverting the Direction of a Cutting Plane Line

Start the Invert Section command. You are prompted to select a section. Click to select the cutting plane line whose direction you want to reverse.



(Note that if you start the command using the shortcut menu, there is no prompt since the section is already selected.)

As soon as you select the section, the direction of the cutting plane is immediately reversed.
