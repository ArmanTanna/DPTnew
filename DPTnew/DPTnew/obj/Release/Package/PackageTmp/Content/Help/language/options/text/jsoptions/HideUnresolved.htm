This item — enabled only if the Symbolic text expansion check box is selected — enables you to control the cases when a symbolic text should be unresolved — that is: should still display the symbolic annotation, not being able to replace it with a proper value. In such cases you can decide whether to display it or to hide it.

  • If the check box is selected, the unresolved symbolic text is not displayed.
  • If the check box is cleared, the unresolved symbolic text is displayed (that is: the symbolic annotation string is displayed in the place of the value).

Please note the option applies to any unresolved symbolic texts, no matter whether included in wider text strings or not.
After selecting or deselecting this check box, click the Redraw Texts button to update the display.
