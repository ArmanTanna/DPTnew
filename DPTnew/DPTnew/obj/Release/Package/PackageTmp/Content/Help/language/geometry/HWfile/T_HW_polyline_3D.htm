Creating a 3D Polyline

In this tutorial, you are going use the Polyline command to create a 3D profile. Open a blank Model (*.e3) document and follow the steps given below.

  1. Start the 3D Profile command. You will notice that the Profile Reference System is displayed on the Graphics area.

  2. If not already visible, activate the display of the Work Plane by using the ViewWork Plane command. Now, change the view to a left-front orientation by using the Left Front View command. Also activate the Static Grid to assist in point selection during the profile creation. The Work Plane should appear as shown in the image below.



  3. Start the Polyline command and ensure that the Work Plane on Last Point check box is selected in the selection list.

  4. Use the Work Plane Origin snap to select the origin as the first point of the polyline.

  5. Move the cursor in the positive X-direction and click on the next point to create a line segment of 50 units. You can make use of the mini-dialog boxes too.
    All references of direction are with respect to the Work Plane.



    You will notice that the Work Plane origin immediately moves to the last selected point. However, the orientation of the Work Plane does not change.

  6. Click on the next point such that the line segment is of 50 units in the positive Y-direction.



  7. Press Q on the keyboard to start the Select command. Notice the various markers displayed on the Work Plane. Double-click on the rotation marker over X-axis; the Work Plane rotates into its new orientation.



    Press Q again to exit Work Plane editing and return to polyline creation.

  8. Click on the next point such that the line segment is of 50 units in the positive Y-direction.



  9. Continue creation of more line segments as shown in the following images.



  10. Once you reach the starting point of the profile, it gets closed and the Polyline command exits automatically.



    Note that some line constraints are displayed over the created lines. These constraints are applied in reference to the Profile Reference System.
