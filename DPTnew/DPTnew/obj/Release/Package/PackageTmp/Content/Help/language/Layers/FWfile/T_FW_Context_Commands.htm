Layers Tab: Context Menu Commands

In the Layers tab, you can use the context menu to access the primary commands used to edit the layers. They are accessible from context menu in the following ways.

  • When you right-click in an empty area of the Layer tab frame, the program displays the following context menu:



    • New Layer
      to create a new layer
    • Filter Layers
      to define the options for the layers search filter
    • Select Layers
       to select only the layers that meet the chosen selection criteria
    • Refresh
      to refresh the layer list by regenerating its contents.


  • When you right-click on a empty layer, the program displays the following context menu:



    • Set Current Layer
      to make the selected layer current
    • Delete Layer
      to remove the selected layer from the layer list
    • Filter Layers
      to define the options for the layers search filter
    • Select Layers
      to select only the layers that meet the chosen selection criteria
    • Rename Layer
      to change the name of a layer in the list.
    • Refresh


  • When you right-click on a layer with entities, the context menu provides access to the following commands:



    • Set Current Layer
      to make the selected layer current
    • Delete Layer
      to remove the selected layer from the layer list
    • Select Entities
      to select entities
    • Filter Layers
      to define the options for the layers search filter
    • Select Layers
      to select only the layers that meet the chosen selection criteria
    • Move Entities
      to move entities from one layer to another
    • Exchange Layers
      to exchange the layer content
    • Rename Layer
      to change the name of the selected layer.
    • Refresh


  • When you right-click on a locked layer, the context menu provides access to the following commands:



    • Set Current Layer
    • Delete Layer
    • Filter Layers
    • Select Layers
    • Protect Layer
    • Rename Layer
    • Refresh
