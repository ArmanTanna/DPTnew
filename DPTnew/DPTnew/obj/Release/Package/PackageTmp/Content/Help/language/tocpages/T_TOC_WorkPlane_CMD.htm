Work Plane

On Entity Enables you to edit the Work Plane
Select Enables you to perform quick editing of the Work Plane
Unlink Breaks the links that associate the Work Plane with an entity
On Absolute Reference System (World) Sets the Work Plane to the one of the world
On View Sets the Work Plane parallel to the active view
On Face Moves the Work Plane onto the face of a solid
On Profile Moves the Work Plane onto a profile
Through 3 Points Defines the Work Plane based on three specified points
Align X-Axis Aligns the X-axis of the Work Plane with a line in the XY-plane
Align Z-Axis Aligns the Z-axis of the Work Plane with the direction indicated by a specified line
Move Enables you to interactively move the Work Plane on the screen
On First Axis Positions the Work Plane on the first plane of the Reference Grid
On Second Axis Positions the Work Plane on the second plane of the Reference Grid
On Third Axis Positions the Work Plane on the third plane of the Reference Grid
Work Plane Here Positions the Work Plane on a planer face or a datum plane.
Work Plane on Surface Positions the Work Plane on a surface.
Work Plane on Curve Positions the Work Plane on a curve.
