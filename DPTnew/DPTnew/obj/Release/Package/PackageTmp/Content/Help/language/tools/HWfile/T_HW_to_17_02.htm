Spreadsheet - Other options used to control variables

You can also use the following options to control variables:
List To view a list of all the defined variables and their values, open the Spreadsheet dialog box. The Spreadsheet dialog box also contains commands which force the updating of variables when this is not done automatically
Modify To modify the value of a variable:
  • select the variable you want to change, display the Edit Driving Dimension dialog box, and then change the value
  • display the Spreadsheet dialog box, select the value, and type the new value
Deassign To deassign a variable (delete the name of a variable without changing its current value):
  • select the variable you want to deassign, display the Edit Driving Dimension dialog box, and delete the variable name without changing its current value
  • display the Spreadsheet dialog box, select the line number corresponding to the variable you want to deassign, click Deassign, and then select the entity from which you want to deassign the variable.
When you deassign a variable, the actual dimension value appears in the drawing rather than the variable name.
Rename To rename a variable, display the Spreadsheet dialog box, select the variable name, and then type the new name
Delete Variables To delete a variable from the list of existing variables, display the Spreadsheet dialog box, select the line number corresponding to the variable you want to delete, and then click Delete Variables. Note that to delete a variable that is used in an expression which defines another variable or dimension in the model, you must fist deassign the variable and those dependent on it before you can delete the variable.
Expand To expand variables (display their values or names in dimensions and text), select the Text tab in the Options dialog box. Then, under Symbolic Texts, choose the desired option for Show variables as (in dimensions) and Show variables as (in texts).
Highlight To highlight features corresponding to selected variables, display the Spreadsheet dialog box, right click on the line number corresponding to the variable whose features you want to highlight, and select the Show Dimensions/Features. Any features which have that variable assigned to them are highlighted in the model.
