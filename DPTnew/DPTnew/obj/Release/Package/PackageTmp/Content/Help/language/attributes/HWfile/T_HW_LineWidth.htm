Setting and Modifying the Line Width

The drop-down list button enables you to:

  1. Set the current Line Width
  2. Modify the Line Width of selected entities

Using the button inside commands
While mode 2 (changing the Line Width of selected entities) can be used only when no command is active, mode 1 (setting the current Line Width) is available also inside commands, which can be very useful. Suppose, for example, you have started the Parallel Line command: once you have selected the reference line, you can change the current Line Width so that it will be applied to the resulting parallel line, which will be drawn using the new Line Width you chose.

Setting the current Line Width

The current Line Width is the width used to draw entities on the screen. For example, if the current Line Width is Line Width 2, all entities you create will have Line Width 2. You can obviously change the current Line Width. When you do it, all entities created after you changed it will be created with the new Line Width (while the ones created before you changed it will not be affected by the change).

Line Width on the screen and Width for printing
Please note that the Line Width drop-down list button enables you to set or change the value of the width used to display entities on the screen.
The Width values to be used for printing, associated with colors, can be set as described in "Set entity width".

To change the current Line Width, proceed as follows.

1 Right-click on the scrolling arrow to display the set of available Line Widths.

2 Two alternative methods are available.
a)
In the drop-down list, select the Line Width to be set as the current one.
b)
Select an entity in the document. The Line Width of the selected entity will be set as the current one.

The Line Width you chose is now the current one: all entities subsequently created will be drawn using this Line Width.

Note that the current Line Width can also be modified using the Line Width or Options/Properties commands.

Modifying the Line Width of selected entities

To modify the Line Width of existing entities, proceed as follows.

1 Select the entities you want to change the Line Width of.


Note that if the selected entities have different Line Widths, the Line Width drop-down list button will display no Line Width value.
 
2 In the Line Width drop-down list, select the Line Width to be applied.

The Line Width of all the selected entities will be changed.

Please note that the Line Width attribute of an entity can also be changed using the Edit Properties and the Assign Properties commands.
