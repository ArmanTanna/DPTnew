thinkparts commands

Catalog Browser Enables you to browse the components catalogs
Catalog Manager Enables you to customize the components catalogs
Create Part Family Enables you to create new catalogs
Find in Catalog Enables you to locate the catalog where the selected part belongs
Import from thinkparts Catalog Enables you to quickly import another instance of the selected part from the thinkparts catalog into the current think3 document
New Standard Part Enables you to quickly insert another copy of the selected component
Open thinkparts Catalog Opens the thinkparts catalog that contains the selected part
Replace Standard Part Enables you to replace one or more components in an assembly with one taken from a thinkparts catalog
Symbolic Mating Enables you to define the part mounting sequence
Unlink Standard Part detaches the selected standard part from its thinkparts catalog and converts it into a think3 local component
