Display information about a selected Plane

If you select a plane, the program displays a dialog box showing:

  • Coordinates of the plane Origin
  • Components of the plane Normal
  • If the plane is bounded, the Trimmed check box will be selected and the Limiting curves and Limiting islands boxes will show the corresponding values.

And also displays arrows within the model indicating the two parametric directions.

If you select the Area check box at the bottom of the dialog box, the program also displays the surface area.
