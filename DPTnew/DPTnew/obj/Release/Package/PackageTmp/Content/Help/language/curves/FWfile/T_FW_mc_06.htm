Modify Curve Invert Orientation


The Invert Curve Orientation command enables you to invert the orientation of one or more selected curves. This can be helpful in all cases where the orientation of the curve is important (as when creating ruled surfaces, surfaces from connected curves, and so on).

First select the curves to be modified (the Curves item in the selection list will turn to green) and then end selection by right-clicking and selecting Continue in the context menu.

Inverting the orientation of a curve means to invert the direction of its parameter: the first endpoint becomes the second, and the second the first. In the case of NURBS, the order of the control points is also reversed.

Once the curves have been selected, an arrow is displayed to show the current orientation of each curve. Move the cursor onto each of the arrow and double-click to invert the orientation. The change will be immediately displayed.

The Associative Mode check box enables you to create curves which retain a link to the surfaces/curves they derive from.

When the check box is selected, the resulting curves will be associative. The new curve will retain a link to the entities it derives from. A specific node displaying both the original entities and the resulting associative curve will be added to the Model Structure.
When the check box is cleared, the resulting curves will not be associative.
