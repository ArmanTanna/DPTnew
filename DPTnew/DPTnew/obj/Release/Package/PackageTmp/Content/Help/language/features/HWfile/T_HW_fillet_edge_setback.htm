Creating Fillet with Setback

In the Edge Fillet command, it is possible to add set backs to fillets created on three or more merging edges. Setback is the distance from the vertex up to which a capping face of a fillet will extend on a filleted edge. It smoothly blend the overlapped surfaces generated from the edges to the fillet vertex.

  1. To create the setback fillet you need to create the rectangular block of size 50*100 and extrude it to 25 units. Next, start the Edge Fillet command and select the merging edges to be filleted and specify the fillet radius values.



  2. Click on More Options to expand it from the selection list. Now, click on the Vertices selector under Setback option and select the vertex of the merging edges which are to be filleted.



  3. To specify individual set back values for each edge, select the Different distances check box and enter the values in the corresponding Distance mini dialogs.



    If the Different distances check box is not selected; a single set back distance will be applied to all the selected edges.

  4. Click or to confirm your selections and fillet the selected edge.

Note:
The Setback option is not available either for the Face Fillet or Solid Fillet commands.
