The Four Viewports command changes the display so that the program shows the modeldrawing in four viewports, with the display oriented so that the model is viewed from the top (looking down along the Z-axis perpendicular to the XY-plane of the World Reference System) in the top-left viewport, from the front in the bottom-left viewport, from the right-side in the top-right viewport, and from a left front isometric position in the bottom-right viewport.
