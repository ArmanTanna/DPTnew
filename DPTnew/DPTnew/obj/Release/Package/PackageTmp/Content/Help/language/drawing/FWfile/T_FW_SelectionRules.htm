The Selection Rules command enables you to define the visibility rules for the Custom and Exploded view commands.

To perform the command, right-click on an empty zone in the Model Structure of the command dialog box and select Selection Rules in the context menu. The command you will actually be enabled to perform depends on your previous choice:

  • None Visible But Selected Entities — enables you to display in the drawing view only entities you select in the model. If you have selected this option, when you right-click on an object icon in the structure, the Add to Non-Visible command will be available, enabling you to hide also the entity corresponding to the selected icon. You can then display the entity by right-clicking on its icon and selecting Remove from Non-Visible
     
  • All Visible But Selected Entities — enables you to display in the drawing view all entities except the ones you select. If you have selected this option, when you right-click on an object icon in the structure, the Make Entity the Only One Visible command will be available, enabling you to display only the entity corresponding to the selected icon. You can then hide the entity by right-clicking on its icon and selecting Remove from Visible.
