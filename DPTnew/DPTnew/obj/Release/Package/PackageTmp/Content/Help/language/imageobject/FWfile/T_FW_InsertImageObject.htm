InsertImage


 

The Insert Image command enables you to import an image into the current model or drawing. The command can be useful to place and use designers' preliminary sketches (created on the paper — for example received through a fax — and turned into image files using a scanner) inside the program's 3D environment or to insert images in drawings , for example when customizing title blocks.

Recognized image file formats are:

.bmpBMP format (Windows Bitmap)
.tifTIFF format (Tagged Image File)
.gifGIF format (CompuServe Graphics Interchange)
.jpgJPEG format (Joint Photographic Expert Group - SA/CCITT)
.pngPNG format (Portable Network Graphics)

A file selection dialog box is displayed — as in the case of the Open command —  which you can use to select the image file to be imported.

Once the file is selected :

  • In the Model environment
    the image is placed onto the current Work Plane, its lower-left corner coincident with the Work Plane origin.
  • In the Drawing environment
    the image is placed with its lower-left corner on the origin.

A set of parameters, in the Image category of the System Options, is available to enable you to control the image colors and the resize quality to be used when importing it into the model drawing.

Note that, in the Model environment, if you move the Work Plane after loading an image, the image will remain in its original position. To make it follow the Work Plane movement, use the Move to Work Plane command.
