Organize Taper Designation

The Organize Taper Designations dialog box enables you to predefine the desired taper dimension designation. You can see this designation only when the taper format is set to Angle or taper designation. If the angle does not match with available designations only the angle will be displayed.

The command dialog box provides you with the following controls:

Text: Enables you to specify the designation to be attached to the taper dimension.
Angle: Enables you to specify the angle value in degrees.
Add Enables to add a taper designation to the Designations list.
Remove Enables to remove a selected taper designation from the Designations list.

All the predefined designations and angles are listed down in the defined order.
