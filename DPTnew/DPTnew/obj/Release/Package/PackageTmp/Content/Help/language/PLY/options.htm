PLY Conversion Options

When saving (see Save and Save As) or loading (see Open and Insert from File) a PLY file by selecting Stanford Polygon File (*.ply) in the Files of type/Save as type drop-down list, the Options button in the file selection box enables you to define the parameters to be used in the conversion process.

A dialog box displaying the following categories shows up:

  • Load
    This category contains the most common parameters to be set when converting from PLY.
    • Log File
      This category contains parameters controlling the name and location of the log file created during the conversion process.
  • Save
    This category contains the most common parameters to be set when converting to PLY.
    • Log File
      This category contains parameters controlling the name and location of the log file created during the conversion process.
