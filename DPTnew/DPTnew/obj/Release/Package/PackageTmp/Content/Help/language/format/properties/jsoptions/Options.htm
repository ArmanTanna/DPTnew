Enables you to:

  • Set the changed values as the current ones
    To do so, select Set Changed Properties as Current in the Options drop-down list and the current values of properties will be changed into the ones you just set.
  • Set entity properties to those of an existing entity
    To do so, select the entities whose properties you want to change, and then start the Edit Properties command. In the Edit Properties dialog box, select Select Properties from Entity in the Options drop-down list. You are prompted to select an entity. Click on the entity whose properties you want to acquire. The properties of the selected entities immediately update to those of the entity whose properties you acquired. Click OK to close the Edit Properties dialog box.
  • Cancel your local changes
    To do so, thus resetting all of the changed values to the ones of the parent style, click Reset to Style.
