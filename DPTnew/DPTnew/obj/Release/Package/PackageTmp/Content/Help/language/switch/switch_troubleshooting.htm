Troubleshooting on documentation help

  • The help system displays online documentation in a language different from the user interface

Cause: The possible cause to get an online documentation displayed in different language is installing the application in a language and is trying to use it in another one (for example, you installed in English and are trying to use it in Japanese, or vice versa).

Solution:

  1. Copy the new language folder (example, thinkdesign/doc/japanese) from the medium you were provided with by think3 into the current installation folder (at the same level of thinkdesign/doc).
  2. Rename the old doc folder to any name (other than doc) and the newly copied folder (example, Japanese) to "doc".
