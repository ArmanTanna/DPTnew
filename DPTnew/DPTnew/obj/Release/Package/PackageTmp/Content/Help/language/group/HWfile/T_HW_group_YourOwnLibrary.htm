Creating a New Group Library

To create a new customized Group Library, so that a new tab will be displayed in the Group Library window, after creating a group — no matter whether shared or not — in a new drawing (or opening a drawing file containing only one group) you can proceed as follows:

1 Start the Save As command.
2 In the Save in selection box, choose the "Group_Libraries" subfolder of your the think3 application installation folder.
3 Select Create New Folder.
A "New folder" icon will be displayed in the box below. Change its name to the one you want. This is the name that will be displayed as the new tab in the Group Library window.
 
4 In the File name box, type the name of the file you want to save your group to.

Use the group name!

We strongly recommend that you use the same name for the group and for the file. This will simplify your operations when you use the Group Library functionalities.
   
5 Click the Save button to save the file.

When the Group Library command is started, a new tab with the folder name defined in step 3 will be displayed, containing the Group you saved to it. The new tab you created will work exactly as the standard ones.

If the Group Library is already activated ...

...when you create the new library folder, the new tab might not be immediately displayed in the Group Library window.
To have it displayed, right-click on an empty zone of the library window and select Arrange TabsDown and Arrange TabsRight (or vice-versa) and the new tab will be displayed.
