The effect of choosing a style name in this drop-down list depends on whether you are setting the entity properties with no entity selected (Entity Properties dialog box) or modifying the properties of one or more selected entities (Edit Properties dialog box).
  • Entity Properties
    The adjacent label reads Current. It enables you to view and change the current style by selecting it in the list of all the available ones.
  • Edit Properties
    The adjacent label reads Apply. It enables you to view and change the style applied to the selected entities by selecting a different one in the list.
