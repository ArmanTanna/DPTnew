ModifySurfaceNormal to Degenerate


 

The Normal to Degenerate command enables you to correctly define the normal of a degenerate surface.

A degenerate surface is a surface with only two or three edges instead of four: each missing edge is reduced to a point (degenerate edge).

The normal to a B-spline surface is precisely calculated at each of the control points of the definition grid. Hence, when an edge is reduced to a point, the normal may not in some cases be defined at the point corresponding to the degenerate edge. This command allows you to correctly define the normal. Remember that this inevitably has the effect of modifying the shape of the surface.

Conversion to NURBS
If the selected surface is not a NURBS, you will be asked to confirm whether you want to convert it into a NURBS (unless you have deselected the Ask before converting check box).
