Error file of the think3 to STEP conversion

When you save a STEP file, an error file of the conversion is created:

  • in the same directory of the file you saved
  • with the same name of the file you saved and the ".err" extension

The error file contains a report of the errors occurred during the conversion process. If no errors occurred, the file will be empty. Please note that the file is created even if the conversion process aborts, enabling you to read the report about the issue that caused the process to abort.
