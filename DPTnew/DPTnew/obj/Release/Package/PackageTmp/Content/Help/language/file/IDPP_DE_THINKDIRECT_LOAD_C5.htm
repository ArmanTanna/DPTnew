Catia V5

Category: Load

Convert not visible entities

When selected, both visible and non-visible entities are converted; when not selected, only visible entities are converted.

Convert hierarchy

When selected, it enables the think3 application to read and convert also the hierarchy information. Each 'leaf' component will be converted to an X-Reference Component. When not selected, information about hierarchy is ignored (default).

Convert solid entities

Controls the conversion of solid entities. When this check-box is not selected, solids are converted into their component surfaces.

Convert non-manifold solid

When selected, non manifold solids are converted. When not selected, only manifold solids are converted.

Convert surfaces

When checked, surfaces are converted. When not checked, they are ignored

Convert curves

When checked, curves are converted. When not checked, they are ignored.

Convert points

When checked, points are converted. When not checked, they are ignored.

Show warnings/error messages

When selected, warnings and error messages are displayed in a popup window.
