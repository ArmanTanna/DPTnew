ToolsConfiguration ManagerReset

The Reset command enables you to restore the default configuration of the application, thereby losing all the changes that have been made to the current configuration. The default configuration is a snapshot of the application settings done after the installation at the first run time of the application.

Note
The reset operation will have effect only when you restart the application. It is important to save the configuration settings in the location different from the default one (normally this is a folder inside your local file system: User folder), as the Reset operation removes all the existing configuration files from the above mentioned default location.
