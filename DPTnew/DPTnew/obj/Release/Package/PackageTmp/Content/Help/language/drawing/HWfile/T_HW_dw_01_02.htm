Creating a Custom View

1  Create a new drawing by using the File FileDrawing command (if you haven't already done so).
2 Start the Custom View command.
3 The first time you insert a view into a new drawing, the think3 application displays the Edit Reference Model dialog box. Select the model from which you want to create the view, and then click OK.
4 As soon as you specify the model, the program displays the Custom View Definition dialog box.
See Managing Views about establishing the custom view.
5 When you have finished establishing the custom view, click OK to continue.
6 The program quickly calculates the view and displays a preview image of the view, attached to the cursor. The program prompts you to enter the view origin. Click to position the view.

After inserting the view, you may want to specify a suitable Drawing Frame, insert a Drawing Frame and Title Block, and then insert other views (such as projected, auxiliary, detail, cross-section, sectioned, developed, or exploded views).

Note

Factors which affect the appearance of the view, such as the visibility of its view name and the visibility of specific types of entities, are controlled from the various categories under Drawing of the Options/Properties... dialog box.

After you create a view, you can change the scale and many of its properties from the various categories of the Drawing View.
