Drawing Views

Main Enables you to create main views.
Derived Enables you to create derived views.
Exploded view Enables you to insert a two-dimensional exploded view of a 3D assembly model into the current drawing.
Flat Pattern View Enables you to create a flat pattern view of a Sheet Metal part.
Auto-Place Dimensions Enables you to automatically reposition all the dimensions in a view after modifying their distance from the related object.
Modify Edge Visibility Enables you to restore smart dimensions to their default positions in the selected drawing view.
