Wrapped Diagonal

A pattern giving a line across the leading diagonal of the texture space. A color source shader providing a diagonal stripe in texture space. The size of the pattern is specified by value Scale; increasing this value will make the pattern appear larger.

The width of the stripe specified as a fraction of the pattern size is given by Width. The color of the stripe and of the background are specified by settings Stripe Color and BackGround Color respectively. A fuzzy edge is applied to the stripe with a width given by Fuzz.
