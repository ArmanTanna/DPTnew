Welding Bead

Bead

The controls in this area enable you to define the aspect of the bead.
The options displayed relate to the bead type selected in the Type: drop-down list.

Type

Type:

Select a bead type from the drop down list in order to see the graphical representation preview and set the relevant properties.

The available bead types are:

Bead
Seam
Plug
Slot
Spot

Spacing

0.0

This option enables you to configure the blank space between two adjacent elements of the chain. It is expressed as a multiplier of the width of the elements:

Blank space = Spacing value * Element width

Drag the slider to the left to reduce the spacing between elements or to the right to increase it.

You are also enabled to enter the spacing value (within the limits indicated by the slider) in the relevant numeric field. The higher the number, the wider the spacing; when the spacing value is set to 1, the width of the element and the blank space between two adjacent elements are equal.

Radius

Radius:

This option enables you to configure the curvature of the circular elements used to represent the bead. It is expressed as the ratio between half the cord length and the arc radius.

Drag the slider to the left to reduce the curvature of the arcs or to the right to increase it.

You are also enabled to enter a value (within the limits indicated by the slider) in the relevant numeric field. The higher the number, the higher the curvature; when the value is set to 1, the arc extends to 180º.
