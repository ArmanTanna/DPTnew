Setting positional continuity with different weight values

  1. Open file "mc_01_04.e3e2".


     
  2. Start the Curve Continuity command.
     
  3. In the Degree drop-down list, select Positional.
     
  4. In the Method drop-down list, select Adjust both.
     
  5. Select the two curves by clicking close to their near endpoints, as shown in the illustration below:



    Since the two curves are circular arcs, you are asked to confirm that you want each one converted into a NURBS. Confirm in both cases.
     
  6. In the 1st Weight and 2nd Weight mini-dialog boxes, type the values of the weights of the two curves (1 and 1).


     
  7. Click to apply and quit the command.
