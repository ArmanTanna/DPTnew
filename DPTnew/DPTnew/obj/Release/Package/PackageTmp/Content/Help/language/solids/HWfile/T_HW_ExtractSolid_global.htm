Creating Copies of all the Faces of a Solid

In the following tutorial you will use the Solid from Faces command to create the static solid corresponding to a parent solid. To accomplish the task, open the ExtractSolid_global.e3 file available in the Samples folder of your think3 installation and follow the steps given below:

  1. Invoke the Solid from Faces command. Ensure that the Mode is set to Global in the selection list. You are prompted to select a solid.

  2. For the Solid selector, select the solid whose all faces will be considered for creating the static solid.

  3. All the faces of the solid get selected.



    Note that to change the selected solid, you can right-click on Solid in the selection list, choose Reset and reselect another solid.

  4. If you want the resulting static solid to retain a link to the original, select the Linked copies check box. In this case, you will get a parametric solid.

  5. Click or to confirm your selections and create the new solid.



    Click to discard your changes.
