Overview table on how to use the keyboard

Here is a map of how to use some of the keys of the keyboard. In order to have the following keys work, the mouse cursor must be positioned somewhere in the Graphics Area.

KEY/Action Graphics Area
Mini-Dialog Boxes		 Selection List
TAB First hit: confirms supplied value.
Second hit: moves onto next mini-dialog box. Cyclical: when positioned on the last one, moves to the first.		 Moves onto next selection list option. Cyclical: when positioned on the last one, moves to the first.
Shift+TAB First hit: confirms supplied value.
Second hit: moves onto previous mini-dialog box. Cyclical: when positioned on the first one, moves to the last.		 Moves onto previous selection list option. Cyclical: when positioned on the first one, moves to the last.
Ctrl+TAB Moves onto first option in selection list. Moves onto its mini-dialog box.
ENTER Within a command:
Applies the command and quits if all the input arguments have been supplied. A warning message is displayed if some input argument is missing.

With no command selected:
Causes the program to start the last command used.		  
SPACE BAR Starts an editing session on current option.  
ESC Aborts the editing session on current option, which remains the current one. When no editing session is active, aborts the command.  
BACKSPACE Deletes the content of current input field, when in editing mode.  
DEL Deletes the content of current input field, when in editing mode.
