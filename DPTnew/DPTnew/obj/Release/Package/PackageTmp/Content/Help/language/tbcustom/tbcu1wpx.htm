Changing the Appearance of a Button

To change the appearance of a button, do the following.

  1. If the button whose appearance you want to change is not already on a toolbar, add it to a new or existing toolbar.



  2. Select ToolsCustomize from the menu bar to open the Customize dialog box.

  3. Move the mouse pointer over the new button and right-click.

  4. Choose Change Button Icon from the toolbar popup.



  5. In the Customize Button dialog box, choose the new button icon.
