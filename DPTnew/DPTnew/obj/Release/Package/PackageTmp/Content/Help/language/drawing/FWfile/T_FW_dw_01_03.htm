The Sectioned view option of the Custom view definition dialog box enables you to insert a custom two-dimensional sectioned view of a 3D model into the current drawing.

Note: This option is only available if the drawing already contains a main view with at least one existing section already created by using the InsertCutting Plane LinesDefine Section command.
