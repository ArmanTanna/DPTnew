Customize GUI Profiles - Set current GUI profile

The think3 application is shipped with the following default profiles:

ThinkDesignDrafting
2D-3D Design
2D Drawing
Industrial Design
3D Learning
Machine Design
Mold Design
Sheet Metal
Viewing Tools

1 In the list of profiles, select the name of the GUI profile you want to use.
2 Click the Set as Current button. The profile is now activated and the name of the current GUI profile will appear in the Name: box.

Note
The newly 'set current' profile can be edited by removing or adding the toolbars or buttons. The changes will be stored in that particular profile. Also you can add customized toolbars to the selected profile.
3 The Reset button can be used to roll back a think3 supplied profile to its original state. For example, if you have added or removed certain toolbars from a think3 supplied profile and if you want to reset that GUI profile to its default setting, you can use Reset.

Note
The Remove function can be applied only on user defined profiles. You cannot delete the think3 supplied profiles.
