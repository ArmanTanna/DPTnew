Shortcut Function Keys

The program enables you to assign shortcuts to commands by associating them with keyboard key combinations. The F1 through F12 function keys, however, cannot be customized. Standard commands have been assigned to them as described in the following table.

Function Key Description
F1 Opens the think3 product Help.
F2 Executes the Rebuild Model command.
F3 Toggles the Window Selection Mode.
F4 Hides or unhides the I/O Window.
Ctrl+F4 Closes the currently active document.
F5 Executes the Redraw command.
F6 Recalls the Previous view.
Ctrl+F6 Activates the next document in the list of the currently open documents. It is useful to cycle through the open documents as an alternative to selecting them from the Window menu.
Ctrl+Shift+F6 Has the same function as Ctrl+F6, but it operates in reverse order.
F7 Temporarily hides or shows mini dialogs.
Shift+F7 Temporarily hides or shows both mini dialogs and handles. This shortcut applies only to commands that display the handles, such as Move Copy Entities, Scale Entities, Multistretch Entities etc.
F8 Orients the View on Work Plane.
F9 Enables or disables the Point Coordinates system. When enabled, a red circle marks the Work Plane coordinate origin that you are enabled to temporarily redefine by pressing the Shift key.
F10 When the Point Coordinates system method is active, switches between cartesian and polar coordinate system. A red circle marks the Work Plane coordinate origin, and, when the polar coordinates are active, a red arrow points from the coordinate system origin towards the positive direction of the X axis. By pressing the Shift key you are enabled to temporarily redefine the origin location, while by means of the Ctrl key you can reorient the X axis.
F11 Reserved.
F12 Reserved.
