FormatRendering Library

The Rendering Library is a palette consisting of a set of predefined materials, light studios and environment. This library is used to apply the material, light studio and environment to a model in the application. The Rendering Library consisting of three tabs as shown in the image below.



The following are the three tabs available:
Material Tab Enables you to apply material from the library files.
Light Studio Tab Enables you apply light studio's
Environment Tab Enables you apply an environment.
