The new GSM Fillet Reduction command

This version introduces a new GSM command called GSM Fillet Reduction for Die Designing. It enables you to modify a set of selected fillet surfaces by changing their curvature, while retaining the desired continuity along borders. It can be used to modify some high curvature areas to let the material to fill properly during stamping.
