System Options

Category: Model Structure

Show entities

Show entities

The controls in this area enable you to define the settings and visualization of entities in the Model Structure.

Solids and profiles

Solids and profiles

If selected, solids ( ) and profiles 2D ( ) & 3D ( ) are displayed in the Model Structure.

If cleared, solids and their respective profiles will not be displayed in the Model Structure.

Components

Components

If this box is selected, components are displayed in the Model Structure.

If this box is not selected, components are not displayed in the Model Structure.

Show part number and description

Show part number and description

When checked, the program displays in the structure tree the Part Name and Description that you enter when you edit the Part Properties of the component or group.

When not checked, the program displays the component or group name.

Used profile and associative curves

Used Profile and associative curves

When the check box is selected, associative curves and profiles already used by some feature are displayed in the Model Structure.

When not selected, associative curves and used profiles are not displayed.

Associative surfaces

Associative surfaces

When the check box is selected, associative surfaces are displayed in the Model Structure.

When not selected, associative surfaces are not displayed.

Datum planes

Datum planes

If this box is selected, datum plane ( ) constraints are displayed in the Model Structure.

If this box is not selected, datum plane constraints are not displayed in the Model Structure.

Other entities with history

Other entities with history

When the check box is selected, all entities (datum lines and datum points) with history are displayed in the Model Structure.

When the check box is not selected, no entities with history are displayed but other static entities will still appear in the Model Structure.

Entities without history

Entities without history

When the check box is selected, all entities without history are displayed in the Model Structure.

When not selected, all entities with the history are displayed in the Model Structure.

Show associated events

Show associated events

The controls in this area enable you to control the visualization of symbolic references and matings in the Model Structure.

Symbolic references

Symbolic references

When selected, Symbolic references ( ) are displayed in the Model Structure.

When not selected, symbolic references are not displayed in the Model Structure.

Matings

Matings

Enables you to show or hide the Mating Structure below the Model Structure in the left panel:

  • When the check box is selected, the Mating Structure is displayed just below the Model Structure:



  • When the check box is not selected, the Mating Structure are not displayed.

More information

More information

When the check box is selected, additional parameters appear in brackets next to the mating constraint involved in the Mating Structure.

When the check box is not selected, only the mating constraints are displayed in the Mating Structure without any additional information.

Events inside hidden solids

Events inside hidden solids

When selected, entities inside the hidden solids are displayed in the model.

When cleared, entities inside the hidden solids are not displayed (and loaded) in the model.

Events inside hidden components

Smart load

When selected, entities inside the hidden components are displayed in the model.

When cleared, entities inside the hidden components are not displayed (and loaded) in the model.
