Mirroring a Set of Entities (2D Environment)

After starting the Mirror Entities command, proceed as follows:
1 Select the entities you want to apply the transformation using a window selection.



End selection by right-clicking and choosing Continue in the context menu.
To change the selected entities, right-click on Entities, select Reset in the context menu and perform selection again.
2 In the Symmetry axis drop-down list, choose the definition mode for the symmetry axis:
Line To define the symmetry axis as an existing line. Once the mode is selected, select the line. To change the selected one, right-click on Symmetry axis, select Reset in the context menu and select another line.
2 points To define the symmetry axis as the line identified by two points. Once the mode is selected, specify the two points. To change the selected ones, right-click on Symmetry axis, select Reset in the context menu and specify two different points.
Parallel to X and through point/ Parallel to Y and through point To define the symmetry axis as a parallel to one of the coordinate axes and through a point. Once the mode is selected, specify the point. To change it, right-click on Symmetry axis, select Reset in the context menu and specify another point.
For example, choose Line and select an existing line.
3 To make a copy of the selected entities, select Copy. The preview shows what you will obtain:



To undo the transformation you can check in the preview, before actually applying it and quitting the command, click the button.
4 Click on More Options to expand it — you are enabled to apply geometric transformations to entities in the model/drawing.
For example, to mirror each single text character, select the Mirror texts check box.
5 Click or to confirm your selections and apply the transformation.



Click to discard your changes.


Note
The Mirror texts option does not apply to True Type Standard font texts; it only affects Vector font texts.
