Deleting a Custom Toolbar

To delete a custom toolbar, do the following:

  1. Select Tools Customize from the menu bar to open the Customize dialog box.

  2. Move the mouse pointer over the toolbar you want to delete and right-click.

  3. Choose Delete Toolbar from the toolbar popup. The toolbar gets deleted.

Note
You cannot delete any of the default toolbars. If you have right-clicked on a default toolbar, the Delete Toolbar option is disabled.

You can also make use of the Delete button present in the Toolbars tab of the Customize dialog box. Click here for details.
