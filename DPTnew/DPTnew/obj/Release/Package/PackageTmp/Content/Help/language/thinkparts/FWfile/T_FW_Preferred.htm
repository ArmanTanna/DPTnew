The Favorite command enables the display of a catalog or index in the Catalog Browser under certain conditions.

You can find this command in the context menu that displays when right-clicking on an index in either the Catalog Browser or the Catalog Manager, or when right clicking on a catalog in the Catalog Manager.

By setting the Favorite flag of an index or catalog, you enable its display in the catalog tree frame of the Catalog Browser provided that the Favorites option of the thinkparts category of the System Options categoryis set. Otherwise the Favorite flag will be ignored.

By default, this flag is not active in imported catalogs, while it is active for new catalogs.

If you wish to conceal, rather than Delete some elements from thinkparts, you can do it with the Non-favorite command, and restore them later with Favorite.
