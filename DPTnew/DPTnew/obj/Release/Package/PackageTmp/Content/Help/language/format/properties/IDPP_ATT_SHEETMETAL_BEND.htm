Entity Properties

Category: Sheet Metal

Bend

Bend

The controls in this area govern bend parameters.

Allowance

Allowance:

Enables you to select the method to be used to create bends:

K-factor to create bends based on a k-factor controlling the position of the neutral radius.
Bend table to create bends based on an external table (see The Bend Table).

K factor

K factor:

Enables you to specify the k-factor to be used when creating bends. The k-factor controls the positioning of the neutral radius: 0.5 identifies the middle of the thickness, 0 identifies an outwards development, 1 an inwards development. For example, very hard materials have k-factors reasonably about 0.1, while k-factors for metals are about 0.5.

Normal side faces

Normal side faces

When this check box is selected, all the side edges of the object (i.e.: the ones cut by the machinery and not the internal existing features) are created vertical.

Inner radius

Inner radius:

Enables you to specify the internal radius value to be used when creating bends. An existing bend table value can be selected if the From table option is checked, which is available on selecting Bend table in the Allowance option.

Thickness

Thickness:

Enables you to specify the thickness value to be used when creating bends, when it is not automatically extracted from the solid, typically with solids created using the Solid Flange command. An existing bend table value can be selected if the From table option is checked, which is available on selecting Bend table in the Allowance option.

Rolling threshold

Rolling threshold:

Enables you to define the value of the radius beyond which the k-factor values is set to 0.5, as the cylindric surface is to be considered as rolled rather than as bent.
