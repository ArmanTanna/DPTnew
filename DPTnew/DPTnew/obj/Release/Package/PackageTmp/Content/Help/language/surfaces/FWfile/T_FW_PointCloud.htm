Insert Surface Specialized Point Cloud

The Point Cloud command enables you to create a surface fitting or interpolating a set of points. You can choose whether to perform a fitting or an interpolation operation in the Surface Type drop-down list:

  • Fit cloud of points
    Enables you to create a surface by fitting a given set of points or an existing surface.
    A reference surface is internally used by the program in order to obtain the final fitted surface. The reference may be either an existing surface of the model, or a stretched virtual one, not actually existing in the model but identified through its boundaries (four or three: in the latter case a segment is considered as the fourth boundary), or through four vertices.
    See further details in "Creating a Point Cloud surface by fitting a set of points".

    Point Cloud Surface Creation Issues
    In some situations, when applying the command, a message may warn you the system is unable to fit the cloud of points, suggesting you to either increase the number of points to fit or decrease the number of control points of the reference curves/surface.
    In such cases you can proceed with one or both of the following actions:
    • Increase the number of points to fit.
    • If Fitting Mode is set to Boundary curves, decrease the number of control points of the boundary curves using the Convert Curve to NURBS command (by specifying the maximum number of control points for the resulting curve in the Max. control points box).
    • If Fitting Mode is set to Surface based, decrease the number of control points of the reference surface using the Convert Surface to NURBS command (by specifying, in the Max. Control Pts U and Max. Control Pts V boxes, the maximum number of control points for the resulting surface along the two parametric directions).

 

  • Grid points interpolation
    Enables you to create a surface that interpolates a grid of points.
    A grid of points is a set of points with some regularity characteristics. The space distribution of the points is such that, when the points are projected orthographically onto a plane, two directions are easily identifiable, almost forming a right-angle, so that if the curves are considered to interpolate the points in the two directions they form a clearly recognizable grid on the plane; within this grid, the points "aligned" in one direction (even if not in a straight line) are called rows.



    See further details in "Creating a Point Cloud surface by interpolating a grid of points".
     

A set of Quality Checks commands is also available.
