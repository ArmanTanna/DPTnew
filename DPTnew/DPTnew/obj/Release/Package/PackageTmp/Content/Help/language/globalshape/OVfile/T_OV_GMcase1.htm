Case 1: Make a linear sweep surface pass through a curve

The problem How to solve it

A linear sweep surface must be edited so that the red curve will lie on it.

GSM solves the issue in a very smart way; the isoparametric curves look much better than hey would in a standard surface modeler.

Anyway, you shall be very careful in generating NURBS output, as it might involve a very high number of control points.
