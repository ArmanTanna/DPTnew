Printing out Annotations


To print out annotations, you will have to:
 

  1. Set the printing Mode to Print in the Print Setup dialog box (see Print).
     
  2. Deselect the Exclude notes in printing check box in the Options category.
     
  3. The Fit to page check box adjusts the print scale to fit the annotations and the model on the page.
