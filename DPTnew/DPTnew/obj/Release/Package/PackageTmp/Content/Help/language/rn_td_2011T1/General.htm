New General Functionalities and Improvements

Version 2011.1 introduces the following general enhancements to the General functionalities.

  • Enhancement in the Model Structure representation
  • New shortcuts for view commands
  • Visual Basic API-COM examples

Enhancement in the Model Structure representation

In this release, any modification or addition of features to a solid or surface will be displayed inside the Model Structure in the same position as the original ones. The new ordering criteria is based on the following differences with respect to the previous criteria:

  • The descending order of solids/skins/curves is based on the original solid/skin/curve. Previously it was based on the final solid/skin/curve.
  • When a feature/modification is applied to a solid/skin/curve, the latter one remains in the same position. Previously it moved on top.
  • Multiple solids/skins/curves created modifying solids/skins/curves are shown in the same position as the original ones. Previously they were placed on top.

New shortcuts for view commands

In this release, brand new and most useful shortcuts for view commands have been added.

Ctrl++ Executes the Zoom In command.
Ctrl+- Executes the Zoom Out command.
Ctrl+1 Executes the Fit command.

Visual Basic API-COM examples

A new set of Visual Basic examples is available, including a step-by-step programming section. See The ThinkDesign Collection of Visual Basic API+COM Examples
