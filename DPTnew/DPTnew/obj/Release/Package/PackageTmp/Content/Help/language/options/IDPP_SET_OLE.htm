Formats

Formats

The controls in this area determine in which formats it is possible to cut, copy, and paste to and from other Windows applications (see Cut Selection, Copy Selection and Paste Selection).

Metafile

Metafile

Windows Metafile Format (a widely used format associated with Microsoft® Windows, especially for exchanging images. Enhanced Metafile Format is also availble.).

Enhanced Metafile

Enhanced Metafile

Enhanced Metafile Windows format (a widely used 32-bitformat associated with Microsoft® to display a picture. A ".emf" file can contain a much broader variety of commands than a ".wmf" file.)

Bitmap (DIB)

Bitmap (DIB)

Device Independent Bitmap format. This is the most common bitmap graphics format used by Microsoft Windows®, usually stored in files with a ".bmp" extension. The "device-independent" adjective comes from the fact colors are represented in a format independent of the final output device. A device driver translates the DIB colors into colors that can be actually output to the monitor or to the printer.

Bitmap (DDB)

Bitmap (DDB)

Device Dependent Bitmap format. This type of file is structured in such a way that it is very similar to the form in which the image is displayed by the device driver. A device dependent image is generally not compatible with other diaplay devices.

Text (if used)

Text (if used)

Text format.

ThinkDesign OLE object

Thinkdesign OLE object

OLE internal format.

Settings

Settings

The controls in this area enable you to define the settings which manage the Drag & Drop operations.

Disable OLE Drag & Drop

Disable OLE Drag & Drop

When selected, disables you to perform drag and drop action.

When not selected, enables you to drag and drop entities inside the same document and also between two documents.

Include background

Settings

  • When selected, it enables you to select the background along with the entities from the Graphics Area and paste it in other supported applications.
  • When cleared, only the selected entitles will be copied, without the background.

Dragdrop semiconstrained entities

Drag&drop semiconstrained entities

The controls in this area enable you to define the settings which manage the Drag & Drop operations for semiconstrained entities.

Preview mode:

Preview mode:

Select the entities that will be taken into account by the preview of drag & drop of components within an assembly:

Dragged entity only Only the selected component will be dragged during the preview.
Entities connected by mating The selected component, the components to which it is connected by multiple matings and any component connected by Mating will be dragged during the preview.

Collision detection

Collision detection

The parameters in this area enable you to control the feedback mode for collision detection.

Feedback:

Feedback:

Enables/disables collision detection when dragging & dropping entities by selecting one of the following options in the drop-down list:

None No collision detection.
Highlight colliding faces Colliding faces are highlighted.
Stop on collision Stops on collision.

Ignore threaded surfaces

Ignore threaded surfaces

Enables/Disables considering threaded entities for collision detection.
