The Make Entity the Only One Visible command enables you to make a an entity the only one visible in a Custom or Exploded view.

The command is available on the context menu you display by right-clicking on an object icon in the Model Structure after the Selection RulesAll Visible But Selected Entities command has been performed.

Once you have performed this command on an entity, you can hide it by right-clicking on its icon and selecting Remove from Visible.
