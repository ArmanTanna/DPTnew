Fit a curve to a set of points stored in an ASCII file

After starting the Fit Curve command, proceed as follows.
 
1 In the From drop-down list, select ASCII file.
2 Click Browse... and select the ASCII point file (file type ".pt") containing the points you want to use to fit the curve (Note that each line in an ASCII Points file contains a coordinate triple, with each coordinate in the triple separated by a blank space).
3 Click the Open button to load the file.
4 Proceed as described for fitting a curve to a set of points.
