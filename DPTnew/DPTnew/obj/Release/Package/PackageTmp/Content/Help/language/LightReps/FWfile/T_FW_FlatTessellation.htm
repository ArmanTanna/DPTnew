The Flat Tessellation command enables you to apply a Light Representation at the lowest, quickest to load tessellation level to a/different X-Reference component(s).

In fact, when you perform the command, you can't view the information on how the component was built, since the Model Structure doesn't display any more the hierarchy of the sub-components and the list of the events.

The command is available from the context sub-menu displayed when right-clicking on the X-Reference (component) and selecting the Light Representations command.

The Flat Tessellation Light Representation is generated automatically.

Note that the command can be applied exclusively to X-Reference components.

Also note that when loading an assembly, you can specify a rule to be applied to all the X-Reference components within the assembly. In particular, you can select a Visual Bookmark, associated to the document, that will be applied at loading, while an additional check-box, Quick Load, enables you to load any X-Reference component in the Lightest Representation, that is in Flat Tessellation, regardless of what is specified in the selected Visual Bookmark.
