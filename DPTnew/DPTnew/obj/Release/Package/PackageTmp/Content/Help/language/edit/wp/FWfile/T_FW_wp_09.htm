EditWork PlaneAlign Z-Axis

The Align Work Plane Z-Axis command enables you to align the Z-axis of the Work Plane along a direction. After starting the command you need to pick a direction to align the Z-axis in the same direction. This can be done by selecting a straight entity that can be either a line or an edge.

The command does not exit automatically after aligning the Z-axis. It enables you to align the X-axis also. In case you need to align the X-axis, you can continue by selecting a line or a point otherwise you need to press ESC key to exit from the Align Z-Axis command.

Note that while the Z-axis of the Work Plane is realigned with the specified direction, the position of the Work Plane origin does not change.

The Align Z-Axis command can also be accessed from the context menu displayed by right-clicking on the Work Plane. You can also use the Select command to align the Z-axis.
