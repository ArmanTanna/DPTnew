Creating a linear sweep solid using Direction

The Linear Solid command creates a solid by sweeping a profile or chain of curves along a linear path. To accomplish the following task, open file "ExtFlan02.e3".

1 Start the Linear Solid command and select the profile.

2 Click More Options to specify additional options.

Use the Direction selection list item to set the sweep direction of the solid.

a. Select Normal to create a normal sweep solid.

b. Select 2 Points to specify the sweep by selecting two different points. Select any two points on the solid.

c. Select Line to specify the sweep by selecting a different line on the solid.

3 Click or to confirm your selections and create the solid. Select Left Front View to view the following solids.

  • If you have selected a., you can see the following solid

  • If you have selected b., you can see the following solid

  • If you have selected c., you can see the following solid

    • Click to discard your changes.
