ModifyCompensatorOffset Mesh

The Offset Mesh command enables you to offset a selected mesh by a specified offset value.

After starting the command select the mesh to offset for the Mesh selector.

The offset value has to be entered in the Offset size text box.

The Always recalculate normals check box enables you to manage the normals of the mesh being offset.

  • When the check box is selected, the normals on mesh are always calculated and reoriented, even if they already exist on the mesh.
  • When the check box is cleared, the normals on mesh are calculated and reoriented only if they do not exist on the mesh.

On selecting the Preview () button the preview of the offset mesh is displayed. An example is shown below.

In case, the offset mesh is self intersecting, a suitable warning is displayed on selecting the Show Warnings () button. Also, the location of self-intersection is highlighted on the mesh.

If satisfied with the preview, you can apply () the command.
