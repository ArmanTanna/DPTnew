Animation List



A model can have one or more Animations, which are embeded in the same file. An animation defines a set of changes of the model in time, so that the behavior of the model during these changes can be checked and/or an AVI showing the model evolution can be generated. The animation list contains the Animations available in the model. The first time the Animation command is run, this list is empty. New animations can be added or existing animations can be removed by right-clicking in this area.
