The References tab of the Define Smart Objects command enables you to define the mandatory geometric references to be requested at insertion time (position point, profile, face, ...).

  • Boxes in the Message column can be edited to change the prompt to be displayed as a request for the reference to be selected at insertion time. After writing the prompt you can add a ";" character and write the label to be displayed on the corresponding item in the Selection List.
  • Boxes in the Type column can be edited to change the type of entity to be selected as a reference at insertion time
