Datum reference

Datum feature

Enables you to determine the contents and orientation of Datum reference tags.

Text

Text:

Enables you to specify the Text to be included in the Datum reference tag.

Frame

Frame:

Enables you to choose the right kind of frame to be used for representing the datum target. Select one from the drop-down list:

<I_DIM_DATREF_FRAME_RECTANGLE>Rectangle
<I_DIM_DATREF_FRAME_RECWITHDASHES>Rectangle with dashes
<I_DIM_FRAME_02>Circle (1c)
<I_DIM_ARROW_FILLED_UNDERLINE>Filled Underline

Datum target

Datum target

The controls in this area enable you to set the parameters for the Datum target symbols.

Text above

Text above:

Enables you to specify the text in the upper portion of the Datum target symbol. Click on the ø (Fi) button to include a diameter symbol.

Text below

Text below:

Enables you to specify the text in the lower portion of the Datum target symbol.

Connected line orientation

Connected line orientation:

Enables you to specify the orientation of the connecting line. You can choose from any one of the following options:

Free
Horizontal

Linkage position

Linkage position:

Enables you to determine whether the arrow of the connecting line must be Attached to the datum target or Free.
