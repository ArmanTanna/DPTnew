System Options

Category: Graphics-Highlight

Highlight

Highlight

The controls in this area enable you to define the setting controlling the highlighting modes.

Without flickering

Without flickering

When checked, no flickering is visible while pre-highlighting is displayed. Removing the flickering effect might slow down your operations. In this case, deselect this box.

In one view only

In one view only

If more than one view are contemporarily active, when this box is checked pre-higlighting is displayed only on the view you are working in, which can be useful in the case of large viewsmodels.
If the box is not checked, pre-highlighting will be displayed in all views.

Hidden Lines

Hidden Lines

When the check box is selected, hidden lines are displayed during the highlight.
When the check box is cleared, hidden lines are not displayed during the highlight.

Standard

Standard

The option set in the drop-down list or check box present beneath will be applied to the standard assemblies and models. The think3 application considers an assembly as 'Standard' or 'Large' as per the settings made in the Large assembly area of the General category.

(current)

(current)

The think3 application considers an assembly as 'Standard' or 'Large' as per the settings made in the Large assembly area of the General category. Depending on the current status of the file, the phrase (current) is displayed either besides Standard or Large assembly.

Large assembly

Large assembly

The option set in the drop-down list or check box present beneath will be applied to the large assemblies. The think3 application considers an assembly as 'Standard' or 'Large' as per the settings made in the Large assembly area of the General category.

With delay

With delay

When checked, a 3 centisecond delay will be applied before performing the pre-highlighting operation. This can be useful to speed the operations up when moving the cursor on large or complex models/drawings.

Line thickness

Line thickness

It enables you to control the highlighting mode of linear entities (lines, curves, edges) when the cursor moves onto them.

  • When the check box is selected, the entities are displayed with a thicker line and highlighted.
  • When the check box is not selected, the entities are highlighted but displayed with their own original thickness.
