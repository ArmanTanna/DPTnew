The Reload Tree command enables you to refresh the tree structure in the browser frame of the Catalog Browser.

This command is located in the context menu available when right-clicking on any object listed in the tree browser frame of the Catalog Browser or Catalog Manager.

It may happen that, for many reasons, changes to the tree structure are not reported in the browser window. The Reload Tree command forces a regeneration of the tree based on the latest information stored in the thinkteam database. The result is a correct update of the information displayed in the browser frame.
