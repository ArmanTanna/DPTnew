Exporting Model to CATIA®

The From Model into Catia Model command enables you to save a model as a CATIA® .model.

Note that only 3d entities are converted.

To convert a model to a CATIA® .model file, proceed as follows:

  1. Start the TD Catia v4 interface Options command from the TD Catia v4 interface toolbar and access the Save tab.

  2. Set the conversion options as you like. Please refer to the TD Catia v4 interface Options command help for more information.

  3. Click OK to save the settings.

  4. Click the From Model into Catia Model command from the TD Catia v4 interface toolbar.

  5. In the Save in box of the file selection box that displays, select the folder where you wish to save the CATIA® model file.

  6. Type the name of the file to be saved, then click the Save button. Any extension you will type will be overwritten with .model

Note that you can save only a selected part of the original model or the active layers only or the whole set of entities, as listed in Save options. You can also choose to save curves and/or surfaces as Bezier entities (approximated by entities defined by an arc), as well as save solid entities as solids (and not as single surfaces).
