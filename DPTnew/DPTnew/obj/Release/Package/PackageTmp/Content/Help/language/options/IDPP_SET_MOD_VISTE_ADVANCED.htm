System Options

Category: Navigation-Advanced

Rotation

Rotation

The controls in this area determine the behavior of the program when you change the current view by zooming, panning or rotating.

Axes

Axes:

Enables you to determine the methods for rotating orthographic views when using the following commands:

  • Rotate Counter-Clockwise about X
  • Rotate Counter-Clockwise about Y
  • Rotate Counter-Clockwise about Z
  • Rotate Clockwise about X
  • Rotate Clockwise about Y
  • Rotate Clockwise about Z

Choose one of the following settings from the drop-down list:

Work Plane

When selected, the above mentioned Rotate ... commands rotate the view about the axes of the Work Plane, by an angle equal to the View rotate step value.

View

When selected, the above mentioned Rotate ... commands rotate the view about the axes of the view projection plane (an orthogonal right-hand reference system based on a plane with its origin at the center of the current view, a horizontal x-axis pointing to the right, a vertical y-axis pointing to the top, and a z-axis pointing directly out from the screen), by an angle equal to the View rotate step value.

Mode

Mode:

Enables you to determine the methods for rotating perspective views when using the following commands:

  • Pan/Zoom/Rotate
  • Rotate Counter-Clockwise about X
  • Rotate Counter-Clockwise about Y
  • Rotate Counter-Clockwise about Z
  • Rotate Clockwise about X
  • Rotate Clockwise about Y
  • Rotate Clockwise about Z

Choose one of the following settings from the drop-down list:

CameraWhen selected, the above mentioned Rotate ... commands change the view by rotating the camera about the target with reference to the Work Plane axes by an angle equal to the View rotate step value.
TargetWhen selected, the above mentioned Rotate ... commands change the view by rotating the target about the camera with reference to the Work Plane axes by an angle equal to the View rotate step value.
Perpendicular This option allows you to perform a planar rotation, where the axis of rotation is always perpendicular to your screen. Note that the center of rotation is defined by axes combination of current view or work plane origin. With this option selected, you can effectively "tilt" the model, with its current view fixed. Note that there is a short cut for the same behavior with SHIFT+CTRL+right mouse button, even if rotation mode is set to Camera or Target.
In a practical design point of view, it is analogous to hand movements during filing or sanding (polishing with sandpaper) your work piece by rotating it in a plane perpendicular to your eye-level.

Target

Target

The controls in this area enable you to manage the behavior of the Set Target command, and the rotation of the model within the graphic area.

Show

Show

You can precisely control the rotation of a model on screen by setting "targets". When you manually set any "targets" for viewing/rotating the model, using View Modify Set Target (or from the View menu in the context menu obtained by right-clicking on the empty space of graphic area), the target you specify is highlighted using this marker:

You can select the Show check box, to toggle the display of the marker in the graphics area.


Note

When you have not manually set any 'targets' for viewing the model, the program tries to find out a point of interest (at that particular instant of time, for that particular model) that can be the 'virtual target' in the graphics window. The program tries to understand and calculates this point behind the screens so that you get the best possible solution. Note that the automatic target is done by the program itself; you don't have to specify anything.

Move camera

99

You can precisely control the rotation of a model on screen by setting "targets". You can manually set any "targets" for viewing/rotating the model, using View Modify Set Target (or from the View menu in the context menu obtained by right-clicking on the empty space of graphic area).


For the Move camera option if No is selected, the target is set in the position that you specify, without any centering or movement. If Seek is selected, then the target is centered and the observer is moved towards the target by an amount specified by the Seek % slider, as described:
0 The target is just centered on the screen.
1 to 99 The target is centered and the observer is moved near the target. For example, if the value is 50% (default value) the observer will be moved toward the target by half a distance from it. As you increase the values from 1 to 99, this will essentially bring the effect of zooming to the target.
