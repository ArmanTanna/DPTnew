Organize Hole Table -

Category: Table Format-Columns

Columns

Columns

This category enables you to customize and format the table columns in the Hole Table.

Available

Available

Displays the available list of columns. Using the arrow buttons (>>/ <<), you can move them to the Used list.

Used

Used

Displays the currently used list of columns. Using the arrow buttons (>>/ <<), you can remove them from the Used list. Using the Up/ Down buttons, you can arrange the columns in the required order.

Up

Down

Enables you to customize the columns order using the following buttons.

Button       Description
Up Moves the selected column up one line. In the Hole Table, the column shifts left one space.
Down Moves the selected column down one line. In the Hole Table, the column shifts right one space.

Down

Down

Enables you to customize the columns order using the following buttons.

Button       Description
Up Moves the selected column up one line. In the Hole Table, the column shifts left one space.
Down Moves the selected column down one line. In the Hole Table, the column shifts right one space.

Title

Title:

Enables you to specify the text to be shown in the Title section of the respective column. You can insert TML syntax to represent a symbol along with the title description.

Width

Width:

Enables you to specify the maximum field width for the column. The entered column width will be considered only when the value is greater then the minimum width required for inserting complete data without any data loss.

Align title

Align title:

Enables you to assign the alignment (Left, Center, Right) to the title text.

Align data

Align data:

Enables you to assign the alignment (Left, Center, Right) to the column data.

Decimal digits

Decimal digits:

Enables you to choose how many decimal digits to be displayed for the Diameter, X and Y values. Hence, this option is grayed out for other columns.

Property

Property:

Enables to insert a property class (UDC-User Defined Class) name to assign the property value in the respective column.
