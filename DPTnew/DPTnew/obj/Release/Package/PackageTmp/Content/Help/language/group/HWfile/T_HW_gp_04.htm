Closing Selected Groups and Views

After starting the Close Group command, select all the groups or views you want to close.

  • To select single non-shared groups and/or views, click on them.
  • To select the non-shared groups and/or views enclosed in a rectangular area, click within the drawing other than on an entity and drag the cursor; the think3 application goes into automatic window selection mode.

As soon as a non-shared group or view is selected, its color changes (see "How entities belonging to a group are displayed").

Close Children check box

  • When the check box is selected, the Close Group command on parent group will close all the child groups inside it.
  • When not selected, only the parent group is closed.

Alternatively, you can apply the command directly:
— from the Drawing Structure by closing the corresponding icon or
— selecting the Close all button on the top right corner of the Selection List.

If you try to apply the Close Group command to the Current Group, you will get a warning message. You can answer Yes or No:

Yes The Current Group will be reset and the groupor view will be closed. If you select the Don't prompt this warning again check box, when you try to close the Current Working Context the message won't be displayed anymore: the Current Group will be reset and the group or view closed without informing you.
No The Current Group will not be closed, but all other open non-shared groups and views will close.
