Converting models to and from the Wavefront format

From Wavefront to think3

  • The result of the conversion is one or more Mesh entities (for details, see "Display information about a selected mesh")

From think3 to Wavefront

  • As a result of the conversion you will get a Wavefront file. Standalone curves and points will be ignored. Surfaces and solids need to be transformed into meshes prior to creating the file, this is performed by using each object's current tesselation. It can be set either individually via the Tessellation category of the Entity Properties or generally using the tessellation slider.
