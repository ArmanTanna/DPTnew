Enables you to determine Horizontal text position for the Radius dimensions. You can choose one of the following options from the drop-down list:

Centered inside Enables you to position the text centerd inside the arc/circle.
Inside arc Enables you to position the text inside the arc/circle.
Outside arc Enables you to position the text outside the arc/circle.
Follow cursor Enables you to position the text based on the cursor position.
Free Enables you to position the text in any position.
