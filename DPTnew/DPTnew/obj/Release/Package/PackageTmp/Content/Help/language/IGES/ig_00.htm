Conversion to and from the IGES format

The Initial Graphics Exchange Specification (IGES) is an American national standard defining a neutral format for the exchange of digital information among CAD systems.

This document describes how to convert IGES files to the format of think3 and vice versa.

Introduction

The think3 application enables you to read and write files in the IGES format ; click here to know about the supported versions.

Conversions of 3D models and of 2D drawings are performed in different ways.

  • When you're working in the 3D environment, curves, surfaces and solids created in the think3 application can be saved to the IGES format. Similarly, curve, surfaces and solids originally saved in IGES files can be imported to think3. See "Converting 3d Models".
  • Also 2D drawing entities can be translated from IGES to think3 and vice versa. See "Converting 2D drawings".

Within the think3 application, you can do the following:

  • To convert from the IGES format to think3, use the Open or Insert From File command. In the file dialog box, under Files of type, select the type of file you wish to convert from by selecting the appropriate file format (".igs").
  • To convert from think3 to IGES, use the Save or Save As command. In the Save dialog box, under Save as type, select the type of file you wish to convert to by selecting the appropriate file format (".igs").

Also, the think3 application supports reading and writing entity names to and from IGES format. Refer to the Entity Name Support for IGES format document for details.

Conversion is controlled by the parameters of the IGES Options dialog box opened by selecting Options. You can save as many different sets of parameter values  — to be used in different conversion situations —  as you like in the Settings library.

Help on parameters controlling the conversion process
You can easily display a short description of each parameter controlling the conversion process by issuing the help request as follows:
  1. Click on the button in the top-right corner of the dialog box displayed by selecting the Options button.
  2. Click on the item of the dialog box corresponding to the parameter you need help on.
