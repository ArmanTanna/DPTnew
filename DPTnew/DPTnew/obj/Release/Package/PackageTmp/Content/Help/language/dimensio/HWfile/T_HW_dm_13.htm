Insert surface texture symbol

1  When you start the command, the program prompts you to select an entity.
2  From the Type drop-down list choose the type of symbol ( Generic, Material removal, Without material removal).
3 
  • In case of ISO 1302: 1978, specify the surface texture value in the Value box.
  • In case of ISO 1302:2002 standard, specify the following in the corresponding text boxes:
    1st requirement Enables you to specify the first surface texture requirement to be indicated at the respective position in the surface texture symbol
    2nd requirement Enables you to specify the second surface texture requirement to be indicated at the respective position in the surface texture symbol
    Manufacturing process Enables you to specify the manufacturing proces to produce the surface, for example, turned, ground, plated etc.
Note: You can also define all the options using the Organize Surface Textures dialog box.
4  Under More Options you can define the following:
Plane Enables you to choose the plane onto which you want the surface texture to be inserted (Work Plane, View, World). This option is available only in the 3D environment.
Text only Enables you to insert the surface texture symbol without selecting a reference entity to attach it to.

5 Select the Entity Properties node to open the properties dialog box to edit the surface texture properties while inserting it.
6 After you set the surface texture parameters, specify the position to insert the surface texture symbol.
