Generating a Bend Data Chart

Follow the steps given below to generate and position a bend data chart.

  1. Generate a Flat Pattern View of a sheet metal model.

  2. Invoke the Bend Data command.

  3. For the Flat Pattern View selector and chose the Flat Pattern View whose bend data chart is to be created.

  4. Click on the graphics area to position the bend data chart.

  5. To reposition the bend data chart, choose the Reset option from the context menu displayed by right-clicking on the Position selector and then click again on the graphics area to reposition the bend data chart.

  6. Finally choose the OK () button to execute the command.
