The Statusbar tab of the Customize dialog box enables you to select sets of commonly used command to be added to the statusbar, the gray band beneath the Graphics Area, so as to have them easily reachable at any time during a work session. To customize the statusbar, proceed as follows.

  1. Statusbar commands in the two working environments (Model and Drawing) can be different. So fFirst of all, in the Document type drop-down list, select the working environment (Model or Drawing).
     
  2. The Objects to be viewed pane displays a list of commonly used commands, organized by category. Select the check box corresponding to the set of commands you want to be available in the statusbar.
     
  3. Select the Show check box to display the commands in the statusbar. Clear the check box to hide the commands from the statusbar.

To change the statusbar customization, select the Reset button. On selecting the button, a confirmation dialog box affirms whether you want to lose the changes made on the statusbar and restore the default configuration. This button is active only for the current opened document. If you change the statusbar type different from the current document, the Reset button will be grayed off.
