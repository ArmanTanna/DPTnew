PARTsolutionsImport from PARTdataManager File


 

The Import from PARTdataManager File command enables you to import a CADENAS native part file in the current document.

The command is useful when you download a CADENAS part from a web server or you wish to bypass the CADENAS PARTdataManager™ to add a part or group to your document.
Make sure you select the Metafile 2D (PS2) and Metafile 3D (PS3) formats in the neutral formats section of the PARTserver Setup page.

When you start the command, the program opens a file browser dialog box that enables you to select the part to import.
Depending on the active environment, the dialog box filters the default file extension:

  • PS2 files (*.ps2) in the 2D environment
  • PS3 files (*.ps3) in the 3D environment

Once you select the file to import, the process continues as if you were running one of the following commands:

  • Import PARTdataManager Group
  • Import from PARTdataManager Catalog
