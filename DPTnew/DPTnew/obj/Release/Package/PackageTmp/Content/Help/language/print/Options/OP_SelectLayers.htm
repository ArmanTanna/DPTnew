Enables you to specify the layers of the drawing/model to be printed.

The list of layers displays the layers used in the model and it includes the layer number, description and the count of entities present in the layer.
You are enabled to:

  • select a layer by clicking the relevant line;
  • select multiple layers by pressing CTRL while selecting the lines;
  • select a contiguous series of layers by clicking the first and last while pressing SHIFT.
