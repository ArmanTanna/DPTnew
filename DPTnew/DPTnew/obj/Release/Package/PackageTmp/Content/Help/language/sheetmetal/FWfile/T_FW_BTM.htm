ToolsBend Table Manager


 

The Bend Table Manager command enables you to use the Bend Table Manager for managing the bend table data. It is possible to view, edit and add new bend tables using the Bend Table Manager.

On starting the command, the Bend Table Manager dialog box is displayed. In this dialog box, select the proper Table type to view the desired bend table data. It is possible to add, delete and copy the existing bend table data.

The Bend Table Manager provides a filter to sort out data as per requirement. On applying the filter, the Bend Table Manager displays only those data that adhere to the filter settings.
