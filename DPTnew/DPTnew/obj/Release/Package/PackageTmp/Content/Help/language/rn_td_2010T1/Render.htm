New Rendering Functionalities and Improvements

Version 2010.1 introduces new Rendering functionalities. There have been major enhancements in the usage of material and its properties. The material and color attributes are distinctly classified and they can be rendered separately on shaded entities. The process of applying the Rendering properties remain the same, but with enhanced and easy to use features. The following is a summary of all the enhancements.

  • Three rendering commands are introduced to render the model in different visualizations.
  • A default material will be associated with each model created.
  • Material is treated as an attribute and all the functionalities and location of the material attribute are similar to that of color.
  • Materials are now available in the status bar, selection filter, and edit properties dialog box.
  • All the old files will be managed with the new material and color attribute.
  • A new weight management function is available in the Part properties and Part Data. This weight value will be visible in the part list, balloons, and title block.

New Rendering Mode commands

The following three commands have been introduced to render the models in the application.

  • The Basic Rendering Mode command enables you to render the model with the model color.
  • The High Quality Rendering Mode command enables you to render the model with the advanced rendering features like environment, lighting, and decal.
  • The High Quality + Material Rendering Mode command enables you to render the model with all the advanced rendering features with the applied material. The effect of material with the environment, lighting, and decal are visualized in this mode.

These commands do not need separate license.

Note
The following commands are no longer available as the functionality of these commands are performed by the three new Rendering Mode commands.
  • ViewDisplayHigh Quality Shaded
  • ViewDisplayHigh Quality Shaded + Boundaries
  • ViewDisplayHigh Quality Shaded + Boundaries + Hidden Lines

Enhancement in the Color Map command

The Density text box has been removed from the Color Map dialog box. Also the Material column has been renamed Description.

The density of the material can be viewed and its value can be specified in the Edit Material selection list.

Enhancement in the Open File Command

The enhancement in the Open File command is to manage the old files and to add material attribute to the entities that does not have a material assigned.

Under the Load category, the Material area has been introduced in the Model file options dialog box to manage old files. Two new radio buttons Material from color and Default material are available in the Material area. These options enable you to choose the mode with which you want to load the model created in previous versions. The following are the functionality of the radio buttons:

  • Material from color: When selected each color on the model is converted to a material and applied to the entities. The following are the attributes of the material created.
    • The density of the color is taken as the density of the material created.
    • Each color is converted as new material on the model. The newly created material has the old color properties.
    • All the materials created will be listed under the Material tree in the Rendering Structure.
  • Default material: When selected, the default material will be applied to all the entities of the model that do not have material. Also the materials applied in the previous versions are retained on the entities.

    Note
    When the Default material is selected, the old model properties such as density are lost.

Enhancement in the Options/Properties-System Options dialog box

  • Enhancement in the Construction category
    In this version the method of propagation of material on models is the same as that of colors while doing Boolean operations as it used to happen for colors. To achieve this goal, the Solid Color area has been renamed Solid color/material.

    Also, the three radio buttons under Solid color/material area have been renamed :
    • First entity color/material
    • Current color/material
    • Keep face color/material
  • Enhancement in the General-Advanced category

    The AutoRename check box has been introduced in the Materials Area. This option is used to manage situations of duplication of material name or its properties in a same working session.
    • When the check box is selected, the material is automatically renamed.
    • When the check box is cleared, you are asked to enter the name of the new material.

Enhancement in the Edit Part Properties dialog box

The new Weight Calculation button has been introduced in the Edit Part Properties dialog box. The weight management will enable you to specify a weight value or automate the process of calculating the weight. When the Weight Calculation button is selected, the following options are displayed in the dialog box.

  • Weight:: text box: This text box enables you to enter the value of the weight. The Automatic check box is available adjacent to the edit box. When the Automatic check box is selected, the weight of the model is calculated by the software and displayed in the Weight edit box.
  • Weight with assembly feature:: It displays the weight of the model with the added assembly feature on the component.

Enhancement in the Status Bar

Materials can be added to the models by using the new Material button from the status bar. The selection of materials from the Rendering Library is similar to that of the colors from the status bar.

In the status bar, when you right-click on the Material button, the following are displayed in the pop up window.

  • Under the Document Materials, the material that is currently set as default is displayed.
  • Under the Current Library Materials, the materials of the current library folder are displayed.
  • On selecting the More Materials option, the More Options Materials dialog box is displayed. You can select the required material from the dialog box.

    • Also the Material drop-down list has been introduced in the Attributes area of the Selection Filter dialog box.

Enhancement in the Make Solid and Break Solid commands

The Color option has been renamed to Color/Material in the Make Solid and Break Solid selection lists. This option enables you to define the propagation of materials during the Boolean operations while using the Make Solid and Break Solid commands.

Enhancement in the Edit Properties dialog box

The Material: drop-down list has been introduced in the Edit Properties-General dialog box. It enables you to change the material of the selected entity by choosing the material from the drop-down list.

Enhancement in the Rendering Environment

The following are the enhancements in the Rendering environment.

  • The new Delete Unused Materials option is available in the context menu on right-clicking the Materials node. When you select this option, all the materials listed in the rendering structure tree that are not attached to the model are removed from the list. A warning message will be displayed when you try to delete the materials that are attached to the model and listed in the Rendering Structure.

  • The new Set as Current Material option is available in the context menu on right-clicking the specific material, listed in the Rendering tree. When this option is selected, the selected material is set as the current material. This option is also available when you right click on the material in the Rendering Library palette.

  • The new Set as Current Library option is available in the context menu on right-clicking a material folder in the Rendering Library palette. When this option is selected then the materials in the selected folder will be listed under the Current Library Materials in the status bar.

  • The new Refresh option is available in the context menu on right-clicking the UserDefined folder. It enables you to update the folder list, after you have manually customized the library folders at the installation location.

Note
The material that is set as the current material will be highlighted in bold in the Rendering Structure and in the Status bar.
