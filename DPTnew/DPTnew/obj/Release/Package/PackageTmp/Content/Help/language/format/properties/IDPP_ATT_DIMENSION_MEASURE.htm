Entity Properties

Category: Dimension-Measure

Current

Current:

Enables you to set the current style by selecting it in the list of all the available ones.

Prefix

Prefix

The controls in this area enable you to specify the Prefix for the dimension created.

Text

Text:

Enables you to choose the text from the available list or type in the text for the prefix to be placed along with the dimension.

Position

Position:

Enables you to specify the Position for the Prefix using the following options from the drop-down list:

Left Enables you to place the Prefix to the left of the dimension
Above Enables you to place the Prefix above the dimension

Suffix

Suffix

The controls in this area enable you to specify the Suffix for the dimension created.

Text

Text:

Enables you to choose the text from the available list or type in the text for the suffix to be placed along with the dimension.

Position

Position:

Enables you to specify the Position for the Suffix using the following options from the drop-down list:

Right Enables you to place the Suffix to the right of the dimension
Below Enables you to place the Suffix below the dimension

Measure

Measure

The controls in this area enable you to determine the Measure for dimensions.

Automatic calculation

Automatic calculation

Enables you to control the automatic calculation of measures in dimensions.

  • When selected, the dimension measure is calculated automatically.
  • When not selected, dimensions will display the value manually entered in the Measure box.

Measure

Measure:

Enables you to enter the values for the dimension manually or select them from the drop-down list. The Measure option will be active only if the Automatic calculation option is not enabled.

Scale

Scale:

Enables you to specify the Scale factor for the dimensions measured automatically. If the Automatic calculation option is not enabled, this option will be grayed out.

Dual dimension symbol

Dual dimension symbol

Enables you to Show dual dimensions symbol, depending on the state of the check box:
When selected, the Dual dimensions will be shown appropriately as set in the Dual settings under the Advanced category of Unit.
When not selected, the Dual Dimensions will not be shown.

Customized format

Customized format

Inside Customized format two very useful tools can be used:

  • The TML coding (think3 Text Markup Language)
  • Some specifically created symbolic notations — along with all the already existing ones — enabling the inclusion of text portions implicitly expanded according to the current property settings.



    They are:
    <MM> The complete dimension according to all current properties (that is: the TML notation emulating the whole dimension as if it were created with no custom format)
    <DM> The dual measure according to current dual measure properties
    <M> The ordinary dimension measure according to current properties
    <T> The ordinary dimension tolerance representation according to current properties
    <S> The fit tolerance symbol
    <ST> Fit symbol and tolerances
    <U> The symbol of the unit of measure for the ordinary dimension measure.

All the above mentioned notations can be automatically applied on selecting the corresponding options available in the drop-down list. Depending on the actual content of the Customized format box, then, some properties will be grayed out or not in order to very clearly display which properties are giving their contribution to the visualized dimension text and which are not.

Some Examples

  • To display the unit of measure after the measure and the tolerances, separated by some additional space (take into account the tolerance values and the dimension unit have to be entered as ordinary properties), the following code can be used:

    <M><T><SP/><U>



  • Some ready-to-use notes spreading across more than one line lines can also be used:

    2<X><SP/><FDIA><M><BR/> <CNB><SP/><FDIA>1.5<BR/><DEP>.062



  • Customizable ways to write tolerances are also available:

    52<VALIGN type="center"><STACK scale="0.60" separator="none">
    <SR> H7(<STACK scale="0.40" separator="none">
    <SR>+0.03</SR><SR><SP/>0</SR></STACK>)</SR>
    <SR> g6(<STACK scale="0.40" separator="none">
    <SR>-0.01</SR><SR>-0.029</SR></STACK>)</SR>
