Creating Copies of Solid Faces

In the following example you will use the Solid from Faces command to create the static solid corresponding to the faces to be pulled up after a parting plane has been created on a starting solid. To accomplish the task, open the ExtractSolid.e3 file available in the Samples folder of your think3 installation and follow the steps given below:

1 — Parting Plane
2 — Starting Solid

  1. Invoke the Solid from Faces command. Ensure that the Mode is set to Local in the selection list.

  2. Select the three faces in the following image. You may need to rotate the model to select the top face — see Zooming, panning and rotating)



    Note that to change the selected faces, you can right-click on Faces in the selection list, choose Reset and reselect the faces.

  3. If you want the resulting static solid to retain a link to the original, select the Linked copies check box. In this case, you will get a parametric solid.

  4. Click or to confirm your selections and create the new solid.



    Click to discard your changes.
