Modify MeshInvert

The Invert Mesh command enables you to invert the normals to the facets of a triangular mesh.

  • Select the meshes you would like to invert. Arrows are displayed to show the current orientation of the meshes.



  • In the Invert Normals drop-down list, choose the way the meshes will be inverted:
    • Manually — you need to manually invert the normals; automatic inversion is not applied.
    • By direction — normals are oriented towards the specified direction.
    • By axis — normals are oriented towards the defined axis.
    • By point — normals are oriented towards the defined point.

  • The direction of all the normals can be simultaneously inverted by using the Exchange All Normals option.


Note that if there are normals to the nodes of the meshes, they are reversed according to what is done on the normal facets.
