Inserting a Cutting Plane Line

1 When you start the Place on Drawing command, the program displays the Section List dialog box and prompts you to select a section or section name.
2 Choose the section on which you want to base the new cutting plane line from the drop-down list, and then click OK.
3 The program prompts you to select a view.
Click to select the view in which you want to insert the cutting plane line.



The think3 application immediate inserts the cutting plane line.

Note

If you select a view that is not perpendicular to the cutting plane you selected, the program displays a dialog box warning you of this condition.
Click OK and then select a view perpendicular to the cutting plane.

If the cutting plane arrows are pointing in the wrong direction, you can reverse them by using the Invert Direction command.

After inserting a cutting plane line, you can use it to create sectioned views.
