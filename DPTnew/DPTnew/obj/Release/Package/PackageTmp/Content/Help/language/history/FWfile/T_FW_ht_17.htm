EditHistoryDismantle


 
The Dismantle command enables you to isolate a selected solid within the Model Structure so that you can cancel selected events within its branch of the tree.



Note

The Dismantle command is significantly different from the Roll back command. Dismantle remains focused on a selected branch, enabling you to modify that branch, whereas Roll back affects the entire tree, thus canceling the selected event and all subsequent events in the Model Structure.
