Case 4: Make a surface pass through some curves with an additional perpendicularity constraint

The problem How to solve it

Create a 5 sided, double bumped surface. It has to pass trough the red curves and has to be perpendicular to the middle plane of the hand-grip

GSM is able to "spread" a plane on the new curve. Actually you could spread any other kind of surface. The result is excellent.
