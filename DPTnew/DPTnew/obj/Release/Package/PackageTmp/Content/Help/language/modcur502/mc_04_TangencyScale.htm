Setting tangency and modifying both curves: changing the scale factor

This example shows the effect of changing the scale factor and leaving all the other values unchanged.
 
  1. Repeat steps 1 to 9 of the previous exercise, (Setting tangency and modifying both curves: weights)
     
  2. Use 3 as the scale factor and leave all the other values unchanged:
     
    • Scale: 3
    • 1st Weight: 1
    • 2nd Weight: 1
    • 1st Propagation: 0
    • 2nd Propagation: 0.5

      Compare the results obtained:

      1st Weight 
      Weight 2: 1 
      1st Propagation: 0 
      2nd Propagation: 0.5
