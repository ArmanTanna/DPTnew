The Organize Settings command enables you to manage the settings in the library that you can recall with the Settings command.

The command displays the Organize Settings dialog box that lists the existing settings and enables you to perform the following actions using the respective buttons:

  • Add a New setting to the library.
  • Load the currently selected settings in the library.
  • Rename the currently selected settings in the library. This command is disabled when a system dependent setting is selected.
  • Delete the currently selected settings in the library. You will be asked for confirmation. This command is disabled when a system dependent setting is selected.
