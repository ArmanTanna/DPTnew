Markers

The options in this area enable you to control the display and placement of the markers.

Display format

Display format

This area enables to select one of the three display formats, which will get appended to any specific marker value provided in the respective Hole Definition file.

A B C D

A1 B1 C1 D1

Enables to choose the following display formats:

A B C D When selected, displays only letters in a sequential order (A-Z; AA, AB-AZ; BA, BB-BZ etc).
1 2 3 4 When selected, displays only number value in an incremental order.
A1 B1 C1 D1 When selected, display the letters along with the Hole Table number in an incremental order of the letters.

Offset

Offset:

Enables you to specify the value (greater then zero) to be used for the placement of the Marker label with respect to the default position; applied only when the marker is outside the circle.
