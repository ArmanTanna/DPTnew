Dimension Format - Precision

Note
All the options in this page are enabled only when the checkboxes associated with them are selected.

Precision Enables you to specify the precision level for the dimensions that you create. You are enabled to specify the precision in one of the following three ways:

Linear Enables you to specify the linear precision limit for the dimension
Angular Enables you to specify the angular precision limit for the dimension
Dual Enables you to specify the precision limits for the dual dimensions as well.
