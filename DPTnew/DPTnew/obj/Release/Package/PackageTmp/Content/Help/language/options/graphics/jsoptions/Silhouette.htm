This option controls the visualization of silhouettes within the program.


The silhouette visualization mode is based on tessellation. In Shaded modes, the program has to perform tessellation computation. In Wireframe mode the program generally skips tessellation computation. Once the program has determined the tessellation, the silhouette mode is readily available.See Wireframe and Shading.


Default The silhouettes are visualized only in Shaded modes, in which mode, the program has to perform tessellation computation. So when this option is selected, silhouettes will not be not visualized in the Wireframe mode.
None The silhouettes rendering is disabled altogether, in any mode.
Always Force the program to render silhouettes every time, even in Wireframe mode, in which mode the program generally skips tessellation computation.
Note, however, silhouettes are not rendered during dynamic transformations (i.e.; Pan, Zoom, Rotate operations).

Silhouette

The silhouette of a surface is normally composed of:

  • The apparent contours of the surface — the curves defined as the sets of the points at which the normal to the surface is parallel to the view plane.
  • The real boundaries of the surface — the isoparametric curves with the lowest and highest parametric values, or in the case of surfaces with parameter limitations, the limiting curves which do not coincide with other boundaries of the surface.
  • The edges of the surface (if any) — the isoparametric curves defined as the sets of the points where the surface has only positional continuity.
For further details, see Silhouette.
