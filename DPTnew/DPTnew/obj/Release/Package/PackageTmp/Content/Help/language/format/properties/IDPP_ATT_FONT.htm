Entity Properties

Category: Text-Font & Frame

Current:

Current:

Enables you to set the current style by selecting it in the list of all the available ones.

Font

Font

The controls in this area enable you to set various parameters for the Font.

Standard:

Standard:

Enables you to specify the TrueType Standard font to be used to create text. Select one of the fonts in the drop-down list.

Alternative:

Alternative:

Enables you to specify Alternative fonts such as Vector fonts when the TrueType Standard font is not specified. Select one of the fonts in the drop-down list.

Size:

Size:

Enables you to specify the text Size.

Aspect:

Aspect:

Enables you to determine the aspect applied to the current font:

Bold Text will be displayed in bold (for example: TEXT). This option applies only to True Type (Standard) fonts.
Italic Text will be displayed in italic (for example: TEXT). This option applies both to True Type (Standard) and vectorial (Alternative) fonts.
Underline Text will be underlined (for example TEXT). This option applies both to True Type (Standard) and vectorial (Alternative) fonts.

Effects:

Effects:

Enables you to determine the orientation to be applied to the text characters:

Upside down Enables you to mirror the text characters about the horizontal axis.
Backward Enables you to display the text characters in the backward direction.
Vertical Enables you to display the text characters vertically.

All the above mentioned options can be applied both to True Type (Standard) and vectorial (Alternative) fonts.

Frame

Frame

The controls in this area enable you to select the color and the type of the text frame.

Color:

Color:

Enables you to specify the color to be used to draw text frames.

Type:

Type:

Enables you to determine the Type of Frame used to display text. You can choose one of the following options:

None
Circle (1c)
Circle (2c)
Circle (3c)
Circle (4c)
Rectangle
Rounded rectangle
Rectangle with broken sides
Rectangle with a broken side
Underlining
Arc over text
Break note
Round brackets
Square brackets
Triangle
Rectangle with dashes

Dimensions:

Dimensions:

Enables you to specify the Dimensions to be provided for, for the texts within Frames.
