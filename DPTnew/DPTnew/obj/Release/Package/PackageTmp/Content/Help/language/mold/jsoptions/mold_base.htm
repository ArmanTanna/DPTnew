The Plates tab enables you select the values for the different plates of the mold base. The values available are corresponding to the series selected in the Basic Size tab.

The Guide Elements tab enables you to specify the values for the guiding elements that support the Mold Base assembly. The guide elements enable you to align the core and cavity plates to the required position. The pictorial representation enables you make effective selection of values from the drop-down list.

The Fasterners tab enables you select the values for the fasterners of the mold base assembly. The fasteners hold the Mold Base assembly clamped to the base.

On the left pan of the wizard, the total configuration of the selected values of the mold base assembly is displayed.

There are options to manage the configuration of the parameter settings in the above dialog box.

Save Configuration It is used to save the current parameter settings of the Mold Base Wizard in a configuration file.
Open Configuration It is used to open and load a configuration file which contains the details of the parameter settings of the Mold Base Wizard.
OK When selected, the Mold Base is created for the configuration selected in the Mold Base Wizard.
