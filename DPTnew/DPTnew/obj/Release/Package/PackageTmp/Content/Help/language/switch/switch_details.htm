Switch tool - Further details

Once you've opened the switch tool (see How to use the switch tool for details), the switch tool dialog box displays the following information:

Keep Open The Switch dialog will close after you click OK. To change the configuration you must start the program again. Click this button to change the setting so that the Switch will stay available after running.
Close The Switch dialog will remain open after you click OK, thus enabling further configuration changes. Click this button to change the setting so that the Switch will close after running.
Help Opens the Switch help page.
Current installation Displays the think3 application's current installation menu path. The information is displayed for your convenience.
Select one of the following installations Lists the different think3 application installation menu paths. Note that when you activate the switch tool, the selected version in the list is the default (current) one.
OK Runs the selected installation and/or the configuration that you have defined. The Switch may close or remain available depending on the Keep Open control.
Cancel Closes the dialog and quits the Switch.
Refresh Updates the selection to the default installation.
Advanced Opens an additional dialog appendix that enables you to set the following options:
Unit Enables you, once you've already installed the think3 application, to redefine for the next launch the default unit of measure for drawings, independently from the one set in the installation.
Actions Enables you to set the following required operations:
Register Automatically updates the Registry Editor. This action is required when switching between different think3 application versions or languages.
Run Runs the selected installation and/or the configuration that you have defined. Note that if this is different from the current one, and you did not check the Register action, you will receive an error message when you click OK.
Launch options Enables you to set the following launch options:
Language Enables you to define the language used in the think3 application, by selecting it in the drop-down list.
Other Reserved for internal use only.

Note

The MS-DOS window being displayed while updating the registry just informs you that the registration process is running.
