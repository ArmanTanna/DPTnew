Visual Bookmarks: Filter and Filter Options

The Filter box on top of the Visual Bookmark Tree Panel can be used to input bookmark names and/or keywords to hide and show Visual Bookmarks.

  • Using the Filter
  • Setting the Filter Options

 

Using the Filter

To facilitate this type of Visual Bookmark browsing, the interface allows for keywords to be entered for each Visual Bookmark (see Creating/modifying a Visual Bookmark).

For example, you may wish to show only those visual bookmarks that have keywords of "ANALYSIS AND STRESS NOT THERMAL". Typing in this Boolean string and hitting the ENTER key will invoke the search. Only those visual bookmarks having the keywords "analysis" and "stress" but not the keyword "thermal" would become visible. All other visual bookmarks that did not satisfy the search would be hidden. The hidden visual bookmarks would not only be missing from viewport but their icons in the tree would also be grayed out. This search example requires that the following items be checked in the Filter Options dialog box: Process all, keyword. The Hide option must also be unchecked.

You may choose to reuse a previously entered search string by selecting the desired string in the drop-down list.
