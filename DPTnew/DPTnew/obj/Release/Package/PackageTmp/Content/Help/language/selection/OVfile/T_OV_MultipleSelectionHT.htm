Selection on the Model Structure

Any item in the Model Structure can be selected by following the rules of pre-selection and post-selection. In other terms, you can either first select the item in the Model Structure and then apply the command, or first select the command and then select the item to which it will be applied.

In addition, multiple selection on the Model Structure is enabled, both for events and for geometric entities, as well as multiple selection in pre and post selection.

In multiple selection, when a homogeneous selection is done, the commands applicable to that selection are available through the Model Structure context menu.

These commands include:

  • Deactivate Feature
  • Reactivate Feature
  • Go to Reference
  • Delete Feature
  • Light Representations
  • Unload Component
  • Reload Component
  • Delete

In case of homogeneous selections, also the Delete key performs the deletion. If the selection is not homogeneous, a specific warning is displayed and no deletion is performed.
