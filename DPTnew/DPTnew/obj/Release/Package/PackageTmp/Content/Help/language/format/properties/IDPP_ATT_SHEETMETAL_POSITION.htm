Sketched bend

Sketched bend

The drop-down list in this area enables you to control the alignment for sketched bends.

Alignment:

Alignment:

Enables you to select the alignment for sketched bends; the first three cases refer to the solid after bending, the last one refers to the unbent solid:

Inner flangethe flange is completely internal relative to the profile position
Outer flangethe flange is external relative to the profile position
Outer flange and bendthe bend and the flange are external relative to the profile position
Bend linedetermines the position of the bend line in the developed solid. The last option can be used to evaluate the solid after bending, starting from the 2D drawing of the solid development with the bend position

Extruded flange

Extruded flange

The controls in this area determine the positioning of Extruded flanges.

Gap:

Gap:

Enables you to specify the gap value to be added between the two extruded flanges.

Alignment:

Alignment:

Enables you to select the alignment for extruded flanges:

Inner flangethe flange is completely internal relative to the selected edge
Outer flangethe flange is external relative to the selected edge

Flange

Flange

The controls in this area determine the positioninig of flanges.

Alignment:

Alignment:

Enables you to select the alignment for flanges:

Inner flange The flange is completely internal relative to the selected edge
Outer flange The flange is external relative to the selected edge
Outer flange and bend The bend and the flange are external relative to the selected edge
Bend line The bend line passes through the selected edge

Length:

Flange Length

Following are the methods by which you can control the length of the flange and can calculate the unbend length.

Inner Enables to calculate unbend length of the flange from the inner face of the flange.
Outer Enables to calculate unbend length of the flange from the outer face of the flange.
Exclude bend Enables to calculate unbend length of the flange excluding the bend radius.
Bend line Enables to calculate unbend length from the bend axis of the flange.
Inner tangent and Outer tangent Enables to calculate unbend length of the flange from the inner tangent and outer tangent. The blue and red lines represent the inner tangent length and the outer tangent length of the flange, respectively
