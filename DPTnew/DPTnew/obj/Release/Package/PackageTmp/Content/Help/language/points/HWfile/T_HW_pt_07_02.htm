Project points orthogonal to a surface

1 Open the file called pt_07_01.e3.



In this example, you are explained about how to project on a plane the points into which a curve has been divided.
2 To do this, start the Project command and you are asked to select the points you want to project.



3 End input.
4 Then select the surface onto which you want to project the points selected.



5 Again, end selection.
6 From the Direction drop-down list, you can select one of the three projection methods: Normal to the surface selected or in any other direction which you want to set.
7 If you decide to have a Normal projection, the projection points are drawn on the surface.
