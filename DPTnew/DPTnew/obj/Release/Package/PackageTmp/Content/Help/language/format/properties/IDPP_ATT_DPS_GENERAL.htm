Document Properties

Category: Drawing-General

General

General

The controls in this area determine general parameters such as the distance between drawing views.

Description

Description:

Contains a description of the drawing. (Note that this same information appears in the Description box on the Document Data tab of the File Properties dialog box).

Distance between views

Distance between views:

Determines the minimum distance between automatic and derived views when they are inserted into a drawing. (Note that if you change the position of a derived view, this minimum distance is maintained if the derived view is Aligned to main , but in that case, the distance is based on the bounding box surrounding the view. In addition, if the size of the model decreases, the views to not move, but if the size of the model increases, the views are moved in order to maintain the minimum distance).

Sheet size related

Sheet size related

When selected, the Distance between views value causes the minimum space between views to be based in proportion to the diagonal dimension of the drawing sheet; when cleared, the Distance between views value specifies the actual distance between views, measured in the current units.

Enable automatic view rearrangement

Enable automatic view rearrangement

When selected, the program is permitted to rearrange the view based on the size of its bounding box when the underlying model has been changed, or when the view definition has been changed.

Next view/section name

Next view/section name

The controls in this area determine the how the name of the next view or section is assigned and whether that name is visible or not.

Show view name

View name visibility

When selected, the name of the next view is displayed within the new view when you insert it into the drawing. To move the text, first enable writing in the view and then use the Move/Copy Entities command.

Show separator

Show separator

Enables you to include a separator symbol in a drawing view notation.

  • When selected, the separator symbol (—) will be displayed in between the view name and its scale factor value.
  • When cleared, the separator symbol will not be displayed.

The Show separator check box is enabled only when the Show view name and Show view scale check boxes are selected.

Show view scale

View scale visibility

When the check box is selected, by default all newly created views will display their own scale factor in the label. In Edit Properties, the Show view scale box will be checked by default (and you can uncheck it to turn off the visualization of the view scale factor).
When not selected, which is the default value, all newly created views will not display their own scale factor in the label, unless you check Show view scale in Edit Properties.


Note
If you load a drawing file (".e2") created with a previous version, the indications of scale factors in the view labels will not be changed (either in format or content). The new scale values will assume the default values.

Automatic names

Automatic names

When selected, the name of the next view is defined automatically using sequential letters (drawings are labeled using lower-case letters, sections in upper-case); when cleared, you can specify the name of the next view and section.

Start from section

Start from section:

Determines the section name to start with when automatically defined. When Automatic names box is checked in, the section name will start with whatever value you specify in the text box.

Next view

Next view:

Determines the name of the next view when not automatically defined. When the Automatic name definition check box is cleared, if no view or section names are specified in the Next view or Next section boxes, if you create a new view or section, the program displays a dialog box, warning you that the view or section name will be defined automatically.

Next section

Next section:

Determines the name of the next section when not automatically defined. When the Automatic name definition check box is cleared, if no view or section names are specified in the Next view or Next section boxes, if you create a new view or section, the program displays a dialog box, warning you that the view or section name will be defined automatically.

Next view

Next view

The controls in this area determine the scale of the next view you create by using any of the Insert=>Drawing Views commands.

Scale

Scale:

Enables to specify the scale for the next view you create.
If you enter only one number — say x — the actual scale assigned by the program will be x:1. Each former scale value will be available in the drop-down list.
