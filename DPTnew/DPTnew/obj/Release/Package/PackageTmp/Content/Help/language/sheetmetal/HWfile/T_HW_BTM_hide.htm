Hiding and Unhiding data in the Bend Table Manager

The following illustration explains the procedure to hide and unhide bend table data in the Bend Table Manager:

1. Start the Bend Table Manager command from the Tools menu; the Bend Table Manager dialog box is displayed.
2. Select the rows that contain the data to hide.


  • To select a single row, click on the row number.
  • To select multiple rows, hold the CTRL key down while making the selection.
3. Right click on the selected rows to display the context menu. Select the Hide option.



The bend table data in the selected rows get hidden.
4. To unhide the hidden data in the Bend Table Manager, right click on a row and select the Unhide All option from the context menu.
5. Choose the OK button to save the changes and close the Bend Table Manager.
