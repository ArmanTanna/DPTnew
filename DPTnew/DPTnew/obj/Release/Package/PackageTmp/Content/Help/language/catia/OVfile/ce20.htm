An Overview on TD Catia v4 interface

TD Catia v4 interface is an external translator with dedicated commands integrated in the application by means of a specialized toolbar that enables you to export and import CATIA® files. Click here to know about the supported versions.

Note
The TD Catia v4 interface imports and exports 3D data, while the transfer of 2D data is limited only to the import operation.

The typical workflow includes the setting of the translation options, and the actuation of the conversion through the use of the commands available to load a CATIA® model or save the current model into a CATIA® model file.
