STEP Format: Standard for the Exchange of Product Model Data

The STEP format (Standard for the Exchange of Product Model Data, the familiar name for ISO 10303) was created in order to provide a representation of product information and all that is necessary to enable product data exchanging: the aim is to have a single and better standard, covering all aspects of the product life cycle in all industries.

STEP is an alternative to IGES in all CAD/CAM/CAE applications and systems. Its main goals are completeness (both for exchanging and archiving), extensibility, efficiency (reduced file size and needed computer resources), compatibility with other standards, no redundancy (only one way to represent a concept), independency (no hardware and software constraints).

The structure of this International Standard is described in ISO 10303-1.

The Application Protocols (parts of the International Standard which specify an application interpreted model satisfying the scope and information requirements for a specific application) regarding the conversion to and from think3's format are:

  • AP203 — (Configuration-Controlled Design) an application protocol concerning the exchange between application systems of configuration-controlled 3D designs of mechanical parts and assemblies.
  • AP214 — (Core Data for Automotive Mechanical Design Process) an application protocol concerning the exchange of information between the applications that support the development process of the mechanical aspects of automotive vehicles.
