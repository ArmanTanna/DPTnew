Display information about a selected Fillet Surface

If you select a Fillet surface, the program displays a dialog box showing:

  • The Surface type box, displaying "Blend surface"
  • Flags indicating if the surface is Closed, Planar
  • If the surface is trimmed, the Trimmed check box will be selected and the Limiting curves Limiting islands boxes will show the corresponding values.

Arrows within the model are also displayed indicating the two parametric directions.

If you select the Area check box at the bottom of the dialog box, the program also displays the surface area.
