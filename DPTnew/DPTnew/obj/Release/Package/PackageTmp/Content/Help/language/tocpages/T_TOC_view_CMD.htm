View

Redraw All Redraws all of the active views.
Previous Restores the current view to previous state.
Next Restores the view that was displayed prior to using the Previous command to restore the previous view.
Work Plane Toggles the display of the Work Plane.
Grid Off Disables both the Static and the Hidden Dynamic Grid.
Hidden Dynamic Activates a hidden grid through which you can limit the values which can be assigned to a geometric item.
Static Grid Displays a grid whose nodes can be used as a reference to help you enter points with the mouse.
Grid Properties Displays the Grid category of the Document Properties containing grid settings.
Modify List of the commands used to modify the views.
Orientation List of the commands controlling the view orientation.
Display List of the commands controlling the display mode.
I/O Window Toggles the display of the alphanumeric Input/Output window.
Hide Entities Enables you to hide selected objects.
Unhide Entities Enables you to display hidden objects.
Hide Inner Entities Enables you to hide entities inside model/assembly.
Show Base Surface Enables you to display the hidden base surface of a curve of the Curve On Surface type.
Hide Base Surface Enables you to hide the base surface of a curve of the Curve On Surface type.
Show Driving Dimensions Enables you to display one or more drive dimensions.
Hide Driving Dimensions Enables you to hide one or more drive dimensions.
Highlight Dimensions Redraws the dimensions on all views, enabling you to immediately appreciate the dimension colors you set.
Fit Texts to View Resizes all text in the view based on the current size of the view.
Highlight All Parts Shows the components to which part data have been assigned.
Highlight All Parts without Balloon Shows the components to which no balloon has been assigned.
Properties Displays the View categories of the Document Properties containing view settings.
Align to Target Re-aligns constrained views after you have moved the target point in the free view.
Synchronize Re-aligns the other constrained views after you have changed one of them.
Virtual Mirror Activates the virtual mirroring mechanism on the current view.
No Mirror Deactivates the virtual mirroring mechanism on the current view.
Orthogonal Projection Changes the projection mode on the current view to an orthogonal projection.
Perspective Projection Changes the projection mode on the current view to a perspective projection.
Set Ground Moves the ground plane based on the current view of the model.
