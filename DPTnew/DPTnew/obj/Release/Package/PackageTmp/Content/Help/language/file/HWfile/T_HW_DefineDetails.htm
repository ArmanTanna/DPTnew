Define Drawing Frame and Title Block Details

When you click the Settings button, the Settings dialog box is displayed. You can then specify the following information:

Load frame fitting current drawing When selected, views are drawn full-size and the application automatically chooses the proper Drawing Frame
Resize drawing to selected frame When selected, you can specify the paper size and orientation:
  • Size: — determines the sheet size when you select Resize drawing to selected frame. Choose the sheet size by selecting one of the predefined sizes from the drop-down list, or select Free and then specify the sheet size by entering the size (in current units) in the X and Y boxes.
  • Orientation — determines the orientation of the drawing sheet when you select Resize drawing to selected frame and then choose a size other than Free from the Size: drop-down. Choose one of the following to determine the orientation of the specified sheet:
    Landscape Specifies a horizontal sheet orientation
    Portrait Specifies a vertical sheet orientation
  • Frame: — select from the drop-down list the Drawing Frame to apply to the drawing. The content of the list depends on the size and orientation specified.
  • Title block: — select from the drop-down list the Title Block to apply to the drawing.
Ask for document data for title block When selected, the program displays the Document Properties dialog box when inserting the Title Block so that you can supply information (such as document name and description) that will appear in the Title Block and will also be associated to the layout.

When you have finished defining the Drawing Frame and Title Block information, click OK to return to the Layout Configuration dialog box.

Note: If you included a Drawing Frame and Title Block and selected the Ask for document data for title block option, when the new drawing is actually created, the program displays the Document Properties dialog box so that you can enter the document name, description, and comments.
