Document Properties

Category: Grid-Reference

Reference grid

Reference grid

The options in this area control display of the Reference Grid.

AutoAdjust

AutoAdjust

When selected, the major and minor lines of the Reference Grid automatically adjust their subdivision with respect to the current zoom level, while performing a zooming operation.
When cleared, no adjustment takes place and the Reference Grid might not be visible under extreme zoom conditions.

Origin axes

Origin axes

When selected, the origin axes of the Reference Grid are displayed boldly with special color to make them prominent.
When cleared, the origin axes of the Reference Grid are not displayed.

On top

On top

When selected, the Reference Grid is visible even if it is positioned behind the model.
When cleared, the Reference Grid is visible only at places where it is above the model.

Size

Size:

The value in this drop-down list determines the percentage extension of the Reference Grid with respect to the bounding box limits. Its value varies from 100% to 300% in the increments of 50%.

Labels

Labels

Text labels with the numeric values of the major lines are displayed on all the four sides of the graphics area when the Reference Grid is being viewed orthogonally. The options in the this area control the display of labels in the Reference Grid.

Enable

Enable

When selected, the labels are displayed on the Reference Grid.
When cleared, the labels are not displayed on the Reference Grid.

Axis name

Axis name

When selected, the name of the axis (X, Y or Z) is displayed in the labels of the Reference Grid.
When cleared, the name of the axis is not displayed.

Font size

Font size:

Specify the font size of the labels in this edit box.

Digits

Digits

Options in this area control the display of measurement units and the extent of abbreviation of the labels.

Display units

Display units:

When this radio button is selected, the drop-down list becomes active. Depending on the value in this drop-down list, the values of the labels get divided by 1, 10, 100, 1000 or 10000 and only the integer part is displayed in the labels.

Measurement unit

Measurement unit

When this radio button is selected, the unit of measurement is displayed in the labels.
