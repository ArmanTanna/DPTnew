Creating points on a curve

After starting the On Curve command ( On Curve/Surface, if you are working in the 3D-Model environment),proceed as follows.

1 Select the curve on which you want to create the point entities.
2

After choosing the curve, the Points option enables you to select the point creation method:

  • By parameter
  • By distance
The Points Along U option enable you to specify the number of points to be placed, on the selectetd curve.
3 After specifying the number of points, specify the initial and the final positions of the u parameter in the U mini dialog box (remember that the u parameter varies within the range [0,1]).
