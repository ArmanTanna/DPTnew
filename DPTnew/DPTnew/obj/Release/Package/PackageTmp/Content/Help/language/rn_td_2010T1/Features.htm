New Part Modeling Functionalities and Improvements

Version 2010.1 introduces the following enhancements to the Part Modeling functionalities.

  • The new Rib command
  • The new Cut/Split command
  • Enhancement in the selection of Work Plane axes in solids/features
  • Enhancement in the Interactive Modeling Move Faces and Interactive Modeling Offset Faces commands
  • Enhancement in the Invert Solid Normals command

The new Rib command

The new Rib command enables you to create a thin wall structure on the component to increase the strength of the whole component. The thickness of the rib is controlled by the Thickness mini-dialog box available in the Graphics Area. The Offset mini-dialog box is available to create a rib at an offset from the profile. Also, the Angle mini-dialog box is available to create a rib at an angle. The following images show an example of a simple rib creation.

The new Cut/Split command

The new Cut/Split command replaces the previously existing Cut command. The new command enables you to cut and split a solid by using a surface, datum plane, or skin.

The Split check box is provided in the Cut/Split Solid selection list.

  • When selected, the selected solid is split by the cutting surface.
  • When cleared, the split operation is not performed.

Cut and split are two separate parametric entities displayed in the Model Structure.

Enhancement in the Interactive Modeling Move Faces and Interactive Modeling Offset Faces commands

The new Fillet Reconstruction drop-down list in the Interactive Modeling Move Faces and Interactive Modeling Offset Faces commands enable you to manage G1 continuous fillets more effectively. Three new options, None, Only Safe, and All are available in the Fillet Reconstruction drop-down list. Also, you can offset a selected fillet surface, without changing the fillet type.

The fillet is reconstructed by evaluating the tolerance value entered in the Fillet Recognition Tolerance text box. Based on the tolerance value, the software computes whether the reconstruction or offset operation can be done to the selected fillet surface or not.

Enhancement in the Invert Solid Normals command

Using the Invert Solid Normals command you can now apply a global inversion to all normals to make them point towards the same direction. As a result, you do not have to select the normals individually to invert their direction. The inversion takes place based on one of the options selected in the new Inversion Mode drop-down list, which is available when the Mode is set to Local. Refer to the Enhancement in the Invert Surface command topic for some illustrations.

The direction of all the normals can be simultaneously inverted by using the new Exchange all normals option.
