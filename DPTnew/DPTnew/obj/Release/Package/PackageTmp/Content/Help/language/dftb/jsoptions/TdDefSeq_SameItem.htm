This option enables you to assign the same number to components that have the same Part number:. When you check Same item position for same part number, in fact, the program recognizes any multiple occurrences of a part and assigns the same sequence number.
On the contrary, when you uncheck it, the program does not recognize any multiple occurrences of a part and assigns a different sequence number to all occurrences. This means that the part list includes a row per single part occurrence.
