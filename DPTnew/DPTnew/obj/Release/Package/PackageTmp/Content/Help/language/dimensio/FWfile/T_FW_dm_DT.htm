InsertDimensionDatum Target

The Datum Target command enables you to insert a datum target.

When you start the command, the selection list displays the following options:

Reference Entity Enables you to select a reference entity to which you want to attach the datum target.
Position Enables you to specify a point/segments of a leader line to position the datum target.
Start from Reference Point Enables you to determine whether the Datum Target will start from the reference point.
  • When selected, the leader line will start from the reference point selected.
  • When not selected, the leader line will start from the first point selected.
Max. No. of Points Enables you to define maximum number of points (less than or equal to 5) that make up the leader line. Although you are always enabled to right-click and insert the datum target with fewer points, when you have entered the required number of points the program inserts the datum target.
Diameter Symbol Enables you to determine whether a diameter symbol is to be included or not.
  • When selected, the diameter symbol (ø) will be included along with the text in the upper portion of the datum target.
  • When not selected, the diameter symbol will be omitted.
Upper Text Enables you to specify the text in the top portion of the datum target.
Lower Text Enables you to specify the text in the bottom portion of the datum target.
Entity Properties Displays the Entity Properties dialog box that enables you to edit the datum target properties.

The following option is available under More Options.

Plane Enables you to choose the plane onto which you want the datum target to be inserted (Work Plane, View, World). This option is available only in the 3D environment.
