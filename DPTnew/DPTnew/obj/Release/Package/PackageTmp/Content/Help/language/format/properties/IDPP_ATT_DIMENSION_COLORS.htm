Entity Properties

Category: Dimension-Color

Current

Current:

Enables you to set the current style by selecting it in the list of all the available ones.

Colors

Text

Arrowheads:

Enables you to determine the Colors for various parameters of the dimensions created. You can select a color for all the following parameters of a dimension:

Text Enables you to determine the Colors from the pull-down list for the dimension text
Tolerances Enables you to determine the Colors from the pull-down list for the dimension tolerances
Arrowheads Enables you to determine the Colors from the pull-down list for the dimension arrowheads

Lines

Dimension

Use current

Enables you to determine the color for Dimension and the Witness lines. Here you have the option to either set a color of your choice using the pull-down list or Use current by selecting the check box.
When the Use current check box is selected, you can use the default color that is used to create all the other dimensions.
When the Use current check box is not selected, you have the option of setting the color of your choice using the pull-down menu.
