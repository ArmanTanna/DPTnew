How to customize the Environment

Settings Wizard

The Settings Wizard allows you to set the style for dimensions and texts, unit of measurement, the look and feel of the toolbar buttons, icon styles and GUI profiles easily by following a wizard driven interface.

Note:
If you are starting the Settings Wizard while the think3 application is already running, the new customizations will be reflected only when you close and restart the think3 application.

Start the Settings Wizard and select the settings as you are taken through the different pages of the wizard. Click Next to proceed to a next page and click Back to go back to a page. Click Finish once at the last page of the wizard to save your settings. Note that you can always rerun the wizard with changed options to create a new setting. Further, the settings created by the wizard can be changed using the System Options and Document Properties/Entity Properties.

System Options

The System Options categories enable you to customize the think3 application by changing the values of program's various parameters and choosing options that match the way you prefer to work with the program.

Document Properties/Entity Properties

The Document Properties and Entity Properties categories enable you to control document properties (such as, for example view modes and units of measures) and entity properties, such as for example color, line type, and line width (line weight) and others like as dimension colors, dimension orientation and position, arrowheads and so on. See Properties (basic concepts) to learn more.
The program comes with several predefined format and style sets designed to meet ISO, JIS, GB, KS and ANSI standards. You can create and save format and style sets for different types of models/drawings or to adhere to other drafting standards and conventions, and reactivate any of them later as the current style. See Styles and Formats for further details.

Keyboard shortcuts

You can customize keyboard shortcuts from the Shortcuts tab of the Customize dialog box.

GUI Profiles

You can customize GUI profiles from the GUI Profiles tab of the Customize dialog box.

Commands and Toolbars

You can determine which toolbars are displayed, customize existing toolbars, and create your own toolbars. See Customizing toolbars for further details.

Statusbar

You can customize the statusbar by adding commonly used commands to it. To do so, use the Statusbar tab of the Customize dialog box.

Drawing Frames and Title Blocks

You can create your own customized Drawing Frames and title blocks.
