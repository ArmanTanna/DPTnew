Balloon

Balloon

The controls in this area enable you to govern the parameters controlling balloons.

Text

Text:

Enables you to enter the Text that appears inside the Balloon.

Frame

Frame:

Enables you to determine the type of Frame used to display Balloons. You can choose one of the following options:

Circle (2c)
Circle (3c)
Circle (4c)

Rounded rectangle

Rectangle with a broken side
Underlining
Arc over text
Break note
Round brackets
Square brackets
Triangle
Rectangle with dashes

Angle

Angle:

Enables to specify the Angle at which the text is positioned. The frame associated to Balloon text also orients along the direction of the text. The angle specified here is measured relative to x-axis and in counterclockwise direction. The default angle is 0.

Label

Label

The controls in this area enable you to define the label setting.

Linkage

Linkage:

Enables you to determine the position of the witness line with respect to the Label. Select one of the Linkage options from the drop-down list:

First line Enables you to link the witness line to the First line of the Label.
Center Enables you to link the witness line to the Center portion of the Label.
Last line Enables you to link the witness line to the Last line of the Label.

Text

Text:

Enables you to enter the actual Text for the Label.

Multiline

Multiline...

Click this button to open the Multiline Text dialog box, enabling you to create multiline text labels. Then, type in your text, that can be:

  • Plain text, in which case:
    • TABS are ignored
    • Blanks are real blanks
    • Hit ENTER to open a new line
  • Text formatted according to the think3 application Text Markup Language. The maximum length, including TML tagging, is 256 characters.

Multiline

Multiline

The controls in this area enable you to determine various options for the Multiline text.

Line spacing

Line spacing:

Enables you to specify the Line spacing for multiline text, as a percentage of the height of the characters.

Angle

Angle:

Enables to specify the Angle at which the text is positioned. The frame associated to Balloon text also orients along the direction of the text. The angle specified here is measured relative to x-axis and in counterclockwise direction. The default angle is 0.

Wrap text

Angle:

Enables to specify the Angle at which the text is positioned. The frame associated to Balloon text also orients along the direction of the text. The angle specified here is measured relative to x-axis and in counterclockwise direction. The default angle is 0.

Enable

Width

Enable

Enables you to control multiline text wrapping.

  • When selected, multiline texts are wrapped around the corner according to the width of each bounding box and/or to the current Width value.
  • When not selected, multiline texts are not wrapped.
