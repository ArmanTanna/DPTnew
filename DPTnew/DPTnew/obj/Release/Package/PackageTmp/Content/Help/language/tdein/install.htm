Installation of ThinkDesign

To perform an installation, run the installation program () from the medium you were provided with. In the wizard that will show up, select Products to access the installation procedure.

Once you have chosen to install ThinkDesign, two installation options are available:

  • Typical
    that will install ThinkDesign Suite enabling you to use ThinkDesign or ThinkDesign Tooling. The online documentation will also be installed. 
  • Custom
    that will enable you to install also all the optional modules as described in ThinkDesign Suite.
    • ThinkDesign Suite
      to be used as a graphical editor: this option will enable you to use ThinkDesign or ThinkDesign Tooling. See "System Requirements" for additional information.
    • TD Catia v4 interface
      to exchange data with CATIA® V4
    • TD Direct
      to exchange data with CATIA® V5, Parasolid V11 (Save) and V16/17 (Open) and Pro/ENGINEER® Wildfire.
    • MicroScribe tool
      the MicroScribe-3D product line enables you to construct detailed computer models by simply tracing the surface features of real physical objects.
    • Coordinates Label
      provides a command to be used to insert labels displaying the coordinates of the selected points.
    • I-interact
      a real-time design platform specifically aimed at the design review phase, providing high control on complex design data through most friendly free hand sketching tools.
    • UConnect
      provides integration among ThinkDesign UDRIVE®, I-Interact and BUNKSPEEDTM UDRIVE®.
    • Compensator
      provides you with all the tools you need to face and solve spring back compensation issues and other Finite Element Analysis compensations, such as warping, shrinkage and so on.
    • ANSYS plug-in
      to integrate ANSYS Workbench 11.0 SP1 in the think3 application.
    • Renderer
      for advanced, high impact life-like and high quality rendering.
    • Reshape
      a reverse engineering solution to import and manage files obtained through 3D digitizing operations, so as to enable fast prototyping, design review, finite element analysis and inspection.
