Modify user defined property to entity

To perform the exercise follow the instructions given below.

  1. Start the Modify UDP command; the Select class(es) to be modified dialog box is displayed.
  2. Select one or more properties to modify the values. To select all the properties in the list, click on All. To deselect a property, click on it again.
  3. Click on the entities you want to modify (the list is restricted to properties which have been assigned to the entity chosen).
    Or click on Select Model to modify the properties assigned to the model.     A dialog box is displayed for assigning the new values.
  4. For each property selected, type the new value in the Value box.
  5. Click on Modify.
  6. Close the property box by clicking on Cancel.
