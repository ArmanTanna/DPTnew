Interface items and graphics interaction

The user interface enables designers to focus their attention mainly on the model drawing and perform any of the possible actions in the most natural way.

The main components of the user interface are:

  • Selection List
  • Mini-Dialog boxes and Handles

Such items make the use of commands very natural and easy to understand, even in those cases in which a number of parameters are required.

In addition to this, the selection highlight mechanism helps you better understand what is happening with your selection. When you have several entities to be selected, it might not be so easy to distinguish what you have already selected from what you are selecting now or you still have to select. The program helps you by highlighting already selected elements with the Alt. selection highlight color, while the current selection is highlighted using the Selection highlight color. Both of the said colors can be changed in the Environment Colors of the Colors and Materials command.
