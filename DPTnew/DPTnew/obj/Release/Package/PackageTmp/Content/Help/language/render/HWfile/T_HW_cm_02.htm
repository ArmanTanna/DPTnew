Chrome

A color source providing simple chrome-like reflections.
The source has a base color, specified by Base Color, which is mixed with bands of colors based on the orientation of the surface relative to a direction specified by Vector.

A Mix value is used to specify the ratio of mixing of base color with the reflection colors. This value should be in the range 0 to 1, where a value of 1 indicates all base color and no reflection colors.
