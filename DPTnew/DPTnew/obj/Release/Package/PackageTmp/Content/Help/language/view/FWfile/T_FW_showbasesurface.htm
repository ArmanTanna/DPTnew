The Show Base Surface command is available in the View (Single Commands) category of the Commands tab of the Customize dialog-box, or in the context menu displayed by right-clicking on a curve of the Curve On Surface type.
It enables you to display the hidden base surface of a curve of the Curve On Surface type.

  • When started from the command button, once you have selected the curve, its base surface will immediately be displayed.
  • When started from the context menu, as soon as you select the command the base surface will be displayed.

Please note that if you move the cursor onto a curve of the Curve On Surface type whose base surface is hidden, a dotted outline of the latter will be displayed.

The capability of controlling the visualization of base surfaces can be most useful especially when working with complex models, when you might need to temporarily hide or how a base surface in order to have a better understanding of the model itself.

To hide the base surface of a curve of the Curve On Surface type, select the Hide Base Surface.
