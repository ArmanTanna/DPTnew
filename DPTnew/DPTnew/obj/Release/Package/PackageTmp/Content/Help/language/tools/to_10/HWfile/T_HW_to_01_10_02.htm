Customize GUI Profiles - Define a new GUI profile

1 Display those toolbars you wish to be displayed in the new GUI profile.
2 On the GUI Profiles tab of the Customize dialog box, click New.
The program displays the New GUI Profile Name dialog box.
3 In the Profile name: box, type the name you wish to assign to the new GUI profile.
4 Click OK.The GUI profile is created and the name added to the list.

Note
The new profile can be edited further by removing or adding the toolbars or buttons. The changes will be stored in that particular profile. Also you can add customized toolbars to the selected profile.
