When modifying a non-associative curve or surface, the options of this drop-down list determine how the program shall apply the change:

Keep the program creates a copy of the original entity and applies the changes to the copy. The original entity remains visible.
Hide the program creates a copy of the original entity and applies the changes to the copy. The original entity is hidden.
Modify the program applies the changes to the original entity.


1 —  Modify
2 —  Keep

Please note that this option does not apply to associative entities, which are always actually modified when they undergo a change, whatever the option value may be.
