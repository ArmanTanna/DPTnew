When selected, the program asks whether specialized curves and surfaces should be converted into NURBS curves and surfaces.

Please note that when more than one entities are to be converted within the same command, a specific option (OK to All) will enable you to confirm conversion for all of them with just one click, without having to confirm each time.
Also note that if you select No the command will be interrupted.

Conversion to NURBS curves and surfaces can occur when using the following commands.

When modifying curves:

  • Curve Continuity
  • Curve Flexibility
  • Extend Curve
  • Modify Curve Shape Constraint
  • Modify Curve through Control Points
  • Modify Curve through Interpolation Points
  • Through Point Curve

When modifying Surfaces:
  • Join Surface
  • Modify Surface Shape Constraint
  • Modify Surface through Control Points
  • Normal to Degenerate Surface
  • Surface Continuity
  • Surface Flexibility
  • Invert Surface

When not selected, the conversion occurs automatically.
