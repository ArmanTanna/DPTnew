Rendering

Rendering is the process of generating an image from a model. A rendered image is a description of three dimensional objects containing geometry, viewpoint, decal, texture, lighting, environment and shading information. To facilitate rendering, three commands are available in the application. The effect of rendering is slightly different in all the rendering commands as mentioned below.

Preview enables you to quickly render the model.
Ray Tracing enables you to render the model as a fully shaded and textured, antialiasing image.
Final Gathering enables you to render a model that yields extremely realistic rendering effects.

The image can be rendered in the Graphics Area or in a separate window. Also you can directly save the image to a specified location in your local machine. All these settings can be optionally selected using the options in the System Options- Rendering category.
