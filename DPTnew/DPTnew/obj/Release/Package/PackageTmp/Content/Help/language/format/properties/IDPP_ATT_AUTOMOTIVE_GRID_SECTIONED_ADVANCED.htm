Entity Properties

Category: Reference Grid-Sectioned Format-Advanced

Major lines

Show label

Enables you to define the display properties of the major lines in the reference grid of a sectioned view.
You can define following properties by selecting from the pull-down list:


Note
Please note that all the following properties apply only to the sectioned grid lines which are created and visible only if the Show sectioned lines check box is selected in the Reference Grid category.

Color Enables you to specify the color to be used to create major line
Line type Enables you to specify the line type to be used to create major line
Width Enables you to specify the line width to be used to create major line

Show label
  • When the check box is selected, the label is displayed attached to the major line.
  • When not selected, the label is not displayed.
