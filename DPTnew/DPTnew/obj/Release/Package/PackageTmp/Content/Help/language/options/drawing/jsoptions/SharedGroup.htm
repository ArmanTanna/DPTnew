The options in this area enable you to specify if shared groups inserted into the drawing from a library (see "Inserting a Group from a Library into your Drawing") and the instances of those added to a library by dragging and dropping (see " Adding a Group to a Library") will be:

Local Locally defined: the shared group and the entities that it contains are stored directly in the drawing.
X-Reference Group X-Reference: the shared group is the think3 application's 2D file and the drawing contains a link to this file.
