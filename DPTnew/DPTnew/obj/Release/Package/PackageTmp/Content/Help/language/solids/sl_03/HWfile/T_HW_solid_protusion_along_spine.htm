Creating a Solid by Sweeping a Boundary Profile along a Spine

The Generic Solid command enables you to create a solid by sweeping a boundary profile along a spine path by using the Along one plane type motion mode, as shown in the image below.

Profile and drive curve Generic Solid

To obtain the above result, you need to open the solid_single_profile.e3 file available in the samples folder of your installation folder and follow the steps given below.

  1. Start the Generic Solid command to display the selection list and you will be prompted to select the spine (drive curve) for path transition.

  2. Select the drive curve for the Spine selector, as shown in the image below.



  3. Now, click on the Boundaries selector in the selection list and select the profile you want to sweep from the Graphic Area, as shown the image below.



    To reselect the profile, right-click on Boundaries in the selection list, choose Reset, and then select required boundary profile.

  4. Next, select the Along one plane option from the Motion Mode drop-down list of the selection list.

  5. To create a shelled solid with a specified thickness, select the More Options to display the 1st Thickness mini-dialog check box. Select the check box to extend the edit box and enter 5 in the edit box.

  6. To create a solid sweep feature you need clear the 1st Thickness check box.

  7. Click or to confirm your selections and create the solid/shelled feature, as shown in the image below.

    8. 1st Thickness check box cleared 1st Thickness check box selected

Click to discard your changes.
