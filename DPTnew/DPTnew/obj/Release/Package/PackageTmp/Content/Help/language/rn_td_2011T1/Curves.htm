New Surface Functionalities and Improvements

Version 2011.1 introduces the following enhancements to the Curve & Surface commands.

  • Enhancement in the Capping command
  • Enhancements in the Linear Surface and Parting Plane commands
  • Enhancement in the Super Capping command

Enhancement in the Capping command

When using the Capping command to create a filling surface, when the Generic Shape drop-down list under More Options is set to Existing surface, it is now possible to select more than just one surface. In fact as soon as the Existing surface option is selected, the Surfaces selector shows up in the selection list, thus enabling you to select several existing surfaces so as to better specify the shape you need.

Enhancements in the Linear Surface and Parting Plane commands

Starting from this version, if you interrupt the said commands, your selections and settings are not lost: they are "frozen" in order to enable you to use them again without having to repeat them all, which might take a long time, at least in complex situations.
Next time you start the command, in fact, the Restore Selection button ( ) will be displayed on top of the selection list. If you select the Restore Selection button ( ), all the selections you made and the settings you defined last time you used the command will be restored, enabling you to go on with no need to repeat those selections/settings.
When the command is started with some pre-selected entities, if you click the Restore Selection button you will be asked to confirm restoring the frozen command selections, thus resetting the current selection. It is not possible to restore the frozen selection after the first use of the preview.

Enhancement in the Super Capping command

The new Symmetry check box in the Super Capping selection list enables you to define the symmetry conditions for the capping surface.

  • When the check box is selected, you will get several additional options to define the symmetry plane, to be chosen in Reference Plane.
  • When the check box is cleared, no symmetry criteria are applied on the capping surface.
