InsertMoldDesignSplit Volume

The Split Mold Volume command enables you to extract certain volume of material either from the core or cavity. This is used in situations where the undercut faces cannot be created on the core and cavity. The extracted material creating a slide core is called an Insert.

On starting the command, you can select the core or cavity for the Part selector from which the material is to be extracted.

The Cut-off Insert drop-down list is used to select the method of extracting the material. The two options are:
By profile The insert is created using a closed 2D profile.
By solid The insert is created using a predefined solid which can be created using the standard solid commands.

Note
It is mandatory that the profile completely encloses or the solid completely engulfs the undercut feature of the core/cavity that has to be extracted.

The type of insert created on the core/cavity can be selected from the Type drop-down list under the More Options. On selecting the following options;

Cut only The insert is created as an X-Reference component, without making any modification in the core and cavity.
Slot only Only a slot is created in the core/cavity to make space for the insert. No separate insert is created.
Cut and slot Both the insert (X-Reference component) and the slot for insert are created. The created insert is the part in red in the image.

The Extension option is available under More Options only when the insert is created using the By profile method. The extrusion of the profile is created by selecting the options from the Extension drop-down list. On selecting the following options;

Single depth The profile is extruded in a single direction and the value can be specified in the Depth mini-dialog box (or by using the corresponding handle). If necessary, the profile can be extruded with an angle and offset using the Offset and Angle check box available on the Graphics Area.
  • Offset: When selected you can specify the offset distance from the profile surface by entering the value in the mini-dialog box or by using the corresponding handle.
  • Angle: When selected you can specify the angle by entering the value in the mini-dialog box or by using the corresponding handle.
Double depth The selected profile can be extruded in both the direction of the profile plane. The values can be entered in the Depth 1 and Depth2 mini-dialog boxes.
