The Import command enables you to add a new catalog or index of components to thinkparts.

This command is located in the context menu available when right-clicking on an index listed in the browser frame of the Catalog Browser or the Catalog Manager.

Upon activation, it presents you with a standard Windows dialog for locating the catalogs to be imported, with extension TPC. After locating the file, press the Open button, and thinkparts will start loading the catalog. A progress bar will keep you updated on the activity. After the catalog has been successfully loaded, you will be able to use the Catalog Browser to navigate through its components or the Catalog Manager to customize it.
The new indexes loaded by this operation will be located just under the index selected before running the command.

You are also enabled to import a catalog by double-clicking on the .TPC file in an explorer window. In this case, the imported catalog will be positioned at the first level in the catalog tree.
If you have the thinkparts Catalog Manager or Catalog Browser open, you may have to reload the tree structure to access the newly imported catalog.

After the catalog has been successfully imported, use the drag & drop feature of the browser tree frame of the Catalog Manager to set its position in your thinkparts catalog tree structure.
