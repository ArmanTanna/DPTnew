Managing the Evolutions

Once an evolution has been added, right-click on its name to perform the following activities on it:

Rename the evolution
  1. Right-click on the evolution and select Rename Evolution from the context menu.
  2. The name of the variable becomes editable; type the new name.
  3. Click on the blank area under the variable name to accept the change and exit the edit mode.
Delete the evolution
  1. Right-click on the evolution and select Remove Evolution from the context menu.
  2. The selected evolution gets deleted along with its variation(s).
Edit the evolution
Not available for Camera evolution
  1. Right-click on the evolution and select Edit Evolution from the context menu.
  2. Depending on the type of evolution selected for editing, the same Animation - Evolution selection list is displayed that was used to add the evolution.
  3. Make the changes and then select the OK ( ) button to continue.
