Delete a class of user defined properties

1. Start the Delete UDC command; the Select class(es) to be deleted> dialog box is displayed.
2. Select the class you want to delete in the class list which is displayed automatically.
3. Click on Delete; the class is immediately deleted from the list.
4. Click on Close to close the dialog box.
Note

To remove a property assigned the model/drawing itself, you can proceed as described in "Delete a class and remove its assignment from the model/drawing", in the Custom tab of the File Properties dialog box.
