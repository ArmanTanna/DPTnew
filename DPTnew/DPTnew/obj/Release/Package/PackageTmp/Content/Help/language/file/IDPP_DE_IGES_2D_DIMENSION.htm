IGES

Category: 2D-Dimension

Dimension

The controls in this area determine general aspects of the conversion of dimensions

Invert arrowheads in angle dimensions

When selected, corrects positioning of arrowheads (entity 214, Leader or Arrow)

Dimension arrowhead scaling

When selected, applies scale factor for the dimension measurement to the arrowheads as well; when not selected (default), the arrowheads are not scaled.

Disassemble angle dimensions

When selected, converts angular dimensions (entity 202, Angular Dimension) as a set of the component entities rather than as dimension entities; when not selected (default), Angular Dimensions are converted to angular dimensions.

Disassemble linear dimensions on loading

When selected, converts linear dimensions (entity 216, Linear Dimension) as a set of the component entities rather than as dimension entities; when not selected (default), Linear Dimensions are converted to linear dimensions.

Loading modes

Determines the conversion of IGES dimension entities, by selecting the appropriate setting from the drop-down list

Arrowheads

Determines the position of arrowheads, by selecting the appropriate radio button

Follow text position

Positions arrowheads in accordance with the dimension text position (internal if internal, external if external)

Keep original position

Arrowheads maintain the position of their Leader (214) entities
