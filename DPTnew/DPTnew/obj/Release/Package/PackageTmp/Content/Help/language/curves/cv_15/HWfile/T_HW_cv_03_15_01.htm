Laying down a curve on a surface

1 Open the file "cv010315.e3". The following model is displayed:
2 Start the Project Curve command.
3 Select the curve to be laid down.



End selection by right-clicking and choosing Continue in the context menu. To change the selected curve, right-click on Curves, choose Reset in the context menu and select another curve.
4 In the Method drop-down list, select Laying down. The Surfaces item is displayed in the selection list.
5 Click on the surface onto which you want to lay down the curve.



End selection by right-clicking and choosing Continue in the context menu. To change the selected surface, right-click on Surfaces, choose Reset in the context menu and perform selection again.
6 Specify the reference point on the curve.



To change the selected point, right-click on Reference point under Curve Reference Data, choose Reset in the context menu and select another point.
7 Specify the reference point on the surface.



A preview is displayed.



To change the selected point, right-click on Reference point under Surface Reference Data, choose Reset in the context menu and select another point.
8 Click or to confirm your selections and create the curve.



Click to discard your selections.
