The Import from report CSV command enables you to add the values contained in a ".csv" file to the selected catalog.

This command is located in the dialog box of the Edit Values command.

To generate a template file with the correct columns layout, use the CSV Report command.

The second column in the ".csv" file contains the row number. To change the order in which the rows are displayed by thinkparts, change the row numbers accordingly.

After editing the ".csv" file, import its contents with Import from report CSV.
Before importing the new values, the program resets the current table. This means that the table of values after the import operation contains only the values as you saved in the ".csv" file: the lines you remove are deleted; those you edit are updated and those you add are appended in the position according to the index value in the second column of the file.

In order to get a correct format of the Microsoft Excel file, the CSV format Separator: parameter in the Report category of the System Options-thinkparts category must be aligned with the List separator parameter in the Number tab of the system Regional Settings that are displayed from Control Panel.
