Enables you to specify the minimum number of characters to be used for the coordinate value.
If the value displayed in a label has fewer digits than this value, as many blanks as required are added so as to reach it.
This option can be very useful if you want all the circles to have the same size, since the radius of the circle is proportional to the number of characters inside the label.


Note
This applies only to the sectioned grid lines which are created and visible only if the Show sectioned lines check box is selected in the Reference Grid category.
