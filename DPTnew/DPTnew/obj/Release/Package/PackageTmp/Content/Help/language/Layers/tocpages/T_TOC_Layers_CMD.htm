Layers

Current Layer Sets the layer of a selected entity as the current one
Layers Manages layers in a document (drawing/model)

Command Buttons

New Layer Creates new layer
Set Current Layer Sets the selected layer as current
Delete Layer Deletes the selected layers
Filter Layers Displays only the layers that meet the selected filter criteria
Select Layers Selects only the layers that meet the chosen selection criteria

Context Menu Commands

Select Entities Selects entities
Move Entities Moves entities from one layer to another
Exchange Layers Exchanges the layer content
Rename Layer Changes the name of the selected layer
Refresh Refreshes the layer list by regenerating its contents
