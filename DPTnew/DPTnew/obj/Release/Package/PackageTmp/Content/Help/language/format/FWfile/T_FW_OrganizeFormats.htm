FormatOrganizeFormats


 

Organize Formats is a collection of all the essential attributes that are most commonly used. This command enables you to change all the attributes at once. You can create your own format and store it in the library, so that you can use it readily.
The dialog box features the following categories and sub-categories that enable you to manage the relevant entity formats:

  • Dimension
    • Prefix/Suffix
    • Tolerance
      • Value Expression
    • Precision
    • Symmetric
    • Custom Text
  • Balloon
    • General
  • General
    • General

All categories feature the following commands that enable you to create new formats and manage existing ones:

  • New Format
  • Set Format as Current
  • Remove Selected Format

The context menu of the available formats, moreover, features the following commands:

  • Set as Current
  • New
  • Remove
  • Rename

Note
If you select an entity first and then choose a format, the new format applies only to that particular entity, thereby overwriting the current properties. However, if you choose a customized format when no entity is selected, it will affect all the entities that will be created.
