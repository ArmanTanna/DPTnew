Aligning a View

1 When you start the Align command, you are prompted to select the reference view. Click to select the view with which you want the view being moved to align.
2 The program then prompts you to select the view to modify. Click to select the view you want to move into alignment with the reference view.



As soon as you select the view, the program aligns it with the reference view.



You can then select another reference view.
