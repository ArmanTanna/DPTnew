Converting models to and from the PLY format

From PLY to think3

  • The result of the conversion is a Mesh entity (for details, see "Display information about a selected mesh")

From think3 to PLY

  • As a result of the conversion you will get a PLY file. Standalone curves and points will be ignored since the generated PLY file contains exclusively meshes and components. Surfaces and solids need to be transformed into meshes prior to creating the file, this is performed according to the current values of the PLY Conversion Options dialog box opened by selecting Options.
  • If there are problems with the objects inside the generated file, check the log file of the conversion.
