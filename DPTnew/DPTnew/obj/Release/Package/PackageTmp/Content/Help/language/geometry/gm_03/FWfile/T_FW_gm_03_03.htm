InsertDraftingCircle and ArcRadius


 
The Radius Circle command enables you to create a circle or arc:
  		• based on its Radius or Diameter (circles only — depending on the option you select in the Option drop-down list) and center point, if the Given drop-down list is set to Center
  		• based on its Radius or Diameter (circles only) and two through points, if the Given drop-down list is set to Point.

    Inside the profile environment, the command works only with radiuses and not with diameters.

    When working in the model (three-dimensional) environment, the resulting circle and arc entities are created on the Work Plane.

    Select the Centerlines check box to automatically draw the centerlines when you create the circle or arc.
