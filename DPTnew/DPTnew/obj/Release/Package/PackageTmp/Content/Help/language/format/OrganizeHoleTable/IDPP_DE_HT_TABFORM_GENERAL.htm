Organize Hole Table -

Category: Table Format

General

Width:

Enables you to control the following properties of the table:

Layer: Enables you to determine the layer to be used to insert the table. You can make a new selection from the editable drop-down list, which can either choose an existing layer or type in the name of a new one to create it.
Note: It is not possible to modify or rename the default layer 0.
Color: Enables you to determine the color to be used to draw the table. To modify it, click on the corresponding down arrow to display the list; make the new selection in it.
Line type: Enables you to determine the line type to be used to draw the table. To modify it, click on the corresponding down arrow to display the list; make the new selection in it.
Width: Enables you to determine the width to be used to draw the table. To modify it, click on the corresponding down arrow to display the list; make the new selection in it.
