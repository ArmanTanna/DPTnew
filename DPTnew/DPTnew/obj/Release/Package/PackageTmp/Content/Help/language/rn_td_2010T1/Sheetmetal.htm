New Sheet Metal Functionalities and Improvements

Version 2010.1 introduces the following enhancements to the Sheet Metal functionality.

  • Improvement in the Step Bend command
  • Enhancement in the Flange command
  • Improvement in the Sheet Metal Wizard command

Improvement in the Step Bend command

It is possible to create steps on a twisted bend surface, as shown in the images below.

Twisted bend surface Step bends created

A message is displayed in case the specified relief diameter is greater or lesser than the required relief diameter.

You need to set the Circular option in the Type: drop-down list of the Entity Properties-Sheet Metal-Corner Reliefs category before applying the Step Bend command on the twisted bend surface.

Enhancement in the Flange command

It is possible to create a flange on the edge having adjacent non-perpendicular side faces. The Adjacent Normal Side Face check box has been introduced in the Flange selection list.

  • When this check box is cleared, the angle of the adjacent face is retained. Also, the side face of the flange is created with the same angle, as shown in the images below.

  • When this check box is selected, the adjacent inclined side face to the selected edge becomes perpendicular to create the flange, as shown in the images below.

Note
The Adjacent Normal Side Face check box is displayed only when the sheet metal model contains inclined side faces.

Improvement in the Sheet Metal Wizard command

With the improvements made in the Sheet Metal Wizard command, the Middle and Full type of rips are now supported for more cases.
