How to Go On

In order to complete the execution of the command, you have to:

  • Supply all the input arguments that are marked in red in the selection list:
    If there are more than one "group" to be selected, you can select the first and omit the others. The markers of arguments whose input is done turn to green.
  • Confirm the command execution selecting
    OK () or Apply ()

Before confirming, you always can:

  • Change the selected entities
  • Change the values of arguments you have already entered
  • Set the values of optional arguments.
