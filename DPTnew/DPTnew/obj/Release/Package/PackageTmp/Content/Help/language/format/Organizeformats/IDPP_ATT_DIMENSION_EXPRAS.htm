Dimension Format - Value expression

Note
All the options in this page are enabled only when the check boxes associated with them are selected.

Value expression Enables you to specify the tolerances as either Deviations or Limits. Using the check boxes, you can choose the Value expression as either of the two types. The preview area, provided on the right hand top portion of the dialog box, enables you to have an updated view and also control the effect of the changes made. After you choose the tolerance type, you can specify the mode, measure, upper and the lower values using the following options:

Mode: Enables you to choose whether to specify the tolerance values as Deviations or as Limits
Measure: Enables you to select the type of dimension the Mode choice will apply to: Linear or Angular
Value above: Enables you to determine the upper tolerance value.
Value below: Enables you to determine the lower tolerance value.
