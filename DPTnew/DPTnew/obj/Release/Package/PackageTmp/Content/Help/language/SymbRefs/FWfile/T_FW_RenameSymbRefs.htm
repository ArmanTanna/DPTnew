The Rename command enables you to rename an existing Symbolic References.

The command is available from the context menu displayed when right-clicking the selected Symbolic References in the Model Structure.

You also get the command from within the View Symbolic References command, when you right-click on a Symbolic References mini-dialog box.

When you apply the command, you enter in edit mode and are allowed to modify the Symbolic References name.

In actual fact, in compliance with Windows Explorer standards, you can rename a Symbolic References also by positioning the selection cursor over its name in the Model Structure and typing it.
