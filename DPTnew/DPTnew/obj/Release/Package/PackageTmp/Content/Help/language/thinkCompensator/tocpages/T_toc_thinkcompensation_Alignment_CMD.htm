Alignment Commands

Pre Alignment Enables you to align two distant meshes using 4 user defined point couples.
Optimize Alignment Once the meshes are brought closer to each other, their alignment can be further optimized using this command.
Contour Alignment Enables you to optimize the alignment along curves and hence, can be useful in cases where the correspondence is about boundaries.
Replicate Alignment Enables you to replicate the transformation generated by the previous commands to other entities.
