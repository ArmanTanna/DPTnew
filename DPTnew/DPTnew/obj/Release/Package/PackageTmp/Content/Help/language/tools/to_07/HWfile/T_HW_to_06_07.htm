Choose units of measurements

To change the units of measurement, click the Units button.

The program displays the Mass Properties Measurements Units dialog box.

You can set the units of measurement as saved to a file.

For length, you can choose one of the following units of measurement:

  • Millimeters
  • Inches
  • Microns
  • Centimeters
  • Meters
  • Microinches
  • Mils
  • Feet

For weight, you can choose one of the following units of measurement:

  • grams
  • kilograms
  • pounds

After setting the desired units, click one of the following buttons:

  • OK—to apply the selected units only to the current calculation
  • Save—to save the settings o that they apply to all subsequent calculations until changed again
  • Reset—to return the settings to their previous values
  • Default—to restore program's original settings
