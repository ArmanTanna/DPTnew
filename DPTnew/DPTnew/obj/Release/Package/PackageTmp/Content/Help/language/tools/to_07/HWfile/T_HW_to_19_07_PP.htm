Checking continuity between two points on two surfaces

After starting the Check Continuity command proceed as follows:

  • In the Type drop-down list select Point on surface - Point on surface.
  • Click on the point on the first surface and then on the one on the second surface:



    A handle is displayed on both of them:

    Drag the handles to the desired positions (or type the proper values into the U and/or V mini-dialog boxes)

The program immediately displays the following data concerning the two surfaces along the selected borders:

  • The distance between the two points
  • The angle between the tangent planes to the surfaces in the selected points
  • The percentage value of the maximum curvature difference
  • The percentage value of the minimum curvature difference at the point of maximum curvature difference.
