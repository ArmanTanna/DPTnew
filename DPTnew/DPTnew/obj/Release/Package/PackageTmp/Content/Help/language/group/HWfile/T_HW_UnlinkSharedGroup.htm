Unlinking a Shared Group

After starting the Unlink Shared Group command, proceed as follows.

The Unlink Shared Group command can also be started by right-clicking on a shared group in the drawing or on its icon in the Drawing Structure and selecting Unlink in the context menu.

1 Select the shared group you want to unlink. It can be either a locally defined shared group or an X-Reference group. To change the selected group, right-click on Group, select Reset in the context menu and select another shared group or X-Reference (groups).

2 In the Name box, type the name to be assigned to the group you are unlinking (default is "sharedGroup#" where # is a sequence number automatically increased by the program each time a new shared group is created using New Group or unlinked using Unlink Shared Group).
3 Click or to confirm your selections and create the group. Click to discard your changes.
